# 静命论师——生平、贡献与历史地位

## 身份概要

| 项目 | 内容 |
|------|------|
| **梵名** | Śāntarakṣita（寂护/菩提萨埵/摩诃萨埵绕卡达） |
| **别名** | 静命、慧命、普明、萨霍堪布 |
| **出身** | 印度东方匝霍（萨霍，今孟加拉国一带）国王珠拉真之次子 |
| **家族** | 大太子达玛燃匝，小公主曼达绕 |
| **出家** | 那烂陀寺，亲教师为有部智藏论师，法名摩诃萨埵绕卡达 |
| **所属** | 中观自续派·瑜伽行中观（开创者） |
| **住世** | 据藏地史料约九百年（学术界认为约105岁） |
| **历史定位** | 与莲花生大士、赤松德赞并称"师君三尊" |

## 佛经中的授记

| 经典 | 授记内容 | 性质 |
|------|---------|------|
| 《文殊根本续》 | "名谓巴者宣净戒"——授记名字首字 | 间接授记 |
| 《楞伽经》 | "导师名智慧，开显五所知，大勇士现世" | 间接授记——以五法涵摄大乘 |
| 《三摩地王经》 | "末法浊世菩萨勇士者，护持如来教之此胜法" | 完整授记全名"摩诃萨埵绕卡达" |

麦彭仁波切指出：这两部经中作授记的原因，正是因为《楞伽经》和《三摩地王经》的教义由静命论师大力弘扬。

## 四种超胜功德

### 一、智慧超群

- 通达五明（声明、因明、工巧明、医方明、内明）
- 在那烂陀寺成为出类拔萃的亲教师，制胜一切外道辩论对手
- 亲见文殊菩萨并获宝剑灌顶——获得砍断一切邪说的智慧利刃
- **外道故事**：南方一位击败内外道所有辩论对手的人前来挑战。至尊者住处，只见文殊菩萨像，不见人影。被告知尊者就在那里。再返回方见尊者——知其已获本尊悉地，当即放弃辩论，皈入佛门。
- 到达藏地后对赤松德赞说："在因明辩论方面，恐怕当今天下再没有比我更擅长的了。"

### 二、戒律清净

- 在印度众多持戒大德中如妙高须弥山王——印度公认的"持戒王""僧王"
- 使者调查时人们说："他每天该忏悔的一直忏悔，该修持的也一直修持，始终不耽误。"

### 三、成就卓越

麦彭仁波切从两个角度论证成就：

| 角度 | 证据 | 推理方式 |
|------|------|---------|
| **内寿自在** | 住世约九百年（等待赤松德赞出世） | 已获长寿持明果位 |
| **外境自在** | 桑耶寺开光时赤松德赞亲见其现为文殊金刚相 | 已获外境自在 |
| **果因推理** | 在谁也无法调伏的藏地成功弘法 | 若非大成就者，不可能在如此困难之地弘法成功 |

堪布由此类推法王如意宝：在文革后极其困难的环境中成功弘法，同理必是大菩萨。

### 四、品行高尚——弘法利生

主要指弘法利生的事业（见下节）。

## 在藏地的弘法事业

### 第一次入藏

- **时代**：拉托托日国王时期
- 带《诸佛菩萨名称经》来到藏地
- 赤松德赞尚未出世、授记时间未到，返回尼泊尔

### 第二次入藏

1. **因缘**：赤松德赞21岁欲修桑耶寺，国师酿·登珍桑波以智慧眼观察到印度东方的萨霍堪布
2. **入藏**：被请至藏地红岩山（布达拉宫附近）
3. **最初传法**：对赤松德赞等传讲十善、十二缘起、十八界（《俱舍论》内容）
4. **遇障**：藏地鬼神障碍严重，显现上无法调伏。审时度势建议迎请莲花生大士
5. **返回尼泊尔**后，再次与莲花生大士同行入藏

> 历史学家分析：尊者菩提心已成熟，实际可以调伏，但**显现上不能降伏的目的是为了在藏地弘扬密宗**——莲花生大士的入藏因缘由此成立。

### 藏地弘法成就

| 事业 | 内容 |
|------|------|
| **修建桑耶寺** | 藏地第一座寺院——与莲花生大士一道考察地势、开光 |
| **开创出家僧团** | 预试七人（以贝若扎那为首）——藏地第一批出家人 |
| **培养翻译家** | 培养108位翻译家，建立译经院 |
| **传讲佛法** | 以讲经说法与听闻方式抉择佛经论典密意 |
| **制伏苯教** | 通过理证制伏苯波教——使之徒剩虚名 |
| **建立戒律** | 在藏地建立起别解脱戒的清规戒律 |

阿底峡尊者后来赞叹："你们前译派教法之兴盛，即使在印度也极为罕见。"

### 圆寂遗嘱

三条遗嘱：
1. **法体安放缘起**：面朝印度→僧少但戒净；不朝印度→僧多但戒杂。最终大臣选择后者。
2. **见解之争的预案**：当出现争执时迎请莲花戒论师——后来果然应验于摩诃衍辩论事件。
3. **亲笔信函**交付莲花戒。

## 学术贡献：中观瑜伽行派的开创

### 核心创见

将大乘两大宗轨合二为一：
- **胜义**：依《中观根本慧论》的胜义理——万法空性
- **名言**：依《释量论》的名言理——唯识安立

此前虽有圣解脱军等从唯识角度解释中观教义，但未以丰富教理建立自宗、驳斥他方。静命论师是第一个以"确凿可靠的理证"系统建立此宗的人。

### 护持者

**印度**：狮子贤、莲花戒（嘎玛拉西拉）、法友、佛智、圣解脱部等

**藏地**：鄂大译师、夏瓦秋桑、荣敦秋吉（因明前派代表），以及宗喀巴大师师徒

### 自续派东方三大论师

静命论师、智藏论师、莲花戒论师——分别著有《中观庄严论》《二谛分别论》《中观光明论》。

## 前世因缘

据《巴协》记载：静命论师、莲花生大士、赤松德赞、绕萨囊四人前世为建造尼泊尔夏绒卡绣佛塔的四位劣种者之子，共同发愿在藏地弘法：

| 前世身份 | 发愿内容 | 今世对应 |
|---------|---------|---------|
| 养鸡者之子 | 在藏地成为国王弘法 | 赤松德赞 |
| 养猪者之子 | 以神变力降伏邪魔外道 | 莲花生大士 |
| 牧马者之子 | 开创别解脱戒和中观宗，以辩论智慧胜伏外道 | 静命论师 |
| 养狗者之子 | "你们有什么事，我可以作通讯员" | 绕萨囊 |

## 与文殊菩萨的关系

- 文殊菩萨亲自以宝剑灌顶
- 住处显现为文殊菩萨像——外道所见
- 桑耶寺开光时现文殊金刚相——赤松德赞所见
- 麦彭仁波切赞叹："蒙文殊剑灌顶者""语狮子尊"
- 金刚手菩萨的化身——因结集了唯识与中观合一的教言

## 苯教的对立叙述

《苯教源流·智者颈饰》声称静命论师是藏地人、苯教叛离者的儿子，为报复苯教而精进修学后回藏摧毁苯教。此说无依据，不被藏学研究者采信。

## 历代大德的赞叹（第127课）

第127课集中汇集了历代大德对静命论师的赞叹：

| 赞叹者 | 赞叹内容 |
|--------|---------|
| **萨迦班智达** | "勤戒胜士具德静命尊，禁行成就莲花生大士，智慧之王莲花戒师等，乃浊世时第二大佛陀。" |
| **宗喀巴大师** | "文殊菩萨亲摄受，抵自他宗之彼岸，获证甚深之意义，具德静命尊前礼。""彼怙雪域首树立，佛教之时善创造，龙树宗兴之缘起。" |
| **麦彭仁波切** | 《中观庄严论释》前后反复赞叹其智慧与文殊无二、佛教宗旨之唯一庄严 |

## 伏藏品中的授记（第127课）

秋嘉德钦朗巴（一百多位伏藏大师之一）的伏藏品中，记录了静命论师对未来的授记：

> "佛教此珍宝，我于雪域弘，凡随学我者，受持佛胜教。赤松德赞王，至尊莲花生，菩提萨埵我，所颁之法令，有情需奉行。"

> "佛教极衰落，众生悲惨时，祈求本师悯，屡屡祈祷吾。外持净戒律，内修菩提心，得密宗成就，杨柳刹成佛，与密主无二，化身护藏教，故定得加持。"

此授记承诺：祈祷静命论师，可获戒律清净、菩提心成就、密宗成就等功德。后来出现的阿底峡尊者等，皆被认为是静命论师的化身。

## 堪布的个人因缘

堪布在第1课中特别提到：他真正接触论著方面的第一部法，就是1985年春天在法王如意宝面前听受的《中观庄严论释》。法王如意宝曾两次传讲此论，堪布对此法"非常有信心"。第127课圆满时，堪布再次提到这一因缘——正式听受的第一部论典即是《中观庄严论释》，此次传讲也是为报答上师恩德。
