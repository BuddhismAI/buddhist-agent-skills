# 中论知识文档·质量验证报告

## 总体评价

本知识库覆盖《中论》全部27品及首义、末义，共30个知识文档（1个序文档、27个品文档、1个结构总览、1个处理日志）加7个推理方法文档，总计约8,700行。整体质量为**A-级**。

**优势面**：结构总览对全论论证策略的分析具有真正的学术洞察力——不是对科判的重新排版，而是展示了龙树"从外围到核心"的辩证退后策略、各品群之间的逻辑依赖关系、以及"破本体+破能立"的递归元结构。各品知识文档在论证脉络追踪方面表现优异：所破/能破识别清晰，推理方法命名明确，颂词引用充分，跨品关联有据。推理方法文档积累了跨品应用，具有真正的复用价值。

**主要不足**：存在一处结构映射错误（结构总览树状图），晚期品文档（品25-27）相比早期品偏薄，部分品文档缺少修行维度或教证维度的充分提取，回遮论证文档的应用案例集中于品24而未收录品25的应用。

## 二、结构总览质量

### 2.1 智识架构

**优秀**。结构总览不是科判的翻版，而是展示了：
- 全论的四步论证策略（破生→破各类法→正面回应→总收）
- "辩证退后"逻辑（从外围到核心）
- 应成/自续分流点的定位
- 各品群之间的逻辑依赖关系（如品08-09的法我空→人我空转折）
- 品15→品18→品24的判教三部曲

品群分析部分（品01-02到品27+末义）是整个知识库中分析密度最高的段落，展示了对全论论证弧线的深入理解。

### 2.2 映射表准确性

二十七品逐品映射表（品号/品名/课次/知识文档）完整且准确。

### 2.3 结构图错误（Must-Fix）

**结构总览第75行的树状图存在错误**：

```
├── 戊三·抉择有实法为空性      ← 品13-14    ← 错误
```

实际上，根据 `outline_fixed.md` 第331-484行，戊三包含：
- 己一·品13（观行品）
- 己二·品14（观和合品）、品15（观有无品）、品16-18（观缚解品→观业品→观我法品）

即 **戊三 = 品13-18**，而非仅品13-14。结构总览自身的品群分析散文部分正确地将品15-18归入"抉择有实法为空性"的讨论，但树状图遗漏了。

树状图中品15-18没有出现在任何节点，品19-21直接归入戊四。这导致品15-18在树状图中"消失"了。

### 2.4 内联引用

结构总览在品群分析中大量引用具体知识文档（如"详见 `品07-观三相品.md`"、"见 `推理方法/四边破.md`"），引用准确且有助于导航。

1. 法相/事相/名相三相框架——知识文档的表格和定义与第21课堪布讲解完全吻合。
2. "空相未有时，则无虚空法"颂词的论证分析——与第21课逐步分析一致。
3. "有相无相中，相则无所住"颂词——知识文档的三路破斥（已有法相/无法相/第三种情况）与源文吻合。
4. 摄义颂互为因果的描述——准确反映了源文的分析。
5. 破有实无实——知识文档准确区分了有部宗（实有虚空）和经部宗（无实虚空存在），与第22课一致。
6. "有无俱非中，知有无者谁"四边破——知识文档的四种情况列举与源文完全一致。
7. "余五同虚空"类推——准确。
8. 呵斥劣见颂——准确引用，并正确标注了麦彭仁波切《中观庄严论释》的引用。

### 5.2 可选字段一致性

- `type`字段：部分品文档有（如品03-06的`unit_overview`），部分没有。不一致但不严重。
- `status`字段：部分品有（如品07的`complete`、品08的`完整`、品15的`完成`），部分没有；措辞不统一（`complete`/`完整`/`完成`/`已处理`）。

- `relates_to`字段：大部分有，但格式不统一——有些包含`.md`后缀（如`品01-观因缘品.md`），有些不包含（如`品01-观因缘品`）。

### 5.3 Frontmatter问题汇总

1. **status措辞不统一**：`complete` / `完整` / `完成` / `已处理` 四种写法混用（应选定一种）
2. **relates_to格式不统一**：有无`.md`后缀混用
3. **lessons格式不统一**：`第X-Y课` / `X-Y` / `[X, Y, Z]` 三种写法
4. **source_refs格式不统一**：`第X课` / `中论讲解第X课` / 数字列表

## 七、缺失知识

### 7.1 缺失的集合级文档

1. **术语表**：全论涉及大量专业术语（自性/他性、法相/事相/名相、有实法/无实法、成实戏论/相之戏论等），目前分散在各品文档中。一个集合级术语表（含梵藏对照、各品出处、定义演化）对LLM检索有高价值。
2. **教证索引**：全论引用了大量经论（《般若经》《宝积经》《三摩地王经》《入行论》《俱舍论》《入中论》《四百论》等），目前分散在各品中。一个集中的教证索引对学术检索有价值。

- **堪布的修行劝导和生活应用**未被充分提取。堪布在讲课中经常穿插修行建议（如品05第21课关于学习因明中观的劝导、品19第68课关于大圆满入门的教言），这些对实际修行者有价值。
