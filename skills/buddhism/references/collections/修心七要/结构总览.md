# 修心七要耳传略释·结构总览

## 一、这部论在修行体系中的位置

《修心七要》是**藏传佛教噶当派**最核心的菩提心修法窍诀之一。恰卡瓦·耶喜多杰格西（与朗日塘巴同为博多瓦格西的两大弟子）将金洲上师传给阿底峡尊者的菩提心教授**归纳为七个要点**：

| 要 | 名称 | 功能 | 对应知识文档 |
|----|------|------|--------------|
| ① | 当修前行法 | 基础（出离心、取舍心） | `要01-前行法.md` |
| ② | 正行修持菩提心 | 核心（胜义+世俗菩提心） | `要02-正行修菩提心.md` |
| ③ | 恶缘转为菩提道用 | 抗逆境保障 | `要03-恶缘转道用.md` |
| ④ | 归结终生之行持 | 生与死的五力 | `要04-归结终生修法.md` |
| ⑤ | 修心圆满之标准 | 验证层 | `要05-修心标准.md` |
| ⑥ | 修心之誓言 | 行为规范"破" | `要06-修心誓言.md` |
| ⑦ | 修心之学处 | 行为规范"立" | `要07-修心学处.md` |

加哲无著尊者（《佛子行》《入行论释·善说海》的作者）对此七要所作的耳传注解就是**《〈修心七要〉耳传略释》**，索达吉堪布于2005年1月1日翻译完成，同年7月6日至14日用八天时间为学院僧众讲解。本wiki即基于这8课讲记及论根本文整理。

## 二、为什么是这个次第——恰卡瓦归纳的论证架构

恰卡瓦格西的七要归纳**不是随意罗列**。它对应一个完整的修行次第逻辑：

```
发菩提心的准备 → 发菩提心 → 菩提心受考验 → 菩提心终生化
     ↓                                           ↓
   前行（要01）                              归结终生（要04）
                                                 ↓
                                           菩提心的外显
                                                 ↓
                           [验证]（要05） → [行为规范]（要06破+要07立）
                              ↑
                              |—————— 中心点：正行（要02）
                              |        │
                              |        └── 遇违缘时：恶缘转道用（要03）
```

### 深度关联：每一要都回答一个具体问题

| 要 | 回答 |
|----|------|
| 要01 | 修菩提心之前，我需要建立什么心态基础？ |
| 要02 | 菩提心具体怎么修？ |
| 要03 | 修到一半遇到违缘怎么办？ |
| 要04 | 怎么把菩提心贯穿到一辈子，直至临终？ |
| 要05 | 我怎么知道自己修成功了？ |
| 要06 | 发了菩提心后，什么事不能做？ |
| 要07 | 发了菩提心后，什么事必须做？ |

这是一套**完整的菩提心生起→维持→验证→持戒**体系。任何单一要都无法独立完成修心，但七要合起来构成一辈子的修行蓝图。

## 三、全论的核心命题——"我执是痛苦之源，众生是成佛助缘"

贯穿七要的两条主线是**一破一立**：

- **破**：我执是一切痛苦的源头（见要03"报应皆归一"，要04"破斥力"，要05"诸法摄一要"，要06十组誓言，要07⑰"不暴躁易怒"中金厄瓦格西的痛斥）
- **立**：众生是成佛必需的助缘（见要02"于众修大恩"的两重恩德论证，要03"于众修大恩"深化，要07⑦"三主要因"）

**朗日塘巴两句话总纲**（在要03的收束段出现）是对这两条主线最简洁的表述：

> 亏损失败我取受，利益胜利奉献他。

这一句贯穿于要02入定时的常念颂、要03的收束、要06的"转欲自稳重"和要07的"不好大喜功"——构成了整个修心七要的"心印"。

## 四、论证架构——理论根据

《修心七要》作为一部**窍诀式**论典，其理论根据完全依赖于另外三部论典：

### ① 《入行论》（寂天菩萨）——主要理证来源

本论引《入行论》颂词**超过十处**，是所有引证中最密集的：

- **要02** 自他相换三颂（"若人欲速疾…""若不以自乐…""如为止自害…"）——见《入行论·静虑品》
- **要03** 五颂我执破斥（"世间诸灾害…""汝虽欲自利…""如是汝屡屡…""昔时受汝制…""自身过患多…"）——见《入行论·静虑品》《智慧品》
- **要03** "修法所依缘，有情等诸佛，敬佛不敬众，岂有此道理？"——众生佛陀恩德等同的关键教证
- **要07⑫** "何需足量革…"（皮革覆地比喻）——伏心胜制外境

**结论**：修心七要可以视为《入行论·静虑品》（自他平等/自他相换）+《入行论·智慧品》（我执破斥）在实修层面的**凝练窍诀**。广大理证见 `collections/入行论/品08-自他平等.md`、`collections/入行论/品08-自他相换修法.md`。

### ② 《中观宝鬘论》（龙树菩萨）——菩提心理论总纲

要01开篇引龙树三正因："欲自世间众，得无上菩提，菩提心为本，坚固如山王，悲心遍十方，不依二边智。"——这是**菩提心、大悲心、无二慧**三正因框架的出处，是整个修心系统的理论坐标。

### ③ 《中观四百论》/《中论》/《入中论》——胜义菩提心的理证基础

- 要02"观心性无生"——观心三时不得，与《中论》破心的抉择同源
- 要03"迷境观四身"——从生/住/灭三时不可得抉择四身，与《中论》观法空同源
- 要07⑥"当学三种难"引《中观四百论》烦恼两种生起方式
- 要02"不依二边智"引《入中论》"无二慧"

详见 `topics/madhyamaka/index.md`。

### ④ 阿底峡尊者三上师传承

要02明确指出：修菩提心的七因果修法（知母-念恩-报恩-悲心-慈心-胜解信-生起）是阿底峡尊者从**达玛绕吉达、宣巴南鸠、金洲大师**三位上师所得。本论所讲就是金洲大师的教规。

## 五、论证的七个关键转折

在修行人可能中途迷失的地方，恰卡瓦和无著都预先布置了"扶正"机制：

### 转折一（要01→要02）：出离心不止步于出离

共同外前行（暇满/无常/轮回过患）指向的不是独善其身的解脱，而是**发菩提心的基础**。如果观修前行没有引出菩提心，那就没有得到前行的真实利益。——`要01-前行法.md`的最终指向。

### 转折二（要02内部）：胜义菩提心凡夫只能相似修

堪布明确——"胜义菩提心只有一地菩萨以上才能真正获得，凡夫只有相似地安住。本论所讲的主要是世俗菩提心。"——这一澄清避免了凡夫误以为自己能修胜义菩提心的错误定位。——`要02-正行修菩提心.md`。

### 转折三（要02→要03）：菩提心必须经过逆境检验

修菩提心**不是顺境中的温情**。要03把菩提心拖到违缘、疾病、诽谤、伤害的场景中——**能在逆境中还能作自他相换的人，才算真正生起了菩提心**。——`要03-恶缘转道用.md`。

### 转折四（要03内部）：胜义+世俗两轨转恶缘

要03不满足于情感上的对境转化，还给出了**胜义转化法**——"迷境观四身""空护为最上"。这把大圆满的"本来清净"抉择直接引入修心七要，使宁玛派讲此论有其独到之处。

### 转折五（要04→要05）：日常化与终极化同时

要04的"归结终生之行持"五力不是随意修，必须**日日、终生、临终都贯穿**。大乘死教言五力（狮子卧+菩提心）是这一要的收官绝技——提供了不依赖颇瓦训练的临终保底方案。——`要04-归结终生修法.md`。

### 转折六（要05）：只是"刚生起的征兆"

要05最大的扶正是——即使达到四个修心圆满标准，也只是**菩提心初生的验相**，绝非可以停止修持的许可证。这防止了**得少为足**的常见误区。——`要05-修心标准.md`。

### 转折七（要06↔要07）：破立并举

要06的"破"（十组誓言）与要07的"立"（十八组学处）构成**完整的菩萨行为体系**——不同于传统菩萨戒以律条编目，此处以**窍诀方式**给出精神实质。两者结合才是完整的菩萨行动指南。——`要06-修心誓言.md`+`要07-修心学处.md`。

## 六、三大关键群——贯穿七要的三条主线

### 主线A：自他相换（lojong / tonglen）

这是修心七要的**签字式修法**，不是某一要的专属：

| 要 | 自他相换的形态 |
|----|---------------|
| 要02 | **入定时**的杂修二取舍+彼二乘风息（呼吸配自他相换） |
| 要02 | **出定时**的三境毒善根+取次从自起（三毒转为自他相换对境） |
| 要03 | 遇害时的自他相换（代受害人的苦，将乐给他） |
| 要04 | 临终时的自他相换（借狮子卧+风息继续代众生受苦） |
| 要07② | 违缘中的自他相换（代一切众生受苦） |
| 要07⑨ | 临终境的自他相换（恰卡瓦格西舍极乐往生代众受苦公案） |

修心七要的自他相换不是一种静态的观修对象，而是**随时随地可以启动**的心理操作。与入行论品08的理论广说互补——详见 `collections/入行论/品08-自他相换修法.md`。

### 主线B：我执破斥

| 要 | 我执破斥的形态 |
|----|---------------|
| 要02 | 胜义菩提心观心无生、对治亦自解——从根本上破我执 |
| 要03 | 报应皆归一——把一切痛苦归于我执 |
| 要04 | 五力之"破斥力"——终生对我执的武器库（菩提心教言/中观空性/大圆满本来清净） |
| 要04 | 临终破斥力——寻"我"不得而断执 |
| 要05 | 诸法摄一要——一切修持都是为了对治我执 |
| 要06 | 十组誓言的精神——爱他对治爱我执 |
| 要07 | 多项学处围绕"不自矜""不邀功""不偏袒"——日常破我执 |

**我执破斥不是哲学问题，是修行的日常战役**。见整论中奔贡甲格西"手持短矛守城门"的比喻（要03）——修行人唯一的任务就是对治我执。

### 主线C：两种菩提心的双运

入定→胜义，出定→世俗，两者**轮番或融合**构成菩提心的全貌。第8课堪布最后总结：

> 入定时主修胜义菩提心，即抉择一切万法为离戏大空性，这实际与大圆满的"本来清净"相同；出定时缘一切有情修自他交换，以此引生强烈的大悲心，这就是世俗菩提心。若能将此二者圆融一味地修持……那就是诸佛菩萨共同修持的道——**空性大悲藏**。

这是修心七要最高的实修境界——**空性与大悲的不二**。与大圆满的"本来清净"见地直接对接，也与入行论品09"智慧品"的智悲双运呼应。

三者可以视为不同侧重的**同一体系**——foundations topic的三大支柱。详见 `topics/foundations/index.md`。

- **菩萨戒的窍诀化**——十破+十八立的日常行为规范

## 八、宁玛派诠释此论的特色

本论的作者加哲无著尊者虽属噶当派传承，但索达吉堪布（宁玛派）的讲解**将修心七要与大圆满见地有机对接**：

### 接口一：胜义菩提心↔本来清净

要02第四颂"道体住赖耶"——堪布解释为**如来藏的光明分**，与大圆满本来清净的见地对应：

| 修心七要颂 | 对应大圆满抉择 |
|-----------|---------------|
| 思诸法如梦 | 心的本体空性=离戏法身 |
| 观心性无生 | （破能取心识） |
| 对治亦自解 | （破空性对治念） |
| 道体住赖耶 | 显现不灭=光明报身；显现空性无二周遍=大悲化身 |
| （四身体性无别） | 本性身 |

### 接口二：四身观↔烦恼即菩提

要03"迷境观四身"——堪布指出这与大圆满抉择烦恼本性的修法一致，是显密共通的最甚深窍诀。"烦恼即菩提"的禅宗传统与大圆满本来清净会师于此。

### 接口三：临终智悲双运↔本来清净

要04末尾——胜义菩提心+世俗菩提心在临终时轮番修持，"以大圆满的说法，这就是一种最高的'本来清净'的死亡方法"。

## 九、金洲大师的教规与传承

本论是**金洲大师教规**，即阿底峡尊者三上师之一（其他两位：达玛绕吉达、宣巴南鸠）传授的菩提心窍诀。

### 金洲大师的地位

阿底峡尊者在一百五十多位上师中有十三位大成就者，但唯独对金洲上师——

- 每月举办忌日供会
- 提及其他上师尊名时合掌当胸，提及金洲上师时**合掌于顶、双目垂泪**

阿底峡尊者自言："我相续中的这少分菩提心，就是依靠金洲上师而得的，因此他老人家对我的恩德最大。"

## 十、时代意义——五浊恶世的修心甘露

无著尊者本人在全论结尾直接点出本论的时代意义：

> 在时浊、众生浊、命浊、烦恼浊与见浊等五浊十分炽盛的时候……如果能将这些恶缘转为修心的助伴而进行修心，则无论有再多的恶缘，也都能让自己的善行不断增长。

堪布在第8课的时代观察：

> 现在世界上的人口越来越多，武器毒药越来越先进，食物越来越没营养，环境污染越来越严重，天灾人祸越来越频繁……人们生活在空虚的世界里，为了填补精神上的空白，不断用种种刺激来麻醉神经，自杀率每年也以成倍的速度递增……

**修心七要对现代人的适配度**：

1. **极简**——适合现代人碎片时间修持
2. **无需离世**——在家、上班、出行都能修（彼二乘风息可以融入呼吸）
3. **抗逆境**——恶缘转道用在压力社会中最实用
4. **无需颇瓦**——临终教言对大多数人是可达的保底方案
5. **窍诀化**——不需要精通五部大论就能修

这是恰卡瓦格西九百年前归纳时就为"后期众生"准备的窍诀。
