# 品09 成熟品——自成熟九相与令他成熟六度

本品是"所思维"三品（真如/威力/成熟）的**收尾品**——真如品讲所证、威力品讲利他之工具、成熟品讲**相续如何成熟以能真正承担这一切**。课21 开宗明义：

> "自相续成熟：是指菩萨需要智慧、安忍、威力以及对大乘佛法的欢喜、信心等等。但与此相似的一些境界，每一个凡夫人都会有——包括其他乘、其他宗教的人。但真正的成熟——前面也讲了什么叫成熟，成熟就是已经可以使用了。"

**"成熟"是本论贯通性的关键概念**——从品5 发心品起的一切菩萨行，都要通过相续的"成熟"才能发挥作用。发心→修行→证悟真如→起用威力→最终汇聚为"自他相续成熟"，构成大乘道次第一条不可分割的脉络。

**本品的双层结构**——**自成熟**（己一 略说 + 己二 九相广说 + 己三 摄义）是**令他成熟**（己一 分类 + 己二 意乐 + 己三 六度加行）的前提。麦彭仁波切密意疏 170 行，课21-23 讲解。

## 二、戊一 己一 略说——九相之名

> **喜信寂静悲，忍慧具威力，**
> **不夺具支上，佛子成熟相。**

九种成熟：**喜 / 信 / 寂静 / 悲 / 忍 / 慧 / 威力 / 不夺 / 具支**——佛子成熟之九相。

**九相的内在逻辑**——从大乘入道的角度看，此九相构成入大乘的完整"成熟序列"：

```
喜  → 信  → 寂静 → 悲  → 忍  → 锐智 → 威力 → 不夺 → 具支
大乘  善知    烦恼   众生   自苦    智慧    广积   不被   断支
欢喜  识信心  薄少   深悲   安忍    锐利    资粮   邪转   相助伴
```

- **喜、信、寂静**是**初入门阶段**（对大乘法生起欢喜 → 对善知识佛陀生起信心 → 相续寂静少烦恼）
- **悲、忍、锐智**是**中间修学阶段**（发大悲心 → 能忍苦行 → 生起锐智）
- **威力、不夺、具支**是**成熟证悟阶段**（积资成威力 → 不被邪魔所夺 → 具足五支获一地）

课21 对听者的严厉提问：
> "大家一定要好好观察自相续成熟没有。"

以下按密意疏三分法（因 / 本体 / 作用）逐一详析。

## 三、戊一 己二 广说九相

每一相都以"因 + 本体 + 作用"三维析明——这一三分架构是本品最严密的安立方法。

### 3.1 庚一 喜成熟相

> **胜友等三勤，竟妙摄胜法，**
> **受持大妙法，悲尊成熟相。**

| 维 | 内涵 | 颂词对应 |
|---|---|---|
| **因** | 三种——殊胜善知识（胜友）、闻法、如理作意 | "胜友等" |
| **本体** | 二种——精进闻思修大乘法 + 究竟妙（对大乘深意获不可思议定解） | "勤竟妙" |
| **作用** | 受持大乘法——"摄集并不失而护持最殊胜的大乘法" | "摄胜法 / 受持大妙法" |

### 3.1.1 喜成熟的核心——护持大乘法

密意疏："欢喜成熟的密意归纳起来，就是受持大乘广大妙法，为此承许是大悲尊者菩萨欢喜大乘真实成熟的法相。"

**《大宝积经》两大教证**（课20）：
> "于如来圣教，敬心而听法，于法恭敬已，速悟大菩提。"

> "菩萨恒沙愿，皆入一大愿，所谓一大愿，摄受如来法。"

后一教证特别重要——**恒河沙数菩萨大愿可归摄为一愿：摄受如来法**。课20："这句话大家一定要记住。护持如来的教法、受持如来的教法，所有菩萨的大愿都可以包括其中。"

### 3.1.2 判断"喜成熟"真假的标准

课20 给出检验：
- 自称喜欢大乘但"让你给别人辅导一遍也不愿意"——不真喜
- "让你听一堂课的话，第一节课的时候还在，第二节课之后就在我的世界当中永远消失了"——不真喜
- **真喜欢 = 受持 + 传讲 + 听闻 + 保护大乘法**——口头语言不算

### 3.2 庚二 信成熟相

> **知德心不退，速得定受果，**
> **诚信本师故，佛子成熟相。**

| 维 | 内涵 | 颂词对应 |
|---|---|---|
| **因** | 了知本师佛陀功德 | "知德" |
| **本体** | 对本师不退转信（一地时真正获得不退转解信） | "心不退" |
| **作用** | 迅速获得大乘首楞严等持 + 能领受等持之果（神通）| "速得定受果" |

### 3.2.1 "知德"——信心必以了知功德为因

密意疏："信心成熟之因：了知本师佛陀的功德。对本师获得不退转信，对本师佛陀信心不会改变、退失。真正的不退转信，获得一地时才对三宝获得不退转的解信。"

课21 直接破盲目信心：
> "有些人说：'我对上师您特别、特别起信心。''为什么？''莫名其妙的，我也不知道什么原因。'这种信心不是成熟的信心，因为你不知道他的功德。你如果真正知道他的功德，以后不会退的。"

### 3.2.2 堪布自证——"对本师不退"的人证

课21 罕见地**自我见证**：
> "我觉得我对佛陀具有真正的、成熟的信心：首先我具足信心的因——知道佛陀的功德；第二，我对佛的信心是发自内心、深入骨髓的，所以在我的有生之年当中，绝对不会退转。"

### 3.2.3 《定解宝灯论》教证——随法信 vs 盲目信

课21 引《定解宝灯论》：两种信心——
1. **利根者对法的信心**（随法者）——基于了知功德之真实定解
2. **钝根者盲目的信心**（随信者）——不知功德之相似定解

**"一个是利根者对法的信心，一个是钝根者盲目的信心"**——此二者正对应《大乘经庄严论》信成熟"知德"的有无。

### 3.2.4 《出曜经》教证——佛德难尽赞

课21："假如一个人有无量的头，每个头都在赞叹佛陀恒河沙数的功德，但即使如此也不能赞尽佛的功德。"

### 3.3 庚三 寂静成熟相

> **极禁断惑念，无碍喜乐善，**
> **佛子除惑故，许真成熟相。**

| 维 | 内涵 | 颂词对应 |
|---|---|---|
| **因** | 以正念正知摄持极其禁止诸根趋入对境，不作恶行 | "极禁" |
| **本体** | 断除烦恼分别念 | "断惑念" |
| **作用** | 无有违缘修行烦恼不善的对治法，欢喜乐于行持六度等善品 | "无碍喜乐善" |

### 3.3.1 《诸法集要经》教证

课21 引：
> "若人心寂静，诸根不散乱，决定趣菩提，此说为安乐。"

### 3.3.2 寂静成熟的方便——对治分别念

不同烦恼对应不同对治（印度注疏）：
- **嗔心** → 修慈悲心
- **贪心** → 修不净观
- **痴心** → 修缘起观

课21 提醒："我们在修行的时候不能变成这种情况：'我现在好像越来越没有任何进步，不但没有进步，还有退步，烦恼更加粗重。'其实，如果你真正修对了，真正很好地闻思，烦恼越来越大是不可能的。"

### 3.3.3 祈祷上师——寂静成熟的外缘

课21 强调："希望很多道友在修行过程中，千万不能离开对传承上师的祈祷，心一定要调柔。"——**寂静成熟非独力可成，需要加持**。

### 3.4 庚四 悲成熟相

> **自慈见他苦，劣心尽断除，**
> **胜进众尊身，悲他成熟相。**

| 维 | 内涵 | 颂词对应 |
|---|---|---|
| **因** | 由种姓所牵自然慈悲 + 所缘见他苦 | "自慈见他苦" |
| **本体** | 完全断除欲求声缘涅槃的小乘心（"劣心"）+ 大悲越来越超胜 | "劣心尽断除 / 胜进" |
| **作用** | 获得成为众生之尊的身体——从一地起获得相好 | "众尊身" |

### 3.4.1 "劣心"——大悲成熟的遮破对象

此处"劣心"**专指声闻缘觉求自寂灭的心**——非世俗不善心。大悲成熟必是**超越二乘自利心的菩萨心量**。

课21 的"悲心次第"：
> "刚来的时候，我们的悲心是相似的，只是对个别人和事做一点简单的慈善，但是这样修下去，最后利益众生的范围就越来越大了，可以对所有的众生，包括对自己从前怨恨的敌人都具有悲心，这就是大悲心的自性。"

### 3.4.2 地道次第中的悲心递增

密意疏：
> "与资粮道、加行道相比，得地者的悲心更为殊胜，二地菩萨的悲心又超胜一地菩萨的悲心，如此类推。"

—— **大悲不是一次达成的目标，而是地地超胜的延续提升**。此是悲成熟之"胜进"义。

### 3.4.3 《大般涅槃经》教证

课21 引：
> "众生无明冥，不见烦恼果，常造诸恶业，不得安隐眠。"

### 3.4.4 悲成熟与菩萨入监狱/贫困地之教诫

课21 罕见开示：
> "本师释迦牟尼佛在因地发心时，他专门选择到我们这个娑婆世界、五浊恶世……现在个别的大德经常去条件比较好、人员素质比较高、各方面资源比较丰富的地方，而比较偏僻、边远的地方，有些人不太愿意去，其实这样不是很好。希望我们弘法利生的时候，有机会要尽量找一些苦难的对境，也许对你个人的修行会有很大的帮助。"

### 3.5 庚五 忍成熟相

> **自坚抉择修，恒耐寒等苦，**
> **胜进喜乐善，许忍成熟相。**

| 维 | 内涵 | 颂词对应 |
|---|---|---|
| **因** | 自坚（种姓力甘受苦）+ 抉择修（闻法分别抉择） | "自坚抉择修" |
| **本体** | 恒时耐受寒热、邪行等极度痛苦 | "恒耐寒等苦" |
| **作用** | 功德越来越超胜，欢喜实行六度善法 | "胜进喜乐善" |

### 3.5.1 安忍因缘的二层——先天 + 后天

- **自坚（先天）**——由如来藏自性具足的忍耐种子，无需教育自然有
- **抉择修（后天）**——通过观修空性/因缘/大乘因果习得

### 3.5.2 两修行人的课堂事例

课21：
> "以前佛学院刚成立的时候，汉族人很少……当时两个出家人因为座位有点争吵，有一个出家人说：'我如果没有学习大乘佛法，按照我原来的脾气，今天肯定揍你。'"

——**后天抉择修安忍的现实例**。

### 3.5.3 《大萨遮尼干子所说经》教证

课21 引：
> "能行忍辱者，见者皆欢喜，怨家舍毒心，皆生亲友想。"

### 3.5.4 忍成熟的现代判准——情绪不被外缘所转

课21 自举："刚才我来上课之前，有两个学生到我家里来。他们看到我年轻时的一张照片。一个同学说：'哎，你以前还是比较好看的。'当时，我心里还是比较淡定，没有生气也没有伤心。另外一个同学马上开始纠正：'现在也可以，现在也可以！'（众笑）所以如果有一些安忍的能力，就能像我一样不伤心也不生气。"

### 3.6 庚六 锐智成熟相

> **报净念闻等，尽知善恶说，**
> **可生大智慧，锐智成熟相。**

| 维 | 内涵 | 颂词对应 |
|---|---|---|
| **因** | 前世闻思异熟果报清净 + 正念 + 能辨善恶说之正知 | "报净念闻等" |
| **本体** | 尽知善说与非理恶说（以正念正知使意无颠倒） | "尽知善恶说" |
| **作用** | 以世间知念锐智为因，使智慧增上，可生圣地大智慧 | "可生大智慧" |

### 3.6.1 智慧成熟必讲前世

课21：
> "锐智成熟相的因缘一定要讲前世，这个是很重要的……我们的智慧实际上是前世各种因缘所带来的一种果报。"

**《三宝感应要略录》"鼠闻律藏"公案**（课21）：
> "有个阿罗汉戒律特别清净……他给别人讲自己的经历说：'我前世是一只老鼠，当时我住在一个山洞里，洞里有一个学戒律的大师在闭关，他每天都诵戒律，诵戒的声音被我听到了。因此我即生当中，一方面对戒律很有兴趣，第二方面我的戒律也很清净。'"

此公案说明即使一只老鼠听律藏，也能后世得成就——**前世因缘决定今生智慧**。

### 3.6.2 《正法念处经》教证

课21 引：
> "若人无智慧，如盲入闇处，则不厌生死，非法诤斗笼。"

### 3.6.3 "善恶说"——智慧本体的辨别功能

锐智的**核心功能是辨别**——
- 善说：三转法轮之真实语
- 恶说：世间虚妄宣传等

课21："有智慧的人对于善说和恶说、善语和恶语能够辨别——这是佛陀的语言；这是一般世间的语言。而没有智慧的人就没办法辨别——他觉得这是同样的两本书，为什么你非要选择这本？"

### 3.7 庚七 威力成熟相

> **二善增二界，依能生果胜，**
> **如愿成众尊，威力成熟相。**

| 维 | 内涵 | 颂词对应 |
|---|---|---|
| **因** | 行持福德智慧两种善资粮 | "二善" |
| **本体** | 福慧二界（种子）在阿赖耶上增长，所依身堪能产生大乘圣道果 | "增二界，依能生果胜" |
| **作用** | 随意成办众生利益——以福获殊胜身受用，以智获超胜妙慧成众生之尊 | "如愿成众尊" |

### 3.7.1 二界（种子）义——弥勒的独特表达

课21 注："种子就是界的意思，弥勒菩萨《辨中边论》：种子义名界。"

—— **"界"与"种子"在弥勒五论中义同**。福慧二界在阿赖耶上种下，遇缘增长，生大乘圣道果。

### 3.7.2 威力成熟与品8 威力品的关系

**品8 威力**是 **外在利他工具**（六通/三明/八解脱等），本品"威力成熟"是 **内在相续积累**（福慧二界）。**内在成熟为因，外在威力为果**。无前者则后者不真——提婆达多即是例——得神通而无威力成熟。

### 3.8 庚八 不夺成熟相

> **理析妙法智，恒无魔作障，**
> **得胜驳他宗，不夺成熟相。**

| 维 | 内涵 | 颂词对应 |
|---|---|---|
| **因** | 以四理分析善妙密义获得无疑智慧 | "理析妙法智" |
| **本体** | 恒时无魔作障——对甚深法义获真挚定解，邪魔外道不能转其道 | "恒无魔作障" |
| **作用** | 对大乘教义获较他殊胜定解 + 能破斥他宗 | "得胜驳他宗" |

### 3.8.1 四理（四种理）

课22 解说：
- **观待理**：任何果观待因而产生
- **作用理**：因对果的产生起作用
- **证成理**：自性因/不可得因/果因——推理证明
- **法尔理**：缘起法尔（如来出世不出世皆然）

### 3.8.2 "不随他转 / 不被他夺"——麦彭仁波切的定解观

课22 引麦彭仁波切《中观庄严论释》与《定解宝灯论》：
- **不随他转**：凭自己理证智慧的力量坚信不移，不依赖他人仰仗他力
- **不被他夺**：获得解信后，一切邪魔外道无法使之背其道而行

**这两种定解是本品"不夺成熟"的宗门诠释**。

### 3.8.3 《大宝积经》教证

课22 引：
> "勇健大智人，了知空性理。能怖畏魔军，彼堪销供养。"

### 3.8.4 不夺智慧 vs 盲目无畏——课22 的警告

课22 严辨二者：
> "当我们有了不被他夺的甚深定解，在世间的任何一个群体中——不管是非佛教徒，还是佛教徒，你都可以自由自在地展现你的辩才和智慧，而不会胆战心惊——'我不敢出现，害怕别人问我。'当然有些人的不怕，很可能是盲目的——'我无所谓，我也没有什么害羞，别人问我答不上也无所谓，反正乱说也没事'，这是一种无耻的无畏。"

### 3.9 庚九 具支成熟相（断支成熟相）

> **积善依堪勤，现喜胜静善，**
> **佛子之具支，真实成熟相。**

此颂最难懂——对应**一地菩萨断功德的五种支**。

### 3.9.1 五断支与颂词对应

"具支"即"**具断分支**"——现量证悟法界后断所取能取（断功德）的**五种助伴**：

| 五支 | 颂词对应 | 意义 |
|---|---|---|
| 1. **信心**（因） | "积善" | 通过积累福慧资粮得——信心为一切功德之母 |
| 2. **无病**（身依堪能） | "依堪勤" | 身所依健康行善无不堪能 |
| 3. **无谄** | "现喜（胜静）" | 极喜断除愦闹的寂静 |
| 4. **无诳** | "现喜胜静善" | 极喜远离作意自利的菩萨殊胜寂静 |
| 5. **智慧** | "善" | 极喜闻思修大乘法的殊胜 |

### 3.9.2 具支对应"因 / 本体 / 作用"

- **因**：积善 = 积累福慧资粮 = 信心
- **本体**：依堪勤 = 身体无病可堪行善
- **作用**：现喜胜静善 = 欢喜寂静 + 欢喜善业 = 无谄无诳智慧

密意疏："首先现量证悟法界后断除所取能取，名为断。其助伴的信心等前面所讲的五种称为断分支。那五种以因、自性、作用相成熟之理。"

### 3.9.3 健康对修行的重要——课22 的现代关切

课22：
> "行持善法必须要有一个健康的身体，如果今天头痛，明天心脏痛，后天走路不方便，一会儿这儿有病，一会儿那儿不舒服，不要说断除一切障碍，连平时听课、参加一次考试都不行……能吃的人可以堪为修行的所依。"

情绪健康的注意：
> "有些人比较敏感，很容易受打击，好像一点点小事就特别伤心，一直反反复复地放在心里，晚上睡不着，白天吃不好饭，经常给周围的人带来负面影响和不安全感。"

## 四、戊一 己三 摄义

### 4.1 庚一 九相成熟者是令他成熟的所依

> **九相成熟尊，能令他成熟，**
> **法性相续增，恒为众至亲。**

**核心命题**：自成熟才能令他成熟。密意疏："以上所说喜等九法堪为成熟本性佛陀之法的主尊，他具足能成熟其他众生的功德，因为**自成熟才能令他众成熟**。"

"恒为众至亲"——具足九相的菩萨恒常作为众生之至亲，**六度诸法的自性相续越来越增上的主尊**能成熟其余众生。

### 4.1.1 世间教育 vs 大乘自成熟的根本区别

课22 的结论性对比：
> "菩萨自相续成熟的最终目的不是为了自己，但世间的教育并不是这样的。我在美国的一个学校里面也讲过，现在世界上所有的教育都是为了自我的成功，最多是对自己的国家和民族有利……教育的理念跟大乘佛教很有差别。"

——**大乘"自成熟"的归属是利他**，与世间"成功"为自我服务根本不同。

### 4.1.2 《十住毗婆沙论》教证

课22 引：
> "如是求二利，心无有疲懈，以无疲懈故，能得诸深法。"

### 4.2 庚二 成熟之比喻——疮熟与食熟

> **如疮可破漏，食可用许熟，**
> **此依于二品，灭受说成熟。**

### 4.2.1 疮熟喻——对应"断烦恼"

密意疏："肿核等脓没有成熟，不可破脓，否则对病不利，当脓成熟时，重新结痂，可以破脓，因此时机已到称为疮熟。"

对应修行——五蕴上的贪嗔痴烦恼如脓，因缘成熟时可通过对治挤出去灭除。

### 4.2.2 食熟喻——对应"享用智慧"

密意疏："葡萄等果实没有成熟或食物没有烹饪等时不可享用，如果已成熟到量，则可享用。"

对应修行——五蕴上的善心、人无我智慧、法无我智慧到因缘成熟时可**享用对治**，即"受用"修行智慧。

### 4.2.3 两比喻的综合义——"灭受说成熟"

- "灭"：所断烦恼如脓，灭之 = 破脓除病
- "受"：对治智慧如成熟食，受之 = 享用起作用

"说成熟"即此二品（断 / 受用）的因缘成熟。

### 4.2.4 根性可改变——课22 的鼓励

密意疏："由于根性下中上可以改变，越来越向上修炼根性，是大要点。"

课22："修行比较好、安忍心比较强的人，任何地方都可以是修行的道场……外面各种各样的事、各种各样的人的言行举止，都不会使你的心烦乱。"

### 4.2.5 《佛性论》（世亲）教证

课22 引世亲菩萨《佛性论》：
> "聪明人次第，数数细细修，灭除自身垢，如金师炼金。"

——**修行三个要素**：按次第 / 持续不断 / 细致入微。如金师炼金。

## 五、戊二 己一 令他成熟——八种分类

> **许令离成熟，普熟极成熟，**
> **随善证成熟，恒熟上成熟。**

**八种令他成熟**——涵盖菩萨成熟众生的所有面向：

| # | 名 | 内涵 | 例 |
|---|---|---|---|
| 1 | **离成熟** | 令众生远离贪嗔等烦恼 | 奔公甲出家前嗔心大带刀，出家后不需 |
| 2 | **普成熟** | 令三乘种姓普皆成熟 | 同时传三乘法度诸根基 |
| 3 | **极成熟** | 以三乘法成熟不复退转（胜于世间道）| 断烦恼种子故不再复生 |
| 4 | **随成熟** | 随所化相续而说（贪者说不净/嗔者说慈/痴者说缘起）| "见人说人话"之大乘式 |
| 5 | **善成熟** | 无堕品、平等+恭敬策励他 | 法王传密法平等传授 |
| 6 | **证成熟** | 自得地后将他众安置于彼 / 以证悟令成熟 | 父母是医生让孩子学医的世间版本 |
| 7 | **恒成熟** | 恒常无误宣说佛法 | 佛学院春夏秋冬不断讲经说法 |
| 8 | **上成熟** | 从信解行地至十地间越来越向上 | 一年级→二年级→……研究生 |

### 5.1 关于"随成熟"——应机施教

课22 对"随成熟"的现实关切：
> "世间人认为'见人说人话，见鬼说鬼话'是一种贬义词，但是我觉得，有时候该说还是要说，如果你见到一个魔鬼，光会讲人话是没办法度魔鬼的。"

对比不善巧的例子：
> "比如全是用出家人的要求来要求世间人。原来有一次我出去的时候，我身边有一个出家人，还有一个根本不信仰佛教的世间人……他一边抽烟一边和我们对话，然后那个出家人很不客气地命令说：'你把烟掐了！拿下来！快快快！'令对方有点难堪。"

### 5.2 关于"恒成熟"——佛学院精神

课22：
> "我们佛学院应该有恒成熟的这种精神吧！我们这里的很多法师，确实除了休息日以外，一直常年地、不间断地给别人传授佛法……这是一种恒成熟的精神。"

### 5.3 八种令他成熟与品6 修行品"十三性"的关系

本品八种令他成熟 **vs** [`06-修行品`](./06-修行品.md) 戊二己二**十三性利他**——两者关系：

- **十三性**（品6）：从**对象/方式的具体分类**讲（善说/令悦/趋入/解义/成熟善/授/住/慧解脱/成就胜功德/生种/授记/灌/佛智）
- **八种成熟**（本品）：从**成熟的质的类型**讲（离/普/极/随/善/证/恒/上）

两者互为经纬——品6 讲**做什么**，本品讲**成熟到什么程度**。合观得完整的大乘利他图谱。

## 六、戊二 己二 令他成熟之意乐

> **佛子以利心，住此成熟众，**
> **父母友于子，及亲非善住。**
> **慈主利乐行，慈爱他众生，**
> **世人自不爱，何况极爱他？**

### 6.1 菩萨利心超越世间父母之爱

密意疏："佛子们以欲求将一切众生安置于暂时与究竟安乐的利益意乐而行住于此世间成熟众生，世间父母友人对各自之子与亲人并不具有这样善妙的意乐。"

**理由**：
- 世间父母对子女有慈爱，但"只是暂时今生能作少许利益"
- 不会"生起将他安置于永乐之因善法中的意乐"
- "有些也让他们造罪"（世间父母为孩子利益可能教其造恶业）

### 6.2 凡夫自爱 vs 菩萨爱他

"世人自不爱，何况极爱他"——世人对自己尚且不求究竟仁慈（因愚痴以杀生偷盗为今生乐），**何况对他人？** 菩萨反之——"菩萨慈爱一切众生，是希望他们随时随地行善永久安乐的慈爱。"

### 6.3 《大宝积经》教证——先施世乐后令发净心

课22 引：
> "成熟彼众生，先施世间乐，后令发净心。"

**这是菩萨度生次第**——先以世乐摄受（使其接受），后以出世法引导（令得解脱）。

### 6.4 《华严经》教证——游行众界利有情

课22 引：
> "菩萨游行诸世界，常能安隐群生类，悉令一切皆欢喜，修菩萨行无厌足。"

## 七、戊二 己三 令他成熟之加行——六度

本科判是本品的**压轴**——以六度（布施/持戒/安忍/精进/禅定/智慧）作为"令他成熟"的实际加行。六度本来用于自利，此处**全部转化为令他成熟的方便**。

### 7.1 布施令他成熟

> **菩萨身受用，无有不施他，**
> **二益成熟他，等施不餍德。**

布施**三特点**：
1. **普施一切**：身体、受用皆不舍施
2. **二益成熟**：今生饶益（圆满意愿）+ 来世饶益（安置善法种解脱种子）
3. **平等不餍**：无贤劣高低亲疏偏堕 + 至轮回际不厌足

**南岳总胜集讲法因缘公案**（课23）：
> "有一位修行人广闻博学……讲得再怎么好，座下听法的人却越来越少……老和尚告诉他：'你应该想办法找一些食物，然后到森林、山里布施给那些虫物……发愿以后为它们讲经说法，让这些众生成为你的眷属。你在三十年中不要出来讲法，三十年以后可以出来。'此人对老和尚所讲很有信心，便依教奉行，三十年当中没有出去转法轮，三十年之后他就出来了。这个时候他传法各方面的因缘都具足了，听课的人成千上万非常多。"

此公案说明**布施创造法缘**——发愿力 + 与众生结缘 = 后来大法会的基础。

**《华严经》教证**（课23）："不可思议无量劫，种种布施心无厌"。

### 7.2 持戒令他成熟

> **坚性尤无害，自喜谨安他，**
> **二益熟等流，德转成熟他。**

五种戒：

| # | 名 | 内涵 | 梵文/义 |
|---|---|---|---|
| 1 | **坚（恒常戒）** | 乃至菩提果间恒时守持 | 时间维度 |
| 2 | **性（自性戒）** | 前世串习力，今生自然守戒 | 习气维度 |
| 3 | **尤无害（圆满戒）** | 不损害众生，梦中亦不作十不善（二地菩萨境界）| 净度维度 |
| 4 | **自喜戒** | 一地极喜地证真谛后法性所得的欢喜戒 | 证德维度 |
| 5 | **谨（不放逸戒，无迷乱戒）** | 一切威仪不生细微堕罪 | 细密维度 |

**菩萨不仅自守五戒，更将他人安置于五戒中**——这是"令他成熟"的持戒加行。

**《大萨遮尼干子所说经》教证**（课23）：
> "恶道大毒龙，无明诸罗刹，见持净戒者，恭敬舍害心。"

**道容比丘尼公案**（《法苑珠林》）：戒律清净故获几朝皇帝赞叹。

### 7.3 安忍令他成熟

> **他害作利想，重害尚耐受，**
> **知方便忍害，令其真行善。**

**三层安忍深度**：
1. **他害作利想**——把害我者当作善知识/父母
2. **重害尚耐受**——即使身骨被一块一块砍下（忍辱仙人公案）亦能忍
3. **知方便忍害**——知道以暂时长久二利方式令作害者行善

**忍辱仙人公案**（课23）：
> "国王说：'那我看看你能不能忍！'就用剑把他的双手双脚割断，又把他身体的其他部位，如耳朵、鼻子等一一割断……忍辱仙人说：'你怎么停止了？不管你对我怎么样，即使你把我粉身碎骨，我不但不会对你生一刹那的嗔恨心，甚至我还要回向：以此因缘，将来我成佛时，即使你没有劝请我，我也要想方设法让你成为首座弟子。'因为世尊如是发愿的原因，后来那个坏国王变成了憍陈如，即五比丘之一。"

**《大集经》教证**（课23）：
> "为一一众生，无量劫受苦，救拔众生故，堪忍世苦恼。"

### 7.4 精进令他成熟

> **佛子依胜勤，成熟有情众，**
> **为他一善心，久劫亦不厌。**

四种精进（此处集中）：
- **披甲精进**
- **加行精进**
- **无厌精进**
- **利他精进**

**精进的极致**——"为他一善心，久劫亦不厌"——**为让其他众生生起一刹那善心，在千劫万劫中精进不生厌烦**。

课23："这种精神，唯一是大乘佛教才能有的，在其他任何世间非常崇高的理论或很多世间宗教里面，都不会有。"

**净精进菩萨八万四千年劝化王子公案**（课23）：
> "当时有一个十六岁的王子，他长得特别好看……特别傲慢……最后他抛弃了所有信仰，不再礼佛。当时广光明佛陀在八万四千位菩萨的群体中说：我们要想办法度化王子，但是度他需要很长的时间。后来一位名叫净精进的菩萨表示愿意去度化这个王子。于是净精进菩萨在八万四千年中，一直去王子那里进行劝化，最初王子天天谩骂他、打击他，但是菩萨一直不放弃，最后王子终于被感化。"

与此对比课23 记录的现代居士："我劝了四次！你知道吗？我劝了四次！"——相差悬殊。

**《父子合集经》教证**：
> "常以悲心观世间，普使众生脱苦厄，未尝一心生疲劳，坚固精进无退转。"

### 7.5 禅定令他成熟

> **自在得无上，他供定摧欲，**
> **令他向胜教，令善真增长。**

菩萨禅定**四层作用**：
1. 心自在——获虚空藏、首楞严等无上等持
2. 他供定摧欲——受他恭敬供养而摧毁一切贪欲
3. 令他向胜教——使无此禅定的他众向往殊胜佛教
4. 令善真增长——以神变、他心通相应说法，增上他相续善法波罗蜜多

**课23 的核心点**：
> "如果一个窍诀修一辈子，不仅一辈子乃至生生世世都会受益，不但不会被各种欲望、贪婪所转，并且自己也会对佛教生起极大信心，能很坚定地趋入佛法。"

**《大方广如来秘密藏经》教证**：
> "常乐独静处，思念于如来，善摄如是法，修行心不乱。"

**佛学院百日闭关**（课23）——即使闭关七日也要观察自心。

### 7.6 智慧令他成熟

> **解真如密义，尽除众生疑，**
> **故彼敬佛法，自他善德增。**

智慧的四层作用：
1. 解真如密义——通达了义不了义、胜义世俗、真如密意非密意
2. 尽除众生疑——根除他相续中的怀疑
3. 故彼敬佛法——自己恭敬佛法 + 将他众安置于此智慧中恭敬佛教
4. 自他善德增——自他相续善法功德增上

课23 的恭敬观：
> "对佛恭敬，一方面需要一种外在的表示，但是更重要的是内心深深被佛所折服，佩服得五体投地。"

**《胜思惟梵天所问经》教证**：
> "以智慧光明，知一切法性，无闇无障碍，是智者行道。"

### 7.7 摄义——有际平等化

> **菩萨悲利生，善趣及三善，**
> **是以下中上，有际平等化。**

**菩萨令他成熟的最终归摄**——

| 层次 | 对象 | 成熟方式 |
|---|---|---|
| **善趣（下）** | 断种姓者遇菩萨 | 安置十善→暂时成熟于善趣 |
| **三善（声闻/缘觉/菩萨三乘）** | 三种姓者 | 安置于各自之道→永乐解脱三菩提 |
| **三清净地（上）** | 大乘中二资殊胜者 | 成熟于八/九/十地 |
| **一至七地（中）** | 大乘中等者 | 成熟于不清净七地 |
| **信解行地（下）** | 大乘二资微薄者 | 成熟于资粮道/加行道 |

**"有际平等化"**——乃至世间际、尽虚空界周遍众生，等同虚空界与众生界未来际存在间，**菩萨以成熟调化众生无有穷尽**。

**地藏菩萨式的摄义**：课23："就像地藏菩萨所说'地狱不空，誓不成佛'，以这样的发愿成熟无量无边的众生，这是第九品的核心内容。"

## 八、本品核心论题归摄

### 8.1 成熟 = 大乘道的"使用前检查"

本品根本立场——**修行不是为了修行而修行，而是为了成熟到"能使用"的程度**。课21 的比喻：
> "世间人也经常这样讲：'这个人很年轻还不成熟——你看他说话那么冲，没有人生经验；这个人很成熟的——他有工作经验、社会经验。'现在很多企业不愿意招大学生，认为大学生欲望很大却没有经验，他们愿意招比较成熟的人。"

大乘道同理——**有境界不等于已成熟，成熟 = 境界 + 能使用**。

### 8.2 自成熟为令他成熟之前提

本品的根本次第——**自成熟→令他成熟**。不可颠倒：

课21 记录的现实问题：
> "今天有一个人给我讲，他刚刚出家就要下去度众生。他说：'我现在在网络上有三百多个人，我正在度化他们。'他问我的建议如何，我说：'按照我们佛学院的规矩，你可能还没有成熟。因为你自己的行为等各方面还没有经过一定的训练，自己的佛性还没有开发出来，刚出家就下去不太合理。'"

**自未成熟而急于令他成熟 = 无基大楼**。此与品6 修行品戊二己三"自己未解脱不能令他解脱"的义理一致。

### 8.3 九相的"因-体-用"三分法是本品的方法论贡献

每一相均以**因-体-用**三维展开——这是大乘论典**安立法**的精致实现。不像中观的破立、因明的论式，本论走的是**如实展开**之路：

```
某成熟相：
  因：从哪里来
  本体：是什么
  作用：能做什么
```

此方法论在后品（品17 六度四摄的每一度、品18 四无量的每一量）反复出现——**九相"因-体-用"实际上是本论方法论的典范展示**。

### 8.4 "成熟"在本品 vs foundations 的侧重不同

本论所讲"成熟"与 [`topics/foundations/practice-path.md`](../../topics/foundations/practice-path.md) 所讲的"成熟"侧重不同：

| 维度 | 本品 "成熟" | foundations "成熟" |
|---|---|---|
| 焦点 | 相续的质变（九相） | 道次第的完成（五道十地） |
| 方法 | 因-体-用三分 | 依次第修持 |
| 结果 | 能使用、能利他 | 能入圣位、获圣果 |

两者**互相补充**——foundations 讲"量"的进展，本品讲"质"的转化。

### 8.5 "受持大乘法"——一切成熟的根本

本品戊一庚一"喜成熟"指向"受持大妙法"。**《大宝积经》"恒沙诸愿摄受一大愿——摄受如来正法"** 是本品最根本的教证——一切成熟都是为"受持大乘法"的不同面向。

### 8.6 "不夺成熟"与"真如品"的衔接

本品戊一庚八"不夺成熟"的四理分析、不随他转/不被他夺定解，直接**承接品7 真如品的证悟次第**——加行道"以分别心认知"+ 见道"现量证法界"即是获得不可夺智慧的基础。**无真如的证悟/胜解，无不夺的成熟**。

## 十、与其他集合/品的交叉引用

### 10.1 本集合内部

- **`03-皈依品`** / **`04-种姓品`**：种姓是成熟的本因——有大乘种姓才有可能进入九相成熟。皈依是成熟的开始——真皈依 = 喜成熟 + 信成熟的起点。
- **`05-发心品`**：发心是"诺言" + "承诺"，但需本品"成熟"才能"兑现"。发心 → 修行 → 证悟 → 成熟 = 菩萨道完整链。
- **`06-修行品`**：修行品戊二己三"八方法"（随众/无倒说/无傲/无我所/通/忍/调/远/无尽）与本品"九相"互相指涉——品6 行的**对外方法**，本品讲**相续内在质变**。特别"调（调柔）"与本品"寂静成熟"、"忍"与"忍成熟"直接对应。
- **`07-真如品`**：真如品证悟 = 本品"不夺成熟"的基础；真如品五道次第 = 本品九相渐次展开的内在机制。
- **`08-威力品`**：威力品的外在工具（六通等）= 本品"威力成熟"的外现。

### 10.2 与其他教典

- **`collections/佛子行/`**：三十七条中"坚固的安忍"、"不舍大乘道"等条目与本品九相对应——佛子行提供实操条目，本品提供义学骨架。
- **`collections/修心七要/`**：修心七要"座间/座下"修法与本品"寂静/忍/精进"成熟相对应。
- **`collections/入行论/品07-精进.md`**：入行论精进品的"铠甲精进"等与本品戊二己三"精进令他成熟"的四种精进义同。
- **`collections/定解宝灯论/`**：定解宝灯论对"随法信/随信者"的辨析直接用于本品"信成熟"之判准。
- **`collections/宝性论/03-如来藏-总纲十义.md`**：本品"自坚（安忍种子）"、"悲成熟自慈"皆指向如来藏的内在种性——两论互补。
- **`topics/foundations/bodhicitta-methods.md`**：本品"自成熟→令他成熟"与菩提心修学"愿→行菩提心"的结构对应。
- **`topics/foundations/practice-path.md`**：本品九相成熟是foundations道次第图谱中"相续成熟"环节的最详细论述。

### 10.3 关于"成熟"与"波罗蜜多圆满"的区分

**本品 vs 品17 六度四摄**的区别——

| 本品（品9）| 品17 六度四摄 |
|---|---|
| 以六度为"令他成熟相续"的加行 | 以六度为"圆满自佛法"的波罗蜜多 |
| 成熟 ≠ 圆满，是中间环节 | 圆满 = 波罗蜜多到彼岸 |
| 重点：成熟他人相续 | 重点：自成佛法 |
| 六度是"工具化"使用 | 六度是"本体"体证 |

两品的六度**义涵不同**——**品17 讲六度的本体波罗蜜多意义**，本品讲六度作为"令他成熟之加行"的应用。不要混淆。
