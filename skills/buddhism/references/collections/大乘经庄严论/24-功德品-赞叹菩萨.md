# 品20 功德品·赞叹菩萨

## 一、本档在全品中的位置与范围

### 1.1 承接上档

品20「功德品」共分两大部分——**己一 修行之功德**与**己二 赞叹之功德**。上档 [`23-功德品-修行之功德.md`](./23-功德品-修行之功德.md) 处理己一的十七种修行功德（课95-102，含"说法安立"完整段），**本档处理己二"赞叹之功德"的全部内容**，对应课103-104 两堂课，密意疏约 140 行（全品 585 行中的最后段）。课105 起进入品21 行为究竟品，见 [`25-行为究竟品.md`](./25-行为究竟品.md)。

### 1.2 本档之核心与品全的收结

己一展示的是**菩萨修行的各种功德**，己二则以**赞叹的方式再度呈现这些功德**——"为了让众生对菩萨功德更加生起信心而赞叹"。本档含三大块：

1. **赞叹对境菩萨之分类（庚一）**——把从信解行地到十地的菩萨归纳为**五种分位**，为下面的赞叹提供"对谁赞叹"的清晰图谱
2. **以因波罗蜜多心力不退赞叹（辛一）**——从修行因位的"**九种菩萨**"角度赞叹，加上**十六种菩萨异名**的总赞
3. **从证悟菩提因果赞叹（辛二）**——以"名副其实"的方式再度赞叹，连结菩萨"名"与"实"的对应

因此本品即是"**先分类，再赞叹因，再赞叹果**"——以九种菩萨 + 菩萨异名 + 证悟五方面 + 菩萨十一分类展开。

## 二、本档在赞叹体裁上的特色

### 2.1 每颂皆以六度作暗线贯穿

本档的九种赞叹（课103-104）虽从九个不同侧面赞叹，但**每种赞叹的颂词都暗含六度**。索达吉堪布在课104 开头即点明："以六波罗蜜多的方式来赞叹九种菩萨"——即九种赞叹各自以六度为内里展开，每一种赞叹有六句（六度），九种共计六十四度面向。这是本论作为"弥勒菩萨庄严诗学"的典型修辞——用不同的修饰角度赞同一菩萨。

### 2.2 与 "己一" 的功德呼应

己一十七功德中：庚四饶益、庚六希求、庚七不唐捐、庚八真实行、庚十五恒常所为等，皆从"菩萨之作"角度说；己二九种赞叹从"菩萨之相"角度说——**二者皆以六度为单位，但一为"动词式"（做什么），一为"形容词式"（是什么）**。如：

| 己一（修行功德） | 己二（赞叹功德） |
|---|---|
| 庚五报恩：自住六度 + 令他行持 | 9. 行饶益菩萨：以六度饶益此生他世 |
| 庚八真实行：无求施、不求后世戒等 | 1. 喜菩提菩萨：不贪、三业净、伏嗔等 |
| 庚十五恒常所为：不著禅味等 | 7. 不放逸菩萨：受用/调柔/守护/行善/乐法不放逸 |

故两档读完可见菩萨功德的**"做+是"双重图谱**。

### 2.3 修辞学上的诗韵特色

课103-104 中弥勒菩萨使用了几种**藏印诗学修辞手法**：

- "**除苦不令苦，忍苦不畏苦，脱苦不计苦，承受苦菩萨**"——同字反复修辞，一颂连用七个"苦"字
- "**不喜喜性法，诋法真勤法，自在法无暗，首行法菩萨**"——一颂连用六个"法"字
- "**受用与调柔，守护与行善，乐法不放逸，不放逸菩萨**"——每句结尾皆"不放逸"
- "**不敬微过惭，不忍退失惭，流散小见惭，他乘惭菩萨**"——每句结尾皆"惭"

这种反复修辞既是艺术也是修法——让学人在颂词的反复中深嵌赞叹内涵。

## 三、庚一 赞叹对境菩萨之分类——五种分位菩萨

**一即具信解，增上意乐净，**
**有相及无相，无有现行行，**
**当知此五种，诸地之菩萨。**

从**信解行地到十地**共计十一地者皆称菩萨，以五种分位差别所摄：

| # | 名称 | 位置 | 特点 |
|---|---|---|---|
| 1 | **具信解菩萨** | 住信解行地（资粮道+加行道） | 没有现前法界遍行法相，但对法界有诚挚信 |
| 2 | **增上意乐净菩萨** | 一地（入一地） | 证法界遍行法相，获自他平等性之心，具足肩负他利之增上意乐及清净见断障碍 |
| 3 | **有相菩萨** | 二地至六地 | 还有各阶段不同的相执——见下表 |
| 4 | **无相菩萨** | 七地 | 断除了六地以前所有相状 |
| 5 | **无现行行菩萨** | 八/九/十地（三清净地） | 以无分别等持无勤任运行持自他二利 |

**有相菩萨之各地差别**（庚一科判下二地至六地菩萨的细分）：

| 地 | 有的相 |
|---|---|
| 二地 | 行持辨别声闻、缘觉、菩萨乘种种相 |
| 三地 | 有俱生的萨迦耶见相 |
| 四地 | 视苦谛、集谛为有害相，视灭谛、道谛为寂灭相（四谛相） |
| 五地 | 趋入种种工巧明并分别其相 |
| 六地 | 分别顺式逆式缘起相 |

对于七地"无相"之名，堪布课103 引《入中论》："彼至远行慧亦胜"——七地菩萨的智慧已超越前六地，故名"无相"菩萨。此与本论的"七地无相"在安立上相呼应。

### 3.1 五种分位的修行意义

**我们在哪个位置**？堪布课103："这五种菩萨除了信解行地菩萨，我们都没办法比。但我们相信诸佛菩萨有这样的境界、功德，我们也发愿将来获得这样不可思议的功德和智慧。"

——初学者的主要任务是：
1. **自身位置** = 具信解菩萨（相信之信解）
2. **所向往境界** = 从增上意乐净至无现行行的四地菩萨果位
3. **正确态度** = 相信诸佛菩萨有此境界，发愿将来获得

## 四、庚二辛一壬一 九种菩萨赞叹

九种赞叹每一颂都含六句（六度），每句以特定形容词对六度之一展开。下面按课103-104 的顺序一一解读。

### 4.1 从欢喜菩提赞叹——喜菩提菩萨（课103）

**不贪三业净，伏嗔勤胜德，**
**法不动见真，喜菩提菩萨。**

| 度 | 赞叹之相 |
|---|---|
| 布施 | **不贪**——不贪五欲妙的受用而行施舍内外事物 |
| 持戒 | **三业净**——身语意三业无有不善（身不杀盗淫，语不妄粗绮恶，意无贪害邪见） |
| 安忍 | **伏嗔**——安住忍辱而胜伏嗔恨 |
| 精进 | **勤胜德**——精勤行持出世间善法殊胜功德 |
| 禅定 | **法不动**——禅定一缘安住法中不动摇 |
| 智慧 | **见真**——以智慧见二无我甚深真如 |

**赞叹主题**：行持波罗蜜多是**欢喜获得大菩提果**，为此修行、回向于大菩提。故名"欢喜菩提之菩萨"。此是**六度清净之功德**。

> 《出曜经》："众施法施胜，众乐法乐上，众力忍力最，爱尽苦谛妙。"（课103 引）——在所有布施中法布施最殊胜；所有快乐中法乐最上；所有力量中忍力最强；断爱之后也能断苦。这四句是"喜菩提"修行的方法论支撑。

### 4.2 从利他赞叹——勤他利菩萨（课103）

**欲利无害见，堪受他害坚，**
**不放逸多闻，勤他利菩萨。**

| 度 | 赞叹之相 |
|---|---|
| 布施 | **欲利**——想以布施利益众生 |
| 持戒 | **无害见**——以持戒不害他众生命及受用；无以妄语欺骗等三门任何损害的见解 |
| 安忍 | **堪受他害**——安忍堪受他害 |
| 精进 | **坚**——发起稳固不退的精进 |
| 禅定 | **不放逸**——安住寂静无散乱，不著禅味 |
| 智慧 | **多闻**——以多闻妙法的智慧 |

**赞叹主题**：以六度**精勤利益他众**——一切波罗蜜多皆为暂时与究竟利益一切有情而回向无上菩提。故名"勤他利菩萨"。这是**从利他功德角度赞叹**。

堪布在课103 强调："持戒是不害，大乘佛教中不害很重要。害的方法各种各样，有些用刀子，有些用语言，有些用行为来扰乱别人。" ——"我们至少要行持一种无害的行为，作为大乘菩萨，即使自己暂时利益不了众生，但也要先做一个不害众生者，这很重要。"

### 4.3 从内安住赞叹——住内菩萨（课103）

**知执过不贪，不恨瑜伽通，**
**三相无恶见，乃住内菩萨。**

| 度 | 赞叹之相 |
|---|---|
| 布施 | **知执过不贪**——了知将受用执为我所之过患（无常、苦、如幻；贪感恶趣），不贪欲妙 |
| 持戒 | 不贪欲妙受用，弃家而出家守持戒律 |
| 安忍 | **不恨**——对他人所作损害不怀恨在心而安忍 |
| 精进 | **瑜伽**——精勤于大乘止观瑜伽 |
| 禅定 | **通三相**——修行一缘等持而精通寂止相、受持高昂相、等舍相三相 |
| 智慧 | **无恶见**——具足无有我见等恶见的智慧 |

**赞叹主题**：**从真实安住于内**——菩萨之修行重在向内调伏，不沦于形象。故名"住内菩萨"。

**禅定三相**（重要概念）：
- **寂止相**——对境散乱时的对治，如思维轮回过患、断欲妙贪执
- **高昂相**——昏沉睡眠时的对治，如思维佛菩萨功德振作精神
- **等舍相**——寂止与高昂平衡的一缘安住

课103 堪布补充《大般涅槃经》："乃至小罪心生怖畏，护戒之心犹如金刚。"——哪怕小戒也应像护眼目般护持，受戒之心应坚如金刚。"住内"菩萨的核心是细微的持戒和细微的禅观。

### 4.4 从一心不乱赞叹——乘不乱菩萨（课103）

**怀慈知惭德，受苦不贪乐，**
**正念入定性，乘不乱菩萨。**

| 度 | 赞叹之相 |
|---|---|
| 布施 | **怀慈**——对苦难众生怀慈悯心能布施 |
| 持戒 | **知惭德**——安住知惭功德而守戒 |
| 安忍 | **受苦**——甘愿忍受他者损害等痛苦 |
| 精进 | **不贪乐**——不贪恋睡眠懒惰等自己的安乐而精进利他 |
| 禅定 | **正念**——恒具正念，不随散乱所转 |
| 智慧 | **入定性**——恒具诸法远离有无边平等性的智慧，故是极其入定的本性 |

**赞叹主题**：**于大乘中心不烦乱**——大乘道修行一心不乱。故名"乘不乱菩萨"。从大乘道一心不乱功德的角度赞叹。

安慧论师《注释》此处判：前三句讲修行六度，第四句讲回向无上菩提。

### 4.5 从苦赞叹——承受苦菩萨（课103）

**除苦不令苦，忍苦不畏苦，**
**脱苦不计苦，承受苦菩萨。**

| 度 | 赞叹之相 |
|---|---|
| 布施 | **除苦**——依布施遣除他众贫穷之苦 |
| 持戒 | **不令苦**——依持戒不以杀生偷盗等令众生痛苦 |
| 安忍 | **忍苦**——安忍自己的一切痛苦 |
| 精进 | **不畏苦**——为利他入轮回精勤利众生而不畏惧痛苦 |
| 禅定 | **脱苦**——获静虑等持而解脱欲界痛苦及意苦 |
| 智慧 | **不计苦**——以智慧不分别痛苦相（不计三轮：受苦者/所受苦/受三者） |

**赞叹主题**：**从承受痛苦的层面**——不舍苦难轮回及不以痛苦厌烦。故名"承受苦菩萨"。

**全颂连用七个"苦"字**的修辞特色——"前面的三句每句有两个，后面有一个，共用七个'苦'字宣说了菩萨不可思议、不舍轮回的精神。"（课103）

**对治软弱心**：堪布在课103 以"没有四肢还开朗的尼克"为例警策："有些特别难忍的痛苦我们也许忍不了，但一般的痛苦，不管是心里的痛苦还是身体的痛苦，我们并非不能忍，主要看你的内心是否坚强。" 引《四百论》："胜者为意苦，劣者从身生，即由此二苦，日日坏世间。"

### 4.6 从行法赞叹——首行法菩萨（课104）

**不喜喜性法，诋法真勤法，**
**自在法无暗，首行法菩萨。**

| 度 | 赞叹之相 |
|---|---|
| 布施 | **不喜**（悭吝法）——不喜悭吝，喜欢施舍（布施） |
| 持戒 | **喜性法**——喜欢无过功德自性的戒律法 |
| 安忍 | **诋法**（嗔恨法）——诋毁嗔恨法 |
| 精进 | **真勤法**——真实精勤善法 |
| 禅定 | **自在法**——以法自在等持的威力获得心自在 |
| 智慧 | **法无暗**——具足于诸法无有愚痴黑暗的智慧 |

**赞叹主题**：**首要行持真实法**——不以财为主而以主要行持妙法。故名"首行法菩萨"。

**全颂连用六个"法"字**的诗学修辞。堪布课104："上个颂词有很多'苦'字，这个有很多'法'字，弥勒菩萨用不同的词来赞叹菩萨的功德。"

### 4.7 从不放逸赞叹——不放逸菩萨（课104）

**受用与调柔，守护与行善，**
**乐法不放逸，不放逸菩萨。**

| 度 | 赞叹之相 |
|---|---|
| 布施 | **受用不放逸**——不浪费财富，用于布施 |
| 持戒 | **调柔不放逸**——具足戒律三门调柔，不染佛制罪与自性罪 |
| 安忍 | **守护不放逸**——不以他者作害扰乱相续，不还以损害反而报以饶益 |
| 精进 | **行善不放逸**（隐含"不懈怠"）——精进行持世出世善法 |
| 禅定 | **乐法不放逸**——获禅定安乐而无有著味之爱 |
| 智慧 | **智慧不放逸**（隐含）——无倒证悟无常等，不执常乐我净 |

**赞叹主题**：**具足如理作意大乘法之不放逸**。故名"不放逸菩萨"。

**"调柔不放逸"的重要性**：安慧论师《注释》中"调柔"一词较其他版本更准确——**不染佛制罪与自性罪的三门调柔才是真正的不放逸**。堪布课104 幽默提及："很多人一听说一个星期没有课，就非常开心，这是一种菩萨的放逸"——以自嘲方式警策放假心态。

### 4.8 从惭愧赞叹——他乘惭菩萨（课104）

**不敬微过惭，不忍退失惭，**
**流散小见惭，他乘惭菩萨。**

**反向赞叹——"菩萨非应做之事，为之即惭愧"**。每句都是"X 惭"——若做 X 则惭愧：

| 度 | 应惭愧之事 |
|---|---|
| 布施 | **不敬**——遇乞丐不布施，甚至以脸色谩骂不敬态度 |
| 持戒 | **微过**——微细的佛制罪自性罪，看不到微小罪过 |
| 安忍 | **不忍**——对他人作害不安忍、相续烦乱 |
| 精进 | **退失**——从行持善法中退失而懈怠 |
| 禅定 | **流散**——身处愦闹、心流散于外境、散乱 |
| 智慧 | **小见**——离小见（仅人无我、视法为相状）——即声缘乘之见 |

**赞叹主题**：不趋入小乘他乘的菩萨。故名"**他乘惭菩萨**"。

> 就像诗学中常用反义词一样，这是**间接赞叹菩萨的精神**——菩萨具足惭愧心故不为上述事。间接说明菩萨具足以上六波罗蜜多的境界。

**实践要点**：堪布课104 引《入中论》例——"甚至自己的肉被一块块切下来，菩萨不但不生嗔心，还以欢喜心对其回向"——普通菩萨做不到，但"听别人说几句马上就反击，也是菩萨该惭愧的"。

### 4.9 从饶益赞叹——行饶益菩萨（课104）

**此他与等舍，具行得自在，**
**相应说大果，菩萨行饶益。**

| 度 | 饶益之相 |
|---|---|
| 布施 | **此（今生有利）**——以布施为贫困者施舍受用，饶益此生 |
| 持戒 | **他（他世有利）**——将他们安置戒律饶益他世 |
| 安忍 | **等舍**——不扰相续，不报损害而平等饶益众生 |
| 精进 | **具行**——作为众生事助伴的精进或行持而饶益 |
| 禅定 | **得自在**——以禅定得神通等功德自在而饶益 |
| 智慧 | **相应说**——随众生信解相应宣说五明尤其内明三乘法 |

**赞叹主题**：以因果行持饶益一切有情——六度大果是获得佛果，乃至虚空际间饶益有情。故名"**行饶益菩萨**"。

**安慧论师判**：此颂是归纳上面的意义——可理解为九种赞叹的**总收结**。

## 五、庚二辛一壬二 宣说菩萨异名——十六种共称

**菩萨摩诃萨，具慧胜光明，**
**佛子佛陀基，能胜如来芽，**
**具力及圣尊，商主与胜称，**
**悲尊大福德，自在及具法。**

菩萨有十六种共称的异名，皆从功德角度安立——**名称与意义相应，不像世间乞丐名字叫"富人"**（课104）。

| # | 名称 | 含义 |
|---|---|---|
| 1 | **菩萨（菩提萨埵）** | 一心虔信二无我自性甚深所知；修行无上菩提意乐坚稳 |
| 2 | **摩诃萨（大菩萨）** | 自在拥有大威力，一汗毛孔能示现三千大千世界成坏劫；或欲求救一切众生远离轮回苦故为大菩萨 |
| 3 | **具慧** | 具足深广智慧 |
| 4 | **胜光明/胜威光** | 解甚深密意，法光放射十方 |
| 5 | **佛子** | 不间断秉承如来种族，生于如来种姓 |
| 6 | **佛基** | 是成佛之因 |
| 7 | **能胜** | 能克胜自他相续具有的一切烦恼 |
| 8 | **如来芽** | 成为胜者佛陀之法——如叶如花如果般的三身、十力、四无畏等之因 |
| 9 | **具威力（勇士）** | 能降伏魔众并越过无明习气地 |
| 10 | **圣尊** | 于一切世间中堪为最稀有 |
| 11 | **商主** | 能令入于轮回海者抵至涅槃城 |
| 12 | **大称/胜称** | 功德美名传扬于十方浩瀚刹土 |
| 13 | **大悲尊者/悲尊** | 具足不舍弃一切有情的大悲 |
| 14 | **大福德** | 具足布施等无量善根 |
| 15 | **自在** | 自在拥有世间出世间一切圆满 |
| 16 | **具法** | 具足无罪无漏法 |

另补："以无畏、勇士、坚稳、众尊等种种异名称呼"——故十六只是主要异名，非穷尽。

### 5.1 关于"菩萨"与"摩诃萨"的差别

密意疏列出几种解释：
- 或修行因故菩萨，救度众生故摩诃萨
- 或住轮回不染故菩萨，发精进故摩诃萨
- 或发心故菩萨，具七大故摩诃萨
- 或观待诸地：前前为菩萨，后后为大菩萨
- 或信解行地为菩萨，圣地为大菩萨
- 或不清净七地为菩萨，清净地为大菩萨

**重要结论**："这两者仅以反体而分的，实际菩萨、摩诃萨二者无有差别成为菩萨的总称。"——二者无本质差别，皆是菩萨的称谓。

### 5.2 异名的体系含义

这十六种异名的**分层**：

- **基础层**：菩萨、摩诃萨、佛子——菩萨的身份名
- **因位功德层**：具慧、胜光明、佛基、能胜——菩萨在因位的本质特征
- **前果层**：如来芽、具力、圣尊、悲尊、大福德——已接近果位但未究竟
- **事业层**：商主、胜称、自在、具法——利他事业

此十六名既可见菩萨的"**种姓、功德、事业**"三位一体。堪布在课104 指出《大乘经庄严论》最常用的几个名字是——佛子、具慧、勇士、无畏者、坚稳者。

## 六、庚二辛二 从证悟菩提因果侧面赞叹菩萨名副其实之功德

辛二是对菩萨从"**名实相符**"角度再度赞叹——既然菩萨的名号是根据功德安立的，那么**菩萨所证悟的内容必然与其名号相应**。辛二分五段（课104），每段以"是故名菩萨"作为结句。

### 6.1 五种证悟差别（第一段）

**证真证大利，证诸义恒证，**
**以及证方便，是故名菩萨。**

五种功德对应菩萨之名：

| # | 证 | 内容 |
|---|---|---|
| 1 | **证真** | 证悟法无我与人无我，证悟善妙真如 |
| 2 | **证大利** | 证悟应成办自他大利 |
| 3 | **证诸义** | 普皆证悟有为、无为、有漏、无漏所摄的一切所知义 |
| 4 | **恒证** | 非偶尔证悟、非有朝一日将灭尽，而是**于无余涅槃法界中智慧也不穷尽**，恒常诸时证悟 |
| 5 | **证方便** | 证悟调化众生如何幻化如何说法等调化的无量方便 |

"以这五种功德为特点，因此名为菩萨。"

### 6.2 证心意识四差别（第二段）

**随我证细见，证悟种种识，**
**证非真分别，是故名菩萨。**

梵语"菩提"（bodhi）是通达之义，"萨埵"（sattva）是具有思维之义——符合思维、通达之定义的功德。此颂证**心（阿赖耶）、意（染污意）、识（六识）**三者之差别：

| # | 证 | 对应 |
|---|---|---|
| 1 | **随我（证心）** | 证阿赖耶识中出现一切器处外境身体相；了知"我"无他法就是自相续阿赖耶识刹那持续，以染污意向内观缘于我 |
| 2 | **细见（证意）** | 染污意缘阿赖耶执著"我"，从属五遍行及我见/我贪/我痴/我慢四者。坏聚见是一切恶见烦恼之因 |
| 3 | **种种识** | 了知六识聚自性——境、根、作意三者所生眼识至意识缘色等六境，连同心所善不善无记相应具受之种种 |
| 4 | **非真分别** | 证第八识——二取本无而分别为有，非真实的分别依他起的自性如幻如梦 |

"由这四种原因，称为菩萨。这是证悟**世俗现相**之理。"

### 6.3 证二谛五法（第三段）

**证未证随证，证无实证生，**
**各别证未证，是故名菩萨。**

五种证悟：

| # | 证 | 对应 |
|---|---|---|
| 1 | **证未证** | 证世间人前所未证的无缘如虚空般的无分别智慧（入定智） |
| 2 | **随证** | 以后得的智慧证诸法等同幻术；或了知顺式缘起是证未证，了知还灭缘起是各别证随证 |
| 3 | **证无实** | 证自相法不存在的遍计法 |
| 4 | **证生** | 了知一切所取能取显现依他起以刹那方式产生 |
| 5 | **各别证未证** | 了知凡夫异生前所未知的法性二空圆成实 |

"以各别证悟上述五种意义的缘故，名为菩萨。这也是从证悟二谛方式而言的。"

### 6.4 证三自性二谛五法（第四段）

**证实无胜义，证诸义所有，**
**证所证证基，是故名菩萨。**

证悟二谛之理的其余五种差别：

| # | 证 | 对应 |
|---|---|---|
| 1 | **证实无** | 证依他起识中产生显现三界一切所取能取，耽著它而持续感受痛苦，但所取能取实际并不成立 |
| 2 | **证胜义** | 证二空自性圆成实 |
| 3 | **证诸义** | 证一切二现法仅是遍计，实际无有 |
| 4 | **证所证** | 证蕴界处等尽所有所摄的一切法 |
| 5 | **证基** | 证所证之境、能证之基证悟者及证悟三轮无缘，即证悟胜义 |

"由证悟如是三自性及二谛五法的缘故，称为菩萨。"

### 6.5 证最后有菩萨五相（第五段）

**证成就证处，证胎证次第，**
**证悟除怀疑，是故名菩萨。**

从最后有菩萨五种差别来名为菩萨：

| # | 证 | 对应 |
|---|---|---|
| 1 | **证成就** | 证悟成就无上大菩提，即证悟所现前三身大密严 |
| 2 | **证处** | 最后有菩萨了知住于兜率天饶益诸天人 |
| 3 | **证胎** | 了知在赡部洲入于母胎并于彼处成熟有情 |
| 4 | **证次第** | 了知入胎后降生、于王宫享用五种妙欲、离开皇宫出家、苦行、于金刚座以化身示现证得菩提（佛十二相） |
| 5 | **证断疑** | 证悟成佛后能遣除众生怀疑而转妙法轮 |

"以这五种缘故，称为菩萨。这是从最后有菩萨了知于密严刹土以报身所依现前大菩提后，以化身在赡部洲示现大菩提相的角度宣说。"

### 6.6 五种证悟段落的宏观排序

辛二壬一的五段证悟赞叹从**"证悟的对境"**逐层深入：

1. 第一段——总说五种证悟（真如、自他大利、诸义、恒证、方便）
2. 第二段——证心意识（世俗现相之理）
3. 第三段——证二谛五法（入定智 + 遍计/依他起/圆成实）
4. 第四段——证三自性与二谛（再次归纳）
5. 第五段——证最后有菩萨（从因位转入佛果位前的十二相）

此五段的走向是**从"菩萨一般性证悟"逐步过渡到"最后有菩萨之佛事示现"**，与品21 行为究竟品的"究竟之相"（[`25-行为究竟品.md`](./25-行为究竟品.md)）自然衔接。

## 七、庚二辛二壬二 分类——菩萨十一种差别

**具慧慧证得未得，善住证悟随证悟，**
**妙说无说有无我，未熟真实成熟慧。**

从**十一种分类**（十一种差别之证悟）的角度称为菩萨：

| # | 差别 | 证悟类型 |
|---|---|---|
| 1 | **过去（得）** | 具慧菩萨了知过去法，慧知已得（过去法各自实法已得到） |
| 2 | **未来（未得）** | 证未来将生之法，知未得（彼法现在自实法尚未成立或获得） |
| 3 | **现在（善住）** | 证现在法，善住（已生未灭） |
| 4 | **内** | 菩萨以自智证悟无常痛苦等，知内（证悟内在自心） |
| 5 | **外** | 由其他善知识宣说之因而证悟（随外缘） |
| 6 | **粗** | 善妙宣说诸法自相共相的智慧——依名称、词句、文字的粗大证悟 |
| 7 | **细** | 内在入定的智慧——无可言说无分别安住的细微证悟 |
| 8 | **贤/劣/有我** | 七地前有我执之细微因，是有我智慧，是平庸道的下劣证悟；或信解行地的证悟 |
| 9 | **殊胜/无我** | 八地以上无分别智慧任运自成，无我执细微因，是善妙证悟；或圣地断除所取能取 |
| 10 | **远/未熟慧** | 七地前智慧，未获无勤任运无分别智，远智慧 |
| 11 | **近/真实成熟慧** | 八地以上智慧，无分别任运自成，近智慧（接近佛地）；或信解行地未成熟 + 圣地成熟 |

十一种差别中（1-3）是**时间分类**、（4-5）是**内外因缘分类**、（6-7）是**粗细分类**、（8-11）是**成熟度分类**。这十一种差别以时间、空间、因缘、成熟度四个维度完整刻画菩萨的修行图谱。

### 8.3 五种菩萨与九种赞叹的实修要点

**五种菩萨（庚一）** + **九种赞叹（辛一壬一）**为修行者提供了**自我定位与努力方向**：

**自我定位**（我在哪里？）：
- 初学者→ 具信解菩萨
- 初证空性 → 增上意乐净菩萨
- 中阶修行者 → 有相菩萨（二至六地）
- 高阶修行者 → 无相菩萨、无现行行菩萨

**努力方向**（我要具备什么？）：
- 欢喜菩提——对大菩提欢喜
- 勤他利——一切波罗蜜多为他众回向
- 住内——内在调伏而非外相
- 乘不乱——大乘道一心不乱
- 承受苦——不舍苦难轮回
- 首行法——以法为根本而非以财
- 不放逸——时时警觉
- 他乘惭——不求声缘乘
- 行饶益——利他之具体行

实修者可对照此九种每日自省——"我今日是哪种菩萨相？缺哪种？"这是本品**"赞叹"的真正实用含义**——不是仪式性的歌颂，而是**指示的明镜**。
