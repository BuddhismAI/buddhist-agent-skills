# 品12 求法品——前行五品中规模最大的一品

## 一、本品在论中的地位与结构

品11 信解品"仅有信解尚不足够，必须要以闻思修慧寻求所信解的大乘深广法"（密意疏开篇），故宣说求法品。**信解 → 求法**是前行五品的第一次递进——信解是因（对大乘生心），求法是第一次实地展开（以闻思修实际追寻）。

本品是前行五品中**规模最大的一品**：
- 课程：课40-50，共十一课
- 密意疏：630 行（本集合中密意疏行数最多的前行品）
- 颂数：覆盖三藏十二部安立、十一种证法寻求、五十一作意、求法果与分别——从教义广度而言近乎"小型总纲"

### 1.1 本品科判

      辛二 寻求获得所缘所说义（以三慧得三所缘）
      辛三 寻求能求作意（十八瑜伽作意）
    庚二 寻求所证义法（十一寻求）

### 1.2 本品的结构性特点

- **"所求之法"两类**——教法（三藏）与证法（十一种抉择）。**教法是所依文本，证法是所证义理**。所求必须兼及两类，缺一不可。
- **"十一寻求"实为小型教义总纲**——涵盖真如、如幻、所知、染净、唯识、法相、解脱、无本性、无生法忍、一乘密意、五明十一项。几乎等于本论的一份浓缩教义地图。
- **"五十一作意"的独创性**——把"如何作意"从三种（闻思修）细化为五十一种，是瑜伽学派精密心理学的典型表现，本论独有。
- **唯识立场的集中表达**——辛五寻求唯识、辛六寻求法相、辛七寻求解脱都直接以唯识三自性与转依框架建立，是全论唯识立场最显性的章节。

## 二、己一 所求之法——庚一 寻求所说教法

### 2.1 辛一 寻求所缘经典（三藏与九因）

颂词：
> **三藏二亦可，摄故因许九。**
> **种习了悟寂，彻证令解脱。**

**三藏**：梵语"毗支迦"译为藏——"藏"者含摄众多所诠义。经、律、论三藏，或摄为**大乘菩萨藏 + 小乘声闻藏**两种。

**十二部**——"契经应颂记别及讽诵，自说因缘譬喻与本事，本生方广希法与论义"。其中**方广与本生属菩萨藏**，其余大多数属声闻与大乘共有。十二部也归于三藏中。

**安立三藏之九种因**：

| 角度 | 三藏 | 对治/所学/所知 |
|---|---|---|
| **观待所断** | 经藏 | 对治疑惑烦恼 |
| | 律藏 | 对治享乐边与苦行边之随烦恼 |
| | 论藏 | 对治我见我慢 |
| **观待所学** | 经藏 | 宣说三学 |
| | 律藏 | 宣说戒学与定学 |
| | 论藏 | 宣说慧学 |
| **观待所知** | 经藏 | 说法与义 |
| | 律藏 | 修法与义 |
| | 论藏 | 无误宣说通达法与义 |

**律藏遮二边**——索达吉堪布课40 举愿文："不要转生到太富裕的家里，不然可能没办法修行，也不要转生到太贫穷的家里，不然也可能没办法修行和出家，但愿我转生到一个中等的家里，恒常有缘出家。"律藏的本质是**贪欲对治**——不开许享乐（他胜、僧残、堕罪、恶作），也不鼓励极端苦行（裸体、五火等外道苦行）。关键不在衣食贫富，而在**有无贪心**：无贪时享用百味食品、价值百千的衣服也无罪；有贪时吃粗茶淡饭、穿粪扫衣也有罪。**蒋阳钦则旺波与华智仁波切的公案**：华智仁波切见上师房子如皇宫而起邪见，蒋阳钦则旺波以他心通言："我对这些财富的贪著，根本没有你对你怀里木碗的贪著大"——无贪则富无罪，有贪则贫有罪。

**求法必要**——"以三慧寻求并实行，通过以听闻在自相续**种下解脱道的习气**、以思维对其意义**获得定解**的方式**了悟**，以修行依靠寂止**寂灭散动分别念**，依靠胜观**完全证悟真如**，而令自相续**解脱**"。——即颂中"**种习了悟寂，彻证令解脱**"一句所概括的五步：**种习**（闻）→ **了悟**（思）→ **寂**（止）→ **彻证**（观）→ **解脱**（果）。

### 2.2 三藏各自的四义

颂词：
> **经对法及律，略摄许四义，**
> **具慧了知彼，获得一切智。**

**经藏的四义**：
> **说处法相法，及义故是经。**

| 四义 | 内容 |
|---|---|
| **处** | 舍卫城等说法地点 + 出有坏说法者 + 请法众生三者 |
| **法相** | 二谛——胜义谛与世俗谛 |
| **法** | 蕴界处缘起等具体法 |
| **义** | 意趣非意趣、秘密非秘密之辨 |

课40 索达吉堪布对**二谛**补充各宗分类：
- 世间人：眼见为实的胜义 / 虚无飘渺的世俗
- 经部/有部：最细微心识与无方微尘是胜义 / 粗大现相是世俗
- 唯识宗：虚妄现象是世俗 / 自明自知的圆成实是胜义
- 中观派：离四边八戏是胜义 / 如梦如幻显现是世俗

**经藏的本质**：这四种意义的综合使"素旦"（梵语经）能**略摄具足**一切教义。

**论藏（对法/阿毗达磨）的四义**：
> **对向故反复，胜证故对法。**

| 四义 | 内容 |
|---|---|
| **对向** | 通过菩提分/缘起等抉择令众生趋向涅槃 |
| **反复** | 对一一法反复辨析（有色/非有色、有漏/无漏等） |
| **胜伏** | 以辩论胜伏反方 |
| **证悟** | 以正理决定方式抉择经义而现证 |

课40 举喻："学习《俱舍论》的人辩论应该不错……如果对《俱舍论》的法义了解得很好，这样的道理再通过因明的逻辑来整合，然后进行推理就更加精彩。"论藏是**教义的逻辑化、辩论化、抉择化**。

**律藏的四义**——律藏的颂词是：
> **堕罪犯起堕，离堕数取趣，**
> **制戒及类别，决定故为律。**

律藏的四义有**两种安立**（共八项）：

第一种四义（聚焦堕罪生灭过程）：
- **堕罪**——五篇：他胜、僧残、堕罪/粗罪、恶作、向彼悔（或他胜、僧残、堕罪、向彼悔、恶作）
- **生堕/犯堕之因**——四种：不知法相、烦恼、放逸、不敬学处
- **起堕/出堕**——以意乐后悔、严禁今后再犯而从罪中起
- **摈堕/离堕**——七种离堕法

**七种离堕法**（依密意疏）：
1. **分别忏**：于一位或四位比丘等前发露忏悔
2. **承诺处治**：无覆心犯他胜者后悔忏悔，同意作众中最末位处罚，再传戒
3. **以变异而摈堕**：佛制时未开者因缘变而开（如行途开过午食）
4. **解**（僧众聚会开解细微戒）——佛涅槃时阿难因忧伤未问开解，后迦叶等作结集时开许。一切有部共许开解细微戒；但安慧论师《大疏》说"僧众聚会开解细微戒前所未有，现在也无有"。课40 索达吉堪布折中："有这种历史和说法"，立场留白。
5. **变性而离堕**：比丘变比丘尼（反之亦然），性相异则戒相异，前戒所犯自然离堕
6. **现证而离堕**：思择四法印义，以智慧见义而从小罪中出离
7. **得法性而离堕**：见谛既从微细罪解脱、亦得不再犯之法性

第二种四义（聚焦戒律安立过程）：
- **数取趣**（依人）——依某人而制戒（如依妙授制淫戒）
- **制戒**——以僧众请问佛陀而制
- **类别**——以差别详解
- **决定**——抉择"此成堕罪/此不成堕罪"

### 2.3 辛二 寻求获得所缘所说义（三慧得三所缘）

颂词：
> **承许所缘法，内外及二俱，**
> **以二义得二，二者不可得。**

**所缘法的三分**：
- **内所缘**：自相续的蕴、内六处（能取）
- **外所缘**：非自相续所摄的外六处（所取）
- **二俱所缘**：外内二俱所摄的染净法

**获得所缘的加行道阶段**：
- **忍位**：现量证得"所取外境不存在"
- **胜法位**：证得"能取也无自性"
- **一地**：获得二者所缘即**不缘诸法的真如**
- 暖位、顶位：以义共相接近，非现量，故不称"获得所缘"

获得法所缘之理（以三慧分）：
> **依意诠说义，净信显现义，**
> **认定唯言诠，心名安住故。**
> **以闻等三慧，获得法所缘。**

| 三慧 | 获得方式 |
|---|---|
| **闻慧** | 依有分别意按经典所说义、欢喜清净信心后如是受持 |
| **思慧** | 认定名言所诠习气之外，外境的本体不可言说，除自己分别念外无有外境 |
| **修慧** | 心入名色所分的名基心与心所，以心假立、唯心无所取能取的空性光明真如 |

### 2.4 辛三 寻求能求作意（十八瑜伽作意）

**十八瑜伽作意**——这是本论第一次作意枚举（后面己二还有五十一作意）。《密意疏》说："承许是摄集一切大小乘的所有作意本性"。

简要列表（详见密意疏原文）：

1. **三种姓作意**（总略）——声闻缘四谛/缘觉缘道缘起/菩萨缘二无我
2. **行事作意**——信解行地一阿僧祇劫积二资粮
3. **所依差别作意**——出家（无害所依） vs 住俗家（有害所依）
4. **安住信解作意**——以七相随念佛（《摄乘论》所说）
5. **欲乐作意**——随念佛后对三宝起强烈欲求
6. **禅定作意**——未至定（下依）/ 正行初至四禅（圆依）；有诠（有寻有伺）/ 无诠（无寻无伺）
7. **具慧作意**——闻思智慧为因、止观瑜伽为性的修所生慧
8. **间杂缘作意**（5种）——缘经藏、自说偈、因缘、受持、传讲
9. **不共缘作意**（7种）——缘名、词、字；缘人无我、法无我；缘依他起所取分、能取分
10. **遍知定解作意**（5种）——所遍知基四谛、义十六相、能遍知见修道、果解脱、解脱智
11. **修行作意**（4种+37）——人无我/法无我/见/智行相 × 三十七菩提分
12. **二道自性作意**——寂止胜观双运
13. **善妙/功利作意**——遣恶业（身4 + 心贪嗔痴）、遣五见（以人无我）
14. **真实得受教授随教作意**——发心者于佛菩萨前得一地等持教言
15. **加行作意**（5种）——解数、解安住趋入二相、解分别、次第解、证悟加行（11种：暖/顶/忍/胜法 + 一地 + 二种所证 + 胜劣 + 后得二种）
16. **自在作意**（3种）——清净烦恼障/清净二障/成就殊胜功德
17. **声缘道作意**
18. **广大菩萨道作意**

这十八摄一切大小乘作意，属于**总纲性作意框架**。后面己二"五十一作意"则更细，专于大乘波罗蜜多。

## 三、己一 庚二 寻求所证义法（十一寻求）

此为本品的**核心大展开**——以十一种寻求开列大乘所证义理的全景地图。

### 3.1 辛一 寻求真如（1 颂 半 × 2）

> **恒离二迷所依者，非诠离戏真如性，**
> **所知断净性无垢，如空金水许惑净。**

**以三自性结构真如**：
- **遍计**：虽恒离二取但显现二取——如绳上无蛇
- **依他起**：成为所取能取显现之所依——**不清净依他起为所断，净化转依的清净依他起为所净**
- **圆成实**：不可言说、无分别念戏论的诸法实相——**各别自证**

**三喻显自性清净而客尘遮**：
- **如空**——自性清净如虚空，为云遮蔽
- **如金**——自性纯净如金，为淤泥覆盖
- **如水**——自性澄清如水，为混浊所障

**"舍真取妄"的颠倒**——密意疏感叹："**世间愚相何其重！**"——**众生本具圆成实而耽著遍计，本有的放弃、本无的耽著**——这是一切颠倒的本相。

**与品07真如品的关系**——此处"寻求真如"与[`07-真如品.md`](./07-真如品.md)的展开**互为表里**：品07 以五义建立胜义（非有无/不一异/无生灭/无减增/非净不净）；此处以三性抉择真如。一义二入：品07 从真如本身的法相入手（戊一），此处从"所求"的角度引入（戊二）。

### 3.2 辛二 寻求如幻（八喻与现空无二）

> **以如幻术说，非真之妄念，**
> **以如幻相说，迷乱为二法。**

**幻术师之喻**：
- 咒语加持小石子等真实显现为马、象——说明**非真妄念显现为所取能取的心心所依他起**
- 马象等虚幻身体支分颜色显似真马象——说明**迷乱为二法的遍计本性**

**现空无二的抉择**：
> **如是幻术等，说为有与无，**
> **彼有非无有，无有非是有。**
> **于幻等亦说，有无无差别。**

有非无（有显现）、无非有（无实体）、有无无别（现空一体）——此三句论证**现即是空、空即是现**。课43 索达吉堪布引《华严经》："譬如工幻师，示现种种色，于彼幻中求，无色无非色。"又引**善财童子参德生童子与有德童女**公案——二位善知识的法门即是"**幻住**"（如幻三摩地）。

**转依时的如幻**：
> **如彼无有时，彼因了可见，**
> **如是转依时，可见非真念。**

**阿赖耶识转依时**，如马象显现不存在而小石子、木块可见一样——
- **入定智慧**：见不缘所取能取的圆成实真如
- **后得智慧**：见显现所取能取的因——非真实妄念分别依他起，**证悟三界唯心**

**对治的如幻**（魔术王喻）：
> **犹如魔术王，为余幻王败，**
> **见法之佛子，彼等无我慢。**

染污法与清净法皆如幻。但**如幻的清净法**可以断除**如幻的染污法**——就像一位小幻术王被大幻术王击败。**国境**即阿赖耶识；**小力幻王**即我见等染心及其从属心所；**大力幻王**即无我智慧与信心等清净心。证此如幻的佛子**无"我已做到如此"的增上我慢**——这是如幻空性修证的**无增上慢功德**（课44）。

**佛陀如幻八喻**（密意疏抉择）：
> **承许六六二，二六三一一。**

| 比喻 | 所喻 | 数码 |
|---|---|---|
| 如幻 | 内六处（眼耳鼻舌身意）——无根无命却现似有命 | 六 |
| 如梦 | 外六处（色声香味触法）——梦中外境 | 六 |
| 如阳焰 | 心与心所——无水而误为水 | 二 |
| 如影像 | 内六处——从前世业生今世六处 | 二六之一 |
| 如光影 | 外六处——外六处是内六处的影子 | 二六之二 |
| 如回响 | 所说之法——虚现无实但耳闻 | 三之一 |
| 如水月 | 等持所修义 | 三之二 |
| 如幻化 | 故意受有身体——菩萨连续受生 | 三之三 |

课44 索达吉堪布补充：《金刚经》六喻（梦幻泡影露电）+ 玄奘译本八喻（星翳灯幻、露泡梦电云）；《杂阿含经》五蕴喻"观色如聚沫，受如水上泡，想如春时焰，诸行如芭蕉，诸识法如幻，日种姓尊说"；无垢光尊者《虚幻休息》、《闻解脱》十二幻化喻等。**八喻 / 六喻 / 五喻不同，同显"现而无自性"一义**。

**现代科学的印证**——课44 述英国伦敦大学戴维尔（量子物理专家）："**宇宙表面上看是很坚固、实有的，但是如果真正去观察，树木花草、山河大地、包括人类动物等等，一切全是虚妄的假象**。"爱因斯坦也讲"时间、空间包括宇宙全是人类的错觉"——量子力学与佛法如幻观殊途同归，"但是真正去观察，中间其实也还有一定的距离"（科学尚未穷达佛智）。

### 3.3 辛三 寻求所知（四类所知）

> **邪分别非正，非邪不分别，**
> **非分不分别，名为诸所知。**

四类所知：

| # | 类别 | 内容 |
|---|---|---|
| 1 | **邪分别** | 凡夫的能取所取分别念（不随同法界与无分别智） |
| 2 | **非正非邪** | 信解行地（资粮/加行道至世第一法以前）之法——有二取故非正，随同法界故非邪 |
| 3 | **不分别** | 法界无分别智（一地以上圣者的入定智） |
| 4 | **非分不分别** | 入定无分别智之后得清净世间智——缘法界、无分别（非分）；见诸法自相共相（不分别） |

**要义**——"**唯心以外的法不存在，因为以心的这四种阶段与法界摄集一切所知**"。**唯识立场贯穿**：所有所知不出心与法界。

### 3.4 辛四 寻求染净

**染污法相**：
> **自界显现二，无明烦恼俱，**
> **尽生转分别，断除二实体。**

- **自界**=阿赖耶识 + 所取能取习气
- 自界显现二法（所取/能取）→ 无明与烦恼俱起 → 分别执取 → 流转轮回
- 实际二者非异体——迷乱心致显现二法，断二实体即是**寻求染污法相**

**清净法相**：
> **获得所缘法，具住自界故，**
> **真如无二现，趋入如皮箭。**

- 信解行地获得所缘法 → 一地安住依他起心之法界（=二空真如）→ 修道诸地修此胜观瑜伽 → 真如无二现 → 趋入佛地

**皮革与竹箭喻**：
- 众生心如被雨淋的坚硬皮革——以寂止（如水与油）软化
- 我见常断等恶见如弯曲的箭——以胜观（如火）烘烤抻直

课44 索达吉堪布对皮革喻补充："牛皮放在水里泡可以变软，但放在太阳下晒又变硬了……对此皮匠有一种方法：牛皮在水里泡软后，用酥油不断地揉，揉完后牛皮即使放在太阳下面晒也不会变硬了。"——**究竟佛地的寂止也是这种"不可逆软化"状态**，种子已断。胜观竹箭也是同样不可逆："无明种子已经全部断除，不可能再产生能取所取的分别念。"

### 3.5 辛五 寻求唯识 ★ 本论唯识立场的核心章节

> **心者显现二，承许现贪等，**
> **或现信心等，非有惑善法。**
> **是心现种种，趋入种种相，**
> **显现彼有无，是故非法性。**

**唯识二义**（密意疏）：
1. **不破心所的唯识**——"心"是阿赖耶等八识及心所，显现所取能取二法，**除心外的所取外境不存在**
2. **破心所的唯识**——仅是心本身显现所取能取二法，**除它以外的心所与所取能取并不存在**

安慧论师《注释》说明：
- 第一种承许**心与心所异体**
- 第二种承许**心与心所非异体**

两种皆为唯识宗内部承许，本论对两种承许并列不偏。

**万法唯识的抉择**：
- 心显现贪嗔等烦恼 / 信心精进等善法——除心外无烦恼善法
- 阿赖耶显现种种所取，趋入能取种种相
- 显现有 / 如现实无 → 非他体法性 → **唯心**

### 3.5.1 寻求唯识的教证与公案（课44）

**《大智度论》公案**——佛陀在王舍城说法路遇水灾，**佛陀在水面敷座说法而不沉**：

> "若比丘得禅心自在，能令大水作地即成实地，何以故，是水中有地分故。"

**外境随心之喻**——同一美丽之人，染污者见之生贪；修不净观者见之觉丑；嫉妒心者见之欲杀——**三人见同一境却三种境**。如果此人真实有美/丑/恨之体，所有人所见应同。"**人也是如此：一个人觉得这个人很好，另一个人觉得很坏……一切万法唯心造**。"

**玉琳国师与顺治皇帝公案**——楞严七处征心：
- 顺治：心在何处？
- 玉琳：你寻觅自己的心一无所得，这就是所谓心的去处
- 顺治：山河大地难道不存在吗？
- 玉琳：你做梦时山河大地是存在的，你醒来时，山河大地不存在

**教证**——《大乘密严经》："内外诸世间，一切唯心现。"《楞严经》："如身受用处，唯心我所说。"

### 3.5.2 本论唯识立场 vs 中观立场（必须诚实交代）

《大乘经庄严论》**在科判与义理框架上属大乘唯识宗的立场**——这是读懂本论的前提。索达吉堪布在课60（结构总览已引）明确指出本论是"用唯识宗的观点宣讲的"。

**"寻求唯识"在本论的作用**：
- **名言上**：成立心外无境（第一层唯识）
- **胜义上**：心也显现虽有、实则无法性——即"显现彼有无，是故非法性"——这已接近中观的立场

**与中观庄严论的对照**——[`collections/中观庄严论/16-破唯识所许实一之识.md`](../中观庄严论/16-破唯识所许实一之识.md) 以静命论师的立场**进一步破唯识**：唯识所许的"自证实有的心识"在胜义中也不成立；但在名言中，**唯识比经部更究竟**（本论的名言立场）。

**瑜伽行中观**——合本论唯识立场与中观空性立场的教判产物，参见 [`topics/madhyamaka/yogachara-madhyamaka.md`](../../topics/madhyamaka/yogachara-madhyamaka.md) 与 [`collections/中观庄严论/07-阿赖耶辩论.md`](../中观庄严论/07-阿赖耶辩论.md)。

**宁玛视角下的调和**（依结构总览的处理策略）：
- **名言上**——与唯识宗可共许"心外无境、阿赖耶等八识"
- **胜义上**——以"显现许有、胜义不成"的中观抉择（类似"非真妄念有、彼非有二法"的《辨中边论》立场）
- **真如的自性**——依宁玛教判可开显为**离戏大圆满/自性光明**，非仅"依他起不变"之自证分实有

本论"寻求唯识"不应当作"承许自证实有心识"的凭据；而应理解为**建立"名言唯识、胜义无自性"的弥勒学路径**——这正是瑜伽行中观的理论本源。

### 3.6 辛六 寻求法相（三性配五事）

> **佛为利有情，真实说事相，**
> **法相及名相，一切细分类。**

佛陀宣说**三相**：**事相（基）、法相（用）、名相（诠）**。

**事相——五事**（与《俱舍论》五事基本相同）：

| 五事 | 对应颂词 |
|---|---|
| **心王**（八识聚） | "心" |
| **心所**（五十一） | "有见"前分（能取趋入） |
| **色法**（五根五境十色法） | "有见"后分（所取对境） |
| **不相应行**（得、未得等20余种） | "彼位" |
| **无为法**（抉择灭、非抉择灭、虚空、真如四种） | "不迁变" |

**五事摄一切所知**——"详分无有量"。安慧论师《注释》分法略异（心=所取五根五境；能取六识聚；心所=有见跃入对境）。

**法相——三性**：

#### 3.6.1 遍计的法相

> **依诠作义想，因与彼习气，**
> **从中亦现义，遍计之法相。**
> **依于名与义，显现名与义，**
> **非正分别因，遍计之法相。**

- **依诠作义想**：依名称表义想
- **因与彼习气**：戏论习气是非真实分别念的因
- 显现二法的戏论习气苏醒 → 现二法 → 执取名相依显现般耽著 → 遍计所执
- "**名义无本性相属**，但分别增益暂时相联"——苹果之名既可为水果亦可为手机，**名义假立无性**（课45）

#### 3.6.2 依他起的法相

> **显现三三相，所取能取相，**
> **非真分别念，依他起法相。**

**所取三相**（依他起之色法方面）：
- **处**——显现器世界
- **境**——色声香味触法六境
- **身**——眼耳鼻舌身意六根

**能取三相**（依他起之心心所方面）：
- **意**——染污意，缘阿赖耶视为我
- **能取**——眼识至身识五根识
- **分别**——意识（分别对境）

**本质**——这六种显现都只是**阿赖耶识上习气界苏醒**的梦显现，不成立异体所取能取。

#### 3.6.3 圆成实的法相

> **无有及有者，有无平等性，**
> **非寂寂无别，圆成实法相。**

- **无有**（无所取能取实性遍计）
- **有者**（有二空、有真如法相）
- **有无平等性**——否定能取所取即肯定真如，肯定真如即否定二取
- **非寂寂无别**——从客尘杂染侧面非寂灭，从自本体分是自性清光明寂灭
- **远离寻思戏论**——非分别即圆成实

**三相与五事**——安慧论师《注释》说每一事上皆具三种：遍计之色 / 依他起之色 / 法性之色——三相彻于五事。

### 3.6.4 瑜伽五地与转依的基础

三相存于五事，能表瑜伽五地（基 → 能生 → 如镜 → 显现 → 处/所依）：

| 瑜伽五地 | 对应道位 | 所表 |
|---|---|---|
| **基** | 资粮道 | 缘等流法如理作意 |
| **能生/安住** | 加行道 | 安住心法界 |
| **如镜** | 一地（见道） | 现量明见所知真如 |
| **显现** | 二地至十地（修道） | 见义有与无，无倒显现诸法自性 |
| **处/所依** | 佛地 | 成等性，入无漏法界 |

**五差别——佛胜声缘**：
1. **清净超胜**——佛断二障细微习气；阿罗汉断烦恼障子而未断习气（如猴子跳跃习、妓女照镜习）
2. **刹土清净超胜**——显现珍宝大地、如意树、甘露河；声闻不能
3. **获得法身超胜**——佛证二无我法身；声闻只证人无我解脱身
4. **圆满受用大乘法超胜**——佛与十地菩萨圆满受用大乘法之报身；声闻不具
5. **以化身利益有情及事业不间断超胜**——佛之化身事业相续未来际；声闻不具

### 3.7 辛七 寻求解脱（四种自在 + 四种解脱差别）

> **种子转依故，显现处境身，**
> **转依无漏界，彼所依遍行。**

**阿赖耶识转依**——种种习气所存的阿赖耶识远离所取能取垢而转依时：
- 显现**处**（器世界）→ 珍宝自性的器世界
- 显现**境**（六境）→ 如意树、甘露池等无漏六欲妙
- 显现**身**（六根）→ 如佛身、每根能做一切根之事、具十二种百数功德

**四种自在**：
> **意能取分别，转依无分别，**
> **刹土智事业，是为四自在。**

| 自在 | 所摄识 | 所得自在 |
|---|---|---|
| 无分别自在 | 染污意转依 | 不再缘阿赖耶为我 |
| 刹土自在 | 能取五根识转依 | 清净刹土显现 |
| 智慧自在 | 分别意识转依 | 四无碍解 |
| 事业自在 | 分别意识转依 | 神通力任运成事业 |

**四自在在哪些地获得**：
> **不动等三地，获得四自在，**
> **一者二自在，余地许各一。**

- **八地**（不动地）：无分别自在 + 刹土自在（两种）
- **九地**：智慧自在
- **十地**：事业自在

**解脱四差别**（密意疏摘）：

**第一差别**（转依角度宣说解脱本体）：
> **具慧证诸有，亦知二无我，**
> **彼知平等已，持彼入真如。**

- 通达人无我、法无我二无我一味，遣除增益常边
- 非一切时处唯无的单空——了知如幻依他起与以二我空的圆成实真如无别平等，遣除损减断边
- 以受持"唯识"趋入真如（暖至忍位）→ 意住唯心明觉（胜法位等持）→ 不显现能取
- **证所取能取二者平等圆成实不存在 → 圣者殊胜智慧远离一切所缘 = 解脱**

**第二差别**（解脱道侧面）：
> **资粮基有生，得见唯名称，**
> **见唯名见彼，后亦不见彼。**

- 资粮（积二资）+ 今生闻思修加行（能生）
- 暖、顶位：见诸法唯名称假立
- 忍位：见外境不存在
- 胜法位：执唯名（唯心）的能取也不见

**第三差别**（生起解脱的次第）——以"如镜、显现、平等"三位立解脱

**第四差别**（解脱依于心）：
> **心即彼堕罪，我见之索缚，**
> **极其而流转，内住许还灭。**

- 心即堕罪：身堕罪（杀生等不善业）、心堕罪（贪嗔痴）；或说烦恼障/所知障二种习气罪
- 我见之索：将补特伽罗视为我、将法视为我的我见
- **束缚是心缚、解脱是心解**——外道的"自在天主宰、裸体派有色解脱"皆不合理

### 3.8 辛八 寻求无本性（三本性的无本性）

经中云："诸法无本性，自性涅槃。"

> **自与自性无，自本体不住，**
> **如执无有故，承许无本性。**

**三本性的无本性解读**：

| 三性 | 无本性义 |
|---|---|
| **依他起（生无本性）** | "自"——自性不生，由因缘聚合生，破数论"由主物生果" |
| **依他起（灭无本性）** | "自性无"——有实法灭仅有实灭，破数论"灭是隐于主物中安住" |
| **依他起（住无本性）** | "自本体不住"——不住第二刹那，破外道"产生后安住一阶段方毁灭" |
| **遍计（本无本性）** | "如执无有故"——分别心所执如绳视为蛇，丝毫不曾有 |

> **前乃后所依，无性无生灭，**
> **本来即寂灭，自性涅槃成。**

**五层连环**：无本性 → 无生 → 无灭 → 本来寂灭 → 自性涅槃成

**总结**：依他起（生灭住三相无本性）+ 遍计（本无无本性）+ 圆成实（胜义自性本来安住）——三者配合说明"**显现所取能取一切法无本体 → 诸法无生无灭原本寂灭自性涅槃 = 圆成实**"。

### 3.9 辛九 寻求无生法忍（八种无生）

> **始真实他性，自相自变异，**
> **染污及差别，名无生法忍。**

**八种无生法忍**：

| # | 种类 | 内容 |
|---|---|---|
| 1 | **初始无生** | 轮回无初，依缘起十二支辗转而生 |
| 2 | **真实无生** | 已生已灭之法不复再生 |
| 3 | **他性无生** | 新有情无新生 |
| 4 | **自相无生** | 遍计自法相本无生，如兔角 |
| 5 | **自性无生** | 依他起由因缘他者而生，自性不生 |
| 6 | **变异无生** | 圆成实无转依之生，如虚空不变 |
| 7 | **染污无生** | 阿罗汉断见修所断烦恼后不复生 |
| 8 | **差别无生** | 诸佛法身一自性无异体，无不同生 |

**三种忍的差别**：
- **小忍**——加行道忍位：以共相堪忍无生义
- **中法忍**——见道一地：现量证悟无生
- **大法忍**——八地：无分别智慧成熟，于本来无生中入定后得不动摇

**得成佛授记**——得忍不退转者得成佛授记，尤其八地称授记地。

### 3.10 辛十 寻求一乘密意（七种密意调和三乘与一乘）

问："有些经中说三乘，有些说一乘，它们的密意是什么？"

唯识宗立场——说**三乘了义、一乘不了义**。那么一乘之密意是什么？

> **法无我解脱，等同种姓异，**
> **得二意乐化，究竟故一乘。**

**七种密意**（依密意疏明列）：

1. **同法密意**——虽三乘道三果位并非无有，但趋入三乘者与一切道法果法**除一个法界外不存在**。
2. **同是人无我**——三乘行人皆具人无我。
3. **同解脱**——三乘之果解脱业惑痛苦等同。
4. **不定种姓者乘不同**——有趋入声缘道、有入大乘道，此乃基种姓异。
5. **得二意乐**——佛陀考虑我与众生平等意乐，声缘决定种姓先想舍菩萨行，后由如来安慰"我也想成佛的意乐"——二意乐得故一乘密意。
6. **幻化调化密意**——佛陀"**我数百次以声闻乘涅槃**"——因佛陀的幻化示现声闻涅槃，但密意指**唯有一乘**。
7. **究竟乘一密意**——究竟唯大乘一乘，非三。

**究竟三乘宗 vs 中观一乘宗的调和**（密意疏明示两宗）：
- **究竟三乘宗**——不定种姓入大乘可成佛，声缘决定种姓永不成佛；断种姓者暂无论；声缘阿罗汉趋无余涅槃时蕴灭不复趋大乘
- **中观一乘宗**——一切众生皆有如来藏，无不可成佛种姓；心性光明、客尘可离；即使入无余涅槃也会以如来光芒劝请而从灭尽法界中起定、以不可思议驾驭受生圆满清净达究竟成佛（《宝性论》立场）

**两宗随补特伽罗界性/根基/意乐**——经中两说皆有。**暂时**三乘各有其道果不可否定；**究竟**《宝性论》所说一乘成立。

### 3.11 辛十一 寻求五明 ★ 菩萨必学

> **若未勤五明，大圣不成佛，**
> **故为折摄他，自遍知精进。**

**五明的内容**：

| 五明 | 内容 |
|---|---|
| **声明** | 语言学、文法学、词句字 |
| **因明** | 能真实辨别现量比量之理的论典 |
| **工巧明** | 身技/书写绘画、歌音七品、世规非规等世间种种技艺 |
| **医方明** | 八支——病因、病本、治病、不生病 |
| **内明** | 妙法，指示永乐解脱与遍知之道 |

**学五明之必要**（密意疏）：
- **一切所知包括于五明中**——不精勤通达五明则无一切智
- **菩萨与声闻的差别**——声闻以"人无我一偈意义"可得自果；菩萨需"通达一切所知"
- **折与摄**——以声明/因明**折服外道**；以工巧明/医方明**摄受众生**；以内明**自真实了知**

**教证**：
- 《报恩经》："假设未学五明，则菩萨何时亦不得无上真实圆满菩提遍知之智，是故为得无上菩提当学五明。"
- 《优波塞戒经》："**世间之事，虽无利益，为众生故，而亦学之**。"

**等流果的积累**——菩萨大乘种姓自性利根，于信解行地精进研学五明、回向成遍知之因，由等流果经多生累世串习。**得地后**以等持、智慧不费力通达尽所有的所知。

**索达吉堪布的教诫**（课48）：
- **批评"五明佛学院"的误解**：有人误以为"五明"指"中观、戒律、般若"等大乘五论，实际上**五明 ≠ 五部大论**。"五明"是声明/因明/医方/工巧/内明五种学科。
- **现代学科映照**：声明≈语言学；因明≈逻辑学；工巧明≈工程/艺术；医方明≈医学；内明=佛学
- **朱清时院士的例子**——对前世后世"主要是从阿赖耶和如来藏方面承认"——**现代科学家依佛学立场的表述有特殊加持力**（与本论"摄受"目的相通）

- **五明涉及因明** → [`topics/pramana/index.md`](../../topics/pramana/index.md)
- **五明涉及内明** → 本集合及整个buddhism技能体系

## 四、己二 以如何作意寻求（五十一作意）

前已说"所求之法"，此说**如何作意以求之**。十八瑜伽作意是"一切大小乘总纲"；此处**五十一作意**是**大乘波罗蜜多界的精密增长法门**。

### 4.1 五十一作意的总纲（密意疏简列）

密意疏开列："得因欢喜作意、随念所依作意、欲求共果作意、信解菩提作意、究竟欢喜作意、不厌倦作意，断违品作意、修对治作意七种，极净作意等直至执我胜作意之间，**思维这些作意后使自相续的波罗蜜多界增长**。"

**核心目的**——"**一切作意能增长波罗蜜多界**"，即**十波罗蜜多的种子不断增长**。

### 4.4 五十一作意的摄义

> **十度因所生，此等善作意，**
> **是为能恒常，增长菩萨界。**

"**十度因所生的这些善妙作意，恒常能增长菩萨相续中十波罗蜜多界**"——这是五十一作意的总结果：**阿赖耶识界中具有的十波罗蜜多种子越来越增长**。

**与后续品的关系**——五十一作意是整部《经庄严论》后半部的"作意地图"。从信解品（已说）→ 求法品（当品）→ 说法品 → 修法品 → 教授随教品 → 方便品 → 六度品 → 四摄品 → 供依无量品 → 菩提分品 → 功德品 → 行为究竟品——每一品都可以定位到五十一作意的某一组。**本论把"怎么作意"细化到与每一品教义对应的程度**，是弥勒学的精密心理学示范。

## 五、己三 宣说寻求之分别（十三种差别）

> **增胜意乐大，许坚稳求法，**
> **有断无所断，如是得自在。**

**十三差别分为两组**：

**第一组 从证悟分的三种差别**：

| 位置 | 称谓 | 本质 |
|---|---|---|
| 异生位至世第一法前 | **增** | 能增长解脱种子的求法 |
| 世第一法位始，一至七地间 | **胜意乐** | 证悟二无我、自他平等 |
| 八至十地间 | **大** | 获得无分别任运自成智慧 |

**第二组 从所断方面的三种差别**：

| 位置 | 所断状态 |
|---|---|
| 信解行世第一法前 | 有所断违缘，意乐不稳，善恶伴改变，二障未断 |
| 一至七地间 | 无所断违缘法，无违缘 |
| 八至十地间 | **自在**——无分别、神通、清净佛土任运自成 |

**第三组 从获得法身角度的四种差别**：

> **菩萨之寻求，无身有得身，**
> **圆身多我慢，细慢无我慢。**

两种诠解（《大疏》 vs 另一种）：

**《大疏》释义**：
- 异生资粮道、加行道忍位之前——**无身**（无法身同类因无分别）
- 世第一法位——**有身**（断所取能取证法身之分）

**另一说**：
- 信解行资粮道之前——无身
- 大资粮道获法相续等持、四加行道证随同法身之因——有身
- 一至七地——现量见谛的有勤无分别智
- 八至十地——**圆满身**（无分别智慧无有功用任运自成）

**从我慢分的三差别**：
- 世第一法位前异生位——**多我慢**
- 一至七地——**细微我慢**（断遍计部分，二地始渐断俱生部分）
- 八至十地——**无我慢**

## 七、己五 以宣说求法殊胜而摄义偈

> **慧善依勇勤，说如实求二法，**
> **恒成众生师，如海满功德。**

四要素之总摄：
- **慧善**——缘本品所说深广诸法的智慧 + 波罗蜜多方便的广大善根
- **依勇勤**——求法依靠无懈怠怯懦及迷乱的勇猛精进

- **恒成众生师**——无误为他众传讲善说之法 → 如大海中盈满水与珍宝般，相续中充满道法果法功德

- 本论"寻求法相"中的三性配五事——与中观庄严论"**破唯识所许实一之识**"（[`collections/中观庄严论/16-破唯识所许实一之识.md`](../中观庄严论/16-破唯识所许实一之识.md)）形成有趣对照：本论**在唯识框架内立自证**；中观庄严论**于胜义破自证实一**
- **整体看**——本论"寻求唯识"与"寻求无本性"构成内部的唯识-中观张力：名言唯识 ↔ 胜义无本性，这是**瑜伽行中观**的本论胚胎

### 8.4 与宝性论·如来藏的对应

- "辛十 寻求一乘密意"中的**中观一乘宗立场**引《宝性论》观点——"**一切众生均有如来藏**"可与[`collections/宝性论/03-如来藏-总纲十义.md`](../宝性论/03-如来藏-总纲十义.md)互读
- 一乘究竟说是本论在种姓品（品4）**三种决定种姓 + 不定种姓 + 断种姓**五种姓说的**另一面**：即使声缘决定种姓，究竟也可成佛（中观宗立场）

## 九、本品的实修教诫与修学定位

### 9.1 索达吉堪布教诫摘要（课40-50）

**求法态度**（课39-40）：
- 三地菩萨仍为求法越三千大千世界火坑（《辨中边论》注释引）
- 《华严经》："为重法故，不惜身命，何况王位，城邑聚落。"
- 普通凡夫求法：身体只是臭皮囊，四大假合；为永久解脱和求法，对生命也并不特别重视

**警惕"跑跑居士"和少闻思**（课38/40）：
- 有人以参加活动、朝拜、放生热闹为"求法"，一说闻思就头痛心脏病
- 佛学院的独特性——**系统性闻思**，这是修学模范
- "求法的人的确越来越多，但是我也担心：有些人只是在形式上求法，从来没有去思维法义，没有反复串习和修习"

**依止善知识**（课42/48）——关于依止上师、广闻博学，本论求法品给出的是**通过教理抉择而非单纯信仰**的路径。

**广闻博学对佛法稳固的重要**（课47/48）：
- 仅以皈依证作佛教徒是不够的——佛教徒至少要具内在信心、智慧、悲心
- 广闻博学是保证"永远不生邪见"的保障
- 五部大论、五明是"**胸有成竹**"的前提

### 9.2 本品的教理定位

求法品是整部《大乘经庄严论》**"怎么学大乘"的总纲**：
- **学什么**——三藏（教法） + 十一证法（证法）
- **怎么学**——十八瑜伽作意（总纲） + 五十一作意（大乘波罗蜜多界细化）
- **怎么评估**——寻求分别（十三差别，从地道看求法的层级）
- **得到什么**——成就功德（四种）+ 断除所断（十种散动分别）

**本品在全论次第上的关键意义**——
- **回答"知大乘→行大乘"之间的过渡**：信解品使人"想修"，求法品使人"知道修什么、怎么开始修"
- **铺垫说法品（品13）、修法品（品14）、教授随教品（品15）**：所求既定，后三品才能分别展开"讲法"、"修法"、"得教授"三个阶段

索达吉堪布在课42 总结本品定位："**这部论典非常好**……跟其他有些论典的讲法、思路有点不同。比如，本论讲信心不是像以前讲的论典那样，讲完三种信心就结束了。过一段时间我们要讲求法品，我原来说自己很喜欢求法品，本来以为求法品可能是讲为了求大乘佛法如何苦行，但是里面其实也不是讲这些，而是**从另一个层面来讲，实际上内容更深**。"

"**所求的法**不是指外在的法本或善知识的摸顶加持——而是三藏十二部 + 十一种深义证法 + 五十一种精密作意**"——这才是《大乘经庄严论》求法品的真实定位。
