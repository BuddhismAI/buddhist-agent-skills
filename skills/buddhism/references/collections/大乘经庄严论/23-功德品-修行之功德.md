# 品20 功德品·修行之功德

## 一、本档在全论中的位置

### 1.1 接续菩提分品，收果位因位之对称

品20 功德品是**正行六品**（品16-21）的第五品。功德品整体分两大部分——**己一 修行之功德（十七种）**与**己二 赞叹之功德（九种菩萨 + 菩萨异名）**。本档处理己一的全部十七种修行功德，对应课95-102 共八堂课（课102 讲完"遍知法义安立"后段至"说法八果、大乘七大与八归摄"），密意疏约 430 行。课103 起进入己二，见 [`24-功德品-赞叹菩萨.md`](./24-功德品-赞叹菩萨.md)。

本品与品10 菩提品（24-36 课、两档）构成本论全局中**因位—果位功德**的核心对称：

| 品 | 功德位置 | 数量 | 档 |
|---|---|---|---|
| 品10 菩提品 | **果位佛陀** 十功德 | 十 | [`10-菩提品-十功德.md`](./10-菩提品-十功德.md) |
| 品10 菩提品 | 果位佛陀 六安立（三身四智） | — | [`11-菩提品-三身四智.md`](./11-菩提品-三身四智.md) |
| **品20 功德品（上）** | **因位菩萨** 修行十七功德 | 十七 | **本档** |
| 品20 功德品（下） | 因位菩萨 赞叹九种菩萨 + 异名 | 九 | [`24-功德品-赞叹菩萨.md`](./24-功德品-赞叹菩萨.md) |

**果位十功德**是佛陀已圆满之功德（无染、无二、无分别等），**因位十七功德**是菩萨从信解行地到十地之间实修所得之功德。二者对照读可见本论"**因果首尾完整**"的体例——品10 先立果境，品20 再示因位菩萨如何一步步接近此果境。

### 1.2 本档的三段结构

密意疏开品即给出**稀有总说三功德 + 别说十二功德 + 说法安立**的三段结构：

> 修行功德包括以**三功德总说**与以修波罗蜜多报恩至首要所为之间**十二功德别说**及以**说法安立**说明善巧功德。
>
> 以三功德总说是指：等流功德、相属功德与正观众生功德。

- **三功德总说**（庚一-庚三：稀有+非大稀有+平等心）——"为何是功德"：修六度者所具之功德、由六度所感之果、对众生恒有之平等。这是功德的**总标**。
- **十二功德别说**（庚四至庚十六）——"功德如何落实"：饶益 / 报恩 / 希求 / 不唐捐 / 真实行 / 退失胜进 / 形象真实 / 调伏 / 授记 / 决定 / 无疑所为 / 恒常所为 / 首要所为，逐项开示菩萨的修行面向。
- **说法安立**（庚十七）——"功德如何利他"：以说法之善巧为最后一种功德。这一庚含**五辛**（所说法差别/遍知法义/五种无量/说法八果/大乘七大八归摄），是本论对"菩萨说法"这件事最系统的一次展开，也是与品13 说法品、品12 求法品并列的"说法三大论域"。

## 二、本档之宏观位置——十七功德的三层结构

### 2.1 从"自性/果/平等心"到"饶益"的展开逻辑

表面上十七功德似乎平列，但内里有**三层**：

- **第一层：总定位（庚一-庚三）** ——菩萨修六度是稀有的（庚一：异生、声闻缘觉皆无法做到），但对菩萨自身是"非大稀有"的（庚二：因具相应意乐与平等心，必然感得此果），根本在于菩萨具**平等心**（庚三：于一切众生慈悯平等，不分亲疏、亲人与仇敌皆同）。
- **第二层：饶益众生的方法与果（庚四-庚十二）** ——从"以六度饶益"扩展到"如七种亲人饶益"（庚四），众生报恩（庚五），菩萨自身的希求（庚六）、不唐捐（庚七）、真实行（庚八）、与退失/形象的对比（庚九-庚十），最终调伏众生（庚十一），获授记（庚十二）。
- **第三层：修行的恒常与究竟（庚十三-庚十七）** ——决定之果（庚十三）、不可不为的事（庚十四）、恒常之事（庚十五）、最胜之事（庚十六）、最究竟的利他方式（庚十七 说法安立）。

这三层既是**横向的菩萨行图谱**，也是**纵向的修行成熟次第**——从"自己是菩萨之稀有"到"饶益众生"到"自己安住菩萨道的恒常与究竟"，体现了"自利—利他—二利圆融"的展开。

### 2.2 "六度"作为贯穿十七功德的明线

这是本品最鲜明的论证特色——**几乎每一功德都以六度为框架展开**：

| 功德 | 与六度的关系 |
|---|---|
| 庚一 稀有 | 以六度各自说明稀有（施自身、舍轮王戒、于弱安忍等） |
| 庚二 非大稀有 | 以六度各自说明为何不稀有（断贪则必施、具悲则必持戒等） |
| 庚三 平等心 | 以六度皆具平等心（布施无偏堕、戒恒常不失等） |
| 庚四 六度饶益 | 直接以六度作为饶益法门 |
| 庚五 报恩 | 以六度报恩（自住六度 + 令他行六度） |
| 庚六 希求 | 希求六度增上、违品减损 |
| 庚七 不唐捐 | 四种不唐捐对应心行 |
| 庚八 真实行 | 六度真实行（无求施、不求后世戒等） |
| 庚九 退失胜进 | 贪受用失布施、失戒失持戒、我慢失安忍等 |
| 庚十 形象真实 | 伪六度 vs 真六度 |
| 庚十一 调伏 | 以六度调伏六违品（悭/破戒/嗔/懈怠/散乱/邪见） |
| 庚十三 决定 | 六度六种决定果 |
| 庚十六 首要所为 | 六度各自最主要之分支（法施/无漏戒/谛察忍等） |

索达吉堪布在课99 特别强调："我们要修六波罗蜜多来调伏众生相续中的各种烦恼" ——**凡是菩萨的修行与利他，归结起来都是六度**。因此本品实为"**从十七个面向展开的六度广论**"，可与品17 六度四摄品（[`18-六度.md`](./18-六度.md)）合参。

### 2.3 本品与《入行论》《大圆满心性休息》之呼应

索达吉堪布在课95 开篇即明："赞叹菩萨功德是以六波罗蜜多的修行为主，因为大乘的修行可以全部包括在六波罗蜜多的修行当中，《大圆满心性休息大车疏》里讲得很清楚。" 本品与：

- [`collections/入行论/`](../入行论/) 的菩萨行展开互为表里——入行论重在**修心次第**（菩提心生起、护持、增长），本品重在**功德分类**（以十七面向认识菩萨功德之广大）
- [`collections/大圆满前行/`](../大圆满前行/) 及 `topics/foundations/practice-path.md` 的道次第——本品的十七功德可作"发心后如何实修"的分类参考

## 三、三功德总说（庚一-庚三）——菩萨功德的三重定位

> 己一（修行之功德）分十七……
>
> 修行功德包括以**三功德总说**与以修波罗蜜多报恩至首要所为之间**十二功德别说**及以**说法安立**说明善巧功德。
>
> 以三功德总说是指：**等流功德、相属功德与正观众生功德**。

三功德总说对应庚一-庚三：

- **等流功德**——菩萨长时间串习有学道，菩萨心相续成道本体，前已生之道稳固，未生之道生起增上。即庚一"施自身……"两颂之**稀有六度 + 有学三果 + 佛果**。
- **相属功德**——庚一后半颂"生如来种姓……"，由一地出世智生于如来种姓，八地获授记，十地灌顶，佛地菩提，此是"由修得果"之相属。
- **正观众生功德**——庚三"犹如具慧者……"三颂，菩萨平等观众生得平等心之因。

庚二"非大稀有"一颂是对庚一的**赞叹之因**——说明因具圆满因故感得如是果非稀有，是对等流功德的论证性补充。

### 3.1 庚一 稀有功德

**施自身持戒，圆满皆能舍，**
**于弱能安忍，不顾身与命，**
**而发起精进，禅定无著味，**
**智慧无分别，具慧之稀有。**

菩萨修六度之稀有，不共于异生凡夫、声闻缘觉：

| 度 | 稀有之相 |
|---|---|
| 布施 | 一般人能布施衣食，菩萨连自身身体与性命皆能施 |
| 持戒 | 舍弃转轮王圆满受用如弃唾液而出家 |
| 安忍 | 对弱小、乞丐、贱种之损害亦能安忍（凡夫对弱者易发作） |
| 精进 | 不顾身命而精进 |
| 禅定 | 无著禅味（不贪一禅等四禅殊胜安乐） |
| 智慧 | 无分别（得辨别诸法智慧，但不分别相状） |

堪布于课95 以实例支持："凡夫人对比自己地位高的人能安忍，对比自己地位低的人不能安忍，而菩萨即使遇到下劣者、弱小者来危害自己，也不会报复和损害他们，这是菩萨安忍的稀有。" 此点特别值得体会——世间人所谓"安忍"多是对权势者不得不忍，菩萨的安忍则不分势位。

**生如来种姓，得授记灌顶，**
**尽获大菩提，亦许大稀有。**

修六度之果：

- **一地**：最初获出世无分别智，转生如来种姓
- **八地**：得无勤任运无分别智，得佛授记
- **十地**：蒙佛陀法王补处灌顶
- **佛地**：获无上菩提

三种有学果（生如来种姓、得授记、得灌顶）加无学果（大菩提），是一切世间人、声闻、缘觉皆不具备的大稀有。

### 3.2 庚二 非大稀有——功德的法尔理

**离贪与慈悲，胜修平等心，**
**是故行彼等，亦非大稀有。**

从**具广大意乐行为的菩萨视角**看：六度与十地之果并非稀有，因为**菩萨具圆满因，必感如是果**。这是"法尔理"的运用——如具生长力的种子必生苗芽，具离贪心者必然布施，具大悲者必然不害众生而守戒，具他胜己心者必然安忍，获无功用殊胜修行者必然精进，于自他获平等心者必然行利他不厌，具品味禅定与不分别智慧者必然圆满后二度。

**"非大稀有"的深层含义**：本品并非说菩萨功德不稀奇，而是说**从因果规律看，功德的生起是必然的**。这与庚一"稀有"构成辩证统一：

- **对凡夫小乘稀有**（庚一立场）——因彼等不具此意乐故
- **对具相应意乐之菩萨非大稀有**（庚二立场）——因因具必然果成故

堪布课95：「我们想起上师的功德时，有时觉得"上师真的很了不起，非常稀有！"有时候又想：上师肯定在多生累劫中已经为众生付出过无数次，所以他这样做是非常正常的。」——这正是庚一与庚二双边视角的实修落实。

### 3.3 庚三 平等心——六度皆由平等心而修

**犹如具慧者，极慈诸有情，**
**众生于自妻，子亲非如是。**
**于求无偏堕，恒常不失戒，**
**普忍一切故，发起大精进。**
**恒常妙禅定，智慧无分别，**
**当知彼菩萨，如是心平等。**

**核心命题**：菩萨之慈**超过**众生对自身与亲人之慈——众生烦乱时会自杀服毒，对妻儿会嗔打，并非生生世世不舍慈悯；但菩萨对一切有情平等慈心。

以此平等心贯穿六度：

| 度 | 平等心之相 |
|---|---|
| 布施 | "于求无偏堕"——不分亲疏、贫富无偏施 |
| 持戒 | "恒常不失戒"——一切时中守护佛制罪与自性罪的粗细戒 |
| 安忍 | "普忍一切"——何时何地对所有众生皆安忍 |
| 精进 | "发起大精进"——为利一切有情不间断 |
| 禅定 | "恒常妙禅定"——清净无漏善妙禅定不退 |
| 智慧 | "智慧无分别"——三轮清净之无分别 |

堪布在课95 以"于求无偏堕"一度举多例：发会供品时对关系好者多给是凡夫行；做贫困助学时不分亲疏是菩萨行（"我母亲的妹妹对我说：你经常帮助那些非亲非故的大学生，却不帮我……世间人有这样的理念，亲怨的差别很大。我们帮助贫困学生，根本不看他是不是自己的亲戚。"）

**平等心与《大幻化网》等性的呼应**：堪布于课末引《华严经》"菩萨摩诃萨，广行无量行，出生于一切，正觉平等法"——六度万行的来源是觉悟万法平等性。本论虽属唯识框架，但此"平等心"与大圆满 "等性" 在因位相续层面高度一致。

## 四、饶益（庚四）——六度饶益 + 七喻饶益

### 4.1 辛一 以六度饶益

**堪为作法器，安置于戒律，**
**于害能安忍，能利而行持，**
**令入此佛教，能断诸怀疑，**
**此许具慧者，饶益有情性。**

六度饶益的标准格式：

| 度 | 饶益之相 |
|---|---|
| 布施 | 以衣食等布施，令贫乏而不能行正法者堪为法器 |
| 持戒 | 自持戒后安置众生于戒律 |
| 安忍 | 对害能安忍 |
| 精进 | 一切时处无有困难而行利他 |
| 禅定 | 示神变使所化众生入佛教 |
| 智慧 | 断众生一切怀疑，说三乘法 |

堪布课95 此处即批评佛教徒在布施方面做得不足："网上看到一群藏族人在接受洗礼……周围藏传佛教的一些人没给他们什么吃的，而其他宗教经常给他们很多布施，他们就改变了信仰" ——以六度中之布施饶益对当代具体的情境仍然有直接指导意义。

### 4.2 辛二 七种比喻饶益——从亲人到师长

饶益七喻（父、母、亲、友、仆、轨范师、亲教师）是本品最富有形象力与实用性的段落，课96-97 两节重点讲。七喻的排序也有深意：**从血缘（母、父）→ 旁系亲属（亲）→ 朋友 → 仆人 → 法师（轨范师）→ 传戒师（亲教师）**，由感情最深的血缘关系逐层过渡到最具法义权威的上师。这正好对应众生对菩萨感情的成熟次第——从视菩萨如亲人到视菩萨如师长。

#### 4.2.1 如母之喻（课96）

**恒时于众生，意乐平等摄，**
**令生圣者地，令善得增长，**
**尽防造罪恶，令听闻纯熟，**
**五事诸佛子，犹如众生母。**

| 母事 | 菩萨事 |
|---|---|
| 怀子不舍，无微不至 | 以慈悲平等摄受众生，尽轮回际不舍 |
| 产子见光 | 令众生生小乘见道乃至大乘十地圣者地 |
| 育子增长 | 令众生世出世间善法增长 |
| 护子远险 | 令众生防造罪恶，远离三恶趣 |
| 教子知名 | 令众生听闻佛法至纯熟 |

> 《大集经》："为诸众生故，能忍一切苦，视于诸众生，如母念一子。"

堪布课96 特别连接到"善知识如母"：文殊菩萨、观音菩萨虽存在但不常显现，真正亲近的善知识就是离我们最近的菩萨。这点关系到大乘之实践——**不能只向抽象的佛菩萨祈求，而应从最近的善知识处领受"菩萨如母"的功德**。

#### 4.2.2 如父之喻（课96）

**恒常于众生，能令起信心，**
**教增戒学等，修行解脱故，**
**祈请正等觉，断除彼等障，**
**五事诸佛子，犹如众生父。**

| 父事 | 菩萨事 |
|---|---|
| 播种生子 | 令众生对三宝起信心（信心如种子） |
| 教子学艺 | 教众生增上戒定慧三学 |
| 为子娶妻享乐 | 令众生修行解脱，获佛法妙乐 |
| 委托友伴关照子 | 祈请诸佛不涅槃、转法轮，使众生有依 |
| 为子还债 | 断众生烦恼障与所知障之债 |

堪布课96 特别开示"信心如种子"：《华严经》"信为功德不坏种，信能生长菩提树，信能增益最胜智，信能示现一切佛"——**有上等信心的人，能获得上等的功德和智慧**。故"令生信心"这一菩萨事业至为根本。

#### 4.2.3 如亲之喻（课96）

**于诸不应说，众前能保密，**
**谴责破学处，赞叹圆满者，**
**能施授教言，救护离恶魔，**
**五事诸佛子，犹如众生亲。**

| 亲事 | 菩萨事 |
|---|---|
| 保护亲人事，不泄于群 | 对智慧浅薄众生不宣大乘深广法 |
| 谴责亲人之非 | 谴责失坏三学者 |
| 赞叹亲人善举 | 赞叹具足三学功德者 |
| 授教言让亲通事理 | 授方便窍诀令众生通达圣道 |
| 救护亲人离怨贼 | 救护众生离欲妙、恶业助伴、魔业 |

**"于诸不应说，众前能保密"是大乘密乘摄受众生的关键**。堪布课96：「讲密法、听密法太公开确实不是很好……因为在根基没有成熟的、智慧浅薄的人面前讲甚深的道理，他会接受不了的」——龙王降雨喻：小花小草不能承受过大雨。菩萨对众生的"保密"是慈悲的具体体现，不是吝法。

#### 4.2.4 如友之喻（课96）

**于诸染净法，自慧不迷乱，**
**世间出世间，圆满尽能授，**
**无厌无退转，恒常欲利乐，**
**五事诸佛子，犹如众生友。**

| 友事 | 菩萨事 |
|---|---|
| 为友指明路 | 以自慧了知染净法，授给众生 |
| 给友两世利益 | 授世间（梵天、帝释、轮王）及出世间（佛陀）圆满 |
| 友不厌烦自己 | 恒常不厌烦利益众生 |
| 友不被离间 | 不因恶魔苦行等而退转 |
| 友恒欲利益 | 恒常欲以世出世功德利乐众生 |

堪布课96 引《贤愚经》公案：有大国王喜欢以十善法利益众生，魔王伪造信让小国王们不行十善。国王走路上见魔王化作人在火海受苦自称"因弘扬十善法受此报"。国王问："那些人获得快乐没有？"魔王答："获得了。"国王说："没事，能让众生获得快乐，自己承受痛苦理所当然！你应该继续弘扬十善法！" ——这就是菩萨"无退转恒欲利乐"的典型写照。

#### 4.2.5 如仆之喻（课97）

**成熟诸有情，恒常具精进，**
**真实说出离，安忍诸邪行。**
**能施二圆满，善巧彼方便，**
**五事诸佛子，犹如众生仆。**

| 仆事 | 菩萨事 |
|---|---|
| 仆欢喜力行主人事 | 对成熟众生恒时具精进，不懈怠 |
| 仆如实向主人汇报 | 无误讲说真实出离苦集之道 |
| 仆忍主人打骂 | 安忍众生种种加害、恩将仇报的邪行 |
| 仆奉献主人所需 | 将世出世间两种圆满施予众生 |
| 仆善巧速办事 | 善巧使众生获世出世圆满之方便 |

"前辈大德讲过：要做佛门龙象，先做众生马牛。"（课97）——如仆之喻尤其对治现代人的"管理者心态"，提醒修行者的底色应是服务。

堪布课97 引"色拉康卓"空行母公案（他翻译的《色拉康卓自传》）：空行母修五十万加行时，达西小僧人欺负她（把肮脏鞋垫草堆到她身上，脚踩她头，说她跟牦牛无别），她始终安忍，视其为金刚道友——**这是大乘修行人"如仆"的实际典范**。

#### 4.2.6 如轨范师之喻（课97）

**欲求令获得，无生之法忍，**
**宣说一切乘，成就瑜伽行，**
**相貌妙庄严，不求报异熟，**
**五事诸佛子，如众轨范师。**

| 轨范师事 | 菩萨事 |
|---|---|
| 广闻博通摄弟子 | 欲令一切有情获无生法忍（八地功德） |
| 应弟子希求说法 | 相应所化相续宣说三乘法 |
| 迅速成就瑜伽 | 修行能成就止观的瑜伽 |
| 对弟子面带微笑 | 不现怒容，和颜悦色、相貌庄严 |
| 不图财物 | 不求回报饶益自己 |

**关键实践指导**：堪布课97 强调"应当尽可能自己调伏烦恼，甚至假装也可以"——内心虽苦但在群体中"要开会、先化个妆"。因为气氛是大家营造的，一股黑气会把满屋善能量吸走。"一些发心人员和佛教徒应该要注意，尤其是在非佛教徒的场合要小心翼翼、战战兢兢。" 此处从菩萨轨范师喻延伸出修行人的**社会仪礼原则**。

#### 4.2.7 如亲教师之喻（课97）

**利生极精进，令圆满资粮，**
**积资速解脱，能断诸违品，**
**世间非世间，圆满皆令行，**
**五事诸佛子，如众亲教师。**

| 亲教师事 | 菩萨事 |
|---|---|
| 赐弟子出家沙门学处 | 极精进利众生，令圆满二资粮 |
| 弟子受近圆戒 | 令积资者迅速解脱于果地 |
| 宣说学处违品 | 调伏众生令断六度一切违品 |
| 财物法衣资具提供 | 令众生世间圆满（财布施） |
| 出世法布施 | 令众生出世间圆满（法布施） |

七喻总体来看：**母-父-亲**偏重情感依止；**友**偏重互相扶持；**仆**偏重自我定位（服务者而非主人）；**轨范师-亲教师**偏重法义指导与戒律传授。七者合起来涵盖菩萨摄受众生的全部情感与角色维度，是"饶益有情"的七种具体形态。

## 五、报恩至希求（庚五-庚六）——菩萨意乐功德

### 5.1 庚五 报恩

**不贪诸受用，戒律悉无失，**
**具有报恩者，复令行持故。**
**如是行六度，一切诸有情，**
**于诸菩萨众，如是报恩德。**

菩萨对众生的饶益如何让众生"报恩"？**不是以财物报恩，而是以"自住六度 + 令他行六度"为报恩**：

- 受菩萨布施的众生以布施报恩：不贪一切受用而同样布施
- 受菩萨戒律饶益的众生以持戒报恩：不失戒律
- 受菩萨安忍饶益的众生以安忍报恩：不因损害而起恶意
- 受菩萨精进饶益者以精进报恩
- 受菩萨禅定饶益者以禅定报恩
- 受菩萨智慧饶益者以智慧报恩

堪布课97 特别警示："最近网上有人以我的名义让大家筹款，希望你们以后不要这样，这不但不是报恩，反而造成很不好的影响……如果一个人传了法，你认为一定要依靠供养财富才能报恩德的话，我肯定是不同意的，不仅是我，所有真正关心、重视佛法的人也不会同意。"

**真报恩之理**：《入菩萨行论》"为令如来喜，止害利世间"；《大宝积经》"己身能奉行，复化他人众"；《佛说未曾有因缘经》"常以慈心，以其所解一切善法，展转开化乃至一人，令其信心成就智慧，展转教化无有穷尽——乃名为报师徒重恩"。

### 5.2 庚六 希求

**恒求增与减，有情尽成熟，**
**于地趋胜进，求无上菩提。**

菩萨的五种希求：

1. **增**——希求自他相续六度增上
2. **减**——希求六度违品退减
3. **尽有情成熟**——令未生信心者生信心、已生信者安置于善法、已行善者成熟
4. **于地趋胜进**——一地得已，希求二地至十地越来越胜进
5. **求无上菩提**——越过十地获无上菩提

> 除此五种希求以外，世间圆满、声闻缘觉道果丝毫也不生希求。

堪布课98 引《入菩萨行论》："是父抑或母，谁具此心耶？是仙或欲天，梵天有此耶？彼等为自利，尚且未梦及，况为他有情，生此饶益心。" ——五种希求的罕有性、殊胜性正在于：连父母、仙人、天人，为自己都没有想过获证正觉之心，更何况为他众生生饶益心。

## 六、不唐捐至真实行（庚七-庚八）——菩萨行为之确定

### 6.1 庚七 不唐捐

**断畏真发心，斩断诸怀疑，**
**修行之教言，佛子恒不失。**

菩萨具四种不唐捐之加行：

1. **断畏**——对大乘深广法断除怖畏（不畏甚深空性与阿僧祇劫修行）
2. **真发心**——真实发无上菩提心（不发求世间圆满或声闻缘觉道果之心）
3. **斩断诸怀疑**——对大乘基道果斩断怀疑（不因不懂修菩提方便而退失）
4. **得修行教言**——得受修行教言，自行并为他说

> 《华严经》："行菩萨行，身语意业，悉不唐捐。"

堪布课98 提醒："如果一个人本来很庄严，但在视频里面显得很难看，这也是一种唐捐"——以自嘲方式警策发心人员。关键是：如果没有前三项（断畏/真发心/断疑）而只有教言，教言也会唐捐；如果有前三项但没有教言，也会唐捐。四者必具。

### 6.2 庚八 真实行

**无求之布施，不求后世戒，**
**一切皆安忍，功德源精进。**
**禅定非无色，智慧具方便，**
**于六波罗蜜，坚稳真实行。**

六度各自的"真实相"：

| 度 | 真实相 |
|---|---|
| 布施 | 不求今生利己，也不求他世自己得大受用 |
| 持戒 | 不求轮回增上生的后世，为得无上菩提守戒 |
| 安忍 | 三门皆忍——不仅身语不报复，心中亦不生嗔 |
| 精进 | 精进于所有波罗蜜多、所有乘道果之精进——即"功德源" |
| 禅定 | 得四禅四无色定但不转生无色界（无色界不能圆满佛法与成熟有情） |
| 智慧 | 具方便——了知空性的智慧与大悲心依四摄成熟有情，双融即是具方便之智慧 |

《圣宝积经》证："不求异熟之布施，不住一切生世之戒律，不嗔一切众生之安忍，真实摄集一切善根之精进，不为无色界所染之禅定，具足善巧方便四摄事成熟之智慧。"

堪布课98 指出："禅定非无色"极关键：菩萨不转生无色界——无色界有禅定却不能成熟众生，故非菩萨所欲之处。"菩萨的任何行为都跟利益众生有关"——这一度特别破斥"我修得安宁就好，众生太麻烦"之偏见。

## 七、退失/形象与真实性（庚九-庚十）——六度的两类歧途与对治

### 7.1 庚九 退失与胜进法之差别

**贪受用有失，我慢及享乐，**
**著味及分别，坚稳退失因。**
**当知住彼等，对治之菩萨，**
**遮除退失故，胜进具缘法。**

六度之退失因（"坚稳退失因"）：

| 度 | 退失因 |
|---|---|
| 布施 | 贪恋欲界受用 |
| 持戒 | 破坏、失毁细微戒条 |
| 安忍 | 对亲教师/轨范师/有功德者的三门不敬我慢 |
| 精进 | 处于贪著纵情享乐之边 |
| 禅定 | 耽著禅味 |
| 智慧 | 耽著实体法的分别 |

**六对治为胜进具缘法**——不贪受用、不失戒律、无我慢、不贪享受、不著禅味、于法不分别相。

此偈特别指出"**安忍之退失因是我慢**"——一般以为嗔恨退失安忍，但菩萨层面的退失是傲慢使其不堪忍恭敬对境，对治关键是"对亲教师、轨范师及有功德行者三门恭敬"。

### 7.2 庚十 形象与真实性之差别

**显示与诈现，外貌显庄严，**
**如是暂时行，身语似寂静。**
**圆满通言辞，彼等离修行，**
**说为菩萨众，非净真实相。**
**反之上等行，彼等真实性。**

六度形象（诈现）：

| 度 | 形象之相 |
|---|---|
| 布施 | 口头说施内外事物而实际不施，当面施而背后取，伪装布施 |
| 持戒 | 平常造应防护之罪，但在他人前诈现守戒 |
| 安忍 | 对他不安忍、有嗔恨，但表面假装安忍，身语庄严柔和 |
| 精进 | 平时无精进，但在大众中暂时精进 |
| 禅定 | 身住静处似寂静，但心不住禅定而徘徊于散乱 |
| 智慧 | 无善了知三乘法义智慧，但以通达声明因明耽著词句，似圆满通达言辞 |

《十住毗婆沙论》"外诈现威仪，游行似贤圣，但有口言说，如雷而无雨"——六度形象如雷无雨，真六度则是真诚无诈。"反之上等行，彼等真实性"一句即指：**凡与上述诈现相反者即是真六度**。

庚九与庚十的对照：**庚九处理"有进有退"的菩萨（有真六度基础但可能退失）；庚十处理"全然是假"的菩萨（只有形象无真实）**。前者的对治是六胜进法，后者的对治是"反之上等行"。

## 八、调伏、授记、决定（庚十一-庚十三）——菩萨的行为之方法、结果与必然

### 8.1 庚十一 调伏

**具慧于诸地，修行布施等，**
**彼等能调伏，众生六违品。**

以"五问答"结构展开：

| 问 | 答 |
|---|---|
| 是什么行者调伏？ | 具慧菩萨 |
| 安住于什么阶段调伏？ | 一地至十地 |
| 以什么方便调伏？ | 修行布施等六度法 |
| 调伏谁？ | 调伏众生 |
| 调伏什么？ | 调伏六度的悭吝等违品、退失及虚假相 |

堪布课99 引月称论师《显句论》最后："能调伏贪心，不一定能调伏嗔恨心；能调伏嗔恨心，不一定能调伏贪心；如果从根本上断除愚痴，就能根除其他所有烦恼。" 故智慧波罗蜜多是调伏六违品之根本——"五度如盲，般若如导"。

### 8.2 庚十二 行者授记分类

**以行者时别，具慧授记二，**
**菩提与授记，此外说为大，**
**获得无生忍，断我慢勤作，**
**与诸佛佛子，成为一本体。**

**授记分类**：

1. **以行者差别四种授记**：
   - 授记安住种姓之行者（未发菩提心但具六度种姓）
   - 授记发菩提心之行者
   - 授记现前之行者（眷属中明显授记）
   - 授记非现前行者（对未来不在场者授记，如授记龙树菩萨、无著菩萨）

2. **以时间差别两种授记**：
   - 时间确定（唯于此劫或此世）
   - 时间不定（未来某时）

3. **授记授记**：彼佛授记彼行者得大菩提及授记某未来佛授记此行者

4. **大授记（最重要）**：八地菩萨获无生法忍时断我慢、断相状勤作，与诸佛、佛子菩萨无有异体而成为一自性或本体之授记。此时菩萨"证悟自己无疑成佛后自己授记自己'我将成佛'"——此即自我授记。

**复彼刹土名，时间与劫名，**
**眷属法住世，许为彼授记。**

授记内容包括：刹土名、时间、劫名、眷属多少、佛法住世多久。

堪布课99 引菩提金刚对法王如意宝的授记："单坚阿拉木天喇荣沟，邬金莲師化身名晋美，彼于菩萨四众眷属中，广弘显密教法如明日，利生事业顶天立地也，清净所化眷属遍十方，凡结缘者皆生极乐刹。"——此是"**授记非现前行者**"的典型，法王并不在菩提金刚的眷属中。

关于世间"自我授记"现象，堪布特别警示："有些人认为自己可以给自己授记，说明他可能获得了八地菩萨的境界。原来我真的遇到过这种人，他说他要授记自己将来变成什么什么，这可能是分别念、猜测或估计——因明里讲的伺察意，不叫授记。" 自我授记必须以八地无生法忍为基础，非凡夫可言。

### 8.3 庚十三 决定——六度决定之果

**具慧一切时，圆满生不厌，**
**恒常能修行，等持无退失，**
**成利决定得，任运之法忍。**

菩萨真实行六度，在究竟大菩提之前的生生世世决定获得六度之六种果：

| 度 | 决定之果 |
|---|---|
| 布施 | 决定得帝释天、转轮王等圆满受用 |
| 持戒 | 决定得随愿转生善趣身 |
| 安忍 | 决定不因痛苦而厌倦，始终利他 |
| 精进 | 决定恒常能修行善法 |
| 禅定 | 决定得等持永无退失，依禅定神通决定成办有情利益 |
| 智慧 | 决定得无分别智任运自成之法忍 |

下面道中决定得同分的无分别无勤作智慧，真正八地时决定得无分别无勤作的智慧。

## 九、无疑所为至首要所为（庚十四-庚十六）——菩萨行为之定位

### 9.1 庚十四 无疑所为

**供养真受戒，慈悲修善法，**
**静处不放逸，闻义不厌足，**
**诸地坚稳者，无疑所应为。**

"无疑"即"不可不行之法"。六度之必行：

| 度 | 无疑所为 |
|---|---|
| 布施 | 以真实财物或意幻观想供养如来 |
| 持戒 | 真实受学处不失毁 |
| 安忍 | 以悲心驱使安忍 |
| 精进 | 精进修行世出世善法 |
| 禅定 | 居寂静处不放逸 |
| 智慧 | 依止善知识，闻经藏等佛法之义无厌足 |

《思益梵天所问经》"譬如大海能悉受，一切众水无满时，此诸菩萨亦如是，常求法利无厌足。"——对应第六度"闻义不厌足"。

### 9.2 庚十五 恒常所为

**了知欲妙过，思维诸迷乱，**
**甘受诸痛苦，修行诸善法。**
**不著禅乐味，不分别相状，**
**诸地坚稳者，恒常所应为。**

六度之恒常：

| 度 | 恒常所为 |
|---|---|
| 布施 | 恒常了知欲妙过患（如浮云、闪电、毒食、火坑、剑锋、蛇穴） |
| 持戒 | 思维身语意一切恶劣迷乱所行 |
| 安忍 | 甘受种种痛苦（思维众生漂泊三苦波涛） |
| 精进 | 修行世出世一切善法 |
| 禅定 | 不贪著禅乐味 |
| 智慧 | 不分别所取能取相 |

堪布课99 此处讲"欲望"的反思："以前西方著名艺术家王尔德说过，人生中最悲剧的有两件事情：一个是得到的悲剧；一个是求而不得的悲剧。" ——对应《大智度论》"诸欲求时苦，得之多怖畏，失时怀热恼，一切无乐时。"菩萨于恒常中思维此理。

庚十四（无疑）与庚十五（恒常）之别：前者说明"必须做"，后者说明"时时做"。二者叠加，既是频率与强度的保证。

### 9.3 庚十六 首要所为——六度各自最殊胜之支分

**法施清净戒，获得无生忍，**
**精进大乘法，竟具住悲心，**
**般若许具慧，波罗蜜多首。**

六度各自的首要（最殊胜之支分）：

| 度 | 首要之支 | 理由 |
|---|---|---|
| 布施 | 法施 | 能成办世间与出世间圆满；《金刚经》七宝布施不如四句偈 |
| 持戒 | 无漏戒（或摄善法戒） | 圣者欢喜之极清净戒 |
| 安忍 | 谛察法忍（获无生法忍） | 对空性法能通达无恐惧 |
| 精进 | 大乘法之闻思修精进 | 超越世间人、声闻缘觉之精进 |
| 禅定 | 第四静虑具悲无量心 | 止观均衡究竟，依此安住悲无量心 |
| 智慧 | 般若度（于前五度中最殊胜） | 前五度必依般若圆满 |

《般若经》："设若欲求圆满布施波罗蜜多，当学修般若波罗蜜多等。"

堪布在课100（即百堂课纪念日）提醒：「修禅定时，有人说"我实在安住不下来怎么办呢？"安住很困难的话，你就修大悲心嘛。大悲心就是经常想到众生的痛苦，以前一些格西经常到蚂蚁窝去观悲心。」——第四静虑与悲无量心之结合，是禅定度首要所为的具体教授。

## 十、说法安立（庚十七）——菩萨说法之系统

### 10.1 庚十七的五重展开

**庚十七 说法安立** 是本品最大的一庚，密意疏占约 165 行，本论对菩萨说法学最系统的展开。分五辛：

- **辛一 所说法之差别安立**（法/谛/理/乘四安立）——课100
- **辛二 遍知法义安立**（四寻求/四遍知/三自性修行/阿赖耶转依/心净国土净）——课101-102
- **辛三 说法者菩萨五种无量功德**（所熟/所清净/所得/堪熟/真说）——课102
- **辛四 说法之果安立**（八种果：发心/无生忍/法眼/漏尽/法久住等）——课102
- **辛五 大乘特法及归摄安立**（七大 + 八归摄）——课102

### 10.2 与品13 说法品、品12 求法品的分工

本品之"说法安立"与 [`14-说法品.md`](./14-说法品.md) 的"说法方式"及 [`13-求法品.md`](./13-求法品.md) 的"所求之法"有何区别？密意疏自设问答：

> 有人会想：这在《求法品》与《说法品》中不是已经宣讲了吗？为何再度重复讲解？
>
> 此处是说明精通菩萨说法四种安立的功德时，虽然与上述无有异义，但**从安立方式不同及所知的摄义有特殊差别的角度宣说**，因此，不仅没有重复的过失，反而有重大意义。

即：
- 求法品（品12）——从"菩萨求什么法"角度（十一种证法、作意、果）
- 说法品（品13）——从"菩萨如何说法"角度（说法方式、四秘密四意趣）
- 功德品之说法安立（品20庚十七）——**从"菩萨说法的知识体系安立"角度**（法/谛/理/乘四安立、遍知法义、说法者五无量、说法果、大乘七大），是对菩萨说法体系的**系统性归纳**

三品合起来构成"求—说—安立"的完整链条。

### 10.3 辛一 所说法之差别安立（四安立）

#### 10.3.1 壬一 法安立——五明为法

**以契经等别，安立为明处，**
**当知一切地，坚稳法安立。**

以十二部经（契经、记别、讽诵等）的差别，安立为**内明、声明、因明、工巧明、医方明**五明处，这是信解行地及十地一切菩萨之"法安立"。

**关键命题**：**五明即是佛法，不是非佛法**。密意疏逐层论证：

- 声明——字母组合、结构、格变形成语言学论典之安立
- 因明——现量、比量、教量三量衡量现前/隐蔽/极隐蔽之理的安立
- 医方明——健身养生、饮食行为药物医疗之安立
- 工巧明——身语意串习之各种安立（含历算、天文、八观察等）
- 声明分支——修辞学、韵律学、戏剧学、词藻学
- 内明——无误安立并宣说三乘各自道果

**五明在佛经中的分布**：
- 因明——《解深密经》四种道理、《集量论》《因明七论》
- 声明——经藏"有碍故为色，领受性之受"等释词
- 医方明——《金光明经》中讲健身养生
- 历算——《时轮金刚续外世间品》
- 工巧明——《方广庄严经》等

> **麦彭仁波切判**："佛菩萨的幻变无量……十方浩瀚无边刹土中五明所摄的无谬论典，也是依靠菩萨的威德力而宣说。认定声明等在佛经中无有而将其一概定为外道宗，是智慧极其狭隘的表现。"

堪布课100 由此引出**喇荣五明佛学院的命名内涵**：「上师如意宝当时对佛学院取名很有包容性，不只是可以对应藏传佛教或汉传佛教，或者单一的密宗、显宗。因为这里有空间，所以很多后学者在此学什么都好像很放松」——**"五明"命名是一种开放性教义立场**，而非名号偶然。

#### 10.3.2 壬二 谛安立——七真如

**依于七真如，安立真实谛。**

依七种真如安立真谛：

| # | 真如 | 含义 | 对应四谛/三自性 |
|---|---|---|---|
| 1 | 转入真如 | 十二支缘起环环相扣持续转入轮回 | 苦谛（持续性） |
| 2 | 法相真如 | 遍计/依他起/圆成实三者法相 | 统摄 |
| 3 | 心识真如 | "三界唯心"——万法心本身显现 | 依他起摄 |
| 4 | 安住真如 | 恒住三苦自性之轮回 | 苦谛 |
| 5 | 倒成真如 | 具业与烦恼法相——集谛 | 集谛 |
| 6 | 清净真如 | 清净苦集过失之灭谛 | 灭谛（圆成实离客尘） |
| 7 | 正成真如 | 无倒修成灭谛之道谛 | 道谛 |

七真如归摄：
- 苦谛 = 转入 + 安住
- 集谛 = 倒成
- 灭谛 = 圆成实自性远离客尘（清净真如）
- 道谛 = 正成真如

**最终归摄为《楞伽经》五法三自性八识二无我**："五法三自性，以及八识聚，二种无我义，归摄诸大乘。"

五法（名/因/分别/真如/正智）：
- 名——以柱子、瓶子等名称安立，执彼彼为遍计
- 因——依他起显现为所取分
- 分别——显现为能取分的八识聚
- 真如——无变圆成实
- 正智——真如的有境无倒圆成实

堪布课100 提示："原来讲《中观庄严论释》的时候要求大家背诵"——此与 [`collections/中观庄严论/`](../中观庄严论/) 有直接贯穿。

#### 10.3.3 壬三 理安立——四种理

**安立理与乘，四种及三种。**
**如理而作意，正见具有果，**
**量辨不可思，当知为四理。**

四种理（与 [`collections/解义慧剑/`](../解义慧剑/) 及 `topics/pramana/index.md` 对应）：

| 理 | 本义 | 适用范围 |
|---|---|---|
| **作用理** | 了知某因生某果 | 从因看果，如种子生苗芽 |
| **观待理** | 了知某果观待某因 | 从果看因，如苗芽观待种子 |
| **法尔理** | 世俗法尔（火热水湿）、胜义法尔（诸法空性） | 不可推理之自性之理 |
| **证成理** | 以现量比量相应诸法实相而衡量 | 一切量辨的根本方法 |

偈颂解读：
- "**如理而作意，正见具有果**"——作用理 + 观待理：信解行地如理作意二无我引出见道正见，见道正见具解脱涅槃之果；因果相属是无欺的缘起法（"具生长力种子产生苗芽"）
- "**量辨**"——证成理（现量比量衡量）
- "**不可思**"——法尔理（胜义中所生能生的量了不可得，胜义法性不可思议）

> 这是唯一运用于真实道的四种道理。以此理为例，应当了知一切衡量意义之理均集于四种道理中。

**与 pramana / 因明的关联**：四种理实际上是因明学"量"（pramana）的本论展开——"**以正量辨别或证悟，即证成理**"是本论与释量论、量理宝藏论的直接桥梁。可参 `topics/pramana/index.md` 对 pramana 诸家的系统化处理。课100 堪布提示："四种理是什么呢？学过《解义慧剑》的人都应该清楚"——[`collections/解义慧剑/四法依.md`](../解义慧剑/四法依.md) 即以四种道理为核心辨析。

#### 10.3.4 壬四 乘安立——三乘五差别

**意乐及宣说，加行与资粮，**
**成就之差别，承许为三乘。**

以**意乐、宣说、加行、资粮、成就**五法之差别形成三乘：

| 乘 | 意乐 | 宣说 | 加行 | 资粮 | 成就 |
|---|---|---|---|---|---|
| **声闻** | 证人无我，希求自己速解脱，下品 | 人无我、四谛小法 | 三世或七世修习 | 福慧资粮下品 | 声闻小菩提 |
| **缘觉** | 证一个半无我，百劫修习坚持，中品 | 顺逆缘起法、所取如梦法 | 百劫 | 比声闻善妙的中等资粮 | 缘觉阿罗汉（天眼见三千大千界） |
| **菩萨** | 证二无我，以大悲缘他利成办二利，广大 | 大乘深广法 | 身语意行持二利 | 广大福慧资粮 | 无上大菩提广大果 |

**大乘七大在下文"大乘归摄"中讲（即辛五）**。

### 10.4 辛二 遍知法义安立——四寻求、四遍知与"心净国土净"

辛二是本品最长且最深的一段（课101-102），密意疏约 45 行。此辛从"四种寻求→四种遍知→束缚→解脱→三自性修行→转依→清净刹土"逐层推进，**核心目的是以唯识宗三自性框架说明菩萨如何依般若慧离相断缚**。这一辛与品7 真如品遥相呼应，是本论又一次 "以唯识方式证成二无我"。

#### 10.4.1 四寻求

**名事相互间，寻求为客性，**
**于此二假立，寻求唯此性。**

菩萨信解行地对四法的"寻求"：

| # | 寻求 | 内容 |
|---|---|---|
| 1 | 寻求名 | 瓶子、氆氇等名称——暂时以分别念将意义假立为名称 |
| 2 | 寻求事/义 | 装水的大腹器具、毛线组合的可穿之物——暂时以分别念将名称假立为意义 |
| 3 | 寻求本性假立 | 地坚硬、水湿润等自相——暂时以分别念安立每个事物的本体 |
| 4 | 寻求差别假立 | 无常、苦、有漏无漏等差别——暂时以分别念安立事物的差别 |

核心洞察：名与义、本性与差别**互为客性**，**唯是假立，不是实有**。若名与事相联是一体自性，则瓶子毁灭时"瓶子"之名也应毁灭，但实际名称与所指无此同生共灭关系。

**驴驮佛像的公案**（课101）：寺院驴背上驮佛像出门，路人皆顶礼，驴误以为是顶礼它；第二次没驮佛像出门又想被顶礼，结果被打。僧人解释："你错了——第一次是顶礼你所驮的佛像。" **此公案正是对"名事相互间，寻求为客性"的通俗呈现**：名声、权力、相貌、财富都是"客"，不是人的本体，一旦失去就不被顶礼。

#### 10.4.2 四遍知

**一切不得故，真实四种智，**
**诸地坚稳者，能成一切利。**

信解行地依四寻求在见道生起四遍知：因名/义/本体/差别四种**皆不可得**，故生起四种智——寻求名/事/本体/差别皆了不可得的智慧。此智能成办十地菩萨的自他一切利益。

#### 10.4.3 解脱束缚的唯识修法

**束缚之所依，受用种子因，**
**心心所及依，有种而束缚。**

**能束缚三因**：
1. **所依**（器世界之相，阿赖耶识显现为所缘对境）
2. **受用**（六识享受六境）
3. **种子因**（一切种子阿赖耶识）

**所束缚**：心（八识聚） + 心所（五十一心所）+ 所依（色蕴/六根身） = 五蕴

**如何束缚**：依处、受用相的习气或种子积于阿赖耶，未断除前受束缚。

**前存相状法，本性而住相，**
**悉灭具慧者，获得大菩提。**

解脱之方便——灭**两种相**：
- **前存相状法**：闻思修所住存心前之浮肿观、骨锁观等——以心刻意观想的相
- **本性而住相**：从无始轮回串习相分别而不需刻意的瓶子柱子等俱生显现

破相之道：先了知以心思择显现的骨锁等相只是浮现在心里，破析行相无有成立之义；再认识瓶子等一切事物本性住相也只是浮现在心，自性不成立。禅宗"凡所有相，皆是虚妄"之依据即此。

#### 10.4.4 依三自性修行——断二障证三身

**缘于真如智，断除能所取，**
**现前习气身，具慧能灭彼。**

三自性修行的三句式：
- **缘于真如智**——圆成实：以缘法界之真如智现前圆成实法相
- **断除能所取**——遍计所执：断二取（能取所取、人我法我）
- **现前习气身，具慧能灭彼**——依他起：具烦恼障所知障习气的阿赖耶识即依他起；具慧菩萨现行三法相而灭尽依他起所具二障

### 10.4.5 菩萨与声闻无相之别

**缘于真如智，修相无异体，**
**现前有无义，名驾驭分别。**

声闻与菩萨修无相的根本差别：

| 乘 | 无相修 |
|---|---|
| 声闻 | 缘**有相轮回 vs 无相涅槃**为异体：视轮回有生老病死为不寂灭相想断除，视涅槃无苦恶相为寂灭相想现前 |
| 菩萨 | 缘**轮涅自性本为涅槃**，所断轮回与所取涅槃的行相无有异体；遍知轮涅一切法为空性 |

《无尽慧经》："法界、轮回界、涅槃界、众生界、诸法之界等同。何故等同？同等为空性故彼为等同。" 

菩萨证得二空自性圆成实时，"**现前有无义**"——"有"指断二障的空性智慧存在，"无"指遍计二我假象不存在。此时**驾驭分别**——想"愿从空虚空降资具雨"也会降下资具雨，此即自在分别。

### 10.4.6 阿赖耶转依与解脱

**凡夫障真性，普现非真实，**
**菩萨遣除彼，普现真实性。**
**不现实无有，显现真实有，**
**彼即为转依，随行名解脱。**

**凡夫 vs 菩萨**：
- 凡夫：执我法习气障遮蔽真如性，普遍显现非真实二现轮回，如眼翳者见飘浮毛发、迷乱识见绳为蛇
- 菩萨：对我法无执，遣除二取显现而普遍显现无二真实性真如，如眼病清净者见无模糊、意识无迷乱者了知绳是绳

**转依定义**："**不现实无有，显现真实有，彼即为转依**"——**不显现实际无有的（二取相），显现实际有的（真如）**，即依他起阿赖耶识转依。

"**随行名解脱**"——转依而离一切相束缚，自心获得随意而行故名解脱。

> 有人想：心解脱了一切束缚，但外在显现的器世界如何消除呢？

### 10.4.7 清净佛刹土之理——"心净国土净"的唯识立场

**境大一切时，他法现同类，**
**是能障碍故，遍知断除彼。**

清净佛刹之理：

1. 显现外器世界的行相是**阿赖耶界中器世界习气的所取分**，内在阿赖耶识缘取而现为所取能取；外在实有的器世界不存在（如梦中显现器世界相）
2. "境大一切时"——器世界似乎一切时处不间断显现，实是阿赖耶识刹那串连持续而如是显现
3. "他法现同类"——前后不同他法的刹那显现为同类，相似错乱为一个
4. **众生有共业的习气**，以习气力，所有他法显现相同；若习气改变（如成就行地下者见入地孔穴），行相显现变异
5. **障碍清净刹土之根源**：耽著外境自相显现为实存
6. **成就清净佛刹之方便**：遍知此障后，断除器世界相及对它的耽著，安住于无相真如中

> 乃至心耽著显现大地等器世界有自相，那就是清净佛土的障碍。如此遍知障碍后，断除器世界的相及耽著它为它，而安住于无相真如中，就会成就清净佛刹大地庄严等圆满布局的清净相。

### 【correctness anchor 关键跨集合对照：本论的"唯识立场心净国土净" vs 维摩诘经的"般若立场心净国土净"】

**此段（辛二辛末）的"心净国土净"与 [`collections/维摩诘经/品01-心净国土净.md`](../维摩诘经/品01-心净国土净.md) 的"心净国土净"教义，是大乘两大思想系统对同一命题的不同进路处理，必须对照理解，不可混为一谈**。

#### 对照要点

| 面向 | 本论（唯识立场） | 维摩诘经（般若/中观立场） |
|---|---|---|
| 命题出处 | 庚十七辛二末："断除器世界的相及耽著它为它，而安住于无相真如中，就会成就清净佛刹大地庄严" | 佛国品："若菩萨欲得净土，当净其心；随其心净，则佛土净" |
| 外境的本体论地位 | **阿赖耶识习气所取分之显现**——外境无实体，但有阿赖耶习气 | "众生之类是菩萨佛土"——**外境即众生所感**，不归为阿赖耶识 |
| 不清净的根本原因 | **耽著外境有自相的习气**障碍清净刹土 | "仁者心有高下，不依佛慧，故见此土为不净"（螺髻梵王语）——**心有高下（分别执着）+ 不依佛慧（未证般若平等智）** |
| 转依/清净的方式 | 阿赖耶识转依——"不现实无有，显现真实有"，安住无相真如 | 修十七种净土因（三心、六度、四无量、四摄等），以"依佛慧"见本来清净之土 |
| 哲学框架 | 三自性（遍计、依他起、圆成实）——依他起转依 | 空性—缘起：外境非独立实体，心生则境生 |
| 与密法的关系 | 密法在此立场上开显为大圆满/本来清净，但唯识宗本身不直接说本来清净 | 堪布特别强调维摩诘经的"心净国土净"需借密法"本来清净"才能透彻 |

#### 两者并非矛盾，而是**侧重差别**

- **本论强调"观察与修行"的过程**：如何从不清净的二取显现转为清净真如，即"转依"工夫。说明**"心不净所以国土不净"的机制在于阿赖耶习气**。
- **维摩诘经强调"本来清净"的宣示**：佛陀神足按地立显净土，证成"国土本来清净，唯众生罪见不到"。说明**"国土的本净本来如此，需般若平等智能见"**。

麦彭仁波切《定解宝灯论》警示：「假设未知此法理，认为轮回不清净，同时观为净天尊，亦如吐物瓶涂香，修持等性金刚乘，如画灯火真可悲！」——即**"心净国土净"若停留在"把不净观为净"层面是假修；必须通达本来清净之理，才是真修**。此警示同时适用于唯识立场（需通达阿赖耶转依的真实义）与般若立场（需通达依佛慧见本净）。

**堪布课102 的一语道破**：
> 按照唯识宗的观点或以大乘名言理来观察，外境真实的本体是不成立的，**显宗、密宗都这样认为**。

即：**无论唯识立场（三自性/阿赖耶转依）还是般若立场（心净国土净/本来清净），在"外境无实"这一点上完全一致**——二者不是诤论关系，而是同一真理的两种进路。学人可从两端同时进入，于本论处从"转依工夫"进入，于维摩诘经处从"本来净土"进入。

### 10.5 辛三 说法者菩萨五种无量功德

**所熟所清净，所得堪成熟，**
**真实解说事，具慧之无量。**

真实智慧解脱相束缚的菩萨有五种无量功德：

| # | 无量 | 内容 |
|---|---|---|
| 1 | **所成熟事无量** | 成熟天边无际有情：声闻/缘觉/菩萨三种种姓以各自乘成熟、不定种姓以三乘任一成熟、断种者安置十善 |
| 2 | **所清净事无量** | 器世界全无深渊荆棘污秽，以珍宝地、甘露流、如意树庄严；世界如锦缎位于整个虚空 |
| 3 | **所得事无量** | 断善不善中一切不善，现前十力、四无畏、十自在等无漏善法 |
| 4 | **堪所成熟事无量** | 除断种姓外的四种种姓堪为三乘所成熟（永久堪成熟——从成熟至解脱角度） |
| 5 | **真实宣说事无量** | 以调伏众生的方便说法：贪欲者宣说不净观、嗔恨者宣说慈心等法门不可思议 |

五者依次对应：**缘于谁** / **安住于什么器世界** / **具足什么所得** / **哪些有种姓者** / **以什么说法能成熟**。

### 10.6 辛四 说法之果安立——八种果

**令生菩提心，令得无生忍，**
**低下无垢眼，一切漏法尽，**
**妙法久住世，知断及受用，**
**了知行于彼，具慧说法果。**

说法八种果：

| # | 果 | 内容 |
|---|---|---|
| 1 | 令生菩提心 | 说大乘法令其他行者发无上菩提心 |
| 2 | 令得无生忍 | 说大乘法使有些人获无生法忍（小忍/中忍/大忍三级） |
| 3 | 低下无垢眼 | 以说声闻法获声闻见道，以见四谛法眼断见断，即解脱见断的预流向 |
| 4 | 一切漏法尽 | 声闻见道依法断三界修所断烦恼，得尽智无生智——阿罗汉 |
| 5 | 妙法久住世 | 说大小乘任何法令妙法长久住于此世间 |
| 6 | 知 | 了知以前未知之诸法自相共相 |
| 7 | 断 | 斩断怀疑 |
| 8 | 受用 | 无不知与怀疑而解法义，受用微妙无罪法喜 |

八果中前四果**一 二**为大乘成就、**三 四**为小乘成就；**五**是大小乘共同成就；**六 七 八**是讲者自身成就。

堪布课102 引《佛说贤者五福德经》复述此理的简版——讲法者本身可获**长寿**（不杀生的等流）、**富裕**（不偷盗的等流）、**相貌端正**（不生嗔的等流）、**名声远扬**（受尊重的等流）、**大智慧**（开他人智的等流）五福德。

### 10.7 辛五 大乘特法及归摄安立

#### 10.7.1 大乘七大

**由从所缘大，如是修行大，**
**智慧精进大，善巧方便大，**
**真实成就大，佛陀事业大，**
**具足此大故，决定谓大乘。**

大乘以七种特点超胜小乘：

| # | 大 | 小乘 | 大乘 |
|---|---|---|---|
| 1 | **所缘大** | 声闻三藏 | 大乘深广无量法藏 |
| 2 | **修行大** | 仅为自利修行 | 为自他二利修行 |
| 3 | **智慧大** | 仅证人无我 | 证人无我与法无我 |
| 4 | **精进大** | 仅于三世或七世 | 三大阿僧祇劫 |
| 5 | **善巧方便大** | 畏轮回求断寂乐，不具善巧方便 | 以悲心不舍轮回，以智慧不被染，如莲花出淤泥 |
| 6 | **真实成就大** | 获阿罗汉果尽处胜处六通后无余涅槃 | 真实成就十力四无畏等永不穷尽无量果 |
| 7 | **事业大** | 获阿罗汉但不现证菩提转法轮 | 现前成佛后住密严宫殿，示现入胎现证菩提尽未来际间广大事业 |

前五是**因乘/道乘**的特点，后二是**果乘**的特点。此七大是大乘所以称"大"的核心定义。

#### 10.7.2 大乘归摄八安立（亦可分为十）

**种姓信解法，发起菩提心，**
**行持布施等，趋入无过患。**
**成熟诸有情，清净佛刹土，**
**不住之涅槃，大觉及示现。**

《般若十万颂》《华严经》等大乘无量经所说归摄为八（或十）种安立：

| # | 安立 | 位置 |
|---|---|---|
| 1 | 具大乘种姓/界性 | 安住种姓阶段 |
| 2 | 依种姓信解大乘法 | 安住种姓阶段 |
| 3 | 以信解发起无上菩提心 | 安住种姓阶段 |
| 4 | 发心后于信解行地行持布施等六度 | 信解行地 |
| 5 | 生起一地远离所取能取智趋入出世无过之圣地 | 一地 |
| 6 | 二地至七地以菩提分法及波罗蜜多法令众生信仰佛教/成熟善法 | 二至七地 |
| 7 | 三清净地修行器情清净——清净佛土；以智慧不住轮回、以悲心不住寂灭之涅槃 | 八至十地 |
| 8 | 越过十地得佛地三身大菩提及示现佛事 | 佛地 |

若后面四者分开计算则为十种安立。这与品10 菩提品之结构相呼应——**功德品的大乘归摄八安立是信解至佛地的完整道次第摘要**，与五种无上意义（品1）和二十一品的整体架构相呼应。

### 11.1 十七功德的综合评估

本品**十七功德**是本论对"**已修完前面所有菩提分法的菩萨具何种功德**"的分类性总览。综合来看：

1. **庚一-庚三（总说三功德）**——从自性角度定位菩萨（六度稀有、非大稀有、平等心）
2. **庚四-庚六（利他三功德）**——饶益、报恩、希求
3. **庚七-庚十（行为四功德）**——不唐捐、真实行、退失胜进、形象真实
4. **庚十一-庚十三（方法果位三功德）**——调伏、授记、决定
5. **庚十四-庚十六（定位三功德）**——无疑所为、恒常所为、首要所为
6. **庚十七（说法功德，最大）**——说法的系统性展开（法/谛/理/乘四安立、遍知法义、五种无量、说法八果、大乘七大与八归摄）

### 11.2 本品对《大乘经庄严论》全论的意义

**本论属正行六品之第五品**，与前四正行品有以下逻辑：

- 品16 方便品（[`17-方便品.md`](./17-方便品.md)）——菩萨修行的三方便
- 品17 六度四摄品（[`18-六度.md`](./18-六度.md)、[`19-四摄.md`](./19-四摄.md)）——具体修行内容
- 品18 供养依师无量心品（[`20-供养依师无量心品.md`](./20-供养依师无量心品.md)）——修行支持
- 品19 菩提分品（[`21-菩提分品-八因.md`](./21-菩提分品-八因.md)、[`22-菩提分品-道体性与差别支.md`](./22-菩提分品-道体性与差别支.md)）——修行的道体性
- **品20 功德品 上（本档）——修行所感之十七种功德**
- 品20 功德品 下（[`24-功德品-赞叹菩萨.md`](./24-功德品-赞叹菩萨.md)）——对菩萨的赞叹
- 品21 行为究竟品（[`25-行为究竟品.md`](./25-行为究竟品.md)）——最终之相、受生、十地、究竟

因此本品在正行六品中占**"已修→所成功德"**的位置，即菩萨修前四品之正行后，所成就的各种功德集中展示。

### 11.3 百堂课的里程碑

课100 是《大乘经庄严论讲解》四年 111 堂课的第 100 课（本品讲到庚十六首要所为），堪布在这一课特别感慨：

> 听100堂课对你们而言很不容易，很多人从第1课开始一直到今天；对我而言也有点不容易，一直坚持着走过来。100堂课加上前后念诵，全部算下来差不多有200多个小时，我们四年时间一直在努力……刚开始发这个愿不容易，中间走这条路也不容易，最后有一个圆满的结尾也不容易。

这段独白正是本品"**精进大、不唐捐、长时修**"在弘法者身上的实际体现——一位菩萨弘法者怎样身体力行菩萨的十七种功德。
