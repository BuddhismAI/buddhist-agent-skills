# 发清净愿——思维自身而发愿

本篇覆盖《极乐愿文大疏》最为实修紧要的一段——**丁四「发清净愿」之戊一「思维自身而发愿」**，共 L89 至 L98 前半，相当于原论整段「发清净愿」(译作第 9-144 行)的前半核心。

这是乔美仁波切《极乐愿文》最脍炙人口、也是最直接用于临终的部分。索达吉堪布在讲记中反复强调：**这一段可以直接作为临终诵读文本逐句念诵**，其中每个偈颂都对应一个临终心念的引导。因此本篇内容远非抽象愿文——它是一部微型的临终诀窍书。

---

## 一、发清净愿在往生四因中的定位

### 1.1 为何是第四因

往生极乐世界的四因结构（索达吉堪布讲记屡屡回溯）：

| 四因 | 功能 | 偏重 |
|---|---|---|
| 一、明观福田 | 观想阿弥陀佛主眷、刹土 | 对境 |
| 二、积资净障 | 七支供、三十五佛忏、金刚萨埵 | 因地 |
| 三、发菩提心 | 为利众生愿成佛 | 动机 |
| **四、发清净愿** | **将善根导向极乐、树立目标** | **方向** |

乔美仁波切在本论中以农业作譬——种庄稼不可缺少地、水、火、风四缘，缺一庄稼不生；往生不可缺少此四因，缺一不成。现代很多净土行人只知「一心念佛」即可（莲池大师的确说过信心极猛利者一心念佛亦可往生），但索达吉堪布根据藏文《无量寿经》的强调，认为对绝大多数人而言四因缺一不可。

### 1.2 发愿与业力、与回向的互补

- **业力**决定你能去哪里（资粮够不够）
- **愿力**决定你想去哪里（方向明不明）

没有愿导向，善业会随宿世习气成熟为人天果报；有愿无业，愿力孤单无所依。二者结合，善业才能顺利导入极乐。这也是喇拉曲智仁波切结尾总结的：**「发愿能得实现之因主要观待积累福德资粮，所以具福德者可以实现一切所愿。」**

### 1.3 戊一的四支科判（思维自身四愿）

```
戊一 思维自身而发愿
  己一 发愿临终面见佛菩萨          (L89)
  己二 发愿死时断除对轮回之贪执
    庚一 对轮回之痛苦生起厌离心     (L90-91)
    庚二 断除贪执                   (L92-94)
  己三 发愿往生极乐世界            (L95 前半)
  己四 发愿往生后所获之功德
    庚一 愿获得妙相随好身           (L95 后半)
    庚二 愿断除疑障                 (L96 前半)
    庚三 愿得授记闻佛法             (L96 后半)
    庚四 愿赴其他清净刹             (L97 前半)
    庚五 愿游不清净刹土             (L97 后半-L98 前半)
```

这四支的内在顺序对应**临终-中阴-往生-往生后**的四个时间段，所以整段可以看作一部迷你往生全程指南。

## 二、己一：发愿临终面见佛菩萨

### 2.1 根本颂

> **与我结缘众，愿临命终时，**
> **化身无量光，比丘僧眷绕，**
> **亲临吾等前，见彼心欢悦，**
> **无有死亡苦。**
> **愿八大菩萨，神力临空中，**
> **指示极乐道，接引得往生。**

### 2.2 「结缘众」的含义——善缘恶缘无分别

乔美仁波切历史上带着「母亲、眷仆、门犬」一同往生极乐世界；萨迦派的索朗则模大师也是携眷属往生。这是本颂「与我结缘众」的实例。

**「结缘」包含两面**：

- **善缘**——法缘、财缘、亲情缘
- **恶缘**——被抢夺、摧毁、病魔陷害、结怨

大乘的关键在于：**不以「带贤善舍恶劣」的贪嗔之心发愿**。索达吉堪布特别批评某些居士只愿带关系好的人往生，拒绝带仇敌——「这个人很不好，如果跟他一起往生极乐世界，那我们在极乐世界也会吵架的」（众笑）。这是世间贪嗔之心，不是菩提心。应以平等心发愿：**凡与我有缘者，无论善恶缘，都愿被阿弥陀佛救度。**

文殊菩萨的十大愿中大多愿度化「轻毁、侮辱自己以及对自己制造违缘的众生」——这是发清净愿的标杆。

### 2.3 死亡相与佛菩萨接引的仪轨次第

**临终四大隐没相**（《大圆满前行》体系）：

1. 身体无气力、食欲尽、衣被皆觉沉重（地大隐没）
2. 双目模糊不见色（水大隐没起）
3. 耳不闻声、感受剧烈痛苦（火大隐没）
4. 气息中断、四大最终瓦解（风大隐没）

**罪业深重者**：出现恐怖景象；心识迷乱；语无伦次；身不由己瘫在屎尿中。

**行善者**（修持四因者）：阿弥陀佛由比丘僧众围绕，降临面前虚空的彩虹光芒中，指示极乐道。依生前善习力，「犹如儿子熟悉母亲一般」认出佛，于安乐、明清、喜悦中死亡，解脱于光明中，无丝毫死亡痛苦。

### 2.4 八大菩萨接引的教证依据

「八大菩萨」——文殊、观音、大势至、普贤、弥勒、除盖障、地藏、虚空藏（依《八大菩萨曼荼罗经》；《药师经》与《七佛八菩萨所说神咒经》的列名略异）。

主要教证：
- 宗喀巴大师《极乐愿文》：「愿我一旦命终时，明现佛陀无量光，如海眷众所围绕……愿现中阴境即时，八大佛子示正道」——**与乔美仁波切说法完全一致**
- 《阿弥陀经》：「其人临命终时，阿弥陀佛与诸圣众现在其前，是人终时心不颠倒，即得往生」
- 《普贤行愿品》：「愿我临欲命终时，尽除一切诸障碍，面见彼佛阿弥陀，即得往生安乐刹」
- 《宝箧经》：「凡听闻六字真言『嗡嘛呢叭美吽』者，命终时由十二佛陀、八大菩萨引路而往生极乐世界」
- 《大乘庄严宝王经》：「若人恒念大士名，当得往生极乐界」

### 2.5 临终十念之力

《观经》：「具足十念称南无阿弥陀佛，称佛名故，于念念中除八十亿劫生死之罪……一念顷即得往生极乐世界。」

《无量寿经》第十八愿：「设我得佛，十方众生，至心信乐，欲生我国，乃至十念，若不生者，不取正觉。」阿弥陀佛已经成佛——愿已成就——故十念法坚信不疑。

**为何临终心力特别强**？索达吉堪布譬喻：「势力强大的国王发布一条命令，下面的人不得不执行。」临终心识力量极大，一心祈祷阿弥陀佛即可。

**《三宝感应要略录》敲鼓念佛公案**：印度婆罗门愚痴不信三宝，妻子给出方便——每次她修法完敲鼓，他就念几句「南无阿弥陀佛」。三年后他病死，五天后复活说：死后堕镬汤地狱，阎罗狱卒用铁锤击打地狱铁锅，他误以为是妻子的鼓声，习惯性念佛，当下地狱化为清凉池，地狱众生都往生极乐。偈颂：**「若人造多罪，应堕地狱中，才闻弥陀名，猛火为清冷。」**

### 2.6 串习决定临终一念——死法公案串

索达吉堪布以《养生篇》（龙树）偈颂总领：**「一切众生之功过，皆以串习为根本，习惯悉皆依自己，是故串习德最胜。」**

生前串习什么，临终显现什么。讲记中反复引用下列临终公案：

| 串习者 | 临终显现 |
|---|---|
| 藏北牧民宰旱獭者 | 喊「请杀掉这些旱獭，快把它们赶走！」而死 |
| 猎人 | 喊「给我拿火枪来，这些鹿子要杀我」惨叫而死 |
| 喜吸鼻烟者 | 作吸鼻烟姿势而死 |
| 裁缝 | 作缝纫姿势而死 |
| 商人 | 说「拿账本来」边数账而死 |
| 竹青寺念施水仪轨老僧 | 念着施水咒「桑巴Ra……」而逝 |
| 辩论法相师 | 一边说「给，杰达秋坚」一边作辩论手势而圆寂 |
| 德国诗人海涅 | 喊「我的笔，我的纸」而死 |

这些都指向一个核心教言：**平时串习阿弥陀佛，临终自然忆念阿弥陀佛。**

### 2.7 临终实修小节（索达吉堪布强调）

- **念佛号 / 持《极乐愿文》**——生前行住坐卧不忘念佛（白居易：「行也阿弥陀，坐也阿弥陀，纵然忙似箭，不废阿弥陀」）
- **面西卧 / 佛陀涅槃卧**——帮助心向西方
- **眼观佛像、耳听佛号**——营造临终氛围
- **家人不哭嚎**（关键！）——《经律异相》公案：一位持戒精进的居士临终时妻子哭喊「你走了我靠谁呀？儿子怎么办啊？」，居士生起贪执一念，结果变成妻子鼻中小虫。索达吉堪布：「按佛教传统，人死后亲人不能哭泣，可是我在医院经常听到『你走了以后我该怎么办啊！』一听到呼叫死者的名字，我就担心：说不定死者会转生为亲友身上的旁生。」
- **助念**——居士群念佛号；若突然横死，依《六中阴》中的刹那往生法——**心识立即专注于头顶上阿弥陀佛**。
- **突然横死时的刹那破瓦**——地震、车祸、大火、落水时，当下忆念头顶上师阿弥陀佛，即可解脱。

### 2.8 临终面见佛是否为魔所变

蕅益大师《阿弥陀经要解》：**「修心之人不作佛观而佛忽现，非本所期，故名魔事。念佛见佛，已是相应，况临终非致魔时，何须疑虑？」**即：平时念佛者临终见佛，是愿与佛相应，不是魔变。临终时不是魔来扰之时机。

## 三、己二-庚一：对轮回之痛苦生起厌离心

### 3.1 根本颂

> **恶趣苦难忍，人天乐无常，**
> **愿生畏彼心。无始至今生，**
> **漫长漂轮回，愿生厌离心。**

### 3.2 双重考虑——恶趣苦 + 人天乐无常

不只是「恶趣可怕」这么简单。**人天善趣也必须生厌离**，因为：

- **人天安乐皆摄于行苦**——三苦（苦苦 / 坏苦 / 行苦）中，人天表面快乐实为行苦
- **《亲友书》**：「帝释堪称世间供，以业感召亦堕地，纵然曾为转轮王，于轮回中复成仆」——帝释天业尽堕地狱；转轮王善业尽沦为仆人
- **「春天阳焰」喻**——所有表面上的幸福快乐都像春天阳焰一样瞬间即逝
- **「乞丐果腹」喻**——人天安乐只是乞丐吃饱一顿的快乐，下午肚子又饿，无任何恒常性

### 3.3 「如杂毒食」喻

**「人天之安乐，犹如杂毒食，愿毫无贪求」**（庚二首颂接续），这是《极乐愿文》最著名的意象之一：

世间幸福是毒与食混杂——表面上好吃（乐），实际吃下去会中毒（苦）。苦乐混杂不可分离，应整个舍弃。

### 3.4 六道痛苦摄义（《念住经》总纲）

**「地狱有情受狱火，饿鬼感受饥饿苦，旁生感受互食苦，人间感受短命苦，非天感受争斗苦，天境感受放逸苦。轮回犹如针之尖，何时亦无有安乐。」**

### 3.5 厌离心生起次第的四个教证

1. **《中观四百论》**：「若凡夫亦知，一切生死苦，则于彼刹那，身心同毁灭。」——凡夫若真知轮回苦，身心将承受不住而同时毁灭
2. **《亲友书》**：「即便见闻地狱图，忆念读诵或造形，亦能生起怖畏心，何况真受异熟果？」——仅看见地狱图都能生怖，何况亲受
3. **《正法念处经》**：「汝闻地狱声，已如是怖畏，何况地狱烧，如烧干薪草？」
4. **圣天论师**：「于此大苦海，毕竟无边际，愚夫沉此中，云何不生畏？」

### 3.6 阿难二外甥公案（小阿罗汉）

阿难的两个外甥因帕吉波王屠杀释迦族而父母双亡，阿难收留他们但不精进修行。目犍连以神通幻化地狱境界——见到一口沸锅无众生，狱卒曰：「阿难的两个外甥懈怠不修，死后到此受苦。」两人惊恐万分：
- 上午想起地狱，午饭难以下咽
- 下午回忆地狱，食物全部呕出

目犍连随后又幻化天宫境，二人精进修行最终证阿罗汉。后佛解释——他们在迦叶佛时代为老比丘，发愿童贞入道、早证圣果，故七岁得果。

**启示**：真正修行人一想到轮回痛苦，就恐怖得对任何事情没有兴趣。花色比丘忆前世地狱苦，毛孔出血染法衣成花色。

### 3.7 福增比丘摩羯鱼公案（《贤愚经》）

一百多岁的福增老比丘出家后被年轻比丘欺负，想投水自尽。目犍连救起后带他至海边，见七百由旬高骨架山，山影令海面漆黑。尊者告知：

> 「远古你是法胜国王（有些经中叫法增），因忙于下棋随口说『依法处治』，大臣处死一人。醒悟后你舍王位入山，但死前未清净杀业，以此业堕大海为摩羯鱼，身长七百由旬。一入睡百年不醒，醒时饥饿张口，海水奔流入口；身上大臣转为小鲸、寄生虫食其肉，它疼痛于水晶山上蹭身杀死小生命，海水百由旬血红。千百年受苦。尸被海浪冲到岸边，日晒雨淋肉尽，剩骨即此山。」

福增听后对轮回极生厌离，从此精进修行。索达吉堪布：**「这个公案不是虚构的动画片，也不是哄小孩的故事，而是历史上真实发生过的事。」**

### 3.8 热扎忏文（宿命通者的悲哀）

热扎回忆宿世作忏文：

> 「骨肉若集等须弥，脓血若集如大海，
> 宿业若积说不尽，辗转生死三界中，
> 仍造无义唐捐事。」

### 3.9 清代「百不管」老太太的实修

《净土圣贤录》：一心念佛，最终往生。法师教诫：「念佛欲求一心，应拔去爱根，放下一切。」她依教奉行，遇事皆以「百不管」推掉，人皆以此名呼之。莲池大师亦作《七笔勾》，将父母、夫妻、儿孙、功名、田园、文章、宴游七者一笔勾销。民国赖祥麟念佛而逝——脚痛不能走路，却说「不是身去，是心去」。截流大师（净土十祖）警语：**「今时净业人，终日念佛，忏罪发愿，而西方尚遥，往生弗保者。无他，爱桩未拔，情缆犹牢故也。」**

### 3.10 告别轮回的豪迈语

索达吉堪布：「我们应当潇洒地说：**『轮回，你安闲而住吧，从今以后我不再陪你，而要前往极乐世界。』** 必须向轮回挥手告别，否则，过去如水滴般的罪恶将来也要感受如海洋般的痛苦。」

## 四、己二-庚二：断除贪执四支

### 4.1 四支结构

```
庚二 断除贪执
  ├─ 不贪人天安乐（「如杂毒食」颂）
  ├─ 不贪亲友财食（「食财亲友朋」颂）
  ├─ 不贪自身（身念处观）
  └─ 不贪故乡属地宅（「如梦境宅」颂）
```

### 4.2 颂词

> **设使人转人，受生老病死，**
> **浊世违缘多，人天之安乐，**
> **犹如杂毒食，愿毫无贪求。**
> **食财亲友朋，无常如梦幻，**
> **愿毫无贪恋。**
> **故乡属地宅，犹如梦境宅，**
> **愿知不成实。**

### 4.3 不贪人天安乐——浊世违缘分析

末法时代莲师授记分析五浊（索达吉堪布以当代事件对应）：

| 浊名 | 内涵 | 当代征兆 |
|---|---|---|
| 劫浊 | 时代整体败坏 | 「空中出现噩星时，偏执宗派起争论」（各宗互诤） |
| 烦恼浊 | 五毒粗大 | 贪嗔痴日增 |
| 众生浊 | 刚强难化 | 根机浅薄，难以调化 |
| 命浊 | 寿短不定 | 人寿短、横死多 |
| 见浊 | 严重邪见 | 「众生乱行罪业时，出现地震降雷雨」 |

阿底峡尊者警示末法八诱：**「魔类擅长谄诳术，以财诱惑大贪者，以假法骗修行人，以自诩诱净戒者，以我慢欺有智者；以歌舞诱散乱者，以韶华诱诸多人，以妙饰诱众多人，以恶友敌骗多人。」**

推论：浊世获人身反不如贤劫时转旁生——因造恶业太易，行善太难。

### 4.4 不贪亲友——缘起教证与公案

**教证**：
- 《宝积经》：「如鸟为求食，不知避网罗，贪爱于女人，被害亦如是」
- 《律藏》：**「出生独自生，死亦独自死，独自受痛苦，轮回无友伴。」**
- 《无量寿经》：「人在世间，爱欲之中，独生独死，独去独来」
- 《生经》：「在三界中，独来独去，无一随者，祸福追身，如影随形」
- 《大圆满前行》：「纵然是数以千计僧人的上师也不能带走一僧一徒，即便是数以万计部落的首领也不能带走一奴一仆」
- 佛陀经语：「若未见凡愚，将恒时安乐」——舍弃凡愚恶友如舍尸体、怨敌

**公案**：
- 一人与他人妻行邪淫被杀，因贪恋女人转生女人胎中寄生虫
- 胜光王被恶生（琉璃）太子篡位后流落至王舍城，吃萝卜喝冷水霍乱而死，口中灌灰尘呃逆而终——「世间第五大王」临终如此凄惨

**教诫**：「如果贪恋亲友们，则死后将在他们中转成魔鬼等，因此万万不要贪执解脱的怨敌（指亲友）。」——亲友被称「解脱的怨敌」。

### 4.5 不贪饮食——贪食堕旁生公案串

索达吉堪布罗列多则公案（《现果随录》《大智度论》《贤愚经》等）：

| 贪执物 | 结果 |
|---|---|
| 雅多老僧的牛腿 | 死后成牛腿被狗追赶；华智仁波切教以烧焦烟作佛事解冤 |
| 小孩的烧饼 | 神识往酥油桶跑；上师烧焦烟回向才超度 |
| 印度沙弥的奶酪（《大智度论》） | 死后转为奶酪中的小虫 |
| 某寺监院藏在地藏华冠中的 80 两银（《现果随录》） | 死后每晚登佛座摸华冠 |
| 贪钵盂的比丘 | 转毒蛇，嗔恼烧森林，一日三次大火 |
| 贪松耳石的比丘 | 转青蛙紧抱松耳石，需四脚朝天倒开水才松开 |
| 藏地念观音心咒老人 | 心爱松耳石藏在帐篷角落，死后成松耳石上的小虫 |
| 哲蚌寺老僧藏银币 | 转蜘蛛在银上发「瑟瑟」声；屠夫（隐藏成就者）一口吃下弹指超度 |
| 贪法衣比丘 | 转蛇 |
| 萨巴施主妻贪自美色 | 船毁人亡后转毒蛇在自己尸体口鼻钻来钻去 |
| 贪石簧牧童 | 死后床下转石簧蛇 |
| 贪打火器老人 | 转乌龟（形似打火器），铁两端分别刺头尾 |
| 舍卫城贫穷施主贪牛 | 死后成牛背伤口寄生虫，日受七次生死 |

**「七金瓶毒蛇」公案（《贤愚经》）**：鹿野苑富人做生意赚七金瓶，吝啬埋地下，死后转毒蛇守金瓶，数万年中辗转转蛇，最后城市空无一人。终生厌离心后请路人将金瓶供养僧众，闻法往生三十三天。此蛇即舍利子前世，路人即世尊前世——**舍利子因一念贪金多次堕恶趣。**

**「金瓶毒蛇」的另一公案**（《毗奈耶经》）：老人拾金瓶暴富被怀疑偷窃，国王没收家产判死刑。老人狱中悟：「世尊说金子是散发毒气的毒蛇，阿难说金子是凶恶的毒蛇，他们说得确实对呀！」国王听后释放并归还家产，老人于佛前闻法得初果。**「愚者为财毁，不寻求来世，毁愿以贪欲，亦可毁自他。」**（《因缘品》）

### 4.6 不贪自身——身念处观

身体被「执为我」是最深的贪根。对治：

- **八万四千虫户**——身内寄生生命无数
- **脓血不净**——九种不净物（发毛爪齿、涕唾垢污、脓血尿便）
- **《入行论》身念处**——「涂着美色的不净物或用绫罗绸缎包裹的荆棘」

萨巴施主妻子贪自美色转毒蛇公案，正是此义的直观警示——每天化妆三四小时的当代女性应省察。

### 4.7 不贪故乡属地宅——梦境譬喻

**颂：「故乡属地宅，犹如梦境宅，愿知不成实。」**

**吝啬老妇转狗公案**——吝啬贪家，死后转自家母狗，日夜围家保护防盗贼。**「如今的大多数聪明的家犬肯定是自己父母家人等吝啬者的转世。」**

**《中阿含经》都提转狗公案**——舍卫国富人都提吝啬，死后转自家白狗。佛对白狗说：「你想一想自己当狗的原因。」白狗闻言卧地闷闷不乐。佛告诉都提之子：「这只狗是你父亲，他因吝啬堕为旁生。家中地下埋有宝藏，只有它知。」儿子回去问白狗，狗点头挖出珍宝。

**省庵大师公案**（净土十一祖）：南京老妇偷丈夫四两银子，死后转儿子家驴子。贪父吝啬变猪。

**1996 年法王如意宝流泪告诫藏民**：「你们最好不要杀自家门前的牦牛，也不要杀附近山里的动物。按照佛经的教证推断，这些旁生肯定是你们死去的亲属，他们因为没有往生净土的因缘，并且临死时贪执自己的家乡和亲人，结果成为你们周围的旁生。」

### 4.8 断贪执的两种方法（讲记核心窍诀）

索达吉堪布用《解义慧剑》框架归纳断贪的两条路径：

**一、胜义方法**——中观破四边：用《中观四百论》《醒梦辩论歌》等理证分析，观察所贪之境连芝麻许实质都没有。圣天论师：**「薄福于此法，都不生疑惑，若谁略生疑，亦能坏三有。」**——对空性哪怕略生合理怀疑，也能坏三有。

**二、世俗方法**——如梦如幻：认识到轮回万法显而无自性，与梦境、幻师变现之象马无别。

龙树菩萨的画师罗刹譬喻：**「如同某位绘画师，描绘恐怖罗刹像，彼像令自生畏惧，轮回患人亦复然。」**画师知罗刹像乃自画，何需害怕；众生轮回苦乃自心所现，何需耽著。

### 4.9 中阴神识清明九倍——最殊胜的破瓦法

这是索达吉堪布反复强调的关键**比例**：

- **梦中神识**比白天神识清明一定倍数
- **临终神识**又比梦中更清明
- **中阴神识**比临终神识**清明九倍**

后果：**活着贪执什么，中阴时会以九倍程度贪执**，瞬间忆念立即投生为低劣之身。

故：**「从即日起就要放下一切，专心意念观想往生极乐世界，在有生之年一直念念不忘，这一点相当重要，这也是往生（破瓦）法。」**

对没有修行境界的凡夫，这种**「念念不忘往生净土」本身就是最好的破瓦法**——不需要「啪的」一吼的复杂观修。**「除此之外再没有能断绝中阴的其他教言了。」**
