# 顶礼支——积资净障第一支

## 一、总纲：七支供对治七种烦恼

### 1.1 积资净障在往生四因中的位置

往生极乐四因依次为：
1. **明观福田** — 前一支已讲
2. **积资净障** — 本支起至"回向支"
3. **发菩提心**
4. **发愿回向**

索达吉堪布开示：明观福田是积资净障的所依；积资净障则是发起愿行菩提心、成就清净愿的前提。《广大游舞经》云："具有福德者，亦成诸所愿。"欲得广大利益，必先有广大福德。

> 有些人在干坏事时一帆风顺、毫无违缘，干好事时却违缘重重，这就是福报浅薄。若具有福报，就会像福力王一样做任何善事都没问题。——L18

### 1.2 七支供的结构——对治七种烦恼

一切积资法门可摄于七支之中。《极乐愿文大疏》丁二科判：

| 支分 | 所对治烦恼 |
| --- | --- |
| 一、顶礼支 | 我慢 |
| 二、供养支 | 吝啬（贪心） |
| 三、忏悔支 | 愚痴 |
| 四、随喜支 | 嫉妒 |
| 五、请转法轮支 | 舍法 |
| 六、祈请不入涅槃支 | 邪见 |
| 七、回向支 | 怀疑 |

此七支并非外在仪轨排列，而是依**所对治之烦恼**而建立。顶礼置于首位，即因我慢是一切积资的第一障碍——傲慢的山岗上存不住功德水。

## 二、顶礼支略说——为何以顶礼摧伏我慢

### 2.1 根本颂

> **敬礼远近无别故，我以三门敬顶礼。**

寂天菩萨云："因慢生傲者，将赴恶趣道。"故为摧毁我慢而略说顶礼。

### 2.2 我慢的过患

具我慢者犹如高山及铁球一般，对他人无有礼拜之心。其果报极为惨重：

- 不仅不能往生极乐世界，反而走向恶趣
- 即使获得人身，也会如堪布根霍所说："愚笨如老猪、丑陋如青蛙、体质虚弱多病痛、受人轻蔑如老狗"（傲慢的等流果）
- 尤以对僧众起傲慢，果报最为惨烈

**公案**（L18）：佛陀时代除粪人尼提，因前世作迦叶佛教法中某寺住持时傲慢役使已证圣果之僧众，以金银盆处理大小便令其清洗，致堕三恶趣无量痛苦，转生为人后五百世作除粪人；以其往昔出家持戒善根，方于佛陀教法下值遇得度，证阿罗汉果。

### 2.3 顶礼对治傲慢——《六祖坛经》"礼本折慢幢"

> 汉传佛教也认为顶礼能摧毁傲慢，《六祖坛经》云："礼本折慢幢。"意为顶礼能折服傲慢的魔幢。——L18

谁相续中有傲慢，他就不可能给别人顶礼，不要说五体投地，连弯腰问讯都不愿意。顶礼时需内心同时发愿："生生世世不要变成傲慢者。"

### 2.4 "远近无别"——信心是关键

阿弥陀佛虽住极乐世界（千百万亿那由他由旬之外），若具信心，则如同就在面前；若无信心，佛陀住在眼前亦如远方。

- **净水明镜喻**：净水、明镜中可显远处日影；浊水、垢镜中连近处色相也不能显现。
- **千江有水千江月喻**（《嘉泰普灯录》）：天月与江河远近与月影能否显现无关；佛陀加持能否融入众生心，关键在有无信心。
- **善星比丘喻**：天天与如来住在一起，但因无信心，终不得利益。

**公案**（L18）：无著菩萨鸡足山苦修十二年不见弥勒，下山途中对残废母狗生起大悲，业障清净后见弥勒菩萨。弥勒云："我与你从未分离过，只是因为你罪障深重所以见不到我。"——将弥勒扛在肩上入城，仅一清净业障老妇见到"一具腐烂狗尸"，他人全然不见。

### 2.5 三门顶礼的要点

**身恭敬**：身体不过于前俯后仰，眼睛不东张西望，顶礼速度不过快过慢；以端正、寂静、调柔方式作顶礼。

**语恭敬**：断除绮语，念诵顶礼偈或佛号。

**意恭敬**：断除贪嗔等一切不善分别念，对佛陀功德生起欢喜心、敬信心。

若顶礼时不具足三门恭敬（内无敬信、口无念诵、身不庄重），华智仁波切说"没有任何实义"；《极乐愿文大疏》作者喇拉曲智仁波切则更进一步说——"**倘若以不恭敬的心态顶礼，则是耗尽福德之因**"。

### 2.6 顶礼的世间利益

堪布亦劝领导人多顶礼——磕长头是最佳全身运动，效果远胜健身操。藏地许多病人（甚至绝症）通过顶礼恢复健康，此亦上师三宝加持所致。

## 三、广说·思维名号不同而顶礼

《极乐愿文》分四颂依次从四种名号顶礼：**法身阿弥陀、遍知无量光、导师无量光、怙主无量寿**。

### 3.1 名号差别总表

| 名号 | 侧重功德 | 颂词要义 |
| --- | --- | --- |
| **法身阿弥陀** | 三身之本体 | 右手化观音、左手化度母、心间化莲师 |
| **遍知无量光** | 智悲遍知 | 昼夜六时慈眸恒视，知众生一切心念言语 |
| **导师无量光** | 临终中阴接引 | 除舍法无间罪外，诚信发愿者皆得接引 |
| **怙主无量寿** | 寿命与遣横死 | 祈祷一次等同一次长寿灌顶 |

喇拉曲智仁波切结偈云：单单因报身化身差别、光芒无边或寿量无边而有不同名称，**千万不要认为有两种本体**。

### 3.2 法身阿弥陀——三身总集（L19）

根本颂：

> **法身无量光部主，右手放光化观音，复化百俱胝观音；左手放光化度母，复化百俱胝度母；心间放光化莲师，复化百俱胝莲师，顶礼法身阿弥陀。**

#### 3.2.1 三身关系

法身是报身与化身的来源。阿弥陀佛智慧法身乃一切佛化身的根本，其色身为诸部之主尊，尤以**语莲花部部主**而摄五方佛中西方法身。

#### 3.2.2 三大放光化身

| 放光处 | 光色 | 所化 | 所化对境 |
| --- | --- | --- | --- |
| 右手 | 白光 | 观世音菩萨 → 百俱胝观音 | 调伏男性士夫 |
| 左手 | 绿光 | 至尊度母 → 百俱胝度母 | 调化女性 |
| 心间 | 五彩光 | 邬金莲花生大士 → 百俱胝莲师 | 降伏寂静方式无法调伏的凶神恶煞、边地藏土弘扬显密圆融佛法 |

"百俱胝"（百千万）并非实数限量，仅标举能化之多；诸佛菩萨行境不可思议，可化现善知识、国王、甚至旁生。

**观音三十二应**（《普门品》）："十方诸国土，无刹不现身。"——观音化身不仅遍及世间高层，连人们讨厌的坏人、乞丐乃至精神思想颠倒者中都可能有化现，故密宗强调观清净心。

#### 3.2.3 莲师是阿弥陀佛化身——净密不二

教证（伏藏品）："**昔日极乐阿弥陀，普陀怙主观世音，达纳郭夏莲花生，纵然示现三化身，实则无二也无别。**"

《涅槃经》（藏文版）："我已灭度后，汝等莫哀伤，无垢彩湖中，较我胜士夫。"——佛预言莲师出世。

此处堪布着重开示：**净土宗与密宗并不相违**。若排斥密宗——只承认右手化观音，半承认半不承认左手化度母（因汉地有度母经典不敢完全不承认，因藏地盛行度母法又不太愿意承认），对心间化莲师彻底不承认——便造严重**谤法罪**。

**缘起公案**（L19）：1987年法王如意宝于五台山云："今后汉地信仰藏传密教的人会越来越多，汉地最主要的神山是五台山，为了创造缘起，我们要在五台山的每一座寺院塑一尊莲花生大士像。"不开许之寺则塑小像藏于其他佛像之后。此后汉地信仰藏密之人激增。

#### 3.2.4 "一摄一切"的修法窍诀

法王如意宝：念修阿弥陀佛，当观想释迦牟尼佛、文殊菩萨等一切佛菩萨都融入阿弥陀佛。

《华严经》："一切诸佛身，唯是一法身，一心一智慧，力无畏亦然。"

喇嘛云："印度人修一本尊得成就，西藏人修百本尊皆无成。"——藏人疑心分别念多，难以成就；若深信"一尊阿弥陀佛是一切皈依境总集"，修成一佛，未修皆成。

**百川汇于一处喻**：一百河谷水一一饮用极难，若喝百川汇流之水，则相当于饮用了所有的水。

### 3.3 遍知无量光——昼夜六时慈眸恒视（L20）

根本颂：

> **佛于昼夜六时中，慈眸恒视诸有情，诸众心中所生起，任何分别皆明知，诸众口中所言语，永无混杂一一闻，顶礼遍知无量光。**

#### 3.3.1 佛陀三遍知

| 神通 | 所照 |
| --- | --- |
| 天眼通 | 众生所作所为 |
| 他心通 | 众生所思所想（信、悲、菩提心及贪、嗔、嫉妒、无记一切分别念） |
| 天耳通 | 众生所言所语（无论公开私下，无论念经诽谤，乃至细微私语） |

#### 3.3.2 教证依据

**《佛说阿弥陀三耶三佛萨楼佛檀过度人道经》第十七愿**：
> "使某作佛时，令我洞视彻听，飞行十倍，胜于诸佛，得是愿乃作佛，不得是愿终不作佛。"

此"遍知"功德乃阿弥陀佛因地大愿所成。

#### 3.3.3 "昼夜六时"的密意

非指除六时外其他时间不关注，而是日日夜夜、时时刻刻注视之义。诸佛菩萨唯一的事业就是利益众生，轮回未空前从不间断。

**譬喻**：一个房间所有巨细物体可互不混杂同时现于一个镜子中；同样，佛陀可同时互不混杂了知三世一切所知。

#### 3.3.4 行者对治

> 阿弥陀佛及其眷属昼夜六时都在以慈悲的智慧眼观照着我，所以我一定不能乱来，这样就能具足正知正念，从而也能行持善法、远离恶业。——L18/L20

即便单手胸前作礼、生起刹那信心、作一次祈祷，都应观想阿弥陀佛已见、已知、已闻，对此无有邪见、深信不疑至关重要。

### 3.4 导师无量光——接引往生之理（L20-22）

根本颂：

> **除造舍法无间罪，诸诚信您发愿者，如愿往生极乐刹，佛临中阴引彼刹，顶礼导师无量光。**

#### 3.4.1 两种往生障碍

阿弥陀佛大愿接引一切众生，暂时唯有**两种人**不得往生：
1. **造五无间罪者**
2. **造舍法罪者**

**《无量寿经·四十八愿》**：
> "设我得佛，十方众生，至心信乐，欲生我国，乃至十念，若不生者，不取正觉，**唯除五逆，诽谤正法**。"

最终此二人亦可往生，但恶业深重，暂时无有机会。为此，法王如意宝要求："要往生极乐世界的人，一定要先念四十万遍金刚萨埵心咒"——忏除此类重罪。

#### 3.4.2 《无量寿经》与《观经》看似矛盾的会通

《观经》下品下生讲造五逆十恶者临终十念可得往生——与《无量寿经》"唯除五逆谤法"是否矛盾？

**答**：并不矛盾。
- 《无量寿经》义：造五无间、谤法罪后**未忏悔清净**则不可往生
- 《观经》义：若**罪业忏悔清净**则可往生

临终地狱相现前，念佛之心极为猛切可灭滔天大罪；且此时无有不信，故不言谤法。密宗同理：造此二罪者若能趋入坛城接受灌顶、以密法加持清净罪障，亦有解脱机会。

#### 3.4.3 舍法罪的范围——极易造作

堪布着重开示：舍法罪远非"烧毁佛经、扔经入河、诽谤前后世"此类直接行为，**平时极易造下**：

**舍教法**之相：
- 劝正在传法的法师"停几天课休息休息"
- 劝听法者"别去听法了，去朝山吧"
- 让人中断**一句一义**的传承（哪怕一天课）
- 上课时与邻座说话影响他人听法

**舍证法**之相：
- 劝他人不要出家（"末法时代哪有清净出家人"）
- 讥讽持少分戒（不饮酒、过午不食、八关斋戒）者为"假装"
- 劝修行人"不用修空性菩提心，念佛就可以了"
- 劝密宗行者"不要学密宗，只要念佛就可以"

**自宗他宗之诤**：
- 月称论师云："贪执己见嗔他见，此等皆为邪分别。"
- 赞自宗、诋他宗即谤法
- 非智者却擅评他人"这是智者、那是愚者"——诽谤某人而间接诋毁正法

**擅改佛经亦为极重谤法罪**——如将《地藏经》开头天龙鬼神段用胶带纸贴上、将"如是我闻"改为"我闻如是"、将"百千万劫难遭遇"改为"百千万劫难相遇"。迦叶尊者叮嘱阿难："佛所说法一言一字，汝勿使有缺漏。"

#### 3.4.4 谤法罪的可怕严重性

- 《般若八千颂》："何人若造五无间，不及相似谤佛法。"——谤法**超过五无间罪**。
- 《宝性论》："何人心嗔法，彼焉有解脱？"
- 印光大师："以五逆罪大，谤法不信，此种罪障，岂悠悠泛泛之修持所能灭乎？"
- 《大乘方广总持经》："佛告文殊师利，我于往昔七年之中，昼夜六时忏悔身口及与意业所作重罪，从是已后乃得清净，经十劫已获得法忍。"

**公案·帕翁卡巴格西**（L21）：格鲁派一格西前世业力所感，经常诽谤宁玛派大圆满法，砸毁拉萨附近莲花生大士等密宗佛像，后与其眷属活活陷入地狱，遗迹至今尚存（《协庆加查佛教史》记载）。

**弘一大师的忏悔**（L21）：
> "大乘各宗中，此宗（密宗）之教法最为高深，修持最为真切。常人未尝穷研，辄轻肆毁谤，至堪痛叹。余于十数年前，唯阅《密宗仪轨》，亦尝轻致疑议，以后阅《大日经疏》，乃知密宗教义之高深，因痛自忏悔。"

**虚云老和尚**（L21）最终定论："密宗的确是释迦牟尼佛的教法。"

#### 3.4.5 五无间罪及相似无间罪

五无间罪：杀父、杀母、杀罗汉、破和合僧、恶心出佛身血。

后三者如今不能直接犯，但可造**相似罪**：
- 破和合僧之相似：上师与上师间、上师与弟子间、弟子与弟子间、僧团与僧团间挑拨制造矛盾
- 恶心出佛身血之相似：毁坏佛像佛经、殴打上师道友、以石头砸上师门窗
- 杀父杀母之相似：即使对父母不孝顺不恭敬

#### 3.4.6 临终与中阴往生

**临终往生的三种情形**：

**上等修行人**：临终四大隐没次第（地→水→火→风→识）现前时忆念阿弥陀佛而自在往生。
- 四大隐没次第感受：地入水（身重如被山压）、水入火（眼泪口水鼻涕，口干舌燥）、火入风（体温从边缘内收）、风入识（呼吸停止但意识未离）
- **明慧法师公案**（L22）：临终对身边道友说"现在是地大融入水大……"

**罪孽深重者**：由罪业自相现阎罗王、狱卒、所杀众生景象，彼时虽无力祈祷，**但仅能忆念也可往生**。律藏云："纵然大海有，离开波涛时，佛陀却恒时，不离所化众。"
- **张善和屠夫公案**（L22）：唐代屠夫临终见群牛索命，僧人令其点香念佛，不到十句便云"阿弥陀佛来接引我了"，随即命终。

**中阴往生**：四恐怖声（四大重构之音）、三险地（贪嗔痴自相——红彤彤十万红日、白茫茫十万月光、黑乎乎廿九夜相）现前，文武百尊显现时忆念阿弥陀佛即刻往生。

**嘎巴公案**（L22）：果青寺附近居士嘎巴生前精进但行为并不很好，临终日令家人："给我洗澡穿上出家人法衣，阿弥陀佛肯定会来接我。"家人三次不理，最后披件法衣便高兴圆寂。

**隐峰禅师公案**（L22）：倒立而圆寂、衣不下垂，其为尼之妹呵斥后以手一推方倒下，火化现众多舍利。

#### 3.4.7 四因对照《弥陀经》大愿

### 3.5 怙主无量寿——寿命与遣横死（L23）

根本颂：

> **您之寿量无数劫，不趣涅槃今住世，一心恭敬祈祷您，除非异熟业果外，寿尽亦可享百岁，遣除一切诸横死，顶礼怙主无量寿。**

#### 3.5.1 阿弥陀佛 = 长寿佛

阿弥陀佛与长寿佛只是报身与化身的差别，本体无二。念阿弥陀佛名号与修长寿法**有同等功德**。

#### 3.5.2 心念要求

> 无论是苦是乐、是上升还是下堕，我无有其余的皈依处、指望处。

此为祈祷时应发之心。邬金莲师："**具诚信者得加持，远离疑惑成所愿。**"半信半疑犹如两尖针，一事无成。

#### 3.5.3 三尽异熟果不可延

唯有三种情形任何长寿法门（包括祈祷阿弥陀佛及药师佛亲降）皆无法延寿：

1. **往昔杀生过多之异熟果而寿尽**
2. **因往昔所积福德耗尽而寿尽**
3. **引业如尽力之箭自然落地般穷尽而寿命完结**

此三之外暂时违缘、横死皆可遣除。正所谓"**药师佛虽亲降临，无法延续寿尽命**"。

#### 3.5.4 可遣的十八种横死

《药师经》九种横死 + 《佛说九横经》《佛医经》所说九种，共十八种：
病不得治、横为王法诛戮、横为非人夺精气、横为火焚、横为水溺、横为恶兽所啖、横堕山崖、横为毒药厌祷咒诅之所中害、饥渴所困而横死、天降霹雳、兵器所害、悬梁自尽、鬼神窃寿、诅咒损毁灵魂等。

#### 3.5.5 业寿四种情形

| 情形 | 相当例证 |
| --- | --- |
| 业尽寿未尽 | 上半生乐、下半生苦 |
| 寿尽业未尽 | 人死转人、地狱转地狱 |
| 业尽寿亦尽 | 人死堕地狱 |
| 业寿皆未尽 | 终生享乐 |

#### 3.5.6 "祈祷一次等同于一次长寿灌顶"

如今人寿百岁末期、违缘重重恶世，无需到远方求灌顶、无需观察灌顶上师，只需坐自己卧室一心一意祈祷阿弥陀佛即可。《中观四百论》："虽见身如怨，然应保护身，**具戒久存活，能作大福德**。"

**吉美赤列沃塞公案**（L23）：无畏王之一，智悲光尊者令其至康区寻瑜伽士久利津（莲师化身）遣寿障。久利津以尿壶液体、发霉青稞粉唾沫丸赐之，并以拐杖敲头三下，最后延寿至二百岁。智悲光尊者云："对我来说，来自他的侮辱远比他人的赞美和加持还要好。"

#### 3.5.7 念佛·长寿·往生不应分老少

堪布破除错误观念——认为"念阿弥陀佛是老年人的修法、求长寿法是年轻人的事"是错误的：
- 年轻人若求阿弥陀佛灌顶被嘲笑"是老年人事，难道你马上要死"——无理
- 老年人若求长寿灌顶被嘲笑"老了还求长寿"——无理

**喇拉曲智仁波切的抉择**：求破瓦法即获《极乐愿文》传承及长寿灌顶；求《极乐愿文》读诵传承即获长寿灌顶与破瓦法传承。三法一体，男女老幼皆应修。
