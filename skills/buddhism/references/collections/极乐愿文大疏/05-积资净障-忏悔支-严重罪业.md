# 忏悔支——身语意严重之恶业

## 单元定位与根本颂

本单元科判: 辛二（忏悔严重身语意之恶业）分五: 一、忏悔五无间罪; 二、忏悔近无间罪; 三、忏悔舍法罪; 四、忏悔诽谤菩萨罪; 五、忏悔恶见罪。共11堂课 (L65-75)。

五支根本颂:

> 杀师父母阿罗汉，恶心损害佛身体，发露忏悔无间罪。
> 杀害比丘与沙弥，污尼毁像塔寺等，发露忏悔近无间。
> 三宝殿经所依等，以彼作证违誓等，发露忏悔舍法罪。
> 诽谤诸菩萨之罪，较杀三界有情重，发露忏悔无义罪。
> 闻善功德恶过患，地狱痛苦寿量等，认为不实仅说法，此罪重于五无间，发露忏悔无解罪。

## 严重罪 vs 一般十不善业 — 为何单独成一类

前一单元 (04-身语意十不善) 处理 "一般" 身语意恶业。此 辛二 单独 成类 的 原因, 在于 罪业的 加重机制:

- **对境殊胜**: 父母 / 上师 / 阿罗汉 / 佛 / 僧 / 菩萨 — 属 "严厉的对境" (藏传 "重对境" 概念). 《杂宝藏经》: "于父母所, 少作供养, 获福无量, 少作不顺, 获罪无量。"
- **意乐极恶**: 恶心出佛身血, 以嗔心毁坏三宝所依等
- **所损功德大**: 损害三宝 / 菩萨 = 断绝众生 成佛之因 / 解脱之源
- **业报不经中阴直堕**: 五无间罪 "舍此身已, 次身即受" (《大乘义章》"趣果无间")

乔美仁波切把这五项单独列为 "严重恶业", 因其 **直接障碍往生极乐**。根本颂明言: "暂时有两种人不能往生极乐世界, 即造舍法罪及五无间罪者" (《佛说大阿弥陀经》)。虽然最终佛陀慈悲仍可摄受, 但因业重, 暂时无成就所愿之机会。这是整个忏悔支的关键 apology — 必须对这五类以强烈追悔心发露忏悔, 否则 即使每天念"南无阿弥陀佛", 也无法与极乐相应。

## 壬一、忏悔五无间罪 (L65-67)

### 五项定义与判定标准

根本颂"杀师父母阿罗汉, 恶心损害佛身体" 五项:

1. **杀害上师阿阇黎** (此处按《祖法金藏论》列入无间罪)
2. **杀父**
3. **杀母**
4. **杀阿罗汉**
5. **恶心出佛身血**

判定要点:
- 共通说法 (《俱舍论》等): 杀父、杀母、杀阿罗汉、恶心出佛身血、破和合僧
- 此处按《祖法金藏论》: 将 "破和合僧" 替换为 "杀害上师阿阇黎等具法眼的诸智者"
- **三种造罪方式均成无间**: 自己造 / 唆使他人造 / 随喜他人造 — 堪布特别警告"随喜心"

> "自己和某人关系不好, 当他造了这些恶业后, 自己也许会产生随喜心: 太好了, 他造了这么严重的罪业, 一定会遭受报应的。这样的随喜心非常可怕" (L65)

### "无间"的四层解释 (《大乘义章》)

1. **趣果无间**: 舍此身已, 次身即受
2. **受苦无间**: 造五逆罪者, 一劫之中于阿鼻地狱受苦, 日日夜夜无有间断
3. **寿命无间**: 于地狱中, 一劫之中寿命无有间断
4. **身形无间**: 阿鼻地狱纵广八万四千由旬, 一人入中身亦遍满, 一切人入身亦遍满, 不相障碍

这是五无间罪 "立即堕无间地狱、不经中阴" 的教理依据。

### 杀父母 — 孝道之严厉对境

**为何父母是严厉对境**: "自己的善妙身体也是以他们的因缘而形成的"。对出家人而言, "父母等同于堪布、阿阇黎, 如果未经父母允许则不可以出家, 如果未经父母开许就出家, 传戒的上师会犯一个轻罪" (L65)。

**关键公案: 小驼背比丘 (L65)** — 阿罗汉亦受业报。前世吝啬之徒, 母亲供僧后, 他将母亲关在空屋, 饿七日, 给一碗灰汤。连续堕恶趣多世, 饥饿而死; 获阿罗汉果后仍七日化缘不得, 最后喝一口灰汤饿死。目犍连、舍利子皆爱莫能助。→ 对父母的恶业 即使 证阿罗汉 仍需 偿还, 因果之严厉如此。

**关键公案: 匝哦之女 (释迦佛因地) (L66)** — 商主之子, 不听母劝, 远行大海取宝, 临行用脚狠踢母亲的头。以供养母之功德, 享天界数千年安乐; 以踢母头之业力, 受燃火铁轮旋于头上、脑浆飞溅之剧苦。最终以 "愿众生痛苦成熟在我身上, 由我代受" 的自他相换菩提心, 立即从痛苦中解脱。

**关键公案: 目犍连圆寂 (L66)** — 神通第一的目犍连被裸体外道徒殴打致死, 身体如苇草。因前世媳妇挑拨, 对父母生起"真该将你们的身体打得像苇草一样"的恶心, 以此恶心 五百世被人打死, 阿罗汉果上仍感余业。

### 不孝 / 殴打父母 = 准无间 (Teacher's treatment)

堪布在 L66 中明确扩展五无间罪的范围到 **不孝顺、不恭敬、殴打、侮辱、指使、跨越** 父母:

> "有些人不一定杀害父母, 但是他们不孝顺、不恭敬父母, 甚至侮辱、虐待父母, 这种恶行也是往生极乐世界的障碍。这是由于父母是严厉的对境, 对他们有不如法行为过患极大。" (L66)

具体范围:
- **殴打父母**: 匝哦之女公案; 和杀父母一样有严重罪过
- **仅生起殴打心**: 目犍连前世即仅因一念恶心而五百世被打死
- **跨越父母身体**: 商主仁慈供施因从母亲头上跨过, 即生堕孤独地狱, 头上转兵器铁轮
- **指使父母做事**: 耶输陀罗怀孕六年之苦即是前世让母亲背奶桶的果报
- **粗语虐待**: 以粗语恶言、残羹剩饭虐待父母, "没有比子女虐待父母更严重的罪业了"

《地藏菩萨本愿经》教证: "若有众生不孝父母或至杀害, 当堕无间地狱, 千万亿劫求出无期。" — 直接把 **不孝** 与 **杀害** 并列入无间罪范围。

堪布引当时社会数据: 大城市 7% 的孩子害怕父母, 13% 讨厌, 50% 以上憎恨 → 超 70% 态度负面。对净土行人的 wake-up: 你对父母的心态都是负面的, 而且还缘他们造了很多恶业, 这些罪业没有忏悔清净以前能往生极乐世界吗?

### 杀阿罗汉

阿罗汉 = "断尽三界烦恼的殊胜僧众" (《宝性论》以一地以上圣者菩萨为真实僧宝)。杀阿罗汉 = 损害僧宝中 最殊胜者, 故属无间。

### 恶心出佛身血 — 提婆达多公案

**核心公案: 提婆达多** (L65) — 以石击佛, 伤佛足流血, 为佛陀在世时造五无间罪的 典型例子。

现代引申 (L65):
> "如今释迦牟尼佛已经示现圆寂, 我们不可能直接损害佛身, 但如果对佛像不恭敬或者诋毁也会造下严重的罪业。"

这是把 **恶心毁坏佛像** 与 **出佛身血** 类比, 为 近无间罪中的 "毁像" 铺垫。

### 破和合僧

定义: "破坏僧众和合会中断法轮, 这样会损害佛法"。
结成条件 (传统标准): 四人以上清净僧团 + 分裂持异见 + 形成两个羯磨团体。
现代语境: 宗派之间互相诽谤, 恶心分裂一方清净僧众。

### 忏悔方式 — 除五无间外还可清净

关键定理: "无论何人, 只要诚心忏悔, 相续中的罪业一定会清净。" 《金光经》: "何人千劫中, 若造严重罪, 一次极力忏, 诸罪得清净。"

具体方法 (堪布 L65 列举):
1. 在阿弥陀佛及其眷属前发露忏悔
2. 安住空性境界忏悔
3. 密宗不共修法 (金刚萨埵心咒、百字明)
4. 念诵《普贤行愿品》— "往昔由无智慧力, 所造极恶五无间, 诵此普贤大愿王, 一念速疾皆消灭。"
5. 念诵《三十五佛忏悔文》 (堪布个人日课)

### 核心 apology — 五无间 + 舍法罪 极难往生

> "暂时有两种人不能往生极乐世界, 即造舍法罪及五无间罪者。然而, 并非是佛陀根本不慈悲摄受他们, 最终这两种人也能如愿以偿。但因为这两种人恶业深重, 暂时无有成就所愿的机会。" (译作, 积资净障 第58行)

这回应了极乐愿文 "除造舍法无间罪" 排除条款。忏悔这两类 = 往生 极乐 的 关键障碍清除。

## 壬二、忏悔近无间罪 (L68-69)

### 五项近无间罪

根本颂 "杀害比丘与沙弥, 污尼毁像塔寺等":

1. 杀害具戒的比丘
2. 杀害沙弥
3. 污尼 (毁坏比丘尼 / 沙弥尼等持戒女子的戒律)
4. 毁坏佛像 (画像、塑像)
5. 毁坏佛塔
6. 毁坏经堂等佛堂 ("等"字所摄)

**为何叫"近无间"**: "它们的过患比前面所讲的五无间罪稍微轻一点, 所以叫做近无间罪" (L68)。

### 本论与《俱舍论》的对比

《俱舍论》说法:
> "染污母亦无学尼, 杀定菩萨有学圣, 夺僧合食毁佛塔, 此等即近无间罪。"

即: 以邪淫侮辱 "既是母亲又是阿罗汉尼者" / 杀住定菩萨 / 杀有学圣者 / 夺僧和合食 / 毁佛像佛经佛塔。

本论取舍原则 (L68): "没有宣说染污既是母亲又是阿罗汉尼、杀害有学圣者等不容易造的罪业, 而是着重宣说了容易造的罪业。" — 实用取向。

### 杀害比丘沙弥

"清净戒律的比丘和沙弥是佛教的根本, 杀害他们就同时毁坏了法宝与僧宝" (L68)。

关键 teacher 论断 — **即使破戒的比丘也不能损害**:
> "不要说具有清净戒律的出家人, 即便是破戒的出家人, 损害他们也有严重的过患。为什么呢? 因为如今的出家人无论再怎么下劣, 最迟也会在胜解佛出世时证得圣果, 从这个角度而言, 任何出家人都不应该受到迫害。" (L68)

**胜解佛之愿**: 贤劫千佛中最后一位太子发愿 "所有在九百九十九尊佛教法下破戒、造恶业的出家人在我的教法下都获得解脱" → 胜解佛。此愿力是保护一切出家人的根本因。

教证:
- 《正法念处经》: "佛说一切法, 出家最第一"
- 《地藏十轮经》: "瞻博迦华虽萎悴, 而尚胜彼诸余华, 破戒恶行诸比丘, 犹胜一切外道众"
- 《赞僧功德经》: "破禁比丘虽无戒, 初心出家功德胜, 百千万亿白衣人, 功德纵多不及彼"
- 麦彭仁波切: "虽说破戒如人尸, 然具剩余之功德, 如牛王死留苦胆, 獐子尸体具麝香"

**关键公案: 尊固狮子** (L68) — 金毛狮子听独觉讲法, 对出家人有极大欢喜心。猎人剃头穿僧衣, 以毒箭射之。狮子醒后欲扑咬, 见猎人身穿僧衣, 立即心念: "如果我杀了这个人, 就是对三世诸佛和圣者起恶心, 我不应该杀他。" 发愿: "愿自丧身命, 终不起恶心, 向于坏色服。愿自丧身命, 终不起恶心, 向于出家人。" 之后而死。→ 旁生都不损害着僧衣者, 人更不应损害。

### 污尼

定义分两层 (L69):
- **狭义**: 毁坏出家女子的戒律 (比丘尼 / 沙弥尼)
- **广义**: 毁坏一切持戒女子的戒律 (女居士 / 八关斋戒女)

"一个人即便具足一分戒律 — 如一分居士戒、一天八关斋戒 — 也属于证法, 此人就成了一切人天大众所供养的佛塔。如果有人毁坏…他就是在玷污佛教, 此人也断绝了解脱道。"

**莲花色比丘尼公案** (L69): 佛在世时, 相貌庄严的阿罗汉尼莲花色。婆罗门生染污心潜入其房, 于她熟睡时作不净行。当她觉醒以神通从床上飞到空中, 大地裂开, 婆罗门直接堕入无间地狱。

**姓许举人公案** (L69, 《寿康宝鉴》): 福州许某挑逗不成强暴尼姑, 第二天发狂咬断舌头而死。

藏地谚语 (L69): "以黑毒之斧, 损坏金佛塔" — 毁坏戒律如用斧头砍坏金色佛塔。法王如意宝说 毁一修行人的戒律 = 毁众多佛塔。

宽度: 持戒的功德 **远远超过布施** → 反之, 制造破戒违缘的过患也超过阻止布施的过患。《宝积经》: "何人以信心, 百年作布施, 何者一日持, 净戒较其胜。"

### 毁像

造佛像功德 (《白莲花经》《妙法莲华经》): 甚至童子游戏画佛像、以沙石堆佛塔, 皆能种下成佛之因。

毁佛像公案:
- **巴米扬大佛** (L69): 2001 年塔利班炸毁。世界第三大佛 (53米), 法显《佛国记》、玄奘《大唐西域记》均记载。毁后当年 "9·11" → 美军空袭阿富汗, "参与毁佛的人大多死于美军的轰炸"。
- **"文革"毁佛当场吐血** (L69): "我家乡有些人毁坏佛像, 有的人当场就吐血而亡"
- **塔姆女人** (L70): 朵芒寺大佛塔被爬上拆毁, 后两眼皆瞎, 半死不活挣扎多年

现代语境: 故意破坏 vs 维修区别 — 残缺不全的经书、笔记和草稿, 若无法使用保管, **没有不好的心态** 可以焚化于清净地。这是对 "毁坏" 的限定性界定 — 出发点是关键。

**毁坏高僧大德像也有严重过患**: "汉地有些人毁坏祖师塔后当场就死去。"

### 毁坏佛塔 / 佛堂

佛塔 = 佛陀意所依。

建塔、转绕功德: 阿底峡尊者: "在一切有为善法中, 没有比转绕的功德更大的。" 《一切如来心秘密全身舍利宝箧印陀罗尼经》: "乃至应堕阿鼻地狱, 若于此塔一礼拜一围绕必得解脱。"

**猪蹭泥补塔公案** (L69): 猪为吃草蹭泥, 恰好补合佛塔裂缝 → 转生天界。
**牛绕塔公案**: 牛为吃草右绕佛塔一圈 → 释迦佛教法下获阿罗汉果 (牛相尊者)。

毁佛堂范围 (L70): 不用说真的毁坏, 就是"在佛堂里睡觉、吐口水、擤鼻涕、丢茶渣或者放屁也将堕入恶趣"。

**《佛说因缘僧护经》三肉柱** (L70): 僧护见地狱三根肉柱 —
1. 迦叶佛时代出家人 私自用佛堂柱子 → 被大火燃烧的肉柱
2. 在家人 刀刮佛像金子 → 狱卒利斧砍杀的肉柱
3. 出家人 将经堂木材随便送在家人 → 燃烧的肉柱

另见: 经堂附近随地大小便 → 形似厕所的众生; 不脱鞋踩地毯 → 铁脚饿鬼。

**历史教训** (L70): "三武一宗" 灭佛. 唐玄宗削减僧尼、拆毁佛堂 → 安史之乱 → 杨贵妃被杀 → 太上皇郁郁寡欢而死。宋徽宗贬佛 → 北宋被金灭 → 五国城坐井观天。1958 藏地"宗教民主改革" 毁大量寺院 → 1959-1961 三年大饥荒 非正常死亡三四千万人。堪布直言这是毁佛运动的果报。

**毁佛堂"变相"** (L70): "有些人虽然没有直接毁坏佛堂, 但盗取建造三宝所依或者佛堂的钱财, 结果造成三宝所依或者佛堂因材料不足而缩小面积, 这种行为的后果和毁坏佛堂是相同的。"

- 绍明和尚公案: 以建佛殿为名四处化缘, 后私自享用 → 病中被大火从头烧到脚, 反复感受
- 盗常住木料建华宅 → 重病打卦方知

## 壬三、忏悔舍法罪 (L70-71)

### 根本颂类型 — 以三宝为证违誓

根本颂 "三宝殿经所依等, 以彼作证违誓等":

此处舍法罪 = 以 **寺院、佛堂、三宝所依** (佛像、佛经、佛塔) 等殊胜对境作证发誓, 而后故意违誓。

具体场景 (L70):
- 说绮语时 以三宝对境随便发誓
- 打官司时 以三宝发誓
- 做生意时 以三宝发誓
- 后故意自食其言

**甚至仅说"天尊为证"后违誓**, 都必须造天尊身像, 否则即舍弃此天尊。

堪布的文化观察: 藏族以三宝作证发誓的传统深, 故藏族造此类舍法罪多; 汉族不常以三宝发誓, 此类较少 (但汉地对天发誓亦同理)。**福州许某对天发誓赖账** (L70) — 2008 年《东南快报》报道: 许某赖五百元礼金, 对天发誓 "如果欠钱就遭天打雷劈", 一分钟后被雷击倒 (抢救活)。

喇拉曲智仁波切直指: "如今有些人连做一个小泥塔或者刻印一句观音心咒的能力都不具足, 竟然口口声声以《甘珠尔》《丹珠尔》作证而发誓…大多数人以开玩笑的方式发大誓, 结果福德在口中耗尽。"

莲花生大士: "劣者自食其誓时, 灭福损失庄稼也。"

### 果报

- 增上果成熟于外境: 转生土地贫瘠、食物无营养、恒时遭地水火风灾难之地
- 果报成熟于自身: 无威力、无智慧、无福德

善誓之力反例: 米拉日巴立誓"若不得殊胜成就, 纵然饿死也不下山" → 成就; 阿那律立誓终不睡眠 → 双目失明转得天眼通。

### 根本颂"等"字 — 一般意义的舍法罪

"以彼作证违誓等" 中的 "等" 字 涵盖一般意义上的舍法罪 (L71), 范围更广:

**舍法罪出现的机制** (L70):
> "有些人依止了一位无有智慧、具偏袒心的上师后, 也跟着上师以贪嗔之心扰乱佛教…便自认为是受持宗派者或者大修行人, 于是以傲慢之心说: 我通过观察认为这一点是合理的, 那一点是不合理的。这些人就像盲人评论色法的贤劣一般, 对内道各教派的见解、教规胡乱破立取舍, 致使很多愚人跟在后面随声附和。结果不仅自己造了舍法罪, 还让别人舍弃了正法, 这个罪过比同时造五无间罪还严重。"

### 教证系列 — 舍法罪重于五无间罪

- 《入大乘论》: "谤法之罪, 重于五逆, 恶道长远, 久受苦报"
- 《般若八千颂》: "何人若造五无间, 不及相似谤佛法"
- 《宝性论》: "何人心嗔法, 彼焉有解脱?"
- 《宝性论》: "应畏谤深法, 及谤法知识, 决定令人入, 可畏阿鼻狱"
- 永嘉禅师《证道歌》: "欲得不招无间业, 莫谤如来正法轮"
- 《妙法圣念处经》: "迷惑诸有情, 谤佛毁正法, 堕落于三涂, 穷劫不能出"
- 《经庄严论》: "随闻得具慧, 于闻作轻毁, 余种种无量, 愚昧何能定?" / "尽解直声义, 生慢失智慧, 舍弃诸善说, 毁自嗔法障"

### 汉藏佛教互不相谤 — 堪布反复强调

堪布 L71 重点论断:

> "现在有些净土宗的法师和信徒认为只有净土法门最殊胜, 其他法门都不殊胜, 经常说一些特别可怕的语言: '藏传密法不是佛说的' '禅宗不是佛传下来的'… 净土法门当然是殊胜的正法, 但其他法门也是殊胜的正法。如果你只提倡净土法门, 完全排斥其他法门, 这样肯定会造下舍法罪。"

> "有些人认为只有净土法门最殊胜…有些人口口声声说: 格鲁派不对, 宁玛派不对, 禅宗不对, 密宗不对, 甚至有的法师也经常说某个法门不是纯正的佛法。还有些学者声称《楞严经》《维摩诘经》是伪经, 《入大乘论》是后人伪造的, 等等。"

**原则**: 不舍一法 + 着重修和自己有缘的法。《楞严经》: "归元性无二, 方便有多门"。珠巴根拉大师: "我遇各种胜法理, 每一窍诀皆结缘。"

**中间态安立** (关键!): 佛陀: "住于中间者, 舍法我未说。" — 对自己不了解的佛法, 既不赞叹也不诽谤, 自心安住于中道, 则不会造舍法罪。弥勒: "意罪自性恶, 非理色非理, 疑法何须说, 等舍善无过。"

### 诽谤密法 — 续部教证

续部专门针对诽谤密法:
- 《等虚空边续》: "密中之胜密, 何人若诽谤, 则弃佛胜密, 永离解脱道"
- 《深智圆满续》: "违背了义罪业者, 定堕地狱诚可悲"
- 《护意续》: "若谤普贤此密意, 乃至虚空存在间, 死后堕入金刚狱" — 这是 **金刚地狱** 的教证来源
- 《将哲续》: 末世人以法造舍法, 偏执经律藏, 赞自贬他法 → 堕金刚无间地狱

堪布重要论断: 诽谤普贤王如来 / 莲花生大士 = 诽谤阿弥陀佛 — 因密意一致。"如果你舍弃了普贤王如来、莲花生大士, 就等于舍弃了阿弥陀佛, 阿弥陀佛肯定不会接引你往生极乐世界的。" (L71) 这是把 Nyingma 传承与净土结合的 关键论断。

### 宁玛派退转信心 / 破誓言

堪布引续部"破誓瑜伽者, 多出行违缘" — 密乘行人舍弃别解脱戒 = 失毁密乘誓言 → 修行障碍, 无成就。

## 壬四、忏悔诽谤菩萨罪 (L72-73)

### 根本颂 — 重于杀三界有情

"诽谤诸菩萨之罪, 较杀三界有情重, 发露忏悔无义罪。"

### 为何如此严重

逻辑 (L72):
> "菩萨是成佛的因, 相当于一切众生的父母, 顶礼初发心菩萨就等于顶礼一切诸佛。"

→ 杀一众生: 损一众生; 杀三界一切众生: 损三界一切。但诽谤一菩萨: **断未来无量众生成佛之因**, 故胜于杀三界。

《信力入印法门经》: "若其有人谤菩萨者, 彼人名为谤佛谤法。" — 诽谤菩萨 = 诽谤一切诸佛和佛法。

### 对境范围 — 不限登地, 包括初发心菩萨

> "此处的菩萨不只是文殊菩萨、普贤菩萨等登地的大菩萨, 而是指一切菩萨。很多经论中都说, 诽谤菩萨罪的对境包括初发心菩萨, 即相续中有无伪世俗菩提心者。" (L72)

即: 汉藏凡受菩萨戒之行人, 皆是此律所护对境。

**汉藏出家人都是菩萨**:
> "雪域藏地是观世音菩萨的所化刹土, 每一个藏族人都会念观音心咒, 《宝箧经》中说凡是会念观音心咒的人都是大乘种性, 所以藏族人应该说都是大乘种性…汉地也是大乘佛法盛行的地方, 汉地的出家人在受戒时, 同时要受别解脱戒和菩萨戒。因此现在所见的出家人都可以说是菩萨, 大家应该对他们观清净心。"

### 不知对境是菩萨的不小心诽谤也感重报 — Key 业果原理

**这是最关键的业果原理之一** (L72):

> "很多自称是凡夫的人其实是真正的大菩萨…获得生死自在的大菩萨可以投生为固定、不固定的种种形象来利益有情。固定的形象如文殊菩萨、观世音菩萨或者世人公认的高僧大德。不固定的形象如乞丐、屠夫、妓女、旁生等下劣众生。"

《学集论》: "彼若未断定, 轻蔑诸菩萨, 如灰覆之火, 其于地狱焚。" — **未断定身份即诽谤, 后果等同已知**。盖着灰的火坑从外面看不到火焰, 但踩下去全身被烧毁。

藏地谚语: "不知盗贼、菩萨在何处。"

**堪布个人忏悔** (L72):
> "从无始至今生, 很多人肯定造过诽谤菩萨的罪业。就我本人而言, 出家之前, 经常说别人的过失; 出家以后, 虽然大多数时间都能以正知正念守护自己的言行, 但凡夫人有放逸的恶习, 一旦遇到恶友就会跟着说过失。在自己诽谤过的人中肯定有菩萨, 所以我肯定造过诽谤菩萨的罪业。"

### 关键公案

**僧崖焚身** (L72, 《续高僧传》): 南北朝出家人, 自称"我是凡夫, 誓入地狱, 代苦众生, 愿令成佛"。晚年烧十指、二臂、最后焚身。燃指时人问痛否, 答: "痛由心起, 心既无痛, 指何所痛?" 焚身后肉身成灰, 心脏完好。→ 自称凡夫而实为大菩萨之示现。法王如意宝荼毗 钢条被熔 心脏未坏。

**文殊菩萨老翁相** (L72, 《杂譬喻经》): 迦罗越设大布施会期望见文殊, 丑陋老翁七次被拽离法座, 梦中方知即文殊菩萨。

**老妇狗牙成就** (L72, 《大圆满前行》): 儿子欺母, 以路边狗牙骗称为佛牙。老妇信以为真, 恭敬顶礼, 狗牙中出现舍利, 老妇成就。→ 信心的力量。

**萨迦法王根嘎酿波向小僧人顶礼** (L72): 见小僧脱僧衣在小溪耍箭, 曰 "请穿上法衣, 我这个老居士要向您们顶礼"。

**康熙射文殊** (L72): 五台山见和尚与女人水池共浴, 怒射右肩。后在菩萨顶见文殊右肩插箭 → "带箭文殊"。

**乾隆普陀山** (L72): 三次见出家人"非法行为" 欲杀, 第三次与之赌博, 皇冠皇袍输给观音罗汉像。

**阿育王顶礼七岁沙弥** (L72, 《阿育王传》): 带至无人处顶礼。沙弥以神通钻瓶, 并叮嘱"也不要告诉他人"。

**锁骨菩萨** (L73, 《太平广记》): 延州妇人不拒男子, 死后骨节皆锁状 → 以欲钩牵引入佛道。《维摩诘经》: "或现作淫女, 引诸好色者, 先以欲钩牵, 后令入佛道。"

**松赞干布顶礼捉虱比丘** (L73): 比丘生我慢念 "这些大国王也向我顶礼", 王以他心通诫之"我尊敬的是释迦牟尼佛的律藏法门, 并不是你这个人, 你不要太傲慢了"。比丘反示神通 — 三大塔放指尖; 国王开发髻显头顶阿弥陀佛像; 比丘剖腹显胜乐金刚坛城。

**阿底峡门口老妇** (L73): 大伏藏师眷属开路时打倒"老妇", 往生铜色吉祥山被益西措嘉空行母阻拦 — 正是那"老妇"。

**达西老猪 / 薄荷猪** (L73): 文殊菩萨化身为官员或老比丘, 指示行人将信交给猪 (达西、薄荷), 猪看信即死。→ 旁生中亦有佛菩萨化身。

### 应对 — 观清净心 + 一切皆菩萨化现

操作方法:
- 不知者沉默: 萨迦班智达 "一无所知的愚者若一言不发, 则对佛法损害不大"
- 《宝积经》: "覆藏自过失, 观察他罪过, 此二如毒火, 智者舍此过"
- 释迦牟尼佛: "除非我与同我者, 无人能量他人心, 若量则犯大罪过"
- 智悲光尊者《功德藏》: "诸行不应生邪见, 圣地数多自在者, 显现劣种恶形象, 恶劣之中极恶劣"
- 寂天: "生佛既同体, 何不敬众生?"
- 惟一例外: "为了摄受众生, 或者你是寺院的管家, 在这种情况下也许不得不说过失"

**诽谤僧众(总体)尤重**: 目犍连地狱所见 "耕舌众生" — 迦叶佛时曾说"这些僧人是破戒者, 他们戒律不清净" → 五百世在地狱感受耕舌之苦, 之后多佛出世仍不得解脱。《集菩萨学论》: "云此说法师有是辩才、无是辩才, 亦名谤法。"

### 忏悔剂量

"念四十万遍金刚萨埵心咒、十万遍百字明或者其他的忏悔文, 并且发誓今后再不造罪业。" (L73)

## 壬五、忏悔恶见罪 (L74-75)

### 根本颂 — 重于五无间的无解罪

> "闻善功德恶过患, 地狱痛苦寿量等, 认为不实仅说法, 此罪重于五无间, 发露忏悔无解罪。"

定义: 听闻善法功德、恶业过患、地狱痛苦与寿量等因果道理后, 认为这些道理不真实, 只是佛教的说法 — 此 **不信因果** 的大邪见, 超过五无间罪。

### 何以称"无解罪"

**关键区别** (L74):
> "经论中说, 一个人造了五无间罪还有忏悔的机会, 而一旦产生上述邪见, 则会从根本上中断善业的相续, 乃至没有对因果重新产生诚信之前, 不仅相续中的善根无法恢复, 而且即使忏悔也无法清净罪业, 无量劫都不能从恶趣中获得解脱。"

机制:
- 五无间 是行为之罪 → 可以行为 (忏悔) 对治
- 恶见 是 **根本见** 之罪 → 断绝整个断恶行善的 因
- 正见是一切修行的根本; 毁正见 = 断一切解脱因

《法王经》: "若作邪见, 即烦恼起; 若无邪见, 烦恼不生。"
《大般涅槃经》: "若有邪见, 命终之时, 即应生于阿鼻地狱。"
《大智度论》: "是人邪见网, 烦恼破正智, 离诸清净戒, 唐苦堕异道。"
《毗奈耶经》观点: 一个人若有恶见, 相续中的戒律从根本上失毁。

《诸经要集》两类邪见:
- **可转邪见**: 非因见因, 非果见果 (可转) — 轻邪见
- **不可转邪见**: 诽谤因果, 言无圣人 — 重邪见 (即本颂所指)

### 现代版本

堪布列举多种现代恶见 (L74, L75):

1. "佛教的道理不可能是真的, 即使它是真的, 我们也无法理解"
2. "善有善报、恶有恶报不一定是真的, 哪有那么玄的事啊?"
3. "即使善法真的有功德、恶法真的有过患, 像我们这样业力深重、具足束缚的凡夫俗子也不可能如理取舍"
4. "很多人把因果当作神话、传说或者虚无缥缈的幻想"
5. "只有米拉日巴、无垢光、宗喀巴那样的大成就者才能终生行持苦行" (以此为懈怠借口)
6. "供一盏灯、烧一支香是很小的善法, 不可能有那么大的功德, 否则所有的众生早就成佛了" — 蔑视微小善恶业

堪布 L74 引自称 2009 年成佛者案例, 斥为末法时代无知之"佛"。

### 密宗语境 — 见行脱离 = 密宗魔

《胜乐续》: "不具见之行, 不具行之见, 此二密宗魔, 堕入无间狱。"

- **不具见之行**: 内无密宗见解, 外造杀盗淫妄恶行
- **不具行之见**: 有密宗见解但不行密宗行为

堪布明确警告 (L74):
> "现在有些所谓的密宗瑜伽士确实太过分了, 他们经常宣称: '修密宗的人什么都可以做, 可以喝酒、吃肉, 杀盗淫妄也是开许的。' 其实, 续部中根本没有这样的开许, 凡是自相的烦恼和恶业, 密宗一律遮止。"

《妙臂续》: "佛陀我说别解脱, 一切清净戒律中, 在家咒士除相轨, 其余皆当实行持。" — 密宗行人仍持小乘戒律遮止烦恼, 仅 "外相 + 仪轨" 与小乘有异。《密集续》: "外护声闻行, 内喜密乘义。"

### 关键公案

**鉴空法师** (L74, 《法华持验记》): 前世迦叶佛时代出家四十年, 日食一餐、一破衣。却因说法掺杂异端邪说, 使人怀疑谤因果犯戒 → 今生潦倒贫困。梵僧以枣使其忆前世。→ 讲经说法掺杂邪见 是 严重业因。

**米拉日巴与希哇俄对话** (L75): 学人疑"我们做不到像您那样" → 尊者回 "你们以为我是佛菩萨的化身, 这对我固然是一种信心, 但对法却是无比的邪见。其实我本来是一个凡夫, 前半生还造了大罪业, 但因为我坚信因果报应, 所以能够抛弃今生的琐事一心修行…如果内心深处对因果有坚定的信心, 像我这样的成就每一个人都能获得。" → 归根结底是 **因果正见** 才是解脱之门。

**老妇"晕头转向"公案** (L75): 上师告功德与过患, 老妇曰"听了善法的功德, 您这么清净的比丘与我这样罪业深重的人都有解脱希望; 听了罪业的过患, 您与我都有堕落危险 — 真不明白哪一种正确"。→ 老妇之见偏颇, 实则 "造善皆有希望, 造恶皆有堕险", 因果法真实无欺。

**莹珂和尚** (L75): 平时喝酒吃肉, 偶读《往生传》后, 毛骨悚然, 认真取舍因果 → 阿弥陀佛三日接引往生。→ 最初不信因果者, 后诚信因果 亦有解脱之机。

**死相比丘与具棘沙弥** (L74-75): 对戒律支分 (过午不食、不触火、不砍树) 起邪见 "制定这些戒律有什么必要呢? 真不知道这个佛陀在说什么" → 断绝善根。

**持牛戒、狗戒外道** (L75): 以有限神通见前世当牛/狗死后升天, 误认为当牛/狗是升天之因 → 持牛戒狗戒。业因果微细, "非凡愚分别心所及"。

**杀羊升天屠夫** (L75, 《杂譬喻经》): 阿阇世王前 宿命通屠夫自述六次杀羊六次升天。佛解释 — 此人曾于辟支佛前起善心, 六次升天实为此故, 然神通有限只见六世未见更远。杀羊果报未熟, 将堕地狱。

### 佛语无欺 — 理性信心的建立

《阿难分别经》: "佛语至诚, 终不欺人。"
《入中论》十力之一: "知处非处智力"
《大萨遮尼乾子所说经》: "业及业果报, 所有果因智, 皆能如实知, 是故名为佛。"
《月灯三昧经》: "因果相应胜法门, 微细难见佛境界" — 因果 唯佛境界, 阿罗汉亦不能彻知。
《宝积经》: "诸异报业因虽少, 当来获果无有量, 或无量因感少果, 善逝遍能如实知。"

寂天: "佛说一切善, 根本为信解, 信解本则为, 恒思业因果。"

## 整段连贯论证 — 严重性递增序列

五支按严重性与对治难度递增:

| 科 | 项 | 对境 | 业的本质 | 可忏悔性 | 判定难度 |
|---|---|---|---|---|---|
| 壬一 | 五无间 | 父母 / 上师 / 阿罗汉 / 佛 / 僧 | 行为 (身) | 可忏悔 (除非加舍法) | 较易 (对境明确) |
| 壬二 | 近无间 | 比丘沙弥 / 持戒女 / 佛像塔寺 | 行为 (身) | 可忏悔 | 需辨身份 |
| 壬三 | 舍法 | 正法 (三藏) | 语 / 意 | 可忏悔但障深 | 易无意造 |
| 壬四 | 诽谤菩萨 | 一切菩萨 (含凡夫示现) | 语 / 意 | 可忏悔但对境隐 | **极难 (菩萨遍在)** |
| 壬五 | 恶见 | 因果正见本身 | 意 (根本见) | **无解** (未恢复正见前) | **自难察觉** |

递进逻辑:
1. **对境升级**: 个别严厉对境 → 正法 → 菩萨 (成佛之因) → 因果本身 (一切佛法的根本)
2. **业类升级**: 身 → 口 → 意 → 根本见
3. **忏悔难度升级**: 明显行为易对治 → 微细语言难察 → 心念难察 → 根本见断绝一切善根

从 "五无间" 到 "恶见" 是一个 **从具体行为到根本见地** 的严重性序列。乔美仁波切 / 喇拉曲智仁波切的安立, 在 壬五 特别加强 — 因恶见 "中断善业的相续", 属净土行人最隐蔽的障碍。

## 忏悔重业的特殊方法

针对此五类重罪, 堪布在各课反复强调 **密乘忏罪的必要**:

### 金刚萨埵修法 / 百字明

- 针对诽谤菩萨罪: "念四十万遍金刚萨埵心咒、十万遍百字明或者其他的忏悔文, 并且发誓今后再不造罪业" (L73)
- 堪布个人日课 (L65): "在自己的相续中, 肯定有能想得起来的以及想不起来的许多罪业, 这些罪业肯定会遮障自己的光明心性, 所以忏悔对自己非常有必要。"

### 三十五佛忏悔文

堪布个人日课 — "每天不管再怎么忙, 只要一有时间就不停地念《三十五佛忏悔文》" (L65)。三十五佛在因地发下清净众生罪业的殊胜大愿, 专应严重罪业。

### 《普贤行愿品》

"往昔由无智慧力, 所造极恶五无间, 诵此普贤大愿王, 一念速疾皆消灭。" — 法王如意宝重点开许之忏法。

### 法海雷音如来名号 (恶见罪)

《药师经》记法海雷音如来因地愿: "若有众生生邪见家, 于佛法僧不生净信…若闻我名至心称念, 由是力故, 无明邪慧日夜消灭, 于三宝所深生正信, 不复退转, 乃至菩提。" → 专治恶见罪。

### 四对治力配合 (详见 06-佛制罪与四对治力)

所依对治力 / 厌患对治力 / 返回对治力 / 现行对治力 — 四力具足才能真正清净此级重罪。本文所列各种忏悔方法都须以四对治力摄持, 否则不过是口头念诵。

## 对 Nyingma 净土行人的 关键整合点

1. **"除造舍法无间罪" 的意义**: 极乐愿文根本颂排除两类, 本辛二单元的核心目的正是把这两类清除掉。忏悔不净, 往生难成。
2. **诽谤普贤王如来 = 诽谤阿弥陀佛** (L71): 堪布反复申论 — 此为 Nyingma 传承与净土信心的不共桥梁。
3. **密乘誓言与别解脱戒不二**: 《妙臂续》《密集续》确立 — 密法行人仍遵小乘戒律, 恶见 / 破誓即断密乘成就。
