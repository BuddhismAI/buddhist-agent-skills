## 一、自性罪与佛制罪的分界（总论）

忏悔支在 *现行对治力* 一科下分四项：忏悔自性罪（见 05 文档，涵盖严重罪业）、忏悔佛制罪、忏悔未认识之自性罪、忏悔未知就犯之佛制罪。理解这四项，首先要分清两个根本范畴：

| 类别 | 定义 | 感果条件 | 典型例 |
|------|------|---------|--------|
| **自性罪** | 以贪嗔痴烦恼引发的一切恶业，本性即是恶业 | 不论在家出家、是否受戒、是否知其为罪，造即感果 | 杀生、偷盗、邪淫、妄语、饮酒（大部分声闻宗观点）、吸烟 |
| **佛制罪** | 本性非恶，因违佛所制戒律而成罪 | 必须先受戒才可能违犯 | 过午食、持金银、未受戒律仪轨而污染资具等 |

喇拉曲智仁波切在大疏中点明分界：「在家男女们如果没有受戒则不会有佛制罪。然而，所造的不善业却不可胜数。」即未受戒者不构成佛制罪，但仍须忏悔一切自性罪。《因缘品》云：「愚者若未知，造作诸罪业，一旦果成熟，各自受其苦。」世间所谓「不知者不为罪」只是俗谛说法——因果丝毫不因无知而减损。

佛制罪按所受戒类又分三层：别解脱戒、菩萨戒、密乘誓言。三戒戒体、所破、忏悔仪轨各不相同，必须「像饲养马牛羊三种牲畜般分别护持」——马在平地、牦牛在山坡、羊在山顶，习性不同，不能混养。

### 三戒分摄对比表

| 范畴 | 别解脱戒 | 菩萨戒 | 密乘誓言（三昧耶） |
|------|---------|--------|-------------------|
| 所缘重点 | 遮止身语的恶行 | 愿菩提心 + 行菩提心 + 饶益众生 | 上师、金刚兄弟、本尊坛城、自性清净见 |
| 戒条本数 | 比丘 250/253/257 等（各律部不同）；沙弥十戒；居士五戒/八关斋戒 | 十八堕（龙树系）或二十堕；或愿/行两类学处（无著系）；中根四堕；钝根一堕 | 十四根本戒、八粗支戒；另有各续部不共誓言 |
| 失戒条件 | 根本戒四他胜一旦违犯即失 | 龙树系可多次重受；无著系最多重受三次 | 犯根本戒后若过三年未忏即无清净机会 |
| 忏悔方法 | 向上座、诸佛菩萨发露，依律部仪轨 | 每日受菩萨戒仪轨、修白法对治黑法 | 金刚萨埵百字明、会供、向上师忏悔（3 年内） |
| 所破烦恼 | 粗分身语烦恼 | 二乘自利心 + 断四黑法 | 一切对自性清净起疑、轻毁心 |
| 堕处 | 犯恶作→复活；恶作以上→黑绳、众合、大号叫、极热地狱 | 堕恶趣 | 金刚地狱（无量劫） |

宁玛派自宗随龙树菩萨体系建立菩萨戒，密乘戒三戒转依可在本体上讲，但「并非初学者相续中细微单戒可混为一谈」——麦彭仁波切《建立三戒一体论》对此有详论。换言之，不能以「我得过密宗灌顶，什么都不要紧」作借口忽视下层戒律，否则「灌顶的誓言尚且不清净，又怎能依靠密乘戒来救护下面的堕罪？」

## 二、庚二·辛一 忏悔别解脱堕罪（第76课前半）

### 比丘戒五堕罪

比丘戒有五种堕罪，每破一种对应不同地狱果报：

| 堕罪类 | 内容 | 堕地狱 |
|--------|------|--------|
| 四他胜 | 杀人、盗、淫、大妄语——出家人最根本戒，犯则直接失戒 | 极热地狱 |
| 十三僧残 | 残损戒体但未失戒，需僧伽前忏悔 | 大号叫地狱 |
| 三十舍堕 + 九十单堕 | 与财物持用相关的中品戒 | 众合地狱 |
| 四向彼悔 | 向某一人忏悔即可 | 黑绳地狱 |
| 一百一十二恶作 | 身语细分威仪 | 复活地狱 |

《佛说犯戒罪报轻重经》中说：即使犯一恶作罪，也将于八万年中在地狱感受痛苦。这是粗线条的说法——「不管犯戒的轻重都算破戒，包括犯一个恶作罪也算破戒，不可理解为平常意义上的破根本戒」。

### 沙弥戒、居士戒、八关斋戒的现代取舍

- **沙弥十戒、居士五戒**：以不饮酒为根本（藏传传统中居士受戒先受酒戒，然后才可受少分/多分/圆分）。为何先受酒戒？「烂醉如泥者连自己的生命都无法保护，又怎么能保护戒律呢？」
- **八关斋戒**：过午不食等在家人的短期出家戒。
- **现代情境判定**：佛制罪的判定需结合原戒条的立义。例如为治病服用少量含酒药物，对受了戒的出家人仍不开许；但若只是偶尔食用电子食品、乘车等现代便利物事，其是否犯戒须依律部广疏判别，不可自行揣度。

### 持戒功德与破戒过患

《正法念处经》：「若人护持戒，此人则为胜，若人舍离戒，是名为死人。」《大般涅槃经》：「持戒则为乐，身不受众苦，睡眠得安隐，寤则心欢喜。」——持戒者「相貌气色好、内心安乐」；破戒者「连僧团行列都无法进入，更何况往生极乐世界」。

《毗奈耶经》：「宁吞热铁丸，犹如猛火焰，不以破戒口，啖他信心食。」破戒者不具资格享用僧众经堂、饮食、资具。

公案：**噶·当巴德协**（噶陀寺创建者）24 岁出家、71 岁守净戒而圆寂往生极乐，证八地菩萨；**弘一法师**坐藤椅前轻摇以让小虫避开；**昌臻法师**（2009 年圆寂）烧出 1000 余颗五色舍利——戒律清净是往生净土的根本所依。

## 三、庚二·辛二 忏悔破菩萨戒堕罪（第76课后半）

### 无著菩萨系：四黑法 / 四白法

无著菩萨所传愿菩提心学处，以舍四黑法、修四白法为要：

| 四黑法（须舍） | 四白法（须修，《迦叶请问经》） |
|---------------|-------------------------------|
| 一、妄言欺骗上师、堪布、阿阇黎 | 一、宁死不明知故妄 |
| 二、令行善者对其善行产生怀疑或后悔（如对受戒者说"你没得戒体"） | 二、于菩萨作本师想 |
| 三、以嗔恨心毁谤高僧大德 | 三、于众生无谄诳而以净心安住 |
| 四、以经商谄诳手段欺骗众生 | 四、将一切众生安置于大乘 |

《十住毗婆沙论》：「如此四黑法，则失菩提心。」

### 龙树菩萨系：十八（或二十）根本堕罪

宁玛自宗随龙树观点。十八堕按身份三分：

**国王五定罪**（势大易犯）：
1. 强夺、盗窃或令他人盗窃三宝财物
2. 舍弃三乘佛法（如说「你学这些法没有意义，不能获得解脱」）
3. 以嗔心殴打抢劫比丘、沙弥，或令他人如此
4. 造五无间罪中任一
5. 产生邪见，不承认因果

**大臣五定罪**：上述前四条 + 用武力摧毁城市、地区、村落、附属地方。

**平凡人八定罪**：
1. 对智慧未成熟者宣说大乘空性，令其生畏（"见到一个刚来的人就说：来，过来，我给你讲一个《中观根本慧论》"——索达吉堪布举此例）
2. 令他人退失大乘发心趋入小乘
3. 令他人舍别解脱戒而趋入大乘
4. 声称声缘道不能断烦恼（自持或令他持此见）
5. 以嫉妒心赞自毁他
6. 为名闻利养而宣称自己证悟空性
7. 令比丘受惩罚并暗收贿赂
8. 把修行者的受用送给求学者，促使对方舍修行

再加《大密善巧方便经》舍愿菩提心、《宝积经》舍行菩提心，共二十根本堕。

此为利根者学处。《集学论》中根四堕：自赞毁他、悭吝财法、嗔不受悔、谤乱正法；《教王经》钝根一堕：不舍愿菩提心。

### 菩萨戒的特点

- **以发心为戒体**：「不能仅根据身语行为判定善恶业，应根据内心动机——动机是贪嗔痴即是恶业，无贪嗔痴即是善业。」
- **失戒条件宽**：无著系犯根本戒后可最多重受三次；龙树系可重受多次。为此宁玛派每日诵《菩萨戒仪轨》《释尊仪轨》《普贤行愿品》以「恢复并增上戒体」。
- **菩萨戒归纳为三聚**：禁恶行戒（上述二十堕）、摄善法戒（行六度万行）、饶益有情戒（无私心利益众生）。

《入行论》教证：「犹如最胜冶金料，垢身得此将转成，无价之宝佛陀身，故应坚持菩提心。」「故如所立誓，我当恭敬行，今后若不勉，定当趋下流。」

菩萨戒的进一步学处可参看「入行论」主题下的菩萨学处专论。

## 四、庚二·辛三 忏悔失毁密宗誓言罪（第77课）

### 密乘戒的极端严重性

阿底峡尊者自白：「我受别解脱戒后戒律一尘不染，受菩萨戒后偶尔出现过一两次过失，而进入密宗金刚乘后毁犯的学处则不计其数。」连尊者尚且如此，何况无密乘戒基础的人。

《誓言庄严续》关键规则：犯密乘根本戒后「最好不要让罪业过夜」；过一天、一月、一年都可忏悔；**过三年即无清净机会**，「即使在上师面前忏悔，上师也不能接受忏悔，否则师徒都会下金刚地狱」。「文革」期间藏地有些人批判上师，事后想忏悔已过时效——此生再无可能清净密乘戒，唯有念金刚萨埵心咒清净部分罪业。

《功德藏》：「入密士夫之去处，恶趣佛外无三处。」如蛇入竹筒非上即下：守护誓言迅速成就佛果，不守护则堕金刚地狱。

### 十四根本戒（诸无上续部共许）

| # | 戒条 | 核心界限 |
|---|------|---------|
| 1 | 詈骂或毁谤金刚上师 | 凡在其前得灌顶/密续/窍诀三恩之一即为"三殊胜上师"；以语言、身体、或内心认定"从此断绝"即犯；不认可与否无关 |
| 2 | 违越佛教言 | 具二条件：违别解脱/菩萨/密乘任一戒 + 有轻毁心（"违犯就违犯，没什么"的态度） |
| 3 | 损害金刚道友 | 对"密金刚道友"（同一上师摄受/同一坛城得灌顶者）以语言、身体诽谤殴打——"金刚内战" |
| 4 | 舍弃菩提心 | 对小蚂蚁以上任一众生起"不度化"或"让他堕恶趣"之心即犯 |
| 5 | 故意出精 | 以邪见或轻毁心对待智慧/秘密灌顶之誓言物；续部七种开许以外 |
| 6 | 诋毁他宗 | 他宗含显宗及外道；不得执密宗至上而贬下 |
| 7 | 对非法器宣说极密之法 | 三条件：知对方会生邪见 + 故意讲/示 + 对方果生邪见 |
| 8 | 视五蕴为痛苦 | 以轻毁心折磨身体；"名言现相"上受八关斋戒不犯，须以净见量看五蕴本体为五部如来 |
| 9 | 怀疑自性清净之法 | 对基道果诸法自性清净、如来藏光明显现生颠倒怀疑 |
| 10 | 本有能力而不降伏十大应诛 | 毁灭佛教者、摧残三宝者、劫夺僧财者、谩骂大乘者、坑害上师者、挑拨金刚兄弟者、障难修行者、绝无慈悲者、背弃誓戒者、颠倒业果者 |
| 11 | 以分别寻思视离戏法性为有相 | 凡夫分别念不能衡量远离四边八戏之大空性；若以因明安立法性为单空则犯 |
| 12 | 舍弃堪为法器之弟子 | 具清净信/欲乐信/胜解信之求法者，上师不但不传反而打骂驱赶 |
| 13 | 应时亦不享用誓言物 | 铃杵、手鼓、托巴等外誓言物；五肉五甘露等内誓言物；会供金刚歌舞——若有轻毁心则犯 |
| 14 | 舍弃智慧女性 | 以"所有女众"为对境起轻毁心并说出诋毁之辞；说个别凡夫女不好不犯 |

### 八粗支戒

1. 依止不合格的明妃
2. 会供时进行争吵
3. 依自力从不符合续部所说法相的明妃体内摄取甘露
4. 对堪为法器者不宣说密法
5. 对以信心求法者宣说他法
6. 于不信密宗者中住七日
7. 无有证悟的智慧而自诩为密咒士
8. 对非法器宣说密法

粗支戒犯后「通过会供等方式都有清净的机会」，不像根本戒有三年时效。

### 毁誓的典型场景与警示

- **嗔上师或产生邪见**：现代多见——师徒关系被视为世俗合作，意见不合即"你走你的路、我走我的路"。此念现前即破戒，与是否"认可"无关。
- **泄密**：三条件齐具方犯；旁人偶尔看到法本、听到一两咒语不构成泄密。
- **于空性生怖畏**：依凭《大幻化网》的一因文字理等理证可避免此失。
- **金刚内战**：大德早逝、密法隐没、世间灾难等现象之一因。

### 上师的六种与"三殊胜上师"

上师有六种：忏悔对境、共同显宗法、皈依师、灌顶、传续部、赐窍诀。其中「灌顶 + 传续部 + 赐窍诀」三者具一即是密宗金刚上师，为第一根本戒严厉对境。

## 五、庚三 忏悔未认识之自性罪（第78–79课）

未受戒者不犯佛制罪，但仍感自性罪果报。本颂重点列举"非梵行、饮酒等"。

### 非梵行（第78课）

对**未受戒在家人**：戒律中只遮邪淫，不遮与己妻行淫。但龙树菩萨《宝鬘论》：「贪嗔痴及彼，所生业不善。」以贪心行淫仍是"轮回胜义不善业"之因。细分：

- **邪淫四非**：非处（非生殖器官）、非器（配偶以外）、非时（斋戒日等）、非道（非正道）。
- **现代警示**：「2009年中纪委官员宣称，在查处的贪污腐败大案中，95%以上的贪官都有女人问题。」

对**梵净行居士**：「宁可燃铁片蒙蔽自双目，不可失正念眼看女人面」（续部语）。一切非梵行包括贪心眼看异性均犯。

**公案**：摩邓女贪执阿难，佛令其剃发出家，思维男身不净（眼屎、鼻涕、大小便）而证阿罗汉果——男女互修不净观均有效。阿难生为男身不净，摩邓女观修而证初果。

**公案**：舍卫城少年因贪欲而在母亲面前撬门、持剑杀母，随即造五无间罪，后出家修行、临终前给僧众造浴室，从无间地狱短暂受苦后转生天界——说明"造罪后若能忏悔修善，仍可速脱"。

### 饮酒（第79课）

**关键争议**：饮酒是自性罪还是佛制罪？大部分声闻宗认为是自性罪，少数认为是佛制罪。本疏随大部分声闻宗说。

**理据**：《毗奈耶经》记载世尊禁止释迦族**病人**饮酒，并说「承认我是导师的人，连草尖许的酒也不能喝」。佛制罪对病人一般有开缘，此处无开缘——故属自性罪。

**世亲论师的温和观点**：「在家人如果不是以强烈的贪执而喝少量酒，也无有罪过，就像普通的饮食一样。」但若以猛烈贪心过量饮用，必导致争吵、战争、灾难。

**密宗开许的辨析**：萨绕哈巴、帝洛巴等大成就者行禁行时饮酒，但有"把美酒转变成甘露的能力"；普通密咒师不在开许之列。莲花生大士：「如果密咒师被美酒迷醉，这是堕地狱的象征。」末法五相之一即「饮酒同时讲法者」。

**饮酒三十五种过患**（龙树《大智度论》、华智仁波切《饮酒之过失》）：「不应嗔而嗔，不应笑而笑，不应哭而哭，不应打而打，不应语而语，与狂人无异，夺诸善功德。」

**公案**：山中僧人被女人要求三选一（杀羊、喝酒、行淫），他选"罪过最轻"的喝酒，结果醉后既杀羊又行淫、又妄语，四戒俱破——佛经「酒乃诸罪之根本」之证。

**卖酒过患**：宗喀巴在世时，一个向僧人卖酒的女人身现为人身量大的恶蝎入岩石中，受苦到老鹰用羽毛拂尽岩石方止——卖酒因果与饮酒同样严重。

### 吸烟（第79课）

佛经中未直说不能吸烟，但世尊曾概括：「凡能产生罪业的行为都必须舍弃。」玛吉拉准空行母授记及伏藏品均说吸烟过患极大。

**烟草起源**（喇拉曲智仁波切说）：烟草由魔女经血种子所生，魔女发恶愿「愿人们一旦吸烟就上瘾，生生世世感受痛苦」。故吸烟不仅生贪心，还激怒天龙鬼神。

### 其他自性罪

吃葱蒜（五荤：葱、蒜、韭菜、兴渠、薤）后步入佛教徒行列或经堂亦属罪业。

总原则：「凡是对今生来世有利无害的事，任何人做都无有罪过；凡是对今生来世有害无利的事，或者是以贪嗔痴三毒而做的事，任何人做都是有罪过的。」

## 六、庚四 忏悔未知就犯之佛制罪（第80课前半）

"受戒后不学戒"属此类。皈依戒尚且有学处，何况菩萨戒、密乘戒：

### 皈依戒学处（阿底峡系共同五学处 + 《大圆满前行》九学处）

**共同五学处**（《三戒论》「不为命奖舍三宝，何等要事不寻他，常供令自他皈依，顶礼所去之方佛」）：
1. 为生命/财富/地位/权势之奖赏不舍三宝
2. 即便遇再重要之事，也不在三宝之外寻救护
3. 不间断供养三宝
4. 令自他虔诚皈依三宝
5. 不管去往何方，都要顶礼那个方向的佛陀

**《大圆满前行》九学处**：三所修（皈依佛→对佛像作真佛宝想；皈依法→对一字一句佛经恭敬；皈依僧→对红黄补丁以上僧衣作僧宝想）+ 三所断（不顶礼世间天神；断恼害众生；不与外道为友）+ 三同分（视上师为佛宝；上师教言为法宝；道友为僧宝）。

### 不知而犯的判定

**不知≠不犯**，但罪较轻。「以前我遇到过一个皈依了十五年的老居士，我问她：皈依三宝后有什么要求？她说：我也不知道，反正佛很好，我就皈依了。我试探地问：那基督教这些行善的宗教也可以皈依了？她说：是是，我对基督教也有信心，有时候也去拜一拜——这就是不懂皈依戒的表现。」

**比丘戒各律部条数差异**（末法现象——"连戒条数目都说不清楚"）：

| 律部 | 比丘戒 | 比丘尼戒 |
|------|-------|---------|
| 四分律 | 250 | 348 |
| 五分律 | 251 | 370 |
| 巴利律 | 227 | 311 |
| 一切有部律 | 253 | 358 |
| 十诵律 | 257 | 354 |

各律部本质无别，按哪种律部受戒都可，但受戒后须按本律部学处行持。

### 忏悔方法

「学佛、受戒已经那么长时间了，可是对要守护什么戒条还不清楚，造了这么多罪业，实在太惭愧了！」——以此强烈后悔心念金刚萨埵心咒、《三十五佛忏悔文》。

## 七、四对治力圆满（第80课后半—第81课）

「要想将罪业真正忏悔清净，必须具足四种对治力。」四力缺一不可。

### 四力基本定义

| 对治力 | 藏文/义 | 颂词对应 | 核心修法 |
|--------|---------|---------|---------|
| 现行对治力 | 以善法的力量强行灭罪 | （贯穿前六颂"发露忏悔..."的一切忏悔行为） | 念菩提心、观修本尊、信解大乘、念咒、拜三十五佛、发菩提心灭罪 |
| 厌患对治力 | 追悔心 | 「若无悔心忏不净，昔所造罪如服毒，以大惭畏悔忏罪」 | 以大惭畏/后悔心，如服毒般追悔 |
| 返回对治力 | 戒心、发誓不再犯 | 「后无戒心罪不净，发誓此后遇命难，亦不造作不善业」 | 立誓纵遇命难亦不造不善业 |
| 所依对治力 | 所依对境的加持 | 「阿弥陀佛及佛子，加持净化我相续」 | 皈依上师三宝、发菩提心；依止金刚萨埵、阿弥陀佛 |

### 己一 现行对治力（前文各颂已含）

《入大乘论》（印度坚意菩萨造）：「所作重恶业，能深自悔责，敬信大乘法，拔除诸罪根。」——对大乘佛法产生强烈信心和恭敬心本身就是现行对治力，因「大乘佛法的思想就是无我和利他」。

具体行为：念金刚萨埵心咒、念阿弥陀佛名号、念《极乐愿文》、拜三十五佛、观修金刚萨埵降甘露、以安住见解灭罪（"世俗如幻忏悔，胜义离戏空性中安住"）。

### 己二 厌患对治力（第80课后半）

「若无悔心忏不净，昔所造罪如服毒，以大惭畏悔忏罪。」

**核心比喻**：误服毒药者毒发时身心剧苦，当下极度后悔。对昔造罪业须有同样强烈之**后悔心、畏惧心、惭愧心**。

**引发追悔的思维**：
1. 思果报——堕恶趣感受极痛异熟果
2. 思善根失坏——造罪如"珍宝装饰的黄金器皿清除呕吐物"，愚蠢至极（《亲友书》）
3. 思无欺天眼——《本生传》「纵然未见中造罪，亦如服毒岂安乐，天众以及瑜伽士，清净之眼定现见」

**标准**：「要害怕得茶不思、饭不想，甚至达到疯癫昏厥的程度」——像**未生怨王**那样。未生怨王受提婆达多唆使饿杀父王频婆娑罗王、造五无间罪，后全身长恶疮、夜不能寐，被耆婆医师引至佛前。佛入月爱三摩地放光，疮愈罪净，后成佛教大护法。

**公案**：北齐僧云废止诵戒，被巨人拖至古墓以刀割刺，从此十年如一日精进忏悔，临终瑞相圆寂。《付法藏因缘传》阇夜多尊者对破戒比丘说"只要你听我的话"，幻化火坑令其跳入，火坑变清凉池，示罪业清净。

**妙高禅师**：悬崖打坐时瞌睡落崖，韦陀菩萨救之；起傲慢心被告"二十世不再护法"；后生起惭愧心，韦陀菩萨再次接救——「一念惭愧心，已经远远超过二十世」。

### 己三 返回对治力（第81课前半）

「后无戒心罪不净，发誓此后遇命难，亦不造作不善业。」

即使已具追悔心，若「日后无有稳固的戒心，则罪业也不能得以清净」。须在金刚萨埵/阿弥陀佛等所依境前立誓：**从即日直至菩提果，纵遇命难亦不造不善业**。《药师琉璃光七佛本愿功德经》：「乃至命难，亦不造恶。」

**重犯如何处理**：若实在不敢纵遇命难之誓言，起码发"尽量不造罪业"的心；对严重罪业（杀生、摧毁他人善根等）无论如何不再造。重犯者须再次念金刚萨埵、再立誓。

**核心原理**：《入行论》「获断恶之心，说为戒度圆」——断恶之心本身就是持戒度圆满。善恶皆以心为界：「心善地道亦贤善，心恶地道亦恶劣」（宗喀巴大师）。《大般涅槃经》：「若不护心，则不护身口。若护心者，则护身口。」——身语是心的仆人。

有人问："我天天观修本尊，念了好多咒语。您看我的罪业清净没有？"堪布答：「我的肉眼看不出罪业清净没有，但是你自己看一看：我有没有不造罪业的决心？如果有这样的决心，那你的罪业肯定已经清净了。」

### 己四 所依对治力（第81课后半）

「阿弥陀佛及佛子，加持净化我相续。」

自力之外必须依所依境加持。**净土法的所依对治力**主要依阿弥陀佛及眷属。

《观无量寿经》：「称佛名故，于念念中，除八十亿劫生死之罪。」——这不是安慰语，而是诸佛菩萨不可思议加持力的真实展现。

**公案（蕅益大师文集）**：甲商人死，乙埋葬并代送遗物。甲妻怀疑乙谋害而散布谣言，乙到甲坟前哭诉求清白，甲鬼魂附乙身同归。路上乙随口念"阿弥陀佛"，鬼说"你为什么放光吓我"；再念几句，鬼说"你每念一句，心间就发出五色光芒，我不能再靠近你了"。乙由此对佛号生大信心，出家念佛。

### 四力共作时的力量

《金光明经》：「何人千劫中，若造严重罪，一次极力忏，诸罪得清净。」《万善同归集》：「如是百劫中所集诸不善业，以佛法力故善顺思惟，可于一日一时尽能消灭。」

智者与愚者的分野（《弥勒狮吼请问经》）：「无知所造罪，一切当忏悔，智者若忏罪，不与业共住。」愚者视罪为功德——微罪如小铁球沉水；智者造罪即追悔——铁球打成薄片不沉于水。

**那先比丘譬喻**：小石头放水上即沉；百块大石放船上因船力不沉。百余罪业依念佛/忏悔力不堕；少许罪业若无救护方便则直堕。

**释迦牟尼佛往昔因地**：曾转生火车地狱拉火车，见同伴被狱卒铁叉刺苦，对其生极大悲心，引生菩提心，当下清净百劫罪业转生天界——「一念菩提心的威力就是这么大」。

**蕅益大师**：「以如幻根，缘如幻佛，灭如幻罪。」罪业体性如梦如幻、不成实有、因缘所生有为法，故可净。「罪业虽无德，忏净乃罪德」。

## 八、关键对治法门

### 金刚萨埵百字明（与《大圆满前行》金刚萨埵一章呼应）

「金刚萨埵心咒是忏悔罪业的最胜咒语」。唐代不空译《金刚顶瑜伽金刚萨埵五秘密修行念诵仪轨》：「金刚萨埵三摩地，名为一切诸佛法，此法能成诸佛道，若离此更无有佛。」

法王如意宝发愿：「凡是发愿往生极乐世界的人，一定要念完四十万遍金刚萨埵心咒，否则很可能因为障碍而无法往生。」

往生二大障碍：**舍法罪**（"毁谤上师道友、评论此法殊胜彼法不殊胜"常构成直接或间接诽谤正法）与**五无间罪**。

### 念阿弥陀佛名号与《极乐愿文》

净土法门自身即是最强忏悔——以名号功德、阿弥陀佛愿力摄持即可忏净。「称佛名故，于念念中，除八十亿劫生死之罪。」

### 拜三十五佛忏

宗喀巴大师每日不断念《三十五佛忏悔文》，索达吉堪布亦依此发愿「现在即使再忙，每天都不间断地念一遍」。

### 菩萨戒十八堕的重受仪轨

宁玛自宗随龙树菩萨体系，犯根本戒后可多次重受。每日诵《菩萨戒仪轨》《释尊仪轨》《普贤行愿品》以恢复并增上戒体。

### 实修次第（极乐法会传统）

1. 诵《三十五佛忏悔文》一遍
2. 诵《总忏悔文》一遍
3. 念《极乐愿文》「父母为主吾等众……加持净化我相续」一边顶礼一边观想四种对治力
4. 坐下念净戒咒七遍：「嗡阿摩嘎协拉 桑巴Ra 桑巴Ra 巴Ra巴Ra 玛哈协达萨多 巴玛波波科达波匝 达Ra达Ra 萨曼达 阿瓦罗格得吽帕的索哈」
5. 再念「具足无垢戒，受持清净戒，持无慢心戒，愿戒度圆满」
6. 再念「断除一切损害他，梦中亦不生恶念，恒时不离菩提心，愿戒圆满诸吉祥」
7. 最后观想皈依境诸尊众安慰"你们的罪业已经清净"，身体发光照射，自他一切罪障清净无余

### 金刚萨埵法会修法

观想金刚萨埵安住前空，自己在其前发露忏悔无始以来一切罪业并立誓不再造；一边念咒，一边观金刚萨埵身降甘露洗罪障，或观想咒语本身。

### 十不善业并忏

除了舍法罪、五无间罪，还须忏**十不善业**：杀生、偷盗、邪淫、妄语、离间语、粗语、绮语、贪心、害心、邪见。

## 九、常见误区与对治

| 误区 | 正见 |
|------|------|
| "我得过密宗灌顶，什么都不要紧" | 密乘戒是最难守的，连灌顶誓言都不清净如何救护下层戒律？三戒须分开护持 |
| "空性见下没有能忏所忏" | 胜义中无忏，但"在世俗如幻的境界中，能忏和所悔都是存在的"；不可以空性为借口轻视忏悔 |
| "净土宗最殊胜，不用学其他" | 善导大师：「各宗同以诸佛为师，以佛法为母而养成，不得赞自毁他，否则即是破坏法眼，法眼既灭，如何能生净土？」 |
| "一门深入，其他不学即是夹杂" | 印光大师：「一门深入，尽废余门，唯打七时方可」，平时非不能学其他宗派 |
| "学密宗容易着魔，净土最安全" | 省庵大师（净土宗十一祖）《念佛着魔辩》：「参禅、念佛，论其难易，固有自力他力之分，若论魔事，二俱不免」 |
| "学密者不能受八关斋戒" | 以轻毁心折磨身体才犯第八根本戒；名言现相行持苦行无过 |
| "我坐火车旁边的人看了密法本，犯泄密了" | 泄密须具足：知对方会生邪见 + 故意讲示 + 对方果生邪见，三条件具足方犯 |
| "饮酒是密宗开许的" | 续部开许仅限具"把美酒转为甘露能力"的大成就者；普通密咒师不在其列 |
| "不知者不为罪" | 世间说法；自性罪不论知与不知均感果（《因缘品》） |
| "念几句佛号就能净无始罪，没那么简单吧" | 铁球打成薄片不沉水；如幻根缘如幻佛灭如幻罪——罪非实有，可净之理成立 |

## 十、本章在《极乐愿文》结构中的位置

本章是"丁二（修七支供）—戊三（对治贪嗔之忏悔支）—己一（现行对治力）"之延续与完结，并衔接 己二、己三、己四 构成四对治力之圆满。

上接 05 文档（严重罪业、舍法罪、谤法）讲忏悔自性罪之重罪；
本章 06 讲忏悔佛制罪三类 + 忏悔未认识自性罪 + 忏悔未知就犯之佛制罪，然后完成四对治力；
下接 07 文档（随喜支——戊四对治嫉妒之随喜支）。

忏悔支的终极目的是**净除往生净土的障碍**（舍法罪、五无间罪、十不善业），以期临终现见阿弥陀佛、顺利往生极乐世界。「如果你的相续中全是罪业，阿弥陀佛也没办法救你」——忏悔之重要性于此尤显。
