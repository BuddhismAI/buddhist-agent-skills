# 有无生因（破果法之推理）

## 方法定义

"有无生因"（藏：yod med skye ba'i gtan tshigs）是中观五大因之一，专门用于遮破**果法**（所生之法）的自性。

**基本结构：**

- **有之法不应生**：若果法已经存在，则不需要再生（已有之故），否则有无穷生、无义生之过；
- **无之法也不应生**：若果法不存在，则是无实法（如石女儿），无任何因缘能令其产生；
- **亦有亦无之法也不应生**：有与无相违，不可能在同一本体上并存，此法不可能有产生。

与"金刚屑因"（破因法/缘）相对，有无生因专门从**果法**的角度遮破生起。

## 与金刚屑因的分工

| | 金刚屑因 | 有无生因 |
|------|----------|----------|
| 破的角度 | 破因（缘）的自性 | 破果（所生之法）的自性 |
| 提问方式 | 生起的方式是什么（自/他/共/无因）？ | 果在因中是否存在？ |
| 核心应用 | 品01、品07（总破四生） | 品01（颂06-07）、品07（别破·破生） |
| 来源论典 | 《入中论》对四生的广破 | 《入中论》有无生因段落 |

## 完整论式（品07别破·破生，第29课）

**所破命题**：有为法的生相（法相）是自性成立的。

**论式**：

```
有法：有为法的生
立宗：（胜义中）不应生起
因：若有不应生，若无不应生，若亦有亦无不应生（有无生因）
喻：如石女儿（无）、如已成立之柱子（有）
```

**逐步展开（颂词）**：

> 有法不应生，无亦不应生，有无亦不生，此义先已说。

- **有之法不应生**：已存在的生相若再生，则有无义生和无穷生之过（生完还要再生，犹如已修成的房子还要继续修建）。
- **无之法不应生**：不存在的生相如石女儿，千百万因缘聚合也无法令其产生。
- **亦有亦无之法不应生**：有与无本质相违（水与火不共存），无法在同一本体上并立，此法连概念都不能成立，更无产生。

**"此义先已说"** — 引品01颂："果先于缘中，有无俱不可"——同一逻辑，品01已从因缘角度用过，品07从法相角度再用。

## 在品01中的预先出现

品01已经出现有无生因的基本思路（第5课，颂06-07）：

> 果先于缘中，有无俱不可，先无为谁缘？先有何用缘？

此处问的是果法在因缘中是有还是无：
- 若果在缘中有 → 缘无义，因为果已存在，不需要缘
- 若果在缘中无 → 缘观待无实的果来成立，不合理

这是有无生因的初次应用，还与"互相观待而破"融合（见 `互相观待而破.md`）。品07别破时，此因被独立、正式地命名为"有无生因"，作为单独一套推理出现。

## 在品20（观因果品）中的多重应用（第70-74课）

品20是有无生因在全论中应用最密集的品，从多角度展开：

**因法维度**（子五·丑一，第73课）：因中空果→非因也无果→不能生；因中不空果→已有→不须生。

**果法维度**（子五·丑二，第73-74课）：果不空→有自性→不变→不生不灭；果空→如虚空→不生不灭。

**果本体总破**（壬二，第74课）："若果定有性，因为何所生？若果定无性，因为何所生？"——有自性的果不须生、无自性的果如石女儿不可生。

品20将有无生因与品01首次应用、品07独立出场构成从因果关系角度的三级应用：品01是总纲中的局部出现，品07是单独成因，品20是多维度系统展开。

## 与品15（观有无品）的关联

品15（观有无品，第49-51课）是专门以"有"与"无"本身为破斥对象的品，届时有无生因会在更宏观的框架中展开。

## 方法的哲学意义

有无生因的根本洞察是：

**"产生"这个概念本身预设了从无到有的变化——而这个变化在任何时态中都无法被安立。**

- 若果已"有"，则没有"从无到有"这回事——生是多余的；
- 若果是"无"，则根本没有一个"果法"在那里被生——生的对象不存在；
- "亦有亦无"是逻辑矛盾，无法安立。

如此，所谓"生起"，只是名言假立的一种迁流显现，不是真正"有某物从某处来到这里"。《入行论·智慧品》中的"犹如芭蕉树，剥析无所有"正是同一洞察的比喻表达。
