# 破有外境派之识续——相识等量深析与外道诸派

## 在全论中的位置

子二·破实一之识 → 丑一·破有外境派之识的后半部分（第61-70课）。先深入破析相识等量中的精细问题（旋火轮辩论等），然后转向破外道各派的实一之识，最后以"共同实一之识不容有"收结整个丑一。

## 一、巳三·破相识等量之观点的深层辩论（第61-65课）

### 旋火轮辩论

对方引旋火轮（旋转火烬形成的圆圈）作为反例：旋火轮显现为一个完整的圆，但实际是多个刹那次第产生的——证明次第取多相的识可以显现为"同时见"。

**麦彭仁波切的驳斥**：

1. **见与忆的对境相违**（酉一）：现量只取当下刹那的对境，不取过去的对境。将前后刹那"结合"起来见到旋火轮，必须是忆念（分别念）的作用，而不是现量。

   > **如是结诸际，由忆念为之，非取过去境，故非依现见。**

2. **若是结合则不应明现**（酉二）：旋火轮的"圆"是错乱显现（相似现量），不是真实现量。过去刹那的火点已灭，不可能明现。

   > **成彼对境者，已灭故非明，为此显现轮，不应成明现。**

### 画面类比的驳斥

对方又引"同时见画面"作反例。麦彭仁波切指出：画面上白色的不同部分（上、中、边）仍有差异，能缘识因此也是种种，不是真正的"一"。

### 关于实一之识的总结

无论有部的无相观点、经部的有相三种观点中的哪一种，识都不能成立为"实一"：
- 相识等量：识有多种行相 → 非一
- 异相一识：识只有一相但外境多相 → 识不是正量
- 相识各一：与现量矛盾

## 二、卯二·破外道之观点（第65-70课）

### 结构

| 科判 | 外道 | 核心主张 | 破析要点 |
|------|------|---------|---------|
| 辰一·总破明智派 | 顺世派等 | 识从四大中产生 | 无情法不能产生明知之识 |
| 辰二·别破 | | | |
| 巳一·破胜者派与伺察派 | 耆那教、吠陀派 | 胜我/梵天是常有实一之识 | 常法不可能有不同认知 |
| 巳二·破顺世派 | 唯物论 | 识是四大聚合的产物 | 无因生过失+无分微尘已破 |
| 巳三·破数论派 | Sāṃkhya | 神我（puruṣa）是常有明知 | 多果→因非一（同破常法） |
| 巳四·破密行派 | Pāśupata | 遍一切的胜我 | 异方相联故非一 |

### 破数论派的精细论证（午一·午二）

数论派认为主物（pradhāna）产生万法，神我旁观。麦彭仁波切的破析：
- 若神我常有不变，为何有时知、有时不知？
- 若主物变化而神我不变，主物的变化由谁认知？
- 若神我有不同的认知活动 → 不是真正的"一"

### 破密行派（巳四）

密行派的"遍一切之胜我"以同样的"异方相联"论证破析——与破虚空等总能遍的逻辑完全相同。

## 三、寅二·共同实一之识不容有——丑一的收结（第70课）

> **此二之摄义。**

不论外境承认为实一（卯一已破），还是识承认为实一（卯一+卯二已破），二者皆不成立。耽著外境为实一 → 已由破微尘瓦解；耽著识为实一 → 已由破三种识境关系和外道识论瓦解。

**共同结论**：承认外境存在的宗派（有部、经部、外道），其所许的"实一之识"无一成立。

## 与下一单元的衔接

有外境派的实一全部不成立后，转向**否认外境存在的唯识宗**——他们虽然破了外境，但将心识（依他起）执为实有。这是"破实一"的最高难度，也是静命论师作为"瑜伽行中观"论师的独特贡献：在名言中承认唯识，但在胜义中破唯识。
