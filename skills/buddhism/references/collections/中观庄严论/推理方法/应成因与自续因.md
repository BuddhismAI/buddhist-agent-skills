# 应成因与自续因

## 核心要点

应成因（prāsaṅgika-hetu）与自续因（svātantrika-hetu）是理解中观两大宗派推理方式的关键。但麦彭仁波切的根本判断是：**这一差别只是支分上的区别，并非区分应成派与自续派的根本标准。**（第29课）根本标准是所抉择的胜义深度——相似胜义还是真实胜义。理解这两种推理方式，必须从这个更大的框架出发，才能不陷入支分的争论。

---

## 一、两种推理方式的结构对比

```
自续因                          应成因
─────────────────────           ─────────────────────
三相具足的量成推理              以他宗自己承认的前提
                                推出他宗不愿承认的结论

有法 ─── 立宗 ─── 因           无需同品喻、异品喻
         │
    同品喻 + 异品喻

自宗有承认、有所立              自宗无有任何承认
以正量建立自己的主张            "以子之矛攻子之盾"
```

### 自续因的具体结构

自续因是标准的三相因明推理，要求：

1. **宗法**：因在有法上成立（对自他双方均以量可成立）
2. **同品遍**：凡具足此因者皆具足所立（同品喻成立）
3. **异品遍**：凡不具足所立者皆不具足此因（异品遍成立）

**《中观庄严论》第一颂词即是典型的自续因：**

> 自他所说法，此等真实中，离一及多故，无性如影像。

- **有法**：自他所说的一切万法
- **立宗**：无有自性（宗）
- **能立因**：离一及多故（因）
- **比喻**：如影像（喻）

这一推理以量成立：即便外道或小乘不直接承认"离一多因"，他们承认各自的法（如外道承认多种果法，小乘承认无分微尘），则逻辑上已不得不承认其法远离一体与多体——因此此因对他宗而言同样可以成立。（第36课详析）

自续派推理的前提是自宗有明确的立场：胜义中一切法无自性，世俗中自相成立，二谛各居其位，以无垢二量分别建立。正是**分开耽著二谛**的这一立场，使得自续因的"量成立"有了稳固的地基。（第28课）

### 应成因的具体结构

应成因不要求完整三相，也不需要比喻。其结构是：

**取对方自己承认的前提 → 推出对方不得不接受的矛盾结论**

中观应成派的自宗立场是"远离四边戏论、无有一切承认"。正因无有任何需要建立的法，才能纯粹以他宗观点为武器，攻破他宗。

**应成因的典型运作（以清辨论师与月称论师之争为例）：**

若清辨论师承认"名言中自相成立"，月称论师即发出应成推论：
- 你的名言中自相成立 → 此自相能否经得起胜义量的观察？
- 若能经得起 → 则已成为成实，有情众生便永无解脱可能
- 若不能经得起 → 则与中观应成派"名言中也不承认自相成立"无异
- 无论哪条路，自续派的承认都陷入困境

龙猛菩萨《回诤论》所言，正是应成因的精髓：

> 若我宗有者，我则是有过，我宗无物故，如是不得过。（第29课引）

> 设若现量有，则容有破立。无破亦无立，是故我无过。（第29课引）

**另一颂词揭示应成因的完整逻辑：**

> 若有不空法，则应有空法。实无不空法，何得有空法？（第29课引）

此处逻辑：若对方承认有一种"不空"之法存在，则说"空法"也是可以的。但无有任何"不空之法"——这是以对方自身承认的前提（存在不空法）为入口，推翻对方对"空法"的安立。自宗不需要另立任何观点。

## 二、因明的应成因 vs. 中观的应成因

第29课专门指出这一极易混淆的区分，是理解本论的关键：

| 维度 | 因明所讲的应成因 | 中观的应成因 |
|------|----------------|------------|
| 推理严密性 | 完全合理的标准推理，三相具足 | 不一定是严密的因明推理 |
| 归属 | **可以包括在中观的自续因之中** | 中观应成派的不共推理方式 |
| 同品喻/异品喻 | 具足 | 不需要 |
| 自宗立场 | 自方有承认，有所立 | 自方无承认，无所立 |
| 功能 | 建立自宗观点，同时驳斥他宗 | 仅以他宗推翻他宗，不另立自宗 |

> 因明所讲的自续因和应成因全部可以包括在中观的自续因中。（第29课）

这一判断至关重要。因明的辩论工具（包括因明的"应成因"）在逻辑上都是量成推理——自方有所立、他方不得不承认。这类推理在中观框架下，属于**自续因**的范畴，因为推理者本身有一个立场在支撑推理。

只有中观应成派的应成因才是真正意义上"以子之矛攻子之盾"——推理者自身无有任何立场和承认，仅以他宗的承认为燃料，点燃他宗自身的矛盾。

第36课进一步说明：在辩论场中，中观应成派不需要说"因为什么，所以什么"，只需要将对方的观点作为推理的因，以这种方式来断除他宗观点。此即月称论师经常对清辨论师说的：你的论式没有讲比喻、没有讲所立（指月称站在应成立场时不需要如此）。

## 三、月称论师与清辨论师的历史争论

这场争论是理解两种推理方式何以成为宗派判准的历史线索。（第29课）

**起点：** 龙猛菩萨在《中论》中抉择了远离四边八戏的空性。

**佛护论师**：以应成因为龙猛菩萨作注解——用他宗观点攻破他宗，无需另立比喻和所立。

**清辨论师**：批评佛护论师的做法不合格——认为应该以具足三相的自续因来建立龙猛菩萨的观点。清辨论师在《般若灯论》中明确以自续因方式重新阐释中观。其背后的预设是：胜义中无自性，世俗中自相成立，二谛各有量为证，推理者有一个稳定的立场可以支撑量成推理。

**月称论师**：在《显句论》中驳斥清辨论师，站在佛护论师的立场建立了中观应成派的不共见解。月称认为：若在推理中承认"名言中自相成立"，实际上已经承认了一种成实，这是自续派的根本缺陷。应成派不需要自宗立场，因为真实胜义中根本无有可以承认的法。

**麦彭仁波切的结论**（第29课）：从究竟来讲，静命论师与月称菩萨的意趣无二无别，因为静命论师在《中观庄严论》最终也破除了分开耽著二谛的执著，汇归于离四边戏论的真实胜义。暂时的差别在于**调化路径**：月称论师直接为利根众生宣说离戏；静命论师分开二谛，先抉择相似胜义，最终破除此执著而汇归真实胜义。

## 四、与中观两派法相的关系

### 麦彭仁波切的根本判准

第29课明确列出各论师对两派差别的不同安立，并给出总结：

**格鲁派的判准**（宗喀巴大师等）：名言中是否承认自相成立。
- 应成派：名言中不承认自相成立
- 自续派：名言中承认自相成立

**萨迦派的判准**（果仁巴等）：抉择胜义时是否运用自续因或应成因。
- 应成派：主要运用应成因
- 自续派：主要运用自续因

**麦彭仁波切的总结**：上述所有判准都只是**支分上的区别**，完全可以归属于以下根本判准之中：

| | 自续派 | 应成派 |
|--|--------|--------|
| **根本法相** | 着重讲解**相似胜义**（单空，具有承认的执著相） | 着重阐述**真实胜义**（离四边戏论，无有一切承认） |
| 推理方式 | 自续因（量成三相推理） | 应成因（以他宗推翻他宗） |
| 对所破的鉴别 | 加"胜义中"等鉴别 | 不加鉴别，直接破四边 |
| 二谛关系 | 分开承认，各居其位 | 入定时二谛一味，后得时方作分别 |
| 名言立场 | 承认名言中自相成立 | 名言中也不承认自相成立 |

"是否使用应成因"与"是否承认名言自相成立"，都是"着重抉择真实胜义还是相似胜义"这一根本差别的**外在体现**，而非根本差别本身。

### 推理方式何以连接于胜义深度

自续因的使用内在要求推理者**有一个立场**——"我在这个立场上以量推论"。这个立场就是：二谛分开，各有量成，胜义中无自性，世俗中自相成立。这恰恰是相似胜义（单空）的立场。

应成因的使用内在要求推理者**无有任何立场**——"我无承认，以你的承认来破你"。这种"无有承认"恰恰对应真实胜义——离一切戏论，无有破立。（第29课）

因此，两种推理方式并非独立存在的技术选项，而是从各自所抉择的胜义深度中**自然流出**的推理姿态。

## 五、静命论师在本论中对自续因的应用

### 本论开篇颂词（第36课）

本论所运用的推理典型是离一多因，第36课专门分析它究竟属于应成因还是自续因，结论是**两者均可**：

从**应成因**角度：外道、小乘、唯识宗虽未直接承认"离一多因"，但他们承认各自的有法（外道承认多果，小乘承认无分微尘的总法），依逻辑推出其不得不承认该法远离一与多。故已满足应成因"以他宗承认为依据"的要件。

从**自续因**角度：遮破性论式中有法可以不在外境成立，只需在分别心前通过遣余成立即可。外道心中的"常有自在天"虽不存在于外境，但在其分别念前可以显现，故宗法可以成立；加上同品遍、异品遍的分析，三相可以具足。

麦彭仁波切引莲花戒论师《中观显现论》的结论：无论以应成因还是自续因安立，均合理，此二者没有任何矛盾。（第36课）

### 自续因的必要性（第28、29课）

静命论师使用自续因，根植于中观自续派的整体教化策略：

> 因而运用各自的能立——自续因自然合情合理。（第28课）

在教化众生的次第中，众生相续中的实有执著极为顽固，需要以量成立的推理循序渐进地破除。以具足三相的自续推理建立"胜义中无自性"，既不诽谤名言的显现，又能有效对治实有执著。这就是为何静命论师选择以自续因为主要工具的原因。

### 应成派的应成因在本论中的位置（第28、29课）

本论究竟要义是离四边戏论的真实胜义。第28课讲到，中观应成派抉择见解时：

> 对四边无需加上成实、胜义等二谛各自的鉴别，一并推翻四边的耽著对境。

这正是应成因的运作方式——不加鉴别，直接破四边，推翻一切承认。本论最终与应成派归于同一究竟意趣（第29课），故本论同时涵盖了自续因（暂时路径）与应成派推理精神（究竟归宿）。

## 附：常见误区

**误区一**：认为应成派就是"只用应成因"，自续派就是"只用自续因"。

修正：麦彭仁波切明确指出这只是支分区别。静命论师在本论第36课的推理中，同一个离一多因既可作应成因解，也可作自续因解。两派在推理工具的使用上有重叠，差别在所抉择的胜义深度。

**误区二**：认为格鲁派判准（名言中是否承认自相成立）是最根本的判准。

修正：第29课中麦彭仁波切以婉转语气指出此观点属于"支分类别"，并驳斥了以此判准划分两派而导致的内在矛盾。

**误区三**：将因明中的"应成因"混同于中观的"应成因"。

修正：因明的应成因是量成推理，属于中观自续因的范畴。中观的应成因不一定是严密的因明推理，其核心是"自宗无有承认，以他宗承认为武器"，这是因明框架所无法容纳的。（第29课）
