# 现基递进破除

## 概述

"现基递进破除"不是某一条单独的因明论式，而是《中观庄严论》所说支分中揭示的一个**宏观论证结构**——各宗派都在寻找万法显现的"基础"（现基），而从外道到中观应成派，每一层宗派都是通过发现前一层"现基"的不合理并加以破除来实现见解的提升。

这一模式在第8-10课的宗派比较中清晰呈现，是理解全论论证方向的重要线索。

## 推理结构

### 核心命题

每个宗派（中观应成派以外）都认为：**万法显现必须有一个真实存在的基础（现基）**。如果这个基础不存在，万法显现就如同"无毛线的氆氇""空中鲜花"般不可能。

中观应成派是唯一彻底拔除所有现基的宗派。

### 层层破除的阶梯

```
第1层：外道
  现基 = 常有实法（主物/神我/天尊/四大）
  破除 = 遮破常有即一并破除
    ↓
第2层：有部/经部
  现基 = 无分微尘 + 无分刹那
  破除 = 以六尘环绕等理证破微尘之实一
    ↓
第3层：唯识宗（第71-75课详细破析）
  现基 = 自明自知的依他起心识
  破除 = 以离一多因分析识之实一——真相唯识三派皆破、假相唯识名言断绝
    ↓
第4层：中观自续派
  现基 = 显现分实有
  破除 = （由应成派超越——仅相似胜义非究竟）
    ↓
第5层：中观应成派
  现基 = 无——一切现基全部拔除
  → 万法无自性，如幻如梦，缘起而显现
```

### 各层的"无毛线则无氆氇"论证

每个宗派都用类似逻辑为自己的现基辩护：

| 宗派 | 论证 |
|------|------|
| 有部/经部 | "无分微尘若不存在→粗法失去基础→如无毛线的氆氇"（第8课） |
| 唯识宗 | "依他起心识若不存在→万法显现如空中鲜花"（第9课） |
| 中观自续派 | 显现分若不存在→名言世俗无从安立 |

中观应成派的回应：缘起本身就够了——不需要"实有的基础"来支撑显现。

## 方法论意义

### 对学习《中观庄严论》的指引

本论按照这个阶梯展开论证——先破外道的实执，再破有部经部的极微执，再破唯识的心识执。理解这个递进结构，就能把握全论论证的推进方向。

### 对修行的意义

第9课堪布特别指出："中观应成派认为，需要将所有的现基、所有的根源一并拔除。对这个问题，大家不应该仅仅在口头上说一说，而应该真正从内心当中认识到这一点。"

检查自相续：你的见解实际上停留在哪一层？
- 如果你认为"万法需要有一个最终的物质基础"→有部/经部层面
- 如果你认为"万法是心的显现，心必须实有"→唯识层面
- 如果你认为"虽然空，但显现分总得存在"→自续派层面
- 如果你真正通达"缘起即空，空即缘起"→应成派层面

## 与其他推理方法的关系

| 推理方法 | 关系 |
|---------|------|
| 离一多因 | 现基递进破除的各层破析中，离一多因是核心的具体工具——用来破每一层的"实一" |
| 金刚屑因 | 从因的角度辅助破各层的现基 |
| 缘起因 | 中观应成派不需现基的正面论证——一切法缘起故无自性 |

## 本文性质说明

此推理模式是从课程的宗派比较叙述中归纳的**结构性洞见**，并非《中观庄严论》颂文中的明确论式。它更像是理解全论论证方向的"地图"。


## 在第50-53课中的具体展开

第50-53课实际展开了第2-3层的递进破除：

### 第2层的具体执行（第50-52课）

破有部/经部的无分微尘——三种组合方式逐一破析（粘合、环绕、无间住），两个核心论证（若无分则粗法不成、若有分则微尘不成），然后级联到破一切多法（十八界全部瓦解）。

关键洞见（第52课）：微尘不成→五根五境不成→五识不成→意根不成→第六意识不成→受想行不成→不相应行不成。**所取不成则能取亦破**——从外境的基础（微尘）被拔除，自然过渡到心识也不成。

### 从第2层到第3层的过渡（第52-53课）

第52课末明确了过渡逻辑：只要对一切万法的来源——微尘剖析为无有自性，其他的一切法也就自然而然无法成立。但接下来不是直接跳到唯识层面，而是先处理有部/经部**在心识层面**的实执——破实一之识（第53课起）。

这说明现基递进破除并非简单的先破物质→再破心识，而是在每一层宗派上，**先破其物质执、再破其心识执**，然后再进入上一层宗派。第50-55课体现的就是在有部/经部这一层上的完整论证：先破微尘（物质面），再破识（心识面）。

## 在第95课中的正面总结

第95课"辛二·现基必为实空"是对整个现基递进破除的**正面收束**。颂词"故谓俗无因，非理亦不然，设若此近取，真实请说彼"直接向所有有实宗发出挑战：如果真正存在一个实有的近取因/现基，请指出来。

静命论师的立场：如果无有任何理证妨害而真实存在，中观宗也会承认。但以上述所有理证已证实一切所知均不成立为实有。

龙猛菩萨《成名言论》以密咒（多字组成）、妙药（多元素组成）、魔术（多因缘组合）三喻说明：一切法缘起而现，不存在单一的实有现基。这是以正面比喻呈现"缘起取代现基"的核心思想。
