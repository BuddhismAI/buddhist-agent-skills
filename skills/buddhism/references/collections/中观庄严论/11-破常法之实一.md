# 破常法之实一——离一多因的首次全面应用

## 在全论中的位置

庚二·建其理 → 辛一·建立宗法 → 壬一·建立离实一 → 癸一·破周遍之实一 → 子一·破别能遍之实一 → 丑一·破常法之实一。

这是正文论证的第一个实战区域（第36-46课）。在立根本因和旁述完成方法论准备之后，离一多因开始系统应用于具体的破析对象。**常法**（永恒法）是离一多因的第一个破析目标——因为常法是"实一"的最强候选者，如果连声称永恒不变的法都不能成立为"一"，后续破析无常法就更加顺畅。

## 一、旁述收结：推理的三维分析（第36-37课）

在进入破常法之前，辛二·旁述对根本因（离一多因）完成了三个维度的分析：

### 壬二·分析真因

**1. 应成因还是自续因？**（第36课）

| 争议 | 回应 |
|------|------|
| 对方：不是应成因，因为外道/唯识/小乘不承认"离一多"| 他们承认所遍法（如金瓶），所遍成立则能遍（瓶子）成立；由此不得不承认离一多因 |
| 对方：不是自续因，因为有法不成立，如石女儿 | 虽然常有自在的我在世界上不存在，但在分别心前通过遣余可以成立——外道徒的心中认为存在，这一概念可作为有法 |

结论：**离一多因既可作为应成因，也可作为自续因**。莲花戒论师《中观显现论》："无论依照应成因还是自续因安立都可以。"

**2. 证成义理因还是证成名言因？** → 证成义理因（建立真实含义，不仅仅是名称）

**3. 无遮还是非遮？** → **无遮**（仅遮破实有，不暗示任何存在）

这三个分析为后续所有破析建立了统一的方法论标准。

### 壬三·阐明喻理

"犹如影像"——镜中影像显现但无自性。影像的特征完美对应离一多因的结论：显现（世俗有）+ 无自性（胜义空）。

## 二、破他宗假立之常物（寅一，第38-40课左右）

### 破析策略

"他宗"主要指外道各派所主张的常有实体：自在天（大自在天）、遍入天、梵天、数论派主物、神我等。

### 核心论证

以自在天为例（第36课已提及论证方式）：

1. 自在天被主张为"唯一的一体法"
2. 但自在天的果却是"多种多样的"（它是万物的作者）
3. 果多→因不可能是"一"（今年的果与明年的果来自不同的因状态）
4. 既然不是"一"→按"一与多互绝相违"，也不是真正的"多"（因为"多"以"一"为前提）
5. 因此自在天既非一亦非多→无自性

### 推广到所有外道常法

同样的推理适用于：遍入天、梵天（吠陀派）、数论派的主物和神我、密行派的胜我。只要被主张为"常有且唯一"的法，一旦承认其产生多种果，就自我瓦解。

## 三、破自宗假立之常法（寅二，第41-46课）

### 破析对象

"自宗"指佛教内部各派所承认的常有法——主要是**无为法**：

- 有部宗：虚空、抉择灭、非抉择灭（三种无为法）
- 更广义的：一切被认为"不生不灭"的法

### 核心论证结构（科判展开）

寅二内部有精细的科判层次：

| 科判 | 内容 | 破析逻辑 |
|------|------|---------|
| **卯一·略说能破之理** | 概括：常法必定不是"一"，因为其不变的自性与产生多种作用矛盾 | |
| **卯二·广说彼理** | 详细展开 | |
| 辰一·以前识之境跟随后者不合理而破 | 如果常法的所知相在前后识中相同（跟随后者），则后识不应有新知 | 常法不应有任何变化，但认识它的心识在变 |
| 辰二·以前识之境不随后者不合理而破 | 如果前识的对境不跟随后识 | 则无为法具有刹那性 |
| 巳一·对境无为法有刹那性之过 | | |
| 巳二·若如是承认则有过 | | |
| 午一·若观待缘则成有为法 | 如果无为法观待缘而变化→已经是有为法，不是常法 | |
| 午二·若不观待则成恒有或恒无 | 如果不观待缘→要么永远显现、要么永远不显现，不可能有时现有时不现 | |

### 关键颂词

> **果实渐生故，常皆非一性。若许各果异，失坏彼等常。**

如果常法能渐次产生果实，则其"常"的本性已被动摇——产生前和产生后的状态不同，就不是真正的"一"。如果承认各果之间相异，那就承认了常法的不同时段有不同状态，等于说它不是常有的。

> **说修所生识，所知无为法，彼宗亦非一，与次识系故。**

被认知的无为法，既然与不同时刻的心识（前识、后识）产生关联，就不可能是没有任何变化的"一"。

### 结尾（寅三）

> **非具功用力，许彼立何用？论黄门俊丑，欲者观何益？**

如果常法根本没有任何功用（既不产生果、也不被认知），那承认它存在有什么意义？就像讨论黄门（无性别者）的美丑对欲者没有任何意义一样。

## 四、方法论要点

### 离一多因在破常法中的应用模式

1. 设定所破：某法被主张为"常有的实一"
2. 检验"一"：如果它产生多果，或被多识认知，则不可能是"一"
3. 如果不是"一"："多"也不成立（因为"多"以"一"为前提）
4. 结论：此法离一离多，无自性，如影像

### 名言中的假立"一"不被遮破（第47课强调）

> "离一多因所遮破的，是任何时间、任何地点、任何形象的一种成实唯一，并未破斥名言中假立的'一'。"

假立的"一"只是将多法的积聚取名为"一"，对此中观宗不遮破。如果遮破假立的"一"，就是毁谤名言。

### "桑达瓦"的场合辨析（第47课）

"成实"在不同场合有不同含义：能起作用的法称为成实；以名言量可得到的法叫成实；经胜义量观察破不了的法称为成实。不辨场合则如"菟丝草"——杂乱无章。
