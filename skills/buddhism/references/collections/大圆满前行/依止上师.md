# 解脱利益与依止上师（第73--83课）

## 丙五、解脱利益（第73课前半）

顶礼偈：

> **多成就者智者所摄受，依照上师言教而修行，**
> **解脱胜道无误示于众，无等上师足下我敬礼。**

### 丁一、解脱之定义

"所谓的解脱，就是指脱离轮回这个大苦海，依靠《三主要道论》所讲的出离心、菩提心、无二慧，断除自相续的一切烦恼，远离三界的一切痛苦，最终获得声闻、缘觉、圆满菩提其中任意一种果位。"世间人的所谓"解脱"（出狱、康复等）只是相似的，唯有根除了一切烦恼种子才是真正的解脱。

### 丁二、解脱之分类

**戊一、能获解脱果位之因**：从人身难得开始以四种厌世心调顺相续，再从皈依到正行圆满之间——每一个修法都有各自的功德，这就是解脱之因。四种厌世心是一切修行的基础，否则"修大圆满、大手印等也将统统成为今世之因"。

**戊二、三菩提之果**：声闻、缘觉、圆满菩提三果皆寂静清凉。"尤其是如今我们遇到了大乘佛法，理所应当唯一希求圆满菩提。"

**四法印**——《佛为海龙王说法印经》："诸行无常，一切皆苦，诸法无我，寂灭为乐。"佛对龙王说，若能受持读诵此四法、深解其意，"即与读诵八万四千法藏功德无异"。"寂灭为乐"就是解脱之果。

《正法念处经》："若人远离众恶业，喜行善法心爱乐，此人现世常安乐，必得涅槃解脱果。"

### 三士道次第详解（《前行备忘录》）

《前行备忘录》对三菩提果位的三士道次第给出了极为详细的区分：

**小士道**：依靠的乘是世间乘，发心是求增上生的发心，善根是随福德分或成办三有的业，见解是世间正见。这些善根既没有以出离心摄持，也没有以空性见摄持，又没有以菩提心摄持——无论行持多少，都是小士道善根，修世间正见得到的是人天果报 (《前行备忘录》)。

**中士道**：依靠的乘是佛教之出世间乘，发心是自私自利的发心，善根是小乘随解脱分善根。教法是三藏，证法是三学——律藏教义（戒律学处）属戒学，经藏教义（修不净观等）属定学，论藏教义（观察人我与五蕴六界是一体还是他体，证悟人无我及粗大法我不存在）是慧学。中士道行人在漫长三世、千年等时间里修行，最终获得有余、无余阿罗汉果位。总之，以出离心摄持、以相似空性的证悟摄持而没有以菩提心摄持的善根，都是小乘随解脱分善根 (《前行备忘录》)。

**大士道**：依靠的乘是大乘，发心是菩提心，善根是大乘随解脱分善根——远道的二资粮、近道的生起次第圆满次第、捷径的本来清净直断和任运自成顿超；见解是三大的见解。修行达到究竟时获得显宗所说佛果、密宗所讲双运金刚持果位。总之，以出离心摄持、以证悟二无我的智慧摄持、以菩提心摄持的善根，不管行持多少，都是大乘随解脱分善根 (《前行备忘录》)。

关于各乘的修行时间：远道因相乘历经三大阿僧祇劫；近道金刚乘中事续等三部五部灌顶后以清净誓言守护，修有相无相瑜伽，三世或五世或七世成佛；玛哈、阿努修方便生起次第和智慧圆满次第，一生一世成就金刚持；光明大圆满获有戏无戏等灌顶，实修直断顿超，数年数月成就大虹身 (《前行备忘录》)。

*此后直接转入共同外前行中分量最重的部分——依止上师。*

## 丙六、依止上师

### 丁一、依师之必要（第73课后半）

**核心论点**："一切佛经、续部、论典中，从来没有提过不依止上师而成佛的历史。我们在现实生活中也可以看出，通过自我造就以及有胆有识而生起五道、十地功德的人，一个也没有。"

**三大教证**：

- 《华严经》："十方法界中，所有诸佛法，汝事善知识，一切皆能入。"又云："若欲成就一切智智，应决定求真善知识。"又："于一切善知识，起如来想。""其有修行善知识教，诸佛世尊悉皆欢喜。"
- 《涅槃经》（《大鼓经》）：佛亲言——"阿难莫哀伤，阿难莫哭泣，我于未来时，化为善知识，利益汝等众。"汉文《涅槃经》："我又示现阎浮提中而作博士，为教童蒙令住正法。"
- 《大般涅槃经》："不能亲近善知识故，虽有佛性，皆不能见。"

《前行备忘录》同样强调：天人、恶魔、世间君主、梵天都不会为我们指示解脱道，慈爱自己的父母、亲朋好友也不会指出正道——只有上师善知识才能指引解脱之道。一切佛经、续部、论典中从来没有提过不依止善知识和善友，而依靠自我造就及有胆有识就得以解脱和成就佛果的历史 (《前行备忘录》)。此外还警告：颠倒宣说见解和行为的人，如果把他作为上师来依止是魔知识，如果作为友伴来依止则是魔友 (《前行备忘录》)。

**总括**："包括自己在内的一切众生，对于邪道倒是很有造诣，所有支分都能圆满具足，而对于解脱圣道，却犹如无依无靠的盲人迷失在空旷荒野中一般知之甚少。"《大集经》："众生心乱故，堕在生死河，如盲无所见，常为苦所没。"不依商主便无法从宝洲取宝，不依善知识则解脱遥遥无期。

对于"只有藏传佛教才重视依止善知识"的偏见，此处严厉反驳：佛陀在许多经典中亲口说自己涅槃后将化现为善知识利益众生。《增一阿含经》中佛亲说——阿难称善知识为"半梵行"，佛纠正说"善知识者即是全梵行之人"，"我亦由善知识成无上正真等正觉"。《宝云经》："于善知识生世尊想，依善知识住。"可见视师如佛、依止善知识是整个佛教的共说而非藏密一家之言。

### 丁二·戊一、观察上师（第73--74课、第75课前半）

《功德藏》以檀香林比喻："如玛拉雅树林中，漂落普通一节木，枝叶滋润出妙香，依止上师随行彼。"

#### 可以依止的上师——六种法相

| 法相 | 含义 | 补注 |
|------|------|------|
| 相续清净 | 从未违犯外别解脱戒、内菩萨戒、密乘三昧耶戒 | 戒律是一切功德的所依 |
| 广闻博学 | 通晓经续论典（显宗五部大论、密宗《大幻化网》等） | 不能连经续名字都说不上来 |
| 具大悲心 | 对无边众生如母对独子般慈爱 | 不能只对亲友爱护而对他人恨之入骨 |
| 通达显密 | 精通外三藏、内密四续部仪轨（包括实修） | 光讲理论自己从未修过则不行 |
| 现前断证 | 依修持实义自相续中现前殊胜断证功德 | 最起码要能摧毁或压服烦恼 |
| 圆满四摄 | 以布施、爱语、同行、共事摄受弟子 | 爱语非迎合——《经庄严论》指讲修行教言 |

《功德藏》总摄："圆满诸胜法相者，浊世力致故难得，三戒清净之大地，多闻大悲润心续，精通如海显密仪，断证净慧硕果丰，四摄鲜花齐争艳，善缘弟子如蜂聚。"

《经庄严论》另列十种法相："依师当调柔，寂灭功德胜，勤富教悟真，善说悲无厌。"即调伏（戒）、寂静（定）、除惑（慧）、德增、有勇、阿含富、觉真、善说法、悲深、离退减。

#### 密宗上师的十二条件

| 序号 | 条件 | 含义 |
|------|------|------|
| 一 | 成熟相续 | 得过不间断能成熟的灌顶，且如理如法、各方面因缘全部具足 |
| 二 | 净持律仪 | 从始至终没违犯过灌顶所受的誓言和其他戒律（如十四条根本戒、《大幻化网》五条根本戒） |
| 三 | 相续调柔 | 烦恼和分别念微弱，不能整天胡思乱想 |
| 四 | 精通密宗 | 精通基道果一切续部意义，否则何为基道果都搞不清楚又怎么教弟子 |
| 五 | 依修圆满 | 最好亲自见过本尊；即使没有，梦境验相中也要见过；所修心咒和验相已尽善尽美 |
| 六 | 解脱相续 | 现量证悟万法皆空实相，对无我之义有所体会 |
| 七 | 唯求利他 | 心相续周遍大悲心，除利他外从不考虑个人利益 |
| 八 | 琐事鲜少 | 断除对今生世间贪执（为弘法利生有些事情不叫"琐事"） |
| 九 | 精进修持 | 修了四厌世心后唯一为来世精进忆念正法 |
| 十 | 厌世劝他 | 现见轮回痛苦生起强烈厌离心，并经常劝勉他人看破世间 |
| 十一 | 摄伏弟子 | 以种种善巧方便——语言、行为示现各种形象摄受调伏弟子 |
| 十二 | 具有加持 | 依照自己上师言教行持从不敢违背，以此缘起自相续具有传承加持 |

《功德藏》："尤其宣讲窍诀师，得灌净戒极寂静，通达基道果续义，念修圆满证自解，悲心无量唯利他，精进念法琐事少，极具厌离亦劝他，善巧传承具加持，依如是师速成就。"

《前行备忘录》特别补充宣讲更高层窍诀的上师的条件：除大圆满自道下面的四灌顶外还须得受有戏无戏等四灌顶以成熟相续；依靠有守护次第的誓言——上师身语意包括的二十七种根本誓言、二十五种支分誓言以及无守护的顿悟誓言——本来清净的无有、平等及任运自成的唯一、任运清净誓言守护相续；现量见到本来清净直断和任运自成顿超的法性；对道之四相（法性现量相等）有不同程度的理解、体验和证悟；对弟子不分贤劣一并摄受；具足金线般未被破誓言之锈所玷污的传承加持 (《前行备忘录》)。

冈波巴大师简要三点：断除此生贪著、以大智慧引导弟子入道、以大悲心不舍弟子。概而言之——"看破今世，有大智慧、大悲心"。

所有上师法相总纲（第79课）：观察上师一切要点，皆可摄于**观察其是否具有菩提心**这一条件中。《前行备忘录》亦明确指出："特别要观察上师的一点，唯独就是看上师具不具备菩提心"——如果具备菩提心，凡是结缘者都受益，结善缘者即生得以成佛，结恶缘者也不会再流转轮回。在观察菩提心的基础上，还要洞察上师在三大见解方面是否具备理解、体验和证悟的境界 (《前行备忘录》)。"有了菩提心的话就可以依止——无论装束如何、出家在家，甚至穿得破破烂烂每天一边传法一边喝瓶五粮液，我们也可以接受也应该依止。"反之，不具菩提心则"法座再高、哈达再多、人民币再多也不是什么法相"。但注意：相似的菩提心不算——"有些人看众生好可怜就闭着眼念'但愿一切众生获得佛果'，口头上说得挺好听，内心中有没有这种境界很难说。"《十法经》："依烟知有火，依鸥知有河，具智菩萨种，由外相而知。"可通过观察其弘法利生的实际行为来大致判断。

#### 四种应舍弃的上师

| 类型 | 比喻含义 | 特征 |
|------|----------|------|
| **如木磨之上师** | 形似磨发出嘈杂声却磨不碎粮食，最终磨损自己 | 无闻思修功德却以家世种姓自居（"像婆罗门世袭门第观念"），或虽有少分功德但所作所为只为今生 |
| **如井蛙之上师** | 井蛙问海蛙"有我这个井的四分之一大否？一半大？这么大？"最后去看大海时头颅崩裂而死 | 无特殊功德却被愚者推上高位，被名闻利养熏得面目全非、骄傲自满、对正士功德视而不见。无垢光尊者：弟子"多如蛆虫也没有意义" |
| **如疯狂向导之上师** | 疯子带路必入歧途 | 从未依止过智者，经续一窍不通，烦恼比凡夫还粗重，显宗戒律违犯密宗誓言毁坏却言谈举止高如虚空，"断掉了慈悲心的吊索"——蔑视因果欺骗信众像大成就者一样行持疯狂之举 |
| **如盲人向导之上师** | 盲人引盲人入深渊 | 不具胜弟子功德，不懂取舍，远离慈悲菩提心，将弟子引入邪道 |

莲花生大士："不察上师如饮毒，不察弟子如跳崖。"忽必烈观察八思巴六年才正式依止；《金刚鬘讲续》："十二年间需观察，若不了知久观察。"

《功德藏》警示："若未详细观察师，毁坏信士善资粮，亦毁闲暇如毒蛇，误认树影将受欺。"——不幸遇到邪知识，一生善资葬送其手中（一生积累的财富也会毁坏），暇满人身白白虚度，如同毒蛇盘绕树下，误认为树影去乘凉结果被咬死。

《正法念处经》："若近恶知识，众苦常不断，当受恶道苦，此苦不可说。"佛陀授记——依止恶知识不仅今生钱财时间耗尽，来世恶道受苦更不用说。《增一阿含经》："人本无有恶，习近恶知识，后必种恶根，永在暗冥中。"

关于观察年限：《事师五十颂》："是故勇士先观察，上师弟子之关系。"宗喀巴大师引《金刚鬘讲续》："十二年间需观察，若不了知久观察。"然而末法时代很多人"不要说十二年，连十二天观察也没有——今天开车回家路上接到电话说某上师来灌顶了，马上急转弯匆匆去依止"。

#### 上师即佛之教理

具德上师是十方诸佛大悲与智慧的本体，为利益众生而显现为人。

从不了义角度讲上师因随顺众生行为跟众生几乎一样；从了义角度讲上师智慧跟佛陀无二致，与众生所作所为完全相违——"就像幼儿园老师为了随顺孩子会跟小娃娃一起玩，但孩子做非法事老师不会跟着做"。

显宗教证：《涅槃经》："怜愍一切诸众生故，而复示现种种色像。"《地藏经》："吾分是形百千亿化度，随其根性而度脱之。"《般舟三昧经》："当敬于师，承事供养，视当如佛。"《华严经》："譬如明净日，出现于世间，生盲虽不见，而能作饶益。"《宝云经》："于善知识生世尊想。"《无二平等最上瑜伽大教王经》："佛为最上阿阇梨……此阿阇梨佛无异。"

上师的十二种功德比喻（《功德藏》原颂）：

| 比喻 | 对应功德 |
|------|----------|
| 如大舟 | 能救度众生脱离轮回大海 |
| 如商主 | 无误指示解脱与遍知佛果之圣道 |
| 如甘露雨 | 能熄灭业与烦恼的熊熊烈火 |
| 如日月 | 遣除无明黑暗，现出正法光明 |
| 如大地 | 容忍弟子忧心劳身及一切邪行，见行广博无所不包 |
| 如劫波树 | 今生与来世一切功德利乐的来源 |
| 如妙瓶 | 不可思议诸乘宗派一切意愿的宝库 |
| 如摩尼珠 | 息增怀诛四种事业大海的源泉 |
| 慈如父母 | 对无边众生无有亲疏爱憎一视同仁 |
| 悲如河流 | 对芸芸苍生广泛怜悯，对无依无怙苦难有情尤其迫切 |
| 喜如山王 | 不为嫉妒之心所转，不为实执之风所动 |
| 舍如雨云 | 自相续不被贪嗔所扰乱 |

"凡是与上师结上善缘者即生便可成佛；结上恶缘者（诽谤、嗔恨、打骂）也会以此因缘最终断绝轮回。"

#### 弟子法相

**弟子四条件**：

1. **精进大铠**——纵遇生命危险也不违背上师意愿
2. **智慧极坚**——心情稳定，不因暂时违缘就"收拾东西找班车离开"
3. **承事上师**——赴汤蹈火在所不辞，付出生命也心甘情愿
4. **依教奉行**——上师所说任何教授言听计从，不夹杂私人要求

《功德藏》："精进大铠慧极坚，不惜身命作承侍，谨遵师命不护己，唯以敬信得解脱。"

**弟子十素质**（《功德藏》："具信慧闻大悲心，恭敬戒律三门调，宽宏慷慨净相愧。"）：

1. **具大信心**——将上师视为真佛。观上师事业、智慧、悲心各方面，凡夫人根本无此能力
2. **具大智慧**——了知上师善巧方便行为的密意，上师度化的众生不是一两个而是无量，其一句话也许你不喜欢但对其他众生有巨大利益
3. **广闻博学**——能受持上师所讲的一切正法，至少听一堂课后要看一两遍
4. **具有悲心**——对一切众生有大悲心，慈爱无依无怙、受苦受难的众生
5. **敬护净戒**——恭敬上师所传授的律仪和三昧耶戒
6. **三门调柔**——身语意寂静调柔，说话做事跟道友接触不会野蛮粗暴
7. **宽宏大量**——内心能容纳上师及道友的一切行为
8. **慷慨大方**——将所拥有一切毫不吝惜供养上师
9. **具清净观**——内心很少出现不清净分别恶念，不观察上师和道友过失（"人看过失的眼睛特别厉害，看功德的眼睛基本没有"）
10. **具惭愧心**——造了恶业立即想到"必受正士大德羞辱"。《增一阿含经》说有惭有愧二妙法拥护世间

**弟子七比喻**：如驯服骏马（上师严厉呵责不怒不恨）、如船筏（上师指派不厌来往）、如桥梁（好事坏事都能承受）、如铁砧（忍耐严寒酷暑一切艰苦）、如奴仆（唯命是从）、如扫帚（断除我慢常处卑位）、如断角牦牛（舍弃骄慢恭敬众生）。

《华严经》另有二十一心（大地心、金刚心、铁围山心、给侍心、弟子心、僮仆心、养母心、傭作心、除粪人心、已熟稼心、良马心、大车心、调顺象心、须弥山心、良犬心、旃荼罗心、犗牛心、舟船心、桥梁心、孝子心、王子心），可摄为寂天菩萨《集学论》九种心（大地、金刚、铁围山、仆使、除秽人、乘、犬、舟船、孝子）。

《华严经》讲依止善知识要有十种心——给侍心、欢喜心、不违心、随顺心、不求利心、一向心、同善根心、同愿心、如来心、同满行心。弟子务必具足。

#### 三种承侍

| 等级 | 方式 | 说明 |
|------|------|------|
| **上等** | **修行供养** | 以坚韧精神实修上师所传正法——**最殊胜** |
| 中等 | 身语意承侍 | 身语意侍奉上师 |
| 下等 | 财物供养 | 供养饮食受用等 |

《前行备忘录》指出依止上师要以三喜承侍的方式进行，并强调："在依师期间，着重的不是修行，而主要是闻思" (《前行备忘录》)。在修学意行时，自相续要得到上师修行的境界——没有得到上师的密意而只是在行为上学，显然就成了诈现威仪，因此一定要得受上师密意以后再学其行为 (《前行备忘录》)。

《华严经》："诸供养中，法供养最。所谓如说修行供养、利益众生供养、摄受众生供养、代众生苦供养、勤修善根供养、不舍菩萨业供养、不离菩提心供养。"

佛陀涅槃前对阿难开示（《长阿含经》）：佛陀前往拘尸那城准备在娑罗双树下入灭，娑罗树虽未到花季却突然开花飘落佛身。佛陀说这些夜叉天神以非时之花供养"并非真正的供养"。阿难问何为真供养，佛答："只有受持我的法、修行我的法，才算是真正的供养如来。"《长阿含经》："人能受法，能行法者，斯乃名曰供养如来。"

"受法"与"行法"有区别：受法指在上师面前听课或翻阅佛经接受法义；行法是明白道理后实实在在去修行。二者都是对上师和佛陀最好的供养。《法句经》："若有智者，见善知识，应当供养，不惜身命。"

### 丁二·戊二、依止上师（第75--82课）

#### 依止的方式（第75课）

"所谓的依止，主要是以佛法依止"——弟子在上师面前听法、上师给弟子传法。《正法念处经》："若为他人说，一句之善法，则为善导师，为众生所尊。"

依止上师的正确心态——《华严经》四想："应于自身生病苦想，于善知识生医王想，于所说法生良药想，于所修行生除病想。"若四想颠倒则"再怎样精进也无济于事"。

依止不一定天天跟着上师形影不离——"若能将上师相续中的智慧、悲心融入自心，逐渐获得上师戒定慧三学功德，这才叫依止。"通过法本、光盘等途径得到上师教言并依教奉行，也是依止上师。

依止上师不像世间关系——"佛教中上师和弟子之间有种不可言说的缘起，这种缘起牵涉到你今生乃至生生世世的解脱，与你的证悟有着密切关系。"

如何判断有缘上师——华智仁波切："面见某上师或听到他的语言，甚至只是听到他的尊名，也会周身汗毛竖立、万分激动，生起无比信心，心情骤然改变——这说明他是自己生生世世的上师，不需要进行观察。"如米拉日巴仅闻玛尔巴之名即不共信心油然而生。但也提醒不可盲目追求感觉——"有些人刚开始遇到上师时互相很激动，过了不久又开始打官司说上师骗了钱"。

"遇到怎样的上师，跟内心清净与否及业力有密切关系。"《佛说观佛三昧海经》：释迦族五百人"见佛色身犹如炭人"，比丘众中一千人"见佛色身如赤土人"——前者因前世以恶心谤佛正法，后者因对正法和上师生邪见。内心不清净即使值遇真佛也不可能将他看作有功德者。

#### 身语意行为规范（第77课）

- **起身**：上师起身时自己毫不迟疑站起来（除非上师开许）
- **安坐**：向上师请安问候、观察时机供养相合上师心意的用品
- **行走**：不可走在上师前面（背对上师不敬）、后面（可能踩上师脚印影子）、右边（右边为首席）——应在上师**左侧稍后**恭敬随行。遇危险可请开许先行。《金刚藏庄严续》："夜晚与过河，处于险地时，请求师开许，先行无过失。"
- **坐垫乘骑**：未经开许不得踩踏上师坐垫、乘坐上师车辆。《事师五十颂》："畏惧如毁佛塔罪，师影尚且不应跨。"若不得已跨越，需一边诵咒（百字明或《金刚经》偈）
- **敲门**：不能猛力敲上师的门。阿底峡尊者弟子大喊索要窍诀，尊者面现不悦："窍诀不是用大声叫喊换取的，唯有以恭敬才能得到。"
- **举止**：断除弄姿作态、嬉皮笑脸、愁眉苦脸，断绝欺人之谈、戏耍玩笑、胡言乱语、无稽之谈。"身口意自然可以寂静调柔。"对上师始终心怀敬畏，杜绝满不在乎的心态。
- 阿底峡尊者对藏人的告诫——"大乘功德不论生多生少都要依靠上师才能生起。你们藏人对上师只作凡庸想，如何能生功德呢？"
- 金厄瓦上师因经常忆念上师、恭敬上师，相续中生起了真实功德。

《功德藏》总摄："上师起时莫安坐，坐时问安供受用，若行莫随前后右，踏垫坐乘等折福，切莫猛厉敲师门，舍弃弄姿笑怒容，妄乱玩笑无关语，三门寂静而依师。"

#### 与诽谤上师者绝交

《功德藏》："詈骂嗔恨上师者，不应为友尽力止，畅所欲言增大罪，失毁一切诸誓言。"

跟破誓言者交往如同与传染病人同住："一滴酸奶可以腐坏一大锅鲜牛奶，同样一个破密宗誓言的人也能毁坏一百个具誓言者的功德。"——金刚持如来亲口所说。

法王如意宝规定：破密宗十四条根本戒者、与金刚上师或道友有严重矛盾者，任何坛城灌顶或法会都不能参加。

**金刚道友三比喻**：如腰带（长久不厌始终如一——"有些人今天认一个上师，过两天舍弃了又寻找新上师，这不像腰带而像卫生纸"）、如食盐（与谁都合得来做什么都行——"不像辣椒跟谁都合不拢"）、如柱子（承受再大压力也能忍耐——"不要稍遇违缘就拍桌子走人"）。

#### 如天鹅与蜜蜂般受持师意（第77课）

"栖身于胜妙水池的天鹅不会搅混池水，而是轻盈嬉戏尽情享受；蜜蜂飞旋花丛中不会损坏花的色香，而是吸取精华后悄悄离去。我们依止上师也是如此——不能扰乱上师及其眷属，有缘得受教言时全心全意恭敬谛听；一旦因缘结束要离开，也是轻轻松松、开开心心。"

"依靠这种信心与毅力，定会将上师相续中的一切功德，如同从一个瓶子倒入另一个瓶子般融入自己的相续。"

#### 生圆次第与上师的关系（第78课）

观修生起次第时——"形象虽显现为本尊，但本体除了上师以外别无其他"。玛尔巴观修喜金刚坛城时先向本尊顶礼而非上师，结果所有本尊化光融入那若巴心间。那若巴言："上师未曾出现前，连佛名字亦未闻，所有千劫诸佛陀，皆依上师而出现。"

圆满次第的智慧也完全依赖对上师的信心——无相圆满次第即观想上师智慧全部融入自己，"这是最了义的本尊修法"。

"能修所修相违"的警告：上师住世时不好好承事、依教奉行，对众生有利的事也不肯做，一旦上师离世才开始绘像修法——"既不具备使上师断证功德融入自相续的诚信恭敬，也不进行祈祷"。"上师住世的目的就是弘法利生，除此以外不可能是为了自己而活着。"

汉地杜顺和尚的公案（第78课）：唐朝终南山杜顺和尚的弟子依止三十多年却非要去五台山拜文殊。杜顺和尚临行前教示："游子漫波波，台山礼土坡，文殊祇这是，何处觅弥陀？"弟子到五台山至诚礼拜时一位老者告诉他："文殊不在这里，终南山的杜顺和尚即是文殊的化身。"弟子赶回却晚了一步——师父前一夜已圆寂。

### 三大依止公案

#### 一、常啼菩萨依止法胜菩萨（第80课）

此公案见于《大般若经》《小品般若经》。常啼菩萨（梵文萨陀波伦，意为"常啼"或"常悲"——因常为众生悲泣而得名）在寂静处修行时闻空中声告知前往东方求般若法，嘱咐他不要在意身体劳累、寒热、昼夜，"不要左顾右盼而当勇往直前"，并说"即使见法胜菩萨享受五种欲妙也不要生邪见，要了知那是菩萨调伏众生的善巧方便"。他猛然意识到未问路途远近，失声痛哭七日七夜。一位如来现前赞叹——"善哉善哉！过去无数如来为菩萨时以精勤苦行求得般若波罗蜜多，就是如你现在这样"——并指路：东方五百由旬"香积城"，法胜菩萨（昙无竭，又名法涌、法上）于七宝宫殿中为六万八千侍女等三时宣讲般若，无量天龙非人恭敬供养。常啼菩萨听后当下处在无所作意的境界中，清晰听到法胜菩萨在远处讲法之声，现前了许多禅定法门，目睹十方诸佛都在说般若并同声赞叹法胜菩萨。

**卖身求法**：常啼菩萨身无分文，决定卖掉自己的身体换取供品。魔王波旬隐蔽其叫卖声，令城中人听不见。唯有一位商主之女因宿世善根不被魔蔽。帝释天化为少年婆罗门考验他，常啼菩萨毫不犹豫"以利刃刺穿右手，割下右腿精肉，到墙角准备砍断骨骼取出骨髓"。

**商主之女的善根**：她见此惊讶下楼问因。听闻空性之义后"汗毛竖立欢喜如狂"，承诺供养一切所需资具并同行。其父母也要同往，于是一家三口、五百侍女浩浩荡荡随常啼菩萨奔赴东方。

**以血洒扫**：到达香积城后，法胜菩萨传法一日即入定七年。常啼菩萨等人站立等候七年，不曾合眼、不曾安坐。七年将满需清扫传法场地洒水压尘，魔王使所有水消失。常啼菩萨刺身出血洒地——"无始以来为五欲丧失身命无数次，未曾为正法舍过身体"。商主之女等人亦刺破血脉同洒。帝释天以红栴檀加持鲜血，使地面变为红色。

**获六百万三摩地**：七年将满时诸天人告知"离法胜菩萨出定还有七天"，于是常啼菩萨等人清扫传法处一由旬以内，铺设狮子座，商主之女与五百侍女各脱一件干净妙衣层层敷设。法胜菩萨入座宣讲般若，常啼菩萨当下获得六百万禅定法门（《大般若经》说六十百千三摩地门），亲睹十方三千大千世界佛陀尊容，从此行住坐卧一切威仪皆为般若幻化，甚至做梦时也常面见如来。

常啼菩萨公案的意义：无著菩萨修弥勒法六年不得见，灰心欲弃——下山后读《般若经》中常啼传记，深感自己苦行远不如常啼，于是返回鸡足山继续闭关，最终得见弥勒菩萨。

#### 二、那若巴依止帝洛巴（第81课）

那若巴（1016--1100）为印度王子，从小有厌世心，父母极力阻挠仍于25岁出家，后成为那烂陀寺大智者。一日专注读经时旁边突然出现一个老妇人，问他是否精通词句和意义。他说了解词句时老妇人高兴大笑手舞足蹈；他为讨好又说也了解意义，老妇人转而落泪大哭——"你说了解词句是诚实的，说了解意义就是在说谎了。"问谁懂意义，答："去找我哥哥帝洛巴。"说完消失。

寻找途中多次遇到上师化现却未认出——如一位无手无脚的麻风女横在路上，他从其身上跨过，麻风女腾空说"若仍受习性左右执著外貌，如何能找到上师？"一条发臭母狗横路中央他闭气跳过，母狗升空说"所有众生都是自己父母，若没有大乘悲心又怎能找到上师？"每次他都昏厥过后懊悔不已。

最后找到帝洛巴时，尊者以乞丐形象在烤活鱼。周围隐士见了不欢喜，他就将鱼吐出来一弹指鱼全部复活游回湖里。经那若巴苦苦哀求才摄受他，此后带他云游却一直不传法。

**十二大苦行**（每年一次，选讲）：

- 从九层楼顶跳下，几乎粉身碎骨——上师问"痛吗？"答"何止是痛，简直成了尸体"
- 竹签涂油烤硬后插入手指脚趾间，关节僵直痛不欲生
- 奉命偷粥被农夫痛打几乎丧命（上师说"我肚子饿了去讨吃的"，他讨回满满一托巴热粥，上师吃得格外欢喜——他想再去讨结果偷了农夫的稀粥被抓住暴打）
- 奉命跳入檀香木烈火中，全身严重灼伤
- 在充满水蛭的深潭中建桥，水蛭蜂拥啃噬导致大量失血
- 奉命抢大臣迎娶中的新娘——被大臣和手下打得遍体鳞伤
- 奉命撞倒国王的王后拖走——被国王和随从痛打至只剩一口气
- 奉命将禁卫森严中的王子拖出来推着绕圈子——被士兵用箭矛剑石打得命如悬丝
- 奉命找一个女人再供养给上师，女人只对那若巴微笑不理帝洛巴——帝洛巴打女人说"你只喜欢那若巴"，那若巴对此不起丝毫邪见（内心苦行）
- 奉命剖脉取血、割头放四肢做曼茶供养，失血过多昏死
- 每次苦行后上师均以加持恢复其身体

二十四次苦行（十二大苦行加十二小苦行）圆满后——

**最终鞋击开悟**：帝洛巴说"你去打水，我在这里生火"。那若巴赶紧去提水。上师生完火后来到他面前，左手抓住他的喉窍说"那若巴，把头伸过来"，右手脱掉鞋子猛击其头——那若巴一下子昏厥完全失去知觉。等到苏醒过来时，"相续中生起了上师的所有功德，师徒二人的意趣成了无二无别"。

这与禅宗的棒喝开悟有异曲同工之妙：黄檗禅师摄受临济禅师时一共打了61棒造就一代宗师；了义禅师被高峰禅师用木棒打下数丈深悬崖后大悟——"大地山河一片雪，太阳一照便无踪，自此不疑诸佛祖，更无南北与西东。"

表面上十几年中上师从未宣说一句正法，弟子也未修过顶礼之类善法。但因遇到成就者上师，弟子百分之百依教奉行，障碍得以清净而证悟。

那若巴的另一教训——违背上师言教的后果：帝洛巴曾嘱咐"不要担任布札马希拉寺护门班智达"，但那若巴经不住请求而接任。后与外道辩论连日不胜，祈祷上师后帝洛巴显现，教他将上师观在头顶以契克印指着外道辩论，才大获全胜。可见遵照上师教言比自作主张更为关键。

《大宝积经》："汝莫惜身命，持心如金刚，正问诸师道，莫舍正信意。"

#### 三、米拉日巴依止玛尔巴（第81--83课）

**背景**：阿里贡塘地方富翁米拉希日嘉村之子，长子名米拉闻喜（即至尊米拉日巴），有一个妹妹。兄妹童年幸福，后父亲去世家产被伯父勇仲嘉村霸占，母子三人沦为乞丐遭受种种艰辛。后从勇敦措嘉、拉吉俄穷处学咒术与降冰雹术——压死伯父的儿子、儿媳等35人，又降了"三墙板高"（约三尺厚）的冰雹致尸横遍野飞禽走兽都死了一片。对所造恶业追悔莫及而生修法之心。

**绒敦拉嘎处未成就**：绒敦拉嘎传大圆满法说"白天修白天成佛，晚上修晚上成佛"，米拉日巴以为自己是具缘者不需修行，终日睡眠玩耍——"正法与相续背道而驰"。绒敦拉嘎判断无法调伏，指示他前往南岩找玛尔巴。

**初遇**：仅闻玛尔巴之名即"欢喜心起、分别念消、汗毛竖立、热泪盈眶"。到来前夜玛尔巴梦到上师那若巴赐琉璃金刚杵和甘露金瓶，师母梦到邬金刹土送来琉璃宝塔——殊胜缘起吻合。玛尔巴装扮成农夫在路边耕田迎接。米拉日巴虽不识却"产生不可言表的喜悦，灭尽敏锐分别念"。他帮农夫耕完田、把留下的酒一饮而尽（满瓶倾泻证悟之缘起），然后才知此人就是上师。

**四次建房**：
1. 东山建圆形房——建了一半上师说"考虑欠妥"要求全部拆掉运回原地
2. 西山建半圆形房——每件材料从几里外山下背到山顶，一半时上师又来拆，说"喝醉了酒没说清楚"
3. 北山建三角形房——三分之一时上师问"谁叫你做的"，否认自己吩咐过要求拆毁
4. 最终在西南险处建四方形九层楼加宝顶共十层——请师母作证保证不拆。打地基时用了三位师兄游戏时滚来的石头，上师要求取出送回原处再搬回来——"不是不要这块石头，而是要你自己搬，不能占人家便宜"

背运石料过程中背上磨出三个血疮，脓血不止（"请上师过目只有挨骂；请师母看又会说在夸功——只有独自哭泣"）。他盖到七层楼时腰上又生一个疮，完成内殿后脊背上又生一个疮——三个疮口流出脓血疼痛难忍，他仍然背上土器继续修建。

多次请求传法或参加灌顶都被拒绝、痛打、赶出。第一次梅敦村波来求胜乐金刚灌顶时他坐在灌顶行列中，上师问要供养，没有供养就被赶出痛打。第二次绒地鄂敦秋多来求喜金刚灌顶，师母将私房嫁妆大松耳石给他做供品，但上师发现后说"师母的一切都是我的，用我的东西供养我算什么"——又遭痛打赶出。

师母一直安慰他"以后肯定能求到法"，还安排他到鄂敦喇嘛处求法。此前米拉日巴曾试图离开——先是假装收拾行李被上师发现遭皮鞭痛打（"你起初来时把身口意都给我了，要走就走但我的糌粑不许拿"），后来真正离开，中途为人家念《般若八千颂》时读到常啼菩萨传记而决定返回。

**师母伪造**：师母达媚玛心软，趁每月初十会供日用三种浓淡不同的酒灌醉上师和喇嘛们，偷出印章和那若巴身庄严、红宝石印，盖了假信送米拉日巴去鄂敦喇嘛处。鄂敦喇嘛见到信和加持物很高兴，也先让米拉日巴去降冰雹（他不得不再次造罪，但特意保全了一位老太太的田——此后那块田再不落冰雹）。鄂敦喇嘛传法灌顶但因没有上师真正开许而"未生起少许功德"。

**暴怒与转折**：玛尔巴召回二人后在会供中身边放了一根长长的棍子，声色俱厉质问鄂敦喇嘛"为什么给大力这个恶人传授灌顶及窍诀"。鄂敦喇嘛吓得发抖，一边磕头一边出示"信件"："是您老人家给了我一封信让我传法。"上师又指着米拉日巴恶狠狠地问东西从哪里来的。米拉日巴浑身战栗几乎说不出话——"那……那……那是师母给我的。"上师一听从座上跳下来拿起木棍就去打师母。师母早知不妙站在外面，一见情势不好拔腿跑进房"嘭"地关门。上师咆哮着追过去用棍子狠狠打门打了半天才回法座。

米拉日巴万念俱灰想到"不但自己受苦还连累鄂敦上师及师母受罪，与其活着造罪不如死了好"，找了把藏刀准备自尽。鄂敦喇嘛劝住他："密乘中寿命未终行转识法都有杀佛过患，没有比自断生命更重的罪了。"

**最终传法**：上师怒气消后命人将众人唤到面前，宣布——"照过去的事看起来谁都没有错。我为了清净大力的罪业故意让他苦行叫他修房子；达媚玛是个女人心肠太软太慈悲难怪她；鄂敦也没有错但先把身庄严和玉印还给我以后再给你。我这个儿子大力如果能受九次大痛苦大折磨将不受后有即身成佛，现在还有一点点余业未净这完全是达媚玛女人心软的缘故。话虽如此但他的大部分罪业已清净了。从今以后我要摄受他，传给他最无上的灌顶和窍诀，还要给他修行的资粮。"传授皈依戒、居士戒和菩萨戒，赐法名"米拉金刚幢"。

**坛城显现**：灌胜乐金刚顶时上师指着绘画坛城说"这是表相坛城，真正的坛城你看——"手指虚空一弹指间，胜乐金刚六十二本尊坛城显现空中。"所有灌顶与窍诀以满瓶倾泻方式毫无保留地传授。"

**预言解说**：上师揭示所有缘起——"你把酒都喝完、田耕得一点不剩，这是你将领受窍诀成为法器达到圆满证悟的征兆；供养四角铜锅灯盏，表示你将成为四大弟子之一；空器供养表示修行时会遭受饥饿痛苦；我装满酥油燃灯是使后半生得大受用；敲灯发出响声是使名声遍于十方。"

米拉日巴公案中值得注意的细节：(1) 他从大圆满上师绒敦拉嘎处因根基不对口而未成就——说明法与修行者的宿缘非常重要；(2) 玛尔巴到来前夜的梦境（琉璃金刚杵、甘露金瓶、琉璃宝塔）精确预示了师徒因缘；(3) 米拉日巴多次萌生自杀念头，鄂敦喇嘛以"密乘中寿命未终行转识法有杀佛过患，没有比自断生命更重的罪"劝阻；(4) 常啼菩萨的公案对米拉日巴产生了直接影响——他在念《般若八千颂》时看到常啼求法事迹，认为自己的苦行远不如常啼，于是决定返回继续依止。

### 丁二·戊三、修学上师之意行（第76--78课、第83课）

#### 对上师行为的清净观

"上师行为变化莫测——为摄受不同根基众生，不能只示现一种行为。产生邪见时应认识是自己心不清净、眼不清净，立即忏悔。"

《大智度论》："于师一心不起增减者，汝于师所尽得妙法。"

#### 帝洛巴吃鱼的真相（第76课）

帝洛巴并非因饥饿杀鱼——"那些鱼是不知取舍道理的恶趣众生，他具有超度能力，为使之与自己结上缘才吃了鱼肉，然后将其神识接引到清净刹土。"旁生无法以其他方式与上师结缘，唯一供养自己肉身——密宗降伏超度法。一般来讲无因无缘超度一个众生相当困难，按照藏地传统哪怕一条哈达一针一线也要用它跟上师结缘。

严厉警告模仿者："手指虽然能弹响，但能否把鱼神识超度到清净刹土，自己应心知肚明。假如你真有这种能力那怎么做都可以；但若没有，最好不要在别人面前装模作样。"

汉地也有类似公案：清朝皇帝咸丰母亲死后托梦说堕入恶趣，请西域寺放生场"养猪的和尚"超度。那位圣僧穿得破破烂烂跟猪一块睡吃猪食，放焰口时在法台上睡着了。第一晚太后托梦说"圣僧救我时拉着衣服要出来了，被人一拉又出不来了"。第二晚不敢动他，焰口将完时他说："老僧活了八十多，从没干过这个活，大的孤魂生净土，小的孤魂吃馒头。"当下成千上万饿鬼超生。

#### 善星比丘——反面教材（第76课）

善星比丘为佛陀亲戚（《法华玄赞》说是太子时所生之子），在佛陀身边当侍者**二十四年**，对十二部经了如指掌可倒背如流。然而他将佛陀一切行为都看作欺诳之举，连芝麻许功德也看不到。

**五件愚行**（《大般涅槃经》）：
1. 佛陀晚上为帝释天传法时他打瞌睡，嫌传法太久，用恐吓小孩的鬼故事威胁佛陀回去睡觉——帝释天见之惊讶不已
2. 跟佛陀进城乞食时故意踏乱抹去佛的足迹——结果抹不掉反惹众人反感
3. 在酒馆旁赞一个喝醉胡言乱语的外道为"阿罗汉中的阿罗汉"
4. 赞说"烦恼无因无缘产生"的外道苦得为最殊胜阿罗汉
5. 佛预言苦得七日后腹痛而死转生食吐鬼。善星跑去告诉苦得让他小心不要被佛言中。苦得六天不饮不食，第七天忍不住吃了黑蜜喝了冷水果然腹痛而死。善星在尸陀林亲见食吐鬼——鬼亲口说"你不要对佛起邪见，不信佛语将来也会如此"——善星仍不信

**最终堕落**：善星对佛说"二十四年为汝仆，除身具有一寻光，芝麻许德吾未见，知法我胜不为仆"扬长而去。佛预言他七天后死于花园转生为具九种丑相的饿鬼。善星六天水米不进想让预言落空，第七天口渴喝了一口水便气绝身亡——"死后在花园中投生为一个具有九种丑相的饿鬼"（《涅槃经》另说堕入阿鼻地狱）。

**教训**：即使佛陀身边二十四年精通一切经，若无信心只生邪见，学问反成堕落之因。"久伴于佛亦见过"——"整个三千大千世界中一切功德圆满、一切过失断除的唯有佛陀一人，但没有信心、具有邪见者即使跟佛陀共住共睡也会觉得他没什么了不起。"

《功德藏》："自己未调自心前，迷乱观察无量罪，善星精通十二部，见师行为狡诈相，善加思维改自过。"自心未调伏前以迷乱错误观念观察上师会有无量过失——善星比丘精通十二部经却以劣见观察上师过失最终堕入恶趣。一旦生起不好念头应意识到是着魔了要马上忏悔。

#### 清净观教证

《功德藏》："诸行不应生邪见，圣地数多自在者，显现劣种恶形象，恶劣之中极恶劣。"

《十住毗婆沙论》："内有功德慧，外现无威仪，游行无知者，如以灰覆火。"——轻贱外现不如法的成就者就像坐在灰覆盖的火上，必被烧伤。又："若以外量内，而生轻贱心，败身及善根，命终堕恶道。"

《般舟三昧经》："当敬善师，不得视师长短。"

上师显现忿怒时——《功德藏》："倘若上师现忿怒，见己过失呵责调，时机成熟应忏悔，如是智者不着魔。"应意识到"上师以寂静方式无法调伏我才严厉呵责，说明得度机缘成熟了"。

对已得过法的上师诽谤的果报极重——《君规教言论》（麦彭仁波切）："佛说如若不恭敬，仅赐一句法上师，百世连续转为狗，复转生于劣种中。"

退一步说，即使无法视师如佛，也要忆念法恩："给你传授一个偈颂以上的上师可以长养你的智慧法身"——如果不是忘恩负义之人就应感激恭敬。

印度阿阇黎佛智的教训（第83课）：上师以乞丐形象来到他讲法现场，他因怕丢脸装作没看见。上师问为何不顶礼，他谎称"没看见"——话音刚落眼珠掉落地上。说了实话并忏悔后蒙上师加持才得以恢复。另一次他在五台山途中遇到在家僧人和丑女，当坛城本尊显现时他选择向本尊顶礼而非上师——本尊、僧人和丑女全部消失，"这才明白所有坛城本尊皆为上师幻化"。

黑行大师的傲慢教训：航海时想到自己眷属受用胜过上师，航船当即沉入海中。祈祷上师后上师亲临解救并说："我不是不致力于寻求眷属受用——如果也将精力放在这上面，成为与你同样的人肯定不成问题。"

**鸠摩罗什大师**的公案也说明清净观之重要（第79课）：大师受国王姚兴强迫接受十个美女还俗。僧人效仿破戒，他当众吞下一碗缝衣针令众人知其境界非凡——"谁若能吞下另一碗便也可以娶妻生子"。每次讲法前提醒："譬喻如臭泥中生莲花，但采莲花勿取臭泥也。"圆寂前发誓"若所译经文无误则尸身焚烧后舌不焦烂"——果然尸身都成灰唯有舌头完好无损。大成就者所示现的任何行为即使显现再不如法，弟子也应观清净心。

#### 参与上师事业的功德（第78课）

"上师在行持菩萨行积累广大福慧资粮时（传法印经书属智慧资粮，建殿堂放生行善属福德资粮），自己也加入其中"——哪怕只供养微薄财物、做力所能及之事（打电话、发信息），甚至仅发起没有嫉妒的随喜心，"上师以菩提心摄持所得功德有多少，自己也同样获得"。

灌顶或传法期间上师与佛无别——《事师五十颂》："得受殊胜之灌顶，金刚上师之面前，十方世界中所住，如来三时敬顶礼。"乔美仁波切《山法》："上师在讲经和灌顶时，即使不是大成就者只是凡夫人，十方诸佛的加持也会融入他与他无二无别。待传法灌顶完毕后，诸佛菩萨才返回各自刹土。"此时对上师诽谤生恶念过失极大；此时以信心欢喜心供养一口食物也比其他时间百千次供养的福德大。

清扫上师居室等日常劳务也是积累资粮的正道。噶当派霞叶瓦格西是金厄瓦弟子——"上师叫他时哪怕写字还差一笔也立即扔笔跑过去，修曼茶时也扔掉曼茶盘跑到上师面前"。有一天打扫卫生把垃圾包在衣襟中准备倒掉，"当下楼梯到第三个台阶时突然入于禅定，面见好多佛菩萨，从此成为大成就者"。萨迦班智达也是在承事生病的根本上师时"完全不顾自己饮食睡眠"，以此缘起亲见文殊菩萨。

**豆粉与糌粑比喻**：两个藏族人同赴拉萨，一个除了少量豆粉外没有其他口粮。他比较聪明，将豆粉跟另一人的一大口袋糌粑混在一起。过了几天那人说"你的豆粉该用完了吧"，结果看了好几次豆粉都没用完——最后只好与所有糌粑一起吃完。同理，自己仅以少许财物或身语劳作与上师善业结缘，"上师所造善根没有耗尽前，你的善根也不会穷尽——就像一滴水融入大海，大海没有干涸前这滴水也不可能干"。因此高僧大德做弘法利生大事时"也应当稍微参与一下，然后将此善根如理如法作回向"。

#### 祈祷上师（第78课）

"上师是无上智慧和无偏悲心的总集。上师如如意宝可赐一切所欲，但没有祈祷就不可能赐予悉地——如同电脑不打开就无法使用，功能再多再齐全对你也没有任何用处。"

**中阴时的重要性**："死后在中阴时若想蒙受上师指引救度，也需要无限的信心恭敬心，如此才会与上师的悲心愿力聚合。假如你不具备虔诚信心、不恭敬上师，无论上师再好也不可能出现在中阴界为你指引道路。"（上师并非亲自去中阴界接人——而是信心与悲心因缘聚合时显现境界。）

《功德藏》："多数愚者绘像修，上师在世不承事，不知师意修实相，能所修违诚可悲，无信中阴难见师。"

平时梦境中遇到困难违缘时若能祈祷上师，是中阴时能依止上师的征兆。修法好时会梦见观音文殊、传承上师、美丽花园等善妙景象；修法差时则梦见恐怖鬼神等。每天修法前都要加上师瑜伽，睡前也要念一遍上师瑜伽——"将一切现象全部融入上师的身口意，很多境界就会显现为清净"。

#### 《事师五十颂》的来历（第83课）

作者跋维谛瓦精通佛法却曾因嗔心修法结手印令海岛陷入大海沉没、众多生灵死亡。后悔之下准备从故乡磕头到文殊菩萨圣地忏悔。度母梦中告诉他："即使这样做也消不了你的罪业。但如果你用多年修行的经验抉择出成佛最快的一条道路，用来指导未来众生让他们能快速成就，这个功德就能抵消恶业。"他经多次闭关修法思维抉择，找到的最快成佛之路就是**如理依止上师**。表面上《事师五十颂》仅五十颂，"但实际上依靠它能迅速清净罪业成就无上菩提"。

《大悲经》："若有说法者，及听佛正法，二俱得福多，能建诸仙幢。"我们对上师的一切言谈举止绝不能视为颠倒，也万万不可怀有狡猾之心，"务必要以正直的秉性老老实实依止上师"。

## 共同外前行总结

**观察上师后依止的总纲**（第80课）：首先善巧观察上师、中间善巧依止上师、最后善巧修学上师意行——"此人必将趋正道"。

所有法相统摄为一：是否具足菩提心。"有了菩提心的话就可以依止；倘若不具菩提心，就算他讲的显密佛法再高深，也只是给人种下善根而已。"《大宝积经》："菩提心者应可亲近，为诸智者所称赞故。"反之不具菩提心的如毒蛇——《大宝积经》："不亲下劣人，见不正直者，见已当远离，犹如避毒蛇。"

上师的恩德远超诸佛——"往昔曾出世了数不清的佛陀，他们的大慈大悲也无法救度我们，直到现在我们仍沉溺在轮回大苦海中。"如寂天菩萨言："饶益众有情，无量佛已逝，然我因昔过，未得佛化育。"弥勒菩萨："如天虽降雨，种坏不发芽，诸佛虽出世，无缘不获善。"如今上师以无量悲心垂念，相应不同缘分以不同身相现于世间——不是用财物或地位而是用佛法进行饶益。《正法念处经》："若人能说法，利益于他人，其人如父母，示以涅槃城。"上师引导我们前往涅槃大城永离痛苦，恩德不可思议。

善知识度化众生必须示现凡夫相——否则在凡夫群体中没办法待下去。"诸佛菩萨度化旁生时必须示现旁生形象——要度化麻雀就要变成麻雀跟它们一起吃睡叫玩。"善知识在不同众生面前示现不同身相，甚至变成屠夫、妓女。《维摩诘经》："众生病则菩萨病，众生病愈菩萨亦愈。"

应随时随地以清净信、欲乐信、不退转信三种信心全力以赴谨慎依止上师。

华智仁波切在结尾总结偈颂中谦虚地说："虽已值遇诸佛菩萨化现的善知识，但仍常被恶劣行为所诱惑；虽已获得即生可超越轮回的显密胜道，却仍一直造恶业。"祈愿诸佛菩萨加持以正法调伏自相续——"不要再如此刚强难化，听了多少法都当成耳边风"。

米拉日巴："无人山谷岩洞中，恒具出离厌世心，上师乃为三世佛，强烈坚信永不离。"

《华严经》："亲近供养诸善知识，是具一切智最初因缘。是故于此，勿生疲厌。"

《华严经》："譬如暗中宝，无灯不可见，佛法无人说，虽慧莫能了。"暗室中虽有珍宝但没有灯就漆黑一片看不见；佛法中虽有殊胜道理但若没有上师以窍诀揭示，纵然世间智慧再超群也不一定能通达深意。

《六波罗蜜多经》："善知识者难遭难遇，若不信受，后悔何追？"

> **虽遇圣士仍为劣行诱，虽获胜道仍漂非道中，**
> **我与如我恶性诸有情，正法调伏自续祈加持。**

此为共同外前行（人身难得、寿命无常、轮回过患、因果不虚、解脱利益、依止上师）全部内容之总结。以下将进入不共加行。
