# 日修闭关要诀

## 来源与定位

此要诀出自第2课后半部分，堪布在讲完序文后即传授此日修指南，称其为"堪布阿琼的窍诀，结合法王如意宝的教言，以及大圆满心滴讲义中的法义"。堪布自述："多年以来，我一直沿循这样的修行道路完成了自己的前半生，后半生肯定也只有这一条路。"

此要诀适用于共同加行和不共加行的全部修法，是心滴派修行人的标准日修框架。

## 一日修行三阶段

### 一、晨起修行

1. **醒来**：观想诸佛菩萨和传承上师摇动金刚铃和手鼓的声音，将自己从无明烦恼的睡梦中唤醒。

2. **醒后即修**：马上念"喇嘛钦、喇嘛钦……"（祈祷上师垂念），或念21遍百字明。

3. **观察梦境**：在床上安住片刻，观察昨晚做的是恶梦还是善梦。恶梦则谴责自己，善梦则随喜回向。

4. **起身礼拜**：起床后最少磕三个头。

5. **日常准备**：洗脸、刷牙、方便等。然后打扫佛堂，在佛像前供水、供香。

### 二、日间正式修行

根据自身时间，可分为两座、三座、四座、六座。**最少保证每天一座**（早上或晚上）。

每一座分三部分：

#### 前行

前行必须具足三个要点（堪布阿琼《前行备忘录》）：

**1. 身要——毗卢七法**

先在舒适坐垫上身心放松，稍加休息，然后：

| 要点 | 方法 |
|------|------|
| 双足 | 金刚跏趺或半跏趺坐 |
| 双手 | 结定印（右手在上、左手在下、拇指相触），置于脐下 |
| 脊背 | 端直——"身端直则脉端直，脉端直则心端直，心端直则觉性自然现前" |
| 颈部 | 稍向前屈 |
| 臂膀 | 后展放松，不向胸前内收 |
| 双目 | 垂视鼻尖，不睁太大 |
| 舌 | 抵上颚 |

**2. 语要——排九次垢气**

| 步骤 | 操作 | 观想 |
|------|------|------|
| 前三次 | 右手握金刚拳压右大腿根动脉，左手金刚掌按左鼻孔，从右鼻孔排三次 | 自他众生以**嗔心**所造罪业，以**白色**气体排出 |
| 中三次 | 左手握金刚拳压左大腿根动脉，右手金刚掌按右鼻孔，从左鼻孔排三次 | 自他众生以**贪心**所造罪业，以**红色**气体排出 |
| 后三次 | 双手握金刚拳压两大腿根动脉，两鼻孔同时排三次 | 自他众生以**痴心**所造罪业，以**黑色**气体排出 |

金刚拳：拇指压住无名指指根，其他四指握成拳头。

排垢气后，念元音咒、辅音咒、缘起咒（不会念可直接念"嗡啊吽"），以诸佛三金刚加持清净语业。

**3. 心要——发心与祈祷**

*发心*：审查此刻的心态——
- 无记心 → 转为善心
- 恶心 → 彻底舍弃
- 世间善心（救畏心、善愿心）→ 舍弃
- 声闻缘觉自利心 → 舍弃
- 唯取大乘菩提心：为一切众生获圆满佛果而修行

*祈祷*：
1. 在前方虚空约一尺高处观想上师（若信心大则观上师现相，若信心不够则观为观音/释迦/莲师）
2. 一心祈祷："十方诸佛菩萨智慧总集根本上师，请垂念加持我，令佛法融入我心……"
3. 若修人身难得，即祈祷"愿我像上师一样，人身难得的境界在相续中生起"——**将祈祷与当日修法主题直接挂钩**
4. 以强烈信心念上师祈祷文7遍或21遍
5. 观想上师化光融入自身，自心与上师智慧成为无二无别
6. 在此境界中安住——不追随过去、不执著现在、不迎接未来的分别念

#### 正行

上师瑜伽可作为前行，也可直接作为正行（一边祈祷一边磕头）。

若上师瑜伽作为前行完毕后，正行即是观修当日的引导文内容：人身难得、寿命无常等。按顺序观修——如昨日修了八无暇，今日修十圆满，逐一思维。

从"人身难得"到"菩提心"之间的修法，按桑吉多杰（堪布阿琼弟子）的传承，**至少需要146天修习**。若结合《大圆满心性休息》，共同四加行至少100天。

#### 后行

观修结束后回向善根：
- 上等：安住无缘中回向
- 中等：念"诸佛菩萨如何回向，我亦如是普皆回向"
- 有时间：念一遍《普贤行愿品》
- 无时间：念"文殊师利勇猛智，普贤慧行亦复然……"两偈

### 三、晚睡修行

1. **收供品**：将一天的供佛、供护法品全部收拾。

2. **磕头**：临睡前先磕三个头。

3. **反省一日修行**：
   - 修得好 → 随喜自己，回向众生
   - 修得差 → 呵责自己——"我在上师面前听了那么多窍诀……真是坏蛋！"如堪布阿琼所说，使劲掐自己的肉以示惩罚

4. **入眠观想**（心滴派教言）：
   - 方式一：观想上师化光融入心间四瓣莲花，于上师发出的无量光芒中入眠
   - 方式二：器情世界融入自己 → 自己融入上师 → 上师化光消于法界 → 在无缘境界中入于光明梦境
   - 若未修过光明梦境，则在无有执著的境界中入眠

## 核心原则

堪布反复强调的几个原则：

1. **修任何正行之前，必先修上师瑜伽**——这是心滴派的不共传统。"有些人念了皈依偈就马上修正行，这样不是很好。"

2. **一个修行窍诀用一辈子**——"不可能看到这本书有个修法就试一试，看到那本书有个修法又开始修一修。"诸传承上师都是在一个窍诀上完成一生。

3. **最少一座**——即使再忙也要保证每天一座修行。"连短短半个小时都不能修，那你可能已经转生到'无暇'中了。"

4. **修行是终生功课**——共同四加行"不是一两天、一两个月、一两年的事情，而要将其视为终生最重要的功课"。
