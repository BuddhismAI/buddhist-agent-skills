# 寿命无常·四观无常（第28–31课）

## 在全论中的位置

寿命无常是共同外前行第二个引导文（丙二）。暇满难得令人珍惜人身，寿命无常则令人**紧迫**——你手中的珍宝随时可能失去。

顶礼偈：

> **现见三有无常幻化相，舍弃今世琐事如唾涎，**
> **苦行修习追随先辈迹，无等上师足下我敬礼。**

科判分七，前四个是从四个维度观察无常的普遍性：

```
丙二（寿命无常）分七
├── 丁一 思维外器世界而修无常 ← 连宇宙都是无常的
├── 丁二 思维内情众生而修无常 ← 一切众生皆必死
├── 丁三 思维殊胜正士而修无常 ← 连佛菩萨也示现涅槃
├── 丁四 思维世间尊主而修无常 ← 连天帝轮王也逃不过
├── 丁五 思维各种喻义而修无常 ← 万事万物的无常实例
├── 丁六 思维死缘无定而修无常 ← 死亡方式不可预知
└── 丁七 思维猛厉希求而修无常 ← 由无常激发修行
```

论证逻辑：从最坚固的（宇宙）到最殊胜的（佛陀）再到最强大的（帝王），全部无常——那渺小如我们的人，凭什么觉得自己可以例外？

## 丁一、思维外器世界而修无常（第28课）

### 器世界的形成过程（《前行备忘录》）

器世界从下向上形成，内情世界从上到下形成 (《前行备忘录》)。起初世界从一劫毁坏到变成空空荡荡的虚空，中间整整经过二十劫。此劫众生由共同福报感召，下基清净的意空形成虚空，其上方形成带有墨绿色辐围的十字金刚杵状风轮，风轮上方由金藏云降雨形成水轮，再依靠上中下三等分辨界的风，分别形成须弥山、金山、四大部洲与铁围山 (《前行备忘录》)。内在有情从有顶到地狱之间，分别念越来越粗，逐步产生第一个地狱有情，这期间历经二十中劫 (《前行备忘录》)。

### 七火一水的毁灭过程

"由众生共同福德所形成，被认为是坚不可摧的四大洲、须弥山、天界以及铁围山等外器世界，虽然存留时间可长达数劫，但它们也是无常的，最终必定将因七火一水而毁于一旦。"

原论详述七个太阳依次出现的毁灭过程：

| 太阳 | 毁灭对象 |
|------|----------|
| 第一 | 一切树木园林烧尽 |
| 第二 | 一切溪流池沼干涸 |
| 第三 | 一切大江河水干涸 |
| 第四 | 无热恼大海滴水不剩 |
| 第五 | 外界大海从一百由旬到八万由旬深度全部干涸，直至蹄印许的水也无余 |
| 第六 | 焚毁大地雪山 |
| 第七 | 须弥山、四大洲、八小洲、七金山、铁围山全部烧成一片火焰；火舌向下焚尽一切地狱，又向上烧燃梵天无量宫殿 |

光明天小天子惊呼大火燃起，老天子安慰："这样的大火以前也是烧到梵天以后就无影无踪了。"

七火之后，二禅天形成水云层，倾盆大雨使一切如盐溶水般毁灭。之后十字杵金刚风翻滚，三禅天以下如风卷尘埃般灭绝。最终"万事万物变成一大虚空"。

《前行备忘录》详述坏劫中的迁移次第：坏劫开始时地狱不再有新生有情，已在地狱者业力渐减升至上界（除造无间罪、舍法罪及破密宗金刚上师誓言者堕金刚地狱外）。饿鬼、旁生亦渐空。人类中有人获二禅心后传播"寂静所生喜悦"，众人修行后转生二禅光明天；一禅天众亦迁至二禅天。此后二禅以下无一呼吸之众生，天不降雨，草木干枯，七日依次出现。火焚须弥山四洲后向下蔓延，遇地狱火——地狱火先向上，然后劫末火因业力强大向下焚尽地狱，无间地狱有情在一眨眼间转生他刹无间地狱。火舌再向上烧尽二禅天以下。二禅天众迁三禅天，三禅天众获四禅心迁四禅天。下基十字架风啸，吹散灰尘，变为空无所有的虚空，经二十中劫。劫的成住坏空合计"八十中劫一大劫" (《前行备忘录》)。

教证——《佛所行赞》："劫火燃烧时，能坏须弥等，四大海竭尽，如同水泡体，况复此人身，而欲求长久？"

《德施请问经》："一劫之末到来时，此世界当毁灭之，大山被火所焚烧，灰尘微许亦不剩。"

堪布补充：现代科学也证实太阳将在约50亿年后膨胀为红巨星吞噬地球——科学与佛法在这一点上一致。

**核心推论**："既然大千世界也有变成空无一物的时刻，那么我们如秋蝇一样的人身，又有什么恒常稳固的呢？"

### 公案：阿育王与老比丘

阿育王建八万四千塔后举办三个月法会，对一位终生无修证的老比丘却特别恭敬。法会上请老比丘说法时，比丘惊慌（出家60年未记住任何经文）。一树神出现教他一偈："高山大地大海水，终归无常归无一。何况大王之王位？大王当于此思维。"阿育王闻此偈深受触动。老比丘后来因此一偈证得阿罗汉果。

## 丁二、思维内情众生而修无常（第29课）

### 身体即是小器世界（《前行备忘录》）

正如外在器情世界无有常恒，我们内在的身体也不例外——"身"相当于器，"识"相当于有情。从母胎中形成到降生是"成"的时期；从婴儿到患病是"住"的时期；从患绝症到死亡是"坏"的时期；白光水大坏灭、红光火大坏灭、黑光风大坏灭、昏厥过去属于"空"的阶段 (《前行备忘录》)。

### 有生必死的绝对规律

"上至有顶下到地狱底层的所有众生没有一个能逃脱死亡。"

《解忧经》："地上或天间，有生然不死，此事汝岂见，岂闻或生疑？"——有生而不死的事情，见所未见、闻所未闻，也绝无"到底死还是不死"的怀疑余地。

### 死期不定的紧迫

《前行备忘录》将死期不定分为三方面——必死无疑、死期不定、死时一切都无济于事，分成入座和座间来修 (《前行备忘录》)。

"自从出生的那一天起我们便一步一步地向死亡靠近。人的寿命从来不会增加而只有减少。"

《因缘品》："明日死谁知，今日当精进，彼死主大军，岂是汝亲戚？"

龙树菩萨《亲友书》："寿命多害即无常，犹如水泡为风吹，呼气吸气沉睡间，能得觉醒极稀奇。"——在睡眠中平缓呼吸之间就可能死去，醒来应当看作"非常稀奇"之事。

### 死亡来时无可依靠

原论以极为生动的语言描述：

> "纵然拥有英勇的军队、强大的势力、丰富的财产、智者的辩才、美女的身色、奔驰的良驹也都无济于事；即便钻进一个无隙可乘的铁箱子里，外面有数十万勇士手持锋利的兵刃围绕保护，也丝毫守护不了、遮挡不住。"

> "勇士无法救护，大德不能吩咐，饮食无法引诱，无处可逃、无处可躲，无依无怙、无亲无助、无计可施，也无有任何尊者的大悲所能庇护，哪怕是药师佛亲自降临也无法延缓寿命已尽的死亡。"

《前行备忘录》以三层论证死期不定：(1) 你并没有"不死的把握"——除非遍知佛陀亲自授记、自己具有无漏神通、或与阎罗法王交上了朋友——否则寿命如强弩之末或油尽之灯；(2) 即便寿命引业未尽，突如其来的外缘（四百零四种疾病、八万魔障、三百六十阴卓鬼等）也可能导致突然死亡；(3) 一旦寿命穷尽，药师佛给灵丹妙药、无量寿佛赐长寿灌顶、金刚手菩萨予以保护都起不到作用 (《前行备忘录》)。

### 佛陀论生命之短

《四十二章经》公案：佛问弟子生命有多长？

| 回答 | 佛的评价 |
|------|----------|
| "几天" | "你不了解佛法" |
| "一顿饭的时间" | "你也不了解" |
| **"生命在呼吸之间"** | **"善哉！你已了解佛法"** |

### 公案：一休与茶杯

一休禅师师父珍爱的茶杯被打碎，一休把碎片藏起来。师父回来后，一休问："什么东西生了就一定会死？"师父答："一切有生之物皆必死。"一休取出碎片："您的茶杯已经死了。"师父微笑，未加责备。

## 丁三、思维殊胜正士而修无常（第30课）

### 核心论证

如果连获得无碍神通、自在驾驭四大的大成就者们都示现了无常——火不能焚、水不能溺——那么我们凡夫人何以幻想永恒？

### 印度圣者

**佛陀本人**——教法缘起：贤劫之前的七佛（毗婆尸佛、尸弃佛等）及其不可思议的声闻缘觉眷属，曾以三乘法利益无量众生，如今均已趋入涅槃，"只剩下释迦牟尼佛的教法"。

佛陀最后的教言："一切和合事物皆是无常，汝等比丘当精进修行。"

佛陀涅槃并非普通人的"死亡"——《涅槃经·金刚身品》中佛陀说："如来之身，是常住身、不可坏身、金刚之身、非杂食身，即是法身。"佛陀的涅槃是为破除众生的常执而示现。

**二胜六庄严**——龙树、圣天（中观）、无著、世亲（唯识）、陈那、法称（因明）、功德光、释迦光（戒律）——均已不在世，仅有传记和著作留存。

**八十大成就者**——如拉瓦巴（一衣师）等具足种种神变，然皆已示现涅槃。

### 藏地圣者

莲花生大士及君臣二十五大成就者、耶瓦八十大成就者；旧派索宿努三师、新派玛尔米塔三师——不可胜数的智者成就者。

原论特引米拉日巴尊者在尼泊尔的公案：猎人向他射毒箭未中，拖入水中丢下深渊也不能伤害，堆柴点火也烧不起来——如此自在，最终仍然示现无常。

### 汉地大德

堪布补充：杜顺大师（华严宗初祖，为文殊化身）、永明延寿禅师（被认为是阿弥陀佛化身）等——均已圆寂。

永明延寿公案：吴越王千僧斋上，一位耳朵奇长的陌生和尚坐了首座。后来有人说出"定光古佛"来了，使者追上时僧人说"弥陀饶舌"（阿弥陀佛多嘴了）便圆寂。使者回头找永明延寿，他也同时圆寂——两位佛陀一日同逝。

### 总结教证

《无常经》："生者皆归死，容颜尽变衰，强力病所侵，无能免斯者。"

> "假使妙高山，劫尽皆坏散，大海深无底，亦复皆枯竭，大地及日月，时至皆归尽，未曾有一事，不被无常吞。"

上师如意宝无常道歌（堪布引）："诸佛菩萨虽如繁星般出世利益众生，然皆示现涅槃以显无常。于此深思，方知万法无一恒常。"

## 丁四、思维世间尊主而修无常（第31课）

### 天界主宰也不能免死

《功德藏》："梵帝自在转轮王，无法摆脱死主魔。"

《解忧经》："大仙具五通，能行于虚空，然而却不能，诣于无死处。"

梵天、帝释天寿达数劫，身高一由旬及一闻距，光彩胜于日月——然而终有一死。

### 帝释天堕驴胎公案

帝释天寿命将尽时，以天眼预见自己将转生为陶匠家的驴。天人五衰相现前（花鬘枯萎、衣服垢腻、腋下出汗、体出恶臭、不乐本座），极度恐惧。忆念佛陀为唯一依怙，急赴灵鹫山，佛正入定。帝释天顶礼三次行三皈依，未及起身，神识已脱离天身投入驴胎。母驴受惊打碎陶器，陶匠怒打母驴，驴胎中帝释天死亡，神识返回天身。佛出定告言："善哉帝释！你的皈依已灭尽恶业。"随后为帝释说无常法，帝释证须陀洹果。

教训：即使天王之尊也无法与死王讨价还价，唯有佛法能救。

### 五通仙人藏身公案

《解忧经》引四位具足五种神通的仙人，预知七日后将死，各想办法躲避：

| 仙人 | 藏身处 | 结果 |
|------|--------|------|
| 第一 | 大海中央 | 海中即死，鱼啖其身 |
| 第二 | 须弥山腹中封闭 | 山中即死 |
| 第三 | 虚空之中 | 如熟果坠地而死 |
| 第四 | 集市人群中 | 人丛中悄然而死 |

佛陀评论："此等愚人不了解生命的本性。自古以来，无论藏身何处，无人能逃脱死亡。"

### 人间帝王的无常

**印度**：自众敬王（人类第一位国王）传承下来的无数君主、三巴拉王、三十七赞扎王——如今仅存历史记载。

**藏地**：涅赤赞布以来——天座七王（去世后升空无遗骨）、地贤六王、中德八王、初赞五王、幸福十三代（含松赞干布、赤松德赞、赤热巴巾等）、极乐五代。

松赞干布"依靠幻化军队征服了上至尼泊尔下至中国的大片领域"；赤松德赞"统辖南赡部洲三分之二的领地"；赤热巴巾"在印度恒河岸边竖立铁碑作为边界标志"——如此威力，"现在都已成了历史记载，除此之外无有任何留存"。

**汉地**：夏商周秦汉……唐宋元明清。秦始皇、汉武帝、唐太宗、康熙——皆已逝。

### 核心反思

原论："我们现在所拥有的住房、受用、眷属、权势等，自以为是何等何等地优越，但与以上诸位先贤比较起来，简直就成了蜂巢一样。这样的世间欲妙又有什么恒常性、稳固性可言呢？"

知玄禅师五岁诗："花开满树红，花落万枝空。唯余一朵在，明日恐随风。"——以花喻人：枝头繁华，终归空落。

### 中阴状态的描述（《前行备忘录》）

《前行备忘录》详述死后中阴经历：神识到了中阴从昏厥中苏醒，起初不知自己已死，见亲友号啕大哭便对他们说"我没有死，我还在这儿"，但亲友视而不见、充耳不闻。于是观察太阳下有无影子、沙中有无脚印、水中有无影像——一无所有，方知已死。随后极度恐惧，出现四大恐怖声音（大地震动声、世界烈火声、狂风声、洪水声）和三险地（贪心自相——红彤彤一片；嗔心自相——白茫茫一片；痴心自相——黑乎乎一片）(《前行备忘录》)。

业力清净者呈现勇士空行相迎、直趋上方的景象；造无间罪者浮现业卒阎罗卒情景而径直下堕。中阴停留七七四十九天（或三七、四七不等），前半时间见前世身体，后半时间见后世身体：转生地狱饿鬼则大头朝下行走，投生畜类则头上有角横行，转生天人人类则头朝上行走、身沐日月光辉 (《前行备忘录》)。

## 四观的递进逻辑

| 维度 | 观察对象 | 核心教训 |
|------|----------|----------|
| 丁一 外器 | 四大洲、须弥山、天界 | 连宇宙物质基础都会毁灭——何况血肉之躯 |
| 丁二 内情 | 一切众生（有顶→地狱） | 有生必有死，死期不定，无处可逃 |
| 丁三 正士 | 佛陀、八十成就者、二胜六庄严 | 连超越四大的圣者都示现涅槃——凡夫何以例外 |
| 丁四 尊主 | 天帝、轮王、历代帝王 | 连权力财富最圆满的也不能免死——世间成就毫无意义 |

从坚固到殊胜再到强大，全部击穿——留下的唯一结论：**修行正法是唯一的出路**。
