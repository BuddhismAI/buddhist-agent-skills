# 轮回过患·总述与三恶趣（第40–51课）

## 在全论中的位置

轮回过患是共同外前行第三个引导文。暇满难得令人珍惜，寿命无常令人紧迫，轮回过患则令人**恐惧与厌离**。宗喀巴大师《三主要道论》："人身难得寿无常，修此可断今生执。无欺业果轮回苦，修此可断后世执。"前两个引导文斩断对今生的贪执，此处所讲的轮回过患与下一个因果不虚，则斩断对来世善趣的耽著——令人真正希求从轮回中彻底解脱。

顶礼偈：

> **了达轮回诸事无实义，唯以大悲利益诸有情，**
> **不贪有寂依教行大乘，无等上师足下我敬礼。**

修法前应先念此偈，闭眼合掌祈祷："请根本上师为主的历代传承上师、诸佛菩萨加持，让我相续中早点产生轮回无有实义的定解，并能生起看破世间的无伪出离心！"

### 科判总览

> 丙三（轮回过患）分二：
> - 丁一、总的思维轮回痛苦
> - 丁二（分别思维六道各自痛苦）分六：
>   - 戊一（地狱之苦）分四：己一·八热地狱（八个庚级）→ 己二·近边地狱（五个庚级）→ 己三·八寒地狱 → 己四·孤独地狱
>   - 戊二（饿鬼之苦）分二：己一·隐住饿鬼（外障/内障/特障）→ 己二·空游饿鬼
>   - 戊三（旁生之苦）分二：己一·海居旁生 → 己二·散居旁生
>   - 戊四·人类之苦 → 戊五·非天之苦 → 戊六·天人之苦

本文涵盖丁一至戊三（总述与三恶趣），三善趣另见《轮回过患·三善趣》。

## 丁一、总的思维轮回痛苦（第40课）

### 总体过患的六方面归纳（《前行备忘录》）

《前行备忘录》将轮回总体过患归纳为六个方面：流转于什么地方、流转为什么众生、在什么时间流转、以什么形式流转、以什么因流转、以实例说明流转，通过观察安住轮番的方式来修 (《前行备忘录》)。

并指出一个根本规律："所有世间俗人都是以快乐开幕，以痛苦告终。所有修行人都是以痛苦开头，以快乐结尾" (《前行备忘录》)。如果依此总体思维仍未生起厌离心，就要进入具体思维六道各自痛苦的阶段。

### 轮回的本质

"所谓的轮回，就像陶师手中的轮盘、井中的水车、瓶中的蜜蜂一样接连不断地旋转。"善趣的人间天境如同瓶内上面的空间，三恶趣就像瓶内下面的空间——无论飞上飞下，都超不出轮回这个瓶子的范围。六道众生以有漏的善业和不善业为因，连续不断地投生流转，为此叫做轮回。

相关经证：《法华经》："以诸欲因缘，坠堕三恶道，轮回六趣中，备受诸苦毒。"《宝鬘论》以旋火轮喻说明轮回无初、无中、无尾。《佛本行集经》喻为竹筒中的蜜蜂。

### 一切众生皆曾为父母

"假设把整个大地的土抟成枣核许的丸子，口中数着'这个众生的母亲是这个，那个有情的母亲是那个'……待到土丸的数量已经穷尽，然而各个众生互相当过母亲的次数却还不能到尽头。"

龙猛菩萨《亲友书》："**地土抟成枣核丸，其量不及为母数。**"

三组量化比喻展示轮回之漫长：
- **肢体堆积**：即便投生为蚂蚁等小含生的所有肢体堆集一处，也比须弥山王还高
- **泪水汇聚**：因饥寒交迫而哭过的泪水若未干涸而收集，远远多于所有的汪洋之水
- **铜汁铁水**：仅仅生在地狱时所喝过的铜汁铁水，也比四大海洋的水还多

### 善趣安乐也不可靠

"即便是坐在天衣铺陈的如意宝座上尽情享受五种欲妙的天王，当寿命结束后也会在睁眼闭眼的瞬间大头朝下堕入地狱，在炽燃铁地上感受痛苦。"

"太阳和月亮尽管拥有普照四洲的光芒，但最后也会有转生到伸手不见五指、漆黑一片之暗处的时候。"

《正法念处经》："天上欲退时，心生大苦恼，地狱众苦毒，十六不及一。"

### 轮回安乐如梦中富贵

"现在暂时的幸福快乐就好似梦中正在兴旺发达之时突然醒来一样，有什么实质可言呢？"《大圆满心性休息》："如梦富足醒时无。"五蕴聚合一旦散开，名声地位荡然无存，紧紧跟随的只有善恶业力。

### 前后世存在的确立

讲授此科之前需确信前世后世存在。讲解中以王阳明金山寺五十年后开关房的公案、各宗教的轮回观点（古印度吠陀经典、柏拉图、孔子"敬鬼神而远之"、道教元神之说），以及现代的前世回忆案例等进行旁证。修行者须通过闻思在理论上彻底说服自己，而非仅以"佛说的"盲目接受。

### 具体观修的四要素与厌离/出离心（《前行备忘录》）

《前行备忘录》明确指出，观修六道痛苦的内容包括四方面：第一住所、第二身体、第三痛苦、第四寿命。关于六道其余痛苦，也要通过细致入微地思维此四要素而生起厌离心、出离心 (《前行备忘录》)。

当静心思维时心里觉得悲悲凄凄、万分伤感，那就是**厌离心**；生起了想从中解脱的心，那就是**出离心**。出离包括出离的果和出离的意乐，此处是指出离的意乐 (《前行备忘录》)。

观修后应做四件事：(1) 对此具足四种对治力进行忏悔；(2) 立誓从今以后哪怕肉被揪成一块块而死也绝不造此等恶业；(3) 对转生在该道的一切有情知母、念恩、生起报恩心；(4) 猛厉祈祷三宝，念诵"奇哉三宝大悲尊"等祈祷文，遵照全知无垢光父子的观点，要间接对作意自私自利深恶痛绝 (《前行备忘录》)。

### 总思维的结论

"我们要下决心：今生今世一定全力以赴脱离轮回大苦海，获得永久安乐圆满正等觉果位。"圣天论师："若凡夫亦知，一切生死苦，则于彼刹那，身心同毁灭。"——如果凡夫能像圣者一样现量了知轮回痛苦，当下身心就会因无法承受而崩溃。

总思维须明确两点：第一，轮回六道确实存在；第二，必须要从中获得解脱。扪心自问："我现在修加行，是为了今生，还是为来世？若为来世，是想获得善趣快乐，还是希望从轮回中永远解脱？"《业分辨经》："三有之苦，犹如毒药、猛兽、野人般恐怖。"

## 丁二·戊一、地狱之苦（第41–44课）

### 己一、八热地狱

八热地狱从上到下层层叠叠如高楼大厦，"地面与周围全部犹如打造的烧铁一般，一经落脚就没有丝毫的舒适感"。

各经论对地狱状况的描述不尽相同——法王如意宝说："地狱其实是众生心识的显现，由于众生心识千差万别，故地狱的状况也成千上万，不一定只有八热地狱、八寒地狱这几种。"此处以归摄的方式大致宣说。

#### 庚一、复活地狱

**痛苦**：众生由嗔恨引业聚于烧铁地上，如同见到不共戴天的仇敌，手持业力幻化的兵器互相残杀，全部丧命。此时空中传来"愿你们复活"的声音，随即死而复生，又一如既往争斗不休。辗转死去复活，轮番交替。有经中说地狱众生"一日一夜万死万生"。

**寿量**：人间50年 = 四大天王天1天 → 四天王天500年 = 此地狱1天 → 自寿500年。以人间年计约**一万六千二百亿年**。

**转生因**：触犯戒律、杀害众生；较轻的恶作罪未忏悔亦堕此处。《前行备忘录》概括转生地狱的因："虽然总体来说是三毒引发的上品十不善业，但主要是嗔心驱使的杀生等业" (《前行备忘录》)。

#### 庚二、黑绳地狱

**痛苦**：狱卒将众生带到炽热铁地上，以黑线划分为四份、八份、十六份、三十二份等，然后用火红铁锯锯割。刚被锯开的部位马上粘连，反反复复感受剖割之苦。

**寿量**：人间100年 = 三十三天1天 → 三十三天1000年 = 此地狱1天 → 自寿1000年。约**十二万九千六百亿年**。

**转生因**：对父母、亲友加以损害，以妄语欺骗。

#### 庚三、众合地狱

**痛苦**（两种形态）：其一，众生被关在大如地域的铁臼内，狱卒挥舞须弥山般的铁锤锤打，众生在气息分解的剧苦中死去，铁锤举起时复原如初。其二，川谷中相对山岭变成自己前世杀害的鹿子、黄羊、山羊等动物的头像，角尖燃火、角抵角相斗，两山碰撞时中间的众生全部碾碎死去，分开时恢复如初。

**寿量**：人间200年 = 夜摩天1天 → 夜摩天2000年 = 此地狱1天 → 自寿2000年。约**一百零三万六千八百亿年**。

**转生因**：杀害猪、羊、兔子等动物。

#### 庚四、号叫地狱

**痛苦**：众生身陷无门的炽热铁室内，倍受煎熬，想到无有从此解脱之时，不禁失声惨叫。

**寿量**：人间400年 = 兜率天1天 → 兜率天4000年 = 此地狱1天 → 自寿4000年。

**转生因**：身语意三门不如法，尤其是说离间语、挑拨离间。

#### 庚五、大号叫地狱

**痛苦**：狱卒将众生驱赶到**双重铁门**的炽热铁室中，以铁锤锤打。众生想到"内外两个门都是用铁水浇铸，即便能逃脱内门也逃不出外门"，遂声嘶力竭大呼大叫。

讲解中引述一则近代旁证：前苏联地质学家阿撒哥夫博士团队钻探至约九英里深时，探测到地底有巨大空间，拾音器收到阵阵凄惨叫声，"仿佛有数不清的人正在受到极大的痛苦"。阿撒哥夫说："作为一个无神论的共产党员，我一向不相信有天堂或地狱之说。但作为一个科学家，我现在却不能不确信有地狱存在。"此录音曾在美、加、英国BBC台播放。

**寿量**：人间800年 = 化乐天1天 → 化乐天8000年 = 此地狱1天 → 自寿8000年。

**转生因**：盗窃佛、法、僧三宝的财物。

#### 庚六、烧热地狱

**痛苦**：大如三千大千世界的铁器装满沸腾铁水，众生在其中遭受熬煮。每当浮到水面，被铁钩钩住、铁锤击头而昏迷；醒来后因对比而"以为那是一种安乐"——始终处于水深火热之中。

**寿量**：人间1600年 = 他化自在天1天 → 他化自在天16000年 = 此地狱1天 → 自寿16000年。

**转生因**：以火焚烧山川、森林、房屋、草原乃至烧害众生。

#### 庚七、极热地狱

**痛苦**：火势熊熊的铁屋内，狱卒用火焰炽燃的**三尖铁矛**从众生两足掌和肛门刺入，从双肩和头顶径直穿出，并用燃烧的铁片在身外缠绕。

**寿量**：**半个中劫**——已无法用人间年数衡量。

**转生因**：对别人作无因诽谤。

#### 庚八、无间地狱

**痛苦**：周围有十六个近边地狱围绕的燃火铁屋内，众生身体与烈火融为一体，"只能发出凄惨号叫的声音而身体却显露不出来"。有时以为火门稍开而企图逃走，又被狱卒毒打，口中灌注沸腾铁水——需**饱尝前七种地狱的所有痛苦**。

**寿量**：**一个中劫**——再没有比这更强烈的痛苦，故名"无间"。

**转生因**："造无间罪业的众生或者进入密乘对金刚阿阇黎产生邪见的人将投生到这一地狱，而以其他业力绝不会转生于此。"五无间罪（杀父、杀母、杀阿罗汉等）及破密乘根本戒。密乘中还有**金刚地狱**，痛苦和寿量远超无间地狱，破密乘誓言者转生于此。《明誓言续》："失毁根本破誓者……所有普通之地狱痛苦合一亦不及彼狱十万分之一。"《大幻化网释》："纵然这个器世界遭到毁灭，金刚地狱众生的痛苦也不会穷尽。"

**寿量递增规律总结**：从复活到烧热，各地狱寿量与天界对应的层级是四大天王天→三十三天→夜摩天→兜率天→化乐天→他化自在天，换算层层递增；极热为半个中劫，无间为一个中劫——最后两层已超出天界年数换算范围。麦彭仁波切教言："现在就是一颗小火星落在身上，你也会觉得疼痛难忍，那么将来真正在烧铁地上，无量狱火灼烧着你，又怎能忍受得了呢？"

### 己二、近边地狱（五种）（第42课）

无间地狱四方各有煻煨坑、尸粪泥、利刃原、剑叶林四个近边地狱（共十六个），四角各一铁柱山。每种地狱的众生都是从无间地狱中业力减轻后走出，看到一种似乎美好的景象而前往——却是新的痛苦：

#### 庚一、煻煨坑

从无间出来看见远处黑漆漆的凉荫或妙壕，欢喜前往——却陷入剧烈燃烧的炭火坑，骨肉焦烂。转生因：依止出离道而不护禁戒。

#### 庚二、尸粪泥

看见远方河流急奔饮水——却陷入人尸、马尸、犬尸等腐烂尸体的臭泥中，淹没过顶，铁喙昆虫竞相啄食。转生因：诈现威仪、邪命养活。

#### 庚三、利刃原

看见赏心悦目的青草原——整个大地长满如草般燃火铁刺，右脚踏上右脚刺穿，左脚踩下左脚刺透，脚抬起恢复如初再踩又被穿透。转生因：宣说邪道、破正法。

#### 庚四、剑叶林

看见枝繁叶茂的森林——铁树上长着叶状利剑随风摆动，将众生碎尸万段，恢复后又再度割截。转生因：阴谋害他，或以刀等兵器伤害别人。

#### 庚五、铁柱山

**毁坏梵净行或行邪淫者**的转生处。由业力牵引听到山顶昔日苦苦爱恋之人呼唤——"念在我俩昔日情分上，你也见到我了，为何不来亲近我？"——攀登时铁树叶朝下刺穿身体，到山顶被乌鸦鹰鹫啄食眼油及一切支分。又听到山脚呼唤向下奔去，树叶转向上方从前胸刺入穿透后背。到山脚，铁男铁女将其拥入怀中，**将头颅活活吞入口中细嚼慢咽，嘴角两边流出白色脑浆**。

《正法念处经》总结："何因故烧？邪欲为因。"

### 八热地狱与近边地狱的观修方法（第42课）

原论明确的闭目观想指导：

> "到一个寂静的地方开始**闭目观想**：我如今已经真真切切地转生在那些充满恐惧和痛苦的地狱之中。又转念沉思：我现在并不是真正生在那些地狱里，仅仅是内心意念，竟然那么恐怖、那么痛苦，而如今真正堕落于地狱中的众生不可胜数，这些有情都是自己前世的父母，也无法确定现世的生身父母、亲朋好友命终之后不堕入地狱。其实，**转生到那些地狱的主因就是嗔恨心**，而我们在前世和今生之中肯定造下了数之不尽的嗔恨烦恼恶业。"

米拉日巴尊者："恶趣主因为嗔心，舍命亦应修忍辱。"

随后要做三件事：（1）忏悔以往所造的嗔恨恶业，立坚定誓愿"宁死也绝不造堕落地狱的恶业"；（2）对地狱有情生起强烈悲心，发愿"但愿这些众生当下从恶趣中解脱"；（3）以加行（发菩提心）、正行（闭目观修）、后行（回向）三殊胜摄持。

### 己三、八寒地狱（第43课）

"处在中央与周围雪山、冰川的环抱中，到处是狂风四起、暴雪纷飞"，众生赤身露体遭受寒冻：

| 寒地狱 | 命名由来 |
|--------|----------|
| **具疱** | 身上不时长出水泡 |
| **疱裂** | 水泡破裂形成疱疮伤口 |
| **紧牙** | 寒冷难忍，禁不住牙关紧咬 |
| **阿啾啾** | 不断发出呼寒叫冷之声（"好冷啊"） |
| **矐矐婆** | 呼寒叫冷的音声已经间断，只能呼呼呼长声叹息 |
| **裂如青莲花** | 皮肤冻成**青色**，裂成**四瓣** |
| **裂如红莲花** | 皮下之肉冻成**红色**，裂成**八瓣** |
| **裂如大莲花** | 冻成**黑红色**，裂成**十六瓣、三十二瓣或无数瓣**，铁喙昆虫爬入冻裂伤口咬噬，同时遭受寒冻之苦 |

#### 寿量（芝麻计算法）

二百藏升的大盆装满芝麻，具疱地狱每一百年从中取出一粒，所有芝麻全部取完之时，才是具疱地狱众生寿量结束的时刻。其余地狱**次第呈二十倍递增**——疱裂是具疱的20倍，紧牙是疱裂的20倍，依此类推直到裂如大莲花。《俱舍论》："芝麻器中每百年，取出一粒至穷尽，即是具疱地狱寿，余寿渐成二十倍。"

#### 转生因

三种主要因：（1）让众生在寒冷中冻死（如将活物放入冰箱）；（2）持断见（否认因果前后世）；（3）毁谤圣者。

公案：瞿伽离比丘因毁谤舍利子和目犍连堕入大莲花地狱——佛陀亲自劝告也不听。

#### 寒地狱观修法

"首先观想：在我们人间，寒冬季节一丝不挂赤裸裸地在外面停留一瞬间，尚且无法忍受，假设真正转生在那些地狱，又怎能忍受得了呢？"以前有些上师在冬天最冷时不穿衣服去亲身感受，由此引发对轮回的猛厉出离心。

观想步骤：（1）想"我以后堕入寒地狱怎么办"；（2）想"我正在寒地狱中亲身感受痛苦"；（3）想"虽然如今我没转生于此，但从无始以来所造的业中肯定有许多寒地狱之因，没有把握将来不会堕落，我该怎么办"。然后转念："我现在并未堕入地狱，忏悔还来得及。"同时想到父母亲友可能正在寒地狱受苦，对他们生起强烈悲心。如前一样具足三殊胜来修。

### 己四、孤独地狱（第43–44课）

"处所没有固定性，痛苦也是变化不定。"

各种形态：夹在山岩间、困在磐石内、冻在冰块里、煮在沸水中、烧在烈火内。有些藏在树木里——当樵夫砍伐时肢体被千刀万剐。还有些**转生为人们日常使用的杵臼、笤帚、瓦罐、门、柱子、灶石、绳子等形象**，"也以识蕴感受这些苦楚"。

阿罗汉目犍连到海边时，见到许多孤独地狱众生——有些在石头里，有些在树木里，以各种无情物形象感受无量痛苦，悲叹："三有无安乐，孤独地狱众，分别为苦迫，如住炽火中。"

关于孤独地狱的命名，无垢光尊者在《心性休息大车疏》中指出：曾有五百比丘共同感受孤独地狱之苦，故此地狱并非以数量微少而安立，而是因处所不固定、痛苦变化不定的特点得名。

#### 卓滚朗吉日巴与雅卓耶湖公案

大成就者卓滚朗吉日巴（1128–1188）向雅卓耶湖中观望后竟然悲泣着说："**哎哟哟，千万不要享用信财！千万不要享用信财！**"

弟子请求上师示现。尊者大显神变令湖瞬间干无一滴——眼前出现一条巨鲸，**庞大的躯体遍及整个湖泊**，身上密密麻麻布满数不胜数的含生争相蚀食，只见它痛苦难忍而满地翻滚。

此鱼是**后藏黑马喇嘛**的转世。黑马喇嘛咒力非同小可，对鬼迷心窍的人只需看一眼就立竿见影，成了前后藏四翼地区众所周知的供养处。"他超度亡人只念一声'啪的'便开始收取大量牛马等牲畜"——死后转生为这条大鱼。

#### 唐东加波与石中青蛙公案

大成就者唐东加波在一块大石头上修气脉瑜伽，以修证力量石头裂成两瓣。里面有一只大青蛙，身体上粘附无数微小生灵在蚀食，"只见它张着黑洞洞的口，苦不堪言"。上师告弟子："这个众生的前世是一个作血肉供养的上师。"

类似公案：大堪布具德护法在德格期间，一天对弟子说"今天俄达河里出现什么都不要放掉"。僧人们守候至午后，打捞上一截随波漂来的树干。劈开后惊见一只大青蛙被众含生蚀食。上师做了沐浴仪轨超度，说这是德格管理信财的监院师哦吉的转世。

华智仁波切因此警示："今生这样肆无忌惮地享用血肉的人，后世要用自己的身体来偿还血债，到那时，孤独地狱中不堪设想的大苦头会恭候着他们！"

#### 昼辛吉尊者所见之孤独地狱（日夜交替苦乐）

往昔世尊时代有屠夫**晚上守持不杀生戒**（白天大开杀戒）。死后堕入孤独地狱：**夜晚**身居美妙宫殿，四名美女供奉饮食受用，快乐无比；**到了白天**，美宅变燃烧铁屋，四美女变恐怖花斑恶狗啃食他。

另有邪淫者**白天守持不邪淫戒**（晚上非法邪淫）。死后与前相反——白天享乐，夜间受苦。二者皆为昼辛吉尊者亲眼目睹。

#### 五百比丘公案

一座环境幽雅的寺庙中住五百比丘。每天中午击犍椎集僧供斋时，经堂即刻变燃烧铁屋，钵碗变兵器，僧人互相殴打。供斋时间过后又恢复如故。此为**迦叶佛时许多比丘午饭时发生争执**的异熟果报。

#### 十八地狱与比丘的笑话

八热八寒共十六个，加上近边地狱和孤独地狱合计十八个——通常所说的十八层地狱。一位自称广闻博学的比丘被问十八地狱是什么，答："八热八寒十六……再加上噶玛巴黑帽、红帽两个。"——并非不敬噶玛巴，而是忘了孤独地狱和近边地狱的名称。

## 丁二·戊二、饿鬼之苦（第45–47课）

### 己一、隐住饿鬼

饿鬼的根本因：《成实论》："于饮食等，生悭贪心，故堕饿鬼。"《正法念处经》："一切饿鬼皆为悭贪嫉妒因缘，生于彼处。"《正法念处经》详列36种饿鬼，各有不同业因。

#### 庚一、外障饿鬼

"**甚至在数百年之中连水的名称也没有听过**，整日饥渴交迫，经常为寻找饮食而四处游荡。"

四种痛苦情形：（1）看见远处清澈江河，以僵直的肢体艰难支撑着巨大的腹部赶去——到了近前所有水已干无一滴，仅剩河床。（2）远望果实累累的绿树赶去——到了跟前所有树木已干枯为树干。（3）看见品种繁多的饮食——到了近前遭到全副武装的看守用兵器殴打驱赶。（4）**四季颠倒**：夏季月亮火烧火燎，冬季太阳寒气逼人。《亲友书》："诸饿鬼界春季时，月亮亦炽冬日寒。"

**昼辛吉尊者求水公案**：尊者到饿鬼境内，中了饿鬼吝啬毒气，口干舌燥。见铁城大门前黑面红眼饿鬼，急问"哪里有水"——话音刚落聚来一大群形似烧焦树干的饿鬼乞求："给我们一点水吧。"尊者："我也得不到水。你们到底是谁？"答曰："**我们自从投生在这座山谷以来已经有十二年了，可是到今天才听说水的名字。**"尊者问转生之因，饿鬼答："在人间时，吝啬惜财、不施与人，还常常嗔恨骂人。"

#### 庚二、内障饿鬼

"**嘴巴小得像针眼一样**，本想开怀畅饮大海里的水，怎奈水却无法流进它们那细如马尾毛的咽喉，而且在这中间已被口中的毒气一扫而光。就算是有一星半点的水进入了咽喉，也满足不了它们那**大如盆地的腹部**。即使腹内稍稍有一点饱的感觉，在夜晚期间腹内也会燃起大火，烧尽心肺等所有内脏。"灰白色茅草般的肢体难以支撑大如盆地的腹部——痛苦到了极点。

#### 庚三、特障饿鬼

"在每一个特障饿鬼的庞大躯体上，都居住着成群的小饿鬼，这些小鬼不停地啖食着大鬼。"

**昼辛吉尊者与美女饿鬼公案**（完整故事）：

尊者进入饿鬼界的一座无量宫殿，见一位婀娜多姿的美女饿鬼，珍宝宝座四条腿上各拴一个饿鬼。美女饿鬼供食并嘱咐："一丁点儿也不要给它们。"

尊者心生怜悯，将食物分给四个饿鬼——结果：第一个所得变为**糠秕**，第二个变为**燃烧的铁丸**（铁锤），第三个变为**它自己的肉**，第四个变为**脓血**。

美女饿鬼回来不满。尊者问关系，她说："这是我前世的**丈夫**，这是我的**儿子**，这是我的**儿媳**，这是我的**仆女**！"

转生因缘：美女饿鬼前世是婆罗门女，于佳节前夕供养嘎达亚那尊者斋食。告知丈夫请他随喜——丈夫火冒三丈："在没有供奉婆罗门之前先供养那个光头，光头怎么不去**吞糠秕**！"告知儿子——儿子："光头怎么不吃**铁锤**（铁丸）！"亲戚送来美味被儿媳独吞，质问时儿媳赌咒："我吃你的食物，还不如**吃自己的肉**！"让仆女捎食物给亲戚被偷吃，质问时仆女赌咒："我偷吃你的食物，还不如**喝脓血**！"

四人的恶口赌咒全部成为在饿鬼道中感受的实际痛苦。

美女饿鬼因供养圣者的功德本应生三十三天，但因对家人发了**恶愿**"但愿我将来转生在能看到他们感受报应的地方"而也堕为饿鬼。她请求尊者转告已沦为娼妓的女儿：取出父亲房间里四个装满黄金的铜锅及金盘金瓶，时常供养嘎达亚那圣者，**念诵我们的名字作回向**——可使业力逐渐减轻直到消尽。

**哲达日阿阇黎与金刚座公案**：饿鬼母有五百个孩子，请尊者捎口信给去金刚座寻食十二年未归的丈夫。尊者问如何辨认，饿鬼母说："大嘴巴、塌鼻子、小眼睛具足九种丑相。"尊者在金刚座找到它时，那饿鬼诉苦："十二年中只有一次在一位清净比丘擤鼻涕时，众多饿鬼蜂拥争夺，我才得到了一点点，**而且争抢时被其它饿鬼打得鼻青脸肿、遍体鳞伤**。"

### 己二、空游饿鬼（第47课）

包括妖精、王鬼、死魔、厉鬼、鬼女、独角鬼等。

"始终都处于提心吊胆、担惊受怕以及恍恍惚惚的错觉状态中，经常居心不良，怀有歹意，精勤于害他的恶业，它们死后的下场多数都是立即坠入地狱等恶趣的深渊。"

特殊痛苦：**不管生前是因病身亡、利刃所毙，还是悬梁自尽等途径横死的，每隔七天都要感受一次以那种方式死亡的痛苦。**它们想把这种痛苦转移给别人——本来满心欢喜来到昔日亲友面前，结果却使他们重病缠身或精神疯狂等。

瑜伽士作诅烧抛仪轨时，饿鬼"被镇压在地下黑咕隆咚的地方长达数劫"，或"被烧在作火施的烈焰中"，或"咒师抛撒驱邪芥子，头颅裂成百瓣、肢体断成千截"。它们还要感受一般饿鬼所感受的**冬季太阳寒冷、夏季月亮酷热**等反常苦恼。

### 饿鬼的根本因

"投生为饿鬼的主因就是自己一毛不拔的吝啬和阻碍他人布施的悭吝。"《业报差别经》列十种饿鬼因：身口意轻恶业、多贪、恶贪、嫉妒、邪见、贪著资生即命终、因饥而亡、枯渴而死。

### 饿鬼观修要点

"想想看，我们这些人，仅仅没有吃早饭，便会觉得何等的痛苦？如果真的转生到长年累月连水的名字也听不到的地方那将如何面对呢？"以三殊胜摄持诚心观修。

日常应施食给饿鬼：念变食真言（纳美萨瓦达塔嘎达 阿瓦洛格得 嗡 桑巴Ra桑巴Ra吽）加持食物七遍，洒水后置于户外或烧掉——烟所接触的众生皆得利益。同时念四位如来名号（南无多宝如来——破悭吝恶业令福德圆满；南无妙色身如来——破丑陋恶形令色相具足；南无广博身如来——令咽喉宽大所施之食恣意充饱；南无离怖畏如来——令恐怖悉除离饿鬼趣）。

此仪轨源于佛为阿难尊者说之施食法——面然饿鬼告阿难三天后将死堕饿鬼道，佛以特殊方便令阿难以一咒即可布施无量饿鬼。汉地寺院每天晚课"放蒙山"即是此法传承。

## 丁二·戊三、旁生之苦（第48课）

### 己一、海居旁生

"在一望无际的大海当中，鱼、鲸、螺、龟、虾等就像酒糟一样密密麻麻。"大者身量可围绕须弥山数周，小者如微尘或针尖。

核心是**弱肉强食**："大的吞食小的，小的径直刺入大的身体蚀食它们，每一个庞大动物身上都有成群的小含生筑窝居住并以其为食。"《大楼炭经》描述海中有身长四千里、八千里乃至三万二千里的大鱼。有的生在暗无天日的岛屿上，多少万年中连自己屈伸肢体也无法看见，格外痛苦。"作为旁生几乎都是呆头呆脑、愚昧无知而根本不懂得取舍的道理，终日处在无边无际的痛苦之中。"

### 己二、散居旁生

**龙王**：常遭大鹏鸟威胁、热沙雨降临之害，而且愚痴呆笨、毒气冲天。

**野兽**："无有主人饲养的野兽等**时时刻刻都处于万分恐惧的心态中**，即使吃一口食物也不得安稳，经常面临互相啖食、遭猎人捕杀、被猛兽吞食等险情。"猎人精通伤害手段——陷阱、网罟、火箭，瞬间使其丧命。

因自身而遭杀：为珍珠采杀海贝，为象牙杀大象，为兽皮杀虎豹水獭狐狸，为麝香杀獐子，为血肉杀野牛野马——"这些动物自己的身体反而成了送命的因"。

**家畜**："由于愚痴呆笨的缘故，就连屠夫手拿刀剑来到面前时，也只是眼睁睁地看着，根本不知道逃避。"被挤奶、驮运货物、阉割、穿透鼻孔、辛勤耕地——"牛马等牲口背上即使已经伤痕累累却仍然要驮运货物、被人乘骑而艰难行路。当它们实在走不动时，狠心的主人就用鞭子抽打或用石头猛击这些牲口，从来也不曾想过身为动物同样有辛苦疲惫、病患疼痛。"

"**成群的牛羊，从身强力壮到老气横秋，只要还没有到派不上用场或气绝身亡之前，就无有休止地被主人使用，一旦衰老得不成样子的时候，或者被主人一刀结果性命，或者被卖给别人，无论如何都摆脱不了被宰的厄运，自然死亡的几乎一个也没有。**"

此外，马戏团中老虎钻火圈、猴子耍火棍、大象单腿站立等表演的背后无不充斥着棍棒交加和残忍手段。《亲友书》："有因珍珠有因毛，血肉骨皮而遭杀。"作为旁生，所受的痛苦实在是常人无法想象的。

### 旁生的观修要点

"每当我们目睹遭受这样痛苦的众生时，应该深深地思索、设身处地观想，如果这般剧烈的苦痛落到自己身上，那将如何忍受得了？"

对总的旁生界有情生起强烈悲心，尤其对自己饲养的动物多一分仁慈。《梵网经》："若佛子常起大悲心……若见牛马猪羊一切畜生，应心念口言：汝是畜生，发菩提心。"乃至虫蝇及细微含生以上的所有动物都同样有苦乐的感受，而且无一例外都当过自己的父母。

### 旁生的转生因

《辩意经》："有五事作畜生：一犯戒私窃；二负债不还；三杀生；四不喜听受经法；五常以因缘艰难斋会。"《大宝积经》："傍生趣者无量无边，作无义行，无福行，无法行，无善行，无淳质行。"作为旁生愚昧无知、没有福德善根、没有行持善法的机会——即使能活上万年也连一句观音心咒都念不了。

## 三恶趣的总结与修行要点

原论总结："**不管是投生在三恶趣中的任何一处，都必然要受苦受难，而且这种苦难的数量也是多不可数，程度也是无比剧烈，时间也是极其漫长。更悲惨的是，三恶趣的有情由于愚昧无知、无有正法光明致使所作所为仍旧逃不出恶趣的因。一经生到恶趣后，就很难再有出头之日。**"

三恶趣有情因愚昧无知而所作所为皆逃不出恶趣之因——"一经生到恶趣后，就很难再有出头之日，也可以说要想再度解脱实在是难之又难"。与之对比，我们作为人，修行再差也能制止恶行，再差也可以每天念一百遍观音心咒——这就是人与恶趣众生的根本差距。

修行要求：（1）诚心忏悔以往恶业，立誓今后绝不再犯；（2）对恶趣众生生起强烈悲心；（3）发愿"将自己三世所积的一切善根回向给沦落恶趣的这些众生，但愿它们能早日脱离恶趣"；（4）发心"我如今有幸遇到大乘正法，一定要刻苦求法、精进修行，将来好接引恶趣所有众生到清净刹土"；（5）以回向等三殊胜摄持来实地修持。

每次行善——听一堂课、念观音心咒、放生、闻思、背诵——都要通过回向文（如《普贤行愿品》）将善根回向三恶趣众生，这是必不可少的日常修行。

龙猛菩萨教言："即便见闻地狱图，忆念读诵或造形，亦能生起怖畏心，何况真受异熟果？"仅仅见到地狱的图画、听到痛苦的描述，有善根的人就会毛骨悚然，何况真正去感受？因此，大家今后对因果一定要注意，对上述道理要踏踏实实去修持。

回向偈：

> **虽得闲暇而乏真实法，虽入佛门而耽非法行，**
> **我与如我愚痴诸有情，获得暇满实义祈加持。**
