# 因果不虚·十不善业与果报（第59–67课）

## 在全论中的位置

因果不虚是共同外前行第四个引导文。前三个引导各有功能——人身难得令人珍惜、寿命无常令人紧迫、轮回过患令人厌离——而因果不虚则揭示**决定去向的机制**：众生在善趣恶趣之间流转，不是造物主的安排，也不是偶然因缘的产物，而是善恶业力的必然结果。

顶礼偈：

> **取舍善恶因果依教行，行为依照九乘次第上，**
> **真知灼见于何皆不贪，无等上师足下我敬礼。**

核心原则："轮回是由业力所生、由业果所成，上升善趣或下堕恶趣并没有其余作者，也不是由偶尔的因缘所生。"《心地观经》云："诸法无不因缘成，若无因缘无诸法……无因无果大邪见，不知罪福生妄计。"

修行人的唯一标志："相不相信因果。如果你不信，表面上再装成什么样子，也不过是在虚假的骷髅上套个面具而已。"

四个引导文的逻辑递进：人身难得→珍惜修行；寿命无常→修法要有紧迫感；轮回过患→不能只求人天福报、必须超出三界；因果不虚→遵循善知识教言行持善法，"将来才会得到快乐，否则前方永远是一片黑暗和痛苦"。

《正法念处经》云："汝自作恶业，汝如是自食，非此人作业，余人受果报。"自作自受是因果的基本法则——造了恶业就要自食其果，不可能让另一个人代替承受。

## 丁一、所断之不善业

不善业虽无边无际，从积业的侧面归纳，就是身三不善（杀生、不与取、邪淫）等十不善业。不善业的引发者是三毒——《中观宝鬘论》："贪嗔痴及彼，所生业不善。"三毒是**本性不善**，由三毒引生的业叫做**相应不善**。(《前行备忘录》)

按照理论的讲法，不善业是由烦恼引发的心和心所；依照窍诀的观点而言，不善业是由烦恼引发的分别意念。在萌生烦恼时，必然要依靠对境而起——依靠悦意的对境生起贪心，依靠不悦意的对境生起嗔心，依靠中等的对境生起痴心。(《前行备忘录》)

### 戊一、身恶业

#### 己一、杀生（第59–60课）

**杀生四支分**（原论逐一详述，以猎人猎杀野兽和宰羊两例说明）：

1. **对境（基）**：无误认知所杀对象是某一众生——"认准了这是某某野兽"
2. **意乐**：生起欲杀之心——"怀着嗔心等烦恼想杀他"
3. **加行**：采取杀害的手段——"用火箭、枪等击中要害"
4. **究竟**：断绝命根——"使它身心的聚合瓦解"

若想杀甲却误杀了乙，虽有罪业但未圆满四支分。未亲手杀而仅具前两支分（了知对境+生起意乐），"如同镜中映现影像一样，罪业已经染污了自相续"。

**三种杀生分类**：

| 类型 | 烦恼驱动 | 典型例证 |
|------|---------|---------|
| 嗔心杀 | 嗔恨 | 将士战场杀敌 |
| 贪心杀 | 贪欲 | 贪图肉食、穿戴皮革 |
| 痴心杀 | 愚痴 | 为亡人杀生祭祀、外道认杀生为善业 |

**无间业**——杀父、杀母、杀阿罗汉，属最严重的杀业，"在今生与来世之间不经过中阴而径直堕入无间地狱"。上等杀生杀阿罗汉/父母，中等杀住道者，下等杀人和旁生。

**杀生的普遍性与连带责任**（原论极详）：

- **施主供僧杀生**：上师和僧人到施主家时，施主宰杀家畜供养——"施主和福田将无任何差别地获得同等杀生罪业"。《楞伽经》："为利杀众生，以财网诸肉，二俱是恶业，死堕叫呼狱。"
- **牛羊间接杀生**：春季虫蝇蚂蚁被牛羊连同草料吞进或前蹄后蹄践踏致死，马粪牛粪中死亡的含生数不胜数，"这些杀生的罪业也将一并落到主人的身上"。
- **羊只的详细分析**（原论极具体）：羊以小蛇、青蛙、鸟蛋等微小含生为食；春季毛纺时每只羊背上"大约有十万含生全部丧命"；冬季产羊羔时一半刚出生便被宰杀，皮制成帽子衣服；母羊精华耗尽后被宰杀皮肉享用；公羊"无论到哪里都只有死路一条"；羊身长虱子剪毛时"每只羊背上大约一亿含生会丧命"。华智仁波切断言："拥有一百头以上羊只的主人必将堕一次地狱。"
- **女人的杀业**：订婚、婚宴杀无数羊只；每次回娘家杀一个众生款待；红面罗刹女不吃素食而"二郎腿一跷，掏出小刀津津有味地吃起来"，背着血淋淋的全牲肉"好像猎人返家一样回去了"。
- **儿童的杀业**：游戏时看见或没看见中杀命无数，夏季用牛鞭抽打大地打死含生不计其数。许多孩子小时候杀青蛙、蝌蚪、蜻蜓、蝴蝶、蚯蚓、蚂蚁。"如果孩子从小没有善良的天性，后天也不培养慈悲心，那他小时候会杀小鸡、小鸟雀、小兔，长大后就会杀牛、杀马、杀狮子、杀大象，甚至一旦成了科学家，还可能会制造原子弹、核武器。"因此童年的慈悲教育极为重要。
- **母牛的恩德**：母牛一生为人类提供牛奶如同父母养育我们，恩深似海——"可我们又是怎样对待它们的呢？就是将它们杀了，喝它们的血，吃它们的肉。"人类在这方面"比罗刹更狠、更恶劣"。

**杀生的罪业为何最大**：因为众生最爱惜自己的生命，为维护生命甚至愿受各种痛苦。《大智度论》："世间中惜命为第一。"不管虫类人类，苦乐感受并无差别——"大人被杀时很痛苦，小孩被杀时也一样。同理可知，小虫虽然身体微小，但被杀害的时候，也照样有剧烈的苦受。"

**不杀生的大乘价值**：《大智度论》："好杀之人，有命之属皆不喜见；若不好杀，一切众生皆乐依附。"不爱杀生的人有一种祥和气质，众生都愿意接近。堪布强调即使修行不精进、偶尔造了些业，"最重要的是下半生不会再杀生了。仅仅是这一个功德，你获得人身也有意义了。"

**关键教义——参与杀生每人得完整罪业非平分**：

> "凡是参与杀生的每一个人都将得到杀生的整个罪业，而不是杀生这一罪业由许多人平均分配。"

《俱舍论》："军兵等为同一事，一切人均如作者。"十人点一条鱼，每人都有杀一条鱼的恶业，不是十分之一。共同做功德亦是同理。

唆使他人杀生，按小乘戒律等同亲自杀生；按大乘菩萨律仪更严重——"因为自己杀只是自己染上罪过而已，但让别人杀的话，既害了自己，又害了别人。"

**人比魔鬼更可怕的公案**（藏地家喻户晓）：盗贼与魔鬼相约害牛。魔鬼只是把一块木屑扎入牛蹄，要等它化脓感染慢慢死去。盗贼不满太慢，亲自去偷宰了一头牛，很快扛回血淋淋的牛肉。魔鬼惊讶道："这么快！莫非你是比我还厉害的鬼？"不敢吃牛肉远远跑开。说明"人类想要杀生的话，当场就可以杀掉、吃掉"，比魔鬼还恶劣。

**杀生果报**（《佛说分别善恶所起经》）：生生世世短命；常遭恐怖；仇敌多；死后堕地狱；从地狱出得人身仍短命多病。杀害一个众生需偿还五百次命债，《念住经》："杀害一个众生，需在地狱住一中劫。"

**以善举为借口杀生罪更重**：帕单巴尊者："依恶建造三宝像，将被后世风吹走。"把上师僧众迎请到家中以荤腥血肉供养——"施主供养成了不清净的供养，上师也成了邪命养活，此种罪业远远超过了所行的善事。"除非上师有起死回生的超度能力，一般人接受这样的供养，"不可能不被杀生的罪业染污"。

《大智度论》："世间中惜命为第一。何以知之？一切世人，甘受刑罚、刑残、考掠以护寿命。"生命是最珍贵的，人被杀痛苦，小虫被杀也照样有剧烈的苦受。

#### 己二、不与取（第61课）

**三种分类**：

| 类型 | 定义 | 例证 |
|------|------|------|
| 权威不与取 | 以非法暴力强取豪夺 | 国王以军队明目张胆掠夺；1840年列强侵占中国 |
| 盗窃不与取 | 趁主人未见暗中窃取 | 盗贼窃取饮食财物 |
| 欺诳不与取 | 以谎言、短斤少两、非法秤斗等骗取 | 经商欺诈——"通过尔虞我诈的欺诳手段经营，无论赢得多少利润都与直接偷盗没有差别" |

**出家人经商的过患——米拉日巴与经商僧人公案**：

一位僧人整夜在床上盘算杀一头牦牛后如何销售——头赚多少、大腿多少、肩胛多少、小腿多少——"除了尾巴以外一切均已计划妥当"。天亮后装模作样念经礼佛，却嘲讽仍在入睡的米拉日巴尊者。尊者答道："本来我平时是不睡懒觉的，但昨夜我一直在考虑如何出售那头被杀的牦牛，没有空出时间来睡觉……牦牛的尾巴还没算好。"一语揭穿僧人内心丑恶。

**经商过程中具足八种不善业**（原论详列）：

| 不善业 | 在经商中的表现 |
|--------|-------------|
| 妄语 | 低劣商品谎称高价买进 |
| 离间语 | 在买卖双方间撒谎制造不和 |
| 恶语 | 口出不逊说对方物品太次，因欠款吵得天翻地覆 |
| 绮语 | 无意义地评论价格，不想买也讨价还价 |
| 贪心 | 想把对方财物据为己有 |
| 害心 | 想把他人弄得一败涂地 |
| 杀生 | 杀羊杀鱼做买卖 |
| 不与取 | 欺诳手段本身即是 |

"十不善业中除了邪见和邪淫以外全部已经直接具足"——间接也可具足此二者。

不与取同样具有四支分。"甚至仅仅给猎人或强盗少许口粮，也将分毫不差地得到他们杀生或不与取的所有罪业。"

**经商导致的毁灭**：经营不顺则倾家荡产，经营顺利则"不论赚了多少也不知满足，一直贪得无厌"。"生意场上的人们平时心里装的就是欺骗他人的阴谋诡计，就像刀刃、矛尖、针尖一样与别人针锋相对，完全背离了饶益他众的菩提心。"即使出了家做生意的人，恶心也很长时间转不过来，"连晚上睡觉时也一直在考虑经营的账目"，最终"身不由己地在迷茫中打转转"，临终时"只能是手抓胸口，悲惨地堕入三恶趣"。

佛陀在《杂阿含经》中教导合理的财富分配：将赚来的钱分为四份——一份自己使用、两份用于投资、一份储存以备不时之需。《妙法圣念处经》："若于有情所，如同父母想，能离不与取，智慧福无量。"

#### 己三、邪淫（第62课）

邪淫是针对在家人的禁戒。出家人则须从根本上杜绝非梵行。

**具体分类**：

- **非对象**：与他人妻子或已付赏钱的女人行淫；与直系亲属行淫；与未成年少女行淫
- **非时**：白天、受斋戒日、生病期间、妊娠期间、忧愁所迫、月经期间、产后未恢复
- **非处**：在三宝所依之处交欢；在口和肛门等处行淫
- **其他**：男人自己出精；与旁生或尸体行淫

《正法念处经》："若灭贪火者，以智慧为水，不灭贪心人，解脱不可得。"断邪淫的根本方法在于以无我智慧对治贪欲。

印光大师《寿康宝鉴》列邪淫十二种危害：害天伦、害人节、害名节、害门风、害性命、害心术、害风俗、害阴骘、害名利、害寿命、害祖父、害妻子。

断邪淫后的功德——《大般涅槃经》："不犯他妇女，自妻不非时，施持戒卧具，则生不动国。"能断除邪淫并对持戒清净者布施卧具，来世即可转生于清净刹土。

**佛教的夫妻伦理**——《善生经》中佛陀详尽阐明丈夫对妻子的五项职责：怜念妻子、不轻慢、为作璎珞严具、于家中得自在、念妻亲亲。妻子对丈夫亦有十三事善敬顺承的规范。

**在家人断淫欲的实例**：印度"圣雄"甘地于37岁立下夫妻终生禁欲誓言并始终奉行不渝；梁武帝50岁起断绝房事。说明"纵然是有地位有权势的在家人，也可以过非常清净的生活"。

### 戊二、语恶业（第62–63课）

#### 己一、妄语

**三种分类**：

| 类型 | 定义 | 严重程度 |
|------|------|---------|
| 一般妄语 | 怀欺骗心所说的一切虚言 | 《正法念处经》："若人离实者，善人说如狗" |
| 大妄语 | 行善无功德、造恶无罪过、佛无功德等 | "再也不可能有比这更严重的弥天大谎"；《大宝积经》："当知妄语，为诸恶本" |
| 上人法妄语 | 未得说得、未证说证——没有登地说登地、没有神通说有 | 末法时代最流行的欺骗 |

**假活佛公案**（堪布列举两例）：一个藏地盗贼因偷盗被罚两万，穿上僧装到汉地冒充上师得几十万供养，回来后笑称"本人的聪明才智加佛陀法衣的加持，盗贼变成了上师"。另一位"活佛"被弟子问何为大圆满，答"不就是个吃的东西"，后以六世达赖情歌冒充灌顶，被录音揭穿。

"我们不能随随便便轻信自欺欺人的大骗子，一定要依止一位非常熟悉、谦虚谨慎、表里如一的修行人作为善知识。"依止上师"不一定要有名声、财富或相貌端严"，最关键的是"有大悲菩提心，具有能摄受弟子的正法，并能如理如实地行持"。法王如意宝一直强调："我不是伏藏大师列绕朗巴的化身，我没有什么神通神变"——真正的大德从不自我标榜功德。

#### 己二、离间语

**两种分类**：

- **公开离间语**：有权势者当面在两人同在的场合挑拨——"这个人暗地里对你如何谩骂，还明目张胆迫害你……"
- **暗中离间语**：在背后对一方说另一方的坏话——"你对他情真意切，可他对你却品头论足。"

破和合僧是离间语中最严重的，《五分律》："众聚和合乐，和合常安隐，若破和合僧，一劫地狱苦。"反过来："若和合破僧，一劫生天乐。"密宗上师与弟子之间、金刚道友之间的挑拨离间"罪业更是重上加重"。

堪布举例：有些上师的弟子、同一位上师的弟子之间"水火不容、势不两立"——"一个人来的时候，另一个人马上离开"，用因明术语叫"互绝相违"或"互不共存"。金刚道友之间若是如此"相当不好"。"既然彼此是同行道友，若连互相说话的余地都没有，那么见到怨敌时该如何对待？面对与自己毫无关系的众生，又怎么以大乘精神去度化？"

离间语的对治：如果有人之间有矛盾，"有能力的话就应想办法去调解，只要能令其和好，即使出钱出力也值得。"《大般涅槃经》："莫坏善知识，远离恶眷属，口常和合语，则生不动国。"

呻号比丘公案（《撰集百缘经》）：前世富翁甲向国王谗言导致乙遭酷刑——以此恶业转生为满身生疮的呻号，后遇佛陀说法当下病愈，精进修行证阿罗汉。

#### 己三、恶语

对相貌丑陋者公开宣扬缺点，对盲人聋人当面称"瞎子""聋子"，起绰号嘲讽身体缺陷——"虽无箭尖利刃语，亦能刺入人心间。"

**劫比罗咒骂比丘的公案**（《根本说一切有部毗奈耶》）：迦叶佛时代，婆罗门之子劫比罗精通三藏但未证圣果。母亲怂恿他与比丘辩论，理屈词穷时按母亲计策咒骂比丘为"马口、骆驼口、狮子口、牛头……"等十八种恶语。以此恶口詈骂僧众之业，命终后堕为摩竭大鱼——身有十八个头、三十六只眼睛，"在达一劫的漫长时间里不得解脱"。佛陀带领比丘亲临现场，以神通加持此鱼回忆前世并讲说人语。法王如意宝经常讲述此公案，其上师托嘎如意宝也常提起并要求弟子不得给人取绰号。

另一公案（《百业经》）：释迦牟尼佛在世时，一位比丘尼讲经说法很厉害、供养也多，她发心以所得财物供养僧尼维持生活。一次她请僧众帮忙急事，大家诵经忙碌不能来。她心生嗔恨，口出不逊："我平时对你们事事关照，你们谁都不肯帮忙，简直像母狗一样！"以此恶口谩骂僧众之业，五百世中转为母狗。

**恶语的深远影响**：修行差的人听100天教言可能一偈也记不住，"但若有一次批评他10分钟，他会记性特别好，永远把它记在心里"。萨迦班智达："伤害他人之恶语，即使怨敌亦勿说，否则如同谷回声，立即自受报复也。"

**恶语也包括温和方式的不善语**：尽管不是粗恶语，但通过温和的方式使对方心不愉快，也属于粗恶语。藏地俗话："粗恶之语无多少，欢喜之食无大小。"《正法念处经》："恶口破慈心，智者能舍离，常乐说软语，则生于天上。"

**对境严厉性**："对一位菩萨妄加诽谤、恶语中伤，这比杀害三界所有众生的罪过还严重。"——《极乐愿文》："诽谤诸菩萨之罪，较杀三界有情重。"

#### 己四、绮语

范围广泛：婆罗门咒语不是正法反认为是正法；世间小说电影；淫秽语言；撩拨贪心的靡靡之音；军事武力抢劫盗窃的高谈阔论——"凡是能引生贪嗔痴的无稽之谈，统统属于绮语"。

"别人诵经念咒时，口若悬河地谈论令人散乱的话题，会断绝别人行善的资粮，罪过极其严重。"僧众行列中一人废话连篇，"全体僧众的善资都会断送在他手中"。

光孝安禅师定中所见：两僧交谈——初谈佛法有天神护卫谛听；后叙旧事天神离去恶鬼来到；再谈施主财物供养，恶鬼唾骂并扫掉他们的脚印。

**信财享用的过患**：黑财"就像燃烧的铁丸子，除非具足生圆次第双运铸铁之牙齿的人才有能力享用"，普通人享用只会"自讨苦吃，焚毁相续"。"黑色信财乃是生命之利刃，过分享用斩断解脱之命根。"有些人念诵仪轨时到了最重要的念咒期间，便打开绮语的"伏藏门"说各种闲话混时间，"结果只会损人害己"。

《正法念处经》："多集绮语句，能令心意乱，破坏于梵行，妨碍涅槃道。"《受十善戒经》："若能不绮语，口常出妙香，犹如优钵罗，生处得值佛。"

### 戊三、意恶业（第64课）

#### 己一、贪心

对他人财物打如意算盘——"这财物为我所有该多好"，并三番五次思量如何弄到手。此处的贪心主要指贪财。《正法念处经》："贪心甚为恶，令人到地狱。"

**鹅王叼宝珠公案**（《毗奈耶经》）：一父亲转生为鹅王，每天从宝洲衔一颗宝珠放在贫穷女儿家门口。女儿不舍得用，日积月累攒了许多。后来她起贪心："它肚子里可能还藏着很多"，便布网想抓住鹅王。鹅王见此"不知恩图报反而恩将仇报"，展翅飞走不复回来。天人偈："不应作多贪，贪是罪恶事，若作多贪者，所获皆散失。"

《佛所行赞》："富而不知足，是亦为贫苦；虽贫而知足，是则第一富。"

#### 己二、害心

对他人痛恨在心，想用武器或语言损害；见他人荣华富贵便不高兴而暗自诅咒；他人遭遇不幸则幸灾乐祸——"凡是对他人生起损恼的心理都属于害心"。

"心善地道亦贤善，心恶地道亦恶劣"——好人坏人以心好心坏来分，不以外在区别。《诸法集要经》："由心乐损害，人神咸不护。"麦彭仁波切："害心后世常受苦，无缘无故遭他害。"

三国太监放鼠屎于蜜汁梅以陷害库吏的公案：太监与库吏有嫌隙，取蜜汁梅时悄悄放入鼠屎献给吴王。吴王命人切开鼠屎发现外湿内干——若早就在蜜中必已湿透，说明是临时放入。太监阴谋败露因欺君罔上掉了脑袋。"害别人就是害自己"。

六祖《坛经》："心地但无不善，西方去此不遥。若怀不善之心，念佛往生难到。"带着害心念佛，"不要说往生，就连今生中大家看到你，也会唯恐避之不及"。

#### 己三、邪见

**无有因果之见**：认为行善无功德、造恶无过患、前世后世不存在、三宝无功德。"若见解没有摆正，学佛肯定只是一种形象。"

**常断见**：

- **常见**：认为神我常有，大自在天、遍入天是造世主，上帝、真主能创造一切。《释量论·成量品》对常有主物进行了驳斥——在胜义中，根本不可能有一法常有不变。
- **断见**：认为一切自然而生，前世后世不存在。来源是往昔天人与非天战争时，天人师长为激励作战而故意造了一部无有前世后世的邪论，后被外道蚁穴师传播至人间。《黑自在书》举五喻——太阳升起不是谁牵的、河水下流不是谁引的、豌豆圆形不是谁抟的、荆棘锋利不是谁削的、孔雀翎艳不是谁画的——以此否认因果。堪布指出此五喻不成立：（一）这些事物"都是由各自因缘形成的"，因缘不具足时包括荆棘刺的锋利也不会产生；（二）即便外境现象自性而生，"也不能说明内心的苦乐也是如此"——要想比喻成立必须两个东西有共同特点，而外境与内心的苦乐并不具足此共同点。

邪见可分为62种或360种等（全知无垢光尊者在《如意宝藏论》中有详述），但所有邪见归纳起来可包括在常见和断见中。

对于外道邪见的具体内容，《前行备忘录》有更详尽的阐述：**断见派**认为万法是由本性而生，身体好似雪、心宛若狼的脚印，当雪融化时脚印也不复存在——前后世、解脱通通不存在，人死时身体消失在四大种界中、心也随之消失。《黑自在书》中说："犹如日出水下流，豆圆荆棘长而利，孔雀翎艳诸苦乐，谁亦未造自性生。"**常见派**虽承许前后世存在，但认为并非由因果产生，而是大自在天创造的——如果供奉自在天则赐安乐，不悦则降不安。或如数论派承认"主物"，认为痛苦、快乐、不苦不乐三者平衡时取名为主物，内在明知的士夫恒常受用它，最终了解到这是主物所幻现之时，主物羞愧逃走、一切现象隐没，士夫离开现象即获解脱——解脱时心里有一个不可思议、拇指大小的常我。(《前行备忘录》)

**杀生与邪见最为严重**——"杀生之上无他罪，十不善中邪见重。"《成实论》："意业势力，胜身口业。"又云："于诸罪中，邪见最重。"圣天论师："宁毁犯尸罗，不损坏正见。"龙猛菩萨亦云邪见者行善果报仍然是苦。

邪见一旦染污相续，"即使奉行善法也不能踏上解脱之道"，而且"造罪也没有忏悔的对境"——因为连佛菩萨都否认了，向谁忏悔？

**一念正见的力量**——屠夫跳崖公案：北宋无德禅师座下一位沙弥走夜路不慎踩死青蛙，禅师令其跳崖谢罪。沙弥在悬崖边号啕大哭时，一位路过的屠夫听闻原委，说："你不过无意间踩死一只青蛙，那我天天杀猪，罪过岂不更无量无边？你不要跳，我来跳！"屠夫以极强烈的忏悔心毫不迟疑纵身跳下，一朵祥云突然托住救了他一命。"一念忏悔，就能清净无量罪业；而一念邪见，则会堕入万劫不复的深渊。"

**行善者临终生邪见堕地狱，作恶者临终起正见生天上**——《成实论》："行善者将命终时，生邪见心，则堕地狱。行不善者死时，起正见心，则生天上。"一念之差有时候特别关键。

**杀生罪业的公案补充**——《杂宝藏经》：一富翁贪吃羊肉，谎称家中富裕是树神庇佑须杀羊供神。儿子信以为真常杀羊祭树神。富翁死后以恶业投生为自家的羊。次年儿子选中此羊要杀时，一位阿罗汉到来加持儿子看到此羊正是父亲转世，儿子遂摧毁神庙从此断恶行善永不杀生。

### 戊四、十不善业之果（第65–67课）

#### 己一、异熟果

异熟果即不同的因最后成熟的果报。十不善业中任何一种：

| 烦恼主导 | 异熟果 | 教证 |
|---------|--------|------|
| 嗔心所导 | 堕入地狱 | 龙猛菩萨《中观宝鬘论》："由贪转饿鬼，以嗔引地狱，痴多成旁生，相反得人天。" |
| 贪心驱使 | 投生饿鬼 | |
| 痴心状态 | 转为旁生 | |

另一种分法：上品恶业（贪嗔痴极其粗重且长期积累）→地狱；中品→饿鬼；下品→旁生。

《本事经》："具足贪嗔痴，无智见明慧，定不能解脱，生老病死等。"反之，"远离贪嗔痴，具智见明慧，决定能解脱，生老病死等。"

《大宝积经》："众生造善恶，如影恒逐形。"业果不一定当下成熟——《亲友书》："有者所造诸罪业，纵未即时如刀砍，然死降临头上时，罪业之果必现前。"但"一旦因缘聚合时必将自食其果"。

造恶业的人若暂时享乐，并不是恶有善报，而是"以前行善的果报"——就像农夫虽已毁坏田地不再耕种，但上年积累的粮食还够享用一段时间。表面上丰衣足食，"可是未来的资粮无法积累，眼前这些享用完了也就没有了"。

**佛陀与鸽子的公案**（《大智度论》）：佛陀身影覆在鸽子身上时它安静不恐惧，舍利子身影覆上却仍战栗不已——因为舍利子三毒已除但细微习气未断。舍利子以神通观察鸽子前后八万大劫都是鸽身看不到边际，佛陀以无碍智说此鸽还要在恒河沙数大劫中为鸽身，灭罪后经五百世方遇佛法而渐获解脱。说明最细微因果连阿罗汉也无法彻见。

#### 己二、等流果

等流果：从异熟果牵引的恶趣中解脱获得人身后所感受的报应。"果似因故说名为等，从因生故复说为流。"（《入阿毗达磨论》）

不仅人会感受等流果，恶趣众生也会——有些旁生前世喜欢杀生，以等流果所感这一世也喜欢杀生；有些旁生前世爱吃草不害其他众生，即生中也会有类似习气。

因果虽不一定立竿见影，但绝不会耗尽——《福盖正行所集经》："设经无量劫，彼业不能坏，果报成熟时，众生决定受。"

**同行等流果**——今世习性延续前世：

- 《萨婆多毗尼毗婆沙》三例：牛主尊者有两个喉咙像牛反刍（前世常为牛）；一比丘走路蹦蹦跳跳（前世猴子）；一比丘虽得圣者果位却常照镜子（前世妓女）。
- 从幼年起即有明显差异——"有些喜欢残杀众生，有些喜欢偷鸡摸狗，有些热衷于行善积福"——皆前世旧习的惯性。
- 动物亦然：鹞鹰豺狼喜爱杀生，老鼠擅长偷盗。
- 佛经："过去生何处，当视今此身；未来生何处，当视今此身。"今生对善法有信心、对恶法厌恶，前世肯定是行善之人；反之，对恶法兴致勃勃、对善法兴趣索然，则前世"专业"是搞恶业的。
- 法王如意宝降生时刚一落地就口诵文殊心咒——这是前世与文殊菩萨有不共因缘的同行等流果。单比尼玛仁波切8岁即能精彩讲述《入行论》，华智仁波切赞叹："宁玛巴的教法正值辉煌期！"

**感受等流果**（十种不善业各有二果）：

| 不善业 | 感受等流果一 | 感受等流果二 | 关键教证 |
|--------|-------------|-------------|---------|
| 杀生 | 短命（婴儿夭折、突遭横祸） | 多病（有生之年几乎不离疾病） | 《大萨遮尼干子所说经》："杀生无善报，短命多诸疾" |
| 不与取 | 受用贫乏 | 财物被强夺或偷走，被迫与敌人共享 | 《决罪福经》："种福得福，种贫得贫" |
| 邪淫 | 丈夫/妻子相貌丑陋、懈怠懒惰 | 双方犹如仇人般争吵不休 | 《华严经》："妇不贞洁""得不随意眷属" |
| 妄语 | 常遭诽谤 | 常被欺骗 | 《正法念处经》："若人妄语说，彼人速轻贱" |
| 离间语 | 眷属仆人格格不入 | 主人遭到攻击，下属不听指挥 | 《正法念处经》："何人两舌说，善人所不赞" |
| 恶语 | 常闻不悦耳的话语 | 所说的话成为争论的话柄 | 《华严经》："常闻恶音""所可言说恒有诤讼" |
| 绮语 | 言语无威力无分量、口才拙劣 | 坦率直言别人也不信以为真 | 《华严经》："所有言语人不信受" |
| 贪心 | 凡事不能称心如意、事与愿违 | 多欲无有厌足（《华严经》） | 《正法念处经》："多欲大恶疮……昼夜不得乐" |
| 害心 | 常为一切人求其长短 | 常为众人之所恼害 | 《大集经》："忍为世间最，忍是安乐道" |
| 邪见 | 生邪见家 | 其心谄曲 | 《法华经》："薄德少福人……入邪见稠林" |

**不与取等流果的深入说明**：前世没有布施果报的人，"就算今生费尽九牛二虎之力也不会有什么收益效果"。那些强抢暗偷的人获取的赃物看似很多，"如果经常性获得那么多，恐怕整个大地也难以容纳"，但最终"往往因为山穷水尽而落得个饿死的下场"。有些人虽然拥有财产却没有权力享用，"由自己吝啬的原因而越是富裕就越发觉得贫困"，"简直就成了饿鬼守护宝藏一样"——《入中论释》讲过守财饿鬼，天天置身于财宝包围中，却只能看而不能用。《大宝积经》："若人多贪求，积财无厌足，如是狂乱人，名为最贫者。"

**布施的福报公案**（《贤愚经·波婆离品》）：一极穷人靠砍柴度日，见一独觉找不到饭就把自己的食物全部供养。后来遇到一只山兔变成一具死人跳到背上，到家后死人变成金人。国王来看只见死尸，穷人看来却是金子。国王封他为大臣。此穷人即阿那律尊者之前世——"遇到殊胜福田，转眼之间也会得到好报"。

**对治要点**：生病时不要只寻医疗方法，"应将精力放在发露忏悔上，下决心痛改前非"——有些病花多少钱治丝毫起色没有，好好念经忏悔"效果就特别明显"。遇到诽谤时应观想"依靠这场风波可以净除我的累累恶业"——持明无畏洲（智悲光尊者）："怨敌反对亦使修行增，无罪遭到诬陷鞭策善，此乃毁灭贪执之上师，当知无法回报彼恩德。"夫妻争吵应认识到"这是各自前世邪淫的等流果所致"，尽可能忍气吞声——帕单巴尊者："夫妻无常犹如集市客，切莫恶言争吵当热瓦。"

#### 己三、增上果

增上果是业力对外部环境产生的作用——不是罪业成熟于地水火风，而是共业形成了相应的生存环境。

| 不善业 | 增上果——转生之处 |
|--------|----------------|
| 杀生 | 深谷、悬崖、险地等威胁生命之处 |
| 不与取 | 庄稼遭霜冻冰雹、树木不结果、饥荒频发之处 |
| 邪淫 | 臭气熏天的粪坑、污秽不堪的淤泥等恶心之处 |
| 妄语 | 财富动摇不定、心常慌张、违缘频频之处 |
| 离间语 | 悬崖陡壁、深渊狭谷等难以行走之处 |
| 恶语 | 乱石堆积、荆棘丛生等令人心神不宁之处 |
| 绮语 | 辛勤务农却颗粒不收、季节反复无常之处 |
| 贪心 | 庄稼荒芜、地时恶劣、痛苦层出不穷之处 |
| 害心 | 多灾多难、战乱频仍之处 |
| 邪见 | 物资鲜少、无依无靠、孤苦伶仃之处 |

《大集经》："造十不善业，少福无供养，法味不纯厚，行法心亦薄。"即使是听法不感兴趣，也是造十不善业所致。

"一个人的心态对外境会起很大作用。行持十善心情快乐，环境就会变得优美柔和；内心十分痛苦，即使身处天堂，也会觉得外境很丑陋。"

堪布以亲身观察说明增上果的道理：到了好的环境时会自然想"这些人以前肯定造过十善业"；看到不好的环境时会觉得"这是当地人所造恶业成熟的果报"。"器世界也是以众生共业而形成的"——《俱舍论》说初劫时人类跟色界众生一样身体自然发光不需饮食，后来因贪执地味等内心渐趋恶劣，外境也随之每况愈下。

当今世间天灾人祸频频不断——地震、海啸、大旱、洪水——也是人们造十不善业的增上果所致。"人们若再不杜绝自己的恶行，恐怕今后日子更不好过了——这并不是谁在授记，也不是谁在预言，而是一种因果规律。"

#### 己四、士用果

"士"指士夫，"用"指功用——如同农夫耕耘得庄稼、商人经营获利润。但原论特别强调的含义是：**善恶业若不中断会像银行利息般不断增长**。

"不管造的是善业恶业，如果没有进行中断，增长率会非常高。就如同把钱存在银行里，存的时间越长，利息越高一样。"

恶业若未以忏悔对治，"随着时间流逝，成熟的果报将与日俱增，痛苦会世世代代辗转蔓延"。

公案——《百业经》：一居士嗔骂僧众"像跛子一样坐在法座上"，以此恶业五百世转生为跛子。"造了一刹那恶业的话，最终的果报不可思议。"

反之，善业的士用果也极可观——"所做的任何善业都会突飞猛进地增长，福德接连不断地涌现"。《功德藏》："无忧树种如芥子，每年果实成熟时，一枝亦增一由旬，善恶果增不可喻。"无忧树种子比芥子还小，但成长中每年树枝增长一由旬之巨，然而即便如此"也无法形容善恶果报的增长程度"。

因果的总结原则——《百业经》："纵经百千劫，所作业不亡，因缘会遇时，果报还自受。"不论经过多久，所造之业绝不消亡，因缘聚合时果报必定亲自感受。如金翅鸟翱翔高空时身影不现，"但是并非没有身影"，无论降落在哪里，"它那黑乎乎的身影就会出现在那里"——所造善恶业的果报虽非立竿见影，"最后不可能不降临到自己的头上"。

## 丁二、应行之善业（第67课）

了知十不善业过患后，应立下坚定誓愿受持严禁恶行戒——不杀、不盗、不邪淫、不妄语、不两舌、不恶语、不绮语、不贪、不害、不邪见。

善业的引发者——《中观宝鬘论》："无有贪嗔痴，及彼生业善。"想到万万不可造十不善业，从此以后纵然遇到生命危难也不造十不善业，这种心念就是**断十不善戒**。特殊的十善是指戒杀后救护生命等等——不仅仅是不造恶，而是进一步积极行善。(《前行备忘录》)

**十善的具体内容**：

| 类别 | 善业 | 实践 |
|------|------|------|
| 身三善 | 不杀生 | 断除杀生，爱护生命，放生吃素 |
| | 不偷盗 | 断不与取，慷慨布施 |
| | 不邪淫 | 断除邪淫，守持戒律，身心清净 |
| 语四善 | 不妄语 | 断除妄语，说真实语 |
| | 不两舌 | 断离间语，化解怨恨，调解不和 |
| | 不恶语 | 断除恶语，说悦耳语、柔和语、调顺语 |
| | 不绮语 | 禁止绮语，精进持咒念诵 |
| 意三善 | 无贪心 | 断除贪心，满怀舍心，知足少欲 |
| | 无害心 | 断绝害心，修饶益心、慈心、悲心、菩提心 |
| | 无邪见 | 弃离邪见，依止空性见、因果观、平等观等正见 |

**受戒方法**：

- 不需要在上师或亲教师面前——自己默默立誓"我从今以后永远不再杀生"便是善业
- 若能力有限，可附条件：在特定时段（一年/一月/一日）不杀生，或不杀特定众生
- 特殊日期功德更大：藏历一月（神变月）、四月（萨噶月）、六月（转法轮月）、九月（天降月），以及每月初八、初十、十五、二十五、二十九、三十
- 在上师、善知识、佛像、佛经、佛塔面前承诺发誓，"功德更为显著"

**受戒发誓的故事**：堪布在藏地开法会时，要求屠夫发愿——有些人天天杀生不愿戒断全部，上师就让他们发愿不杀雪山狮子、不杀恐龙，或不杀美国的众生——"如此发愿也是有功德的"。原来有个人特别爱喝酒，在法王面前发愿不杀生时不慎说成"我不喝酒"，"法王边说好边给他作加持，结果加持完了后，他才发现自己说错了"。

**短期受戒的利益**——屠夫半日戒杀公案（《根本说一切有部毗奈耶·皮革事》）：嘎达亚那尊者劝一屠夫白天不杀生未果，便让他承诺晚上不杀生持戒。死后堕入孤独地狱——白天在炽燃铁屋受苦，夜晚身居天宫由四天女围绕享乐。天子偈："昔时白日损他命，夜则持戒勤修行，以此因缘生此中，今受如是善恶业。"此公案在原论和讲解中都被特别强调，说明即使是极短时间的受戒也有明确的善报。

**十善业的四种果**：

- **异熟果**：转生在相应的三善趣中
- **同行等流果**：生生世世喜欢行善，善举蒸蒸日上
- **感受等流果**：断杀生→长寿少病；不偷盗→具足受用无盗敌；断邪淫→夫妻美满怨敌鲜少；断妄语→受众人称赞爱戴；断离间语→受眷属仆人恭敬；断恶语→恒闻悦耳语；断绮语→语言有威力；断贪心→如愿以偿；断害心→远离损恼；断邪见→相续生起善妙之见
- **增上果**：与十不善业增上果恰恰相反，具足圆满功德
- **士用果**：所做善业突飞猛进增长，福德接连不断涌现

《华严经》："人中随意得受生，乃至顶天禅定乐，独觉声闻佛乘道，皆因十善而成就。"一切功德的根本来源即是行持十善。

"即使你无法守持其他戒律，但不杀众生、具慈悲心的戒条，作为大乘佛教徒也不能舍弃。"——《阿难请问经》："不舍一法，即守持一切胜法。何为不舍一法？不舍一切众生是也。"

**善星比丘出家的意义**（《涅槃经》）：迦叶问佛既然早知善星比丘会生邪见堕地狱，为何还开许其出家？佛答：若不出家他会继承王位以强大力量毁坏佛法，在恶趣的时间更长。虽然最后生邪见，但"因为出家种下了善根，将来必会获得无上佛果"。说明哪怕只受一天出家戒也有功德。

**微善与微恶**：寂天菩萨："刹那造重罪，历劫住无间，何况无始罪，积重失善趣。"《贤愚经》："莫想诸罪微，无害而轻视，火星虽微小，能焚如山草。"反之，微善也产生大果——我乳轮王前世以穷人身份向佛撒一把豌豆，四粒入钵两粒触胸，以此统治四大部洲八万年、在帝释天平起平坐执掌国政。"莫想诸善微，无益而轻视，水滴若积聚，渐次满大器。"

**根本在于起心动念**——《功德藏》："树根为药芽亦药，根为毒芽何用说，唯随善恶意差别，不随善恶像大小。"如果内心清净纯正，纵然外观像是造恶业，实际已成善举（如大悲商主杀短矛黑人以保护五百菩萨，反而圆满七万劫资粮；星宿婆罗门以悲悯心与婆罗门女结成夫妻，圆满四万劫资粮）。反之，以贪嗔动机行表面善事，只会变成不善业。因此善恶的判断，"关键还是要看人的起心动念"。

但此类开许仅限于"没有丝毫自私自利、内心无比清净的菩萨"——"以私欲、在贪嗔痴的驱使下，在何时何地对何人也没有开许"。

堪布的结语告诫："善恶因果的见解得以坚固，那你做什么事情、行持什么善法都很容易。相反，倘若这个见解动摇不定，就算得了再高的灌顶、再无上的窍诀，对你恐怕也不一定有长期的利益。"
