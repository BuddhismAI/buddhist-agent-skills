# 寿命无常·喻义与修行（第32–39课）

## 丁五、思维各种喻义而修无常（第32–36课）

此节是寿命无常中最长的部分，以具体可观察的现象全方位展示无常的普遍性。原论总结为五个规律——**生际必死、积际必尽、合久必分、堆际必倒、高际必堕**——加上亲怨无常、苦乐无常、贤劣无常。

### 劫之增减（第32课）

往昔初劫时众生"凭借自身光芒照明，依靠神变行走空中，身体高达数由旬，以甘露为食，幸福美满可与天人相媲美"。因烦恼与不善业，福报日趋直下，直至如今寿命不足百岁。

《俱舍论》："先前有情如色界，而后渐次贪执味。"

未来人寿递减至十岁时，三大灾劫依次出现：

| 灾劫 | 持续 | 特征 |
|------|------|------|
| 饥馑劫 | 7年7月7日 | 无雨，五谷不生，人掘古骨煮汤 |
| 疾疫劫 | 7月7日 | 遍地病死，药石无效 |
| 战争劫 | 7日7夜 | 一切手中物皆成武器，人相残杀 |

之后弥勒菩萨化身传"断杀生法"，人寿从十岁渐增至八万岁时弥勒佛出世成佛转法轮。如此往返增减十八次后，人寿达无量岁时胜解佛出世，"此佛住世寿量是前面贤劫千佛寿量的总数，饶益众生的事业也等同于千佛事业的总和"。此后贤劫也终归消逝。

### 四季变迁

原论用藏地四季直观说明："夏季……五颜六色的鲜花争奇斗艳……真好似天境一般；秋季，瑟瑟的冷风将绿野变成黄色……到了冬季……就算经过许多马路寻觅也找不到夏季生长的一朵鲜花。"

### 生际必死（第33课）

> "无论是任何人，即便他高如天空、厉如霹雳、富如龙王、美如天仙、艳如彩虹，可是当死亡突然到来之时，他也没有刹那的自由……就像从酥油中抽出一根毛般独自而去。"

> "纵然是数以千计僧人的上师也不能带走一僧一徒，即便是数以万计部落的首领也不能带走一奴一仆，哪怕是拥有南赡部洲一切财产的主人也无法带走一针一线，就连自己最为珍爱、精心保护的身体也必然要舍弃。"

米拉日巴尊者："见而生畏之尸体，本为现在之身体。"——现在你精心保养的身体，死后就是让人恐惧的尸体。

**米拉日巴回乡公案**（第33课）：尊者回到故乡，发现祖屋荒废、田地长草、经书被雨水浸烂被鸟筑巢。在废墟中发现母亲的遗骨，悲痛至极，安住大手印定中七日，以定力超度母亲和父亲的神识往生净土。随后取出残损的经书和母亲遗骨发愿："见故园毁灭、亲人离散，我透彻明白了人生的虚幻本质……我放下贫穷与世人的轻蔑，将余生献给解脱修行。"

### 积际必尽（第34课）

"一切积聚终将散尽……即使是统治南赡部洲的国王，也有沦落为乞丐的时候。"

历史公案：
- **石崇**（西晋首富）：八王之乱中赵王司马伦看中其家财欲杀之。石崇悔言"是财多而招致了杀身之祸"。司马伦直言相告："知道是财富害了你，为何不早一点把财富散了？"石崇言："晚了！"
- **胡雪岩**（清代红顶商人）：富甲一方，后因政治斗争（左宗棠与李鸿章之争）成为牺牲品，倾家荡产
- **2009福布斯全球富豪排行榜**：拥有超过10亿美元资产的富豪人数，由上一年的1125人减少近三分之一，只剩793人（富豪**人数**锐减，非财富蒸发）

### 合久必分

> 帕单巴尊者："夫妻无常犹如集市客，切莫恶语争吵当热瓦。"

原论："现在朝夕相处的道友、家人等在不久的将来必定各奔前程，所以彼此之间千万不要怒气冲冲、恶语中伤、争吵不休……在极为短暂的岁月中理当互敬互爱，和平共处。"

堪布以自己的经历说明：三十年后重访童年冬牧场，邻居面目全非，只有当年刻在石头上的六字真言依旧清晰——人事全非，唯石上法印不变。

### 堆际必倒（第34课）

原论列举：

- **桑耶三层宝顶宫殿**：赤松德赞时期由幻化工人修建、莲师开光——后遭火灾毁于一旦
- **红山宫殿**（松赞干布时期布达拉宫前身）："如今已片瓦不存，连基石也无影无踪了"

堪布补充当代例证：
- **世贸双子塔**：110层现代工程，两架飞机在数秒内化为废墟
- **圆明园**：清朝"万园之园"，雨果赞其超越欧洲一切宝藏——1860年英法联军洗劫后焚毁，仅余残垣断壁
- **长城**：童年以为"永不可摧"，实际不过砖石砌成，已部分残损

**噶当四依处**——原论引为修行者面对无常的正确态度：

1. **心依于法**——心不依于世间事务
2. **法依于贫**——修法不依赖于财富（非强制贫穷，而是不贪著）
3. **贫依于死**——以死亡准备的心态面对贫困
4. **死依于干涸之壑**——选择僻静处迎接死亡，不依赖医院器械

### 高际必堕（第35课）

> "无常的地位又有何用呢？所以我们必须修成无衰无退、人天应供、圆满正等觉的果位。"

**我乳转轮王公案**：统治四大洲的金轮王，兼主三十三天与帝释平起平坐，战场上无往不胜。最终因贪念"我应取代帝释"，福德当即耗尽，坠落人间死于贪欲。临终悟言："虽然欲望如雨降，贪者终不能满足，智者知欲乐为苦。"

### 亲怨无常（第33、35课）

**嘎达亚那尊者化缘公案**——尊者以神通观察一户人家：

| 现世关系 | 前世真相 |
|----------|----------|
| 正在吃的鱼 | 施主的父亲转世 |
| 被石头打的母狗 | 施主的母亲转世 |
| 怀中抱的儿子 | 前世杀害施主的仇人转世来讨债 |
| 妻子啃的骨头 | 前世丈夫的遗骸 |

尊者偈："口食父肉打其母，怀抱杀己之怨仇，妻子啃食丈夫骨，轮回之法诚稀有。"

**志公禅师婚礼公案**——禅师观察一场婚宴：

> "孙儿娶祖母，女食母之骨，子打父皮鼓，猪羊炕上坐，六亲锅里煮。众人来贺喜，我说：苦！苦！苦！"

（新娘是亡祖母转世；院中小伙子所打驴皮鼓，其驴皮来自其父前世造罪堕入驴身；炕上坐的亲戚是前世被吃的猪羊；锅里炖的肉却是前世六亲眷属。）

### 苦乐无常

有些人上半生富裕快乐，下半生穷困潦倒；也有上半生痛苦，下半生幸福。

关键区分："为了求法，尽管暂时历尽苦行、饱经沧桑，但终会苦尽甘来，最后将获得无上安乐——就像往昔出世的诸佛以及米拉日巴尊者等前辈那样。"反之，以造罪积累的享乐虽暂时快乐，终将感受无边痛苦（尼洪国先降粮食雨、衣服雨、珍宝雨，最后降土雨将所有人活埋）。

### 贤劣无常（第36课）

世间层面："能言善辩、知识渊博之人也有一败涂地的时候。"出世间层面更为警醒："具证年老求学问，舍事年老积财物，法师年老成家长。"

有人上半生是清净戒师，下半生却子嗣成群；有人上半生唯造恶业，下半生唯一修法获得成就。"在尚未断除我执、没有生起无我之智慧获得圣者果位之前，贤劣的表象都是无常的。"

唐巴比丘公案：一人因密友成仇步入佛门，修至风心自在可在虚空飞行。一日施食子时鸽群聚来，他念头一转"若有此般大军可灭仇敌"——此恶念未转为道用，后来还俗成为军队首领。教训：即使高度修行，一个未对治的恶念就可能让一切功亏一篑。

## 丁六、思维死缘无定而修无常（第37课）

### 死缘极多生缘极少

圣天论师："死缘极众多，生缘极稀少，彼亦成死缘。"

死缘：火、水、毒、险地、野人、猛兽、疾病（人体有404种疾病、80000种魔障）、食物中毒、交通事故等无穷无尽。

生缘（衣食住行等）本身也可能成为死缘——吃饭可能噎死，穿衣可能窒息。

### 当代具体死缘

堪布列举大量当代实例：
- 2004年佛山逾百万人因生食鱼片感染肝吸虫病
- 1983年、1988年上海甲肝大流行（2万人和31万人）源于生食毛蚶
- 有毒保健品、假药、食品污染等——"自以为是生缘的衣食等也有成为死缘的"

### 死期全然无法确定

"有些在母胎中便已死亡；有些刚出生就已断气；有些刚学会爬行就已夭折；有些在壮年时离开人世……"

中国俗语："黄泉路上无老少。"

**万丈深渊比喻**：人悬挂于万丈深渊的草绳上，黑白二鼠（昼夜）不停啃噬绳子，下方是张着巨口的猛兽（恶趣），上方蜂巢滴蜜（五欲）。人在品尝蜜滴的同时不知不觉走向死亡。

**八十老翁公案**（《法句譬喻经》）：舍卫国一位年近八十、家财万贯、生性吝啬的老婆罗门，为自己建造华丽巨宅，亲自监工指挥。佛陀智慧观察得知他只剩一天寿命，亲自前来劝化。老婆罗门虽向佛介绍房屋用途却称"太忙"，不肯坐下听法，推说"过几天再聊"。佛陀见其冥顽不化只得离去。佛陀走后不久，老婆罗门亲自上屋顶安大梁，大梁突然坠下将他当场砸死。

## 丁七、思维猛厉希求而修无常（第38–39课）

### 临终时唯有正法有益

寂天论师："当鬼使来时，亲朋有何益？唯福能救护，然我未曾修。"

萨迦班智达："不知何时死，修何法无定，故于一切时，恒修利后法。"

### 三士道修行对死亡的不同价值（《前行备忘录》）

《前行备忘录》将对临终有益的正法按三士道明确区分：小士道行人了知三恶趣是痛苦的本性，以追求增上生人天果位的心态修行三种福德事——成就了小士道对死亡有益的法，却不是成就中士道、大士道对死亡有益的法。中士道行人认识到三有三界轮回是痛苦，在渴求自我解脱的出离心驱使下以想获得解脱的意乐实修三学——能成就中士道对死亡有益的法，却并非大士道对死亡有益的法。大士道以利他菩提心驱使，由经远道因相乘、近道金刚乘或捷径光明大圆满修成对死亡有益的法，才是真正的大士道清净法 (《前行备忘录》)。

尤其是，通过光明大圆满道修成对死亡有益的法，要以两个誓愿铁钩钩住："宁死也要修行了脱生死、达到无死本来坚地的法要"和"不舍此誓言" (《前行备忘录》)。

### 十种日常行为中的无常观修（第38课）

原论列出十个日常领域，每个都以"无常→正确应对"的结构：

| 领域 | 无常 | 应当 |
|------|------|------|
| 行路 | 道路无常 | 一举一动如理如法；"视轭木许无心乱"（《般若摄颂》） |
| 居住 | 处所无常 | 将之观为净土 |
| 饮食 | 受用无常 | 享用禅定美食（如米拉日巴重禅定食轻世间食） |
| 睡眠 | 卧躺无常 | 将迷乱修成光明境界 |
| 财富 | 珍宝无常 | 依靠圣者七财（信、戒、闻、施、惭、愧、慧） |
| 亲友 | 邻近无常 | 栖身静处激发出离 |
| 名位 | 地位无常 | 恒常身居低位 |
| 言语 | 话语无常 | 督促自己念咒诵经 |
| 信心 | 出离也无常 | 为誓言稳固而精进 |
| 验相 | 证悟无常 | 务必至达法界尽地——"到那时已经了脱生死" |

原论最终目标描述："获得无死的坚地，就像雄鹰翱翔在虚空中一样，死亡到来也无所畏惧。"

米拉日巴尊者："吾初畏死赴山中，数数观修死无定，已获无死本坚地，此时远离死畏惧。"

### 噶当派修行传统（第39课）

噶当派格西们"晚上睡觉之前常常思维：不知道明天早晨还用不用生火。因此他们往往不盖火并且将碗也是倒扣放置"——以每晚为最后一晚的心态入睡。

### 佛陀论无常修行之殊胜

佛陀金口玉言：

> "多修无常，已供诸佛；多修无常，得佛安慰；多修无常，得佛授记；多修无常，得佛加持。如众迹中，象迹为最，佛教之内，所有修行，观修无常，堪为之最。"

《毗奈耶经》："对我眷属中如妙瓶般的舍利子及目犍连等百名比丘供斋供物，不如刹那念有为法无常更胜。"——一刹那的无常观念，功德超过供养一百位阿罗汉。

### 博朵瓦格西的五重无常修行框架

一居士问："如果专修一法，修什么最重要？"格西答：**无常**。展开五重递进论证：

**第一重——入道之因、勤善之缘、证悟之伴**：
- 初：修无常 → 知道时间紧迫 → 进入佛法
- 中：修无常 → 精进修善，不敢懈怠
- 终：修无常 → 一切法等性（证悟诸法无常即证悟空性）

**第二重——断今生、舍轮回、趣涅槃**：
- 初：修无常 → 斩断对今生的贪执
- 中：修无常 → 不再贪恋任何轮回快乐
- 终：修无常 → 趣入涅槃圣道

**第三重——生信心、起精进、生智慧**：
- 初：修无常 → 对三宝生信（唯三宝能在死时帮助）
- 中：修无常 → 产生精进力
- 终：修无常 → 生起无我智慧

**第四重——求法、修法、证法**

**第五重——擐甲精进、加行精进、不退精进**

### 帕单巴尊者的教言

"如果相续中生起了无常观，最开始可作为步入正法的因，中间可作为精进的鞭子，最终也能获得光明法身。"

帕单巴对西藏修行者的尖锐批评（原论引）：

> "在西藏修行人当中，没有看见一个人有死亡的念头，也没有发现有一个人遗留在世。身着僧衣的人累积财产，难道是要供养阎罗王吗？……谁具广闻我慢高，修行好者积财宝，谁依静处多散乱，谁离故乡无羞愧，彼为形象修法者。彼等喜爱造恶业，虽已见到他人死，然却不知自将亡，此等一切诸过患，皆由未修无常致。"

### 无常定解的标准

**喀喇共穹格西**：在后藏觉摩喀喇山洞修行，洞口荆棘丛常挂衣服。每次想砍时就想"也许会死在洞中，不知还有没有出去的机会"，每次出洞又想"不知能否再度返回"——连续多年修行至获得成就，始终没有砍除荆棘丛。

**持明无畏洲（智悲光尊者）**：有一个秋季七月沐浴的水池缺少阶梯。弟子建议修一个，他回答："不知道明年还有没有在这里沐浴的机会，那么费事有什么用呢？"

**真正的定解标志**（原论引如来芽尊者）："我无论看见世间如何高贵、如何权威、如何富裕、如何俊美之人，也不会生起羡慕之心，而注重前辈高僧大德的事迹，这就是因为自己的相续中生起了少许无常观的缘故。我除了无常以外也再没有其他更殊胜的教言传授给别人。"

堪布解读：定解真正成熟时，就像呕吐症患者不愿取油食一样——对世间繁华自然丧失兴趣，不需刻意勉强。

### 佛陀因地求无常偈的公案（第38课）

佛陀前世为雪山苦行菩萨。帝释天化为可怖罗刹，念出半偈："诸行无常，是生灭法。"菩萨闻半偈欢喜渴求后半偈。罗刹以食肉为条件。菩萨答应以身供养。罗刹说出后半偈："生灭灭已，寂灭为乐。"菩萨将此偈刻于岩石树木上，然后攀上高树纵身跳下。帝释天恢复原形接住他。以此半偈法舍身之因缘，释迦佛迅速圆满十二劫资粮，较弥勒菩萨提前成就无上佛果。

### 塔波仁波切的三阶段修行

> "开始的时候，害怕生死所追，务必像鹿子逃出笼子一样义无反顾；中间的时候，务必像农夫辛勤耕耘田地那样做到死而无憾；到了最后，要像大功告成的人一样做到心安理得。"

另一组三阶段：
- 初：如箭中要害——知道没有空闲
- 中：如死了独子的母亲——专心致志
- 终：了达无所作为——"如敌赶走牧童牛"（牛被赶走后牧童愣在原地不知所措——比喻超越了一切造作）

### 结尾回向偈

> **无常现前反而执常有，老年到来反而以为幼，**
> **我与如我邪念诸有情，相续生起无常祈加持。**

## 修行实操要点总结

1. **修行方式**：随时随地观修死亡——行住坐卧都想"这可能是最后一次"。晚上睡时想"今夜可能死去"，早上醒来想"今天可能命绝"。
2. **修行时长**：没有固定期限。"在相续中没有生起不加改造的无常观之前就要修持"——可能需要100天、200天、3-5年甚至更长。
3. **三殊胜摄持**：加行发菩提心，正行调心观修，后行回向。
4. **修行次第**（博朵瓦格西窍诀）：无常 → 慈悲心 → 空性——此三步是净除恶缘的完整窍诀。
5. **晚课发愿**："是日已过，命亦随减，如少水鱼，斯有何乐？"
