# 《究竟一乘宝性论》结构总览

《宝性论》(弥勒菩萨造 / 梵 Ratnagotravibhāga / 藏 Uttaratantra "无上续") 是如来藏思想的根本论典——"弥勒五论"之一，也是第三转法轮的核心论著。索达吉堪布 2019 年基于**觉囊派朵洛瓦尊者**《大乘宝性论释》讲授 42 堂课，以宁玛视角重新疏通此觉囊派传承。

本集合的知识文档既呈现朵洛瓦注释的精细科判与觉囊派风格的如来藏阐释，也同时保留索达吉堪布融入麦彭仁波切《如来藏大纲要狮吼论》立场的解读——**双层并置，不得以任何一方遮盖另一方**。详见 [`topics/tathagatagarbha/index.md §Editorial Policy`](../../topics/tathagatagarbha/index.md)。

## 本论的核心论证——"四因三果"的因果架构

《宝性论》的全部义理以**七金刚处**为骨架，而七金刚处本身又被安立为一个**因果链条**——这是全论最容易被忽略的基础立场。不理解这个立场，整部论就变成七个孤立主题的拼盘。

```
    四金刚处 (因)                        三金刚处 (果)
    ────────────                        ────────────
    如来藏   ─┐
    (有垢真如)│                          佛宝 (断证圆满)
            │  ──(修行)──▶              法宝 (灭道二谛)
    菩提     │                          僧宝 (证悟之众)
    (无垢真如)│
            │
    功德     │
    (64种)   │
            │
    事业     ─┘
    (任运)

    "彼生妙三宝，见胜义者境"——颂2
```

**核心论式**（第6课对应颂词）：

> 有无垢真如，无垢佛德业，
> **彼生妙三宝**，见胜义者境。

读懂这句话，就读懂了本论的全部骨架：**三宝由四处所生**。这个次序与世间对佛法僧的直觉理解**相反**：
- 世间直觉：先有佛法僧三宝（可见的皈依对境），后有"佛性"等概念
- 宝性论立场：先有**如来藏**这个因（每个众生本具），通过修行净除客尘转为**菩提**（转依），圆满**功德**，自然显发**事业**，最后于世间现起三宝形象

换言之：**佛法僧不是从虚空中无因而来，也不是从凡夫心流通过信愿就能生起的——其终极因，是众生相续中本具的如来藏**。去掉如来藏，佛果无从成立（"无因生的过失"，课1）。

这就是为什么《宝性论》一定要从"三宝本体"讲起，再回头讲"三宝从何而来"——这是整部论最高级的教判策略：**以三宝为果引出四因，使如来藏的安立成为必须而非多余**。

### 四个"不可思议"的理由

每一金刚处为什么都"不可思议"——即非凡夫、二乘所能通达？课6 以一颂概括了四个理由，这是本论最精妙的并列判教：

| 金刚处 | 不可思议的理由 | 表面相违处 | 实际道理 |
|---|---|---|---|
| 如来藏 (有垢真如) | **清净具染故** | 自性本来清净 × 客尘遮蔽 | 虚空喻——虚空本净，云雾不改其净 |
| 菩提 (无垢真如) | **无染清净故** | 本体无染 × 需修道方显 | 金矿喻——金性本具，除锈方显 |
| 功德 | **无分之法故** | 众生本具64种功德 × 凡位不现 | 功德以不可分方式存在于法界 |
| 事业 | **任运无念故** | 无分别 × 能无误利生 | 如意宝、日光——无思而有作 |

四个理由并非独立，而是同一实相在四个施设位上的四种显示：**本净、离染、本具、任运**。这是"如来藏—菩提—功德—事业"本为一如来藏的不同面向的最清晰证明。

## 本论的讲授立场——觉囊与宁玛的双层并置

索达吉堪布在第1课明确声明：本论讲解以**朵洛瓦《大乘宝性论释》**为根本参考（他本人翻译），科判和内容主体依此。但他本人站在宁玛派立场，又受法王如意宝（曾讲此论前九课）的影响，故在关键节点上以麦彭仁波切的立场柔和地调整朵洛瓦原义。

**读者须理解的三层关系**：

1. **弥勒颂词本文** — 共 405 颂，是根本；各家注释都必须紧扣颂词
2. **朵洛瓦注释层** — 觉囊派典型的"义他空"解读：如来藏是实有法性，非空性所能遮破；第三转为了义，第二转为不了义
3. **索达吉堪布宁玛重释层** — 以麦彭立场疏通：如来藏即离戏大双运/自性光明大圆满，二转三转所诠无二；朵洛瓦的"不空"应理解为"不空于功德"，非"不空于自性"

这三层在本集合的知识文档中**不得混为一谈**。引用颂词、对应科判、阐释义理时，应清楚标注是在哪一层发言。详细规则见 `topics/tathagatagarbha/index.md §Editorial Policy`。

### 法王如意宝圆寂的密意

索达吉堪布反复提到：法王如意宝曾讲《宝性论》前九课（由麦彭注释本），讲到"**佛外无有余涅槃**"（课14 "以日轮比喻决定"）时，示现圆寂。索达吉堪布视此为深密意——法王以身示教，印证**究竟唯一乘**（声缘无余涅槃终须回大乘成佛）。这是本集合在"菩提品"章节处理时必须提及的教内重要事件。

## 科判骨架（大甲级）

全论科判（朵洛瓦），分四：

```
甲一契义之论名
甲二成办大义总顶礼
甲三具殊胜义之正论                           ← 主体
  乙一安立论体
    丙一总纲义
    丙二对应经教
    丙三次第固定之理
  乙二广说分支
    丙一广说所得三宝之分支                    ← 三宝本体
      丁一三宝各自本体 (佛/法/僧)
      丁二安立三皈依之理
      丁三以释词宣说
    丙二广说得法四处之分支                    ← 四金刚处 (主体大半)
      丁一共同宣说 (四处为圣者境/不可思议/理由/因缘安立)
      丁二各自本体
        戊一所证法界之义                     ← 如来藏品
        戊二证悟本体菩提之义                 ← 菩提品
        戊三证悟分支功德之义                 ← 功德品
        戊四能证方便事业之义                 ← 事业品
    丙三彼等功德                              ← 利益品
甲四解说圆满造论事宜                          ← 后记
```

**朵洛瓦注释文** 将科判重组为**六品**（与上述一一对应）：

| 品 | 科判对应 | 学时 | 核心内容 |
|---|---|---|---|
| 总说品 | 甲一–乙一 + 丙一三宝 + 丙二丁一共同宣说 | L1–L5 | 七金刚处框架、三宝本体、四处概论 |
| 第一如来藏品 | 丙二丁二戊一 | L6–L20 | 如来藏十义、九喻、不空如来藏、五过 |
| 第二菩提品 | 丙二丁二戊二 | L21–L28上 | 转依、二清净、三身、恒常不可思议 |
| 第三功德品 | 丙二丁二戊三 | L28下–L33 | 十力、四无畏、十八不共法、32相 |
| 第四事业品 | 丙二丁二戊四 | L34–L40 | 任运/不间断、佛事业九喻 |
| 第五利益品 | 丙三 + 甲四 | L41–L42 | 闻信此法超胜余善之理、造论事宜 |

## 教证来源

《宝性论》最主要依据的经典（课6）：
- **《如来藏经》** — locus classicus: "一切众生，虽在诸趣烦恼身中，有如来藏常无染污，德相备足，如我无异。"
- **《不增不减经》** — "如来藏者，是法界藏，是法身藏，是出世间上上藏，是自性清净藏"
- **《大般涅槃经》** — "我者即是如来藏义。一切众生悉有佛性。"
- **《胜鬘经》（吉祥鬘）** — "心自性光明难以证悟，心真实随烦恼难以证悟。"
- **《大方等如来藏经》** — 九喻的经典出处
- **《宝女经》《陀罗尼自在王请问经》《法界赞》** 等

《宝性论》的主要异译与参考：
- 汉译本：北魏勒那摩提（宝意）《究竟一乘宝性论》——未标作者
- 梵文本：约 1935 年印度学者罗睺罗在中亚发现，1950 年牛津琼思敦校订出版
- 藏文译本：多版本，朵洛瓦《宝性论释》是其中的觉囊派代表注释
- 其他重要注释：麦彭仁波切注释（法王如意宝传）、贾曹杰注释（格鲁派，有汉译）、廓诺·迅鲁伯《宝性论广释》

作者问题：藏传公认弥勒菩萨造；汉传、日本、韩国学界（源自法藏大师判定）多认为是坚慧论师造。1997 年英梵合订本确认弥勒菩萨说。
