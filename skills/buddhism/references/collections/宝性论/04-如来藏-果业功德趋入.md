# 《宝性论》如来藏品之二——果业功德与三分位趋入

本文件是 [`03-如来藏-总纲十义.md`](./03-如来藏-总纲十义.md) 的直接续接。上文在庚一 "本体因清净" 处立下的 **"四障 / 四对治 / 四佛子因"** 论式，在此文 "果端" 得到对称展开——**四对治 → 遣四颠倒 → 得四果**。这是十义的 **核心推论完成节点**，也是全论 "如来藏不是外道的我" 这一 correctness anchor 的教内论证位置。

本文处理讲记第 8 课完整（庚二 + 庚三）与第 9 课的前部分（庚四 / 庚五 / 庚六）。第 9 课末尾已进入庚七 "恒时无变" 的第一辛科（辛一 "宣说无变之分位" 颂 "过失客尘性 ..."），但此颂属于 "三分位恒常" 主题，归下一文档 [`05-如来藏-恒常无变与功德无别.md`](#) 处理。

## 一、庚二果及业成就之义——四对治何以能得四果

科判：戊一·己三·庚二（果及业成就之义）下分 **辛一共同宣说所得能得** + **辛二解说各自本体差别**。辛二下再分 **壬一所得果之义**（三癸）与 **壬二能得作业之义**（二癸）。

### 1.1 辛一共同宣说所得能得——四果与四业的并列总颂

> **净我乐常德，波罗蜜多果，**
> **厌苦欲得寂，发愿之业者。**

朵洛瓦注：

> "如是修行四种对治而净化四障，由此获得具足如来藏法身净波罗蜜多、我波罗蜜多、乐波罗蜜多、常波罗蜜多——四种胜义功德波罗蜜多，即是果之义。厌离轮回痛苦、欲求获得涅槃寂灭及发愿的业是有佛性的作用。"

此颂以 "所得果（四波罗蜜多）" + "能得业（厌苦 / 欲寂 / 发愿）" 两端总摄 —— **果与业不是两个无关的主题，而是 "所得" 与 "能得" 的因果双边**。此处 "业" 并非佛教因果业力通称义，而是 "如来藏在众生位发起的作用"—— 即 "若无如来藏众生就不会有求解脱的心" 这一立场的颂词基础。

### 1.2 壬一所得果——癸一断除所断颠倒之理

此癸一仅一颂，是庚二的总门：

> **略摄此等果，是以于法身，**
> **遣除四颠倒，对治而安立。**

朵洛瓦注：

> "简略地说，信解、智慧、禅定与悲心——法界能净的这四种因的果是以依次遣除对法身不净、无我、苦、无常四种颠倒的缘故，获得以对治而安立的净、我、乐、常四种功德波罗蜜多。"

索达吉堪布（课8）：

> "这四个波罗蜜多若要现前，最关键的还是'信'，信仰或者信心是很重要的……无著菩萨的注释里面引用《胜鬘经》所云：'或有众生信佛语故，起常想、乐想、我想、净想，非颠倒见，是名正见。'"

#### 四因 → 四对治 → 四颠倒遣 → 四果完整对应

这是全论最关键的一张五列表——**连结庚一（因端）与庚二（果端）的主骨架**：

| # | 所障 | 行人 | 四对治（四佛子因） | 所遣颠倒 | 所得四果 |
|---|---|---|---|---|---|
| 1 | 嗔大乘法 | 大欲者（断善阐提） | **信解** | 执法身为 **不净** | **净波罗蜜多** |
| 2 | 我见（五蕴执）| 外道 | **智慧**（二无我） | 执法身为 **无我** | **我波罗蜜多**（胜我） |
| 3 | 畏惧轮回苦 | 声闻 | **等持**（三摩地） | 执法身为 **苦** | **乐波罗蜜多** |
| 4 | 无视众生利 | 缘觉 | **大悲** | 执法身为 **无常** | **常波罗蜜多** |

此表与 [`03-如来藏-总纲十义.md §六壬二能净因`](./03-如来藏-总纲十义.md) 的 "四障 / 四对治" 表接合 —— **03 仅做 "障 → 因 → 佛子" 的半边论证，此处完整补上 "因 → 遣颠倒 → 得四果" 的另一半**。

#### 四果针对四种行人——对症下药

索达吉堪布（课8）补充：

> "这四种果实际上是针对大欲者、外道、声闻、独觉而安立的……对大欲者讲大乘佛法，使他们对大乘佛法产生清净的心，然后断除不净的相，最后获得净波罗蜜多的果位；对于外道，因为外道有各种我执，所以给他们宣讲无我的智慧，最后也得到我波罗蜜多的道理；对于声闻而言，他们比较怕大乘空性法门，通过禅定获得乐波罗蜜多的境界；对于缘觉而言，给他们讲一些慈悲心的法，最后他们也得到常波罗蜜多的境界。"

即 —— 四果并非四个平列果位，而是四个对治道次第在 **四类行人机宜** 上的契入。

### 1.3 壬一所得果——癸二以此而得四果之理

此癸二有二颂，展开四果各自的安立方式。

#### 净波罗蜜多与我波罗蜜多

> **彼性清净故，断习气故净，**
> **灭我无我戏，是故为胜我。**

朵洛瓦注：

> "究竟果如来法身，它的总法相自性本来普皆清净的缘故，差别法相无余断除客尘连同习气的缘故是净波罗蜜多。完全寂灭执著外道等遍计之我的戏论及执著声闻等遍计之无我的戏论，因此是胜我波罗蜜多。"

**净波罗蜜多** 的双重成立：
1. **总法相自性清净**——即 "自性清净"（众生位本具）
2. **差别法相无余断除客尘习气**——即 "离垢清净"（佛位现前）

索达吉堪布特为辨明：

> "众生有自性清净，但没有离垢清净。而佛、菩萨，以及声闻，断除习气的程度也有一定的差别。"

以鸽子躲佛影与舍利子影的公案（出《大方便佛报恩经》）—— 声闻断烦恼障而习气未尽，故影下仍战栗；佛三阿僧祇劫断二障习气，故鸽影下泰然。此喻是 "净波罗蜜多必须自性清净 + 离垢清净双具" 的具体教证。

**胜我波罗蜜多** 的 **三重灭除**：
1. 凡夫的 **俱生我执**
2. 外道的 **遍计常我执**（数论神我、胜论离蕴我等）
3. 声缘的 **"无我" 戏论**（将"无我" 本身执为实法）

三重皆破之后所余，非 "有我" 亦非 "无我"，勉强名之为 **胜我 / 大我 / 我波罗蜜多**。

##### Correctness anchor 必读辨：如来藏之"我" ≠ 外道神我

这里必须引入 topic 层最重要的 correctness anchor（详见 [`../../topics/tathagatagarbha/index.md §Correctness Anchors "如来藏不是外道的「我」"`](../../topics/tathagatagarbha/index.md#correctness-anchors)）：

此 "我" 必须以 **经过二无我抉择** 为前提——先以中观破尽五蕴之我与法我，所余之 **离戏自性光明** 方名 "胜我"。绕过自空中观而直说 "我" 即外道神我。

索达吉堪布（课8）明确辨别：

> "外道经典里也有'我''无我'这些讲法，但是与佛教所讲的如来藏，意义和境界完全不同，在《首楞严经》中，佛陀对大慧菩萨讲了很多这方面的道理。佛教所讲的如来藏，远离四边八戏，并非恒常不变、不可毁坏的一种坚固的东西，我波罗蜜多也并非我们凡夫那种非常坚实的我执。"

> "在世间修行时，首先破掉坚实的我执，达到'无我'，然后再把这个'无我'彻底清净，清净后虽然也称之为'无我'，但这个'无我'实际上是'胜我'，不是凡夫阶段的'有我'，也不是声闻缘觉的'无我'。"

以 "看山是山 → 看山不是山 → 看山还是山" 作三阶段比喻 ——字面相同，境界层次迥别。

##### 编辑注：此段觉囊立场与宁玛立场的张力点

朵洛瓦注将 "胜我波罗蜜多" 作为 **如来藏实有的积极德相** 宣说，不怕使用正面语汇。索达吉堪布讲解时 **两个动作同时做**：

1. **保留朵洛瓦本义**——明立"胜我"为大乘真实四果之一
2. **补入宁玛防滑门**——强调此 "我" 须经二无我抉择、远离四边八戏、不是 "坚固不坏之物"

**此即索达吉堪布典型的双层并置**—— 不把 "胜我" 消解为单纯空性术语（那会削弱朵洛瓦注），也不把 "胜我" 实有化（那会滑入义他空）。读者须在此双层结构中理解，不可偏取一端。参见 [`../../topics/madhyamaka/debates.md §1 第九诤辩`](../../topics/madhyamaka/debates.md) 与 [`../../topics/madhyamaka/classifications.md §1 自空/他空`](../../topics/madhyamaka/classifications.md) —— 宁玛对 "不空如来藏" 的定位是 "不空于功德而非不空于自性"。

#### 乐波罗蜜多与常波罗蜜多

> **除意自性蕴，彼因故为乐，**
> **证悟轮与涅，等性故为常。**

朵洛瓦注：

> "由无余灭尽痛苦而断除意自性的蕴及所有集谛，结果也就消除了它的因无明习气地的一切烦恼及无漏业，为此是乐波罗蜜多。如是证悟三有轮回与寂灭涅槃究竟意义自性等性的缘故不间断二利，是常波罗蜜多，那也是因为胜义法界自性清净。轮回无常的过患丝毫无所减，因此未堕于断边；本来任运自成的缘故，涅槃的功德丝毫无所增，因此未堕于常边。由此成立远离二边的涅槃。"

**乐波罗蜜多** 的特殊含义：

索达吉堪布强调 —— "乐波罗蜜多并非指特别开心和快乐，而是 **断除了声闻缘觉相续中意识形态的痛苦和痛苦之因**。"

此 "苦" 指三乘圣者相续中仍有的四种相（无著菩萨注）：
- **缘相**——缘无明习气地而产生的烦恼
- **因相**——此烦恼的因 (无漏业)
- **生相**——有烦恼 + 业而产生的意生身相
- **坏相**——有坏灭与不可思议生死（变异生死）

**重点**：声闻阿罗汉虽断三界烦恼，仍有 "无明习气地"—— 仍有烦恼习气的残余，仍有 "无漏业" 所感的意生身，仍有 "变异生死"（非凡夫的分段生死）。课 8 以众多阿罗汉习气公案作证：

| 尊者 | 残余习气 |
|---|---|
| 难陀 | 贪心习气（多世为王） |
| 舍利弗 | 嗔恨心习气（多世为蝎子） |
| 毕陵伽婆蹉 | 傲慢习气（多世为大婆罗门） |
| 周利盘陀 | 愚痴习气 |
| 牛身比丘 | 奔跑习气（多世为牛） |
| 迦叶 | 舞蹈习气 |
| 阿难 | 歌唱习气 |

故 "乐波罗蜜多" 不是 "凡夫更强的乐"，而是 "**声缘的无余涅槃之上再断意生身之苦与无漏业之因**"。

**常波罗蜜多** 的成立理由：

"证悟轮涅等性"—— 非 "只取涅槃舍轮回" 之常，而是 **不堕断 / 常二边的双运常**：
- 轮回无常过患丝毫无所减 → 未堕 **断边**
- 涅槃功德丝毫无所增 → 未堕 **常边**
- "智不住三有，悲不滞涅槃" → 轮涅无二之 "常"

此 "常" 与凡夫概念的 "恒久持续" 义根本不同—— 它是 "**不间断二利任运成就**" 的境界性常。

### 1.4 壬一所得果——癸三以得而解脱二边之理

> **以慧无余断爱我，爱众悲尊非得寂，**
> **此依智悲觉方便，圣者不住轮涅边。**

朵洛瓦注：

> "以证悟无我的智慧无余斩断对蕴贪爱为我的缘故，不是像大贪欲者一样堕于有边；以大悲慈爱一切众生而成办他们的利益，因此，慈悲尊者菩萨也不是像声闻缘觉一样堕于仅得寂灭痛苦之边。如此依靠证悟无我的智慧以及缘众生的大悲这两种获得无上菩提的殊胜方便而修行的佛子圣者们，不住三有轮回或寂灭涅槃任意一边，现前不住之涅槃。"

此颂是整个壬一的 **收束点**—— 四果安立究竟成就 **"不住涅槃"** 的大乘道：

| 二边 | 能断 | 所对 |
|---|---|---|
| **有边**（堕轮回）| 无我智慧 | 大贪欲者 |
| **寂边**（仅自寂）| 缘众生大悲 | 声闻缘觉 |

索达吉堪布（课8）：

> "因此，佛教真正的意义在于智悲双运——以智慧而不住轮回，以悲心而不入涅槃。即便是住于轮回，也不会心生畏惧和痛苦。"

《大般若经》为教证：

> "诸菩萨摩诃萨。若观生死而起厌怖。欣乐涅槃则堕非道。不能利乐一切有情，通达如来甚深境界。"

——菩萨若喜涅槃厌生死， **已堕非道**。此是大乘与二乘的根本分野，也是本品 "如来藏之业" 讨论的直接理论入口。

### 1.5 壬二能得作业之义——若无法界应成不证舍取

科判：庚二·辛二·壬二下分 **癸一若无法界应成不证舍取** + **癸二有种姓故证悟功过之作用**。

此二癸是 **本论 "如来藏必有" 的正面论证**——**若众生无如来藏，则世间现见的 "厌苦求寂" 心无从产生**。即以 "厌苦求寂之心事实存在" 反证 "如来藏必存在"。

#### 癸一若无法界应成不证舍取

> **设若无佛性，于苦不厌离，**
> **不欲乐涅槃，亦无希求愿。**

朵洛瓦注：

> "假设不存在佛陀界性或法界自性清净如来藏，则应成见到众生流转痛苦的过患也不厌离，见到涅槃安乐的功德也无有向往获得它的欲乐，也不会希求它的方便及心现行其义的发愿。但事实并非如此。"

此是一个 **应成式反证**——
- **设**：若无如来藏
- **应成**：则众生见苦不厌、见涅槃不求、不发愿求解脱
- **事实**：众生见苦而厌、见涅槃而求、能发愿
- **结论**：故如来藏必存在

《胜鬘经》教证：

> "若无如来藏者，不得厌苦、乐求涅槃。"

即 —— "厌苦求寂的心" 本身就是 **如来藏在众生位已经发挥作用的明证**。此与 [`./01-总说-七金刚处.md §六四因三果`](./01-总说-七金刚处.md) 的 "佛法僧不是从虚空中无因而来，其终极因是众生相续中本具的如来藏" 的论式完全同构—— **去掉如来藏，成佛 / 求解脱都无因可得**。

#### 癸二有种姓故证悟功过之作用

> **见有痛苦过，涅槃安乐德，**
> **此有种姓故，无种姓无故。**

朵洛瓦注：

> "见到三有轮回痛苦的过失与寂灭涅槃安乐的功德后厌离三有痛苦的过失，欢喜涅槃安乐的功德，这些是来自于有佛陀种姓。"

此颂从正面再说 —— **"有种姓" 的功用表现为 "见过 / 见德 / 发心求解脱"**。

##### "无种姓者 = 一阐提" 的不了义说

索达吉堪布（课8）特别处理了 **"一阐提无如来藏"** 这个经常被误引的说法：

> "'一阐提没有如来藏''这个人一点佛性都没有'是一种**不了义**的说法……他的佛性暂时被深重业障所遮蔽而看不到，所以暂且称为'无种姓者'……这种人是不是真的没有如来藏呢？不是，如来藏依然具足，但可能还要再过几百个劫才会成熟善根。"

此说与 [`./03-如来藏-总纲十义.md §六癸一四障`](./03-如来藏-总纲十义.md) 中 "三类阐提" 的区分（无种性 / 断善 / 大悲阐提）完全贯通。 "无种姓" 是约客尘深障暂不显现立说，非约如来藏本体立说—— 本体上 **一切众生皆具如来藏**，此是《宝性论》所判决定立场。

##### 四轮教证——众生皆可苏醒佛性

《亲友书》四轮教证：

> "身处圣地、依止善知识、发大宏愿、造甚深殊胜之福德，此四大轮汝具足。"

四轮具足 → 众生即使 "看似无佛性" 亦终将佛性苏醒 ——此立场是如来藏品与《大般涅槃经》 "一切众生悉有佛性" 的主线一致。

## 二、庚三具众功德之义——因功德与果功德

科判：戊一·己三·庚三（具众功德之义）下分 **辛一共同宣说因果功德** + **辛二解说各自本体差别**。辛二下分 **壬一具因功德** + **壬二具果功德**。

### 2.1 辛一共同宣说因果功德

> **如大海无量，德宝无尽处，**
> **具无别功德，本性故如灯。**

朵洛瓦注：

> "如同极其广阔的大海汇集了无量的珍宝与水一样，法界也在极其广阔的信解法器，成为无量智慧、等持功德珍宝及大悲水无尽之处的缘故，是具足因功德之义。神通、无漏智慧、无漏断德是具足无别功德之本性的缘故，如同灯火的光、暖、色无别一样存在，即是具足法界果功德之义。"

两喻对应：

| 功德层 | 喻 | 三德 |
|---|---|---|
| **因功德**（众生位本具，可开显） | **大海** | 无量（水器）/ 德宝（珍宝）/ 无尽（水） |
| **果功德**（佛位任运显发） | **灯** | 光 / 暖 / 色（三德无别而俱） |

### 2.2 辛二壬一具因功德之义

> **摄法身佛智，大悲界性故，**
> **以器珍宝水，说与大海同。**

朵洛瓦注：

> "由于无余摄集或具有清净法身之因与获得佛智之因、趋入大悲之因或界性，故以具足广大容器、珍宝与水三种同法，说明法界与大海等同。"

#### 因功德三德与四佛子因的对应

索达吉堪布（课8）特将此处的 "大海三德" 与庚一的 "四佛子因" 相对：

| 大海三喻 | 功能 | 对应四因 |
|---|---|---|
| **水器**（无量）| 承载一切功德 | **信解**（信心是所有功德的器） |
| **珍宝**（德宝）| 具足势力与威力 | **智慧 + 等持**（珍宝喻威力能解决问题） |
| **水**（无尽）| 滋润万物 | **大悲**（润体之力） |

《入行论》教证：

> "佛说一切善，根本为信解。"

故 "如来藏因功德" 即 **庚一能净四因之摄持**—— 信心为根本器，智慧与等持为威力宝，大悲为润物水。此处是 **十义第一 ~ 第五义首尾呼应** 的关键节点。

### 2.3 辛二壬二具果功德之义

> **无垢处神通，智无垢真如，**
> **无别故与灯，光暖色同法。**

朵洛瓦注：

> "自性无垢之处如来藏中五通、无漏智、无垢真如无余转依之断德无别存在的缘故，与灯火的光芒、温暖、颜色无别存在同法。"

#### 果功德三德与灯喻

| 灯喻 | 果功德 | 断除 / 遣除 |
|---|---|---|
| **光** | **五通 / 神通**（他心通、天眼通等） | 照见之前看不到的东西 |
| **暖** | **无漏智**（如所有智 / 尽所有智） | 烧毁业及烦恼（灯暖烧尽灯芯灯油） |
| **色** | **漏尽通 / 无垢真如转依之断德** | 清净 / 无垢 / 光明（断烦恼障、所知障及清净二障） |

灯的 "光 / 暖 / 色" **不能相分离**—— 故以此喻 **果功德三德无别而俱的特性**。

#### 《不增不减经》教证与《法界赞》对应

《不增不减经》：

> "舍利弗，如世间灯，所有明色及触不离不脱。"

—— 灯光、灯色、灯触（即热）不离不脱，是果功德无别的经典教证。

#### 此处 "果功德" 的宁玛调整

此节 **朵洛瓦立场最容易被读成 "如来藏于众生位已实有具足五通 / 十力等果功德"**。课8 末尾索达吉堪布作了柔和调整：

> "对我们而言，果法的功德比较遥远，但是因法的功德：信心、悲心、智慧，每一个人都具足。所以，大家要将如来藏中最主要的功德——因与果的四种法，铭记在心。"

即 —— 宁玛讲记层通常 **在实际修行落处强调 "因法功德每人具足 + 果法功德须修显发"**，而不在颂义上做硬性修改。这与 [`../../topics/tathagatagarbha/index.md §Correctness Anchors "不空如来藏 = 离戏大双运"`](../../topics/tathagatagarbha/index.md#correctness-anchors) 一致——"不空" 者非 "众生位已实有具足佛果一切功德"，而是 "光明分本具、客尘遮蔽而终可开显"。

## 三、庚四以行人分类趋入之义——凡/圣/佛三所依

科判：戊一·己三·庚四（以行人分类趋入之义）下分 **辛一以三所依趋入宣说之理** + **辛二彼等亦次第清净之理**。

### 3.1 辛一三所依趋入

> **分别入凡圣，圆佛之真如，**
> **见真于众生，说此如来藏。**

朵洛瓦注：

> "以分别趋入异生之真如、圣者之真如、圆满佛陀之真如入手，现量照见真如，而对成为遍知所化的有缘众生，极其明了宣说此如来藏法界自性清净。"

**三所依真如**：

| 所依 | 真如 | 垢染状态 |
|---|---|---|
| 异生凡夫 | 凡夫真如 | 有垢真如 |
| 圣者菩萨 | 圣者真如 | 半垢染半清净 |
| 圆满如来 | 佛陀真如 | 无垢真如 |

索达吉堪布（课9）：

> "佛从兜率天降神于母胎宣讲的《广普经》云：'法性如大海，不记有是非，凡夫贤圣人，平等无高下，唯在心垢灭，取证如反掌。'"

即 —— 三所依不同，但 **如来藏本体无高下之分**，只依垢染程度立三名。

### 3.2 辛二彼等亦次第清净之理

> **异生乃颠倒，见真谛者反，**
> **如来则如实，无倒无戏论。**

朵洛瓦注：

> "凡夫异生想、心、见是颠倒的；现量见到四谛法性真如的圣者，则与他们相反，想、心、见无有颠倒，是真实见到；如来如实照见一切法摧毁颠倒及二障连同习气，是无有戏论性。"

**三行人的认知状态**：

| 行人 | 想 | 心 | 见 |
|---|---|---|---|
| 凡夫 | 颠倒（常乐我净颠倒）| 颠倒（贪嗔痴慢嫉） | 颠倒（常见 / 断见 / 六十二恶见） |
| 菩萨 | 无颠倒 | 无颠倒 | 无颠倒（现量见四谛） |
| 如来 | 如实 | 如实 | 无戏论（摧二障及习气） |

**关键教证**（《大般涅槃经》，索达吉堪布课9 引）：

> "一切众生虽不能见。十住菩萨见少分故。如来全见。十住菩萨所见佛性如夜见色。如来所见如昼见色。"

即 —— 凡夫不见，菩萨见少分如夜见色，如来全见如昼见色。此节引出下一庚 "三分位唯名" 的主题。

## 四、庚五分位唯名之义——名异义一的立场

科判：戊一·己三·庚五（分位唯名之义）下分 **辛一三分位立三名称之理** + **辛二以三名摄六义之理**。

### 4.1 辛一三分位立三名称之理

> **不净不净净，极净依次第，**
> **是名为众生，菩萨与如来。**

朵洛瓦注：

> "如来藏客尘不清净、不清净与清净兼有、极其清净三种阶段，依次以众生、菩萨与如来三种不同名称来说明。"

#### 三分位 / 三名称 / 染污 / 功德显发结构表

| 分位 | 清净状态 | 名称 | 染污 | 功德显发 |
|---|---|---|---|---|
| 不净分位 | 客尘极遮 | **众生** | 烦恼 + 所知 + 习气 + 俱生我执等全部 | 仅有因功德潜伏（信心等尚未必生起）|
| 不清净清净分位 | 半染半净 | **菩萨**（一地至十地）| 已见四谛真如 + 渐断修惑 + 所知障尚存 | 渐次显发五通、禅定、智慧 |
| 极清净分位 | 无垢 | **如来** | 烦恼障 + 所知障 + 一切习气皆断 | 果功德任运显发（五通 + 无漏智 + 断德）|

索达吉堪布特别强调此颂的重要性：

> "这个教证还是很著名的，大家应该背下来……《大乘法界无差别论》中也有类似说法：'不净众生界，染中净菩萨，最极清净者，是说为如来。'"

#### 《不增不减经》三位对应

《佛说不增不减经》从 "烦恼缠缚" 角度立三位：

| 经文描述 | 对应三分位 |
|---|---|
| "众生被无边烦恼缠缚，往来生死、动荡漂流" | 众生位（不净）|
| "菩萨正在从烦恼中出来，舍诸欲求、修菩提行" | 菩萨位（不清净清净）|
| "如来则已完全远离烦恼垢染，住清净法、得自在力" | 如来位（极清净）|

### 4.2 辛二以三名摄六义之理

> **本性等之义，此六摄界性，**
> **于诸三分位，是以三名说。**

朵洛瓦注：

> "本性之义，以及'等'字包括的因之义、果之义、作用之义、具足之义、趋入之义，这六义摄集了上面广说的界性清净。这是指，对于不清净、不清净与清净兼有、极其清净这三种阶段，是以众生、菩萨、如来三种名称安立说明不同，而其本体实际中丝毫也无差别。"

#### 十义前六义 → 三名摄六义的收束

从十义（庚一 ~ 庚八）到三名的收束关系：

```
十义前六 (本性/因/果/业/具/趋入)
    ─────── 摄 ─────────▶ 三分位三名 (众生/菩萨/如来)
十义后四 (分位/普行/恒常/无别)        只是本体唯一之不同面向
```

**关键立场**：**本体无差别 + 所依有分别**——此即 "名异义一"。

《现观庄严论》教证（索达吉堪布课9 引）：

> "法界无差别，种姓不应异。由能依法异，故说彼差别。"

#### 水三态喻——名异义一

索达吉堪布以 "水三态" 作妙喻：

> "同样是水，它在冰箱里是固态的冰；拿出放到茶壶里烧化，就成了液态的水；如果继续烧下去，就化成气态的水蒸气了。所以如同水的自性不曾改变，却有固态、液态、气态三个阶段那样，本自清净的如来藏，也分别有众生、菩萨和佛陀的三个阶段。"

#### 编辑注：此处与外道我的终极辨析

**"众生 / 菩萨 / 如来本体同一"** 这一立场表面看似与外道 "万物皆是梵" / "众生本是神我" 类似，但根本区别在于：

| 外道"我" | 如来藏 |
|---|---|
| 未经二无我抉择 | 以二无我抉择为前提 |
| 恒常实有之体 | 离戏自性光明大双运 |
| 独立主宰 | 空性与光明无二 |
| 修行即回归神我 | 修行即除客尘显本具 |

课9 中索达吉堪布引《入楞伽经》作根本辨析：

> "无垢如来藏，远离边无边，本识在蕴中，如金银在矿。"

"**远离边无边**" 是关键—— 如来藏本身已离 "有边 / 无边" 两端，故虽言 "本具"，非外道恒常实有义。

## 五、庚六如虚空般普行之义——遍众生而不染

科判：戊一·己三·庚六（如虚空般普行之义）下分 **辛一以无分别周遍之理** + **辛二总相无可分之理**。

### 5.1 辛一以无分别周遍之理

> **如无分别性，虚空普随行，**
> **心性无垢界，如是普行性。**

朵洛瓦注：

> "犹如无有分别、无有障碍之本性的虚空无别普皆随行周遍一切色法一样，心的自性光明本来无垢的法界也如是无别普行周遍一切阶段的行者。"

#### 虚空喻——遍色法而不与色法混杂

索达吉堪布（课9）详说：

> "虚空无有任何分别、在四大中也无任何阻碍，还可以产生功用，并且它普随一切色法。瓶子里外皆有虚空，人的身体里外也有虚空……同理，我们的心性光明本无垢染，就像虚空能遍于所有色法一般，如来藏亦平等周遍于所有众生，包括所有人类以及一切动物，哪怕微小如蚂蚁、蚯蚓、麻雀等，也不例外。"

**虚空喻的四重含义**：

1. **无分别性**—— 虚空本身不分别瓶内瓶外
2. **无障碍性**—— 一切色法无法阻挡虚空
3. **普随行**—— 一切色法内外皆有虚空
4. **不染**—— 虚空不受色法所染

同样，如来藏：
1. **无分别周遍**—— 不分凡夫菩萨佛
2. **无障碍**—— 烦恼业苦皆无法阻碍如来藏本体
3. **普行**—— 一切众生（包括蚂蚁蚯蚓麻雀）皆具
4. **不染**—— 受烦恼遮蔽而本体不染

#### 《大乘经庄严论》教证

> "虚空恒遍行，许佛亦如是，虚空遍行色，佛遍行众生。"

此颂是 "虚空喻 / 如来藏遍众生" 的 locus classicus，课9 明引。

#### 《涅槃经》佛陀尼连禅河外道公案

课9 详讲五百婆罗门依止佛陀公案：

- 婆罗门疑佛持断见（"无我" → 无主体持戒无主体解脱）
- 佛答：**"我亦不说一切众生悉无有我，我常宣说一切众生悉有佛性。佛性者，岂非我耶？"**
- 婆罗门闻后即发无上菩提心

此公案出《大般涅槃经》，是 "如来藏 = 真我" 的经典出处之一。

##### 编辑注：此处的朵洛瓦 / 宁玛张力点

此段若逐字读取—— "佛陀承许有我 / 如来藏即是我"—— 极易滑向 "**如来藏是实有常住法性** / **与梵我无本质区别**" 的义他空立场。 **朵洛瓦觉囊派立场就倾向于此读解**（如来藏是实法）。

索达吉堪布（课9）的 **宁玛补遗** 直接接在此公案后：

> "后来佛陀在经中也说：'虽然我宣说众生有佛性，是"有"法，但这只是**暂时引导众生趋入佛道的善巧方便，并非最了义之说**。'"

即 —— **"如来藏 = 我" 是善巧方便说，非最了义**。最了义仍是离戏大双运。这是本文中最明显的 **觉囊 / 宁玛立场公开张力点**，索达吉堪布用《入楞伽经》 "远离边无边" 紧接着调整。

读者请参 [`../../topics/tathagatagarbha/index.md §Correctness Anchors "如来藏不是外道的「我」"`](../../topics/tathagatagarbha/index.md#correctness-anchors) 与 [`../../topics/madhyamaka/debates.md §1 第九诤辩`](../../topics/madhyamaka/debates.md)。

### 5.2 辛二总相无可分之理

> **彼总法相遍，过功及究竟，**
> **犹如虚空遍，下中上色相。**

朵洛瓦注：

> "法界自性清净是一切法的总法相或法性的缘故，同等随入周遍于有过失的异生、具功德的菩萨及功德究竟的如来一切阶段，如同虚空随行周遍于下等土器、中等铜器、上等金器一切色法形相一样。"

#### 三等色法喻

| 色法等级 | 容器材质 | 对应三分位 |
|---|---|---|
| 下等 | 土器 | 凡夫异生（具过失）|
| 中等 | 铜器 | 菩萨（具一定功德）|
| 上等 | 金器 | 如来（功德究竟）|

**要点**：同一虚空遍入三等色法器， **不以容器质地有所增减**。同样，同一如来藏遍入三分位， **不以分位有所差别**。

索达吉堪布（课9）：

> "这些只不过是我们眼中看到的现象，而实际上，乃至蚂蚁以上的每个众生都具足如来藏……虚空随着不同的器皿而变化，如来藏随着习气而有不同的表象，但众生都是未来佛，如来藏存在于每一种身份的众生之中。"

#### 常不轻菩萨立场的理论依据

**常不轻菩萨** 见一切众生皆顶礼 "汝当成佛，我不敢轻" —— 其理论依据正是此庚六 "如虚空普行"。索达吉堪布：

> "常不轻菩萨之所以不轻视任何众生，源于他对如来藏的特殊认识。"

此处是《宝性论》义理与《妙法莲华经》菩萨行的直接连结点之一。

## 七、小结——本文所立关键论式

1. **四对治 → 遣四颠倒 → 得四果**（庚二核心论式，承接庚一）：
   - 信解 → 遣不净执 → 净波罗蜜多
   - 智慧 → 遣我执 → 我波罗蜜多
   - 等持 → 遣苦执 → 乐波罗蜜多
   - 大悲 → 遣无常执 → 常波罗蜜多

2. **四果圆满 → 解脱二边**（癸三收束）：
   - 无我智慧断有边（不堕轮回 / 大贪欲者）
   - 缘众生大悲断寂边（不堕涅槃 / 声缘）
   - → 不住涅槃 = 大乘真实菩提

3. **反证如来藏必有**（壬二）：
   - 若无如来藏 → 应成众生不厌苦不求寂
   - 事实：众生厌苦求寂
   - 故如来藏必具

4. **因 / 果功德无别**（庚三）：
   - 因功德：大海三德（水器 / 珍宝 / 水）= 信心 / 智慧+等持 / 大悲
   - 果功德：灯三德（光 / 暖 / 色）= 五通 / 无漏智 / 断德
   - 因与果功德本体无别，唯客尘遮否而异

5. **三分位 / 三名 / 本体唯一**（庚四 + 庚五）：
   - 三所依：凡 / 圣 / 佛 → 三真如：有垢 / 半垢 / 无垢
   - 三分位：不净 / 不净净 / 极净 → 三名：众生 / 菩萨 / 如来
   - **本体无差别，所依有分别 = 名异义一**

6. **普行如虚空**（庚六）：
   - 虚空遍色法而不与之混杂
   - 如来藏遍众生而不受众生染污
   - 此即 "一切众生皆具佛性" 的理论根据
