# 《宝性论》如来藏品之一——十义总纲与本体因清净

本文件进入《宝性论》主体——**第一如来藏品**。总说品已安立三宝为果、如来藏等四处为因（详见 [`01-总说-七金刚处.md`](./01-总说-七金刚处.md) §六四因三果）。现开始广说四处中的第一处——**有垢真如 / 如来藏**（戊一所证法界之义）。本文覆盖如来藏品最前部分的三个总纲段与第一个广说段——**己一法界周遍之理 / 己二以总纲略说能遍 / 己三广说总纲义总说 + 庚一本体及因清净之义**（讲记第 6–7 课）。

这是进入如来藏品的第一把钥匙：弥勒菩萨并不在此处以譬喻铺陈 "如来藏像什么"，而是先以 **十种范畴** 把如来藏的所有意义结构性地展开——本体、因、果、业、具、入、分位、普行、恒常、无别——再从第一义 "本体及因清净" 切入广说。读懂此十义的并列关系与第一义的 "四障 / 四对治 / 四佛子因" 论式，后续所有如来藏材料（九喻、五过、不空、分位、九垢等）都只是这个骨架上的延展。

## 一、法界周遍之理——如来藏必然遍于一切众生

科判：戊一·己一（法界周遍之理）。

### 三理推证：佛身 / 真如 / 种姓

如来藏品开篇第一颂即立下本品全部基调 —— **为什么一切众生必然本具如来藏？**

> **佛身遍现故，真如无别故，**
> **具种故众生，恒具如来藏。**

全知麦彭仁波切在《如来藏大纲狮吼论》中以此颂为框架展开。索达吉堪布（课6）指出：此颂由三个推理合成一个结论——

| 因 | 推理 | 对应"四理" |
|---|---|---|
| 佛身遍现 | **以果推因**——佛法身之功德非无因生，其因即如来藏 | **观待理** |
| 真如无别 | **体方面**——众生心之本体与真如无二，轮涅本性一味 | **法尔理** |
| 具种 | **以因推果**——众生相续本具佛种，以此种而必得佛果 | **作用理** |

> "以上观待理、法尔理，还有作用理，我们在《解义慧剑》里面也曾讲过。通过这三种推理，我们可知'三界众生皆具如来藏'。"（课6）

三理合成——**果 / 体 / 因** 三侧同证，故结论坚不可破：**众生恒具如来藏** 。

朵洛瓦注释（第一如来藏品）以一句概括：

> "因为圆满佛陀的法身能起现周遍于一切法，因为一切轮涅的法性真如无有差别，因为一切有情均具有法界的自性清净、障碍可净的如来种姓，所以，所有众生从无始以来就恒常相续不断具有胜义如来藏。"

### 汉传对照——《宗镜录》同构推理

索达吉堪布特别指出汉传亦有相同推理。延寿大师《宗镜录》引弥勒菩萨立场：

> "一切众生，有如来藏。何等为三？一者如来法身，遍于一切众生心识；二者真如之本体，与一切众生无二无别；三者一切众生，皆悉等有真如佛性。"

此与本颂三理一一对应 —— 可见《宝性论》三理推证法并非藏传孤传，而是弥勒教法跨汉藏的共同结构。

### 第二颂：佛智入众生

紧接着的第二颂是前颂的开演：

> **佛智入众生，性无垢无二，**
> **佛种择果故，众具如来藏。**

- "佛智入众生"——呼应"佛身遍现"（法身佛智普入一切众生）
- "性无垢无二"——呼应"真如无别"（无垢佛性与众生无二无别）
- "佛种择果故"——呼应"具种故"（佛种逐渐抉择成熟显现佛果）

此颂在梵文本、魏译本中有时与前颂合为一颂，在藏文本中独立。朵洛瓦注释与无著菩萨注释皆未广详，以 "大意与前颂相同" 处理。

### 教证与信心门径

索达吉堪布以多处教证支持众生本具佛性的道理：

- 《如来藏经》："一切众生，虽在诸趣烦恼身中，有如来藏常无染污，德相备足，如我无异。"
- 《喜金刚续》："众生本来佛，然以客尘遮，除彼乃真佛。"
- 傅大士偈："夜夜抱佛眠，朝朝还共起。"——我们时时与心之本体佛性同在。
- 《首楞严经》："顶礼如来藏，无漏不思议，愿加被未来，于此门无惑。"

《宝性论》自此处起具有 **密续意味** —— 索达吉堪布在此处再次强调："《宝性论》实际上与密法基本上是相同的，因此有些大德要求，听闻前需要灌顶"（课6）。此即本论藏文名 "无上续" 意味所在。

### 编辑注：三理与宁玛的 "离戏法界" 对接

此段极其容易被读成 "如来藏是一个实有的常恒之物存在于众生相续中"。三理推证 ——果、体、因皆似指 "有某物"——如不慎即滑向义他空立场。

须依 correctness anchor（见 [`../../topics/tathagatagarbha/index.md §Correctness Anchors`](../../topics/tathagatagarbha/index.md#correctness-anchors)）明辨：三理所证之 "众生恒具如来藏"，其 "所具" 不是一实有法性，而是 **离戏自性光明大双运**。麦彭仁波切《如来藏大纲要狮吼论》明示：

> 凡是承许无变之法界为成佛种性，首先需要认识所谓的法界是于何施设之基——真胜义二谛大双运极为不住的中观义。

读者须在进入下文 "十义" 时始终带此问题意识 —— 十义所描述的如来藏诸面向，其施设基是 "大双运"，不是 "成实之法"。

## 二、以总纲略说能遍——十义框架的提出

科判：戊一·己二（以总纲略说能遍）。

### 十义总颂

> **体因果业具，入分位普行，**
> **恒常德无别，胜义界密意。**

弥勒菩萨以一颂将后续对如来藏的所有展开结构化为 **十个范畴**——这是本品的总骨架，与总说品的 "七金刚处" 为相同级别的关键并列教判。

### 十义一览

| # | 范畴 | 一句话概括（朵洛瓦定义） | 广说位置 |
|---|---|---|---|
| 1 | **本体** | 法身真如种姓自性清净的本性 | 庚一壬一（本文§四） |
| 2 | **因** | 信解等能遣除客尘的方便——即四对治 | 庚一壬二（本文§五–§六） |
| 3 | **果** | 法界远离客尘于佛地获得四种功德波罗蜜多（净我乐常） | 庚二壬一（下一文档） |
| 4 | **业** | 因有种姓，见轮回过患而欲出离，见涅槃功德而欲获得 | 庚二壬二（下一文档） |
| 5 | **具** | 具因功德（信解等）与果功德（神通等）两方面 | 庚三 |
| 6 | **趋入** | 由凡夫/圣者/佛三种行人所依，能依法性也三分 | 庚四 |
| 7 | **分位** | 三阶段（不净/染净/极净）本无差别，唯以三名说 | 庚五 |
| 8 | **普行** | 一切分位中，法界如虚空般普周遍入 | 庚六 |
| 9 | **恒常** | 普行所依不因过功而改变 | 庚七 |
| 10 | **无别** | 究竟断证功德在恒常不变真如中无二无别而具 | 庚八 |

朵洛瓦在列完十义后，特别加一句压顶之言：

> "应当了知，这十种意义是以善加分析了法性胜义谛法界自性清净之密意而抉择的。"

即 —— **十义皆是 "法性胜义谛法界自性清净" 一法之不同侧面的展开**，非十个独立命题。此与总说品四因三果 "本净 / 离染 / 本具 / 任运" 四侧面一如的立场贯通——详见 [`01-总说-七金刚处.md §九四个"不可思议"`](./01-总说-七金刚处.md#九-四个不可思议同一如来藏的四侧面)。

### 十义与"本体因清净 → 果业成就"大分架构

己三广说将十义重新组合为八个庚级科判 —— 本体 + 因合为 "庚一本体及因清净之义"，果 + 业合为 "庚二果及业成就之义"，其余六义各为一庚。这种合并不是随意的：

- **本体 + 因**：所净 + 能净——**从因端说清净**
- **果 + 业**：所得 + 能得——**从果端说成就**

故前两庚是 "从因说清净" 与 "从果说成就" 的对称结构，是十义的核心论式。本文处理第一对（庚一），下一文处理第二对（庚二 + 余六庚）。

### 索达吉堪布的劝学

索达吉堪布（课7）对此十义特意提醒：

> "佛教徒经常讲如来藏，到底什么是如来藏？刚才讲的这十个意义恐怕一些人也很难答得上来。如果有人问你什么是如来藏？你把这十个意义给他宣讲后，他可能已经迷糊了……应该依弥勒菩萨的教义宣说，使他们从根本上了解如来藏的含义。"

即 : 十义并非学术分类，而是当被问 "如来藏是什么" 时能以结构性作答的工具表。

## 三、己三总说——从"本体因"到"果业"的次第

科判：戊一·己三（广说总纲义）总说。

己三下分八个庚，但在正式广说之前没有独立总纲颂 —— 弥勒菩萨直接进入庚一的总颂。然而朵洛瓦注释在此处已作出科判层面的结构声明（见 §二表格最右列），为后续展开奠定秩序。

注意一个重要细节：**前两庚连带的颂词分布在 L6 末尾（"如宝空水净" 颂）到 L8 初（"略摄此等果" 颂）之间** ——讲记跨越三课，反映本段密度。本文仅处理庚一（L7 完整完成），庚二及其后将在下一文档处理。

## 四、庚一本体及因清净之义——"所净 / 能净" 总颂

科判：戊一·己三·庚一（本体及因清净之义）。先由辛一 "共同宣说所净能净" 总立框架，再由辛二 "解说各自本体" 分壬一所净本体 / 壬二能净因两端详论。

### 辛一总颂——所净三喻 + 能净四因

> **如宝空水净，自性恒无染，**
> **信法殊胜慧，等持悲所生。**

**前两句**——**所净**——如来藏本体以三个比喻说其自性恒无染：

| 喻 | 所表 |
|---|---|
| 如意宝珠自性清净 | 如来藏具满愿的威力 |
| 虚空自性清净 | 如来藏自性不变异 |
| 水自性清净 | 如来藏悲慈润化 |

**后两句**——**能净**——四种清净因：

| 四因 | 义 |
|---|---|
| **信法**（信解大乘） | 对大乘佛法的胜解信 |
| **殊胜慧**（智慧波罗蜜多） | 证悟人无我、法无我的空性智慧 |
| **等持**（三摩地） | 虚空藏三昧、楞严三昧等如海禅定 |
| **悲**（大悲心） | 缘众生而生的大悲 |

朵洛瓦总释："如同珠宝自性清净、虚空自性清净、水自性清净一样，如来藏法界也是自性恒常无有垢染、清净就是本性义。信解大乘法、证悟无我的殊胜智慧与具足安乐的等持、缘众生的大悲是能清净客尘的因，从中所产生的菩提是**离系果**。"

关键点 : 能净四因所生的果是 **离系果** —— 即 "除客尘后显发的本有功德" ，非 "修行新造之法"。此立场与总说品 "菩提 = 无垢真如 = 如来藏离客尘后之圆满显发态" 一以贯之（见 [`01-总说-七金刚处.md §六`](./01-总说-七金刚处.md#六四因三果本论最核心的因果论式)）。

如来藏本体因之清净即七金刚处中第四金刚处的展开（见 [`01-总说-七金刚处.md §九四因不可思议`](./01-总说-七金刚处.md#九-四个不可思议同一如来藏的四侧面)）——此处的 "清净具染" 即如来藏第一个 "不可思议" 的理由。

## 五、壬一所净本体——"威力 / 不变 / 润体" 三德

科判：庚一·辛二·壬一（所净本体之义）。

### 颂词与三德对应

> **威力不变异，润体自性故，**
> **与珠宝虚空，水功德同法。**

朵洛瓦将前颂三喻与如来藏三德严格对应 ——

| 三德 | 同法喻 | 义 |
|---|---|---|
| **威力**（成实一切所愿） | 如意宝珠 | 如来藏具应一切众生所愿的威力 |
| **不变异**（自性永不转） | 虚空 | 如来藏于一切阶段自性不变 |
| **润体**（慈悲滋润） | 水 | 如来藏周遍一切众生以慈悲滋润 |

### 索达吉堪布的展开（课7）

**威力**：众生学出世间法能学有所成，学世间法也能挖掘潜力 —— 这是如来藏之威力。"如果没有种子，再努力也徒劳无益，就如石头什么都学不会。"

**不变**："真如在任何阶段自性未曾变异过，如虚空一般，仅是有云和无云的不同，虚空本体没有任何差别。"

**润体**："佛性（如来藏）能周遍一切众生，正因为如此，他的慈悲可以滋润我们的本性。"《大乘法界无差别论》："譬如劫尽火，不能烧虚空，如是老病死，不能烧法界。"

### 《妙法莲华经》与"小佛 / 大佛"

索达吉堪布特别引《妙法莲华经》中常不轻菩萨："每一众生都具足佛性，现在是'小佛'，以后会变成'大佛'"——故常不轻菩萨对每个众生（包括侮辱他的人）都恭敬顶礼。这是 "如来藏润体" 在修行中的具体落处。

### 莲花生大士的证心之言

课 7 特引莲花生大士之言：

> "如果通达自心，即使牧人也能解脱；如果没有通达自心，即使班智达也是迷惑懵懂的。"

索达吉堪布补充：现在讲的如来藏，对自心有所认识的人 "即能了知如来藏现前的时候是菩提，未现前的时候是众生"——无需向外寻求，当下自心中具足。这是索达吉堪布典型的 **宁玛补遗动作**：将朵洛瓦紧守的 "如来藏是一实有法性具足种种威德" 立场，以 "自心光明" 之宁玛视角连接到大圆满的观心法门 —— 详见 [`../../topics/tathagatagarbha/index.md §Editorial Policy`](../../topics/tathagatagarbha/index.md#editorial-policy)。

## 六、壬二能净因——四障 / 四对治 / 四佛子因

科判：庚一·辛二·壬二（能净因之义）下分癸一断除所断四障之理 + 癸二以对治成佛子之理。

**此节是本文的核心论式**——如未读懂此节，全论后续 "四果净我乐常" "五过" "九喻对治九垢" 皆失去轴心。

### 癸一断除所断四障

> **嗔法及我见，畏惧轮回苦，**
> **无视众生利，大欲者外道，**
> **声缘之四障，净因胜解等，**
> **法即有四种。**

#### 四障的具足者

朵洛瓦与索达吉堪布共同给出的对应表：

| 障 | 障之内容 | 所障之行人 |
|---|---|---|
| 嗔法 | 嗔恨大乘法 | **大欲者**（一阐提／断善阐提） |
| 我见 | 执人等为我 | **外道**（数论／胜论等） |
| 畏惧轮回苦 | 因畏苦而唯求自寂 | **声闻** |
| 无视众生利 | 只求自证十二缘起而不度化众生 | **缘觉** |

#### 四障详析（课7）

**嗔法**——索达吉堪布对此义特重：

> "现前如来藏的最大障碍首当其冲的就是嗔恨心……嗔恨心是摧毁一切善根……因嗔恨心而暂时断了善根的阐提。"

《入行论》："一嗔能摧毁，千劫所积聚，施供善逝等，一切诸福善。"

"大欲者"即"一阐提"。索达吉堪布引经论分三类：
1. **无种性阐提**——《楞严经》《大乘经庄严论》所说"断种性不能成佛"者（此为不了义说，宁玛立场终须皆可成佛）
2. **断善阐提**——因嗔恨心暂断善根者
3. **大悲阐提**——因大悲殷重而长住轮回不求涅槃者（即菩萨）

此处所指为第二种——"断善阐提"。

**我见**——包含 "我 + 我所" 见，通常数论外道、胜论外道具足。引《大智度论》："若信诸法空，是则顺于理；若不信法空，一切皆违失。"——通达人无我、法无我的智慧可断此执。

**畏惧轮回苦**——声闻为免轮回痛苦唯求自寂，可依虚空藏等三摩地之境界来对治。"若能恒时安住于三摩地的境界中，对轮回中所谓的痛苦也能安之若素、泰然处之。"

**无视众生利**——缘觉既不求佛果，也不愿度化众生，唯独希求自己获得十二缘起的证悟。"除了极少数的缘觉会对个别有缘众生示现种种神变，大多数缘觉几乎不会度化身边的众生。"对治：对众生生起真实无伪的大悲心。

#### 四障 → 四对治对应表

这是本论最清晰的并列判教：

| # | 四障 | 具足者 | 对治（四因／四法） | 所断之颠倒 |
|---|---|---|---|---|
| 1 | 嗔法 | 大欲者 | **信解**（胜解信） | 不净执（需下文四果颠倒表确认） |
| 2 | 我见 | 外道 | **智慧**（般若波罗蜜多） | 我执 |
| 3 | 畏惧轮回苦 | 声闻 | **等持**（三摩地） | 苦执 |
| 4 | 无视众生利 | 缘觉 | **大悲** | 无常执 |

（四颠倒对应将在庚二果门下一文详说；本文仅点出，因 L7 未详谈此一一对应 — — L7 讲的是四障 → 四对治， L8 讲的是四对治 → 四果遣四颠倒。合起来才完整。）

#### 课7 的补充：三聚对应

> "在其他注释中也有'邪定聚''不定聚''正定聚'，相当于我们经常讲的'无种姓''不定种姓''决定种姓'的说法。我们可以用邪定聚者对应嗔法的大贪欲者，不定聚者对应外道，后面讲的正定聚者对应声闻缘觉。"

此为旁注对应，非必坚持，朵洛瓦原注以上表四类为主。

### 癸二以对治成佛子——转轮王四喻

> **信解胜乘种，慧生佛法母，**
> **禅乐胎处悲，乳母佛生子。**

此颂以 "**转轮王降生** 须具备的四缘"喻 "**佛子菩萨诞生** 须具备的四因"：

| 四因 | 转轮王喻 | 义 |
|---|---|---|
| **信解** | 父方种子 | 对大乘法之胜解为菩萨功德之根本因 |
| **智慧** | 生母 | 证悟二无我之智慧出生佛陀一切法 |
| **禅乐（等持）** | 胎处 | 虚空藏等持等禅乐为不失增上之所依 |
| **大悲** | 乳母 | 大悲心为抚育成长之缘 |

**只要四法具足，即生为能仁圆满佛陀意中所生之佛子菩萨。**

朵洛瓦注释："具足这四法，就是由能仁圆满佛陀之意中生、作为子嗣的佛子菩萨。"

#### 为何以转轮王作喻

索达吉堪布（课7）：

> "一般来说，转轮王若要降生于世间，首先需要父亲方面的种子，其次需要母亲，之后入胎的时候需要胎藏，最后降生后需要乳母的喂养，然后才能真正地变成转轮王。"

释迦牟尼佛传记记载，佛降生后有四种乳母：给予乳汁的、沐浴清洁的、给予饮食的、负责生活起居的，每种两位共八位乳母。此处以"乳母" 喻大悲，正取其"抚育成长使不夭折" 之义。

#### 四因教证

- 信解 = 种子：
  - 《宗镜录》："佛法如海，唯信能入。信则道原功德母，一切善法由之生。"
  - 《华严经》："信为道元功德母，长养一切诸善法。"
  - 《入大乘论》："所作重恶业，能深自悔责，敬信大乘法，拔除诸罪根。"
- 智慧 = 生母：需闻思修，非平白而生。
- 禅乐 = 胎处：如无胎藏，孩子无从降生。
- 大悲 = 乳母：《入中论》从初 / 中 / 后三处讲大悲心的重要性（"悲性于佛广大果，初犹种子长如水，常时受用若成熟，故我先赞大悲心"）。

#### 信解的特殊重要性

在四因中， **信解列首** 。索达吉堪布（课 7）于此反复强调传承加持力的作用：

> "如果传承上师的意传加持入于心间，修行很容易获得成就，就好比你本身具有种子，要经灌溉浇水后，种子才有成长和成熟的机会。而我们虽本具如来藏的种子，但若没有上师的加持给予'灌溉'，也许这辈子很难苏醒自己的佛性。"

这是本论中由义理回到修行的关键转折 —— **四法不是抽象清净因的概念，而是上师加持下能被激活的实际修行品质。**

### 编辑注：四果"净我乐常" vs 声缘"常乐我净"——对治的对治

下一庚（庚二）的四果 "**净 / 我 / 乐 / 常**" 字面看与凡夫的 "常 / 乐 / 我 / 净" 四颠倒相同，也与声缘破此四颠倒所立的 "无常 / 苦 / 无我 / 不净" 所对治的东西相同。这里存在一个 **双重辩证**：

1. 凡夫颠倒执 —— 无常执为常，苦执为乐，无我执为我，不净执为净
2. 声缘对治 —— 以 "无常 / 苦 / 无我 / 不净" 破凡夫颠倒
3. 如来藏四果 —— **以 "净 / 我 / 乐 / 常" 破声缘所立的 "不净 / 苦 / 无我 / 无常"**

故如来藏之 "四果" ，乃是 "**对治的对治**" 所显之真实德相 —— 不是回到凡夫颠倒执，而是 **越过声缘的 "仅遮颠倒" 而安立的大乘四德**。此一细微但极重要的层次辨析，正是《宝性论》超越二乘法的关键所在。

这个 "如来藏之我"（大我 / 我波罗蜜多）**与外道 "神我" 根本不同** —— 须以二无我智慧为前提方可安立。详见 [`../../topics/tathagatagarbha/index.md §Correctness Anchors "如来藏不是外道的「我」"`](../../topics/tathagatagarbha/index.md#correctness-anchors)：

> 此"我"必须以**经过二无我抉择**为前提——先以《中观》破尽五蕴之我与法我，所余之**离戏自性光明**方名如来藏"真我"。**绕过自空中观而直说"我"即外道**。

《大般涅槃经》："我者即是如来藏义。一切众生悉有佛性，即是我义。如是我义，从本已来常为无量烦恼所覆，是故众生不能得见。"——注意此 "我" **必须** 在宝性论 "二无我智慧为能净因之二" 的框架下理解，否则与印度教梵我不二论无法区别。

四果的详细展开（癸一断除所断颠倒 + 癸二以此而得四果 + 癸三以得而解脱二边）将在下一文档处理。

## 七、小结：如何用这份框架进入如来藏品

1. **十义不是十个平行主题**——十义是 "法性胜义谛法界自性清净" 一法的十个侧面。遇到任何一义，应回问它与其他九义的关联（特别是 "本体" 与 "因 / 果 / 业" 的因果链， "分位 / 普行 / 恒常 / 无别" 四者的无变性群）。

2. **庚一本体因清净是核心论式的 "因端"**——四障 → 四对治 → 四佛子因 → 生佛子。此 "因" 与庚二 "果端" （四果净我乐常）构成本品的主要论证骨架。下一文档将从果端接续。

3. **三喻（珠宝 / 虚空 / 水）= 威力 / 不变 / 润体**——这三德贯穿整部论典。后续九喻（败莲佛 / 蜜蜂蜜等）也是在 "自性清净被客尘遮蔽" 同一立场上的进一步展开。

4. **四因对应四对治同时对应四果**——记住这个连环表：

   | 障 | 所障行人 | 对治因（四佛子因） | 四果（庚二将详说） | 所破颠倒 |
   |---|---|---|---|---|
   | 嗔法 | 大欲者 | 信解 | 净波罗蜜多 | 不净执 |
   | 我见 | 外道 | 智慧 | 我波罗蜜多 | 我执 |
   | 畏惧轮回苦 | 声闻 | 等持 | 乐波罗蜜多 | 苦执 |
   | 无视众生利 | 缘觉 | 大悲 | 常波罗蜜多 | 无常执 |

   这张表是 "**四因 / 四障 / 四对治 / 四果 / 四颠倒**" 五元结构的汇归 —— 弥勒菩萨以此一组数把对治与所得圆合无缝。

5. **宁玛警觉：在读 "如来藏具足威力 / 不变 / 润体三德" 时，以 "离戏自性光明大双运" 解读，不以 "实有法性" 解读** —— 详见 §六编辑注与 [`../../topics/tathagatagarbha/index.md §Editorial Policy`](../../topics/tathagatagarbha/index.md#editorial-policy)。 "四果" 中 "我" 字必经二无我抉择方立，否则即是外道神我。
