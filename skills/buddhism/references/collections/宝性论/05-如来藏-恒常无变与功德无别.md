# 《宝性论》如来藏品之三——三分位恒常无变与功德无别

本文件接续 [`03-如来藏-总纲十义.md`](./03-如来藏-总纲十义.md) 与 [`04-如来藏-果业功德趋入.md`](./04-如来藏-果业功德趋入.md)。03 立十义框架并处理庚一 "本体因清净"; 04 处理庚二—庚六 "果业 / 具功德 / 三所依 / 分位唯名 / 如虚空普行"。本文处理 **十义最后两义 —— 第九义 "恒常不变"（庚七）与第十义 "功德无别"（庚八）**。

此两庚共占讲记第 10—14 课五节, 是如来藏品 **科判最深、义理最密、公开 doctrinal 分量最重** 的单元。庚七以 "三分位 —— 凡夫 / 菩萨 / 佛 —— 各各无变" 铺开, 穷尽法界恒常之所以然; 庚八以 "法身 / 如来 / 圣谛 / 涅槃" 四名收摄为 "一味一体"。**并且 L14 是法王如意宝讲《宝性论》前九课时示现圆寂所讲的颂词** —— "佛外无有余涅槃" —— 以身示教印证第三转法轮的究竟立场。这是本集合处理中必须以独立小节郑重对待的教内事件。

## 一、庚七恒时无变总纲——三分位皆无变的立场

科判: 戊一·己三·庚七(恒时无变之义) 下分 **辛一宣说无变之分位** + **辛二广说彼理**。辛二再以三分位分为 **壬一凡夫位无变 / 壬二菩萨位无变 / 壬三佛位无变** 三支展开, 穷尽所有可能的 "存在阶段"。

### 辛一无变之分位——破 "先过后德" 之疑

L9 末尾的总纲颂定基调——**如来藏之所以无变, 是因为 "过失客尘 / 功德本具" 前后一如**, 并非 "前染后净" 的真实转变:

> **过失客尘性，功德性具故，**
> **如前后亦然，无变之法性。**

朵洛瓦注:

> "先前不清净阶段，烦恼等过失是可断除的客尘性，并不是自性中存在的缘故；后来清净的阶段现前力等功德，是自性中本来具足，并不是突然产生的缘故。如先前轮回界中存在一样，后来涅槃也这样存在永远、无有变化，这就是此法界的法性或自性。"

此颂同时立起 **两个立场**——**过失是客尘不是自性** + **功德本具不是新生**。前者与庚一 "自性本净" 呼应 (见 03 §五), 后者与庚二 "果功德来自离系而非修造" 呼应 (见 04 §1.2)。如来藏之无变不是 "静止不动" 的僵死, 而是 "客尘有进退, 法性无增减" 的动态恒定。

### 三分位无变结构总览

庚七的整体推理架构是 "**以最难立处立**"——最容易被误以为 "如来藏会变" 的就是 "凡夫阶段被烦恼遮蔽", 故先破之; 其次菩萨阶段 "有悲心示现生灭的显相"; 最后佛位 "有降生与涅槃的显相"。三分位皆破疑后, 恒常立场方真正坚固。

| 分位 | 对应众生 | 障碍 | 恒常如何显现 | 所用比喻 |
|---|---|---|---|---|
| **不清净** | 凡夫 | 烦恼障+所知障+习气未断 | 如虚空本无变, 客尘遮显而自性不染 | 虚空 / 器世界四大 / 三火不焚 |
| **不清净清净** | 菩萨 (一地-十地) | 未究竟断障, 有无漏业与意生身 | 虽悲心示现变化(入胎/老/死/病), 本体无变 | 莲花生水不染 / 长者救子粪坑 |
| **极清净** | 佛 | 无任何障, 二障习气皆断 | 远离生死变, 具常/坚/寂/永四德 | (另立四名无别而论, 见 §五) |

全部推理汇归于一立场: **三分位仅是客尘遮否之差, 法性本体无丝毫改变**。此即 [`03-如来藏-总纲十义.md §七`](./03-如来藏-总纲十义.md) "法性胜义谛法界自性清净一法的十个侧面" 在 "分位" 这一侧面的穷尽展开。

## 二、壬一凡夫位无变——虚空喻与 "骤然成轮回"

科判: 庚七·辛二·壬一(因障碍不清净分位亦无变之理) 下分 **癸一如虚空无变之理** + **癸二解说彼等差别**。

### 2.1 癸一虚空无变总颂（L10 子一略说）

> **如虚空普行，细微故无染，**
> **如是于众生，安住此无染。**

虚空作为无为法, 有 **三个特点**——周遍、细微、无染。索达吉堪布（课10）析:

> "虚空是一种无有因缘造作的无为法，同时又是一种无有阻碍、非常细微的法。……以细微的缘故，色法虽有无常、易坏的性质，但虚空从未被其染污过。"

**四重对应**:

| 虚空 | 如来藏 |
|---|---|
| 周遍一切色法 | 周遍一切众生 |
| 细微无阻 | 离戏无阻 |
| 色法无常,虚空不染 | 众生烦恼,法性不染 |
| 非因缘所摄 | 非业惑所生 |

唐不空译《百千颂大集经地藏菩萨请问法身赞》为根本教证:

> "法界亦不生，亦不曾坏灭，一切时不染，初中常无垢。"

"初 / 中 / 常" 即对应众生位 / 菩萨位 / 如来位三分位, 皆无垢染。

#### 文殊菩萨调化两百比丘公案的教理意义

L10 详讲《大宝积经》文殊菩萨化身比丘随众同出, 以同理心引导两百比丘观心本体 "无颜色 / 无常无常 / 无里外 / 无真实圆成实法", 最后令其当下开悟。此公案的教理核心是——**即使 "一切法皆是法界自性、清净无染、无解无缚" 听起来令人 "不想修行", 但正是由于心本体即空即净, 方有解脱可言**。文殊菩萨答须菩提: "胜义中无有调伏, 世俗中如幻如化而调伏, 无有真实。" —— 净见量与观现世量之别即在此处。

### 2.2 癸一·子二·丑一有法现生灭之理（L10）

总颂:

> **世间诸一切，空中生与灭，**
> **如是无为界，根等生与灭。**

#### 三层生灭辨析

这里最核心的结构是 "**生灭的三层包裹**":

| 层 | 所指 | 生灭方式 | 所依 |
|---|---|---|---|
| **器世界层** | 地/水/火/风 | 刹那刹那成坏 | 虚空 |
| **众生有法层** | 蕴/界/根 | 胎生卵生湿生等有生灭 | 如来藏(无为界) |
| **法界层** | 如来藏本体 | **无生无灭** | 不依不住任何法 |

坚慧菩萨《大乘法界无差别论》教证:

> "如一切世间，依虚空起尽，诸根亦如是，依无为生灭。"

此即 L10 最重要的一句立场——**显现层的生灭, 从不触动法界层的无生无灭**。

### 2.3 癸一·子二·丑二法界无灭之理——三火不焚

> **虚空未曾为，诸火所焚毁，**
> **如是此不为，死病老火焚。**

三火与生老病死的对应 (L11 子二详说三火差别):

| 三火 | 焚烧对象 | 对应众生生灭 |
|---|---|---|
| **劫末火** | 整个器世界 | 死 (劫末式结束生命) |
| **地狱火** | 有情身心苦楚 | 病 (如地狱式剧痛) |
| **自性火**（平常火） | 柴薪 / 身体老化 | 老 (渐变如柴尽) |

无著菩萨注引《胜鬘经》: "世尊, 死者诸根坏。世尊, 生者新诸根起。" —— **生死依 "根" 安立, 而如来藏不在 "根" 上, 故不在生死中**。

### 2.4 癸一·子二·丑三器世界与内心世界的同构对应

**比喻**:

> **地依水依风，风住于虚空，**
> **虚空则非住，风水与地界。**

**意义**:

> **如是蕴界根，依于业烦恼，**
> **业惑恒常依，非理之作意。**
> **非理作意者，依存清净心，**
> **心之自性法，不住于一切。**

**对应**:

> **当知犹如地，一切蕴处界，**
> **当知如水界，有情业烦恼。**
> **非理作意者，观如风之界，**
> **自性如虚空，不依亦不住。**

即建立 "**外器 / 内识同构**" 的五层对应:

| 外器(喻) | 内识(义) |
|---|---|
| 地轮 | 蕴界处(五蕴十八界十二处) |
| 水轮 | 业与烦恼 |
| 风轮 | 非理作意 |
| 虚空 | 清净心 / 如来藏 |
| 虚空不依任何法 | 心自性不住于一切 |

《大方等大集经》教证:

> "喻如大地依水界住，大水依风界住，大风依虚空住，虚空无所依住。如是大地无所依住，而假有依住之名。"

此段被《大圆满心性休息》《七宝藏》反复引用, 是 "以外喻内" 的 locus classicus。

### 2.5 癸一·子二·丑四骤然成轮回之理（L11）

> **非理作意者，住于心自性，**
> **非理作意者，促成业烦恼。**
> **业与烦恼水，生诸蕴处界，**
> **如彼坏与成，形成生与灭。**

**关键论式**——**本无轮回, 骤然而成** (这是宁玛 / 大圆满的直接衔接点):

| 骤然生起 | 本位 |
|---|---|
| 非理作意 → 业 → 烦恼 → 蕴界处 → 生老病死 | 如来藏清净光明本体 |
| (客尘因果链条) | (本无客尘故曰 "骤然") |

索达吉堪布 (课11) 将此颂密续化解释:

> "所谓'非理作意'就是常、乐、我、净的执著，**密宗阐述为在本基当中显现基现的时候，由于没有认识自心本面，依靠遍计无明和俱生无明，从而现前形形色色的外境，继而漂流于轮回当中。而普贤王如来等诸佛却能认识自己的本性，所以不再流转轮回。**"

此为 **宁玛 "本基显基 / 本净与客尘二无因" 立场** 的显教根据——《宝性论》此颂明文支撑大圆满 "本来清净 + 自现客尘" 的根基观。故 L11 明确说: "《宝性论》是连接显宗和密宗的纽带"。

此即为什么本论被藏文称为 **"无上续"** (rgyud bla ma) 的主要依据——颂义本身已通达续部立场, 故部分前代大德要求讲闻《宝性论》前需灌顶。

### 2.6 癸一·子二·丑五自性无变之理（L11 核心）

> **心性如虚空，无因亦无缘，**
> **无聚无有生，无灭亦无住。**

虚空的 "四无"——**无近取因 / 无俱生缘 / 无因缘聚合 / 无生住灭**——悉皆应于心性如来藏。此颂是 L11 点名 **最重要的教证** 之一:

《华严经》: "设一切众生, 一时成正觉, 若成若未成, 菩提无增减。"

《大集经》: "如空常住, 无有败坏, 诸法亦尔, 常住法界。"

麦彭仁波切《定解宝灯论》即依此立场直指心性:

> "尔时如视前虚空，自心正在动念时，亦为空性之定解，必须深深而生起。"

索达吉堪布（课11）将此颂作为 "显宗与密宗打通" 的直接接点——虚空在显宗是比喻, 在密宗是直指心性的窍诀。第一世敦珠法王即常以 "直视虚空" 作大圆满认识心性之门。此处应参 [`../../topics/tathagatagarbha/index.md §Cross-Topic Connections → tantra`](../../topics/tathagatagarbha/index.md#cross-topic-connections)。

### 2.7 癸一·子三摄义——客尘不染的最终立场

> **心自性光明，无变如虚空，**
> **邪念生贪等，客尘不染彼。**

**三杂染全属客尘** (据《大乘阿毗达磨集论》):

| 三杂染 | 内容 |
|---|---|
| 烦恼染污 | 非理作意生贪嗔等 |
| 业染污 | 以烦恼造善恶业 |
| 生染污 | 以业惑流转生轮回 |

**三者皆 "客"**——客人暂住非主人。《释量论·成量品》为根本教证:

> "心自性光明，诸垢是客尘。"

龙猛菩萨《释菩提心论》以 "甚奇" 赞叹光明如来藏与业果无欺二者同时成立:

> "知此诸法空，而依业果者，甚奇此甚奇，稀有此稀有。"

## 三、壬二菩萨位无变——悲心示现变化（L11 下–L12）

科判: 庚七·辛二·壬二(不清净清净分位亦无变之理) 下分 **癸一真实宣说不以生等改变之义**(子一略说 + 子二广说) + **癸二认清十地佛子功德**(子一—子五)。

### 3.1 略说——证真如 + 悲心入世二一合

> **解脱生死与病老，如实证悟此自性，**
> **离生等祸以彼因，具慧悲众故依存。**

此颂是菩萨 "**智不住三有 / 悲不滞涅槃**" 立场的根本颂基——菩萨已解脱生老病死, 却因悲心而继续安住世间。

《维摩诘经》教证:

> "或示老病死，成就诸群生，了知如幻化，通达无有碍。"

### 3.2 丑一不以痛苦变化之理

> **圣者已根除，死病老痛苦，**
> **以业惑感生，彼无故无彼。**

核心论式——**"彼无故无彼"**: 菩萨以见道修道已断非理作意与业惑, 故老病死之因已无, 故此 "彼" (痛苦) 也无。魏译本 "老病死诸苦, 圣人永灭尽, 依业烦恼生, 诸菩萨无彼。"

**结论**: 菩萨为利众而示现生老病死, 并非真受生老病死之苦。索达吉堪布: "凡夫看到菩萨为了利众而示现生老病死等感人事迹的时候, 会误以为他们很痛苦, 但事实上, 这些菩萨没有丝毫疾病、衰老和死亡的痛苦。"

### 3.3 丑二由悲心显现变化之理

> **真见真如故，已超越生等，**
> **然悲尊示现，生死及老病。**

**长者入粪坑救子喻** (出《海慧菩萨品》 /《佛说无上依经》, L11 详引):

| 喻 | 所表 |
|---|---|
| 粪坑 | 三界轮回 |
| 长者之子 | 众生 |
| 长者父 | 菩萨勇士 |
| 母亲及亲属 (躲外哭唤) | 声闻缘觉 |
| 跳入救子 | 大乘方便 (悲心示现轮回) |

此喻是 "大乘道与二乘道之根本差别" 的 locus classicus——**同情众生却不敢入, 与不惜自身跳入粪坑救子, 差别在此**。

### 3.4 癸二十地佛子功德五层（L12）

科判癸二下五子:

| 层 | 所指菩萨 | 核心功德 | 关键颂 |
|---|---|---|---|
| **子一发心** | 一地菩萨 | 现量证悟无变法性, 众生反见生等稀有 | "佛子证悟此, 无变法性已" + "获得圣行境, 示于凡行境" |
| **子二入胜行** | 二—七地 | 智不染轮回, 悲不舍世间 (莲花生水不染喻) | "彼离诸世间, 世间中不动" + "如莲生水中, 不为水所染" |
| **子三不退转** | 八地 | 利众心恒如火燃, 同时入定寂戏论 | "成办所为事, 心恒如火燃, 恒常入定寂, 静虑等至中" |
| **子四最后有** | 十地 | 离分别任运利生, 四种应机(说/色身/行/威仪) | "先前引业感, 离诸分别故" + "或说或色身, 或行或威仪" |
| **子五成办二利差别** | 十地 vs 佛 | 利他同佛, 自利远差 (大地/微尘海/蹄迹水喻) | "然如地与尘, 海与蹄迹水, 佛陀与菩萨, 差别亦如此" |

#### 一地发心——"混迹众生而成至亲"

一地菩萨现量证悟无变法性, 自身已脱生死, 却在被无明翳蔽的众生面前示现生老病死。《入行论·智慧品》教证:

> "为度愚苦众，菩萨离贪惧，悲智住轮回，此即悟空果。"

菩萨 "得圣行境 / 示凡行境" 的悲心方便, 不是 "迁就众生的妥协", 而是 "融入众生而至亲"——通过四摄 (布施 / 爱语 / 利行 / 同事) 使 "猎人、妓女等任何众生" 成为密友, 方得引入解脱道。

#### 二—七地入胜行——莲花喻

> **如莲生水中，不为水所染，**
> **此虽生世间，不染世间法。**

莲花喻的关键是 "**长期浸润而不染**"。布匹、木头水中浸渍即染污, 但莲花终生生于水中愈加艳丽。《华严经》:

> "菩萨种种方便门，随顺世法度众生。譬如莲花不着水，如是在世令深信。"

#### 八地不退转——火燃喻

八地 "心恒如火燃" 是 **任运无勤** 的直接立场——如干柴遇强火自然燃烧, 不需任何勤作。同时 "恒常入定寂" —— 在利生的同时安住于离戏根本定。这是凡夫难以想象的 "**入定与后得无二**" 境界。

#### 十地最后有——任运利生四方便

十地菩萨以先前九地发愿引业 , 所知障基本断尽, 离一切分别任运利生, 四种方便:

| 方便 | 内容 |
|---|---|
| **说** | 宣说大小显密法门 (或缄默) |
| **色身** | 显现国王/婆罗门/凡夫/乞丐等身相 |
| **行** | 六度四摄, 显密行仪 |
| **威仪** | 行住坐卧皆利生方便 |

**文殊菩萨加持魔王说法**公案 (L12 详引): 文殊加持魔王波旬示佛身说法, 五百人证无生法忍, 又加持舍利弗示佛身, 共一千三百人得度。示 "十地任运 / 加持无碍" 之极致。

#### 十地与佛的自他差别

> **然如地与尘，海与蹄迹水，**
> **佛陀与菩萨，差别亦如此。**

**两组比喻**:

| 比喻 | 十地菩萨 | 圆满佛 |
|---|---|---|
| 微尘 vs 大地 | 微尘功德 | 大地功德 |
| 牛蹄迹水 vs 大海 | 蹄迹水智慧 | 大海智慧 |

无著菩萨注引《宝髻菩萨品》城门喻: 长者一足入城门一足未入, 想起城外独子遂返身。此喻部分十地菩萨暂不入 "大涅槃城", 留于世间救众生。

## 四、壬三佛位无变（L13）

科判: 庚七·辛二·壬三(极清净分位亦无变之理) 下分 **癸一略说无变之理由** + **癸二广说彼义**。

### 4.1 癸一略说——四因成立佛位恒常

> **无尽具法故无变，无后际故众皈依，**
> **不分别故恒无二，无作性故无灭法。**

四因对应佛位恒常四德:

| 因 | 果 | 所对 |
|---|---|---|
| **无尽具法** (法身具无量功德) | **无变** (常) | 色身生灭 |
| **无后际** (乃至轮回边际恒救度) | **众皈依** (坚) | 无常皈依处 |
| **不分别** (无轮涅过功分别) | **恒无二** (寂) | 二取分别 |
| **无作性** (非因缘造作) | **无灭** (永恒) | 有为坏灭 |

《大般泥洹经》教证: "从今于如来, 莫念非常想, 当知如来性, 长存不变易。"

**关键立场** (承 [`02-总说-三宝本体.md`](./02-总说-三宝本体.md)): **胜义皈依处唯一佛**——因为法、僧二宝皆有变化可能, 唯佛的法身无后际。

### 4.2 癸二·子一·丑一远离生等四德

> **无生亦无死，无害亦无老，**
> **彼乃常坚故，寂灭永恒故。**

**四德专名**——须背诵:

| 专名 | 对治 |
|---|---|
| **常有** | 无生 |
| **坚固** | 无死 |
| **寂灭** | 无疾病损害 |
| **永恒** | 无变异衰老 |

宝意汉译《究竟一乘宝性论》: "不生及不死, 不病亦不老, 以常恒清凉, 及不变等故。" —— 常 / 恒 / 清凉 / 不变对应常 / 坚 / 寂 / 永。

《文殊师利菩萨问法身经》教证:

> "法身无有生，若有生乃有灭，法身者不生故，知佛而不灭。"

#### 佛示现生灭与水晶喻

常见疑问: 佛于蓝毗尼园生 / 拘尸那城涅槃, 明明有生灭。**《文殊师利问经》水晶喻** (L13 详引):

> "如琉璃珠，清净无垢，若值白物，青、黄、赤物，此琉璃珠则随物色，琉璃无心令见异色。……如来亦尔。或有众生，见佛涅槃、转法轮，见降众魔，见并现神通……如众生意悉见，如来如是。"

水晶自体清净透明, 周围有色光即随现色相; 佛自体恒常无变, 于不同众生业感前即示现不同身相。

《金光明最胜王经》妙幢菩萨疑问 (L13 详引): 佛具不害众生 / 布施之因, 何以寿量仅八十岁? 四方佛答: "一切诸海水, 可知其渧数, 无有能数知, 释迦之寿量。" 八十寿量是 **为示无常令众生起珍惜心** 的方便示现, 非佛法身真有寿量。

### 4.3 癸二·子一·丑二对应广说——菩萨之"生死病老" 于佛位皆无

> **彼即意性身，不生常有故，**
> **不可思死殁，不死坚稳故。**
> **细微习气病，不害寂灭故，**
> **现行无漏业，不老永恒故。**

这个对应是 "佛超越菩萨" 的关键 —— 菩萨的四种 "生死病老" 都是极微细的变异, 佛位皆无:

| 菩萨之四种变异 | 佛位如何无 |
|---|---|
| **意性身** (意识形态身, 随愿 / 无漏业化生于净土) | 不以意性身生,故常有 |
| **不可思死殁** (菩萨在一刹土隐没至另一刹土, 不可思议的"死") | 不以此死, 故坚稳 |
| **无明习气地之病** (最细微所知障习气) | 不以此损害, 故寂灭 |
| **现行无漏业** (以愿力 + 无漏业所感意生身) | 不以此衰老, 故永恒 |

**重点**: 声闻无余涅槃尚有 "意生身 + 无明习气地 + 无漏业 + 变异生死" 四种微细生灭, 故 **非究竟**。这一立场是后面庚八 "佛外无余涅槃" 的直接理论根据——声缘涅槃还没断尽这四种, 怎能与佛位究竟涅槃同日而语?

### 4.4 癸二·子二摄义承接 + 子三安立常有等之因

**摄义**:

> **二句如是二，二二依次第，**
> **了知无为界，常有等之义。**

**安立因**:

> **具无尽德故，非变性常义，**
> **等同边际故，皈依性坚义，**
> **不分别性故，无二法性寂，**
> **无改功德故，无毁义永恒。**

此即将癸一四因与子一四德对应收束——**功德无尽 → 常 / 等同边际 → 坚 / 不分别 → 寂 / 无改功德 → 永恒**。

## 五、庚八功德无别总纲（L13 末–L14）

科判: 戊一·己三·庚八(功德无别之义) 下分 **辛一以名义略说** + **辛二广说彼等本体**。辛二再分壬一名称差别 / 壬二意义本体 / 壬三比喻决定。

### 5.1 辛一以名义略说——"彼即法身即如来" 核心颂

**这是本论全部最重要的颂词之一, 也是法王如意宝圆寂前讲授的颂词**:

> **彼即法身即如来，圣谛胜义之涅槃，**
> **故如日光德无别，佛外无有余涅槃。**

朵洛瓦注:

> "如来藏极其清净的分位，它是一切佛陀的法身，同样也是如来，也是圣谛、胜义涅槃，因此这些仅是名称的差别。为此，就像太阳与阳光无别一样，十力等胜义功德无别，因此，除圆满佛陀外，无有其余真正的涅槃。"

**核心立场**: 法身、如来、圣谛 (胜义谛)、涅槃 **四个词是同一实相从四个侧面立的名**; **佛位以外没有真正的涅槃**。

### 5.2 壬一·癸一—癸二四义四名对应 (L14 主体)

总颂:

> **略摄无漏界，以四义类别，**
> **可知法身等，四种名差别。**

对应颂:

> **佛法无有别，获得彼种姓，**
> **无妄无欺性，自性本寂灭。**

**完整四名四义对应表**:

| 四名 | 四义 | 定义 | 所立角度 |
|---|---|---|---|
| **法身** | **无二** (佛法无有别) | 佛不共功德(十力/四无畏/十八不共法)无别具足 | 果法无别侧面 |
| **如来** | **种姓** (获得彼种姓) | 如实而来 / 如实而去, 无始安住法性 | 本来如是侧面 |
| **圣谛** (胜义谛) | **无妄无欺** | 永不虚妄恒常无欺的圣者证境 | 真实性侧面 |
| **涅槃** | **自性本寂灭** | 本自离戏寂灭所有客尘 | 本净侧面 |

教证 (《不增不减经》, L14 明引):

> "甚深义者即是第一义谛，第一义谛者即是众生界。众生界者即是如来藏，如来藏者即是法身。"

—— 四名为一, 层层等同: 第一义谛 = 众生界 = 如来藏 = 法身。

### 5.3 **重点张力** —— 二转 / 三转 "法身" 义之别

L14 明确作出 **宁玛关键判教宣示**——这是本节最重要的 doctrinal 分层:

索达吉堪布:

> "第二转法轮所讲的法身是空性的，尤其是密法中讲到本来清净时，本体空性叫法身，自性光明叫报身，都是从空性层面来讲的；而**第三转法轮有所不同，在第三转法轮中，法身的概念是什么呢？即是具足如海一般广博的佛法——佛不共的功德**，包括刚才讲的十力、四无畏，这些全部都是智慧。那具足这些智慧叫做什么呢？这就叫做法身，所以用'无别'来说明法身。"

**二转 / 三转 "法身" 双照表**:

| 法轮 | 法身义 | 所诠侧面 |
|---|---|---|
| **第二转** (般若经/中论) | 本体空性 (离戏 / 无遮) | **空分** |
| **第三转** (如来藏经/宝性论) | 具足如海广博佛不共法 | **光明分** |
| **二者合参** | 空性与光明无二 — 即大双运 | 同一实相 |

**这是全《宝性论》最显著的 "二转三转融通" 教点**——两转所诠实相无二, 仅诠说侧面不同。见 [`../../topics/tathagatagarbha/index.md §Correctness Anchors "第二转与第三转所诠实相无二"`](../../topics/tathagatagarbha/index.md#correctness-anchors) 与 [`../../topics/madhyamaka/debates.md §1 第九诤辩`](../../topics/madhyamaka/debates.md)。

朵洛瓦觉囊派倾向 "第三转胜第二转" (二者所诠有高下别), 而索达吉堪布承麦彭立场柔和解为 "双照"——保留朵洛瓦对第三转 "法身具功德" 的直白表述, 同时以 "空分 / 光明分侧面不同" 避免义他空倾向。

### 5.4 壬二·癸一广说无二之义

> **诸相正等觉，断垢及习气，**
> **佛陀与涅槃，胜义无二性。**

**两面收为一面**:

| 面 | 内容 | 立名 |
|---|---|---|
| 证德 | 以尽 / 如二智证一切相 | 圆满正等觉 (佛陀) |
| 断德 | 无余断除二障连同习气 | 涅槃 |

二者在无漏法界中 **一味一体**, 仅从不同反体立不同名。

**明镜喻** (L14):

| 擦镜角度 | 所立 |
|---|---|
| 将微尘擦净 | 断德 |
| 微尘擦净后自然显影 | 证德 |

**因明立场**: 相续中智慧增长, 障碍自然断除——并非如外在两微尘碰撞一败一胜的物质相违。《现观庄严论》 "三大" 章有详说。

### 5.5 壬二·癸二广说涅槃之义

> **一切相无数，无思无垢德，**
> **无别相解脱，解脱即如来。**

**四层功德** (涵盖从广大 / 甚深 / 清净三个维度):

| 层 | 功德 | 所含 |
|---|---|---|
| 广大 | **一切相** (种姓无量无边, 声缘/菩萨/佛三种姓皆具) | 十力 / 四无畏等 |
| 别法广大 | **无数** (别法无量, 如 "处非处智力" 知一切法) | 恒河沙数万法全通 |
| 甚深 | **无思** (甚深难测, 超越一切声缘 / 凡夫 / 外道揣度) | 如 L13 菩萨妙幢之疑 |
| 清净 | **无垢** (清净二障连同习气) | 二障尽断 |

**重要立场** (L14): 大乘涅槃与声闻乘涅槃 "**内容有差异**" —— 声闻的 "得解脱" 是断烦恼证寂灭; 大乘 "得解脱" 是了知 "本来已经现前正等觉" (《大般涅槃经》迦叶问答)。两种 "解脱" 从结构上就不同。

《大宝积经》教证:

> "如来出世。若不出世。法不增减不遍增减。诸法自性本性常住。"

## 六、壬三·癸一图画喻——六度皆全方成法身

> **犹如绘画者，精通各有别，**
> **知某分支者，去他处不定。**
>
> **复次君主王，赐布命彼等，**
> **汝等所有人，于此绘吾像。**
>
> **彼等听从命，尽力行画事，**
> **正当行彼等，一人赴他境。**
> **因彼往他境，不全故彼画，**
> **一切诸分支，不能皆圆满。**
>
> **能绘一切者，具施戒忍等，**
> **一切相之最，空性说成像。**

**六度 (六画师) 与法身对应**:

| 画师 | 六度 | 所 "绘" 法身功德 |
|---|---|---|
| 1 | 布施 (无贪) | |
| 2 | 持戒 (无罪垢) | |
| 3 | 安忍 (无扰乱) | |
| 4 | 精进 (喜善法) | |
| 5 | 禅定 (一缘专注) | |
| 6 | 智慧 (辨别善恶) | "一切相之最" 空性 |

**核心立场**: **六度缺一不成法身**。只修空性智慧不行布施持戒等, 法身画像缺角。《大集经》教证:

> "若有一行不成就者，不名具足如来正法。是故要当具足诸行，名为成就无上菩提。"

索达吉堪布 (L14) 以此颂直接破斥 "只修大圆满不用修加行 / 持戒 / 布施" 的立场——此等立场 "六度不全如画像缺头, 成了抽象画看不出是国王"。

**"空性" 含义的重要辨析** (L14):

> "'空性'指具足一切方便之最的空性，**既不是一个单空，也不只是离四边八戏**。……具足了从布施、持戒，直到智慧波罗蜜多等方方面面的一切功德，六波罗蜜多全部具足、三轮体空的空性智慧，如此这般显空双运的这种空性，就叫做法身。"

**再次宣示二转 / 三转之别**:

> "我们这里所讲的法身，与二转法轮所讲的'法身'不同。"

—— 三转 "法身" = 显空双运 + 六度具足 + 一切相之最的空性; 二转 "法身" = 单从空分抉择的离戏。本质相同, 侧面不同。

## 七、壬三·癸二日轮喻 + **"佛外无余涅槃"**

### 7.1 子一日轮四同法

> **慧智及解脱，明射清净故，**
> **无异故与光，光辉日轮同。**

**三德与日轮三德对应**:

| 佛三德 | 日轮三德 | 作用 |
|---|---|---|
| **如所有智** (妙慧) | **能明** (光) | 打破黑暗 |
| **尽所有智** (本智) | **能射** (光辉) | 照耀万物 |
| **清净二障解脱** | **能净** (日轮清净本体) | 清净所有障碍 |

三者无异无别, 即同一日轮的光 / 辉 / 净三面, 不可拆分。

### 7.2 子二 **佛外无究竟涅槃** 的最终颂

> **故未成佛前，不能得涅槃，**
> **如离光光辉，不能见日轮。**

朵洛瓦注:

> "无始以来安住的界性自性清净如来藏中（十）力等一切胜义功德无别，现前它就是究竟解脱，因此，在没有证得照见一切所知无碍智慧的佛果之前，不会获得解脱一切障碍的究竟涅槃。"

**立场核心**: **佛位以前所有 "涅槃" 皆非究竟**——包括声闻无余涅槃、缘觉涅槃、菩萨十地涅槃, 皆未断尽习气 (见 §4.3 菩萨四种微细变异), 故终须回大乘成佛。

教证:

- 《大乘法界无差别论》: "法身即涅槃界。涅槃即是如来。"
- **一乘究竟立场**: 与《妙法莲华经》 "唯有一乘法, 无二亦无三" 合流。

#### 跨 topic 关联——与定解宝灯论 Q2 的关键会通

此 "佛外无余涅槃" 立场与 [`../定解宝灯论/Q2-声缘证二无我耶.md`](../定解宝灯论/Q2-声缘证二无我耶.md) 所辨的 "声缘证悟是否究竟" 同构 —— **宁玛立场**: 声缘未证尽法无我, 其涅槃仅是 "自解脱阶段性果位", 终须回入大乘方成究竟涅槃。《宝性论》此颂即是此立场的本颂根据之一。

## 八、**法王如意宝示现圆寂的密意** ——核心教内事件

此节是本集合全部 doctrinal 处理中必须郑重对待的教内事件, 当独立陈述。

### 8.1 历史事件

2004 年 1 月, 法王晋美彭措 (法王如意宝) 在喇荣五明佛学院讲授《宝性论》。法王先圆满传讲麦彭仁波切《宝性论释》 (即所谓 "前九课"), 接着开始讲觉囊派全知朵洛瓦《宝性论释》。

讲至庚八辛一 "彼即法身即如来, 圣谛胜义之涅槃, 故如日光德无别, **佛外无有余涅槃**" 此颂时, 法王于所化众生面前示现圆寂, 色身融入法界。索达吉堪布在其法本此处亲笔标注:

> "讲到此处，法王如意宝色身显现上融入法界。"

此前 2003 年极乐法会, 法王已明示:

> "这可能是我最后一次参加极乐法会了，虽然我也希望不是最后一次，但确实很有可能。如果我们师徒即生无法再遇，就让我们发愿，在西方清净刹土、在阿弥陀佛的座下，依靠宏大的愿力，我们能够再聚。"

### 8.2 索达吉堪布的两段咏叹 ——完整保留

**L13 课末段** (法王下山治病前日):

> "第二天法王出发时，恭送法王的队伍从山那边一直排了过来。雪花零星飘扬，缓缓洒落，我和齐美仁增堪布一路走到居士林，到那里的僧众已经寥寥无几。当车辆途径我们这边时，法王将车窗摇下来，向我摆了摆手……从此以后，我再也无法见到这一幕了。
>
> 多年以来，法王一直为我们传讲佛法，最后止于这个偈颂而示现圆寂。"

**L14 开篇**:

> "现在《宝性论》讲到'如来藏品', 本品共宣说了十种功德, 分为八个科判, 今天我们讲最后一个科判, 即'功德无别之义'。上节课我们讲了'以名义略说', 讲到法身、如来、圣谛、涅槃, 其实这些功德都是无二无别的, 也就是说, 除了成佛以外没有其他的涅槃。**我们的大恩上师——三界导师无上正等觉法王如意宝晋美彭措讲到上面这个偈颂的时候, 暂时在所化众生面前示现涅槃, 色身融入法界。**"
>
> "法王老人家在时局那样艰难的年代, 也坚持不断地弘扬佛法, 最后在讲到《宝性论》这里的时候, 示现圆寂。
>
> 不管是否亲见过法王如意宝, 我们都应深深地思考——为什么是这样的? **法王老人家在宣讲其他经典、论典的过程中, 都没有示现圆寂, 而恰好在讲解第三转法论的《宝性论》'如来藏品'中'除佛以外没有其他涅槃'的时候而示现涅槃呢?**"

### 8.3 密意解读

索达吉堪布 (L13) 的回溯式反思:

> "当年我们无法参透其甚深含义。如今时光流转，十几年一瞬即过，这才恍然明白，原来当年法王如意宝示现的圆寂，和普通人的死亡是有所不同的。为什么呢？**我们认为上师已经涅槃了，要开法会等缅怀纪念。而这里颂词'佛外无有余涅槃'，其实也正意味着，上师已经获得了最高的境界。**"

**三重密意**:

1. **以身示教印证颂义**——颂曰 "佛外无余涅槃", 法王即在此颂处示涅槃, 正是宣示自己所入非声缘式无余涅槃, 而是 "佛外无余" 的究竟涅槃。
2. **荼毗瑞相印证正等觉位**——《七宝藏》续部教证所列 "圆满正等觉相" (光团 / 椭圆形 / 梯阶状 / 竖立状) 在荼毗日空中显现。
3. **一乘究竟宣示**——与 "法身无余涅槃界中也不穷尽具足胜义无量功德法" 的教义一以贯之; 上师即是法身之体, 色身融入法界即法身之显发位。

曲恰堪布亦是讲《经庄严论》未圆时示现圆寂, 法王曾云: "我以后也会这样。" —— 传法中示现圆寂是教内极稀有的身示传承方式, 汉地遇安禅师入棺公案亦同此理。

### 8.4 教内典故的处理原则

本集合叙述此事件时, 严守三原则:

1. **尊师重道的敬语表述**——不可当作一般传记事件叙述
2. **doctrinal 核心不可偏离**——法王圆寂的重心是 "**以身示教印证佛外无余涅槃究竟唯一乘**", 不是偏重灵异瑞相
3. **与 [`../../topics/tathagatagarbha/index.md §Correctness Anchors "佛外无涅槃"`](../../topics/tathagatagarbha/index.md#correctness-anchors) 立场绑定交叉引用**

## 十一、小结——本文立下的关键论式

1. **三分位恒常** (庚七主论):
   - 凡夫位: **虚空喻 + 器世间四大 / 三火 + 骤然成轮回** → 客尘遮而法性不染
   - 菩萨位: **解脱生死 + 悲心示现 + 莲花喻 + 长者粪坑喻 + 十地五层功德** → 悲心显变化而本体无变
   - 佛位: **断证圆满 + 四因 (无尽具法 / 无后际 / 不分别 / 无作性) + 四德 (常 / 坚 / 寂 / 永) + 四种微细变异皆无 (意性身 / 不可思死殁 / 习气病 / 无漏业)** → 无变恒常

2. **功德无别** (庚八收束):
   - **四名四义**: 法身 / 如来 / 圣谛 / 涅槃四名, 无二 / 种姓 / 无妄无欺 / 自性本寂灭四义, 一一对应
   - **日轮喻四同法**: 慧智解脱三德 = 光光辉日轮清净三德无异
   - **图画喻六度**: 六度缺一不成法身, 空性须显空双运而非单空或离戏
   - **佛外无余涅槃**: 究竟涅槃唯一乘——声缘涅槃终非究竟

3. **二转 / 三转法身双照** (L14 关键):
   - 二转 "法身 = 空性" (空分侧面)
   - 三转 "法身 = 具足如海广博佛不共法" (光明分侧面)
   - 合参即大双运 = 离戏大中观 = 离戏自性光明

4. **法王如意宝示现圆寂** —— "以身示教, 佛外无余涅槃" 教内重大事件
