# 《宝性论》教证索引

本文是《宝性论》集合内 13 篇知识文档 + 结构总览所引**经、论、赞、续**等原始圣教量的汇总索引。以所引典籍为索引键; 每条记录被引的 doc 位置、原文 (或关键句)、以及论证功能 (**经证**=以经为据成立义理; **辅证**=加强主论立场; **对治教导**=实修指引)。

如来藏教典的特殊性在于: 它**不是从理论推演中建立, 而是以佛语为根据显示**。因此, 教证索引对如来藏 topic 的 agent 回答**至关重要**——任何 "如来藏论述" 若缺乏经教支点, 即滑入臆说。

## 使用规则

1. **引用一条教证前, 先看此索引确认原文**, 不要凭记忆补句。
2. 同一教典多次被引时, 每次引用皆独立记录, 因为**上下文 (哪一 doc、哪一 §、说什么义)** 决定了引文的效力。
3. 本索引只列**明引的教证**, 隐引意趣 (如论中 "释论云云") 不列。
4. 覆盖不完全 (有些处仅列典籍名无引文), 需查原 doc 补。
