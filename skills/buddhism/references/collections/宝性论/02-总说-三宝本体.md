# 《宝性论》总说品之二——三宝本体与胜义皈依处

本文件覆盖丙一"广说所得三宝之分支"全部——三宝各自本体（丁一）、安立三皈依之理（丁二）、以释词宣说（丁三），对应讲记第 3–5 课。本文是《宝性论》**最核心的教判动作**：**重新界定佛法僧**——从显宗一般认知的"色身佛像、经教文字、出家僧团"，转向 **胜义层面的三宝本体**，并进一步抉择"**三宝中唯一佛宝是究竟皈依处**"。

这项动作至关重要：它既是后续四金刚处（如来藏/菩提/功德/事业）的教义铺垫——"果" 端有了胜义本体的界定，"因" 端的如来藏才能被看作真正的成佛之因；也是修行上的基础——皈依若止于相似三宝，修行的最终方向便无法定位。

## 一、胜义三宝 vs 相似三宝——一场重新界定

索达吉堪布第3、4课再三强调，《宝性论》里讲到的三宝，"和我们平时所说的佛法僧三宝，**名称虽然一样，但在内容的深度方面有所差别**"（课2）、"**我们认为的这种三宝只是相似的三宝，这里所讲到的才是真正的三宝**"（课4）。

| 层面 | 相似三宝 | 胜义三宝（本论立场） |
|---|---|---|
| 佛宝 | 色身、化身、唐卡佛像、寺院塑像（释迦牟尼、药师佛等形象） | 具足二利八功德的**法身佛** |
| 法宝 | 经文、法本、教法文字 | **灭谛与道谛**所摄、具足六/八种功德的无漏法 |
| 僧宝 | 比丘四众、出家僧团 | 具足**如所有智与尽所有智**不复退转的一地以上圣者菩萨相续 |

这个转换并非贬低世俗三宝，而是区分 **暂时的应供处** 与 **究竟的皈依义**。课3云：

> "我们皈依的是最究竟的佛——法身佛，佛陀的法身和色身还是有一定差别的。所以在这里，弥勒菩萨宣讲了何为真正了义的佛。"

《金刚经》"若以色见我，以音声求我，是人行邪道，不得见如来"与本论立场直接呼应——**真正见佛，须见法身**。《大乘造像功德经》中佛陀告莲花色比丘尼"第一个见到我的是须菩提"——须菩提当时正在耆阇崛山石窟中"安住在如来不可思议、不生不灭的法身境界中"，即是此义（课3）。

## 二、佛宝本体——二利八功德

科判：丁一·戊一（究竟皈处佛宝本体）三（己一以赞叹真实宣说 / 己二分析彼义 / 己三彼等对应广说）。

### 赞叹门（己一）——六功德摄略赞颂

> **无初中末寂，佛自觉觉已，**
> **为未觉觉说，无畏恒常道，**
> **持胜智悲剑，金刚斩苦苗，**
> **见林绕疑墙，尽摧佛前礼。**

索达吉堪布（课3）指出，此颂从"能说的层面"赞叹佛陀六种功德：

1. **无初中末**——有为法的三种法相（生、住、灭）全然不具，故是无为法。
2. **寂**（寂灭）——身语戏论与心分别皆灭。
3. **自觉**——不依比喻、推理、善知识等他缘，以"各别自证"的自然本智觉悟。
4. **觉已**——现量通达一切万法实相，智慧圆满。
5. **为未觉觉说，无畏恒常道**——为未觉的众生宣说令得无畏、不退的恒常之道，悲心圆满。
6. **持胜智悲剑，金刚斩苦苗，见林绕疑墙，尽摧**——持智悲宝剑斩断轮回苦苗（苦因苦果），持智悲金刚摧破萨迦耶见、邪见、常见、断见的密林所围之疑墙，威力圆满。

索达吉堪布对"密林"譬喻的体会（课3）：藏地森林中放牦牛，走入深林常迷失方向、猛兽出没，恐惧无依。世间邪见对众生之遮蔽，亦复如是——唯以佛之智悲金刚方得破之。

### 分析门（己二）——八功德总摄教证

> **无为任运成，非依他缘证，**
> **具足智悲力，具二利佛陀。**

索达吉堪布（课3）：

> "这是经常引用的教证，如果不能将《宝性论》全部背下来，希望大家能够背诵这个教证。我在 1986 年刚学《宝性论》时，这也是我第一个背诵的教证。"

以**实体**分六功德 + 以**反体**分自利、他利 = **八功德**：

| 归类 | 功德 | 义 |
|---|---|---|
| 自利三（自证圆满） | 1. 无为 | 不依因缘而生，无有迁变（非与虚空、抉择灭等无为法相同，乃真如无为，与法性一味一体） |
| | 2. 任运成 | 无有勤作、造作；身语意戏论无余寂灭，法身本性，事业自然成办（对应颂中"寂") |
| | 3. 非依他缘证 | 各别自证，自然本智现前（对应颂中"自觉"） |
| 他利三（利他圆满） | 4. 智 | 世俗、胜义、世间、出世间通达无碍，如持庵摩罗果于手掌 |
| | 5. 悲 | 将未觉众生暂安各位，究竟安置佛果 |
| | 6. 力 | 以智悲断众生苦因（烦恼）与苦果（痛苦） |
| 反体二 | 7–8. 自利、他利圆满 | 八 = 六实体 + 自利/他利反体 |

### 对应门（己三）——逐一释义

朵洛瓦注释（戊一·己三）：

> **是无初中末，自性故无为，寂灭法身故，名谓任运成。**
> **各别自证故，非依他缘证，证此三故智，说道故为悲。**
> **威力以智悲，断除苦惑故，前三成自利，后三成他利。**

此颂逐一对接："**前者为因，后者为所安立的道理**"（课3）——

- 以 "无初中末" 为因 → 立 "无为" 之功德
- 以 "寂灭法身" 为因 → 立 "任运成" 之功德
- 以 "各别自证" 为因 → 立 "非他缘证" 之功德
- 以上三德 → 合立 "智慧圆满"
- 以 "说道" 为因 → 立 "悲心圆满"
- 以 "以智悲断苦惑" 为因 → 立 "威力圆满"
- **前三自利、后三他利**——"因为法界无为法任运自成成为自证智慧的行境，断除一切有漏法，具足无漏大乐，究竟自利……以智慧照见所化界，以大悲心趋入，以事业也能如愿无余成办所欲之事"（朵洛瓦注释）。

### 编辑注：法身佛宝与义他空的风险

"真正的佛身，即法身，是不生不灭的"——此立场本身没问题，但在觉囊派读解中易向 "法身 = 实有的不空法性" 滑动，成为义他空所承许的实有成实之法。宁玛立场需在此郑重加一笔：**法身即断证圆满、离戏大双运**——空性与光明无二，非离戏之外另立一个常恒实有之佛身本体。课3 的索达吉堪布讲法身时特别引入《入中论》的陶师比喻 ——"刚开始陶师需要费力勤作，后来因为有惯性，就可以'任运'而转了"（"如是佛住法性身，现前虽然无功用，由众生善与愿力，事业恒转不思议"）——这是将朵洛瓦的觉囊立场与自空中观连接的典型的宁玛重释动作，读者须留意。详见 `../../topics/tathagatagarbha/index.md §Editorial Policy` 与 correctness anchor "不空如来藏 = 离戏大双运"。

## 三、法宝本体——灭道二谛所摄之八功德

科判：丁一·戊二（彼之圣教法宝本体）四（己一以赞叹真实宣说 / 己二分析彼义 / 己三彼等归摄为二谛 / 己四彼等对应解说理由）。

### 赞叹门（己一）

> **非无有二俱，非俱无可择，**
> **远离诸释词，各别自证寂。**
> **无垢具智光，于诸所缘境，**
> **摧破贪嗔痴，正法日前礼。**

索达吉堪布（课4）分析此偈：

**前四句归灭谛所摄——三功德**：

1. **无思**——
   - "非有"（从觉囊立场，世俗客尘法无生，故不堕有边）"非无"（在各别自证前无欺存在，不堕无边）"非二俱""非俱"——**破四边**
   - "远离诸释词"——**非诠**，语言、名言、比喻、推理无可诠表
   - "各别自证"——**圣知**，圣者入定自证行境，不依他证
   - 合此三者 → "**无思**"（名言心识无可思维）
2. **寂无二**——寂灭轮回根本（业与烦恼），故无二
3. **寂无分别**——寂灭业与烦恼之因（非理作意），故无分别

**后四句归道谛所摄——三功德**：

4. **净**——远离烦恼障（包括随眠习气），本体清净
5. **明**（具智光）——远离所知障，智慧光明现量了知一切相
6. **对治**（摧破贪嗔痴）——于净/不净/中等三对境上，摧破贪、嗔、痴

### 分析门（己二）——六/八功德

> **无思二分别，净明对治品，**
> **所能离贪者，具二谛相法。**

朵洛瓦注释：

| 实体六功德 | 反体二功德 |
|---|---|
| 无思 / 无二 / 无分别 / 净 / 明 / 对治 | **所离贪者**（灭谛）+ **能离贪者**（道谛） |

合计八功德——"具二谛相法"即正法。

索达吉堪布（课4）要求背诵此颂："这个偈颂大家最好能背诵下来……大家要把这个关于法宝的偈颂背下来，这是本节课的要求。"

### 归二谛（己三）

> **远离贪欲者，灭谛道谛摄，**
> **彼等依次第，三三功德知。**

三皈依的通俗语："皈依佛，两足尊；皈依法，**离欲尊**；皈依僧，众中尊。"——法宝正是"远离贪欲者"。

朵洛瓦注释清晰地把结构理顺：

- 灭谛 = 所离贪之果 → **无思 + 无二 + 无分别** 三功德
- 道谛 = 能离贪之因 → **净 + 明 + 对治** 三功德
- 合 = 具二清净之二谛 → 八功德的法宝

### 对应理由（己四）

> **非择非诠故，圣知故无思，**
> **寂无二分别，净等三如日。**

朵洛瓦注释以 "**日轮喻**" 归摄道谛三功德：
- 日轮**清净**——喻道谛远离一切随烦恼之垢
- 日轮**照色**——喻道谛照亮一切所知相
- 日轮**破黑暗**——喻道谛为见真如一切障碍之对治

唯道谛配日喻而未配灭谛——索达吉堪布课4 引一段具有玩味意味的注解："也许弥勒菩萨可能会说：'道谛有变化，灭谛则一无所有，不用顶礼。'会不会如此大家可以慢慢思维分析。"——其背后的原因将在"三皈依非究竟"一节显现：灭谛（声闻灭谛）本身即是非究竟的无有之法，故不以日喻。

### 编辑注：《金刚经》与本论

"法尚应舍，何况非法"——此言教法如过河之舟，抵岸即舍。正法本体"无思、无二、无分别"——灭道二谛所摄——方为所皈依处。"很多人最多将正法理解成法本，而不能理解其甚深的法义与内涵"（课4）——此错认即相似法宝与胜义法宝的落差。

## 四、僧宝本体——如所有智 + 尽所有智

科判：丁一·戊三（受持佛法者僧宝本体）三（己一以赞叹真实宣说 / 己二证成彼等摄为二功德 / 己三彼等对应而广说）。

### 赞叹门（己一）

> **心性即光明，见惑无本性，**
> **真证诸众生，无我寂灭边。**
> **具见佛随行，一切无障慧，**
> **具见无量众，有境智前礼。**

索达吉堪布（课4–5）点明：此"僧"特指**一地以上至十地之间不退转菩萨**，非包括加行道、资粮道的凡夫菩萨，亦非指出家僧团。

**如所有智**（前四句）——三特点：

1. "心性即光明"——以对治自性现前光明（心性清净、光明）
2. "见惑无本性"——照见烦恼本体本来无生
3. "真证诸众生，无我寂灭边"——现量证悟一切众生心自性人无我、法无我，寂灭戏论法性

**尽所有智**（后四句）——三特点：

1. "具见佛随行"——以照见法性圆满佛陀之法身**周遍随行于一切众生**的智慧
2. "一切无障慧"——无有对境之障的出世间智慧
3. "具见无量众"——遍知一切众生之有境智慧

### 与一般大乘论典中的"二智"之别

索达吉堪布第5课特别点出一个重要差别：

> "其他大乘论典中所讲到的尽所有智，是指通达世俗中万事万物的显现；通达胜义中一切万法的空性，叫如所有智。但这里有点不同，**通达我和众生心的本来面目叫如所有智；而通达如来藏遍于所有三界众生，是尽所有智**。"

| 术语 | 一般大乘用法 | 《宝性论》用法（本处） |
|---|---|---|
| **如所有智** | 通达胜义空性 | 通达心性本来面目（空、净、光明、烦恼本尽） |
| **尽所有智** | 通达世俗万法显现 | 通达**如来藏周遍一切众生** |

此处的二智带有密宗智慧的意味——"因此，佛教不同经典中的某些名词，尽管字面上一模一样，但实际含义却可能大相径庭。比如，所谓的'自然本智'或'本智'，《俱舍论》中所说的智慧，与中观里所说的智慧，以及密宗大圆满中的'觉性本智'，这三者词句虽无大异，意义却未必相同"（课5）。

### 证成二功德（己二）

> **如所尽所有，内智见清净，**
> **具慧不退僧，具无上功德。**

朵洛瓦注释：以如所有智见清净 → 断**贪著障**（烦恼障）；以尽所有智见众生普具法界 → 断**质碍障**（所知障）。二智超胜声缘，故是"具无上功德"的不退转僧。

此偈也须背诵（课4 要求）。

### 对应广说（己三）——四科判

该科判下分四小段（庚一如所有 / 庚二尽所有 / 庚三清净 / 庚四最殊胜皈依处），第5课完成：

**庚一如所有智**：
> **证众寂法性，故为如所有，彼亦自性净，烦恼本尽故。**

"获得圣者果位的菩萨，具有现量见到法性的智慧——证悟包括自己在内的一切众生心的本性。就像《中论》当中所讲的一样——不生不灭、不来不去、不常不断、不一不异，这样远离四边八戏的本体"（课5）。

**庚二尽所有智**：
> **证所知究竟，慧见众生具，遍知之法性，故为尽所有。**

"以证悟对境一切所知法及其究竟真如的出世间智慧了知一切，而以各别自证智慧现量见到尽所有的一切众生自性普遍具有遍知的法性如来藏的缘故，是了知尽所有的有法之智。那种现见也是从一地起产生，因为证悟了法界普行（遍行）之义"（朵洛瓦）。

**庚三清净**：
> **如是证悟者，各别自智见，无垢界无著，无碍故彼净。**

二智所证，对境清净——贪著障与质碍障皆已断除。此清净"观待相似智慧所见也是极其清净"（朵洛瓦）。

**庚四一切中最殊胜皈依处**：
> **智见清净故，佛智无上故，不退之圣者，众生皈依处。**

"断"方面烦恼障、所知障断除；"证"方面佛智无上、与佛智相近；由此得行不退、位不退、念不退三不退转（课5）。

### 关于"植物有无如来藏"

课5 侧笔触及一个重要界定：

> "包括草木在内的具有'生命'的这些植物有没有如来藏呢？按照这种道理来推，应该不具足。从事物本身的角度来讲，一棵树、一根草、一块石头，将来是不可能成佛的。
>
> 当然，个别大乘经典中说，从一些觉悟者的自现，即从万法唯心造的理念来推，一切法的法性也可以称为如来藏，也可以称为现空无二的自性。从自性和本体方面来推，一切万法也具有如来藏的自性，这一点，在全知麦彭仁波切和无垢光尊者的个别论典中有所分析。"

此处索达吉堪布暗作了一次宁玛重释——将朵洛瓦紧守的 "有心识之众生本具" 立场，以麦彭、无垢光的 "万法皆有法性如来藏" 角度加以补遗，显出宁玛的显密贯通视角。

## 五、观待必要安立三皈依之理（丁二·戊一）

> **依佛法弟子，信解于三乘，**
> **及三作用众，安立三皈处。**

朵洛瓦与索达吉堪布（课5）共同指出：安立三皈依对应六种行人的意乐：

| 皈依 | 所信解 | 应机行人 | 成就 |
|---|---|---|---|
| 皈依佛 | 信佛为 **二足尊**（功德圆满） | 大乘菩萨（追求佛果） | 佛果 |
| 皈依法 | 信法为 **离贪尊** | 缘觉乘（自证十二缘起） | 独觉果 |
| 皈依僧 | 信僧为 **会众尊** | 声闻乘（随师学习） | 声闻圣果 |

朵洛瓦总结："以三种意义针对六种行人而安立三皈依处，这是为众生渐次趋入而宣说的。"即**三皈依是为利益不同根机而方便建立的一个次第安排**，并非本论立场上三者地位平等。

课5 中索达吉堪布引《四童子三昧经》"能归依佛者，彼人得大利，百千诸劫数，更不见诸苦"、《大智度论》"若人爱敬佛，亦当爱敬僧，不当有分别，同皆为宝故"等教证，强调在修行中对三宝应等恭敬。

## 六、法宝与僧宝非究竟皈依处（丁二·戊二）

> **舍欺有法故，无故有畏故，**
> **二法及圣众，非胜永皈处。**

此颂是本论最有震撼力的教判。朵洛瓦注释逐字解读：

| 原因 | 所指 | 析 |
|---|---|---|
| **舍** | **教法** | 教法是名称、词句、文字组成的义共相之所依，证道后如过河之舟所舍。《金刚经》"法尚应舍，何况非法" |
| **欺有法** | 证法·**道谛** | 一到十地菩萨相续中的功德是有为法，有生住灭，如浮云般无常、虚妄、欺惑 |
| **无** | 证法·**灭谛**（声闻式） | 按声闻方式所证的灭谛只是"灯火相续中断"的无有之法，仅是灭尽烦恼与痛苦的本性，非积极的佛果本身 |
| **有畏** | **僧宝**（阿罗汉、菩萨） | 阿罗汉虽灭粗烦恼，仍怕细微有为法（不可思议之投生、意生身）——"如同关监狱的人看到刽子手持剑的恐惧"；菩萨虽胜于阿罗汉，仍有所学，未达无畏极地 |

**结论**：教法（舍）、证法·道谛（欺）、证法·灭谛（无）、僧宝（畏）——"教法与证法二法，以及佛陀以外的其他圣者僧众是暂时的皈依处，在获得究竟时它们也不是最殊胜的永久皈依处"（朵洛瓦）。

## 七、抉择胜义皈依处——唯一佛宝（丁二·戊三）

> **胜义诸众生，皈处唯一佛，**
> **能仁法身故，僧究竟亦彼。**

索达吉堪布第5课开解："这里的'胜义'不是我们学中观的时候学的'胜义谛'和'世俗谛'，意思是，**究竟意义上的皈依处就是佛**。"

核心论式：
- **法宝之究竟 = 佛的法身** —— 法身具足二十一种无漏法，与法宝究竟实相无二无别
- **僧宝之究竟 = 佛的断证** —— 佛陀具最究竟的断证功德，三乘僧众之断证皆未究竟圆满
- 故 **唯一心一意皈依佛，即能具足皈依三宝的一切功德**

朵洛瓦注释：

> "在胜义中成为救度无量众生离苦、无灭恒常的殊胜皈依处，就是唯一的圆满佛陀。然而，法与僧说为暂时的皈依处是指，他们**究竟就是佛陀**……能仁具足清净二谛的法相，是究竟离贪之法身的缘故；三乘的僧众也以断证来分，究竟也是现前佛陀的法身。"

### 这意味着什么？

这一抉择不是在贬低法僧，而是在说明：**法僧之终极都融入佛宝**—— 法宝的最究竟即佛之法身，僧宝的最究竟即佛之断证圆满。**三宝在究竟层面归为一宝**。这为下一步的"**佛外无余涅槃**"（究竟唯一乘，详见菩提品）立下伏笔 ——既然一切圣者之究竟皆融入佛法身，则声缘无余涅槃亦非终点，终须回大乘成佛。

### 与"四因三果"的呼应

回忆 `./01-总说-七金刚处.md §六` 的 "四因三果" 架构：三宝是果，如来藏等四处是因。此处的"胜义皈依处唯一佛"与四因三果的果端相应——**佛宝即是菩提 + 功德的安立位**，而菩提即如来藏离客尘后之圆满现前。**故胜义皈依佛宝 = 皈依自心本具的如来藏之圆满显发态**。此见解是下一品"如来藏品" 的基础。

### 编辑注——显密皈依的呼应

此"胜义皈依处即佛法身"的立场，与 **`../大圆满前行/皈依.md`** 所讲的 "显密四层皈依" 直接相应——**显宗皈依**以佛法僧三宝为对境，**密宗外皈依**以上师 /本尊/空行为对境，而 **密宗内皈依**与**密密皈依**最终以 **自心之自性光明/如来藏/觉性本智** 为究竟皈依处。本论所说"胜义皈依处唯一佛（法身）"即是这一显密贯通的显宗教证基础。两文配套阅读，方能看到从《宝性论》的三宝本体立场 → 《前行》的四层皈依立场是同一立场在显密两个层次上的展开。

## 八、释词——"宝"的六种同法（丁三）

> **稀有无垢故，具力世庄严，**
> **是最殊胜故，无变故为宝。**

梵语"绕那"（Ratna）兼含"稀世之珍"与"珍宝"二义。此颂以 **珍宝之六同法** 释三宝之"宝"义（朵洛瓦与课5）：

| # | 宝性特征 | 三宝对应 |
|---|---|---|
| 1 | **稀有** | 未经累劫修行二资粮则不会值遇 |
| 2 | **无垢** | 何时何处都无有过患之垢 |
| 3 | **具力** | 具神通等不可思议的功德威力，遣除众生衰败、赐予圆满 |
| 4 | **世庄严** | 为一切众生善心之因，作为一切世间最殊胜庄严（圣者七财等） |
| 5 | **最殊胜** | 超越世间，于一切法中堪为最胜 |
| 6 | **无变** | 无为法自性，不以众生信不信等外缘改变（如黄金不因时代国度而失去价值） |

此外《大乘本生心地观经》宣说**十种功德**（坚牢/无垢/与乐/难遇/能破/威德/满愿/庄严/最妙/不变），《宝性论》此处六种为其摄略。

索达吉堪布课5末引《大庄严论经》："虽无诸珍宝，及以资生具，**能信三宝者，是名第一富**"——与其在 80 年代 "万元户" 的比喻，显得朴实温情：对三宝有真正信心者，即是世间第一富翁。此处从义理的 "释词宣说" 回到修行的信心劝勉，圆合了本品的整体节奏 ——三宝不仅是抽象教判，更是修行者实际的皈依处。

## 九、三宝本体所蕴含的修行涵义

本文的教判抉择对修行有三重核心涵义：

1. **皈依不是仅限于对相似三宝的信仰**——修行者须逐步认识到佛之法身之意味、法之灭道二谛之意味、僧之二智之意味，令皈依由形入义。
2. **胜义皈依处的唯一性指向自心**——既然佛之法身即如来藏圆满显发之态，且一切众生本具如来藏，则胜义皈依的最终落处即是认识自心本具之佛性。此与禅宗"即心即佛"（课5 引《景德传灯录》）、密宗"自心大圆满"皆同趣。
3. **三皈依之对机差别是方便安立**——读者应理解为度不同根机众生的渐次趋入，而非认为法僧地位低于佛。在修行次第上仍应等敬三宝，在义理上方见究竟归一。
