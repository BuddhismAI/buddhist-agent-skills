本 doc 覆盖事业品下半 (L38-L40):

- **己二·以比喻决定** 之 **庚二·辛四-辛九** (后六喻: 梵天/日轮/摩尼宝/回响/虚空/大地)

- **庚四·复次归纳宣说差别**

至庚四末, 事业品 (戊四) 整体完结。下一 doc (`13-利益品.md`) 将从 L41 起开始 "丙三·彼等功德" 最后一品——利益品 (戊五)。

## 〇、承前启后——从前三喻到后六喻

### 0.1 九喻总纲回顾表 (retrieval 核心)

本 doc 承 11 §1 "九喻总纲一表" 所立, 但从**身/语/意密与事业层面映射**的角度, 给出更深度对应表:

| # | 比喻 | 所表 | 身/语/意对应 | 核心义 | 本 doc 详讲 |
|---|---|---|---|---|---|
| 1 | 天王 | 示现 (身显) | 身 (色身显现) | 琉璃地现天王影, 无勤利他 | see doc 11 §3 |
| 2 | 天鼓 | 语宣说 | 语 (显音说法) | 天鼓自鸣四法印, 遍劝天人 | see doc 11 §4 |
| 3 | 雨云 | 意周遍 | 意 (智悲周遍) | 云遍虚空, 无勤降雨成熟庄稼 | see doc 11 §5 |
| 4 | **梵天** | **幻化 (身语)** | **化身** (应世幻现) | 梵天不动于一禅, 幻化于欲界六天 | ✓ §1 |
| 5 | **日轮** | **智慧散射** | **智身** (妙法光明) | 日无分别照耀, 莲开睡莲闭 | ✓ §2 |
| 6 | **摩尼宝** | **意密满愿** | **意密** | 摩尼宝无念满一切愿 | ✓ §3 |
| 7 | **回响** | **语密无字** | **语密** | 空谷回响非内非外 | ✓ §4 |
| 8 | **虚空** | **身密周遍** | **身密** | 虚空无相而遍容万物 | ✓ §5 |
| 9 | **大地** | **大悲趋入** | **悲心** | 大地无念普承万物 | ✓ §6 |

**关键结构性观察** (L40 §壬二"归纳宣说喻义"颂词):

> **示现语周遍, 幻化散智慧,**
> **意语身之密, 获得大悲性。**

此颂即九义次第总摄: 前三喻 (示现 / 语 / 周遍) + 中二喻 (幻化 / 智慧) + 后三喻对应**身语意密** (意 / 语 / 身) + 终喻 (大悲)。九喻既非并列罗列, 也非机械分配——而是从粗显 (身显幻化) 到微细 (三密) 再到终极遍依 (大悲地) 的次第展开。

### 0.2 九喻次第的 doctrinal 逻辑——"前后由断除违品法而说"

L40 庚四辛二颂:

> **此诸比喻摄, 义是此次第,**
> **前后由断除, 违品法而说。**

即: 九喻不是随机举例, 而是**每前一喻的不足由后一喻的特点来遮破**。同法 (相同处) 继续引用, 违品 (不足处) 由后喻超胜。如此九喻连成一环环相扣的教学链, 至大地喻到达顶端 (世出世一切善法所依), 再往后无法找到更超胜的比喻——故止于九。

此 "渐进超胜" 结构, 构成本 doc §8 "归纳差别" 的核心 doctrinal。

### 0.3 三密显密 anchor 预告 (§7)

后六喻中, **第 6/7/8 喻** (摩尼宝 / 回响 / 虚空) 直接对应佛之**意密 / 语密 / 身密**, 第 9 喻 (大地) 对应**大悲**。此 "三密 + 大悲" 结构在事业品的显宗语境内是一个关键落地, 与密宗 (尤大圆满) 所说 "三密" 名同而层异。

本 doc §7 设立独立 correctness anchor **"事业品三密: 显密同名异层"** ——承 09 §1.7 "三身显密差别" anchor 的进一步展开。

## 一、辛四·以梵天喻决定幻化之理 (L38)

科判: 戊四·己二·庚二·辛四分二——**壬一** 于具缘者前示现之理 (三癸) + **壬二** 于无缘者前不现之理。

### 1.1 情节核心

**梵天** = 色界第一禅大梵天王。他常住自己的无量宫 (一禅梵住) **不离本位**, 同时能**无勤幻化**出许多梵天身, 入欲界六天 (四天王 / 三十三天 / 夜摩 / 兜率 / 化乐 / 他化自在) 利益天人。

**核心义**: 身在本位**不动**, 同时幻化**遍入**所化界; 此 "不动而幻现" 即佛法身不动而化身遍利的本喻。

### 1.2 壬一·于具缘者前示现之理

#### 1.2.1 癸一·以无勤等同

**颂**:

> **梵天于梵住, 无有迁移中,**
> **于一切天境, 无勤示显现。**
> **佛于法身中, 不动于诸界,**
> **众具缘者前, 无勤示幻现。**

梵天在一禅无量宫不迁移他处, 同时**无勤幻化**于欲界六天。同样, 佛陀在**胜义法身**中永不动摇, 于无量世间界的具缘所化众生前**无勤示现**世俗色身种种幻变。

**《华严经》教证** (L38):

> "犹如大梵王, 普应现大千, 其身无别异, 诸佛法如是。"

大梵王无勤无分别于大千世界现种种身, 体性与自身无二无别。佛陀亦如是——法身不动, 化身在不同世间以不同形式救度众生, 诸化身与法身同体不二。

《华严经》亦广说佛于不同众生前种种示现: 有者见佛正在成道, 有者见佛发菩提心, 有者见佛修六度, 有者见佛已入涅槃, 有者见佛住世无量劫, 有者见佛一须臾灭度, 身相有高大矮小, 光大光小, 寿八十岁或无量岁——皆随顺众生缘起而现。

> "虽顺世间如是现, 体性清净同虚空。" (《华严经》)

随顺世间所现相不灭, 但体性如虚空清净无变——"千江有水千江月" 即此缘起之理。

#### 1.2.2 癸二·以除罪等同

**颂**:

> **梵天恒于宫, 不动入欲界,**
> **天人见见彼, 亦断喜对境。**
> **善逝法身中, 不动于世界,**
> **有缘见见彼, 恒行除诸垢。**

梵天不动住一禅宫, 同时于欲界六天显幻化身。欲界天人见梵天庄严身相、声音、妙香, 羡慕向往, 于是**断除对自身欲妙的贪著**, 追求上界 (色界) 的功德——此即 "以梵天之现而除欲界天人之罪"。

同样, 佛陀法身不动, 于清净菩萨前显**报身**, 于不清净凡夫前显相好庄严等**化身**。众生见佛庄严身相, 对佛陀庄严希求, 进而通过修行**断除相续中的烦恼障 / 所知障 / 习气障**。

**"见贤思齐"** (索达吉堪布 L38): 见贤者自思 "我亦当得", 是大乘修行的积极心态。《大宝积经》"众生见如是, 光明大神力, 多发无上心, 我等亦当得"——见佛光明神通而发无上菩提心。

#### 1.2.3 癸三·观待缘分等同

**颂**:

> **往昔自愿力, 天人善业力,**
> **梵天无勤现, 自生化身同。**

梵天之所以能现于欲界天人前, 依**双因缘**:
1. 梵天**自己往昔要利益天人的发愿力**
2. 欲界天人**积累见到梵天的共同善业力**

两因缘和合, 梵天无勤示现。

同样, 众生见到自生圆满正等觉的佛陀化身, 也依**双因缘**:
1. 佛陀**因地的发愿力**
2. 众生**自身善根力**

**"深奥奇妙的因缘法则"** (L38): 佛陀不同于世间 "以牙还牙" 的报偿逻辑, 不想 "他祈祷我, 我要去帮助他"——是依**无欺缘起力**无勤显现。人与人之间见面有缘无缘, 有者从未被害却看他不舒服, 有者从未被帮却见他生欢喜——皆是甚深因缘之显现。

**承 11 §6 "任运" 正确理解**: 此 "双因缘力" 即事业任运的缘起自显机制——非造物主单方赐予, 非众生单方感招, 而是佛宿愿 + 众生善根二缘和合之自然显现。

### 1.3 壬二·于无缘者前不现之理

**颂**:

> **天降入胎诞生住, 王宫喜享静处行,**
> **降魔证觉说寂道, 无缘眼前佛不现。**

此颂即佛陀**十二相成道** (承 09 §1.4.2) 的再列: 天降 / 入胎 / 诞生 / 王宫 / 喜享 / 静处 / 苦行 / 至菩提迦耶 / 降魔 / 证觉 / 说法 / 寂 (涅槃)。

佛陀以八相 / 十二相成道遍利众生, 但**无缘者见不到佛**。

**舍卫城三分教证** (L38, 引《大智度论》):

> 舍卫城九亿人——这里的亿是十万——只有三分之一见过佛陀, 三分之一听说过但没见过, 还有三分之一既没听过也没见过。

佛陀在舍卫城前后居住二十五年 (时间最长), 尚且三分之二的人未得见。古人云 "佛在世时我沉沦, 佛灭度后我出生"——无缘则不遇, 是因缘法则的如实展现。

**宁玛派重释** (宁玛派视角): "无缘不现" 非佛不慈悲, 非佛有偏私, 而是**缘起力的必然**——佛之加持是无欺显分, 但无信心 / 无善根者心中无容器承接 (呼应 11 §5.1 "清净水器方现日影")。索达吉堪布此处明示: 上师如意宝在世时色达一带也有人未见, 默默无闻的上师更不用说——"没缘分也见不到佛, 也不会祈祷"。

### 1.4 梵天喻核心 doctrinal 总摄

| 喻义 | 比喻侧面 | 佛法义 |
|---|---|---|
| 无勤示现 | 梵天一禅不动而欲界幻现 | 佛法身不动而化身遍现 |
| 除罪等同 | 见梵天断欲界贪 | 见佛身断二障习气 |
| 缘分观待 | 梵天愿力 + 天人业力 | 佛愿力 + 众生善根 |
| 无缘不现 | 无业力者不见梵天 | 无缘者不见佛 |

梵天喻的**核心贡献**: 承天王喻 (身显) / 云喻 (智悲周遍) 之后, 以 "**身语幻化**" 层面落地——强调化身在色界 (法身位) 不动而于欲界 (众生位) 遍幻的**同体不二**结构。

## 二、辛五·以日轮喻决定智慧散射之理 (L38-L39)

科判: 戊四·己二·庚二·辛五分四——**壬一** 不分别利害之理 + **壬二** 不分别利他之理 (三癸) + **壬三** 依缘分次第显现之理 (二癸) + **壬四** 较比喻更超胜之理 (二癸)。

### 2.1 情节核心

**日轮** = 太阳, 于虚空无分别而放射光芒, **照莲花令开, 照睡莲令闭**; 不分别利害, 不分别利他, 不分别应照何山——皆依缘次第显现。但太阳有**三大不足**:
1. 不能照无边刹土 (有局限)
2. 不能遣无明痴暗 (只遣色暗)
3. 不能永久恒显 (白天 / 晚上交替)

此三不足为佛智日**超胜**之处。

### 2.2 壬一·不分别利害之理

**颂**:

> **如日热一时, 莲开睡莲闭,**
> **开德与闭过, 无念圣日同。**

莲花又名 "日亲", 日光至则开; 睡莲是日之 "敌", 日光出则闭。日轮有无 "睡莲是坏蛋 / 莲花是好的" 的分别取舍? **绝对没有**。

同样, 佛陀展现智慧时: 提婆达多一直想取代佛, 六外道对佛反感, 目犍连 / 舍利子 / 文殊等极为欢喜——佛陀自身**有无 "对我好的要帮助, 对我不好的要远离" 的分别念**? 没有。

**L38 佛陀应对五千人退席** (教证): 佛讲《法华经》, 开讲时五千人退席, 佛陀仍说 "善哉善哉"——无分别利害心的示现。一般法师遇此会 "气坏, 脸红, 动来动去"——佛陀的无分别是究竟证量。

**承 11 §6.4 "祈祷不必节约"**: 佛不因某人祈祷多而偏爱, 不因某人诽谤多而疏远——无分别利害即无分别利他的前提。

### 2.3 壬二·不分别利他之理 (三癸)

#### 2.3.1 癸一·无分别利他之理

**颂**:

> **如日无分别, 顿时放自光,**
> **令莲花绽放, 令余得成熟。**
> **如是如来日, 妙法之光芒,**
> **于所化众莲, 无有分别入。**

太阳无分别地顿时放光, 令莲花绽放、庄稼成熟。同样, 如来日放射**三乘 / 十二部经 / 三藏**的妙法光芒, 令有缘所化众生的莲花自然绽放。

**猫头鹰教证** (L38): 太阳照耀时, 大多数众生欢喜, 但猫头鹰眼睛受不了, 白天睡觉夜间出没——就像佛陀正法对某些众生无法接受, 非日轮错, 非佛陀错, 是众生**无缘无业**。

《华严经》: "譬如日月游虚空, 照临一切不分别; 世尊周行于法界, 教化众生无动念。"

#### 2.3.2 癸二·放光利他之理

**颂**:

> **法身与报身, 菩提空中升,**
> **遍知日智慧, 放光于众生。**

日轮光能驱散世间黑暗。同样, 佛陀法身 + 色身 (报) 远离戏论, 于现前菩提果的虚空中升起**遍知日智慧**, 放射妙法光芒, 驱散众生无明黑暗。

#### 2.3.3 癸三·顿时显现利他之理

**颂**:

> **因于诸清净, 所化水器中,**
> **善逝之日影, 顿时现无量。**

众生相续中烦恼垢染清净的水器里, 佛陀色身如日影同时顿显无量尊。若水器浊垢则不现。

**《华严经》教证**:

> "心器常净, 常见佛身, 若心浊器破, 则不得见。"
> "随顺众生故, 普入诸世间。"

**实修应用** (L38 索达吉堪布): 不必 "佛陀辛苦, 今天我不祈祷让他休息"——佛陀无量世间同时入, 只要祈祷 + 心清净, 加持的影像定现于心。对佛反感 / 有邪见者, 加持无法融入; 对上师挑毛病 / 生嗔恨者, 传承加持难入相续。

### 2.4 壬三·依缘分次第显现之理

**颂 (略说)**:

> **恒时遍一切, 法界虚空中,**
> **佛日应机落, 所化之山上。**

**颂 (广说)**:

> **具千光日升, 明世间次第,**
> **落高中低山, 佛日渐照众。**

太阳次第照高山 → 中山 → 低山。同样, 佛陀加持次第显现: 先照缘分好的, 再照中等的, 最后照下根的。

**"法本无主人"** (L39): 佛陀对提婆达多和罗睺罗完全同等看待, 没有 "文殊对我好我传他最深法, 提婆达多是坏蛋我报复他不传" 的分别。法是没有主人的, 看谁有缘分。法王如意宝传法时同一时间同一场合同一群人同一法, 但弟子得到的不同——有得甘露, 有得清水, 有得脏水。

### 2.5 壬四·较比喻更超胜之理

#### 2.5.1 癸一·以光超胜之理

**颂**:

> **日无照射天边刹, 非除痴暗显所知,**
> **悲尊散放多彩光, 于众明示所知义。**

**太阳三不足**:
1. 不能照无边刹土
2. 不能遣无明痴暗
3. 不能显示所知实相

**佛智光超胜**:
1. 周遍虚空际一切刹土
2. 能遣无明痴暗
3. 能显示所知如所有 / 尽所有义

《佛本行经》: "日明照昼不及夜, 不曜天上三恶趣, 佛光昼夜三千世, 及至一切众生心。"太阳只照白天, 不能照晚上 / 天上 / 恶趣; 佛智光昼夜照三千大千, 直入众生心。

#### 2.5.2 癸二·以作用超胜之理

**颂**:

> **佛至城中时, 诸盲人复明,**
> **无义众远离, 见佛得受益。**
> **堕落三有海, 见暗覆痴盲,**
> **依佛之日光, 得见未见处。**

太阳令众生见色法, 但不能遣轮回苦 / 通达空性如来藏。佛陀**超胜日轮之功德**:
- 至城时诸盲人复明
- 聋人闻声
- 疯狂者恢复正常
- 贫穷者家现宝藏
- 丑陋者变美丽
- 三恶趣苦得遣
- 见诸恶见覆痴盲者得见前所未见的**法界甚深真如**

### 2.6 日轮喻核心 doctrinal 总摄

日轮喻的核心贡献: 承 11 云喻 (智悲周遍) 之后, 以 "**智慧散射**" 层面落地——不分别利害 / 利他, 不分别应机, 而能遍照法界, 超胜世间日轮三不足。此喻是事业品内 "智身放光" 最详展开的一喻 (4 壬 + 6 癸), 显明佛智非世间因光可比拟。

## 三、辛六·以摩尼宝喻决定意密 (L39)

科判: 戊四·己二·庚二·辛六分二——**壬一** 以满愿说等同 (二癸) + **壬二** 以难得说等同。

### 3.1 情节核心

**摩尼宝** (如意宝 / Cintāmaṇi) = 龙王最主要的宝贝, 在海底大地之下。两大特性:
1. **无分别而满愿**: 祈祷者想要衣食 / 药物 / 处所等, 摩尼宝无念顿时各不混杂满足一切
2. **难得**: 福报浅薄者得不到

此喻**专表佛之意密**——意 (心) 无分别而遍知一切所化心愿。

### 3.2 壬一·以满愿说等同

#### 3.2.1 癸一·以无分别满愿之理广说

**颂**:

> **犹如摩尼宝, 无念顿时间,**
> **能满住境者, 各自诸心愿。**
> **依佛如意宝, 不同意乐者,**
> **得闻种种法, 彼不分别彼。**

如意宝无分别顿时满不同众生的不同心愿。佛陀如意宝亦如是——三种姓 (声闻 / 缘觉 / 菩萨) 等信解不同意乐的所化, 随各自缘分听闻三乘等种种法, 而佛陀丝毫无 "要为某众生说此法" 的分别心。

**《如来庄严智慧光明入一切佛境界经》教证** (L39 脚注 1):

> 诸佛如来无有如是分别之心。此众生信上法, 此众生信中法, 此众生信下法……诸佛如来智日光轮, 远离一切分别异分别及诸戏论。依诸众生种种善根, 诸佛如来智日光轮种种别异。

即: 佛智日光本体无分别, 而随众生**善根差异**自然别异呈现——此即 "意密" 的缘起义。

#### 3.2.2 癸二·以无勤恒常安住之理略说

**颂**:

> **如摩尼宝无念勤, 所求财物赐他众,**
> **能仁无勤而应机, 利他恒常住世间。**

摩尼宝无念无勤赐祈祷者所需。大能仁释迦牟尼佛亦无勤应机, **恒常住三界轮回中不断利益众生**——此即承 11 §2.10 "有际不间断" + 09 §2.2 "色身恒常 = 不间断" 在意密层面的再度落地。

**《入中论》教证** (L39):

> "此寂灭身无分别, 如如意树摩尼珠, 众生未空常利世, 离戏论者始能见。"

佛寂灭身无分别却能利他; **众生界未空之前, 利他事业不间断**——此即 topic anchor "佛外无涅槃" 在事业意密层面的第三度展开 (承 05 §6 + 11 §2.10)。

**《华严经》教证**: "譬如如意宝, 能满一切欲, 最胜亦复然, 满诸清净愿。"

### 3.3 壬二·以难得说等同

**颂**:

> **海地下摩尼, 欲者极难得,**
> **劣缘惑困众, 此心难见佛。**

如意宝处大海底 + 大地下, 福薄者不得。同样, 住轮回的众生若**缘分下劣 + 烦恼随烦恼所困 + 业力深重**, 则有垢染之心难见善逝二身。

**《华严经》教证**: "无信众生不见佛。" / "善逝不生念, 我利益众生, 其怀恶心者, 不睹如来身。"

佛陀虽无 "要利益众生" 的分别念, 实际能利益众生; 佛陀虽无 "害对他怀恶心者" 的想法, 起恶心者却根本见不到如来身。

**承 1.3 "无缘不现"**: 梵天喻已立 "无缘不现", 摩尼宝喻以 "难得" 再度强化此立场——意密之难显, 不在佛有偏堕, 而在众生业缘深重。

### 3.4 摩尼宝喻核心 doctrinal 总摄

| 喻义 | 比喻侧面 | 佛法义 |
|---|---|---|
| 无分别满愿 | 摩尼宝无念满一切求 | 佛意无分别而应机说三乘 |
| 无勤恒常 | 摩尼宝常安住而赐 | 佛恒常住世乃至轮回未空 |
| 难得难遇 | 摩尼宝在海底大地下 | 佛对劣缘众生难遇难得 |

摩尼宝喻核心贡献: 首次在九喻中明示 "**意密**" 层面——佛之心 (意) 无分别却能周知 / 遍应一切众生意乐, 是 "**随缘别异而本体无别**" 的神妙机制。

## 四、辛七·以回响喻决定语密 (L39-L40)

科判: 戊四·己二·庚二·辛七分二——**壬一** 比喻 + **壬二** 意义。

### 4.1 情节核心

**回响** = 空谷回声, 在山间大吼 "啊——", 整个山间给你传出 "啊——" 声音。**三大特性**:
1. 声音在**听者的觉知**中显现, 非自住
2. 岩山无分别 / 无造作 / 无勤作
3. 声音**非内非外**——本体不可得, 不住于外岩山, 不住于内发声处, 不住中间

此喻专表佛之**语密**——语无自性而能说法。

### 4.2 壬一·比喻

**颂**:

> **犹如回响声, 他者觉知现,**
> **无分别造作, 不住外与内。**

空谷声依种种发声之缘, 其他听者**随各自所喊**从觉知中产生显现。岩山等**无动机分别念, 无造作或勤作**。声音自本体**既不住岩山外面, 也不住里面, 哪里都不可得**——只是发声者声音的反弹。

### 4.3 壬二·意义

**颂**:

> **如是如来语, 他者觉知现,**
> **无分别造作, 不住外与内。**

佛陀说法时, 声音在**听众的觉知**中现前, 但佛陀根本没有想 "我要说什么"。表面看佛陀如人, 通过喉间 / 声带发声, 实际佛陀所说妙法是**无生无灭**的——这样的声音在佛身里里外外 / 上上下下**少许也得不到**。

**《如来不思议秘密大乘经》教证** (L39):

> "譬如应声而对响, 音响非内亦非外, 人中尊出妙言语, 非内非外亦如是。"

**《华严经》教证**: "非有方所, 非无方所。"

缘起法就是这样——**不是真正有方所, 也不是没有方所**。声音在听者心中显现, 本体不立, 但相显不灭。

### 4.4 回响喻核心 doctrinal 总摄

| 喻义 | 比喻侧面 | 佛法义 |
|---|---|---|
| 他者觉知现 | 声音在听者心中显 | 佛语在众生信心觉知中显 |
| 无分别造作 | 岩山无勤无念 | 佛语无说者念想 |
| 不住外与内 | 声音本体不可得 | 佛语无生无灭, 非内非外 |

回响喻核心贡献: "**语密**" 的缘起性安立——佛语无实说者 (无 "我要说什么" 的分别), 无实说相 (非内非外), 但**依听者缘而觉知显现**。这是中观应成派 "缘起无自性" 原则在佛语层面的直接应用。

**与 11 天鼓喻对比**: 天鼓喻 (显音说法) 侧重 "**无勤自鸣四法印**"; 回响喻 (语密) 侧重 "**声本体不可得而遍一切觉知**"。前者是 "语的运作方式", 后者是 "语的本体层面" ——前显后密, 互为表里。

## 五、辛八·以虚空喻决定身密 (L40)

科判: 戊四·己二·庚二·辛八分二——**壬一** 比喻 + **壬二** 意义。

### 5.1 情节核心

**虚空** = 无为法, **无相无作而遍容万物**。六大特性:
1. **少无** (无少许实法)
2. **无显现** (根识前无显)
3. **无缘** (非所缘对境)
4. **无所依** (无能依对境)
5. **超眼境无色** (非色法)
6. **无示** (无法指示)

虚空虽有时看见 "高/低/圆形/蓝色/黄色", 但**胜义中并非真实存在**。

此喻专表佛之**身密**——身遍而无形。

### 5.2 壬一·比喻

**颂**:

> **少无无显现, 无缘无所依,**
> **超眼境无色, 无示之虚空,**
> **虽见高与低, 然彼非如是。**

虚空的法相: 不成立实法 / 根识无显 / 不成所缘 / 无能依基础 / 超眼根境 / 无碍色法 / 无可显示——是**无为法**。见高低圆方等种种, 皆非真实。

**色达吹牛者教证** (L40 有趣公案): 原色达有人骗人说 "我骑马去摸虚空, 摸不着, 拿棍子敲到, 咚咚咚"——实际不可能敲到虚空, 因虚空无实。

### 5.3 壬二·意义

**颂**:

> **如是虽见佛, 然彼非如是。**

虽然所化众生见佛**入胎 / 诞生 / 涅槃 / 散步 / 站立 / 种种身色 / 不同刹土**等无量幻变, 但**胜义的佛陀并非像显现生灭等种种那样真实存在**——因为他是**无变的**。

**此颂是本 doc 最深的 doctrinal 落地之一**——承 09 §2.2 "色身恒常 = 不间断" + 11 §2.10 "有际不间断" + 本 doc §8.1 "色身生灭而法身无生灭" 的一贯立场, 以虚空喻的**身密**层面完成最深度辨析:

- 佛身**见为有生灭** (随众生信心缘起显现)
- 佛身**本体无生灭** (法身无变)

此即**显宗三身中"法身无相"与密宗三身中"身密虚空性"**的共通处——见于 §7 显密 anchor。

### 5.4 虚空喻核心 doctrinal 总摄

| 喻义 | 比喻侧面 | 佛法义 |
|---|---|---|
| 少无无显 | 虚空不成实法 | 佛身胜义非色 |
| 无缘无依 | 虚空非所缘 | 佛身非能所对境 |
| 超眼境 | 虚空非色可见 | 佛身非粗色法 |
| 虽见非如是 | 显现非真实 | 示现生灭而法身无变 |

虚空喻核心贡献: "**身密**" 的无相 / 周遍 / 无变同时安立——佛身虽示现八相 / 十二相, 本体却是如虚空般的无为法, 恒常不变。

## 六、辛九·以大地喻决定大悲趋入之理 (L40)

科判: 戊四·己二·庚二·辛九分二——**壬一** 比喻 + **壬二** 意义。

### 6.1 情节核心

**大地** = 万物所依。三大特性:
1. **能生一切** (草木林园鲜花皆依大地生)
2. **无念无勤** (无 "这是怨 / 亲" 之分别)
3. **增长 / 稳固 / 广大** (前所未有令其增长, 已有令其稳固, 一切越来越广大)

此喻专表**佛大悲**——无分别念而周遍作一切众生善法所依。

**九喻以大地终**: 九喻次第以大地为顶峰, 再往后无法找到更超胜的比喻 (参 §8.2)。

### 6.2 壬一·比喻

**颂**:

> **如地生一切, 依于无念地,**
> **增长稳固广。**

大地生草木林园鲜花等一切, 依于无念无勤作之大地, **前所未有的令其增长, 已有的令其根深蒂固, 一切越来越广大**, 作为众生的**生存之因**。

**《华严经》教证** (L39):

> "譬如大地一, 能生种种芽, 于彼无怨亲, 佛福田亦然。"

大地能生各种苗芽, 但无 "这是怨敌 / 这是亲人" 的想法。佛陀福田亦如是——能出生种种善根庄稼, 但佛陀本身无此分别。

### 6.3 壬二·意义

**颂**:

> **依无分别念, 圆满佛大地,**
> **众生诸善根, 无余得增长。**

无分别念的圆满正等觉佛陀就像大地, 依靠他能增长众生的**菩提心善根 / 信心善根 / 智慧善根**等等。

**承 1.2.3 "双因缘" 立场**: 要依靠 "众生善根 + 佛陀发愿", 因缘具足才增长——大地喻再度呼应梵天喻 "缘分观待" 之理。

### 6.4 大地喻核心 doctrinal 总摄

| 喻义 | 比喻侧面 | 佛法义 |
|---|---|---|
| 能生一切 | 大地生草木鲜花 | 佛出生众生一切善根 |
| 无念无勤 | 大地无怨亲分别 | 佛无所化分别 |
| 增长稳固广大 | 万物依地渐广 | 善根依佛渐稳广 |
| 一切所依 | 大地是一切生存因 | 佛是世出世一切善法之基 |

大地喻核心贡献: "**大悲趋入**" 的总摄——九喻之终喻, 不再只是 "身 / 语 / 意" 某一侧面, 而是**作为一切善法根本依的大悲本性**。大地承载万物 = 佛大悲遍承一切众生, 这是事业品最终、最遍的喻义。

## 七、【核心 correctness anchor】事业品三密: 显密同名异层

### 7.1 安立缘起

事业品后六喻中, 第 6 / 7 / 8 喻 (摩尼宝 / 回响 / 虚空) 明确标举佛之**意密 / 语密 / 身密**, 第 9 喻 (大地) 标举**大悲**。此 "三密 + 大悲" 结构, 是显宗《宝性论》中对 "佛三业" (身 / 语 / 意) 的 "密" 化安立——指其**无分别无勤作而遍利** 的微妙运作。

但 "三密" 也是密宗 (尤大圆满) 的核心术语——**本尊身密 / 真言语密 / 禅定意密**。名同而层异, 一旦粗糙等同, 即会误解两者。此 anchor 承 09 §1.7 "三身显密安立的层次差别" 的结构性处理, 在事业品 "三密" 层面作第二度 correctness 落地。

### 7.2 显密三密对照表

| 层次 | 显宗 (《宝性论》事业品) | 密宗 (大圆满 / 一般密宗) |
|---|---|---|
| 身密 | 佛身如虚空, 遍而无相, 示现种种而本体无变 (§5) | 行者观本尊身 = 佛身, 身密 = 本尊坛城瑜伽 |
| 语密 | 佛语如回响, 无实说者 / 无实说相, 依听者觉知而显 (§4) | 行者念持真言咒语, 语密 = 真言持诵瑜伽 |
| 意密 | 佛意如摩尼宝, 无分别而满愿, 遍知一切所化心愿 (§3) | 行者入三摩地与佛意相应, 意密 = 禅定 / 生圆次第 |
| 大悲 | 佛悲如大地, 无念而遍承一切众生善根 (§6) | 菩提心 / 大悲空性无二; 修行中大悲摄持一切 |

### 7.3 同名异层的 doctrinal 根据

**显宗《宝性论》三密的立足点**: 佛**果位功德**的运作方式
- 佛身 / 语 / 意**本已圆满无勤作 / 无分别**
- 众生见闻觉知以缘起力自然受益
- 重点是**佛作为说明对象** (佛是什么样的 / 佛如何起用)

**密宗三密的立足点**: 行者**因位修行**与佛果三密相应
- 行者以 "我即本尊 / 我语即咒 / 我心即佛意" 的三密瑜伽修持
- 通过与佛三密相应而**迅速现前佛果三密**
- 重点是**行者作为修行主体** (我如何修与佛相应)

### 7.4 为何不可粗糙等同

若将显密三密等同, 会落入以下**两种过失**:

1. **过密过失**: 将显宗《宝性论》所说 "佛三密" 直接理解为密宗意义的身密本尊 / 语密咒语 / 意密生圆次第——违《宝性论》原文, 原文只说 "佛身如虚空 / 佛语如回响 / 佛意如摩尼宝", 并未立 "咒语 / 本尊 / 三摩地" 的修行次第。
2. **过显过失**: 将密宗三密等同于《宝性论》事业品三密——把本尊身密理解为 "佛身如虚空般遍一切", 把真言语密理解为 "佛语如回响无实", 把禅定意密理解为 "佛意如摩尼宝满愿"——违密宗续部的明示立场, 密宗三密明确是行者因位修持法门而非佛果位运作方式。

### 7.5 贯通而不等同——《宝性论》作为显密津梁

承 09 §1.7 anchor 立场, 《宝性论》事业品 "三密" 不是与密宗三密**隔绝**的——二者有**结构性的贯通点**:

- 显宗 "佛身如虚空" ↔ 密宗 "行者身本具佛身体性" (即身成佛基础)
- 显宗 "佛语如回响" ↔ 密宗 "行者语本具佛语体性" (持咒相应原理)
- 显宗 "佛意如摩尼宝" ↔ 密宗 "行者意本具佛意体性" (禅定相应原理)

显宗立的是**佛果位三密的本来**, 密宗立的是**行者因位三密与佛果相应的修道**——**同一三密体性, 因果两侧立言**。

法王如意宝 / 麦彭仁波切等大德共许此立场: 《宝性论》特别深奥, 需受密法灌顶方可听闻 (参 01 §总说品 §"《宝性论》在密续中的地位")——因其所讲的 "三密事业" 本身即是密续所许 "佛三密体性" 的显宗开演。

## 八、庚三·以说必要摄义 (L40)

科判: 戊四·己二·庚三分二——**辛一** 必要及喻义之分类 (四壬) + **辛二** 解说以比喻表示之理殊胜 (二壬)。

### 8.1 辛一·必要及喻义之分类

#### 8.1.1 壬一·成办必要及最极必要之理

**颂**:

> **未见无勤作, 而行何事故,**
> **为断所化疑, 宣说九比喻。**

普通人的心境何时也不曾见过 "丝毫无勤作任运自成而行某事", 因此所化心中会生 "佛事业无勤而利生相违" 的怀疑。为**斩断此疑**, 才以九种比喻明显宣说 "无勤也能行事之理"。

**《大智度论》教证** (L39):

> "有福德之人, 安坐无事, 福禄自至; 薄福之人, 勤设方便, 斗战乃得。"

有福德者 "福德自至"——不勤作即自然获得; 薄福者 "精进而得"——勤奋才能获得。佛陀因地精进, 果地任运——此即 "福德自至" 的究竟相。

**颂**:

> **此等九比喻, 宣说详解说,**
> **是于经藏中, 宣说彼必要。**

九比喻在**《入诸佛境界智光明庄严经》** (《趋入佛境智光庄严经》/《如来庄严智慧光明入一切佛境界经》, 不同译本) 中详细宣说。

**两层必要**:

| 层次 | 必要 |
|---|---|
| **暂时必要** (近必要) | 遣除众生 "佛事业无勤相违" 的怀疑 |
| **最极必要** (必要之必要) | 让众生很快趋入佛陀无勤作的境界 |

**颂**:

> **由闻所生慧, 广大光庄严,**
> **具慧速趋入, 佛陀诸行境。**

由听闻此类经及注释所生的智慧清净广大光明庄严, 具慧菩萨以此能迅速趋入, 最初以分别方式趋入佛陀一切行境的无勤任运事业, 最终以**行持方式**趋入。

#### 8.1.2 壬二·归纳宣说喻义

**颂**:

> **彼义是以说, 琉璃帝释影,**
> **彼等九比喻, 认清彼摄义。**
> **示现语周遍, 幻化散智慧,**
> **意语身之密, 获得大悲性。**

此颂即**九喻所表九义**的总摄次第:

| # | 比喻 | 摄略义 |
|---|---|---|
| 1 | 帝释 (琉璃影) | 示现——身显 (随顺所化示现种种身幻) |
| 2 | 天鼓 | 语——随缘以语言宣说妙法教言 |
| 3 | 云 | 周遍——心的智悲周遍一切 |
| 4 | 梵天 | 幻化——身语幻化无量事业 |
| 5 | 日轮 | 散智慧——意事业依智慧放妙法光芒 |
| 6 | 摩尼宝 | 意——无分别念也能赐所需的意密 |
| 7 | 回响 | 语——无文字也能善说的语密 |
| 8 | 虚空 | 身之密——无质碍也为所化见到的身密 |
| 9 | 大地 | 获得大悲性——大悲作为一切功德之依处 |

**结构性观察**: 此九义次第**不是随机排列**:
- 1-3: 粗显三业 (身 / 语 / 意周遍)
- 4-5: 化身与智身 (显现的层面)
- 6-8: **微细三密** (意 / 语 / 身密)
- 9: 终极大悲所依

粗到细, 显到密, 至大悲所依而止——"前后由断除违品法而说" 的完整结构 (参 §9)。

#### 8.1.3 壬三·对应三义解说

以三相推理方式证成佛利生之无勤无分别合理。

**颂**:

> **灭诸勤作续, 无有分别意,**
> **如无垢琉璃, 现帝释影等。**
> **灭勤是立宗, 无分别意因,**
> **为自性义成, 喻帝释影等。**

**因明论式**:

| 要素 | 内容 |
|---|---|
| **有法** (所诤事) | 佛陀 |
| **立宗** (所立) | 远离一切勤作 (灭勤) |
| **因** | 无分别意故 |
| **比喻** (同喻) | 如无垢琉璃地现帝释影 |

**三相推理**:
- **宗法**: 佛陀无分别 (已成)
- **同品遍**: 凡无分别者皆无勤作 (如琉璃地 / 天鼓 / 云 / 大地 / 虚空 / 回响等九喻皆成立)
- **异品遍**: 凡有勤作者皆有分别 (反面——无法找到 "有勤作而无分别" 之事例)

**索达吉堪布 L40 论式应用**: 每一个比喻都可作论式喻——"佛陀灭一切勤作, 无分别故, 如同天鼓 / 云 / 大地 / 虚空 / 空谷声……" 每一喻皆成立, 反面有实宗举不出 "一切万法有自性之因及共许比喻" 的对立论式——此即因明层面九喻成立之根据。

#### 8.1.4 壬四·本义无勤作之理

**颂**:

> **此是此本义, 示现等九种,**
> **本师离生死, 无勤尽趋入。**

示现神变 / 宣说教言等九种意义所表达的本义: 本师圆满佛陀远离生死等有为法, 无分别念, 无思 "此是此" 的勤作, 但以成办事业的方式**完全趋入**示现在所化众生前。

**索达吉堪布 L40 实修应用**:

> "这是个什么人? 这是男众, 这是女众, 这是摄像机, 这是茶杯……" 心里没有这样想, 行为上也不会偏向, 所以佛陀远离是此是彼的分别, 不会有这样的想法: "今天某个人对我祈祷, 特别好, 我要好好加持他。"

佛以无勤任运自成的方式趋入有缘众生界, 照样度化众生。我们的障碍是自己的负面行为自然带来的缘起——如种毒药长毒花, 种妙药长妙果。

### 8.2 辛二·解说以比喻表示之理殊胜

#### 8.2.1 壬一·宣说比喻分类

**颂**:

> **犹如帝释天鼓云, 梵天日轮摩尼珠,**
> **回响虚空地有际, 无勤利他瑜伽知。**

九喻重述完成——"乃至轮回未空之间, 佛陀在无有勤作的情况下可以利益众生, 这样的甚深缘起, 唯有佛陀或得地圣者才能了知, 其中微妙殊胜的道理, 一般世间人很难想象。"

此颂重申 topic anchor "**佛外无涅槃**" (轮回未空之间事业不间断)——承 05 §6 + 11 §2.10 + 本 doc §3.2.2 的第四度教证。

**"瑜伽知"**: 九喻之理**唯佛与得地圣者**能通达, 一般世间人难以想象。十地菩萨对有些道理亦比较模糊——"佛陀不断地为我们宣说正法, 后学者也不断学习, 智慧逐渐增长, 进入佛的境界后, 对这些功德才有话语权。"

#### 8.2.2 壬二·对应意义解说

圆满佛陀利他, 九种意义 (复诵式展开):

**颂 1**:

> **示现如宝现天王, 善妙传授如天鼓,**
> **遍主智慧大悲云, 遍及有顶无量众。**

- 1 身显现如宝地现天王——法身不动而色身幻变无量
- 2 善妙传授如天鼓——语善巧传讲教授 (入定 / 解脱)
- 3 遍主智慧大悲云——意周遍一切所知的尊主, 了知如所有尽所有, 悲心周遍, 趋入直至有顶无量众生

**颂 2**:

> **如梵无漏处不动, 示现众多幻化相,**
> **如日智慧放射光, 如净摩尼宝珠意。**

- 4 如梵不动现幻化——无漏法界不动而化身语相应众生信解
- 5 如日智慧放光——智慧放妙法无量光
- 6 如净摩尼宝珠意——无分别而赐所需的意密

**颂 3**:

> **佛语无字如回响, 身如空遍无色常,**
> **如地众生善法药, 一切之基是佛地。**

- 7 佛语无字如回响——众生能听种种而不成立文字本体
- 8 身如空遍无色常——身密遍一切而不成立色法, 是恒常性
- 9 如地众生善法药——众生善法庄稼 / 滋补药无余增长稳固广大的基础, **即是获得具足无余转依之大悲的佛地**

**"身如空遍无色常" 再度落地**: 此句是本 doc 对 09 §2.2 "色身恒常义辨析" + 05 §4.2 佛位四德 + 11 §2.10 有际不间断的**事业品内最终综合展开**——身密如虚空遍一切, 不成立色法, 是**恒常性** (非刹那不变, 而是**不间断**)。

## 九、庚四·复次归纳宣说差别 (L40)

科判: 戊四·己二·庚四分二——**辛一** 色身虽现生灭然法身无生灭之理 (二壬) + **辛二** 以同法虽显现等同然以不同超胜之理 (二壬)。

### 9.1 辛一·色身虽现生灭然法身无生灭之理

#### 9.1.1 壬一·以天王比喻无生灭之理

**疑问**: 如果佛陀恒常无生无灭, 那我们经常在佛经中见到佛陀降生、最后示现涅槃, 这不是生灭吗?

**颂**:

> **如净琉璃心, 现见佛陀因,**
> **不为不信退, 信根得增长。**
> **善根生灭故, 佛陀色生灭,**
> **能仁如帝释, 法身无生灭。**

清净琉璃地上可以显现帝释天之影像。同样, 众生**心如清净琉璃**时能见佛陀; 见佛陀的因是**心清净**, 不被不信等违品退转, 信根得增长。

**众生善根生灭故, 佛陀色身显现生灭**; 但如帝释天自己于三十三天不动摇一样, **能仁遍知于自本体胜义法身中无有改变, 是无生无灭**。

**实修应用** (L40 索达吉堪布):

> 上师在世时, 弟子信心也有; 一旦上师们离开了, 那时说明弟子的信心可能退了, 上师在弟子面前显现的因缘没有了。

善知识是否住世, 包括佛陀生灭, **跟众生福德和信心有关**——如果具足不退信心, 他们会显现; 一旦因缘遮障, 高僧大德会 "沉没于涅槃西山"。

**此段是本 doc 最深度呼应 09 §2 "二身恒常" + 05 §5-§6 "二转三转法身差别 / 佛外无涅槃" 的收束**:
- 色身显现生灭 = 众生善根缘起 (世俗 / 显现)
- 法身无生灭 = 佛本体 (胜义 / 本体)
- 二者以 "琉璃地 + 天王" 之喻完整统摄

#### 9.1.2 壬二·一切不间断之理

**疑问**: 如果佛陀无有勤作, 那事业不间断的原因何在呢?

**颂**:

> **如是无勤作, 法身无生灭,**
> **于世永行持, 示现等事业。**

帝释天虽无勤作, 但依显现影像等也能利他。同样, 佛陀**无分别念等任何勤作**, 但于本来无生无灭的法身中**不动摇的同时**, 于所化无边轮回世间存在期间, **任运自成不间断**行持身幻变 / 语教言 / 意智悲周遍等事业。

**"于世永行持"**: 是 topic anchor "**佛外无涅槃**" 在事业品 "归纳差别" 层面的最终教证 (本 doc 第五度展开, 承 §3.2.2 + §8.2.1)。

### 9.2 辛二·以同法虽显现等同然以不同超胜之理

#### 9.2.1 壬一·略说

**颂**:

> **此诸比喻摄, 义是此次第,**
> **前后由断除, 违品法而说。**

九喻次第**固定**, 原因是 "前后由断除违品法而说"——**前前比喻不能无余表示佛事业**, 通过后后比喻依次**断除前喻违品的部分**, 宣说**同法的途径**来说明。

**教学结构**:
- 同法 (相同处) → 前喻好处继续引用
- 违品 (不足处) → 后喻特点遮止, 超胜前喻

如此九喻**连成环环相扣的教学链**, 渐进超胜至大地喻止, 再无更超胜可举。

#### 9.2.2 壬二·广说——九喻超胜逐一

**颂 1**:

> **佛陀如影像, 无音非同彼,**
> **如天鼓非能, 普利非同彼。**

| # | 前喻 | 前喻特点 | 前喻不足 | 后喻超胜 |
|---|---|---|---|---|
| 1→2 | 天王影像 | 无分别示现 | 无妙音, 仅可见 | 佛陀有说法妙音 (如天鼓) |
| 2→3 | 天鼓妙音 | 宣说四法印 | 只在天界, 非周遍 | 佛音周遍三界 (如云) |

**颂 2**:

> **如密云非断, 无义种非同,**
> **如大梵天非, 永熟非同彼。**

| # | 前喻 | 前喻特点 | 前喻不足 | 后喻超胜 |
|---|---|---|---|---|
| 3→4 | 密云周遍 | 成熟庄稼 | 不能断贪嗔烦恼种 | 佛断无义烦恼种 (如梵天断欲) |
| 4→5 | 梵天断贪 | 断欲界贪执 | 暂时非永久 | 佛永熟众生 (如日永照) |

《华严经》: "譬如大药王, 消灭一切毒, 诸佛圣福田, 能灭烦恼患。"

**颂 3**:

> **如日非永久, 驱暗非同彼,**
> **如摩尼宝非, 难得非同彼。**

| # | 前喻 | 前喻特点 | 前喻不足 | 后喻超胜 |
|---|---|---|---|---|
| 5→6 | 日永熟 | 成熟庄稼光明 | 白天夜晚交替, 非恒久 | 佛永摧黑暗 (如摩尼宝昼夜放光) |
| 6→7 | 摩尼宝昼夜放光 | 恒久照明 | 旁生 (龙) 亦得, 非难得 | 佛难遇 (如回响之妙说) |

戒律教证 (L40): 出家人去王宫需先看国王摩尼宝光收回没——"黎明时国王侍者把摩尼宝收回, 这时可以去王宫"。

**颂 4**:

> **如回响缘生, 亦非与彼同,**
> **如虚空非是, 善基非同彼。**

| # | 前喻 | 前喻特点 | 前喻不足 | 后喻超胜 |
|---|---|---|---|---|
| 7→8 | 回响妙说 | 空谷声难得 | 依外缘而生, 非自然无为 | 佛是自然无为法 (如虚空) |
| 8→9 | 虚空无为 | 无为法特性 | 不能成为善法根本所依 | 佛是一切善法之基 (如大地) |

**颂 5 (大地)**:

> **世间出世间, 众生诸圆满,**
> **彼处之所依, 是故如大地。**
> **依于佛菩提, 生出世道故,**
> **善业道禅定, 无量及无色。**

**大地喻止于顶端**——再无更超胜比喻可举。佛陀作为世出世间一切圆满之所依:

- **出世间** (三乘解脱道): 依于佛菩提而生
- **世间欲界圆满之因** (十善业道): 依佛妙法而生
- **世间色界圆满之因** (四禅 + 四无量心): 依佛妙法而生
- **世间无色界圆满之因** (无色定): 依佛妙法而生

一切世出世圆满**无有一法**不依佛菩提而生——此即九喻以大地终的最深义。

### 9.3 九喻超胜结构总表

| # | 喻 | 相同处 | 不同处 (不足) | 断违品 |
|---|---|---|---|---|
| 1 | 天王 | 无分别示现 | 无妙音 | 由天鼓断 |
| 2 | 天鼓 | 有妙音说法 | 非周遍 | 由云断 |
| 3 | 云 | 周遍成熟 | 不断烦恼种 | 由梵天断 |
| 4 | 梵天 | 断欲界贪 | 非永久 | 由日轮断 |
| 5 | 日轮 | 永熟 | 昼夜交替 | 由摩尼宝断 |
| 6 | 摩尼宝 | 昼夜放光 | 非难得 | 由回响断 |

| 8 | 虚空 | 无为法 | 非善法所依 | 由大地断 |
| 9 | 大地 | 一切所依 | (顶端, 无更超胜) | — |

此颂自然引向**第五利益品** (戊五·丙三彼等功德) —— 既然佛陀是世出世一切圆满之所依, 那**此教法 (《宝性论》第三转法轮 / 如来藏教义) 为何殊胜? 为何超胜布施 / 持戒 / 修行? 六度之间如何相摄?** 此即利益品所答的核心问题。

利益品是《宝性论》的终品, 总结 "此教为何是最极必要", 回归首品 "三宝本体" 的价值论完成——从 **三宝立信** (总说品) → **如来藏立因** (第一品) → **菩提立果** (第二品) → **功德立果体属性** (第三品) → **事业立果体作用** (第四品) → **利益立教法殊胜** (第五品) 的完整七金刚处论证闭环。

### 10.4 修行实践应用要点

事业品 (11 + 12 两 doc) 实践性 takeaway:

1. **祈祷无需节约**: 佛 "无量世间同时入", 祈祷不会加重佛陀负担 (本 §2.3.3)

## 十一、双层立场张力汇总

事业品 (11 + 12 两 doc) doctrinal 张力较轻——觉囊派与宁玛派对 "任运 / 不间断 / 九喻" 基本共许, 主要双层诠释点如下:

### 11.1 "佛外无涅槃" 立场

- **觉囊派朵洛瓦**: 佛陀常恒不变, 事业不间断 = **如来藏功德不灭的外显**; "佛外无涅槃" 即二转三转所诠 "灭度非实灭" 之立场 (倾向不空实有)
- **宁玛派索达吉堪布**: 承朵洛瓦立场, 再以宁玛 "空性光明双运" 调和——"佛外无涅槃" = 法界本无二涅槃, 非 "有一个不空的佛身永恒存在"; 参 05 §6 + 11 §2.10

本 doc §3.2.2 + §8.2.1 + §9.1.2 四度重申时均以双层并置模式处理, 不粗糙滑入 "实有常住佛"。

### 11.2 三密显密差别

- **显宗**: 事业品三密 (§3/§4/§5) 是佛**果位运作方式**的描述——佛身如虚空 / 佛语如回响 / 佛意如摩尼宝
- **密宗 (尤大圆满)**: 三密是**行者因位修行法门**——本尊身密 / 真言语密 / 禅定意密
- **贯通立场**: 承 09 §1.7, 同一三密体性, 因果两侧立言——《宝性论》作为显密津梁, 承许二者结构性贯通但**不等同**

详 §7。

### 11.3 "无缘不现" 的诠释

- **朵洛瓦**: 从 "众生缘起力" 立场, 无缘者心中 "如来藏功德种子未现起", 故不见佛——偏业缘感召
- **宁玛派重释**: 承朵洛瓦基础, 再以 "心器清净方现日影" 立场 (11 §5.1), 强调**心清净 > 业感召**——邪见 / 嗔恨者即使业缘多亦难见佛, 信心清净者虽业微亦可感佛; 参本 §1.3 + §3.3

### 11.4 "九喻非唯喻" 立场

- **觉囊派**: 九喻不仅是说明方便, 更是**佛三业事业如其性的显示**——影像 / 音声 / 光明等本身即佛功德的外显
- **宁玛派**: 九喻是遣疑 device (承 11 §6 立场), **功德显现非实有**——本 §8.1.1 "暂时必要 + 最极必要" 双层必要的强调即此立场

本 doc §8 保留两层, 不强合。
