# 《宝性论》第四·事业品 (上) ——任运不间断与前三喻 (天王/天鼓/云)

本文件是《宝性论》第四品**事业品** (戊四·能证方便事业之义) 上半 doc, 承 [`10-功德品.md`](./10-功德品.md) 功德品终篇 "六十四功德: 32 离系 + 32 异熟" 立场, 展开七金刚处之第四 "事业" ——即具足六十四功德的佛陀**如何无勤利益众生**。

本 doc 覆盖事业品的前半部分 (L34-L37):

- **己一·宣说任运不间断之理** (庚一略说 + 庚二广说)

## 〇、事业品整体定位

### 0.1 事业品在七金刚处中的位置

**弥勒五论七金刚处结构** (参 [`../../topics/tathagatagarbha/index.md`](../../topics/tathagatagarbha/index.md)):

```
佛宝 ── 三宝          ← 所得果 (goal)
法宝 ──  
僧宝 ──

如来藏 ── 四处         ← 因果安立
菩提  ──  (因 → 果 → 功德 → 事业)
功德  ──
事业  ──  ← 本 doc
```

四处之第四为 "事业" ——佛果所具六十四功德如何**外显为利生事业**。承菩提品 (戊二 = 果体) + 功德品 (戊三 = 果体属性) 后, 事业品 (戊四) 立**果体属性的运用侧面**。

### 0.2 从菩提品 → 功德品 → 事业品的 doctrinal 延伸线

| 品 | 科判 | 核心义 | 关系 |
|---|---|---|---|
| 戊二·菩提品 | 二清净 / 三身 | 佛**之体** | 本体论 |
| 戊三·功德品 | 64 功德 | 佛**所具功德** (静态) | 属性论 |
| **戊四·事业品** | **任运不间断 + 九喻** | **功德如何外显利生** (动态) | **作用论** |
| 戊五·利益品 | 四处难证 / 胜布施持戒 | 此教为何殊胜 | 价值论 |

L34 开篇索达吉堪布明宣:

> "前面已经学习了三品, 第一品首先讲如来藏, 每个众生都具有如来藏; 第二品讲到清净障碍以后现前法身、报身、化身的菩提; 第三品讲到菩提的六十四种功德——法身离系的三十二种功德, 报身异熟的三十二种功德。这样的功德做什么呢? **利益众生**。下面紧接着是事业品。"

即: 功德是果体所**具**, 事业是果体所**作**; 二者一静一动, 一体两面。事业品承功德品而立, 不可离开功德品独立理解。

### 0.3 事业品核心问题——佛入灭后, 事业如何继续?

L34 事业品开篇即点出本品的核心 doctrinal 关切:

> "成佛以后利益众生无有任何勤作, 就像自动化机械一样, 是自然运转的, 不需要人工操作。"
>
> "有些人可能会想, 成佛以后佛陀每天为了那么多众生不间断做利他的事业, 会不会很累? 个别宗教也认为, 造物主日日夜夜在帮助众生, 是否和这种说法一样? **实际上是不一样的**。"

即: 佛陀事业不同于 "造物主式的有分别勤作"; 事业是**无勤任运 + 不间断永续**的法性自显。佛入灭后事业不断——正是 [`../../topics/tathagatagarbha/index.md §Correctness Anchors "佛外无涅槃"`](../../topics/tathagatagarbha/index.md#correctness-anchors) 立场在事业层面的具体展开 (参 05 doc §6 + 09 doc §5.2)。

承 09 doc §2.2 "色身恒常义辨析" 立场: **化身之恒常不是 "刹那不变", 而是 "不间断"**——此 doctrinal 立场在事业品 "不间断" 法相中得以最完整展开。

## 一、事业九喻总纲 (retrieval 核心坐标)

虽然本 doc 仅详讲前三喻, 但**九喻总纲必须先列全**, 以作事业品整体导读。下文所有前三喻的展开, 皆应对照此总纲理解。

### 1.1 九喻一表总览

**总纲颂** (L35 / 译作044):

> **如帝释天鼓, 雨云与梵天,**
> **日轮如意宝, 回响虚空地。**

**九喻对应表**:

| # | 比喻 | 所表事业层面 | 核心喻义 | 本 doc 详讲 |
|---|---|---|---|---|
| 1 | **帝释天王** | 身显示现神变 | 天王于天界不动, 于人间琉璃地显影像, 人行善升天 | ✓ §3 |
| 2 | **天鼓** | 语宣说教言 | 天鼓无勤自鸣四法印, 劝勉放逸天人 | ✓ §4 |
| 3 | **雨云** | 意智悲周遍 | 云遍虚空, 无勤降雨长庄稼 | ✓ §5 |
| 4 | 梵天 | 身语幻化 | 梵天于色界不动, 幻化利欲界天人 | doc 12 |
| 5 | 日轮 | 智慧散射 | 日光无分别照耀, 遣除黑暗 | doc 12 |
| 6 | 摩尼宝 | 意密满愿 | 摩尼宝无念赐所需 | doc 12 |
| 7 | 回响 | 语密说法 | 空谷回响无实体而成声 | doc 12 |
| 8 | 虚空 | 身密周遍 | 虚空无色而周遍一切 | doc 12 |
| 9 | 大地 | 大悲趋入 | 大地无念, 为万物所依而长善法 | doc 12 |

### 1.2 九喻略说 (L35)

L35 索达吉堪布以略说方式一次铺陈九喻, 本节保留此 teachings-rich 段落作为事业品整体导读:

> 一、"帝释", 如来身体的幻变无边无际, 就像天界的帝释天。他在没有任何动摇的情况下在我们人世间能显现他的影像, 依靠他人们会行善而趋入善趣。
>
> 二、"天鼓", 佛陀的语言宣说种种遣除众生烦恼的教言, 但宣说时没有任何分别念, 如同天人的天鼓, 没有勤作的情况下自然而然发出四法印的声音, 利益无量天人。
>
> 三、"雨云", 佛陀的意智慧在有情界当中周遍地成熟众生相续, 就像天上的云遍于整个天空, 在没有勤作、没有分别念的情况下可以降下雨水, 成熟地上各种不同的庄稼和植物。
>
> 四、"梵天", 佛陀的身体和语言的事业也是在没有动摇、没有勤作的情况下可以利益众生, 就像梵天自己在住处毫不动摇, 但是他的化身可以到欲界天利益天人。
>
> 五、"日轮", 佛陀的意智慧能遣除众生心里的愚痴黑暗, 就像日光在没有分别念的情况下次第地照耀世界、遣除世间一切黑暗一样。
>
> 六、"如意宝", 佛陀秘密的意行、秘密的智慧无勤当中可以利益众生, 如同如意宝在没有分别念的情况下可以对一切众生赐予所需。
>
> 七、"回响", 佛陀语言的秘行可以利益众生, 相当于空谷回响, 本体不成立, 但是可以宣说种种不同的意义。
>
> 八、"虚空", 佛陀身体的秘行利益众生, 就像虚空周遍一切, 但不属于有为的色法。
>
> 九、"地", 佛的大悲心利益众生就像大地一样, 没有勤作、没有分别念的情况下自然而然饶益万物, 成为一切有情世界的依处。

### 1.3 九喻的宣说必要 (L35)

> "有些人疑惑, 佛陀是无生无灭的无为法的法相, 他怎么会在乃至轮回存在之间事业任运自成、永不间断? 为了遣除这样的怀疑, 让大家对佛不可思议的行境生起定解, 下面以九种比喻说明。"

即: 九喻不是装饰性的比喻, 而是为遣除 "佛无为法如何起用" 的根本诤问而设立的教学 device。承 09 doc §2.2 "色身恒常 = 不间断" 立场, 事业品以九喻在 "作用层面" 完成此 doctrinal 辨析。

**教证依据** (L35): 《如来庄严智慧光明入一切佛境界经》(昙摩流支 / 法护译) 为九喻最早经典依据。

## 二、任运 + 不间断两法相 (核心 doctrinal)

### 2.1 事业品的双法相结构

事业品以**两大法相**统摄佛陀事业:

| 法相 | 义 | 对治 |
|---|---|---|
| **任运自成** (无勤) | 无勤作地显发, 不分别却有所成 | 对治 "造物主式有分别勤作" 误解 |
| **不间断** (恒常) | 事业不因佛入灭而断, 法性恒常显发 | 对治 "佛入灭 = 事业终结" 误解 |

两法相合 —— 佛功德在**无勤作的同时不间断地利益众生**。这是事业品区别于其他宗教 "创造神持续干涉世界" 的最核心立场。

**L34 核心宣示**:

> "佛陀不会如此, 他是什么样的呢? 所有的事业任运自成, 相当于大自然的客观规律一样, 成佛以后利益众生无有任何勤作, 就像自动化机械一样, 是自然运转的, 不需要人工操作。"

### 2.2 庚一·略说任运自成之义 (辛一, L34)

**总颂**:

> **化界化方便, 化界调化事,**
> **彼境应时往, 遍主恒任运。**

任运自成分五义 (五方面不分别勤作):

| # | 五义 | 颂字段 | 内涵 |
|---|---|---|---|
| 1 | **化界** | 化界 | 了知所化众生不同根基、界性、意乐、爱好、种姓、烦恼随眠, 但无 "此好彼差" 的分别 |
| 2 | **化方便** | 化方便 | 了知度化所需方便: 寂静相 / 忿怒相 / 显宗 / 密宗 / 各乘各形式 |
| 3 | **化界调化事** | 化界调化事 | 将所化安置于人天增上生或三菩提决定胜, 无执著分别 |
| 4 | **彼境** | 彼境 | 了知调化众生的环境: 舍卫城 / 迦毗罗卫 / 王舍城等, 无分别执著 |
| 5 | **应时往** | 应时往 | 了知度化时机, 不过早不过晚 |

"遍主恒任运" ——遍主佛陀恒时以无勤作方式起用。

**无分别却善巧的教证** (L34):

> "其自性虽离贪嗔, 贪心者前示似贪, 因所化嗔示嗔恨, 顶礼住大菩提彼。"

佛自性虽离贪嗔, 但因所化不同, 于贪心者前示现贪者, 嗔心者前示现嗔恨 ——佛的相续实未被染污, 是**任运随机应化**。

**教证** (L34, 《华严经》):

> "譬如日月游虚空, 照临一切不分别, 世尊周行于法界, 教化众生无动念。"

### 2.3 庚一·略说不间断之义 (辛二, L34)

**总颂** (长颂):

> **具胜功德珍宝聚, 智水大海福慧日,**
> **无余决定修行乘, 无中边广如空遍,**
> **佛见无垢功德藏, 于诸众生皆无别,**
> **以佛大悲风吹散, 烦恼所知之云雾。**

**六事不间断之理** (略说阶段已列不间断的六因):

| # | 六事 | 比喻 | 义 |
|---|---|---|---|
| 1 | **出离之因** | 大海 | 因地具总持、等持等功德珍宝, 以无垢智水充满菩萨十地 |
| 2 | **显示之因** | 日轮 | 福慧二资粮如日光般成熟众生 |
| 3 | **果 (菩提)** | 虚空 | 得无中无边广大如虚空般周遍之大菩提 |
| 4 | **摄受众生** | 宝藏 | 照见众生皆具如来藏, 摄受他们 |
| 5 | **所断障碍** | 云雾 | 众生相续中烦恼障、所知障如云雾遮蔽 |
| 6 | **能断之缘 (大悲)** | 狂风 | 以大悲风吹散二障云雾 |

此**六因次第**是事业不间断的 doctrinal 骨架。因圆满 (1 + 2) → 果圆满 (3) → 利生圆满 (4) → 障染现起 (5) → 对治显现 (6), 步步相因, 事业永续。

### 2.4 庚二·广说任运自成之义 (辛一, L34)

**总颂**:

> **何者依何化, 何为何处时,**
> **无彼分别故, 能仁恒任运。**

**并重颂**:

> **于何所化界, 依众方便化,**
> **所化之所为, 何处与何时。**

广说与略说核心义一致, 以 "何者 / 依何化 / 何为 / 何处 / 时" 五方面再次宣示任运自成。

**广说关键 teachings** (L34):

> "大地不偏于任何方向, 非常平等地让整个大地上所有的草木、万物依靠它自然生长。大地不会想'我要让东边的树长起来'……同样的, 佛也没有这样的分别, 心没有高下, 但他仍然以六度和他的法语来利益众生。"

此即大地喻 (九喻之末) 在广说阶段已预立, 意象连贯。

**索达吉堪布 L34 "千江有水千江月" 论**:

> "月轮在虚空当中, 虽然没有分别, 但依靠缘起力, 它的光芒直接现于所有水器当中。……有再多的人祈祷都不用担心, '现在很多人正在祈祷的时候, 佛陀他肯定很忙。我不祈祷, 不然佛陀太辛苦了, 我的业力很深重, 我的业障会不会沾染了他。'……**如果我们真正学了了义经典, 包括《宝性论》中的道理, 应该会明白**。"

此是事业品最 pragmatic 的应用——破除 "佛有限勤作" 的实执误解。

**教证** (L34, 《华严经》):

> "如来净法身, 饶益无量众, 法身亦无念, 我利诸群萌。"

### 2.5 庚二·广说不间断之义 (辛二) ——五壬结构

不间断广说分**五壬**——这是事业品 doctrinal 最精密段落, 从 "名 / 义 / 喻 / 喻义对应 / 归纳" 五层展开。本节 §2.6-§2.10 依次展开五壬。

### 2.6 壬一·宣说名称分类 (L34)

**总颂**:

> **出离彼显示, 彼果彼摄受,**
> **彼断彼障缘, 无有分别故。**

宣说事业不间断的**六种名称**:

1. **出离** (出离之因)
2. **彼显示** (显示之缘)
3. **彼果** (果——菩提)
4. **彼摄受** (摄受具如来藏众生)
5. **彼断** (所断障碍)
6. **彼障缘** (能断障之缘——大悲)

L34 索达吉堪布强调:

> "有些外文注释中说, 这几个次第不是依次的, 但这种说法是不对的, 应该一步一步上来, 先因地再果地, 再摄受众生。在摄受众生时, 发现众生有障碍, 然后用大悲风将其吹走。这几个并不是分开的, 应该这样来理解。"

即六名是**因次第有序**的, 非平列。

### 2.7 壬二·解说义本体 (L34)

**总颂** (长颂):

> **出离即十地, 二资是彼因,**
> **彼果胜菩提, 菩提摄众生,**
> **无量障烦恼, 随烦恼习气,**
> **诸时摧彼缘, 即是大悲心。**

六名之**义本体**:

| # | 名 | 义本体 | 对应 |
|---|---|---|---|
| 1 | 出离 | **菩萨十地** (极喜地 → 法云地) | 从无明中次第出离 |
| 2 | 彼显示 | **福慧二资粮** | 令十地功德越来越增长 |
| 3 | 彼果 | **胜菩提** (圆满正等觉) | 因圆满所得之果 |
| 4 | 彼摄受 | **菩提摄众生** | 得菩提后摄受众生 |
| 5 | 彼断 | **无量障 (烦恼 / 随烦恼 / 习气)** | 众生如来藏上无量障习 |
| 6 | 彼障缘 | **大悲心** | 一切时摧毁障垢之能摧缘 |

**华智仁波切立场** (L34 引):

> "华智仁波切在《现观庄严论》的修行次第中讲到, 成佛并不是做其他的, 就是利益众生。十地圆满以后现前菩提, 之后摄众生。"

此与 09 doc §1.4.3 "化身四阶段引导" 立场一致——**成佛 = 利生**。

### 2.8 壬三·以比喻表示彼等之理 (L34)

**总颂**:

> **当知此六理, 依次如大海。**
> **日轮及虚空, 宝藏云与风。**

六义配六喻:

| # | 六义 | 六喻 |
|---|---|---|
| 1 | 出离 (十地) | **大海** |
| 2 | 彼显示 (二资粮) | **日轮** |
| 3 | 彼果 (菩提) | **虚空** |
| 4 | 彼摄受 (众生界) | **宝藏** |
| 5 | 彼断 (烦恼) | **云聚** |
| 6 | 彼障缘 (大悲) | **狂风** |

**L34 teaching 要点**:

> "一个世间很成功的老师, 知识如大海般非常丰富; 对他所有的专业也是非常明白, 如日轮一般; 最后他有虚空一样广大的思想; 这个时候他要了解他的所化弟子、学生, 看看他们相续中如云一样的障碍是什么样的, 然后他有特别好的爱心, 把自己拥有的智慧传给他们。"

### 2.9 壬四·广说喻义对应 (L34)

**总颂** (六颂):

> **具智水德宝, 故地如大海。** (出离喻)
> **有情生存故, 二资如日轮。** (显示喻)
> **广无中边故, 菩提如虚空。** (果喻)
> **真佛法性故, 众生界如藏。** (摄受喻)
> **客尘遍不成, 故惑如云聚。** (所断喻)
> **散彼近住故, 大悲如狂风。** (能断缘喻)

**六喻六义对应广说一表尽览**:

| # | 喻 | 喻义所以然之理 | 承接前文 |
|---|---|---|---|
| 1 | 大海 | 菩萨十地具有无垢智慧水 + 总持等持等功德珍宝 | 承 03 doc §4 "十地" + 10 doc §3 "十力" |
| 2 | 日轮 | 福慧二资粮能饶益众生生存, 成熟果德 | 承 07 doc §1 "因方面" + 10 doc §6 "异熟果" (福德资粮) |
| 3 | 虚空 | 菩提周遍一切而广大、无中边而甚深 | 承 09 doc §1.6.2 "色身住于法身如虚空" + 08 doc §1.1 "虚空喻" |
| 4 | 宝藏 | 真正圆满佛陀法性功德无尽财富原本任运自成安住 —— 众生界自性清净如来藏如大宝藏 | 承 03 doc §3 + 06 doc §3 "地下宝藏喻" |
| 5 | 云聚 | 客尘自性能遍, 有垢染遮障, 但胜义不成立 (可断客尘) | 承 06 doc §4 "九垢" + 07 doc §2.7 "客尘可除" |
| 6 | 狂风 | 大悲能吹散烦恼乌云, 是示道近住 (具足示道)  | 承 09 doc §1.4.3 "化身四阶段引导" |

**"真佛法性故, 众生界如藏" 的 doctrinal 要点** (L34):

> "佛陀得到菩提以后发现, 真正圆满正等觉的因其实在众生位的时候就有。每一个众生相续中具有如来藏, 所以佛陀一定要摄受这些众生。"

此是事业品与如来藏品的 doctrinal 接合点: 佛陀摄受众生的所以然之理, 根本在于**众生具如来藏**——非佛外在赐予, 而是佛照见众生本具 (承 03 doc §3 + 06 doc 九喻立场)。

### 2.10 壬五·归纳不间断之理 (L34)

**总颂**:

> **依他而出离, 自见众生同,**
> **事业未圆故, 有际不间断。**

归纳立场:

- **依他而出离**: 佛因地 (十地二资) 发愿度一切众生 → 依他利众生而出离 (此即菩提心)
- **自见众生同**: 佛得菩提后, 自见与众生法界自性清净无别 → 见度化有义
- **事业未圆故**: 大悲事业未圆满 (众生未尽)
- **有际不间断**: 乃至所化轮回未空, 佛陀事业不间断

**核心教证** (L34, 《入行论》):

> "乃至有虚空, 以及众生住, 愿吾住世间, 尽除众生苦!"

此颂是事业不间断的 **locus classicus**。索达吉堪布 L34 明宣:

> "轮回的边没有间断, 佛陀的事业也就不会间断。"

此承 [`../../topics/tathagatagarbha/index.md §Correctness Anchors "佛外无涅槃"`](../../topics/tathagatagarbha/index.md#correctness-anchors) + 05 doc §6 "佛外无余涅槃" + 09 doc §2.2 "色身恒常 = 不间断" 立场, 在事业品中完成 doctrinal 闭环 ——佛果非终点, 而是无限利生的起点。

**引入 "一乘究竟" 立场**:

如果众生有边际、不具如来藏, 佛陀就 "下岗了"; 但实际上乃至虚空存在之间, 佛陀事业不会终结。此即 09 doc §1.4.3 "小乘者以大乘成熟 → 解脱" 引导链 (化城喻) 在事业不间断立场上的根据。

## 三、天王喻——示现神变之理 (辛一, L35-L36)

### 3.1 天王喻定位

**九喻首喻**, 表**身显示现神变** (身方面的事业)。科判分三 (L35):

- 壬一·虽无分别也成他利之理 (能表喻本体 + 所表义本体)
- 壬二·善加对应宣说彼理
- 壬三·归纳宣说无生灭之理

天王 = 帝释天 (三十三天之主)。**核心情节**: 清净大地 (琉璃) 中显现帝释天影像, 依彼影像, 地上人们行善升天。类比: 佛于法身不动中显现色身于清净众生心中, 依彼影像, 众生行善得佛果。

### 3.2 能表喻本体 (壬一·癸一)

分四子——依四子展开比喻情节。

#### 3.2.1 清净大地显现影像之理 (子一)

**颂**:

> **犹如净琉璃, 自性此地上,**
> **净故见天王, 天女会众俱。**
> **美妙尊胜宫, 彼天境处外,**
> **无量宫种种, 以及众天物。**

**情节描绘**: 整个大地变成琉璃般, 非常光滑、自性清净。因大地特别清净, 有缘的人在此见到:

- 帝释天王 + 天女眷属
- 美妙的尊胜宫
- 无量的天宫、天人
- 金银琉璃等七宝组成的鲜花、琥珀
- 天界美妙的供品、装饰

**L35 现代类比**:

> "看电视时, 电视上面没有任何人和物体, 天人也好、鬼神也好, 什么都没有, 但是因缘具足的话, 在里面什么都可以显现。"

#### 3.2.2 依彼行善业之理 (子二)

**颂**:

> **尔后男女众, 安住地上者,**
> **得见彼显相, 而发如是愿,**
> **愿我等不久, 犹如此天王。**
> **是为获得彼, 真实行善住。**

地上人见到帝释天影像, 发愿 "愿我等不久犹如此天王", 为此真实行善——受八关斋戒、布施、供养等。

**教证** (L35): 难陀公案——佛带难陀至天界看天女, 难陀回凡间后精进修行以求升天。

#### 3.2.3 依彼业获得所欲果位之理 (子三)

**颂**:

> **彼等依善业, 虽不了此相,**
> **如是显现理, 此逝转天界。**

人们不知所见只是影像 (非真帝释天本身), 但依行善之力, 临终确能升天。

#### 3.2.4 无勤成义之理 (子四)

**颂**:

> **彼相无分别, 亦无有动摇,**
> **然于地上者, 具有大意义。**

琉璃地上显现的帝释影像, **无分别、无动摇**, 但对地上众生具大意义 (令行善升天)。

**核心立场**: **无分别而有大义**——此即事业品的根本命题在天王喻中的首次落地。帝释天在尊胜宫中从未动摇, 但地上人们确实无欺见到, 并行善得天果。

### 3.3 所表义本体 (壬一·癸二)

能表义四子与所表义四子**一一对应**。

#### 3.3.1 清净心显现色身之理 (子一)

**颂**:

> **信等无有垢, 修信等功德,**
> **自心现佛陀, 妙相具随好。**
> **散步及站立, 安坐及安眠,**
> **做种种威仪, 宣说寂灭法,**
> **不说而入定, 行种种幻变,**
> **具大威光尊, 众生得现见。**

**义对应**: 如琉璃大地显帝释, 众生清净心显佛陀色身 (具 32 妙相 + 80 随好)。

"信等无有垢" 之 "信等" —— **信 / 精进 / 正念 / 智慧 / 禅定** (五根) 完全清净, 没有违品 (不信等) 垢染。

**所见佛之行相**:

- 散步 / 站立 / 安坐 / 安眠 (四威仪)
- 宣说寂灭法 / 不说法入定
- 行种种幻变 / 具大威光

**关键 anchor** (L35):

> "若以色见我, 以音声求我, 是人行邪道, 不能见如来。" (《金刚经》)

从了义真实义角度, 这些相都没有; 但暂时从真世俗角度, 见到佛色身是有功德的——每个有缘众生根据缘分不同都能见到 (梦中 / 觉受中 / 白天日常生活中)。

**公案** (L35):

- **智者大师**: 亲入法华大会, 见佛金色身相
- **玄奘参拜佛影窟** (《大慈恩寺三藏法师传》): 至心礼拜数百次, 洞中佛影终现; 五强盗同见, 皆皈依

#### 3.3.2 成就所求果位之理 (子二)

**颂**:

> **见彼亦希求, 为佛极力行,**
> **彼因真实行, 能得所求果。**

见佛后, 不仅 "我要发财升官" 的低愿, 而应为成佛精进行持**六度万行**。

**因果链**: 发菩提心 (意乐) → 行六波罗蜜多 (加行) → 成佛 (果) ——此与 07 doc §1 "二清净因" + 08 doc §1 "二利事业" 立场对接。

**L35 建议**: 见本尊 / 上师时, 应发菩提心, 勿发世间小愿 (如 "我要一辆车"、"织布变熟练" 之类)。

#### 3.3.3 无勤也有义之理 (子三)

**颂**:

> **彼现亦无有, 分别及动摇,**
> **如此于世间, 亦以大义住。**

佛色身显现于有缘众生前时, 佛陀**无分别、无勤作**。所以然之理: **两个缘起具足** ——

1. 佛三阿僧祇劫发愿力 (佛方)
2. 众生心清净 (众生方)

二缘和合, 佛在有缘众生前显现; 非 "佛跑去某人处" 的有分别勤作。

**教证** (L35, 《华严经》):

> "如来普现前, 令众生信喜, 佛体不可得, 彼亦无所见。"

名言世俗中见如来相有意义; 胜义中如来体不可得, 所见亦无。——**二谛双照**立场。

#### 3.3.4 次第得究竟法身之理 (子四)

**颂**:

> **异生虽不知, 此乃自心现,**
> **如此见色身, 彼等亦具义。**
> **次第依见彼, 安住此乘者,**
> **将以慧眼见, 内胜义法身。**

凡夫异生虽不知所见是自心清净显现 (非真佛), 但见佛色身即有大义; 依见色身逐渐安住大乘见修行果, 最终以入定无分别智慧眼, 见**内在胜义法身** (本自具足的如来藏)。

**核心立场**: **从外相显现 → 内心本具**——修行次第由外向内。最初依外在佛像、佛经、寺院、僧人趋入, 最终见**自相续之佛**。

**L35 教证** (《妙法莲华经》):

> "一称南无佛, 皆已成佛道。"

即使以散乱心见佛像佛塔, 亦是很有意义的。

### 3.4 善加对应宣说彼理 (壬二, L35)

**总颂** (长颂):

> **地离余惨境, 无垢之琉璃,**
> **明妙具珠宝, 离垢妙功德。**
> **上平净故现, 种种天王影,**
> **渐离地功德, 彼复不显现。**
> **为得彼事乐, 斋戒布施等,**
> **男女之众人, 以信散花等。**
> **为得净琉璃, 心现能仁王,**
> **具有极喜心, 佛子发大心。**

此段作**整体喻义对应**:

| 喻 | 义 |
|---|---|
| 大地离悬崖深渊等惨境, 成无垢琉璃 | 众生心离垢, 成清净心 |
| 琉璃地上平、清净、如镜 → 现天王影 | 心清净 → 现佛色身 |
| 地渐离明澈功德 → 影不再现 | 心渐失清净 → 佛色身不再显 |
| 人们为得天王福报, 受八关斋戒、布施、散花 | 佛子菩萨为得佛果, 发大菩提心, 行六度 |

**"具有极喜心, 佛子发大心" 的强调** (L35):

> "见到上师、见到佛、见到菩萨, '我好欢喜! 我好开心!' 不能光这样, 应该要发大愿, 发大愿以后, 才能获得广大的果报, 不然光是欢喜心也起不到什么作用。"

### 3.5 归纳宣说无生灭之理 (壬三, L35)

**总颂**:

> **如净琉璃地, 现天王身影,**
> **众生净心地, 映现佛身影。**
> **众前影现没, 随无浊浊心,**
> **如世显现相, 不见有与坏。**

**核心结构**:

| 琉璃地 (喻) | 众生净心 (义) |
|---|---|
| 琉璃清净 → 现天王影 | 净心无垢 → 现佛色身 |
| 琉璃有垢 → 影消没 | 心有浊垢 → 色身隐没 |
| **天王本身于自处不动摇, 不见有生坏** | **佛胜义法身永远不见 "有生与坏灭", 因无为法无变** |

**核心 anchor**:

> "天王的影像显现与否, 其实也和因缘是否具足有关, 同样, 众生面前佛的影像现前还是不现前, 和众生的心不浊或污浊有关系。"
>
> "但是有一个地方不同, 是什么呢? '不见有与坏', 佛陀的法身不见有与坏, 法身是无为法所摄, 不管怎么样, 长期存在, 没有生灭。"

**法身无生灭 + 色身随缘现没**——这是天王喻归纳的 doctrinal 要义, 承 09 doc §2 "三身恒常" + 09 doc §3.3 "色身也不可思议" 立场在事业品中的落地。

### 3.6 天王喻 doctrinal 小结

- **身显示现神变**: 佛以身幻化无边无际, 随缘显现
- **二缘起要点**: 佛宿愿力 + 众生清净心 → 和合而现 (非佛有分别地 "来")
- **无生灭之理**: 色身随缘现没, 法身无为不动
- **修行次第**: 外见色身 → 内见法身; 初依外相, 终见自相续之佛

## 四、天鼓喻——说教言之理 (辛二, L36)

### 4.1 天鼓喻定位

**九喻第二喻**, 表**语宣说教言** (语方面的事业)。科判分三 (L36):

- 壬一·如天鼓般说妙法之理
- 壬二·较比喻更超胜之理
- 壬三·彼等依缘趋入之理

核心情节: 天界天鼓无勤自然响起, 宣说四法印, 劝勉放逸天人。类比: 佛陀法音无勤宣说, 遣众生烦恼。

### 4.2 如天鼓般说妙法之理 (壬一, L36)

分二·癸: 广说喻义等同之理由 + 善加对应归纳。

#### 4.2.1 癸一·子一无勤之理

**颂** (长颂):

> **如天境天人, 以昔白法力,**
> **无有勤作处, 意色分别念。**
> **无常苦无我, 寂灭法鼓声,**
> **屡屡而劝勉, 放逸诸天人。**

**天鼓的 "四无"**:

| # | 四无 | 义 |
|---|---|---|
| 1 | 无勤作 | 不似凡人说法需努力 |
| 2 | 无发音处 | 无喉咙、舌头、声带等发音器官 |
| 3 | 无意色 (意识形态的身体) | 无有情身躯 |
| 4 | 无分别念 | 无动机 |

然天鼓依天人**往昔白法善业威力**, 自然传出四法印声音, 劝勉放逸天人。

**四法印**:
1. 诸行无常 (一切有为法无常)
2. 有漏皆苦 (一切有漏法痛苦)
3. 诸法无我 (一切法无我)
4. 涅槃寂静

**义对应颂**:

> **遍主无勤等, 佛语普周遍,**
> **无余所化众, 于有缘说法。**

遍主佛陀亦远离四勤, 但佛语周遍一切有缘众生, 随三乘根器应机宣说。

**关键 anchor**: 天鼓声仅限天界 (如三十三天); **佛语周遍整个世界**——佛超胜天鼓。

#### 4.2.2 癸一·子二由业所生之理

**颂**:

> **如天境天人, 鼓声自业生,**
> **能仁所说法, 亦由自业生。**

**业所生对比**:

| 现象 | 生因 |
|---|---|
| 天鼓四法印声 | 天人**共业** (往昔行持白法之善业) |
| 佛陀宣说种种法 | 众生**自业** (各自信心和业力) + 佛发愿力 |

**核心立场**: 佛所宣说的法不是 "佛单方面给予", 而是**众生业感**。众生业力是法能显现的增上缘。

此立场与《入行论》"地狱狱卒兵器以业力现前" 平行——**业感 / 缘起义**的普遍立场。

**L36 类比**: 两人同至喇荣, 一人觉得殊胜, 一人觉得不行——**不共业**差别显现。

#### 4.2.3 癸一·子三成就寂灭之理

**颂**:

> **离勤处身心, 鼓声令行寂,**
> **远离勤作等, 此法令成寂。**

天鼓远离勤作 / 发音处 / 意形身 / 分别念四法, 但能令天人闻四法印后, **灭尽散乱放逸**, 入修行状态。

同理, 佛语远离四法, 众生闻之能**寂灭烦恼热火、得成就**。

**教证** (L36, 《摄大乘论》):

> "譬摩尼天鼓, 无思成自事, 如此不分别, 种种佛事成。"

#### 4.2.4 癸二·善加对应归纳

**颂** (两颂):

> **天城法鼓声, 因生赐无畏,**
> **惑赴沙场时, 胜非天除戏。**
> **世间禅无色, 因生尽摧毁,**
> **众生烦恼苦, 说寂无上道。**

**天鼓四作用 / 佛语四作用对应**:

| # | 天鼓作用 | 佛语对应 |
|---|---|---|
| 1 | 与非天作战时赐无畏 (除嗔) | 众生烦恼战场中摧灭烦恼之畏惧 |
| 2 | 天人贪恋欲乐时息散乱 (除贪) | 令众生修四禅四无色之等持 |
| 3 | 传四法印入寂 | 传四圣谛寂灭法 |
| 4 | 遣除战胜非天怖畏 | 宣说寂灭涅槃无上道 |

### 4.3 较比喻更超胜之理 (壬二, L36)

分二·癸: 略说 + 广说差别。

#### 4.3.1 癸一·略说——佛语胜天鼓的四方面

**颂**:

> **遍众生利乐, 具足三神变,**
> **是故能仁音, 胜天物铙钹。**

**佛语胜天鼓四方面**:

| # | 四胜 | 义 |
|---|---|---|
| 1 | 遍众生 | 周遍一切所化, 非片面 |
| 2 | 利 | 以决定胜果位利益 (非仅增上生) |
| 3 | 乐 | 暂时得增上生欢喜 |
| 4 | 具足三神变 | 出离痛苦 |

**三神变**:
- **神境神变** (神通神变): 身体幻化成各种形式, 十方示现利生
- **记说神变**: 依无分别智慧深入众生心, 照亮心里的黑暗 (实为智说法)
- **示导神变**: 依语言比喻推理, 令众生出离苦海 (语说法)

身 + 意 + 语三种神变皆具, 故佛语超胜天鼓。

#### 4.3.2 癸二·广说差别——四超胜颂

**四超胜颂** (L36):

> **天界大鼓声, 不入住地耳,**
> **佛鼓声遍行, 轮回地下界。** (1. 范围超胜)
>
> **天界多铙钹, 为增欲火传,**
> **悲尊一妙音, 为灭苦火入。** (2. 作用超胜——灭苦非增欲)
>
> **天妙铙钹声, 增心掉举因,**
> **悲尊如来语, 劝专思等持。** (3. 导向超胜——入定非散乱)
>
> **总之诸世间, 天地安乐因,**
> **彼遍照世间, 尽依妙音宣。** (4. 根本超胜——安乐之因唯佛语)

**一表尽览**:

| # | 超胜面 | 天鼓 / 天乐 | 佛语 |
|---|---|---|---|
| 1 | 范围 | 仅限所在天界 | 周遍三千大千世界 (轮回地上天下) |
| 2 | 作用 | 部分息散乱, 部分增欲火 | 为灭苦火而入 |
| 3 | 导向 | 增心掉举, 散乱 | 劝专思等持 |
| 4 | 根本 | 非出离因 | 安乐根本, 遍照世间 |

**教证** (L36, 《大庄严论经》):

> "佛语至天道, 及以解脱道, 此语决定至, 中间终无错。"

**L36 目犍连公案**: 目犍连欲测佛声能传多远, 神变至九十九恒河沙数世界外光明如来刹土, 佛声不增不减, 目犍连神变已到极限, 依佛加持方得返回。——示佛声遍行之不可思议。

### 4.4 彼等依缘趋入之理 (壬三, L36)

**颂**:

> **犹如诸聋人, 不闻细微声,**
> **具有天耳者, 亦非闻一切。**
> **细微殊胜法, 细微智行境,**
> **亦唯无烦恼, 具慧者得闻。**

**双层对比**:

| 世间例 | 对应佛法例 |
|---|---|
| 聋人不闻细微声 | 无缘者对佛八万四千法门 "捂耳离开" |
| 具天耳者亦非闻一切世间声 | 具世间有漏神通的天人, 亦非闻一切佛语 |
| 细微声 | 佛细微殊胜法 (甚深、光明、离戏) |

**闻法器的要求**:
- **无烦恼** (非嗔贪嫉妒忿心严重)
- **具慧** (非一窍不通)
- **具缘**

**教证** (L36, 《四百论》): 具慧、正直的人才是闻法器。

**教证** (L36, 《佛说护国尊者所问大乘经》):

> "具胝劫数难值佛, 所说正法亦难闻, 刹那暂遇若依行, 彼人必证菩提果。"

### 4.5 天鼓喻 doctrinal 小结

- **语宣说教言**: 佛语无勤自成, 如天鼓自鸣
- **业所生立场**: 法依众生共业 + 佛愿力共同显现, 非单方赐予
- **四超胜**: 范围 / 作用 / 导向 / 根本, 佛语皆远超天鼓
- **三神变**: 身神境 + 意记说 + 语示导, 佛事业三轮齐具
- **闻法器条件**: 无烦恼 + 具慧 + 具缘

## 五、云喻——智悲周遍之理 (辛三, L37)

### 5.1 云喻定位

**九喻第三喻**, 表**意智悲周遍一切有情**。科判分四 (L37):

- 壬一·对应增长庄稼之同法
- 壬二·对应器成之同法
- 壬三·对应无观待之同法
- 壬四·对应灭火之同法

核心意象: 云遍虚空, 无勤降雨; 雨因地不同成不同味; 雨无观待众生感受; 雨灭火。四分对应佛智悲的**四种周遍作用**。

### 5.2 对应增长庄稼之同法 (壬一, L37)

分二·癸: 略说无勤之理 + 广说依缘趋入之理。

#### 5.2.1 癸一·略说无勤之理

**颂** (长颂):

> **犹如夏季云, 庄稼茂盛因,**
> **雨露无勤中, 持续降大地。**
> **大悲云中降, 佛陀妙法雨,**
> **众善庄稼因, 无有分别念。**

**喻义对应**:

| 喻 | 义 |
|---|---|
| 夏季浓云密布 | 佛大悲云 |
| 雨露无勤持续降大地 | 佛妙法雨无勤任运降 |
| 庄稼茂盛 (雨之果) | 众生善根庄稼增长 |
| 云雨无分别念 | 佛法无分别念 |

**L37 核心立场**: 云不想 "庄稼干旱, 我去滋润", 雨不想 "我要降下去"——以**农民福报 + 众生因缘**, 雨自然降下, 万物自然成长。佛法亦如此。

**重要限定** (L37):

> "当然众生也要祈祷, 这方面需要一种因缘, 否则佛陀的法雨不一定能融入众生的心。"

即: 佛法无勤, 但**众生需具缘**; 无缘则不得利益。

#### 5.2.2 癸二·广说依缘趋入之理

**颂**:

> **如世入善道, 风生云降雨,**
> **悲风众善增, 佛云降法雨。**

**缘起链**:

- 世间缘起: 众生行十善、修六度、不放逸 → 依天人威力风搅拌 → 云聚 → 降雨
- 佛法缘起: 众生信心精进善根增上 → 佛大悲风搅拌 → 佛智慧云 → 降法雨滋润众生

**佛智悲本体颂**:

> **于有大智悲, 不染变不变,**
> **住空定持海, 佛云善稼因。**

承 09 doc §2 "恒常" + 07 doc §2 "二清净" 立场:

| 佛之本体面 | 义 | 对应 |
|---|---|---|
| 大智慧 | 通达万法空性 → 不染变化轮回 | "智不住三有" |
| 大悲 | 摄受无量众生 → 不染不变涅槃 | "悲不滞涅槃" |
| 住空定持海 | 住虚空般空性境界, 成就无漏禅定、总持、等持如海功德 | 承 10 doc §0.2 "64 功德" |

故**佛云 = 增长善法庄稼之因**。

**立场**: **智不住三有, 悲不滞涅槃**——此为佛出离二边的经典立场, 承 09 doc §1.5.1 "胜涅槃" (佛陀七名之一) 立场。

### 5.3 对应器成之同法 (壬二, L37)

分二·癸: 比喻 + 意义。

#### 5.3.1 癸一·比喻

**颂**:

> **犹如凉香柔, 轻水云中出,**
> **与地盐等系, 成极多种味。**

**云中降下之雨水本具八功德** (八支一味水):

| # | 八功德 | 义 |
|---|---|---|
| 1 | 清凉 | 所触非酷热 |
| 2 | 香甜 | 味道甘美 |
| 3 | 柔和 | 不伤身 |
| 4 | 轻爽 | 不重不硬 |
| 5 | 洁净 | 特别干净 |
| 6 | 无臭 | 无臭味 |
| 7 | 不伤腹 | 不拉肚子 |
| 8 | 不伤喉 | 不伤喉咙 |

然落到地上后, **因地性 (盐地等) 差异**, 味道随之变 (苦 / 涩 / 酸 / 甜……), 功效不同。

#### 5.3.2 癸二·意义

**颂**:

> **八圣支甘霖, 广悲云藏出,**
> **众相续处别, 成具多种味。**

**喻义对应**:

| 喻 | 义 |
|---|---|
| 八功德水 | 佛法**八圣道**甘霖 (正见 / 正思维 / 正语 / 正业 / 正命 / 正精进 / 正念 / 正定) |
| 一味一体云中出 | 八圣道出自佛广大大悲云藏, 真如一味 |
| 因地性不同成多种味 | 因众生相续 (种姓 / 根基 / 界性 / 意乐) 不同, 法显现三乘多种味 |

**教证** (L37, 《华严经》):

> "如来雨正法, 大悲一味水, 随应受化故, 种种差别说。"

**核心立场**: 佛法**真如中一味**, 因**法器不同**显现多种 ——声闻种姓前成声闻乘人无我, 缘觉种姓前成十二缘起, 菩萨种姓前成二无我空性法。同一法雨, 不同法器, 不同取用。

**月称旃扎辩论公案** (L37): 月称所说法, 有众生得利 (正确理解空性), 有众生错解 (把空性当断灭而毁相续) ——法本无害, 众生根基不同故。

### 5.4 对应无观待之同法 (壬三, L37)

分三·癸: 三喻略说 + 广说其义 + 无观待趋入之理。

#### 5.4.1 癸一·三喻略说

**颂**:

> **于胜乘净信, 中等与嗔群,**
> **三种如众人, 孔雀与饿鬼。**

**三种法器 vs 三种受众**:

| # | 三种法器 (对佛法态度) | 喻 |
|---|---|---|
| 1 | 对胜乘净信 (决定信解) | **人** (喜雨) |
| 2 | 中等 (既不欢喜也不痛苦) | **孔雀** (中性) |
| 3 | 嗔群 (不信、排斥、诽谤) | **饿鬼** (厌雨) |

#### 5.4.2 癸二·广说其义

**颂** (长颂):

> **春末无云人, 不行空中鸟,**
> **夏季雨降地, 饿鬼则痛苦。**
> **大悲云聚中, 法雨现不现,**
> **欲法嗔法者, 世间彼比喻。**

**广说三喻**:

| # | 喻 | 情态 | 对应众生 |
|---|---|---|---|
| 1 | 春末无云 / 有雨 | 农民: 无雨痛苦, 有雨欢喜 | **欲法者** (正法降则喜, 不降则不喜) |
| 2 | 孔雀 (不飞空中鸟) | 下雨不下雨都可以, 无所谓 | **不定种姓** (法雨降与不降, 无欢无嗔) |
| 3 | 饿鬼 | 夏季降雨时, 业感为热沙雨、燃烧金刚火 — 特痛苦 | **嗔法者** (法雨降时嗔恨, 诽谤) |

**L37 教导**: 即便遇嗔法者的诽谤言辞, 不应以嗔恨心对待。

**教证** (《佛说华手经》):

> "诽谤佛法的人, 我们最好不要对他们生嗔恨心, 如果生嗔恨心, 不是真正的沙门, 也不是沙门法。"

**教证** (《大般涅槃经》):

> "于余焦木莫生恶心。何以故? 一切众生因恶心故堕于地狱。"

#### 5.4.3 癸三·无观待趋入之理

**颂** (长颂):

> **大滴雨降下, 热沙金刚火,**
> **细微至山者, 云聚不观待。**
> **细广巧方便, 智悲之密云,**
> **诸时无观待, 惑净我见眠。**

**喻义对应**:

| 喻 | 义 |
|---|---|
| 大滴雨、大冰雹降下 | 佛智悲法雨广大降下 |
| 饿鬼感为热沙雨, 地狱众生感为金刚火 | 我见随眠坚固者, 法不显利益 |
| 细微含生 (蜜蜂、小虫) 被摧毁 | 根基不足者 (烦恼深) 难得利益 |
| 至山者 (深山小动物) 无损害 | 善根深厚者 (智慧资粮足) 得大利益 |
| 云聚无观待 (无 "害地狱 / 利人" 之想) | 佛智悲云**诸时无观待** (不分亲疏) |

**核心 anchor**: **佛无观待, 法雨平等; 众生随缘感果各异**。

**L37 重要立场** ——善知识亦无偏袒:

> "任何一个善知识出现在世间时, 从善知识的角度来讲, 可能没有分别'我一定要把相续当中的所有意趣传给某个人', 只要有缘, 对谁都很好。"

**教证** (L37, 《如来兴显经》):

> "譬如雨一味, 而悉遍蒙润, 如来亦如是, 行等无偏邪。"

**教证** (L37, 《释尊仪轨》——麦彭仁波切):

> "诸佛的慈悲没有远近, 谁祈祷都可以得到加持和悉地。"
>
> "何者作意佛, 佛安住彼前, 恒时赐加持, 解脱一切罪。"

**L37 应用立场**: 不必担心 "现在半夜三更, 佛会不会睡了, 我祈祷会不会打扰他"——**佛无所不在, 随时祈祷皆得加持**。此应用立场直接承 §2.4 广说任运 "千江有水千江月" 立场。

### 5.5 对应灭火之同法 (壬四, L37)

分三·癸: 能灭痛苦烈火之理 + 次第生起能灭智慧之理 + 依了知而修行之理。

#### 5.5.1 癸一·能灭痛苦烈火之理

**颂** (长颂):

> **生死无初始, 行彼道有五,**
> **不净无妙香, 五趣无安乐。**
> **彼苦恒如火, 刃盐等触生,**
> **悲云降妙法, 大雨能灭彼。**

**五道 (五趣)**: 地狱 / 饿鬼 / 旁生 (三恶趣) + 人 / 天 (二善趣, 阿修罗归天趣)。五道众生在轮回中**无安乐**, 如不净粪中无妙香。

**轮回三苦三喻**:

| # | 苦喻 | 义 |
|---|---|---|
| 1 | 如火焚烧 | 轮回苦不间断 |
| 2 | 如刀刃砍割 | 苦如利刃之伤 |
| 3 | 如伤口撒盐 | 苦苦 (雪上加霜) |

**义**: 佛大悲云降下妙法大雨, **若实修其义, 则能灭尽痛苦烈火**。

**朵洛瓦立场** (L37):

> "实修其义则能灭尽痛苦的烈火。"

**实修** 的强调——光知理论不修, 烈火不灭。

#### 5.5.2 癸二·次第生起能灭智慧之理

**颂** (两颂):

> **了悟天死堕, 人有求苦故,**
> **具慧亦不求, 天王人君尊。**
> **随从于智慧, 虔信于佛经,**
> **此苦此是因, 此灭智见故。**

**智慧生起的次第**:

1. **了悟天死堕、人有求苦** → 不希求**天王 / 人君尊**之果 (帝释 / 梵天 / 自在天 / 转轮王 / 国王 / 总统)
2. **随从宿慧 + 今生虔信佛经** → 证悟四谛:
   - 有漏果一切是**苦** (苦谛)
   - 业惑是苦因 (**集谛**)
   - 轮回因果灭尽是**灭谛** (涅槃)
   - 无我清净智慧现见彼 (**道谛**)

**教证** (L37, 《佛所行赞》):

> "诸天亦迁变, 人中岂得常?"

**教证** (L37, 《涅槃经》):

> "众生无明暗中行, 具受无边百种苦, 世尊能令远离之, 是故世称为大悲。"

#### 5.5.3 癸三·依了知而修行之理

**颂**:

> **当知疾病当断因, 当得乐住当依药,**
> **苦因灭彼如是道, 当知当断当证依。**

**四谛 = 医病四步**:

| # | 医病步骤 | 对应四谛 | 动作 |
|---|---|---|---|
| 1 | 知疾病 | 知**苦谛** | 当知 |
| 2 | 断病因 | 断**集谛** | 当断 |
| 3 | 得无病乐住 | 证**灭谛** | 当证 |
| 4 | 依妙药 | 依**道谛** | 当依 |

**四谛四字总摄**: 知 / 断 / 证 / 依。

**教证** (L37, 《佛说医喻经》): 病 → 病因 → 治疗方法 → 永愈, 对应佛所说四谛。

**教证** (L37, 真谛译):

> "苦如病, 集如病因, 灭如病愈, 道为治疗。"

**教证** (L37, 《教诫经》):

> "月可令热, 日可令冷, 佛说四谛, 不可令异。"

**真理不变立场**: 四谛是**客观规律**, 不因任何情形而变。

### 5.6 云喻 doctrinal 小结

- **智悲周遍**: 佛智悲云遍虚空, 无勤降法雨
- **同一法雨四面相**:
  1. **增长庄稼** (壬一): 善根成熟
  2. **器成** (壬二): 八圣道一味, 因根基成多种味 (三乘不同法)
  3. **无观待** (壬三): 法雨平等, 众生三种法器 (人 / 孔雀 / 饿鬼) 感受不同
  4. **灭火** (壬四): 四谛灭轮回烈火, 次第: 知苦 → 断集 → 证灭 → 依道
- **实修强调**: 光知不修, 烈火不灭
- **不观待立场**: 佛不作 "度谁不度谁" 的分别, 法雨无偏; 众生**自业感**决定受益多少

## 六、【编辑注】事业品的双层立场处理与 "任运" 正确理解

### 6.1 事业品的双层立场张力 (相对较轻)

事业品在觉囊读法下是佛功德的**外显**, 宁玛补遗较少——因为事业层面无重大 doctrinal 张力。觉囊派与宁玛派在事业品上基本共许:

- **任运**: 共许 (无勤起用)
- **不间断**: 共许 (事业永续)
- **九喻**: 共许 (经典教证)

不存在如 06 doc §7 "不空如来藏" 或 07 doc §1.5 "二清净" 层级的 doctrinal 张力。

### 6.2 主要 anchor——"任运" 的正确理解

事业品的**核心诠释重点**是**"任运"(无勤作) 的正确理解**——非无记空寂的无作, 而是**离戏智悲的无勤起用**。

#### 6.2.1 "任运" 的三种误读

| # | 误读 | 对治 |
|---|---|---|
| 1 | **造物主有分别勤作** | "佛陀不像造物主日日夜夜帮助众生" (L34) ——佛事业是法性自显, 非有神论 |
| 2 | **无记空寂 / 佛入灭即事业终结** | "事业不间断" —— 佛入灭后事业继续, 不因个体佛入灭而断 |
| 3 | **只要不努力就是任运** | 此为对空寂的误解——任运是**离戏智悲**的无勤自显, 不是无作无为 |

#### 6.2.2 正确立场——离戏智悲的无勤起用

**"任运" = 离戏大双运在利生层面的自显**。承 07 doc §1.5 "二清净" + 09 doc §1.8 "三身一体" + 10 doc §8.2 "64 功德 = 大双运 64 侧面" 立场, **任运**是大双运在 "作用面" 的自然显发:

- **非有勤作** (离增益): 不是造物主那样的有动机起用
- **非无作** (离损减): 不是无记空寂的死寂停滞
- **离戏大双运**: 智悲自显起用, 不住二边

**索达吉堪布 L34 关键立场**:

> "佛的境界我们确实不太懂, 如果真正是佛, 他有这样无为的智慧, 祈祷他的时候自然而然能得到加持。"

"无为" 不是 "不做事", 而是**非造作的智悲起用**——离戏本然的利生。

#### 6.2.3 "任运 = 缘起自显" 的 madhyamaka 对应

承 [`../../topics/madhyamaka/classifications.md §他空中观`](../../topics/madhyamaka/classifications.md) 立场, 任运即大双运在缘起面的自然起用:

- 佛方: 宿愿力 (因地所发)
- 众生方: 信心 + 清净心 + 业力
- **二缘和合 → 自然显发**

此立场避开义他空 "佛为实有主宰" 读法, 也避开断见 "佛入灭事业断" 读法。

### 6.3 双层立场张力点汇总表

| 位置 | 朵洛瓦 / 觉囊立场 | 索达吉堪布 / 宁玛调整 | 处理方式 |
|---|---|---|---|
| §2 任运不间断总颂 | "具足功德, 任运利生" 正面语汇 | 兼通: 任运 = 大双运智悲自显, 非 "实有佛主宰" | 本 doc 正文保留正面语汇; §6.2 集中辨析 |
| §3.3.4 "次第得究竟法身" | 外见色身 → 见内胜义法身 | 兼通: "内胜义法身" 非实有, 是离戏自性光明的显发 | 承 09 doc §1.7 显密三身立场 |
| §5.2 "住空定持海" | 佛智悲住空性境界, 成就无漏禅定如海功德 | 兼通: "住空" 非住于实有空性, 是大双运离戏 | 承 07 doc §1.6 二清净立场 |

### 6.4 事业品在修行实践上的意义

事业品对**修行实践**的直接 takeaway (L34-L37 贯通):

1. **祈祷不必 "节约"** (L34): 不担心佛太忙, 祈祷是缘起和合, 非打扰佛
2. **清净心是见佛关键** (L35): 心清净见佛色身, 心不清净见佛色身不现
3. **发大菩提心而非世间小愿** (L35): 见本尊时发菩提心, 不发 "升官发财" 等小愿
4. **闻法器条件** (L36): 无烦恼 + 具慧 + 具缘三者缺一不可
5. **四谛实修** (L37): 光知理论, 烈火不灭; 依道谛修习方能证灭谛

### 7.1 "佛外无涅槃 (究竟唯一乘)" (主 anchor)

九喻对三身的完整对应在 doc 12 (后六喻) 完成: 梵天 = 身语幻化, 日 = 智慧, 摩尼 = 意密, 回响 = 语密, 虚空 = 身密, 大地 = 大悲 —— 身密/语密/意密/大悲是密宗三身的核心侧面, 故九喻天然是显密三身津梁。

详参 09 doc §1.7 + [`../../topics/tathagatagarbha/index.md §Correctness Anchors "三身显密安立的层次差别"`](../../topics/tathagatagarbha/index.md#correctness-anchors)。

### 7.3 "不空如来藏 = 离戏大双运"

§2.9 "真佛法性故, 众生界如藏" (宝藏喻义) —— 佛照见众生具如来藏而摄受, 承 06 doc §3 "地下宝藏喻" (九喻之第 5) + 03 doc §3 "如来藏总纲" 立场; 此 doctrinal 立场是事业品与如来藏品的直接接合点——**佛事业之所以能成立, 根本在于众生本具如来藏**。

如来藏的 "不空" (具功德) 在事业品中以 "佛事业不间断地利生" 作最 pragmatic 的展开。

### 7.4 "二转三转所诠实相无二"

§5.2 "于有大智悲, 不染变不变, 住空定持海, 佛云善稼因" —— **智不住三有 (空分), 悲不滞涅槃 (光明分/现分)** —— 二转自空立场 + 三转光明立场在事业品的融通展开。

承 05 doc §5.3 二转三转双照立场在事业动态层面的落地——佛的事业之所以能 "任运不间断", 根本在于智悲双运、不住二边。

## 九、小结——本 doc 立下的关键论式

### 9.1 事业品整体定位

- 菩提品 (戊二) + 功德品 (戊三) 已立 "佛之体 + 功德"
- 事业品 (戊四) 立 "佛如何无勤利益众生" —— **功德之外显 (动态)**
- 核心问题: 佛入灭后事业如何继续 —— **任运不间断** 答此问

### 9.2 任运 + 不间断两法相

- **任运自成**: 无勤作的显发, 不分别却有所成; 五义 (化界 / 化方便 / 化事 / 彼境 / 应时)
- **不间断**: 事业不因佛入灭而断; 六因 (出离 / 显示 / 果 / 摄受 / 所断 / 能断缘) 次第 + 六喻对应 (大海 / 日轮 / 虚空 / 宝藏 / 云 / 风)
- 二法相合 —— 佛功德在无勤作的同时不间断地利益众生

### 9.3 九喻总纲 (retrieval 坐标)

九喻 = 身 (帝释 / 梵天 / 虚空) + 语 (天鼓 / 回响) + 意 (云 / 日 / 摩尼) + 大悲 (大地) 的完整身语意事业展开。本 doc 详讲前三 (天王 / 天鼓 / 云), 后六入 doc 12。

### 9.4 天王喻关键论式

- **情节**: 清净琉璃地显帝释天影, 人见彼行善升天
- **义**: 众生清净心显佛色身, 见者行六度成佛
- **二缘起**: 佛宿愿力 + 众生清净心和合而现
- **无生灭之理**: 色身随缘现没 (似有生灭), 法身无为不动 (实无生灭)
- **修行次第**: 外见色身 → 内见胜义法身

### 9.5 天鼓喻关键论式

- **情节**: 天鼓四无 (无勤作 / 无发音处 / 无意色 / 无分别念) 自然出四法印声
- **义**: 佛语四无, 周遍三乘有缘众生
- **四超胜**: 范围 (佛语遍大千 vs 天鼓仅天界) / 作用 (灭苦火 vs 增欲火) / 导向 (入定 vs 掉举) / 根本 (安乐之因唯佛语)
- **三神变**: 神境 (身) / 记说 (意) / 示导 (语) 全具
- **业所生立场**: 法不是佛单方给予, 是众生业力 + 佛愿力共同显现

### 9.6 云喻关键论式

云的**四同法**对应佛智悲四面相:

| # | 云的作用 | 佛智悲的对应 |
|---|---|---|
| 1 | 增长庄稼 | 长养众生善根 |
| 2 | 器成 (因地不同成多种味) | 八圣道一味, 因众生根基成三乘多种味 |
| 3 | 无观待 (不分亲疏) | 法雨平等, 众生三种法器 (人 / 孔雀 / 饿鬼) 自业感异 |
| 4 | 灭火 | 四谛 (知苦 / 断集 / 证灭 / 依道) 灭轮回烈火 |

**智不住三有 (空分), 悲不滞涅槃 (光明分)** —— 佛出离二边立场在事业品的落地。

### 9.7 "任运" 的正确理解 (编辑注要点)

"任运" = **离戏智悲的无勤起用**, 非无记空寂的无作, 亦非造物主式的有分别勤作。缘起自显 —— 佛宿愿 + 众生缘 → 无勤和合显发。此立场承 madhyamaka 他空中观 + tathagatagarbha 离戏大双运, 是事业品最重要的诠释 anchor。
