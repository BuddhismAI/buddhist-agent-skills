# 品09 ·《入不二法门品》前半：三十一菩萨逐一说入不二之门

## 一、本品的地位——全经顶点、承上启下

### 1.1 全经顶点定位

《入不二法门品》是《维摩诘经》的**结构性顶点**。堪布在第48课开篇即锚定其特殊地位：

> "《入不二法门品》非常精彩……很多前辈大德认为：**这一品之前都是趋入不二法门的阶梯**，就像密法中的共同加行和不共加行一样；**这一品则相当于密法的正行，直接讲远离二边的中道之法**。还有大德认为：第八品和第九品宣讲真实义，主要涉及自利；第八品侧重讲世俗法，这一品侧重讲胜义法。"

| 判法 | 品1-8 | 品9 |
|---|---|---|
| 加行/正行 | 共同/不共加行 (铺垫) | 密法正行 (远离二边直入) |
| 世俗/胜义 | 侧重世俗 | 侧重胜义 |
| 利他/自利 | 教义演示 | 自利证悟 (利他之前提) |

本品是**从言说到寂默**的结构拐点：品1 心净国土净 (能所地基) → 品2-4 案例破有住 → 品5-8 教义说破二 → **品9 默然显不二** → 品10-14 不二行世。无前八品铺垫，品9 默然即无着力处；无品9 顶点，前八品即成空谈。

### 1.2 何以名"不二"而非"不一"——破二即破四边

> "前辈注释中讲：**只要破了两个边，三边、四边、五边、六边等所有边都很容易破**……众生的执著多是二法——你和我、白和黑、上和下……如果破掉了这些二元对立的法，其他所有法都很容易破掉。"

**关键窍诀**：破二即破四边。众生最牢固的执著在二元对立；破二，三边 (二俱)、四边 (二非) 自然崩塌。

### 1.3 不存在实有的"不二"

> "是否存在一个实有的'不二'呢？并不存在。**因为'有'不存在，'无'不存在，有无二俱、有无二非都不存在，万法是远离四边八戏的**……'远离四边八戏'，也只是佛经或前辈大德为了表示破除一切戏论、一切边的说法而已，并非实有。"

**前置警告**：若将"不二"执为实有 (一种"超越二元的统一体")，已落第三边。**本品从头到尾在破二，但不立"不二"为实法**。品后文殊"无言"、维摩诘"默然"均破此重隐执。又引《文殊师利问经》"诸法无有二，亦复无有三，此中道具足，名为真实道"，并强调"'中'也不存在，名言上只能这么讲"。

### 1.4 与他经类似结构、闻法姿态

类似结构经典：**《楞严经》25 圆通**、**《圆觉经》12 菩萨问答**、莲花生大士 25 弟子窍诀、《印度八十四大成就者传》。共通教学 intent：以多个证量菩萨多角度讲说，摄受不同根基。

闻法姿态 (堪布)：**"观想自己在这些菩萨面前接受直指心性"**——非文字研讨，乃接受 33 位证量者直指。堪布严批"假装看虚空"的表演式开悟。

## 二、维摩诘启转轮

> 尔时，维摩诘谓众菩萨言："诸仁者！云何菩萨入不二法门？各随所乐说之。"

**以"讲考"为教学手段**——居士主动启问，令众菩萨各展所证。堪布建议读者自我对读："如果让你讲生灭不二，你会怎么讲？"

## 三、三十一菩萨 · 逐一列清

按经文顺序，每位含**所二之对 / reasoning / doctrinal breakpoint / 堪布与古德注**。

### 3.1 法自在菩萨 — 生灭不二

> "诸仁者！生灭为二。法本不生，今则无灭，得此无生法忍，是为入不二法门。"

**所二**：生 / 灭 (最根本时空二分)。**reasoning**：法本不生 → 既无生故无灭 → 得无生法忍。**breakpoint**：无生法忍贯穿加行/见道/八地各层次。**堪布注**：引《中论·观因缘品》"诸法不自生，亦不从他生，不共不无因，是故知无生"为根据，并引《中论》八不颂"不生亦不灭，不常亦不断，不一亦不异，不来亦不去"——**生灭不二即八不中道之首**。本位是 31 段起点，破最根本的执。

### 3.2 德守菩萨 — 我我所不二

> "我我所为二。因有我故，便有我所；若无有我，则无我所。"

**所二**：我 / 我所。**reasoning**：linear 推导——我所依我而立，破我即破我所。**breakpoint**：与华严菩萨 §3.26"从我起二"形成呼应 (华严是 root level；德守是 linear reach)。**堪布注**：引《入中论》"最初说我而执我，次言我所则著法，如水车转无自在，缘生兴悲我敬礼"——**轮回根在我执起我所执**。又引《大宝积经》"一切内外法，我已知空幻，愚夫颠倒执，分别我我所"。

### 3.3 不眴菩萨 — 受不受不二

> "受不受为二。若法不受，则不可得；以不可得，故无取无舍，无作无行。"

**所二**：受 (取五蕴流转) / 不受 (五蕴寂灭涅槃)。**reasoning**：四重递进——不受→不可得→无取舍→无作行。**breakpoint**：染法/净法的 paradigm 性破，为 §3.13 善意"生死涅槃不二"的雏形。**堪布注**："不眴"即不眨眼——他常观佛陀，慧眼永不闭合。藏/英/日译皆作"不眨眼睛的菩萨"。南怀瑾以鱼喻 (昼夜精进、慧眼不闭)；汉地木鱼即此意。

### 3.4 德顶菩萨 — 垢净不二

> "垢净为二。见垢实性，则无净相，顺于灭相。"

**所二**：垢 / 净。**reasoning**：互待破立——见垢实性则无净相 → 顺灭相。**breakpoint**：以《心经》"不垢不净"为同层教证。**堪布注**："德顶"获最顶层实相禅定之功德。**密法关联**："密法中以享用五肉五甘露等禁行来增上修行"——正是因证垢净无二；但堪布立刻警示"一般人对净与不净还是有执著的"。**梵/藏本与汉译此位顺序略异**，但菩萨数无差。

### 3.5 善宿菩萨 — 动念不二

> "是动是念为二。不动则无念，无念则无分别。"

**所二**：动 / 念 (能所二取的最细层)。**reasoning**：不动→无念→无分别。**古德三解**：(a) 动=所取念头，念=能取；(b) 动=细微散动，念=粗意念；(c) 动=外散，念=意念——共通"动为念因"。**禅宗关联**：**《六祖坛经》风幡公案**——"既不是风动，也不是幡动，而是我们的心在动"，六祖随后讲"甚深不二法门，即常、断、罪、福等一切法都是无二"。**大手印教法**：心安住/动念/明觉三方面认识本性，初期分开，最后达动念无二之境。

### 3.6 善眼菩萨 — 一相无相不二

> "一相无相为二。若知一相即是无相，亦不取无相，入于平等。"

**所二**：一相 (有相) / 无相。**reasoning**：一相 = 无相 → **亦不取无相** → 入平等。**breakpoint**：本条首次出现"亦不取无相"的更深一层破——若执无相则无相成新边。**堪布注**：引《金刚经》"凡所有相皆是虚妄"、"离一切诸相则名诸佛"；又引《中观庄严论释》引《三摩地王经》"所谓有无为二边，净与不净亦是边，**是故尽断二边后，智者亦不住中间**"——"不住中间"即"亦不取无相"，是本品诸位菩萨的底层 pattern。**堪布长篇展开"执著令人脆弱"**——以现实中因执一句话而自杀、父母骂"该死"孩子跳楼、骑摩托车意外父母相继撞墙等悲剧为例："**没学过不二法门的人真的特别可怜**。"

### 3.7 妙臂菩萨 — 菩萨心声闻心不二

> "菩萨心、声闻心为二。观心相空，如幻化者，无菩萨心，无声闻心。"

**所二**：菩萨心 / 声闻心。**reasoning**：观心相空如幻化 → 无菩萨/声闻心。**breakpoint**：**乘法的自我否定**——连修行所建立的"菩萨心"也要空掉，否则成新执。**堪布注**："妙臂"喻通达二谛之二智 (双手喻二谛)。**异文**：藏/梵本中无"空"字而作"心相如幻化"，实义同。引《金刚经》"过去/现在/未来心不可得"，及来源/住处/去处/颜色/形状多角度破心。

### 3.8 弗沙菩萨 — 善不善不二

> "善不善为二。若不起善不善，入无相际而通达者。"

**所二**：善 / 不善。**reasoning**：不起善不善 → 入无相际。**breakpoint**：道德层 doctrinally 极敏感——易被误读为否认因果。**堪布注**："弗沙"为 28 星宿中最吉祥者；佛陀成道降生时此星出现。**核心展开 (本品最重警示)**：**禅宗"不思善不思恶"经证 + 百丈野狐公案**——六祖与惠明"不思善不思恶，正与么时，那个是明上座本来面目"是禅宗最直接经证；但《五灯会元》百丈公案中老人前世法师答"不落因果"五百世堕野狐身，百丈以"不昧因果"纠正。**堪布重申**："**如果稍微听到一点实相法、无二法，就认为因果不存在、一切都是空无的，并不合理**"——以"是故见比虚空高，取舍因果较粉细"收束。"不起善不善"是证量现前，非凡夫借口放任。

### 3.9 师子菩萨 — 罪福不二

> "罪福为二。若达罪性，则与福无异，以金刚慧决了此相，无缚无解。"

**所二**：罪 / 福。**reasoning**：达罪性 (本空) → 罪福无异 → 金刚慧决了 → 无缚无解。**breakpoint**：**金刚慧** = 通达实相的空性智慧；金刚从山顶坠落直达金刚大地无物可挡，无一法可经其观察。**堪布注**："师子"无所畏惧。引《观普贤菩萨行法经》"**一切业障海，皆从妄想生，若欲忏悔者，端坐念实相，众罪如霜露，慧日能消除**"——汉地实相念佛/实相忏悔的经证；又引永嘉《证道歌》"无罪福无损，寂灭性中莫问觅"。**修法应用**：金刚萨埵忏悔可安住此境界。罪福不二 ≠ 不忏悔，而是**实相摄持的忏悔**。

### 3.10 师子意菩萨 — 有漏无漏不二

> "有漏无漏为二。若得诸法等，则不起漏不漏想，不著于相，亦不住无相。"

**所二**：有漏 / 无漏。**reasoning**：得诸法等 → 不起漏想 → **双遣** (不著相 + 不住无相)。**breakpoint**：**为何也不住无相？因无相也是一种相**——与 §3.6 善眼"亦不取无相"形成经内 echo。**堪布注**：有漏法 = 与有漏六门 (漏自性/相属/所缚/相续/随顺/所因) 相连之法。**藏文"有漏"作岗萨即补特伽罗**。堪布戏言："有些人听课一点都记不住，也可作有漏的比喻。"引智者《金光明经文句》三过患心 (散心/覆器/障闻慧 · 忘心/漏器/障思慧 · 倒心/秽器/障修慧)，与藏传《大圆满前行》闻法轨理三过 patterns 一致。

### 3.11 净解菩萨 — 有为无为不二

> "有为无为为二。若离一切数，则心如虚空，以清净慧无所碍者。"

**所二**：有为 / 无为。**reasoning**：离一切数 → 心如虚空 → 清净慧无所碍。**breakpoint**：**"数"指数目**——以清净智慧观察则一/二/三/四乃至百千万之数皆不存在；有为无为同属"可数"范畴。**堪布注**："净解"清净胜解；有译本作"信解菩萨"。藏译此对作"快乐/不快乐"——实义同 (无为不带痛苦，有为执破时令人伤心)。引《中论》"**生住灭不成，故无有有为，有为法无故，何得有无为？**"——本对最经典中观 reasoning。

### 3.12 那罗延菩萨 — 世间出世间不二

> "世间出世间为二。世间性空，即是出世间。于其中不入不出，不溢不散。"

**所二**：世间 / 出世间。**reasoning**：世间性空 = 出世间 → 不入不出 + 不溢不散。**breakpoint**：**"世间性空即是出世间"** 是本品关键句法公式——**染法的本性就是净法**，不需离染另寻净。echo 品7"淫怒痴性即是解脱"、品8"一切烦恼为如来种"，本品再以更精简形式确立。**堪布注**："那罗延"金刚大力士，不被违缘动摇；亦勇坚三摩地。引《守护国界主陀罗尼经》"世及出世法，此二亦平等"。藏译"不入不出"作"不入不出，不来不去"。

### 3.13 善意菩萨 — 生死涅槃不二

> "生死涅槃为二。若见生死性，则无生死，无缚无解，不生不灭。"

**所二**：生死 / 涅槃。**reasoning**：见生死性 → 无生死 + 无缚无解 + 不生不灭。**breakpoint**：**本对是 31 中 doctrinally 最核心**——"生死/涅槃不二"是大乘对二乘寂灭观的终极超越。"心的本体也称为涅槃，并非人死以后才叫涅槃。"**堪布注**："善意"有译本作"调伏心识的菩萨"。引《华严经》"**生死及涅槃，二皆不可得，虚诳妄说者，生死涅槃异**"。**与《中论》"涅槃与世间，无有少分别；世间与涅槃，亦无少分别"** (观涅槃品) 同一命题——生死涅槃不二是中观宗根本标志。

### 3.14 现见菩萨 — 尽不尽不二

> "尽不尽为二。法若究竟尽，若不尽，皆是无尽相。无尽相即是空，空则无有尽不尽相。"

**所二**：尽 (无常有为可灭尽) / 不尽 (无为常住)。**reasoning**：尽 + 不尽 皆无尽相 → 无尽相即空 → 空则无尽不尽相。**breakpoint**：**双破**——"烦恼本体如石女儿，根本无法灭尽；但说'烦恼不尽'也不对"。**堪布注**："现见"现量见诸法本性。**僧肇注**："有为虚伪法，无常故名尽；实相无为法，常住故不尽。"**藏/唐译异文**：作"无尽相即刹那性，刹那性无有尽与不尽相"——空性 = 刹那性 (刹那灭即非实有)。**关键分辨**：此处"无尽相即是空"——**空非单空而是大空性** (堪布明说)；单空执则落一边。

### 3.15 普守菩萨 — 我无我不二

> "我无我为二。我尚不可得，非我何可得？见我实性者，不复起二。"

**所二**：我 / 无我。**reasoning**：我不可得 → 非我更不可得。**breakpoint**：**"我尚不可得，非我何可得？"** ——破我同时自动破无我；无我是观待我而立的否定相。与 §3.2 德守不同——德守破 linear (我→我所)；普守破 polar (我↔无我)。**堪布注**："普守"既守实相又守菩提心。**核心教证**：**真谛译《宝行王正论》**"我无我二义，如实捡不得，是故如来遮，我无我二边"。**堪布发现**：《宝行王正论》实即《中观宝鬘论》(真谛 5 世纪南北朝译本)，古词如"增上生"作"乐因法"、"决定胜"作"解脱法"——为该论古汉译版本首次锁定。

### 3.16 电天菩萨 — 明无明不二

> "明无明为二。无明实性即是明，明亦不可取，离一切数，于其中平等无二者。"

**所二**：明 (智慧) / 无明 (十二缘起首支)。**reasoning**：无明实性 = 明 → 明亦不可取 → 离一切数 (十二支皆离) → 平等无二。**breakpoint**：**"无明实性即是明"** echoes §3.12 那罗延"世间性空即是出世间"；进一步破明 → 离十二支。**堪布注**："电天"是三摩地名 (电天三昧)；《维摩经略疏》"观第一义天发电天三昧"。**核心教证**：**《有德女所问大乘经》**——有德女问佛"无明在内或在外"，佛皆答否；"无明自性从虚妄分别生，非真实生；从颠倒生，非如理生"；有德女推论"无明无自性故行等定不可得"，佛证"万法毕竟空，凡夫迷倒造业受苦；实相中诸业不存，我随顺世间宣说"。

### 3.17 喜见菩萨 — 色色空不二 (五蕴空)

> "色色空为二。色即是空，非色灭空，色性自空；如是受想行；识识空为二，识即是空，非识灭空，识性自空。"

**所二**：五蕴 (色受想行识) / 五蕴空 (即《心经》"色即是空"命题)。**reasoning**：色即是空 ≠ 色灭空 → 色性自空。受想行识同。**breakpoint**：**《心经》locus classicus 之一**——必须明辨**自空 vs 他空**：色不是灭了才空，**色本身性质就是空** = 自空；他空 = 色灭后变成空。"我们正在看到的、听到的……六根前显现的六境，**当下就是空的**，只是我们现在还未通达此理。"**堪布注**："喜见"——一切众生都喜欢见的菩萨。指出《杂阿含经》讲五蕴空时也只讲色与识，中间受想行略——**印度佛教的特殊做法**。引《入楞伽经》"色与空无异，无生亦复然，不应执为异，成诸见过失"。

### 3.18 明相菩萨 — 四大空大不二

> "四种异、空种异为二。四种性即是空种性，如前际后际空，故中际亦空。"

**所二**：地水火风四大 / 空大。**reasoning**：四大性 = 空大性 → 前际空 + 后际空 → 中际亦空。**breakpoint**：**三时证空 reasoning**——以前后际类比推出中际，非直接破当下。**堪布注**："明相"通达一切相之实相。**窥基精细判 (《说无垢称经疏》)**："**初有十六。菩萨以二为二。后有十五。以非二为二**"——前 16 位以**真正成对的二法**为二 (生灭/我我所/垢净/善不善/罪福/生死涅槃等天然二分)；**后 15 位以本来非二之法**为二 (如四大与虚空本非成对，凡夫强执)。本位即此分界点。

### 3.19 妙意菩萨 — 根境不二 (六根六境)

> "眼色为二。若知眼性，于色不贪不恚不痴，是名寂灭。如是耳声、鼻香、舌味、身触、意法为二，若知意性，于法不贪不恚不痴，是名寂灭。"

**所二**：六根 (眼/耳/鼻/舌/身/意) 与六境 (色/声/香/味/触/法)，共 6 对。**reasoning**：知眼性 (本空) → 于色不生贪恚痴 → 寂灭。其他五对同理。**breakpoint**：按《俱舍论》与因明，五根识无分别念，到意识才有；但通达根性空，意识也不生贪恚痴。**堪布注**：以自身《俱舍论》戏言："我以前学过的《俱舍论》现在已经跑到虚空中去了，全部入于不二法门了。如果你们也是这样，可以很开心地说：'我已经证悟《维摩诘经》中的不二法门了！'"（众笑）。《杂阿含经》以人无我抉择空性，本处以法无我抉择，层次更深。

### 3.20 无尽意菩萨 — 六度回向不二

> "布施、回向一切智为二，布施性即是回向一切智性；如是持戒忍辱精进禅定；智慧、回向一切智为二，智慧性即是回向一切智性。"

**所二**：六度各与"回向一切智"为二，共 6 对。**reasoning**：布施性 = 回向一切智性 → 入一相。余五度同。**breakpoint**：**因法 (六度) / 果法 (回向)** 本为能/所回向之二分；以空性融——因果同体，无别别因果。**堪布注**："无尽意"具无尽智慧。推荐《无尽意菩萨经》"内容非常殊胜"。引《入中论》"此中犹如已觉位，乃至未觉三皆有"——名言中三者存在，胜义中皆空。**实修启示**："终生学修之法"——每人应找一两部最重要的法终生研修 (喻如密法息增怀诛四事业可依本尊或最有信心法本)。

### 3.21 深慧菩萨 — 三解脱门不二

> "是空、是无相、是无作为二。空即无相，无相即无作。若空无相无作，则无心意识，于一解脱门即是三解脱门者。"

**所二**：空 (体) / 无相 (因) / 无作 (果) 三解脱门互为二。**reasoning**：空 = 无相 = 无作 → 无心意识 → 一即三。**breakpoint**：**三解脱门同一本体**——"三个人通过不同门到达同一城市"。**堪布注**："深慧"智甚深难测。**展开心/意/识三分**：心 = 8 心王 + 51 心所 (大海)、意 = 染污末那 (波涛)、识 = 六识 (浪花)；阿赖耶在底，意在中，六识在表。**本品窍诀**："**每位菩萨讲的都是破二边之法，并未讲破第三边和第四边，这是一种窍诀**"——破二即破四，最经济直接。

### 3.22 寂根菩萨 — 三宝不二 (佛法僧)

> "佛法众为二。佛即是法，法即是众，是三宝皆无为相，与虚空等，一切法亦尔。"

**所二**：佛 / 法 / 僧 (三者互非二)。**reasoning**：佛 = 法 + 法 = 僧 → 三宝皆无为相 = 虚空 → 一切法亦然。**breakpoint**：**究竟三宝无为相**——佛无相 (《宝性论》)、法无为、僧无修，故三者一体。**堪布注**："寂根"六根清净。慧远："若能观三宝常住同真谛。"**世俗皈依不否定**——堪布特别提醒"念诵'皈依佛/法/僧'确实能遣邪魔；遇艰难或恶梦念 108 遍四皈依能遣违缘"；《随念三宝经》仍教世俗无欺功德——**实相三宝不否定世俗三宝**。

### 3.23 心无碍菩萨 — 身身灭不二

> "身身灭为二。身即是身灭。所以者何？见身实相者，不起见身及见灭身，身与灭身无二无分别，于其中不惊不惧者。"

**所二**：身 (存在时) / 身灭 (死后)。**reasoning**：身即是身灭 (因刹那性) → 见身实相者不起二见 → 不惊不惧。**breakpoint**：**"身即是身灭"** ——假合身正显现时其本体即灭法；非死后才空 = 自空 (再次出现，echo §3.17)。**堪布注**："心无碍"通达二谛/三谛 (智者注)。**异文**：梵/藏/日本作**"不生不灭"**；汉鸠摩罗什本及汉地古德注作**"不惊不惧"**——实义可通；布达拉宫 1999 年发现的梵文本与藏/日译一致。引《入行论》"如是身若无，岂有男女相？"；"众缘聚合时，见石状似人，如是于手等，亦见实有身"——身如人形石/稻草人/假人模特，是假合。

### 3.24 上善菩萨 — 三业无作不二

> "身口意善为二。是三业皆无作相，身无作相，即口无作相，口无作相，即意无作相。是三业无作相，即一切法无作相。"

**所二**：身/口/意三业互为二。**reasoning**：业无自性故无作相 → 身=口=意无作相 → 一切法无作相。**breakpoint**：**业无自性故无作相**——三业虽名言有善/不善/不动，实相中皆无作相；扩展到一切法。**堪布注**："上善"善调顺菩萨；心易调伏。**异文**：梵/藏/唐本作"律"或"戒"、"净行"；几个版本作"善"。堪布认为**作"律"较好解释**——身口意业实即戒律/净行 (不限于善业)，但不必改动汉译。

### 3.25 福田菩萨 — 三行空性不二 (福行/罪行/不动行)

> "福行、罪行、不动行为二。三行实性即是空，空则无福行，无罪行，无不动行。"

**所二**：福行 (欲界善) / 罪行 (欲界不善) / 不动行 (色无色界业)。**reasoning**：三行实性空 → 三行皆无 → 不起分别。**breakpoint**：**引入十二因缘"行"支的三分**——"最究竟的不动业是色界第四禅以上的业，因为四禅以下还有欢喜、勤作等细微的动摇"。与 §3.8 弗沙补充 (弗沙破二，此位破三)。**堪布注**："福田"——众生之良田。本段长篇展开**"行为如法"**——反对密宗名义吃肉喝酒抽烟、反对瑜伽士"半出/半在"混搭形象——"信心比一般的玻璃更易碎"，《入行论》"犹如乌云暗夜中，刹那闪电极明亮"——摧毁他人信心过失极大。

### 3.26 华严菩萨 — 从我起二 (一切二的 root)

> "从我起二为二。见我实相者，不起二法。若不住二法，则无有识。无所识者。"

**所二**：**从"我"起的一切二** (非具体某对，而是所有二对的 root)。**reasoning**：见我实相 → 不起二 → 无我识 → 入不二门。**breakpoint**：**与德守 §3.2 截然不同**——德守破"我/我所"具体一对；华严破"**因我而起的一切二**"——我执是一切二元对立的根源；破我即破一切二。**本条在 31 菩萨中承担"总纲"地位**。**堪布注**："华严"花之庄严。**教证**：**《维摩经略疏》"我性即佛性"** ——见"我"本性即见佛性如来藏。**堪布长篇展开"人际烦恼之源"**——我执炽盛 + 过度敏感 + 心胸狭窄 + 视野狭小 = 现代人际烦恼四根；"很多发心部门只有四五个人却分成几个团体"，"造成这种情况的根本原因，就是我们今天讲的'从我起二为二法'"。**本条是本品最具时代针对性的实修段**——以"从我起二"作诊断工具看待人际烦恼。

### 3.27 德藏菩萨 — 有所得相不二

> "有所得相为二。若无所得，则无取舍。无取舍者。"

**所二**：有所得 (内"我") / 相 (外所缘相)。**reasoning**：无所得 → 无取舍。**breakpoint**：**得 = 内得 (能取"我"); 相 = 外相 (所取境)** ——能所二取最本质表达。《维摩经义记》"若有所缘对心说二，所缘既无不可将之对心说二，故曰不二"。**堪布注**："德藏"证德 + 断德 + 恩德之宝藏。引《海意菩萨所问净印法门经》"若法无所得，彼即无分别，佛及诸佛法，一切法皆然"——**无所得故无分别**。

### 3.28 月上菩萨 — 明暗不二

> "闇与明为二。无闇无明，则无有二。所以者何？如入灭受想定，无闇无明，一切法相亦复如是。"

**所二**：暗 / 明 (物理或喻无明 vs 智慧)。**reasoning**：无暗无明 → 以灭尽定为喻 → 一切法相亦然 → 平等入。**breakpoint**：**以灭尽定为喻**——入灭受想定者受想已灭、只余极微细心识；无明暗分别。一切法相亦如灭尽定境。古大德解为二乘灭尽定 (《维摩经义疏》)。**堪布注**："月上"如空中月投影于每个水器，菩萨功德在每众生心水器中呈现。《说无垢称经疏》"了本真理，则无明暗"。**中观推理普适性**：堪布指出"**很多西方学者从小就喜欢逻辑推理，他们不一定能接受大圆满直指法而安住不二境界，但可按《中观根本慧论》观察方法推理，得出明暗无二的结论**"——对此品 reasoning 方式的方法论判。

### 3.29 宝印手菩萨 — 乐涅槃厌世间不二

> "乐涅槃、不乐世间为二。若不乐涅槃，不厌世间，则无有二。所以者何？若有缚，则有解。若本无缚，其谁求解？"

**所二**：乐涅槃 / 厌世间 (修行情感取舍二分)。**reasoning**：不乐涅槃 + 不厌世间 → 若本无缚谁求解 → 无缚无解 → 无乐厌。**breakpoint**：**"不住涅槃、不住生死"** = 大乘菩萨行根本特征——小乘乐涅槃厌生死；大乘不住两边。与 §3.13 善意 (理论上不二) 不同，本条偏**情感取舍**层。**堪布注**："宝印手"印持万法实相永不改变。**"厌离世间之歧途"专段**：堪布明言"**对涅槃的欢喜和对轮回的厌离都是边**——作为初学者理应远离人群寻寂静处为修行创造顺缘，但从大菩萨行境来讲，这也是一种歧途。真正菩萨不会如此厌烦世间，他们只要能利益众生，再复杂、再混乱的环境甚至地狱界、饿鬼界都愿意去。" 引吉藏《维摩经义疏》引《华严经》"**生死非杂乱，云何而厌？涅槃非寂静，云何而乐？**"；龙猛《中论·观缚解品》"缚者无有解，不缚亦无解，缚时有解者，缚解则一时"——专破缚解实有；"涅槃无有生，轮回亦无灭"。

### 3.30 珠顶王菩萨 — 正道邪道不二

> "正道邪道为二。住正道者，则不分别是邪是正，离此二者。"

**所二**：正道 / 邪道 (八正道 / 八邪道)。**reasoning**：住正道者不分别正邪 → 离二。**breakpoint**：**"住正道者则不分别正邪"** ——本条承担禅宗核心命题的经证地位。**堪布注**："珠顶王"虽未必实有如意珠顶饰，但能满众生愿。**禅宗公案**：**紫柏禅师与张拙《见道偈》公案**——紫柏觉"断除烦恼重增病，趣向真如亦是邪"不对，改为"断除妄想方无病，趣向真如不是邪"；僧告"你错他不错"；紫柏后某日吃斋饭顿悟正邪无二。引**《六祖坛经》"邪正俱不用，清净至无余"**。堪布补："对于刚接触大乘佛法的人来说，禅宗的见解和《维摩诘经》中的不二法门确实有点高……就像刚来到海拔四千米的藏地高原，有点高原反应，迷迷糊糊的"——破执不能急。

### 3.31 乐实菩萨 — 实不实不二

> "实不实为二。实见者尚不见实，何况非实！所以者何？非肉眼所见，慧眼乃能见，而此慧眼，无见无不见。"

**所二**：实 (实有) / 不实 (非实有)。**reasoning**：实见者尚不见实 → 何况非实 → 慧眼无见无不见。**breakpoint**：**31 位最末，也是最高级的破**——连"实"都无可见、何况"非实"。慧眼"无见无不见"——双遣式 (无可见实有，但也无不可照见)。本条破实又破非实，破有见又破无见——接近文殊"无言"，是 31 位到文殊间的桥梁。**堪布注**："乐实"喜实相法门。**核心教证**：《中论》**"一切实非实，亦实亦非实，非实非非实，是名诸佛法"** ——四句判。**堪布展开四边渐破的三种根基与三种修法**：
- **初学者** (单空见)：万法无实
- **中等根基** (二谛)：通达名言有/胜义无
- **上等根基** (离四边)：万法非实非非实，二俱不存非二俱亦不存——离四边八戏的实相胜义
**修法分判**：渐次根基阶梯式破 → 应成派直接破一切边 → 密法根基依证悟上师表示方法**直指心性当下认识**。**乐实菩萨作为 31 位之末，正好收束整个"破二"序列，迎接文殊"无言"与维摩诘"默然"的更高层次**。

| 类别 | 菩萨 (#) | 共通 |
|---|---|---|

| **道德/业果类** | 弗沙(8) 师子(9) 上善(24) 福田(25) | 道德范畴二分；最易被误读为否认因果，堪布反复强调因果不坏 |

| **染净/烦恼智慧类** | 德顶(4) 师子意(10) 电天(16) | 烦恼 vs 清净；echo 品8"一切烦恼为如来种" |
| **世间/涅槃类** (大乘根本立场) | 那罗延(12) 善意(13) 宝印手(29) | 大乘对小乘寂灭观的超越；ontology + ethics 互补 |
| **修道法相类** | 无尽意(20) 深慧(21) 寂根(22) 珠顶王(30) | 破修行所立诸法相 (六度/三解脱门/三宝/正邪) |

**总判**：8 + 4 + 6 + 3 + 3 + 3 + 4 = 31。涵盖凡夫一切二元执著的全面遍覆——本体论 / 道德论 / 认识论 / 我执根本 / 烦恼智慧 / 世出世间 / 修道法相。

**窥基精细判**：前 16 位以**真正成对二法**为二 (天然二分)；后 15 位以**本来非二之法**为二 (强执非二为二)。

## 五、共通 reasoning pattern ——一切 31 段的骨架

### 5.1 三步骨架

```
Step 1 — 设二:    A 和 B 为二法 (世俗显现中二法存在)
Step 2 — 观本无: A 之本性即是 B / A 自性空 / A 不可得
Step 3 — 由无分别即不二门: 故不起 A/B 分别 → 不住相 → 入不二法门
```

### 5.2 深化 sub-patterns

- **Sub-pattern A · 双遣式** (既破 A 又破 ¬A，不住中间)：§3.6 善眼"亦不取无相"；§3.10 师子意"亦不住无相"；§3.15 普守"非我何可得"；§3.31 乐实"无见无不见"。
- **Sub-pattern B · 同性合一式** (A 之性即是 B)：§3.12 那罗延"世间性空即出世间"；§3.16 电天"无明实性即明"；§3.17 喜见"色性自空"；§3.20 无尽意"布施性即回向一切智性"。
- **Sub-pattern C · 互待破立式** (二者互待故破一即破另)：§3.2 德守"无我则无我所"；§3.4 德顶"见垢实性则无净相"；§3.8 弗沙"不起善不善"；§3.29 宝印手"本无缚则无解"。
- **Sub-pattern D · 三时证空式** (前后际空故中际亦空)：§3.18 明相。
- **Sub-pattern E · root-cause 式** (破一根则破一切分支)：§3.26 华严"从我起二"。

### 5.3 为何贯穿——reasoning 的 universality

凡真正的"不二" 都只能通过**设二 → 观本无 → 破二归一 (并破一)** 三步达成。31 段本质上是同一 reasoning 的 31 个应用例。**这就是为何 31 段可以"各说不二"却最终指向同一实相**——reasoning 同构，对象各异。

## 六、堪布 Nyingma 视角

### 6.1 本品与中观应成派的对应

堪布在多处显式打通本品与应成派破法：

- §3.1 法自在 → 《中论》"诸法不自生"四句破
- §3.11 净解 → 《中论》"生住灭不成"破有为无为
- §3.13 善意 → 《中论》"涅槃无有生，轮回亦无灭"
- §3.29 宝印手 → 《中论·观缚解品》"缚者无有解，不缚亦无解"
- §3.31 乐实 → 《中论》"一切实非实，亦实亦非实"四句判

**总判**：本品"二而不二"正是中观应成派"破一切二执而不安立任何边"的经证。**应成派"无承认"** (对论敌不立自方命题) 与本品 31 菩萨的"设二 → 观本无 → 入不二门" 结构同构——**以破为立，不立新的实法**。

### 6.2 与大圆满"离分别即解脱"的对应 + 三种根基修法

§3.28 月上段堪布明示**两种到达不二的路径**：(1) 中观推理路径：以逻辑推理 + 空性抉择阶梯式破；(2) 大圆满直指路径：依证悟上师表示方法当下认识心本来面目。

§3.31 乐实段进一步明示**三种根基的三种修法**：渐次根基次第破 / 应成派直接破一切 / 密法根基直指心性——殊途同归于不二。本品 31 菩萨之说属前两种；维摩诘默然 (品9 后半) 接近第三种"直指"。

**严格界定**：本经未入密法修法语境 (无生圆二次第、灌顶、气脉明点)；堪布仅作教义旁证。**Cross-link (教义同源印证)**：`topics/tantra/index.md` §等净无二。

### 6.3 禁行与证量的对应

§3.4 德顶段堪布点出"密法中以享用五肉五甘露等禁行来增上修行——正是因证知垢净无二"，但紧接警示。**与品8 "圣者行三要"** (因断贪 + 体断不净 + 果饶益) 同层警戒——三要不具足即凡夫行；不得借"不二"为掩护滥行染法。

## 九、关键经文摘录

### 9.1 维摩诘启问

> "诸仁者！云何菩萨入不二法门？各随所乐说之。"

### 9.2 31 位菩萨核心句 (逐一)

1. **法自在**：法本不生，今则无灭，得此无生法忍。
2. **德守**：因有我故，便有我所；若无有我，则无我所。
3. **不眴**：若法不受，则不可得；以不可得，故无取无舍，无作无行。
4. **德顶**：见垢实性，则无净相，顺于灭相。
5. **善宿**：不动则无念，无念则无分别。
6. **善眼**：若知一相即是无相，亦不取无相，入于平等。
7. **妙臂**：观心相空，如幻化者，无菩萨心，无声闻心。
8. **弗沙**：若不起善不善，入无相际而通达者。
9. **师子**：若达罪性，则与福无异，以金刚慧决了此相，无缚无解。
10. **师子意**：不起漏不漏想，不著于相，亦不住无相。
11. **净解**：若离一切数，则心如虚空，以清净慧无所碍者。
12. **那罗延**：世间性空，即是出世间。不入不出，不溢不散。
13. **善意**：若见生死性，则无生死，无缚无解，不生不灭。
14. **现见**：法若究竟尽，若不尽，皆是无尽相。无尽相即是空。
15. **普守**：我尚不可得，非我何可得？
16. **电天**：无明实性即是明，明亦不可取，离一切数，平等无二。
17. **喜见**：色即是空，非色灭空，色性自空。
18. **明相**：四种性即是空种性，前际后际空，故中际亦空。
19. **妙意**：若知眼性，于色不贪不恚不痴，是名寂灭。
20. **无尽意**：布施性即是回向一切智性。
21. **深慧**：空即无相，无相即无作。
22. **寂根**：佛即是法，法即是众，是三宝皆无为相，与虚空等。
23. **心无碍**：身即是身灭。
24. **上善**：是三业无作相，即一切法无作相。
25. **福田**：三行实性即是空，空则无福行，无罪行，无不动行。
26. **华严**：从我起二为二。见我实相者，不起二法。
27. **德藏**：若无所得，则无取舍。
28. **月上**：如入灭受想定，无闇无明，一切法相亦复如是。
29. **宝印手**：若有缚，则有解。若本无缚，其谁求解？
30. **珠顶王**：住正道者，则不分别是邪是正。
31. **乐实**：实见者尚不见实，何况非实。慧眼无见无不见。

## 十、实修要点

1. **"破二即破四边"是本品核心窍诀**——破二即全破。
2. **"不二"非新实法**——若执"不二"为实有已落第三边；本品破二不立不二。
3. **自空 vs 他空辨别**——§3.17 喜见、§3.23 心无碍：当下即空，非灭后才空。
4. **三根基三修法** (§3.31)——渐次次第破 / 应成派直接破一切 / 密法根基直指心性。
5. **闻法姿态**——观想自己在 31 菩萨面前接受直指心性，非文字研讨。
6. **不得以"不二"为借口否认因果**——§3.8 弗沙 + 百丈野狐：见比虚空高，取舍因果较粉细；"不起善不善"是证量现前而非凡夫言说。
7. **"从我起二"为一切二的 root**——§3.26 华严 + 人际烦恼四根 (我执炽盛 / 过度敏感 / 心胸狭窄 / 视野狭小)；破我即破一切二元烦恼。
8. **禁行与证量的严格对应**——§3.4 德顶警示：密法禁行依证量垢净无二而立，凡夫不得滥用；"口头不二内心无定解"是最危险的修行歧路。
9. **世俗三宝不否定**——§3.22 寂根：实相三宝无相，但世俗皈依功德真实不虚。
10. **"不住涅槃不住生死"**——§3.13 善意 + §3.29 宝印手：大乘菩萨行根本标志；初学可寻寂静处，大菩萨不应以此为终极。
11. **"一切实非实，亦实亦非实，非实非非实"** 四句判 (§3.31) 为渐次根基最佳修法。
12. **31 段同一 reasoning pattern**——设二 → 观本无 → 入不二门；可以任一二对为入处体会，应用于日常任何二元执著。
