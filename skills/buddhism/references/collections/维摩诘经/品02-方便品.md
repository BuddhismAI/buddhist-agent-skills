# 品02 ·《方便品》：维摩诘居士登场与因病说法

《方便品》是《维摩诘经》全经叙事真正启动的一品。前一品《佛国品》以"心净则国土净"奠定本体论地基，本品则让全经的灵魂人物——**维摩诘居士**——正式登场，并以一场"示疾说法"的戏剧动作，把抽象的般若空性翻译成五百千万人都能感受的"身体"语言。

这一品在论证结构上承担三个功能：**(一)** 通过对维摩诘居士功德的逐项铺陈，确立"在家亦能臻于极高证境"这一全经命题；**(二)** 通过维摩诘居士在二十余种社会场合中以不同身份饶益众生的纲领性叙述，把"方便波罗蜜多"由抽象概念转译为具体行持；**(三)** 通过"示疾→说身→趣佛身→发菩提心"的因病说法链条，把般若空性教义植入最切身的"身体"对境，并以"令无数千人发阿耨多罗三藐三菩提心"作为这一段教法效用的现量印证。

索达吉堪布于第 9-13 课讲此一品，逐字依鸠摩罗什译本而对照藏译本、玄奘译本、吴支谦译本、僧肇《注维摩诘经》、智者《维摩经略疏》、窥基《说无垢称经疏》等多家注疏；同时反复以宁玛派"善巧方便"的视角发挥——尤其在"于淫舍示欲过""于酒肆能立其志""现身有疾以为方便"等段落，将本经与密乘"烦恼即菩提"、大圆满"自解脱"教义相印证。

---

## 一、品名释义——"方便"的深度

世人讲"方便"多取其浅义——办事的便利之法。佛教所谓"方便" (upāya)，是与"智慧" (prajñā) 相对而立的菩萨德相，**指六度万行或大悲心的妙用**，意义远比世俗之"便利"深邃。

堪布开宗明义指出：

> "在佛教当中，'方便'与'智慧'相对，指六度万行或大悲心的妙用，意义非常深邃，一般人很难透彻诠释。"

窥基大师在《说无垢称经疏》中将菩萨方便细分为**十二种**：
- **依内修证有六种**：(1) 于有情悲心俱行、顾恋不舍；(2) 于诸行如实通知；(3) 于无上菩提妙智深心欣乐；(4) 顾恋有情不舍生死；(5) 知诸行故轮转生死而心不染；(6) 欣佛智故炽然精进。
- **依外成熟有六种**：(1) 能令有情以少善根感无量果；(2) 令有情以少功力引大善根；(3) 于圣教憎背有情，除其恚恼；(4) 处中住者，令其趣入；(5) 已趣入者，令其成熟；(6) 已成就者，令得解脱。

本品的全部内容——维摩诘居士的功德、他在不同人群中的善巧、他的"示疾说法"——皆可视为这十二种方便的具体展开。

## 二、维摩诘居士登场：在家修行人的极致典范

### 2.1 登场之缓——直至第二品才真正出现

堪布以"电视剧"的比喻点出本经独特的叙事节奏：

> "就像一部十多集的电视剧，在第一集中，只是提到了主人公的名字；到了第二集，他才真正登场。"

本经第一品仅在序分中提一次"长者子宝积与五百长者子俱来诣佛"，并未引出维摩诘居士。直至《方便品》开篇——"尔时，毗耶离大城中有长者，名维摩诘"——这位与文殊菩萨齐名、令十大弟子四大菩萨皆"不堪问疾"的核心人物才正式入场。

### 2.2 经文系统铺陈——维摩诘居士的功德序列

经文以连续的并列短句，依次展开维摩诘居士的德相。这一段是全经"在家成就观"最重要的教证依据。堪布逐句疏解，并与藏译/玄奘译/吴支谦译做对照：

| 经文 | 含义 | 堪布释要 |
|---|---|---|
| **已曾供养无量诸佛，深植善本** | 多生累劫供佛、植善根 | 《万善同归集》:"菩萨三种无厌：供佛、闻法、供僧"；维摩诘居士即三者俱无厌 |
| **得无生忍** | 一地或八地菩萨境界 | 对空性无丝毫恐惧；《阿弥陀经》称"阿鞞跋致"；无学地则只称"无生"，无"忍"之概念 |
| **辩才无碍** | 口业功德 | 七种辩才一一具足 (《华严经疏》：捷辩、无断辩、迅辩、应辩、无错谬辩、丰义味辩、一切世间最上妙辩) |
| **游戏神通** | 身业功德 | 五眼六通；身体腾空、发火焰、借声传真理 |
| **逮诸总持** | 意业功德 | 不忘陀罗尼；与世间记忆力 (扑克牌过目不忘等) 不同，维摩诘所持是经藏论藏的法义 |
| **获无所畏** | 四无所畏 | "通达无我，则四魔当下皆被降伏" |
| **降魔劳怨** | 降伏四魔 (烦恼魔、蕴魔、死魔、天魔) | 不依武器，唯以智慧降之 |
| **入深法门，善于智度** | 通达空性、证人法二无我 | 胜义层面认识万法本来面目 |
| **通达方便，大愿成就** | 世俗层面的善巧方便、大愿圆满 | 如阿弥陀佛四十八愿、药师佛十二大愿皆已成就 |
| **明了众生心之所趣** | 知众生意乐 | 藏译/唐译分"意乐"与"行为"二者；鸠摩罗什合译为"心之所趣" |
| **又能分别诸根利钝** | 知众生根基 | 凡夫无他心通故易看错人；维摩诘清楚一切众生根基 |
| **久于佛道，心已纯淑** | 多生安住佛道、心已纯熟 | 不像"外熟内不熟的芒果"——出家多年内心仍不堪能 |
| **决定大乘** | 大乘种姓决定 | 《俱舍论》：不定种姓者会在小乘大乘间转换，决定种姓者不会 |
| **诸有所作，能善思量** | 智慧前导，详察后行 | 不人云亦云、不盲目行动 |
| **住佛威仪** | 行住坐卧如理如法 | 虽未受比丘戒，威仪仍庄严 |
| **心大如海** | 心广深如大海 | 僧肇《注维摩诘经》引鸠摩罗什：海有三德——深广无边、清净不受杂秽、藏积无量珍宝；菩萨三德义同 |
| **诸佛咨嗟** | 受十方诸佛赞叹 | — |
| **弟子释梵世主所敬** | 声闻、帝释、梵天、四大天王皆敬 | "世主"=玄奘译"护世"，藏文"杰定炯瓦"，即四大天王 |
| **欲度人故，以善方便，居毗耶离** | 因度人之故住人间 | 维摩诘住天宫无碍，但特喜人类，故以方便居广严城 |

### 2.3 在家成就观——本经的根本立场之一

堪布在第 9 课开篇即点明本经传讲的现实意义：

> "现在很多人以为：要想学修佛法，就一定要看破世间而出家修行，以在家身份是很难获得成就的，更不用说去度化众生了。但事实并非如此。"

本经的立场是：**佛教修行人有两种**——一种舍俗出家寻求寂静，一种以在家身份修行度化众生。本经第二品至第十一品几乎都在阐述后者的可能性与高度。"佛陀得知维摩诘居士生病后，就分别让十大弟子前去探望，但他们都不敢去"——这一全经叙事的根本张力，本质上就是"在家成就"对"出家声闻"的胜义性彰显。

堪布反复强调三点伦理准则：
1. **在家人完全可以修行佛法**，且修得好其境界可远远超过某些出家人；
2. **出家人不能轻视在家人**——其中有"远远胜过声闻乘出家人"的居士菩萨，应恭敬顶戴；
3. **非佛教徒亦可以智慧研究佛教**——佛教并非世俗分别念所揣测的样子。

藏汉历史上的实例不少：龙猛菩萨的上师萨绕哈巴、莲花生大士、第一/二世敦珠法王皆示现在家相而戒律纤尘不染；汉地鸠摩罗什大师晚年的某些行为虽难为人接受，"殊不知大师的境界确实非同一般"；藏传佛教史上智悲光尊者、麦彭仁波切等大德"显现上都是在家人，但其修行境界和度化众生的能力远远超过很多出家人"。

## 三、六度摄受——维摩诘居士的行门

第 10 课进入维摩诘居士行持六度而摄受众生的详述。这是本经把"六度万行"翻译成"利他行门"的最简洁段落：

| 度 | 经文 | 所摄受之众 |
|---|---|---|
| 布施 | **资财无量，摄诸贫民** | 贫困之众 |
| 持戒 | **奉戒清净，摄诸毁禁** | 毁戒之众 |
| 安忍 | **以忍调行，摄诸恚怒** | 嗔恨、嫉妒、害心重者 |
| 精进 | **以大精进，摄诸懈怠** | 懒惰昏昧之众 |
| 禅定 | **一心禅寂，摄诸乱意** | 散乱、昏沉、掉举之众 |
| 智慧 | **以决定慧，摄诸无智** | 愚痴无明之众 |

堪布点出此段的伦理核心：

> "对于大乘佛教徒来说，有两件事最为根本：一是自己好好修行，二是尽量利益众生。这就是我们的志向和人生目标。"

**摄受 (saṃgraha) 的双向性**：英文译本作"吸引"——以自己的行为为他人树立榜样，令其学习。这与"控制"或"教训"截然不同。维摩诘居士的六度不是孤立的自利行，而是**每一度都同时是利他行**——这是大乘菩萨道的根本结构。

## 四、维摩诘居士的"在家清净行"——既入世又超世

下一段经文以"虽……常……"的对偶结构，展示维摩诘居士在家身份与出世境界并行不悖的德相：

| 经文 | 释要 |
|---|---|
| **虽为白衣，奉持沙门清净律行** | 在家身份，行如沙门；藏文"迦嘎" (印度) 即"穿白衣服的民族" |
| **虽处居家，不著三界** | 不耽欲界财色名食睡，不耽色界禅定妙味，不耽无色界寂静 |
| **示有妻子，常修梵行** | 藏译/吴支谦译还说有小妾；"梵行居士"——居士五戒基础上过与出家人基本相同的生活 (《三戒论》：梵行居士断非梵) |
| **现有眷属，常乐远离** | 眷属虽多，皆乐寂静；吴支谦译"虽有家属，常如闲居" |
| **虽服宝饰，而以相好严身** | 适当装饰随顺世俗——观世音、文殊、报身佛皆有装饰；佛教不排斥装饰，居士不必学佛后变如乞丐 |
| **虽复饮食，而以禅悦为味** | 享用美食而不贪执，以禅悦为真正美味 |
| **若至博弈戏处，辄以度人** | 经常去赌场戏院，非贪赌博，乃因彼处有所化众生 |
| **受诸异道，不毁正信** | 学外道、世间各派而不沉迷其中 |
| **虽明世典，常乐佛法** | 通达五明而不失佛教见解；慧远大师听《般若经》后觉"儒道九流皆糠秕耳" |

**堪布于此段反复强调"中道"原则**：

- 大乘菩萨"既要心胸宽广，又要在戒律誓言上严谨保守，这是无误之举" (《胜道宝鬘论》)
- 善巧方便要观待自己的境界——"如果内在境界很低，行为却相当高，行为与见解就南辕北辙了，最后很可能堕入恶趣"
- 借南怀瑾所言批评"一身佛气、满口佛话"的形式主义；真正的大乘佛教徒"与怨敌、恶魔都要合得拢"
- 出家人不应为融入社会而还俗或穿西装高跟鞋，但要避免"鱼离开水"式的不适应

这是本经"善巧方便"教义最具世俗操作性的一段：**入世而不染、随顺而不流、变通而守本**。

## 五、二十种身份的善巧方便——菩萨在社会各阶层中的行持

第 10-12 课讲完个人德行后，经文以连续二十句的纲领句，展示维摩诘居士在二十种不同社会场合中以不同身份饶益众生。这是大乘菩萨"普门示现"教义在社会维度上的展开，也是本经最具现代意义的一段：

| 场合/对境 | 经文 | 行持要点 |
|---|---|---|
| 一切人见 | **一切见敬，为供养中最** | 受人天最上供养；受欢迎源于内在境界与跟所有人合得来 |
| 长幼老少 | **执持正法，摄诸长幼** | 此处"正法"指世间礼仪规矩；藏译分老年/年幼/青年三层 |
| 经商 | **一切治生谐偶，虽获俗利，不以喜悦** | 善于经商，获利不喜——经商只是利众方便；《入行论》"贪金涣散人，脱苦遥无期" |
| 云游 | **游诸四衢，饶益众生** | 玄奘译"为益含识游诸市衢"；非游山玩水，乃为引导各地众生 |
| 治政 | **入治政法，救护一切** | 进入司法、执法部门，监督公正、维护苦难众生权利 |
| 讲论 | **入讲论处，导以大乘** | 进入辩论场，以大乘慈悲度化外道；如文殊度化富楼那未度之外道 |
| 学堂 | **入诸学堂，诱开童蒙** | 启蒙儿童青年；500 释迦族童子入学堂，太子提 64 问，老师答不上 |
| 淫舍 | **入诸淫舍，示欲之过** | 入妓院非道德败坏，乃欲贪聚集处易示其过；妙金光妓女公案 (详 §六) |
| 酒肆 | **入诸酒肆，能立其志** | 入酒馆令醉生梦死者清醒立志；"慈悲生祸害，方便出下流"——方便有危险性 |
| 长者 (大商主) | **若在长者，长者中尊，为说胜法** | 商人贪财、求财神，故为说出世解脱法 |
| 居士 (七轮宝施主宝) | **若在居士，居士中尊，断其贪著** | 富裕信士，劝其上供下施 |
| 刹利 (国王种姓) | **若在刹利，刹利中尊，教以忍辱** | 当权者面对反抗易生嗔；优婆鞠多劝阿育王勿杀公案 |
| 婆罗门 | **若在婆罗门，婆罗门中尊，除其我慢** | 学识高贵者易我慢，为说断慢法 |
| 大臣 | **若在大臣，大臣中尊，教以正法** | 执行国王命令，需公正无私 |
| 王子 | **若在王子，王子中尊，示以忠孝** | — |
| 内官 (后宫) | **若在内官，内官中尊，化正宫女** | 调化嫔妃嫉妒邪行 |
| 庶民 | **若在庶民，庶民中尊，令兴福力** | 鼓励行持世俗善法增上福德；阿底峡尊者特别重视转绕 |
| 梵天 | **若在梵天，梵天中尊，诲以胜慧** | 梵天禅定虽胜，缺人无我胜观 |
| 帝释 (欲界天) | **若在帝释，帝释中尊，示现无常** | 五欲妙乐沉溺者，须了无常 |
| 护世 (四大天王) | **若在护世，护世中尊，护诸众生** | 以佛法成熟其相续 |

### 经文总结句

> **长者维摩诘，以如是等无量方便，饶益众生。**

堪布于此段强调一项当代大乘佛教徒急需的素质：

> "真正有智慧、有能力的人，会像《胜道宝鬘论》中所说的那样：可以独自安住，又能处众不乱。这就是大乘佛教善巧方便的体现。"

许多年轻人"特别害怕面对生活，完全不想接触外界，很容易得抑郁症、焦虑症等心理疾病；还有一些修行人，没有一点修行境界，无法将违缘转为道用"——本品所讲的善巧方便，正是这两种困境的对治。

## 六、关键公案——僧肇《注维摩诘经》引出的妙金光女故事

僧肇大师在《注维摩诘经》中讲过一则妙金光妓女的公案，索达吉堪布在第 11 课于"入诸淫舍，示欲之过"段以相当篇幅讲述，这是本品"以欲断欲、烦恼即菩提"教义在汉地古注疏中最具力量的承担：

**妙金光女公案**：

外国有妓女名妙金光，身体呈金色。一商主之子达暮多罗以千两金约她入竹林同游。文殊菩萨观度化因缘成熟，化现为衣着异常庄严之白衣男子。妙金光见之即生贪心，舍商主之子而求化人："你可否把你的衣服给我？"

化人答："可以，但你要发菩提心。"
妙金光问："何等为菩提心？"
化人答："**汝身是也**——你的身体就是菩提心。"
"为什么说我的身体是菩提心？"
化人答："**菩提性空，汝身亦空**——菩提心是空性的，你的身体也是空性的。"

妙金光曾于迦叶佛前闻空性法门，宿善苏醒，**当即证得无生法忍**。她欲随文殊出家，文殊不肯，命她"还是跟商主之子走吧"。

回到竹林，妙金光忽然示现死亡，尸体迅速腐烂、发出恶臭、变为白骨。商主之子大惊，跑去见佛，佛为其说法，**亦获法忍**。

**公案的本经义**：(一) 文殊以化人度妙金光——以最强烈的贪欲对境作为菩提心的起点，此即"以欲断欲"之极致；(二) 妙金光化为尸体度商主之子——同一肉身，在贪欲炽盛者眼中是欲爱的对境，在证得无生法忍者的方便手中是观无常、观不净的真切教材；(三) "菩提性空，汝身亦空"一句，把空性教义植入最具身体性的对境，是本经"烦恼即菩提"教义在《方便品》的预演——为后文品8《佛道品》"非道是佛道"做铺垫。

堪布并以创巴仁波切的"大麻 party"故事印证："谁有大麻？拿来，我们一起开 party！"弟子兴奋而至，结果上师让大家把口袋里的大麻全部扔进火炉——"**这就是我们的我执！这就是我们的贪欲！**"——这是同一类方便的近代演示。同时堪布警示："**有些行为到底是善巧方便还是疯狂行为，的确难以分辨**" (《胜道宝鬘论》: 成就者和骗子容易混淆)，故无境界者切勿盲目效仿。

## 七、示疾——"方便"的极致动作

### 7.1 示疾的双重方便

第 12 课经文转入本品的核心戏剧动作：

> **其以方便，现身有疾。**

堪布点出"示疾"包含两重善巧方便：
- **身体方便**：示现疾病，引来众人探望，创造说法对境；
- **语言方便**：来探病者一到，即有机会传授佛法。

> "生病是度化众生的良好机缘。不说维摩诘居士吧，我自己都觉得，在医院里能找到所化众生或能结缘的众生。"

堪布并以自身住院遇藏族癌症晚期病人的经历印证此教义。"以前的大德们，示现生病时可以度很多病人，因为他们有共同的感受；也可以召集无病的众生来到自己面前，为其宣讲佛法。"

### 7.2 探病者的盛况

> **以其疾故，国王大臣，长者居士婆罗门等，及诸王子并余官属，无数千人，皆往问疾。**

维摩诘居士的威望由此可见。堪布于此段插入"关怀病人和临终者"的现代延展——佛陀亲为病比丘擦身打扫房间的公案；《梵网经》"看病福田第一福田"；西方"安宁疗护""临终关怀"服务等。这是本经"示疾"教义在现代社会的应用：**生病不是修行的中断，而是修行的扩展场域**——既是被度者的良缘，也是医病者的福田。

## 八、因病说法的核心教法——身体的彻底剖析

第 12-13 课进入本品最重要的教法段落。维摩诘居士对来探病者展开了一段佛教史上最系统、最密集的"身体观"教法。这一段是全经"以观身入空"的根本教证段，也是本经把抽象空性教义植入最切身的"身体"对境的关键操作。

经文开端即点明三层教法结构：

> **诸仁者！是身无常无强无力无坚，速朽之法，不可信也！为苦为恼，众病所集。**

堪布释要：**先讲无常，再讲苦，后讲空性与无我**——这是佛教剖析身体的标准次第。

### 8.1 无常的四相

| 相 | 含义 |
|---|---|
| **无常** | 身体会变、会坏 |
| **无强** | 老来身体脆弱 |
| **无力** | 病时连吃饭走路都费劲 (华智仁波切《大圆满前行》："人生病后，身体就像被石头击中的麻雀") |
| **无坚** | 死后身体不坚——天葬师以刀斧顷刻分解 |

悉达多太子四门出城见老、病、死，正是此身体三相的对境化呈现。窥基《说无垢称经疏》将无强、无坚、无力分别对应欲界、色界、无色界身——堪布认为不必如此对应，"不如将其理解为对身体无常的不同描述"。

### 8.2 八大比喻——身体的非实有性

经文以八连串比喻描述身体的本质：

| 比喻 | 经文 | 释要 |
|---|---|---|
| 1. 聚沫 | **是身如聚沫，不可撮摩** | 水沫一抓即灭；保养之徒劳 |
| 2. 水泡 | **是身如泡，不得久立** | 阳光照射即破裂 |
| 3. 阳焰 | **是身如炎，从渴爱生** | 阳焰水生于渴心；身依爱执假合 |
| 4. 芭蕉 | **是身如芭蕉，中无有坚** | 一层一层，无实可得 (《中论》《四百论》共用此喻) |
| 5. 幻 | **是身如幻，从颠倒起** | 五蕴假合执为实有，如电视电影中人 |
| 6. 梦 | **是身如梦，为虚妄见** | 梦中身体醒来不存 |
| 7. 影 | **是身如影，从业缘现** | 身体高矮胖瘦由前世业决定 |
| 8. 响 | **是身如响，属诸因缘** | 身依内外众缘聚合 (《浴佛功德经》"诸法从缘起，如来说是因，彼法因缘尽，是大沙门说") |

并补充"浮云"、"闪电"二喻：

| 比喻 | 经文 | 释要 |
|---|---|---|
| 浮云 | **是身如浮云，须臾变灭** | 仔细观才察觉，变化无常 |
| 闪电 | **是身如电，念念不住** | 猝死即此喻之极致；中国每年 180 万人猝死 |

### 8.3 关键教法——"是身如影，从业缘现"

堪布在第 12 课和第 13 课两次特别强调此句的实修意义：

> "这一世你的身体无论好看还是难看，自己满意还是不满意，都要归结于前世的业，而不能怪罪父母，因为他们只是提供了助缘而已。"

> "如果父母真能决定孩子的长相，每对父母都会把自己的孩子生得很好看。决定自己长相的，唯一是自己前世的业。"

堪布特地把藏文单独记下了这句。这是本品对汉地佛教徒最具针对性的教言之一：很多人对相貌、健康、命运的种种抱怨，都可以由"是身如影，从业缘现"这一句重新整理——"自己的身体是在前世的'工厂'里造的，父母只负责运输，就像佛像在工厂里造好后再由司机运过来一样。所以，如果佛像本身有瑕疵，是不能怪司机的。"

此段亦与基因学的"基因突变解释"形成对照——堪布点出佛教因果观与现代科学解释模式的不同。

### 8.4 四大无我——身体本无主、无我、无寿、无人

经文以四大对应四相：

| 大 | 经文 | 含义 |
|---|---|---|
| 地 | **是身无主，为如地** | 大地无常主，强者得之 (《注维摩诘经》)；身亦如是 |
| 火 | **是身无我，为如火** | 因缘具足时燃烧，因缘不具足时不存——身体亦然 |
| 风 | **是身无寿，为如风** | 寿命无定如风向难测；龙猛《亲友书》"呼气吸气沉睡间，能得觉醒极稀奇" |
| 水 | **是身无人，为如水** | 二解：(a) 水中捞月——身上找不到人我；(b) 河水奔流——身体如水般无实本体 |

藏译/玄奘译还多一句"身如虚空无自性"——加上四大成五大。鸠摩罗什译本无此句。

### 8.5 总结教言——四大假合

> **是身不实，四大为家。是身为空，离我我所。是身无知，如草木瓦砾。是身无作，风力所转。**

堪布以《入中论》"如支分积聚，故称之为车，如是依诸蕴，俗谛名众生" 印证："车是各种零件的组合，有车轮、方向盘等，这些支分的积聚被称为'车'；同样，在世俗谛当中，人们也将四大、五蕴的积聚名为'众生'。而在胜义当中，依靠四大而命名的'身体'和'我'都不存在。"

**身体之所以会动，是因风脉明点中之风** (下行风、上行风、平行风、持命风等，密宗对此讲得比较多)；离开风，身体即"如如不动"——如断绳的机关。

### 8.6 不净观与五尘喻

> **是身不净，秽恶充满。是身为虚伪，虽假以澡浴衣食，必归磨灭。是身为灾，百一病恼。是身如丘井，为老所逼。是身无定，为要当死。**

堪布引《入行论》、《四百论》三段教证印证身不净：
- 《入行论》："涂身微妙香，旃檀非她身，何以因异香，贪着她身躯？" (是香水之香，非身之香)
- 《入行论》："宜人冰片等，米饭或蔬菜，食已复排出，大地亦染污。"
- 《四百论》："若一切净物，后触成不净，智人谁能说，彼中有净性。"

并引米拉日巴尊者："**见而生畏之尸体，本为现在之身体。**"——"我们现在这个身体就是能行走的尸体，只是还没有倒下去而已。"

### 8.7 死定无疑

> **是身无定，为要当死。**

《入行论》"若思今不死，安逸此非理，吾生终归尽，死期必降临。"《解忧经》"大地及天上，三界与四生，未曾得见闻，不受无常者。"秦始皇求长生不老药未得；现代医学也无法令人不死。

### 8.8 终极总结——五蕴十八界十二处的危险性

> **是身如毒蛇，如怨贼，如空聚，阴界诸入所共合成。**

按藏文解释：身受**五蕴**控制，如被毒蛇缠缚必死；身由**十八界**所成，如被怨贼束缚；身由**十二处**所成，如同空空的村落。

僧肇《注维摩诘经》引一公案为此段释义：

> 有人得罪国王，国王想秘密处死，于盒中装四毒蛇令其守护，又派五怨贼持刀看守。一善知识告之"快离开"。其逃至空村，善知识又告"恶贼将归"。其再逃至大河，造小船渡河至彼岸。

**四毒蛇**=四大；**五怨贼**=五蕴；**空聚中的恶贼**=十二处、十八界；**小船**=八正道；**彼岸**=解脱。这是把佛教解脱论嵌入身体观最经典的一则比喻。

## 九、由观身彻底转向法身——从"厌身"到"求佛身"

### 9.1 厌离身体的目的——求取佛身

> **诸仁者！此可患厌，当乐佛身。**

经过前面长篇的身体剖析，维摩诘居士不停于"厌身"，而是把厌身导向**求取佛身**。这是大乘的关键转向——**不是离世厌生，而是转所观对境**。

> **所以者何？佛身者即法身也。**

最究竟、最坚固、永不毁坏的就是法身。有些注疏说法身是色身的来源。

### 9.2 法身从无量清净法生——本经的菩萨道纲领

经文以连续多句"从X生"的并列结构，列出法身所从生的众多清净法：

| 类别 | 经文 |
|---|---|
| 福慧二资 | 从无量功德智慧生 |
| 五分法身 | 从戒、定、慧、解脱、解脱知见生 |
| 四无量心 | 从慈悲喜舍生 |
| 六度 | 从布施、持戒、忍辱、柔和、勤行、精进、禅定、解脱、三昧、多闻、智慧诸波罗蜜生 (注：经文将六度展开为十一项) |
| 善巧 | 从方便生 |
| 神通 | 从六通生 |
| 三明 | 从三明生 (宿命明、天眼明、漏尽明) |
| 道品 | 从三十七道品生 |
| 止观 | 从止观生 |
| 不共法 | 从十力、四无所畏、十八不共法生 |
| 对治 | 从断一切不善法、集一切善法生 |
| 见地 | 从真实生 |
| 行持 | 从不放逸生 |
| 总结 | **从如是无量清净法生如来身** |

堪布特别强调本段的**修行论意涵**：

> "佛陀的诸多功德并非轻易获得的，要积累无边资粮、精勤净除罪障才能得到。有些人觉得，只要修行就能马上成佛，其实没那么简单。"

引《菩提资粮论》："少少积聚福，不能得菩提，百须弥量福，聚胜乃能得。"

这一段实际上是把整部大乘菩萨道的纲领——六度、四无量、三十七道品、五分法身、十力四无畏十八不共法——压缩在"法身从生"的句式中，与"是身从业缘现"形成精确对照：**色身从业缘生，故无常无主；法身从清净法生，故坚固永存**。这是大乘菩萨道最简洁的纲领之一，可与《宝性论》如来藏教义、《现观庄严论》八现观、《大乘经庄严论》菩萨地行对读。

### 9.3 发菩提心——本品的归宿

> **诸仁者！欲得佛身，断一切众生病者，当发阿耨多罗三藐三菩提心。**

由"观身→厌身→求佛身→发菩提心"的次第，本品的整个因病说法链条至此完成。这一句是维摩诘居士因病说法的归宿，也与《佛国品》末"八万四千人皆发阿耨多罗三藐三菩提心"形成首尾呼应。

经文末尾以现量印证：

> **如是长者维摩诘，为诸问疾者，如应说法，令无数千人皆发阿耨多罗三藐三菩提心。**

"令无数千人皆发阿耨多罗三藐三菩提心"——这是整品教法的实际效用证明。"示疾"作为方便，最终在听闻者发心的现量上被证成是真方便。

## 十、本品的论证结构

把本品的论证结构整理如下：

```
登场 (品名"方便"之下)
  └── 维摩诘居士功德序列 → 确立"在家成就"立场
        └── 六度摄受 → 把功德转为利他行
              └── 在家清净行 ("虽……常……") → 入世而不染
                    └── 二十种身份善巧 → 普门示现
                          └── 示疾 → 创造说法机缘
                                └── 因病说法
                                      ├── 身体八大比喻 (无常)
                                      ├── 四大喻 (无主无我无寿无人)
                                      ├── 不净/虚伪/灾患/丘井 (苦)
                                      ├── 毒蛇怨贼空聚 (空、五蕴假合)
                                      └── 厌身→求佛身→法身从清净法生
                                            └── 发菩提心
                                                  └── 现量印证：无数千人皆发心
```

每一层都既是论证的下一步，也是"方便"的具体展开。"方便"不是一种孤立的德目，而是贯穿全部菩萨行持的根本品质——**身体可以是方便 (示疾)，社会身份可以是方便 (二十种身份)，世俗才能可以是方便 (经商、辩论、艺术)，疾病可以是方便 (因病说法)，乃至贪嗔痴本身在菩萨手中也可以是方便 (妙金光女、入诸淫舍)**。

## 十一、堪布的宁玛派视角——本品教义与密乘的关联

堪布在讲本品过程中，多次将其与密乘视角相印证。这些点是本品在宁玛派传承中的特殊处理：

1. **"善巧方便"的密乘含义**：本品所讲"于诸场合不被染污"的菩萨行，在密乘中以更明确的"转烦恼为道用"教义展开。妙金光公案中"菩提性空，汝身亦空"的安立，即是密乘"贪欲即智慧"教义在显宗经典中的预演。

2. **创巴仁波切大麻 party 公案**：堪布以此故事说明真实的密乘行人如何处理弟子的烦恼——既不压制也不放纵，直接以方便令其当下醒觉。但他同时严格点出："**成就者和骗子容易混淆**" (《胜道宝鬘论》)，无境界者切勿效仿。

3. **示疾说法与"自显疾即智慧自显"**：本品"现身有疾"的方便，在密乘语境下可与"显现即解脱"教义连接。生病不是修行的违缘，而是修行的扩展场域；这一立场在大圆满法本中以更彻底的方式展开。

4. **法身从无量清净法生 vs 法身本来圆满**：本品讲"法身从清净法生"是因地修证的角度；如来藏教义则讲"法身本来圆满，唯被客尘所障"。两者并不矛盾——前者是道次第之相，后者是法尔之理。可与《宝性论》对读。

5. **身体观的密乘补充**：本品以无常、苦、空、不净观身体；密乘则以"五蕴即五方佛、五大即五佛母" (《大幻化网》) 转身体观为本尊观。两者次第有别——显宗破身执后，密乘以清净观重立本尊。此处堪布只点不展开，因本经自身未进入密乘语境。

## 十二、与本经其他品的呼应

- **品 1《佛国品》**："心净国土净"是本经的本体论地基；品2"方便"则是把这一地基转为行持。心净故能取于净土，方便故能于秽土行净行——两品互为表里。
- **品 3-4《弟子品》《菩萨品》**：本品所讲"在家成就胜过出家声闻"的立场，将在品3-4 通过"十大弟子皆言不堪问疾"的叙事被戏剧化坐实。每位弟子被维摩诘呵责的细节，皆是对小乘"住"于某一定见的破斥——本品已为其铺好对照。
- **品 5《文殊师利问疾品》**：本品末"示疾"创造的说法对境，将在品5 以文殊问疾的高峰对话深度展开——"以空病空"等教义即是"因病说法"在哲学层面的极致表达。
- **品 8《佛道品》**："非道是佛道"教义在本品妙金光公案、入诸淫舍/酒肆段落中已露其端。
- **品 9《入不二法门品》**：本品"虽……常……"的对偶结构 (虽白衣常清净、虽处家不著三界、虽服宝饰常以相好严身……) 是"不二"教义在生活行持层面的预演。

## 十三、与其他集合的交叉

- **`collections/大圆满前行/寿命无常`**：本品身体八喻、四大喻、"是身如丘井为老所逼"、"是身无定为要当死"等段，与《大圆满前行·寿命无常》的修法纲领高度相通——本品偏重观察分析，前行偏重观修修法，两者互为表里。华智仁波切《大圆满前行》中"被石头击中的鸟雀"喻，堪布在讲"无力"一句时直接引用。
- **`collections/入行论/智慧品`**：本品破身执的引文中多次引用《入行论》——"涂身微妙香，旃檀非她身"、"宜人冰片等"等。寂天菩萨以推理破身执，本经以比喻破身执，殊途同归。
- **`collections/中论/`、`collections/中观四百论/`**：本品"是身如芭蕉中无有坚"直接对应《中论》《四百论》共用的芭蕉喻。"如支分积聚故称之为车"则对应《入中论》七支寻车。
- **`collections/宝性论/`**："法身从无量清净法生"与如来藏"法身本具清净功德"的关系——前者是因相，后者是果相；可深化"本来清净 vs 修证清净"的双面理解。
- **`topics/foundations/practice-path.md`**：本品所列六度、四无量、三十七道品、五分法身等，是大乘修道纲领最简洁的一段经证。
- **`topics/madhyamaka/`**：身体八喻 (聚沫、水泡、阳焰、芭蕉、幻、梦、影、响) 是中观破自性最常用的喻例库；本品提供的是经证版本。

## 十四、本品所植入的实修要点

可从本品提取的实修要点：

1. **"方便"必须有般若摄持**——纯方便而无智慧易堕"风流变下流"。前提：自心不被烦恼染污、有正知正念、能不离菩提心。
2. **观待境界行持**——若内在境界低，宁守声闻乘行；若境界相应，可学维摩诘式的入世方便。"自己相续的状态要清楚"。
3. **"是身如影，从业缘现"**——抱怨父母、抱怨命运皆不合理；身相、健康、长相皆前世业感。
4. **观修身体八喻**——在日常起执身相时，可逐一以聚沫、水泡、阳焰、芭蕉、幻、梦、影、响观之。
5. **"见而生畏之尸体，本为现在之身体"** (米拉日巴) ——破身贪的最直接观法。
6. **生病时的修行**——(a) 修自他相换；(b) 趁机度医生护士同病房之人。"开开心心地去医院"。
7. **关心病人**——《梵网经》"看病福田第一福田"；佛陀亲为病比丘擦身。
8. **不要过于形式主义**——避免"一身佛气、满口佛话"；与不同人群都能融入。
9. **法身从无量清净法生**——成佛非一日之功；忍耐与长期坚持比一时热情重要。

## 十五、关键经文摘录

为便于引用与背诵，本品核心经句摘录：

> **其以方便，现身有疾。**

> **诸仁者！是身无常无强无力无坚，速朽之法，不可信也！为苦为恼，众病所集。**

> **是身如聚沫，不可撮摩；是身如泡，不得久立；是身如炎，从渴爱生；是身如芭蕉，中无有坚；是身如幻，从颠倒起；是身如梦，为虚妄见；是身如影，从业缘现；是身如响，属诸因缘；是身如浮云，须臾变灭；是身如电，念念不住。**

> **是身无主，为如地；是身无我，为如火；是身无寿，为如风；是身无人，为如水。**

> **是身不实，四大为家；是身为空，离我我所；是身无知，如草木瓦砾；是身无作，风力所转。**

> **诸仁者！此可患厌，当乐佛身。所以者何？佛身者即法身也。**

> **诸仁者！欲得佛身，断一切众生病者，当发阿耨多罗三藐三菩提心。**

核心教证 (堪布引):

> 《入行论》："涂身微妙香，旃檀非她身，何以因异香，贪着她身躯？"
>
> 《四百论》："若一切净物，后触成不净，智人谁能说，彼中有净性。"
>
> 米拉日巴："见而生畏之尸体，本为现在之身体。"
>
> 龙猛菩萨《亲友书》："寿命多害即无常，犹如水泡为风吹，呼气吸气沉睡间，能得觉醒极稀奇！"
>
> 《菩提资粮论》："少少积聚福，不能得菩提，百须弥量福，聚胜乃能得。"
