# 品01 ·《佛国品》序分与菩萨功德赞

本文档涵盖《佛国品》的前半——从"如是我闻"到宝积菩萨赞佛偈结束 (对应第 2-6 课)。此段为全经搭建场景、展现法会庄严、铺陈主要人物并通过宝积赞佛将众生注意力聚焦于佛的不共功德。核心教义 (净土之行、心净国土净、舍利弗-螺髻梵王辩) 接续于 `品01-心净国土净.md`。

第 1 课不涉经文本文，为堪布讲本经之缘起、闻法要求、译本选择 (鸠摩罗什译本)、所参考二十七种注疏等总介绍，其要点并入下列各节。

## 一、经名、译者与维摩诘其人

### 1.1 经名的安立——依主安立

佛经经名的安立方式有多种：依地名 (《楞伽经》)、依时间 (《父子相会经》)、依祈请者 (《梵天请问经》)、依意义 (《十地经》)、依偈数 (《般若十万颂》)、依比喻 (《金刚经》《妙法莲华经》)、依作者安立。

**《维摩诘所说经》属第七类——依作者安立**。"维摩诘"是音译，意为"无垢"；玄奘译作《说无垢称经》。藏译名《无垢称经》("哲玛美巴"无垢+"扎巴"称扬+"丹比多"所说经)。经中大部分关键教义由维摩诘居士宣说，最后由佛陀予以认可，故以之命名。本经又名《不可思议解脱经》。

### 1.2 维摩诘的身份

经中:
- 第一品身份：**居士菩萨**
- 第十二品揭示：**妙喜世界无动如来座下菩萨**
- 余经说法：**古佛再来，佛号金粟如来**
- 《大方等大集经》说：即东方无量世界五功德佛国土中的**日密菩萨的化身**，"汝非彼土维摩诘耶？"

此多层身份显示诸佛菩萨化现不可思议、一变多多变一，不可以二取分别念揣测。凡夫具烦恼障所知障，不能完全了知此等境界——此是本经一开篇即埋下的基调。

### 1.3 译者鸠摩罗什

- **汉传佛教四大译师之一** (其余为真谛、玄奘、不空)
- 生于龟兹国，父鸠摩罗炎为印度丞相之子、示现出家云游为国王之妹耆婆所得
- 七岁随母出家，后于克什米尔、印度拜诸善知识
- 前秦苻坚派吕光率七万兵讨伐龟兹请之，滞留凉州 17 年未弘法
- 公元 401 年至长安，建中国第一个译经场"草堂寺"，译经十余载，71 岁圆寂
- **誓愿印可**：圆寂前誓"如我译文无误，愿火化后舌不焦烂"；荼毗后舌化坚硬舍利
- **译经风格**：简练流畅，将重复语言凝练表达；译三百余卷
- **加持力特异**：虽玄奘译本几与藏文本逐字对应更为精准，但鸠摩罗什译本"读起来特别顺，也会自然而然令人感受到一种特殊的加持"——堪布讲汉传经论几乎皆选鸠译为讲本
- **禅宗前行**：学者认为罗什所译《金刚》《法华》《维摩诘》等为禅宗弘扬打下基础；达摩祖师后来传不二法门能被接受，与此有关

**本经版本**：《维摩诘经》汉文共有 7 译，现存 3 种 (吴支谦、鸠摩罗什、玄奘)。鸠摩罗什本十四品；玄奘本与藏/梵本同为十二品 (弟子品-声闻品合并、法供养品-嘱累品合并)，内容完全对应。梵文本 1999 年日本大正大学学者在布达拉宫书库"智光明庄严经"梵文抄本中发现，2004 年出版《梵藏汉对照〈维摩经〉》。

## 二、五种圆满——序分

> **如是我闻：一时，佛在毗耶离庵罗树园，与大比丘众八千人俱。**

经文前这四句交代五种圆满 (亦称"五事"、"通序")：

| 圆满 | 原文 | 要点 |
|---|---|---|
| **法 (如是)** | 如是我闻 | 佛陀《涅槃经》中对阿难嘱"一切经初当安'如是我闻'" (僧肇《注维摩诘经》特释)。表明结集者如实记录，未夹杂分别念。阿难、迦叶、优波离等依不忘陀罗尼立为文字，"比录音质量还好" |
| **时** | 一时 | 不定日时。佛陀观待天龙人等不同众生根基说法，时间概念各异，故言"一时"。爱因斯坦"时间空间都是错觉"为近代印证 |
| **处** | 佛在毗耶离庵罗树园 | 毗耶离 = 广严城，印度六大城市之一，阿难尊者遗塔所在；庵罗树 = 芒果树，《亲友书》以之喻人四种成熟；此园为庵罗树女所供 |
| **法主** | 佛 | 本师释迦牟尼佛 |
| **眷属** | 与大比丘众八千人俱 | 藏译本加小乘不共特点：皆阿罗汉，已灭尽烦恼、断一切垢染、得解脱；多为"破惑比丘"。此外有菩萨三万二千、梵天万、帝释万二千、天龙八部、四众弟子 |

堪布特别点出："如是我闻"体现佛经记录的高度准确性——比丘们奉行的是"如实记录而不夹私义"的原则，讲记整理者当随之。

## 三、菩萨三万二千众功德赞

序分之后，进入**对菩萨众功德的一组短颂式赞叹**。每句或几字，如同菩萨功德的清单，但每一条都蕴含相当具体的教义内容。

### 3.1 整体定位

> 菩萨三万二千，众所知识。

菩萨为大善知识，如日月一般——"有眼目的人都能看到"。不同于世间以财富、长相、才艺闻名的人物，菩萨以相续中的世出世间功德闻名。

### 3.2 总功德总摄

| 经文 | 教义要点 |
|---|---|
| **大智本行，皆悉成就** | "大智"= 证悟空性、通达万法真相的智慧；"本行"= 六度万行、利益众生之菩萨行；合言即**智与悲**。此二摄大乘见修行果——见即无我智，行即利他行，果即智悲双运 |
| **诸佛威神，之所建立** | 玄奘译"诸佛威德常所加持"。菩萨的功德非自力独成，皆得诸佛加持。堪布以"加持之线"比喻：佛子与上师三宝相连，便于弘法利生；失此加持之线，智慧辩才能力虽大也难以长久成就 |

### 3.3 大乘行相

| 经文 | 堪布释要 |
|---|---|
| **为护法城，受持正法** | "法城"实相也 (僧肇)；"受持正法"即安住实相。《地藏十轮经》"千俱胝劫中智者勤修定……不如护我法"——末法时代护持正法功德最大 |
| **能师子吼，名闻十方** | 狮吼喻空性智慧之不可破斥。堪布强调学中观关键即在此："如果中观学得好乃至有所证悟，到哪里去都不会感到畏惧" |
| **众人不请，友而安之** | 菩萨不须祈请而自作众生善友——不同于凡夫须人请方行帮助 |
| **绍隆三宝，能使不绝** | "绍隆"即继承弘扬；三宝延续是每个佛教徒本有之职责 |
| **降伏魔怨，制诸外道** | 四魔 = 烦恼魔、蕴魔、死魔、天子魔；内心执著尽则四魔皆伏。外道总分断见派、常见派，皆违缘起与现代科学 |

### 3.4 内在功德

| 经文 | 堪布释要 |
|---|---|
| **悉已清净，永离盖缠** | 五盖 = 贪欲、嗔恚、睡眠、掉悔、疑；十缠 = 无惭、无愧、嫉妒、悭吝、掉举、悔、昏沉、睡眠、忿、覆 (依《俱舍论》有部宗说) |
| **心常安住，无碍解脱** | 藏译作"断除一切障碍的解脱"。《入行论》"遭遇任何事，莫扰欢喜心"为修持要领 |
| **念定总持，辩才不断** | 念 = 正念 (观察三门、心系善法)，定 = 光明空性无二禅定，总持 = 不忘陀罗尼 |

#### 七种辩才 (《大智度论》《华严经疏》)

菩萨所具辩才七种：

1. **捷辩**——通达一切法，别人问什么都能直接回答
2. **无断辩**——回答问题滔滔不绝，思路无有停顿
3. **迅辩**——回答迅速，无有迟疑
4. **应辩**——根据对方心态、文化背景及不同时间地点而回答
5. **无错谬辩**——契合真相，与真理毫不相违
6. **丰义味辩**——从教证、理证、比喻、意义等多方面宣说
7. **一切世间最上妙辩**——具五德：(一) 意义甚深如雷音；(二) 声音清彻远闻；(三) 哀雅动听如迦陵频伽；(四) 能令众生入心敬爱；(五) 闻者欢喜无厌

### 3.5 六度十度与证位

| 经文 | 堪布释要 |
|---|---|
| **布施持戒忍辱精进禅定智慧及方便力，无不具足** | 玄奘译本作"十度"：布施、持戒、安忍、精进、静虑、般若、方便、愿、力、智。此译本"无不具足"亦涵盖愿力等 |
| **逮无所得，不起法忍。已能随顺，转不退轮** | 已至无有所得的空性境界，获无生法忍，能随顺众生转不退转法轮 |
| **善解法相，知众生根；盖诸大众，得无所畏** | 菩萨四无畏：得闻持总持无畏、知众生根无畏、能答无畏、除疑无畏 (与佛四无畏不同) |

### 3.6 功德、名称、信心

接续长篇偈颂式赞叹：

| 经文 | 堪布释要 |
|---|---|
| **功德智慧，以修其心** | 福德 (有相世俗善根) 与智慧 (胜义空性慧所摄无漏善根) 严饰自心 |
| **相好严身，色像第一，舍诸世间所有饰好** | "相由心生"——前世所造善恶业直接影响此生相貌。外在佛教徒仍要"随顺众生"，非须邋遢如乞丐 (《君规教言论》：菩萨无刻意打扮心，然为引导世间人故意身着妙装束) |
| **名称高远，踰于须弥** | 菩萨美名自地狱至无色界皆闻，非同人间明星之局部闻名 |
| **深信坚固，犹若金刚** | 藏译作"于佛法僧三宝深信坚固"；唐译作"于诸佛法得不坏信"。《宝性论》"自然之胜义，是以信所证，日轮璀璨光，无目不得见"——胜义谛唯依信心证悟 |

**堪布关于信心的五点提示**：

1. **信心与智慧**：菩萨信心源于智慧。无智慧之信为迷信
2. **信心与佛教**：佛教不同于世间知识——世间知识懂义即足，佛教以信心为要
3. **信心与福德**：对三宝一念信心，其善根不败亡 (《大悲经》)
4. **信心与串习**：初学者信心不稳，须经常供灯、供水、顶礼、念佛以长养
5. **信心属圣者七财**：《胜道宝鬘论》"可惜之十法"应加"初有信后失"一条

### 3.7 辩才、演法、境界

| 经文 | 堪布释要 |
|---|---|
| **法宝普照，而雨甘露；于众言音，微妙第一** | "微"甚深，"妙"广大庄严；菩萨以多种语言宣说令众生闻而不厌 |
| **深入缘起，断诸邪见，有无二边，无复余习** | 断除常、断、有、无等一切边执邪见。**龙树菩萨《中论》二十七品末颂"瞿昙大圣主，怜愍说正法，悉断一切见，我今稽首礼"** 为此句最佳注脚——佛陀之独特处即在"悲悯众生之心"与"证悟空性之智慧" |
| **演法无畏犹师子吼，其所讲说乃如雷震** | 僧肇："法音远震开导萌牙，犹春雷动于百草也" |
| **无有量，已过量** | 藏译："菩萨已经超越了量与无量之法"——超越世俗之量 (颜色大小轻重) 与空性之边 |
| **集众法宝，如海导师** | 法 = 福德 + 智慧二资粮 |
| **了达诸法深妙之义，善知众生往来所趣及心所行** | 《华严经》"分别一切法，不取诸法相，了达一切法，其心无所染" |
| **近无等等佛自在慧，十力无畏，十八不共** | 藏译"智慧的灌顶"；玄奘"获无等等佛智灌顶" |
| **关闭一切诸恶趣门；而生五道以现其身** | 加行道忍位即不堕恶趣，何况此三万二千大菩萨 |
| **为大医王，善疗众病，应病与药** | 《四百论》："如善巧良医，少有不治症；获巧力菩萨，非所化甚少" |
| **无量功德皆成就，无量佛土皆严净** | 能清净无量刹土 |
| **其见闻者无不蒙益，诸有所作亦不唐捐** | 《华严经》"虚空可度量，海水可渧数，菩萨功德海，无可为譬谕" |

### 3.8 菩萨名号 (五十二位具名代表)

> 其名曰：等观菩萨、不等观菩萨、等不等观菩萨、定自在王菩萨、法自在王菩萨、法相 (法幢) 菩萨、光相菩萨、光严菩萨、大严菩萨、宝积菩萨、辩积菩萨、宝手菩萨、宝印手菩萨、常举手菩萨、常下手菩萨、常惨菩萨、喜根 (常笑) 菩萨、喜王菩萨、辩音菩萨、虚空藏菩萨、执宝炬菩萨、宝勇菩萨、宝见菩萨、帝网菩萨、明网菩萨、无缘观菩萨、慧积菩萨、宝胜菩萨、天王菩萨、坏魔菩萨、电德菩萨、自在王菩萨、功德相严菩萨、师子吼菩萨、雷音菩萨、山相击音菩萨、香象菩萨、白香象菩萨、常精进菩萨、不休息菩萨、妙生菩萨、华严菩萨、观世音菩萨、得大势菩萨、梵网菩萨、宝杖菩萨、无胜菩萨、严土菩萨、金髻菩萨、珠髻菩萨、**弥勒菩萨、文殊师利法王子菩萨**，如是等三万二千人。

几位名号的义理解:

- **常举手菩萨**——"常举手"非指永举一手，而指常向众生伸出慈悲援助之手
- **常下手菩萨**——以慈悲心行持布施
- **常惨菩萨**——见众生可怜常怀悲悯
- **喜根菩萨**——或译作"常笑菩萨"
- **不休息菩萨**——从不休息的菩萨

《注维摩诘经》说："这些菩萨，越往后功德越大" —— 故**弥勒、文殊师利**排在最后。此二亦是娑婆世界本土菩萨，在本经中份量最重。"等"字代表其余众多菩萨。

#### 文殊菩萨公案 (三月安居)

法王如意宝最喜引用《宝箧经》公案：佛在给孤独园三月安居，文殊不见。安居将结束做羯磨时文殊始至，众以不满相视。迦叶问之，文殊答：在波斯匿王宫住一月，童子学堂住一月，淫女舍住一月。迦叶欲以违律开除，击犍槌集僧。佛悄告文殊："你今天可以给这些声闻展示一下菩萨的境界。"文殊入**现一切佛土三昧**，令恒河沙数诸佛世界显现，每一世界有一文殊菩萨、一迦叶击犍槌。佛问迦叶："这么多文殊菩萨，你准备开除哪一位？"

迦叶顶礼忏悔，往问文殊如何度化众生。文殊答：

> 不是所有众生都能以讲经说法来度化，有些众生要用游戏、唱歌、跳舞、贪求等方式度化；有些众生要用比较强硬的方式来调伏，如现忿怒相、说粗恶语；有些众生要以寂静的方式来调伏……总之，要先与众生结缘，再通过讲经说法来利益他们。

《文殊师利般涅槃经》：仅闻文殊名号，消十二亿劫生死之罪，百千劫中不堕恶趣。文殊心咒"嗡阿RA巴匝纳德"应常念。

## 四、天众与其他闻法者

| 类 | 数量与描述 |
|---|---|
| 梵天 | 万梵天王尸弃 (顶髻)，从"余四天下"或"余方四大洲界"来 |
| 帝释天 | 万二千，从余四天下，属地居天 (住须弥山顶)。色/欲界天人至此，无色界不至 (《俱舍论》说无色界无闻慧；但《地藏经》说无色界亦至忉利天听法) |
| 天龙八部 | 诸天、龙、夜叉 (药叉，飘空守天宫)、乾闼婆 (天界乐神)、阿修罗 (非天、不饮酒神)、迦楼罗 (大鹏金翅鸟)、紧那罗 (人非人，似人有角)、摩睺罗伽 (腹行、大蟒神) |
| 四众 | 比丘、比丘尼、优婆塞、优婆夷 |

堪布插话鬼神之存在：若仅承认肉眼见闻之事物，即同唯物主义；人病/做噩梦/附体反应/险恶之地频发车祸等现象皆为鬼神存在之佐证。念烟施仪轨、观音心咒、用转经轮可饶益饿鬼非人。

### 譬喻须弥显海

> 彼时佛与无量百千之众，恭敬围绕，而为说法；譬如须弥山王显于大海，安处众宝师子之座，蔽于一切诸来大众。

佛如须弥山王安处众宝师子座，光芒映蔽大众所有光——菩萨天人之身光如同萤火对日月。

## 五、宝积菩萨登场与供盖变现

### 5.1 宝积与五百长者子

> 尔时，毗耶离城有长者子，名曰宝积，与五百长者子，俱持七宝盖，来诣佛所，头面礼足，各以其盖共供养佛。

- **长者**：具大福德、大智慧、大才华者
- **宝积**：一位已发菩提心的大菩萨，《大宝积经》中许多问答即为其与佛对话。堪布现代解："智慧的'富二代'"
- **五百长者子**：与宝积同心，皆各国最富最有智慧之青年才俊

**供养宝盖之十种功德** (《业报差别经》)：

1. 处世如盖覆护众生
2. 身心安隐，离诸热恼
3. 一切敬重，无敢轻慢
4. 有大威势
5. 常得亲近诸佛菩萨大威德者以为眷属
6. 恒作转轮圣王
7. 恒为上首，修习善业
8. 具大福报
9. 命终生天
10. 速证涅槃

### 5.2 宝盖变现

> **佛之威神，令诸宝盖合成一盖，遍覆三千大千世界。而此世界广长之相，悉于中现。**

五百盖合为一盖，遍覆三千大千世界；三千大千世界诸须弥山、雪山、目真邻陀山、摩诃目真邻陀山、香山、宝山、金山、黑山、铁围山、大铁围山、大海江河、川流泉源、日月星辰、天宫龙宫诸尊神宫——**全部显现于此宝盖中**。

更进：**十方诸佛在四方四隅上下之刹土中说法的景象，亦现于宝盖中**。僧肇释此义：佛陀以此显现诸佛土殊好不同，为长者子启发"修净土之志"。即：这一神变其实是**为接下来的"净土之问"埋下一条铺垫**。

此段用神变把"净土、佛、说法"三者同时摆进视野，是接下去宝积请问"净土之行"的直接缘起。

## 六、宝积菩萨赞佛偈

宝积随即以偈颂赞佛，共计十数偈，藏译与玄奘/鸠译略有开合。此段为全品中最重的"教义化赞文"之一，几乎每偈都承载一具体教义。

### 6.1 赞佛身、心、业

> **目净修广如青莲，心净已度诸禅定，**
> **久积净业称无量，导众以寂故稽首。**

- "目净修广如青莲"——佛目如印度青莲青白分明。堪布附：打坐时眼半闭而非全闭 (佛像半闭眼样式之出处)
- "心净已度诸禅定"——超越明、乐、无念的世间禅定，达出世间禅定极致
- "久积净业称无量"——三大阿僧祇劫积身口意清净业
- "导众以寂故稽首"——佛以寂灭法引导众生

### 6.2 赞佛神变

> **既见大圣以神变，普现十方无量土，**
> **其中诸佛演说法，于是一切悉见闻。**

神通非佛独有 (其他宗教亦有)，但佛陀示现只是"吸引摄受众生之方便"，真正有智慧者更关注佛陀所说正法。

### 6.3 赞佛法王自在——核心教理

> **法王法力超群生，常以法财施一切，**
> **能善分别诸法相，于第一义而不动，**
> **已于诸法得自在，是故稽首此法王。**

- "法王"——因佛于一切法得自在 (如国王大权在握)
- "法财"——佛法之财富，非指法布施与财布施并列
- **世俗谛方面**：佛以尽所有智毫无杂乱、了了分明地照见诸法自相、总相、分类、法相，能清楚无误宣讲
- **胜义谛 (第一义) 方面**：佛照见万法法性，**无论如来是否出世法性毫无动摇**，将胜义空性法门为众宣讲

**宗喀巴大师《菩提道次第广论》教证** (堪布特重):

> 诸佛非以水洗罪，非以手除众生苦，非移自证于余者，示法性谛令解脱。

> 佛陀不是用水洗罪，不是用手除苦，也不是将自己的证悟转移给别人，他唯一是通过宣讲法性真谛令众生解脱。

### 6.4 赞佛所说法——中观立场核心颂

> **说法不有亦不无，以因缘故诸法生，**
> **无我无造无受者，善恶之业亦不亡。**

**本偈为全经立中观见解之一：法性中不堕常边不堕断边，世俗中依因缘生起。** 详解：

- **不有亦不无**——胜义远离一切戏论，世俗依因缘生
- **无我、无造、无受者**——诸法无自性、无造作者、无感受者
- **善恶之业亦不亡**——虽无我无造受者，业力永不耗尽

《入中论》对比: "如是一切法虽空，从空性中亦得生，二谛俱无自性故，彼等非断亦非常。"

道宣律师《净心诫观法》：

> 虽无我人法，善恶亦不亡，三界轮回苦，六道生死长。

《永嘉证道歌》：

> 梦里明明有六趣，觉后空空无大千。

### 6.5 十二相成道一段

> **始在佛树力降魔，得甘露灭觉道成，**
> **已无心意无受行，而悉摧伏诸外道。**

佛陀在菩提树下以智慧力降魔，证无死甘露寂灭觉道——**此证悟没有心、没有意识、没有受、没有行** (不似外道承许万物有造作者)。摧伏外道之方式非武力，而是**降伏自我，令外魔自然消失**。藏译"外道群邪所不测"——外道根本无法通达此甚深法理。

### 6.6 三转法轮

> **三转法轮于大千，其轮本来常清净，**
> **天人得道此为证，三宝于是现世间。**

- 三转法轮：初转 (四谛)、二转 (空性)、三转 (实相)
- 本来常清净——"我从成道至涅槃，未说一字"
- 依然令人天证悟：初转令五比丘等证果 (吉藏说只有憍陈如初转即得初果)；二转灵鹫山许多菩萨证悟；三转亦多众证无上果

### 6.7 妙法救众与顶礼

> **以斯妙法济群生，一受不退常寂然，**
> **度老病死大医王，当礼法海德无边。**

- 一受不退——受佛教得果即不再退为凡夫
- 常寂然——常住于无分别、寂然光明境界
- 大医王——度老病死。不同于世间医王只能除身疾
- **"善知众生病，起患之所由，又知对治法，除灭众疑惑" (《华严经》)** = 四谛之教：知病即知苦谛，知患由即知集谛，知对治即知道谛，灭疑惑即知灭谛

### 6.8 毁誉不动与善恶平等慈

> **毁誉不动如须弥，于善不善等以慈，**
> **心行平等如虚空，孰闻人宝不敬承。**

- 八法不动 (唐译)——赞毁苦乐利衰称讥
- 五百婆罗门诋毁佛陀，佛丝毫不动；后彼等以语赞叹，佛心依然不动
- **"于善不善等以慈"**——对行十恶业的极恶之人与极善之人平等慈心饶益 (吴支谦译"无诫与诫等以慈"对具戒/破戒者平等以慈)
- 《出曜经》："犹若安明山，不为风所动，叡人亦如是，不为毁誉动。"
- 《佛本行经》"调达怒放石，罗云立其前，俱以慈眼视，见毒栴檀等"——视提婆达多与罗睺罗等同，视毒树与旃檀香等同

### 6.9 一音演法三偈——佛之不共三德

接连三偈以"佛以一音演说法"开头，从三个不同角度彰显佛的不共威神：

> **佛以一音演说法，众生随类各得解，**
> **皆谓世尊同其语，斯则神力不共法。**

**第一角度——音声**：佛以一种语言 (梵语) 演说，不同众生各以自己的语言听懂。藏听藏语、汉听汉语、蒙听蒙语。原理：智慧因光明与空性无二本基自然具足，众生自心未融法界故未显；佛已证最甚深现空无二，自然通晓各种语言。

> **佛以一音演说法，众生各各随所解，**
> **普得受行获其利，斯则神力不共法。**

**第二角度——身相** (或解为根基意乐)：众生各随根基意乐而解。喜持戒者得戒律智慧，喜布施者得布施智慧。亦可解为同一佛身，人见人相、天见天相、旁生见旁生相。

> **佛以一音演说法，或有恐畏或欢喜，**
> **或生厌离或断疑，斯则神力不共法。**

**第三角度——心态变化**：佛一音说法能令众生：

- **恐畏**——讲轮回苦令未生恐怖者心生恐怖
- **欢喜**——讲解脱功德令听者生欢喜
- **厌离**——讲轮回过患令贪执者生出离
- **断疑**——对世间出世间疑惑当场断除

堪布附记："以前法王如意宝讲课具有非常强大的感染力。他讲《释迦牟尼佛广传》时，令众多弟子对佛陀生起了强烈的信心……法师们讲课时，首先要感动自己，然后才能让别人有所触动。"——此是"一音演法令众生各得利益"在讲经场上的投影。

### 6.10 顶礼十力、无畏、不共、结缚、彼岸

> **稽首十力大精进，稽首已得无所畏，**
> **稽首住于不共法，稽首一切大导师，**
>
> **稽首能断众结缚，稽首已到于彼岸，**
> **稽首能度诸世间，稽首永离生死道。**

- **十力** (《入中论》《大乘经庄严论》详)——处非处智力、三世业报智力、诸禅解脱三昧智力、诸根胜劣智力、种种解智力、种种界智力、一切至所道智力、天眼无碍智力、宿命无漏智力、永断习气智力
- **四无畏**——正等觉无畏、漏尽无畏、说尽除道障无畏、说尽苦无畏
- **十八不共法**——身无失、口无失、念无失、无异想、无不定心、无不知己舍、欲无灭、精进无灭、念无灭、慧无灭、解脱无灭、解脱知见无灭、一切身业随智慧行、一切口业随智慧行、一切意业随智慧行、智慧知过去世无碍、智慧知未来世无碍、智慧知现在世无碍
- **能断众结缚**——《俱舍论》结缚皆烦恼异名；三结 (萨迦耶见、怀疑、戒禁取见) 一地即断
- **已到彼岸**——六度圆满
- **能度诸世间**——不只自解脱，还能度天下众生
- **永离生死道**——已断细微所知障、一切生死之流

### 6.11 究竟赞句

> **悉知众生来去相，善于诸法得解脱，**
> **不著世间如莲华，常善入于空寂行，**
> **达诸法相无挂碍，稽首如空无所依。**

- 了知众生来去相——轮回流转情况
- 善于诸法得解脱——自身可住轮回而不为所缚
- **不著世间如莲华**——出淤泥而不染
- 常善入于空寂行——常入空性寂灭最深境界
- 达诸法相无挂碍——通达诸法法相故心无挂碍
- "稽首如空无所依"——虚空无依，佛心无挂
- 《心经》："无挂碍故，无有恐怖，远离颠倒梦想，究竟涅槃"
- 唐译/藏译于末尾多"一切愿满无所愿，大威神力不思议"两句

## 七、宝积请问净土之行——过渡到核心教义

> **尔时，长者子宝积说此偈已，白佛言：**
> "世尊！是五百长者子，皆已发阿耨多罗三藐三菩提心，愿闻得佛国土清净，唯愿世尊说诸菩萨净土之行！"

这一问是全品的**戏剧转折**——赞佛既毕，方出问。五百长者子已久发菩提心，到了建立自己佛国的阶段。

佛陀随喜:

> 善哉！宝积！乃能为诸菩萨，问于如来净土之行。谛听！谛听！善思念之，当为汝说！

——由此切入《佛国品》的核心教义。**接续见 `品01-心净国土净.md`**。

## 本文档总结

本段 (第 2-6 课) 承担《佛国品》的四项功能:

1. **交代法会时空与主体——五种圆满** (第 2 课)
2. **广赞菩萨众功德——为后文入不二法门铺底** (第 3-4 课前半)：此间种下"菩萨之智悲"、"演法如狮吼"、"深入缘起断诸邪见"等后文反复回响的关键词
3. **引入天龙八部与四众，以须弥喻佛庄严——法会庄严渲染** (第 4 课中-第 5 课前半)
4. **宝积菩萨供盖、变现三千大千世界及十方诸佛说法、赞佛十数偈——导出"净土之问"** (第 5-6 课)

其中宝积赞佛偈中的两偈具独立教义价值：

- **"说法不有亦不无，以因缘故诸法生，无我无造无受者，善恶之业亦不亡"**——中观缘起性空立场
- **"佛以一音演说法，众生随类各得解……各各随所解……或有恐畏或欢喜"**——一音三解的佛不共威神

此段虽以叙事和赞颂为主体，却是**理解本品核心命题"心净国土净"的前置条件**——若无对菩萨 (修因者) 与佛 (果地法王) 的具体功德认知，后文"菩萨净土之行"将失去支撑的人物与境地。

序分、功德赞、宝积赞佛偈所展开的一整个法会庄严场面，本身也是一次关于"佛国"的现量指示——佛陀此时此地所处之境，本就是佛国的显现。舍利弗后来所疑的"此土不净"，恰与此段所描绘的"如须弥显于大海"形成鲜明反差，为核心教义作了最深的戏剧铺陈。
