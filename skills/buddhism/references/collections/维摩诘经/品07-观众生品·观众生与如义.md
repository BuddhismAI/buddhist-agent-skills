# 品07 ·《观众生品》前半 ：观众生与如义、真实四无量、无住为本

## 一、承上启下 —— 由神变离分别到教义离分别

品6《不思议品》以**神变**直接示现"离一切分别即解脱"——大小、广狭、长短、能所、染净诸二元在菩萨手中悉皆消融。但神变是**现量境界**，对凡夫而言只是"听一个传奇"——若无与之配套的**教义展开**，离分别仍只是少数神通菩萨的境界，与凡夫修行无关。

品7 由此承担起**把品6 的现量境界转译为教义系统**的功能。文殊菩萨开篇问"菩萨云何观于众生"，正是把品6 的"以神变示离分别"落到一个具体可操作的修行问题上：**菩萨度生之际应如何看待众生**？答案不再是神变，而是**教义级的离分别**——观众生十一喻 + 二十喻 (人无我、法无我) → 起真实四无量心 (悲心不被执取所障) → 善依如来功德 (信心) → "X 以 Y 为本"递归至无住 (离戏)。本品后半天女戏剧将以现量演示完成本品教义 (见 `品07-观众生品·天女公案.md`)。

**结构上看，品6 与品7 是同一教义在两种媒介上的展开**：
- 品6 以**神变**显大小广狭染净 离能所分别 → 现量境界
- 品7 以**教义 + 戏剧**显观众生离实执、四无量离取著、烦恼即解脱、男女相非男非女 → 教理与戏剧并举

更深一层：**品7 把品5 末尾"调伏其心"段中所提到的"诸法如" (如幻、如焰、如水中月) 这一教学种子全面展开**——品5 已埋下"如"义为菩萨观众生与观自心的共同视角，至品7 由此种子开出十一喻 + 二十喻的完整"如观"系统。**品5 的"如"是种子，品7 的十一喻是花，品9 的"不二"是果。**

> 索达吉堪布在第34课开篇直接锚定本品定位：
>
> "《不思议品》主要讲能化者菩萨的功德，包括不可思议的神变、法门等。第七品《观众生品》主要讲所化众生。"

**品名考**：藏文译本与梵文本作《观天女品》——以本品后半部分天女与舍利弗对话为标题；唐译《说无垢称经》作《观有情品》——以"有情"译"sattva"。三个名称分别从能观对象 (众生)、戏剧主角 (天女)、术语 (有情) 三个角度命名同一品。

## 二、文殊问"菩萨云何观众生" —— 教义离分别的发起

> 尔时，文殊师利问维摩诘言："菩萨云何观于众生？"

文殊菩萨此问看似平常，实是直触菩萨道的根本困境：

**一边是大悲**：菩萨平等关爱每一个众生，如慈母疼独子，必生救度之心；
**另一边是空慧**：若执众生实有，度生之行被实执所摄，"实有执著非常坚固、猛厉，修悲心或度众生时就会因为难以承受而引发心理疾病。实执的根本是我执，如果没有打破我执，最后很可能放弃度化众生的誓愿"(堪布)。

**两边的张力**即菩萨道之所以"非易行"的根源——既要救度，又不可执取；既悲心遍及，又观空如幻。文殊此问相当于：**这一张力如何在见地上化解？**

维摩诘的回答是"观众生为如幻、如水月、如镜像、如阳焰、如响、如云、如沫、如泡、如芭蕉坚、如电"等十一喻 (鸠摩罗什译本中实为十一种比喻，藏译梵文本多几个、少几个)。**每一喻都同时含两层教义**：(a) 众生在世俗谛中如梦如幻地显现 (不堕断边)；(b) 众生于胜义谛中无微尘许实有可得 (不堕常边)。**两层合观即"现而无自性"——这就是品7 全部教义的总锚定**。

按前辈大德的判析 (堪布从智者《维摩经略疏》中引)，维摩诘的回答分三段：
- **幻观** (前十喻)：众生现而无自性 → 人无我
- **空观** (中十喻)：众生于"无"中亦不可得 → "空与不空"双面观
- **中观** (后五喻 / 唯识立场之中观，非远离四边八戏之中观)：既非有亦非无，离两边

## 三、观众生十一喻 + 二十喻 —— 全部清单

### 3.1 幻观（人无我·十喻·诸法现而无自性）

| # | 比喻 | 维摩诘原文 | 教义重点 | 堪布所引/解读 |
|---|---|---|---|---|
| 1 | 幻师所见幻人 | "譬如幻师，见所幻人，菩萨观众生为若此" | 幻化师明知幻人非实，故无贪著；菩萨观众生亦然 | 《四百论》"若谁见众生，如机关幻人，彼等极明显，能趣入胜位"；木匠与画师机器美女公案 |
| 2 | 智者见水中月 | "如智者见水中月" | 月影现而无自性 | 《入中论》"众生犹如动水月，见其摇动与性空" |
| 3 | 镜中面像 | "如镜中见其面像" | 寻之不可得 | — |
| 4 | 热时焰 | "如热时焰" | 远看似水近看无水，眼识所惑 | 阳焰为中观经典所喻 |
| 5 | 呼声响 | "如呼声响" | 空谷回声内外不可得 | — |
| 6 | 空中云 | "如空中云" | 因缘聚散无定 | 粗 (无常) + 细 (财富相貌青春刹那变化) 双层 |
| 7 | 水聚沫 | "如水聚沫" | 五蕴聚合无实质 | 《增一阿含经》"色如聚沫，受如浮泡，想如野马，行如芭蕉，识为幻法" |
| 8 | 水上泡 | "如水上泡" | 微风/日照即灭 | 《因缘品》"今日活着的众生，到明天就不一定还在了" |
| 9 | 芭蕉坚 | "如芭蕉坚" | 里里外外无坚实 | 中观破析的经典喻 |
| 10 | 电久住 | "如电久住" | 闪电不久住，喻人代代更迭 | "众生的五蕴一直在刹那变化" |

(在梵文本与藏译中，每一喻后都有 "菩萨观众生为若此" 重复句；鸠摩罗什据汉地语言习惯做了删减——"鸠摩罗什大师之所以这样做，是因为他在清净显现中得到了文殊菩萨的开许和加持。")

**前十喻要点**：从中观角度，这是**世俗当中的显现分**——众生现而无自性、如梦如幻。"《般若经》中说：诸法皆如梦如幻，预流果、一来果、不来果、阿罗汉果乃至涅槃和佛果，无一不是如梦如幻的显现。"

### 3.2 空观（中十喻·众生于"无"中亦不可得）

| # | 比喻 | 教义点 | 备注 |
|---|---|---|---|
| 11 | 如第五大 | 四大之外无第五大 (声闻乘不立虚空为大)，众生亦同 | — |
| 12 | 如第六阴 | 五蕴外无第六蕴 | 此喻藏文/梵文本无 |
| 13 | 如第七情 | 六根/六识/六处外无第七 | 唐译"第七根" |
| 14 | 如十三入 | 十二处外无第十三处 | 藏文/梵文本无 |
| 15 | 如十九界 | 十八界外无第十九界 | 藏文/梵文本无 |
| 16 | 如无色界色 | 无色界无粗色 (空) / 有细色 (不空) | 《维摩经略疏》两面分析；《如意宝藏论》: 若无色界完全无色，则转生色界欲界后所产生的色法有无因生过失，故应有细微色 |
| 17 | 如焦谷牙 | 焦谷不能发芽 (空) / 酥油灌溉可发芽 (不空) | — |
| 18 | 如须陀洹身见 | 一般预流果无身见 (空) / 特殊情况偶有 (不空) | 唐译为 "观一来果受第三有" |
| 19 | 如阿那含入胎 | 不来果不入欲界 (空) / 菩萨特意转生 (不空) | — |
| 20 | 如阿罗汉三毒 | 阿罗汉断界内三毒 (空) / 界外细微习气仍可有显现 (不空) | — |
| 21 | 如得忍菩萨贪恚毁禁 | 得忍菩萨不毁戒 (空) / 个别显现上有 (不空) | — |
| 22 | 如佛烦恼习 | 佛无烦恼习 (空) / 必要时为度生显现 (不空) | 《大智度论》：佛骂提婆达多为"死人"——苦切语有二种：垢心瞋骂 / 怜愍众生欲教化故；《大般涅槃经》：佛已渡烦恼大海而仍娶耶输陀罗 |

**关键教义工具**："**空与不空**"双面分析法 (智者大师《维摩经略疏》特有的解读) ——堪布特别采用此法："这些比喻都是从空与不空两个角度来分析的。从了义角度讲……从不了义的角度讲……"。这一分析法对应中观"二谛圆融"——无即有、有即无。

### 3.3 中观（后五喻·非有非无·离二边）

(堪布于第35课明示："这里的'中观'是唯识宗观点——既非空，也非不空，不堕于二边，并非远离四边八戏的'中观'。")

| # | 比喻 | 教义点 |
|---|---|---|
| 23 | 如盲者见色 | 盲人不见外色 (空) / 可见内识形态色 (不空) |
| 24 | 如入灭尽定出入息 | 灭尽定无息 (无分别故) / 有细微意识则有难察出入息 |
| 25 | 如空中鸟迹 | 凡眼无 / 天眼可见 (《维摩经略疏》引《央掘经》) |
| 26 | 如石女儿 | 中观论典常喻不可得 / 业报转化或医疗手段亦可有 |
| 27 | 如化人起烦恼 | 佛之化身无业烦恼 (悲愿游舞) / 显现上示三毒 |
| 28 | 如梦所见已寤 | 醒后不见梦境 / 醒后亦可回忆 |
| 29 | 如灭度者受身 | 阿罗汉不受身 / 个别意幻身界外变易受生 (堪布引迦叶尊者鸡足山入灭尽定故事+柏克森案例) |
| 30 | 如无烟之火 | 无因之火不可得 / 燃烧铁球虽有火却无烟 |

**总判**："总的来说，前面十个比喻是说菩萨观众生如幻，后面又讲菩萨观众生为空性。也就是说，众生在显现上似乎存在，但本体上并不存在，是空性的。"

**实修要点**：发了菩提心的人观自他众生时，应**两边不堕**——
> 一方面，不要觉得众生都是实有的，从而生起强烈执著，因为众生毕竟是空性的；另一方面，也不要认为众生就像石女儿一样根本不存在，从而漠不关心，因为众生毕竟有如梦如幻的显现。在这两者之间，要把握一个合适的度。

此即**世俗大悲与胜义空慧的结合点**——是品7 全部菩萨道教义的根基。

## 四、菩萨观众生即起真实四无量心

> 文殊师利言："若菩萨作是观者，云何行慈？"

文殊接着问的关键问题：既然众生如幻如空，那如何对其修慈悲喜舍四无量？是否会因为"空"而生不起慈悲？维摩诘的回答 (按窥基大师判)，从**九方面缘真如慈 + 五方面缘法慈 + 多方面缘众生慈**展开。

### 4.1 真实慈心 —— 三层结构

**总判**："菩萨作是观已，自念：'我当为众生说如斯法。'是即真实慈也。" ——**为众生讲经说法即是真实慈心**——这是真实慈的根本表征，不是普通意义上的善意。

> 《大法炬陀罗尼经》："慈心说法成大功德，能令佛法久住于世。"
> 《大乘密严经》："一切诸众生，生于种种见，欲令断诸见，为说于空理。"

#### 4.1.1 第一层：**缘真如慈** (九种)

| 慈名 | 经文 | 教义 |
|---|---|---|
| 寂灭慈 | "行寂灭慈，无所生故" | 缘真如无能取所取，无近取因之实质 |
| 不热慈 | "行不热慈，无烦恼故" | 菩萨已断烦恼，行慈不带烦恼 |
| 等之慈 | "行等之慈，等三世故" | 慈心周遍三世无间 |
| 无诤慈 | "行无诤慈，无所起故" | 已断我执，无人我之争 |
| 不二慈 | "行不二慈，内外不合故" | 内慈不与外缘相合 |
| 不坏慈 | "行不坏慈，毕竟尽故" | 法身证得不生不灭故慈不坏 |
| 坚固慈 | "行坚固慈，心无毁故" | 慈心如金刚，《华严经》"无量无数劫，观察众生界，众生非众生，坚固士所行" |
| 清净慈 | "行清净慈，诸法性净故" | 慈心无染污，相合胜义清净 |
| 无边慈 | "行无边慈，如虚空故" | "菩萨以大慈心爱着每一个众生——爱，也许是对慈心最贴切的表达" |

**核心命题**：真正的菩萨大慈不是无感情、不是冷静——而是**断除"自他能所烦恼染著"四重障碍后的纯净大爱**。"如果对众生没有任何感情，那是石头，不是菩萨"——堪布反复破"佛教慈悲是冷漠"的误解。

|---|---|---|

| 菩萨慈 | "行菩萨慈，安众生故" | 安稳成熟众生，圆满自他二利 |
| 如来慈 | "行如来慈，得如相故" | 证得实相 |
| 佛之慈 | "行佛之慈，觉众生故" | 自觉觉他 |
| 自然慈 | "行自然慈，无因得故" | 第一义谛慈，自然流露不依因缘 |

(唐译还有"修独觉慈"。)

(唐译还有妙愿慈/大力慈/若那慈/神通慈/摄事慈五种。)

### 4.2 真实悲心、喜心、舍心

> 文殊师利又问："何谓为悲？"
> 答曰："菩萨所作功德，皆与一切众生共之。"
>
> "何谓为喜？"
> 答曰："有所饶益，欢喜无悔。"
>
> "何谓为舍？"
> 答曰："所作福祐，无所悕望。"

**关键观察**："此处对于悲心的解释，与其他一些大乘经论有所不同。" ——本经的"悲喜舍"以**菩萨的姿态**而非"对苦众生的同情"来界定：
- **悲** = 将一切功德回向众生 (不留己用的回向心)；
- **喜** = 利他时无悔欢喜 (不因损己而后悔的承担心)；
- **舍** = 不求回报 (无交易性的纯粹施与)。

《文殊师利问经》："日月照诸花，无有恩报想，如来无可取，不求报亦然。" ——日月照花不求回报，菩萨利众生亦然。这是"舍"的精髓——离能所占有相、离回报期待。

**与品4 弥勒段、大圆满前行四无量心的呼应**：本品"真实悲" = 大悲心 (拔苦)、"真实喜" = 随喜心 (无悔)、"真实舍" = 平等舍 (无染)。其与《大圆满前行》第十一次第发心节"四无量心"是同一教义在两个集合中的表述——前行是"修法次第"角度，本经是"证境层面"角度。

## 五、十二问答系列 —— 从生死到无住

文殊菩萨进而代初学菩萨问十二个问题，构成一条**从畏死 → 依如来 → 度众生 → 除烦恼 → 行正念 → "X 以 Y 为本"递归**的教义链。本节是品7 教理推进的最重要段落。

| 序 | 文殊问 | 维摩诘答 |
|---|---|---|
| 一 | 生死有畏菩萨当何所依 | 当依如来功德之力 |
| 二 | 欲依如来功德力当于何住 | 当住度脱一切众生 |
| 三 | 欲度众生当何所除 | 除其烦恼 |
| 四 | 欲除烦恼当何所行 | 当行正念 |
| 五 | 云何行于正念 | 当行不生不灭 |
| 六 | 何法不生何法不灭 | 不善不生善法不灭 |
| 七 | 善不善孰为本 | 身为本 |
| 八 | 身孰为本 | 欲贪为本 |
| 九 | 欲贪孰为本 | 虚妄分别为本 |
| 十 | 虚妄分别孰为本 | 颠倒想为本 |
| 十一 | 颠倒想孰为本 | 无住为本 |
| 十二 | 无住孰为本 | **无住则无本。从无住本，立一切法** |

### 5.1 前六问 —— 菩萨道与如来加持的相应

**第一问**："菩萨于生死畏中，当依如来功德之力。" ——菩萨自力面对生死有困难，需依如来加持。
**第二问**："菩萨欲依如来功德力者，当住度脱一切众生。" ——**与如来相应的最有效方式不是祈祷，而是度生**。"因为如来心系众生，如果我们尽心尽力地救度众生，如来肯定会很欢喜的。反之，如果经常害众生、讨厌众生、嗔恨众生，同时又求佛保佑，那就完全无法与佛相应了。"

**阿底峡尊者公案** (堪布引)：阿底峡尊者对仲敦巴说："我的手很痛，你给我加持一下，因为你有一颗善良的心。" ——**善心即是加持力的本源**。

**第三问**："欲度众生，除其烦恼。" ——度生 = 除其烦恼，不仅仅是讲经/做佛事的形式。
**第四问** ："欲除烦恼，当行正念。" ——《佛所行赞》"正念存于心，众恶悉不入"；禅宗"不怕念起，只怕觉迟"。
**第五问、第六问**：行正念 = 行不生不灭 = 不善不生善法不灭。"这里的'不生不灭'，并非中观所讲的'不生不灭'" ——是世俗层面的四正勤含义 (已生恶令断、未生恶令不生、未生善令生、已生善令增长)。

> 《法句经》："诸恶莫作，诸善奉行，自净其意，是诸佛教。"
> 藏传三乘解释："诸恶莫作"=声闻乘，"诸善奉行"=菩萨乘，"自净其意"=无上密法。

### 5.2 后六问 —— "X 以 Y 为本"的递归 直至 无住

**这是经中最关键的教证段之一**。文殊连问六次"以 Y 为本"的根，每次维摩诘退一层：

```
善/不善 ──[以]──> 身  ──[以]──> 欲贪 ──[以]──> 虚妄分别
                                                   │
                                                   ▼
                                                 颠倒想 (常乐我净四颠倒)
                                                   │
                                                   ▼
                                                 无住
                                                   │
                                                   ▼
                                                 无住则无本
```

**邪见的根追到底**——所有邪见 (六十二见) 之根 = **萨迦耶见** = **欲贪** = **虚妄分别** = **颠倒想 (常乐我净)** = **无住**。

> 第十二问的总判——**经中关键教证之一**：
>
> "无住则无本。文殊师利！从无住本，立一切法。"

**双层含义**：
1. **空性面**："无住之本"指一切万法本体不常不断、远离一切边——此即空性。
2. **缘起面**："立一切法"指虽万法本无实质，但于业感众生面前可显现——此即缘起显现。

> 《入中论》："如是一切法虽空，从空性中亦得生，二谛俱无自性故，彼等非断亦非常。"

**与中观四边破的关联**：本经"无住则无本"是中观"无生而生、无住而住"的另一种表述方式——空性与缘起圆融。**此教证是品7 教义部分的最高点，也是品9《入不二法门》的直接预演** (不二即"无住" — 离能住所住)；同时为下半部分天女戏剧 (`品07-观众生品·天女公案.md`) "言说文字皆解脱相、淫怒痴性即是解脱、菩提无住处" 一系列教证铺出根基。

## 七、堪布宁玛派视角

**(1) "如" 义与大圆满"本来如是"** —— 维摩诘观众生十一喻 (如幻、如水月、如镜像、如阳焰、如响、如云……) 的"如"字，与大圆满"诸法本来如是" (gnas lugs ji bzhin) 同源——均指"超越分别衡量的本然显现"。前者从凡夫修行入手 (观众生离实执)，后者从果位心性直指 (本来如是)，但底层教义同一。**品7 是大圆满见地在显宗经典中的预先演示**。

**(2) "真实四无量"与利他自然** —— 本经把四无量定义为"说法、共功德、无悔、无悕望"——已超出"修法对治"层面，进入"行处自然" 层面。这与大圆满"无勤任运" 的修行底色相通——真实菩萨道不是"勤修慈悲"，而是慈悲已成自然流露。

**(3) "无住为本"与大圆满"安住本住"** —— "从无住本，立一切法"与大圆满"住于无住之本性"是同一意趣的不同表述。下半部分天女"言说文字皆解脱相"段会进一步引出本解脱/自解脱教义 (见 `品07-观众生品·天女公案.md` §堪布宁玛派视角)。

## 八、关键经文摘录

**观众生十一喻 (人无我)**：
> "维摩诘言：'譬如幻师，见所幻人，菩萨观众生为若此。如智者见水中月，如镜中见其面像，如热时焰，如呼声响，如空中云，如水聚沫，如水上泡，如芭蕉坚，如电久住。'"

**空观/中观二十喻 (法无我·空与不空双面)**：
> "如第五大，如第六阴，如第七情，如十三入，如十九界。如无色界色，如焦谷牙，如须陀洹身见，如阿那含入胎，如阿罗汉三毒，如得忍菩萨贪恚毁禁，如佛烦恼习。如盲者见色，如入灭尽定出入息，如空中鸟迹，如石女儿，如化人起烦恼，如梦所见已寤，如灭度者受身，如无烟之火。菩萨观众生为若此。"

**真实慈心总判**：
> "菩萨作是观已，自念：'我当为众生说如斯法。'是即真实慈也。"

**真实悲喜舍**：
> "菩萨所作功德，皆与一切众生共之。" / "有所饶益，欢喜无悔。" / "所作福祐，无所悕望。"

**X 以 Y 为本递归 + 无住总判 (经中关键教证)**：
> "善不善孰为本？答：身为本。身孰为本？答：欲贪为本。欲贪孰为本？答：虚妄分别为本。虚妄分别孰为本？答：颠倒想为本。颠倒想孰为本？答：无住为本。无住孰为本？答：无住则无本。文殊师利！从无住本，立一切法。"

**跨品教证 (堪布引)**：
> 《四百论》："若谁见众生，如机关幻人，彼等极明显，能趣入胜位。"
> 《入中论》："众生犹如动水月，见其摇动与性空。" / "如是一切法虽空，从空性中亦得生，二谛俱无自性故，彼等非断亦非常。"
> 《增一阿含经》："色如聚沫，受如浮泡，想如野马，行如芭蕉，识为幻法。"
> 《大法炬陀罗尼经》："慈心说法成大功德，能令佛法久住于世。"
> 《大乘密严经》："一切诸众生，生于种种见，欲令断诸见，为说于空理。"
> 《华严经》："无量无数劫，观察众生界，众生非众生，坚固士所行。"
> 《文殊师利问经》："日月照诸花，无有恩报想，如来无可取，不求报亦然。"
> 《佛所行赞》："正念存于心，众恶悉不入。"
> 《大乘经庄严论》："首荷有情担，众尊缓行丑，自他种种缚，精进当百倍。"
> 《四百论》："佛知作不作，应说不应说。"
> 古德注疏：智者《维摩经略疏》(空与不空双面分析)、窥基大师 (慈心九 + 五 + 缘众生分类)、湛然《维摩经略疏》(三层判)。
