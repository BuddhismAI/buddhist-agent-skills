# 《维摩诘经》结构总览

《维摩诘所说经》(梵 Vimalakīrti-nirdeśa，藏译名《无垢称经》) 是大乘佛教最重要的叙事经典之一，以戏剧性手法演说般若空性与菩萨道。索达吉堪布 2018-2019 年依姚秦鸠摩罗什译本(十四品)传讲 66 课，融入藏地《无垢称经》传承及汉地古大德注疏(僧肇《注维摩诘经》、慧远《维摩义记》、智者《维摩经玄疏》、吉藏《净名玄论》等，参考书目二十七种)。

## 本经的核心架构——戏剧张力中的般若

《维摩诘经》不同于《金刚经》《心经》等以直陈空义为主的般若经，而是**用剧场式叙事展开空性教义**：维摩诘居士"示疾"为线索，通过佛弟子、菩萨、文殊、诸天女的依次登场，层层推进直至"默然无言"的顶点。理解此经不能只按十四品的线性目录去读，必须读出**其内在的戏剧张力与教理递进**。

```

    ─────────                        ─────────                      ─────────
    品1 佛国    →    心净国土净     品3 弟子品   十大弟子      品5 文殊问疾    文殊独承
    品2 方便    →    居士示疾       品4 菩萨品   四大菩萨          ↓
```

### 架构解读——为何品序如此

**品 1-2 (建基与铺垫)**
- 品1《佛国》以"心净则国土净"(详见 `品01-心净国土净.md`) 奠定全经的本体论立场：**一切外境由心所现**，因此佛国、净土、修行对境均非离心他法。此是后文谈"不二"的地基——若国土实有，则能所二分不可除。本品序分部分 (法会因缘、菩萨功德赞、宝积供盖赞佛偈) 见 `品01-序分与菩萨功德.md`。
- 品2《方便》以"维摩诘示疾"启动叙事：**疾非实疾，乃方便**。"方便" (upāya) 是菩萨以无住之心触世间法的核心德相，也暗含全经"烦恼即菩提"的种子——若疾为方便，则任何境皆可转为道用。

**品 3-4 (十大声闻、四大菩萨退却)**
- 这两品表面上是"人物铺垫"，实则承担极严肃的教义功能：**通过弟子/菩萨的"不堪"，显示小乘及初阶大乘所执诸定见 (如宴坐、乞食、持律、入灭定、观空、说法、持戒……) 皆有所住，皆被维摩诘以般若破**。
- 每一段"过去我曾……维摩诘呵责我曰……故我不堪"的叙事，都是一次对小乘(或有住大乘)知见的**个别破斥案例**。
- 此二品为后面"不二"铺好对比：小乘知见必有二(凡圣、染净、戒破、定散)，而本经终将归于"不二"。

**品 5 (戏剧转折点)**
- 只有文殊菩萨承担前往问疾的任务，带出全经中段。**文殊代表"解"，维摩诘代表"证"**，二圣相见即是"般若之眼"与"般若之行"相照——由此开启从品5直到品9的主体对话。
- 问疾对话中"以空病空"等教义为全经论证空性最犀利处。

**品 6-9 (核心教义的四重展开)**
- 品6《不思议》以**神通示不可思议解脱**：须弥入芥子、三万二千师子座、大千国土放掌中……"不可思议"非夸饰，而是直显**离分别心境界**即是解脱。此即经之异名"不可思议解脱经"之出处。
- 品7《观众生》由天女散花、舍利弗转女身等故事，点出**"如"义** (诸法如、众生如、佛如)——为品9"不二"做语义锚定。
- 品8《佛道》提出"**非道是佛道**"，即烦恼、贪嗔、外道、五无间皆可转为道——这是"烦恼即菩提"教义最集中的展开。(此处与宁玛派"显密圆融"教义高度相通，堪布特别于本品多处点出与大圆满直指心性的关联——见 `品08-佛道品·非道是佛道.md` + `品08-佛道品·菩萨行与莲华喻.md`)
- 品9《入不二法门》是全经顶点：**三十一位菩萨各说自己的"入不二之门"(染净、善恶、菩提烦恼、生死涅槃……)，文殊总结"无言无说是真入不二"，维摩诘以"默然无言"作最后回答**。从言说到寂默，本经的论证结构到此抵达极致。

**品 10-11 (扩展与回照)**
- 品10《香积佛》以"香积国"的菩萨众形成对照：香积国诸菩萨**以香供养即说法**，显示说法方式多种多样，众生根基悬殊。
- 品11《菩萨行》综合前文所有教义，整理为菩萨在秽土行道的完整纲领。

**品 12-14 (尾声：本源、法胜、付嘱)**
- 品12《见阿閦佛》揭示**维摩诘的出处——妙喜国无动如来刹土**，呼应品1"心净国土净"：诸佛净土其实不离此土。
- 品13《法供养》确立**"法供养胜于一切财供养"** 的判定，为经典流通建立教证。
- 品14《嘱累》是佛陀将经典付嘱弥勒等未来菩萨，圆满结束。

### 全经的核心命题

若以一句话概括《维摩诘经》的全部教义，即是：

> **不二——染净不二、生死涅槃不二、烦恼菩提不二、凡圣不二、能所不二；而此"不二"之终极表达不是任何一句对立统一的话，而是寂默。**

一切品目均服务于此命题。品1 由"心净国土净"说**能所不二**的地基；品2-4 以叙事展示一切"住着" (住空、住定、住戒、住慢) 皆是二；品5-9 层层说破二；品9 以默然显不二；品10-14 以不二之见再回照秽土行道。
