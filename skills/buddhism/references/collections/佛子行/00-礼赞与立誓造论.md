# 礼赞与立誓造论

本单元对应《佛子行释》科判中的**甲一（礼赞与立誓造论）**，为注释者根桑秋札尊者开篇的三颂礼赞，以及无著菩萨原论四句礼赞立誓（「那莫罗给夏Ra雅」+ 两颂）。在此之后，论文自「已获暇满大舟时」起正式进入甲二（所造论之自性）——即三十七颂主体。

## 一、作者、注疏者、译讲者的身份

### 根论作者：无著菩萨（藏地·无著贤，1297–1371）

此处的「无著菩萨」**是藏地的无著贤（Ngulchu Thogme Zangpo），并非印度六庄严之一的无著（Asanga）**。索达吉堪布在 L1 特别点明：

> 作者首先顶礼诸佛菩萨，再顶礼《佛子行》的作者无著菩萨。此处是指藏地的无著菩萨，而不是印度的无著菩萨。

- 藏传传统尊其为**观世音菩萨的化身**。
- 其生平以「**对亲怨同等修慈心**」闻名，《佛子行》即是其终生修行的归纳。
- 传记参照益西坚赞大师撰写的《无著菩萨传》（索达吉堪布曾翻译，L2 推荐研读）。

### 注疏作者：根桑秋札尊者

> 作者根桑秋札尊者，是道孚很了不起的一位格西，也是法王的前世。（L1）

- 道孚地区的格西。
- **法王如意宝晋美彭措的前世**——这一点使此注疏与宁玛派法脉直接相连。

### 译讲者：索达吉堪布（2022）

- 2022年4月起翻译并讲授，共53课。
- L2 中堪布提示：「正在学习的这部注释，是法王的前世所造。虽然我没有传承，但我们这样读一遍，也算是一次共同学习。」

### 注疏副标题「甘露妙瓶」

L1 中堪布开示「甘露妙瓶」的含义：

> 此《佛子行释》的副标题是"甘露妙瓶"。我们现在接触的全是"甘露"，前段时间刚喝了"治病甘露"，现在又是一瓶甘露。

此处「治病甘露」指麦彭仁波切的《治病甘露流》，堪布于2022年4月14日传讲圆满，紧接着便开始传讲本论——两瓶「甘露」相续而出。

## 二、注释者的三颂礼赞

根桑秋札尊者在开篇以三颂分别顶礼三重对境：**诸佛菩萨、无著菩萨本人、以及《佛子行》所诠的菩萨道本身**。

### 第一颂：顶礼诸佛菩萨

> 以大心力肩负引，遍空众至菩提任，
> 已获将获大果位，佛陀菩萨尊前礼。

颂意（L1）：以极大的心力肩负起引导遍虚空界众生直至菩提的重任，在**已获得、将获得**圣果的诸佛菩萨前恭敬顶礼。

### 第二颂：顶礼无著菩萨

> 恒不舍以大悲眼，凝视有情之誓愿，
> 观音幻化之法王，佛子无著尊前礼。

颂意（L1）：恒时不舍以大悲慧眼凝视所有有情的誓愿，在**观世音菩萨的化身法王——无著菩萨**面前恭敬顶礼。

此颂明确确立：**无著菩萨 = 观音化身**。这是后文将《佛子行》视为观音加持所流出的基础。

### 第三颂：立誓宣讲《佛子行》

> 所有如来与佛子，已行将行胜津梁，
> 由乐至乐彼妙道，佛子行理此宣讲。

颂意（L1）：所有佛菩萨已行持或将行持的殊胜津梁，从快乐前往快乐的必经妙道，就是《佛子行》。

此颂为注释者本人的**立誓**——相对于下文无著菩萨原论本身的立誓句，此为「注释之立誓」。

## 三、原论礼赞与立誓四颂

### 梵文顶礼句（略说礼赞）

> 那莫罗给夏Ra雅！

**意为顶礼观自在菩萨**（注疏原文）。

按科判为「乙一（礼赞）·丙一、略说」。

### 广说礼赞颂

> 虽见诸法无来去，然唯精勤利众者，
> 胜师怙主观音前，三门恒时敬顶礼。

注疏释义：

> 虽然以智慧如实照见了一切所知万法有无、来去、常断、一异等戏论八边均不成立的真如实相，但仍然以大慈大悲一心一意精勤行持利众生事业并为我们开示大乘道的殊胜上师，实际上与诸佛大悲自相的圣者、成为无依无怙苦难有情的依怙——观世音菩萨无二无别。在他们面前，我三门恒常毕恭毕敬地顶礼。

此颂是**智悲双运**的标准表达：

- 「虽见诸法无来去」——照见戏论八边不成立的**智慧分**。
- 「然唯精勤利众者」——精勤利生的**大悲分**。
- 所顶礼对境是**无著菩萨的上师**，而这位上师「与观音菩萨无二无别」。

L1 中堪布进一步点出三层同体：

> 此处讲到无著菩萨的上师与观音菩萨无二无别，实际上，**他自己也与观音菩萨无二无别**。在某种意义上，**文殊菩萨和观音菩萨也是无二无别的**。

为何以观音为礼敬对境？因为：① 无著菩萨本身即观音化身（见注释者第二颂）；② 本论所讲菩萨行以大悲为骨干，观音为大悲本尊；③ 这使整部《佛子行》成为一部**观音加持所开显的菩提心论典**。

### 立誓造论颂

> 利乐之源诸佛陀，修持正法而成就，
> 亦依了知其行故，于此宣说佛子行。

注疏释义：

> 作为暂时之利益、究竟之安乐的源泉——圆满正等觉诸佛，是从什么因中产生的呢？唯一来源于学道阶段如理修行大乘妙法。那到底是怎样的呢？这绝对依赖于了知最初以大悲心为根本而发心、随后积累布施等方便分、证悟诸法无自性的智慧分即六度所摄的二种资粮等行为。因此，为了利益那些想得到佛果而修学大乘道的人，在此宣说实际修学出自大乘三藏及注疏的所有佛子之意乐、行为的道理。
>
> 这以上是立誓句。

此颂的论证结构（L2 堪布开示）：

1. **果**：暂时利益、究竟安乐之源——诸佛。
2. **因**：唯一来源于学道阶段如理修行**大乘妙法**。
3. **如何修**：以**大悲心发菩提心** → 修**布施等方便分** + **证悟无自性之智慧分** → 即**六度所摄的二种资粮**（福德资粮、智慧资粮）。
4. **立誓**：故为想得佛果的大乘行人宣说佛子的**意乐与行为**。

堪布补充造论缘起（L2）：

> 为什么这次会传讲《佛子行释》呢？因为很多人把《佛子行》当作日常课诵，每天都在念。依靠《佛子行》，我们的相续当中一定会生起菩提心。

## 四、立誓颂所引两大教证

注疏在「这以上是立誓句」之后，以两段教证为立誓颂作支持。

### 《普明现前菩提续》

> 密主，遍知智慧由悲心之根本中生，由菩提心之因中生，依方便至究竟。

L2 开示：菩提心和悲心很重要，由此可生佛陀遍知智慧。**悲心对境是所有众生**，并非仅限于对自己好、自己喜欢的人——是否对陌生人、关系不好的人也生起悲心，是检验悲心真伪的标尺。

### 《宝鬘论》（龙树菩萨）

> 欲自世间众，得无上菩提，菩提心为本，坚固如山王，悲心遍十方，不依二边智。

L2 释义：若想令自他众生获得无上菩提，就需要**坚固如山王般的菩提心**，而菩提心的根本是**大悲心**与**无二慧**（即不依二边智）。

两段教证共同把立誓颂定位在「**悲心 → 菩提心 → 方便与智慧二资粮 → 佛果**」这一因果链条上。

## 五、无著菩萨具备造论的三种因缘

注疏接下来解释：这位大菩萨之所以能为他众讲解佛子行为，是因为具足造论的**三种因缘**（L2 展开）：

### 1. 依止过众多善知识

L2 堪布补充：

> 无著菩萨之所以能为众生讲解佛子的行为，因为他曾依止过**四十位善知识**，以及两位如意宝般的根本上师——**仁钦西日和索南扎巴**。

注疏脚注细分此四十位：广闻博学时依止的十位、对其恩重如山的十位、于其前听闻甚深法要的十位、于其前修习广大清净行的十位。

### 2. 精通浩如烟海的大乘经论

### 3. 面见殊胜本尊

L2 补充：「面见了**以观音菩萨为主**的殊胜本尊」——再次呼应原论以观音为礼敬对境的脉络。

### 4. 行持上的亲证

注疏强调「尤其按照其生平事迹中记载，他本人也是**如理修学佛子的一切行为**」——作者资格不仅在于学识与见地，更在于**修行与所造论一致**。L2 提到离拉萨不远有「水银宝洞」是无著菩萨闭关之处。

这四条合起来构成**造《佛子行》论典所需的完整因缘**：依师、博学、本尊加持、亲证修行，四者缺一即失去说此论的资格。

## 六、时代语境（L1 堪布特别开示）

索达吉堪布在 L1 传讲「那莫罗给夏Ra雅」时，特意将论典放入2022年的时代背景：

> 现在全世界的疫情很严重，包括老年人和医务人员，国内外已死了很多人。与此同时，有些国家之间的战争也很惨烈，导致许多难民流离失所……整个世界真的很苦，今天我们学习了《佛子行》——**观世音菩萨加持的殊胜论典**，等会儿共同念观音心咒，将此功德回向给正在受苦的所有众生。

这一层时代语境不是泛泛开场，而是指向论典的**现实修行功能**：在众生苦相毕现之时讲此论，正契合观音大悲救护众生的本愿，也让《佛子行》从一部文本回到其最初的用途——**在苦难时代生起并保任菩提心**。

## 七、本单元在全书结构中的位置

按注疏开篇科判：

- **甲一、礼赞与立誓造论**（本单元） → 乙一 礼赞（丙一 略说「那莫罗给夏Ra雅」/ 丙二 广说「虽见诸法无来去」颂） + 乙二 立誓造论（「利乐之源诸佛陀」颂）。
- **甲二、所造论之自性** → 乙一 入前行法之方式（七支）→ 乙二 正行宣说三士道。从第5颂「已获暇满大舟时」起正式进入。
- **甲三、结尾**。

因此读者读完本单元，应当已经：

1. 理解「无著菩萨」在此特指藏地无著贤，与印度无著不可混；
2. 知道注释来自根桑秋札尊者（法王前世），副标题为「甘露妙瓶」；
3. 明白为何全论以观音为加持源（无著＝观音化身 / 其上师与观音无二无别 / 本尊面见中以观音为主）；
4. 把握立誓颂所立的论证轴：**大悲心 → 菩提心 → 二资粮六度 → 佛果**；
5. 对作者造论的四重资格（依师、博学、本尊加持、亲证）有基本概念。

由此进入甲二的第一颂——「已获暇满大舟时」——便能接上三士道前行的第一环「暇满难得」。
