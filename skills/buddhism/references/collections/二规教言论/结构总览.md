# 二规教言论 · 结构总览

## 论典与讲解渊源

《二规教言论·积聚天德之宝》为全知麦彭仁波切（麦彭嘉样杰布让当）造于宗萨扎西拉哲，由欢喜行持高尚妙规的"嘉靠公确"劝请而作。本论藏文原名《规矩教言论》，索达吉堪布因其内容兼摄佛规（出世间规矩）与世规（世间人格规范）两层，译名为《二规教言论》。

**本论在法王如意宝座下的传承地位**：据索达吉堪布 2006 年讲解时忆述，法王晋美彭措生前对此论反复宣讲，尤其 1987 年在五台山清凉寺文殊菩萨加持大石旁的森林中曾专门传讲。法王座下众高僧大德云集，其"不可思议的超胜功德之外，做人方面极为圆满"是关键原因之一。这一背景是理解本论定位的钥匙：它不是"低级入门"，而是高深法门的**根基**。

**索达吉堪布的两次讲解版本**：
- **2003 年版（课程038）**：1998 年首次翻译即讲过一遍；2003 年再讲一遍时道友据录音整理，与 1998 年文字稿合编成讲记；收入本 collection 的是此版精校本，分为序 + 十二章。
- **2006 年版（课程039）**：作为《入菩萨行论》讲解期间的配套教言，共 12 课。不以分别念添枝加叶，字面意思为主，但配以堪布亲历的当代公案与实修反思。

两版覆盖相同颂词，但侧重不同：
- **2003 版**更接近"字面讲义"：引《格言宝藏论》《亲友书》《宝鬘论》《水木格言》等教证更密集，术语辨析更细（如"应供"、"沙门四法"等），适合作为论义底稿。
- **2006 版**更接近"活的教言"：堪布结合菩提学会学员的实际情况、当代社会现象（网络、赌博、会供饮酒、出家人作风等）、个人修行反思反复观照颂词，更具对治性。

## 本论的立论策略

### 为什么麦彭仁波切要造这部论

开篇明确点出："在能取所取未消融于法界前，本论都可称得上是任何一位佛教徒在修行过程中必不可少的指导闻思修的宝典。"——换言之，除非证悟二取双亡，否则不能跳过此论。

**核心命题**：`世法即是佛法根`（颂3）。世间贤善人格不是大乘的"预备动作"，而是高深佛法的**生长土壤**。六祖"佛法在世间，不离世间觉，离世觅菩提，恰如求兔角"一句颂在本论被反复引用，是全论的世间-出世间关系纲领。

这一命题对治当前修行界两种常见颠倒：
1. **上品颠倒**：希求大圆满、大威德、中观般若等甚深法，却对做人基础不屑一顾——注酥油的金器用陶罐代替，再殊胜的法也无法住心。
2. **下品颠倒**：以"我已证悟空性"为借口放纵行为——但一个装模作样的禅师，来世做牛还债的公案（详见 `04-不放逸.md`）说明：不到真正空净地步，胡说大话是要感果报的。

### 十种功德与大树譬喻

全论以"大树"为核心譬喻（见颂：`正士高尚行为者，树根妙慧与稳重...`），将**正士高尚行为**分解为十种不可或缺的功德，配以**两种善果**：

```
                        树果：安乐 + 声誉（得果）
                         ↑
              树花：信心 + 布施
                         ↑
              树叶：知恩 + 利他
                         ↑
              树枝：正直 + 誓言坚定
                         ↑
              树茎：有愧 + 不放逸
                         ↑
              树根：妙慧（智慧） + 稳重
```

这不是装饰性的比喻，而是**功能性的层级结构**：
- **根（智慧+稳重）**：决定能否吸收养分。没有取舍智慧则一切功德无以判别，没有稳重则智慧如墙头草随风倒。二者互为条件。
- **茎（有愧+不放逸）**：决定养分能否输送。有愧是止恶的内在约束（"二规之妙衣"），不放逸是行善的持续动力（如"居于危崖"之心态）。
- **枝（正直+誓言坚定）**：决定树形能否展开。正直是"天人道"，狡诈是"邪魔道"（颂："趋入天道住天门，趋入魔道住魔门"）；誓言坚定则分上中下三等（石刻文字 / 土中写字 / 水中花纹，出自《亲友书》）。
- **叶（知恩+利他）**：决定遮荫利他能力。知恩是正直的天然延伸，利他则是全论通往大乘的核心闸门。
- **花（信心+布施）**：决定能否开花结实。信心对三宝、四谛、因果为境，是"佛法之根"；布施则是断贪积福、转无常财富为来世安乐的唯一方便。
- **果（安乐+声誉）**：前十种齐备后，**自然生起**今生来世寿长无病、悦意安宁、财富圆满、众人恭敬、名誉遍世——"如天鼓声遍满世间"，不需自我宣传。

**此等缺一不可也**——这不是修辞，而是命题。一个功德缺失，整棵树就不完整，已得的功德也会泄漏。

## 本论的内部逻辑张力（品与品的关系）

这不是十二章的平铺，而是有严格递进：

**第一张力：智慧 vs 稳重**（树根内部）。2006 版第5课末过渡时，堪布点明："有智慧当然最好，但没有也不要紧，只要有了稳重这一功德，其他的功德自然会具足"（详见 `02-稳重.md`）。智慧易得而稳重难守，故麦彭仁波切将稳重置于与智慧并列的根本位。二者的对治对象不同：智慧对治愚痴蠢动，稳重对治飘浮散漫。汉地金山寺持律师的公案（起初因愚笨被调走，却以稳重背诵多部经论终成法师）是稳重胜智慧的典型。

**第二张力：有愧 vs 不放逸**（树茎内部）。有愧是"知道不能做什么"的羞耻底线，不放逸是"知道必须做什么"的持续警觉。二者对应止恶与行善两面——无有愧则堕为无耻裸奔者，无不放逸则堕为虚度光阴者。本论对"无愧九相"与"放逸九过"的并列排布，是全论最繁复也最针对时代的诊断。

**第三张力：正直 vs 誓言坚定**（树枝内部）。正直是空间维度（不偏不倚），誓言坚定是时间维度（不变不移）。正直决定**当下判断**的公正性，誓言坚定决定**跨时段一致性**。两者合一才能让行为有可预测、可信赖的形态——"树枝"之所以重要，是因为它决定了一棵树是否"能被别人抓住"。

**第四张力：知恩 vs 利他**（树叶内部）。知恩是**对过去的承认**（已受者），利他是**对未来的承诺**（将予者）。一个不知恩的人不可能真正利他，因其连自身最切近的恩人都无感觉。2006 版第11课开篇堪布紧接着由"知恩图报"进入"利他"，正是这一逻辑递进。

**第五张力：信心 vs 布施**（树花内部）。信心是**内在所缘**（三宝、四谛、因果），布施是**外在行为**（财、法、无畏）。没有信心的布施是世俗慈善，没有布施的信心是空口愿力。故此二者构成大乘入门的心-行一对。

## 与其他 foundations 典籍的定位比较

**与《入行论》**：入行论从菩提心为纲铺展六度四摄，是"大乘菩萨道的完整哲学与实践"；《二规教言论》则在入行论开讲**之前**或**同时**提供的"做人根基校准"。堪布 2006 年讲《入行论》期间配讲本论，原因即在于"《入行论》虽然甚深，但没有人格基础就如金器盛乳汁倒进陶罐会裂开"（第1课）。两论为"纲-基"关系，非平行。

**与《佛子行》**：佛子行是 37 颂的压缩菩萨行手册，强调具体情境下的转化窍诀；《二规教言论》则停留在**人格层**，不进入"被骗/被伤/被辱"等菩萨行境界。先有二规人格，再学佛子行对应具体逆境。

**与《修心七要》**：修心七要是菩提心的窍诀化归纳（胜义+世俗+恶缘转道用等）；《二规教言论》在其之前一层——**连修心的法器都还没打磨好**时，先用二规造出合格的"人格容器"。

**与《大圆满前行》**：前行的共同外前行（暇满-无常-轮回-因果-解脱利益-依止上师）是**从见解层面**生起出离心；《二规教言论》是**从行为层面**校准做人基础。两者对出离心的关系是"见"与"行"的互补，非替代。

故本 collection 在 foundations topic 中的定位独特：**它不在五加行 / 菩提心 / 六度任一传统框架内，而是所有这些框架能否发挥作用的"底层土壤"**——堪布在 2006 年第12课末总结时说："我这次传讲……主要是看到外面很多居士……基本的做人不太如法，所以很希望大家都懂得这个道理。"

## 本论的对境：谁最需要学这部论

麦彭仁波切与索达吉堪布在讲解中反复点名几类人需要本论为镜：

1. **自认为已学佛多年但人格始终未转变者**：只背诵仪轨颂词，但说话不可靠、做事半途而废。
2. **追求"高大上"法门而跳过基础者**：刚皈依就求大圆满，闭关专修托嘎却没修加行（堪布引慈诚罗珠堪布"白修了，徒劳无益"公案，见 `00-总论·树喻与二规基础.md`）。
3. **发心者与辅导员**：菩提学会里发心部门负责人、辅导员、当管家的道友——本论的稳重、正直、誓言坚定三品是这些角色的直接工作手册。
4. **末法时代的在家居士**：在赌博、网络、情感浪潮中迷失的佛弟子——本论的"无愧九相"几乎每条都可对号入座。
5. **所有藏汉新一代修行人**："老一代修行作风在年轻人身上很难发现，精华差不多都漏完了"（第2课对"汉茶过滤器"颂的诠释）。

## 导读路径建议

- **初学者**：从 `00-总论·树喻与二规基础.md` 入手，把握世法即佛法根 + 十德大树两个纲领；然后选最需对治的一品（若问自己"最缺什么"）深入。
- **有入行论基础者**：本论与入行论品1-3、品4-6、品10 主题高度互补。看到入行论中某个概念时，可来本论查对应品（如入行论正知正念↔本论不放逸；入行论忏罪↔本论有愧；入行论供养与回向↔本论信心与布施）。
- **佛教团体负责人 / 发心部门负责人**：优先阅读 `02-稳重.md`、`05-正直.md`、`06-誓言坚定.md`——堪布在 2006 版中反复以"菩提学会""发心人员""法师""管家"为对境作对治开示，是这些角色的底线规范。
- **修密法者**：本论的"三宝永远不会欺惑"信心观（见 `09-信心.md`）、"泄露秘密即为无愧者"（见 `03-有愧.md`）、"上师三宝面前发誓永不违越"（见 `06-誓言坚定.md`）是密乘誓言的外前行基础。

## 未处理点与存疑

- 两版讲记中个别藏译名（如"拉塔"将领、"玛森"猎人、"嘉靠公确"劝请者）在本 collection 中保留译音，未进一步考据藏文原名。
- 颂词数共约 90 余颂，本 collection 按章节归档，未另出颂词全录本；如需完整偈颂索引，可于各品文档内查对应颂词块。
- 部分公案（如黄檗禅师探母、商主女劫后出家、顺治皇帝出家偈）在两版中讲解深浅不一，本 collection 以 2006 版为主、2003 版为补。
