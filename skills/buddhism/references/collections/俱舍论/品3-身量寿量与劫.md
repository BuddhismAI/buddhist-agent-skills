# 身量·寿量·劫·成住坏空

## 定位

品3的最后一章——把空间结构（器世间）补上**时间与尺度**：众生的身量、寿量、器世界形成与毁灭的周期，以及劫中的佛陀/缘觉/转轮王出世规律。此章包含品3最具神话意味的部分——**初劫人类**如色界天般化生、地味/地脂的渐次出现、人类文明史的退化叙事。

## 一、身量（第70课）

> **南赡部洲众生量，四肘以及三肘半，东胜身洲西牛货，北俱卢洲二倍增。**

### 人类四洲的身量

| 洲 | 身量 |
|---|---|
| 南赡部洲 | 4肘（高）/3.5肘（矮）——"**尘肘**" |
| 东胜身洲 | 8肘（南赡部洲两倍） |
| 西牛货洲 | 16肘 |
| 北俱卢洲 | 32肘 |

（以南赡部洲为标准；按各自洲则只有4肘。）

### 欲天身量

> **具有贪欲天人身，四分之一闻距至，一闻距半之间增。**

| 六欲天 | 身量 |
|---|---|
| 四大天王天 | 1/4闻距 |
| （递增）…… | 递增 |
| 他化自在天 | 1.5闻距 |

### 色界身量

> **一色界身半由旬，彼上半半而递增，少光天众之上天，身量均成两倍增，无云天减三由旬。**

| 天 | 身量 |
|---|---|
| 梵众天 | 0.5由旬 |
| 梵辅天 | 1由旬 |
| 大梵天 | 1.5由旬 |
| 少光天 | 2由旬 |
| 无量光天 | 4（少光天以上两倍递增） |
| 光明天 | 8 |
| 少净天 | 16 |
| 无量净天 | 32 |
| 广净天 | 64 |
| 无云天 | 125（两倍递增但减3由旬——特殊情况：128-3=125） |
| …… | …… |
| 色究竟天 | 16000由旬 |

**恶趣**：身量不固定，不讲。
**无色界**：无身体，无身量。

## 二、寿量（第70-71课）

### 善趣寿量

> **北俱卢洲寿千年，二洲半半而递减。此赡部洲不一定，最终十岁初无量。**

| 洲 | 寿 |
|---|---|
| 北俱卢洲 | 1000年（固定） |
| 西牛货洲 | 500年 |
| 东胜身洲 | 250年 |
| 南赡部洲 | 不定（递减最后10岁，初劫无量岁） |

### 六欲天寿量

> **人类众生五十岁，最下欲天之一日，如是自寿五百年，上天二倍二倍增。**

| 天 | 一日 | 自寿 |
|---|---|---|
| 四大天王天 | 人50年 | 500岁 |
| 三十三天 | 人100年 | 1000岁 |
| …… | 2倍递增 | 2倍递增 |
| 他化自在天 | 人1600年 | 16000岁 |

**三十三天以上昼夜问题**：无日月但有昼夜概念——**时间花闭/鸟不鸣/打瞌睡**=夜；反则白天。光明来自身体光芒。

### 色界寿量

> **色界无昼夜时寿，劫数等同自身量。**

- 色界诸天无瞌睡——寿量由**衡量身量的由旬数换为劫**来计量
- 颂词替换："由旬"→"劫"，"身量"→"寿量"

### 无色界寿量

> **无色界寿二万劫，向上依次而增长。**

| 无色界 | 寿 |
|---|---|
| 空无边处 | 2万劫 |
| 识无边处 | 4万 |
| 无所有处 | 6万 |
| 非想非非想处 | 8万 |

### 中劫vs大劫

> **少光天起为大劫，彼之下天半大劫。**

- 少光天以上：**大劫**
- 少光天以下~大梵天以上：半大劫（40中劫）

### 恶趣寿量

> **复合地狱等六狱，日渐等同欲天寿，是故彼等之寿量，亦与欲天年相同。极热地狱半中劫，无间地狱一中劫。**

| 地狱 | 一日 | 自寿 |
|---|---|---|
| 复合地狱 | 四大天王天500年 | 500岁 |
| （递增）…… | 六欲天递增 | 同六欲天数 |
| 极热地狱 | — | 半中劫 |
| 无间地狱 | — | 1中劫 |

**旁生**：最长1中劫（如八大龙王）；其他无定。
**饿鬼**：一日=人1月，自寿500年。
**寒地狱（第71课）**：
> **芝麻器中每百年，取出一粒至穷尽，即是具疱地狱寿，余寿渐成二十倍。**

- **具疱地狱**：摩竭陀国80克芝麻容器盛满，每100年取1粒，直到取尽的时间
- 疱生地狱等后后以**20倍**递增

### 中间死亡

> **除开北俱卢洲外，其余均有中死亡。**

除北俱卢洲（寿量固定）外，所有众生均可**非时死亡**。

## 四、劫（第72-73课）

### 劫的分类

> **劫有多种，即坏劫、成劫、住劫、大劫。**

### 坏劫

> **多劫坏劫无转狱，直至器世灭尽间。**

**坏劫**：从无间地狱众生空无时起，至器世界灭尽之间。

**坏劫的进程**：
1. 南赡部洲人寿上增到8万岁 = 住劫圆满+坏劫开始（同时）
2. 无间地狱众生寿终死亡——即便积累转生无间地狱之业，也投生到其他世界地狱
3. 从无间地狱向上依次空
4. 海居旁生、饿鬼次第变空
5. 人/天中存在的旁生饿鬼同样变空
6. 南赡部洲人依**法性力**获一禅（他众听后同样获一禅，转梵众天）
7. 东胜身洲、西牛货洲次第同此
8. 北俱卢洲因异熟障不能离贪——后来转生欲天，欲天次第获一禅转梵众天
9. 一禅众生以法性力获二禅→二禅天（前有十九中劫）
10. 诸天不降雨→草木干枯→第二太阳使小河水塘干涸→至第七太阳使江河/无热恼海/外海干涸→四大洲及须弥山冒烟燃起大火→**一禅所摄的火**焚毁一禅天无量殿（此一中劫，共二十劫）
11. 水毁：二禅天以法性得三禅→三禅天；风毁：三禅天得四禅→四禅天

### 成劫

> **成劫即指从初风，至有地狱众生间。**

**成劫**：从最初微风至无间地狱一众生出现之间。

进程：
1. 一切成大虚空时前兆风吹起
2. 梵众天~离诤天形成（**仅此不足形成整个器世界**）
3. 之前的风数年呼啸→下基风轮~三十三天次第形成（此1中劫）
4. 光明天众生寿福业灭尽→转生梵天无量殿（最后灭尽=最初形成）
5. 自此向南赡部洲逐步形成（**此时南赡部洲人类均化生**，以禅悦为食，身体发光，神变行空，寿量无量岁）
6. 向下依次至一众生转无间地狱之间（19中劫）
7. 无量岁减至8万岁+一众生堕无间地狱+成劫圆满+住劫开始——**同时**

### 住劫

> **中劫即从无量岁，直至人寿十岁间，最终上增为一次，彼等寿即八万间。**

**住劫**：共20中劫。

**每一中劫的量**：
- 人寿无量岁减至8万岁
- 再减至10岁 = 1中劫
- 接着上增至8万岁并减至10岁 = 1中劫
- 如此上增下减算为1，共18中劫
- 最后上增为1中劫（寿10~8万岁）

**为何初终二劫单独而中间往返**：初终二劫时间过得缓慢、中间过得快速——因此实际时长仍相等。

### 大劫

**成、住、坏、空**各20中劫——总计**80中劫 = 1大劫**。

## 五、大劫旁述（第73-74课）

### 佛陀出世

> **三无数劫现成佛，下减百岁间出世。**

- **三大无数劫**积累资粮成佛（"无数"非因不可计数，而是以60数位称"无数"）
- 佛陀出世时机：
  - **上增时不出世**——众生对轮回不生厌离
  - 从8万岁下减开始至人寿100岁之间出世
  - **百岁之后不再出世**——五浊极为炽盛，众生不堪为佛教之法器

### 缘觉出世

> **缘觉增减皆出世，麟角喻百劫得果。**

- 缘觉在**上增下减时均出世**（主要自利故）
- 麟角喻独觉：100大劫积累资粮得果

### 转轮王出世

> **诸转轮王现世间，不越人寿八万间。**

- 转轮王出世不超越人寿无量岁~8万岁之间（8万之后众生不堪为圆满法器）
- 释名：依宝轮统治四洲

**四种转轮王**：
| 转轮王 | 统辖 | 胜伏方式 |
|---|---|---|
| 金转轮王 | 4洲 | 初十受长净戒，金轮落手中，空中传声愿行，腾空而起——诸小国前来迎接 |
| 银转轮王 | 3洲 | 亲自前往即获胜 |
| 铜转轮王 | 2洲 | 来到面前摆开战场即胜 |
| 铁转轮王 | 1洲 | 准备降下兵器而制胜（对方恭敬顶礼） |

**一洲不可同时有两位**（无与伦比的福报者，同佛陀）。
**不伤命**（断除十不善）。
**身具32相**——但较佛陀在庄严/明显/圆满方面有差别。

### 初劫人类的演化史（第73课）

> **先前有情如色界，尔后渐次贪执味，懈怠者作积蓄后，具执著者封地主。**

1. 最初众生如色界天——化生、神变行空、身体发光
2. 出现**地味**（色如新鲜酥油、味如熬蜂蜜）——一人贪爱美味用指尖取尝，其他人跟随
3. **身体变坚固沉重，光芒消失**
4. 天空自然出现日月
5. 享地味多少差别→色彩好坏→相互侮辱→地味消失
6. **地脂**出现→同样侮辱→隐没
7. **苗圃**（早割晚显/晚割早显）出现→同样消失
8. **自然稻**出现→享用使男女根显露→生起贪欲→做非梵行→口出恶语
9. 建造住舍→有房屋
10. 懒惰者取明日粮→其他人取七日粮→粮具皮壳糠秕
11. **偷盗出现**→任命端严者为田地长官→**众敬王**
12. 众敬王→光严→善王→善胜→长净→**顶生王**（五先王）：统治四大部洲的转轮王
13. 过分造不善业→寿命逐渐变短→人寿10岁

### 中劫结束方式（三劫）

> **劫以兵疾饥荒尽，次第七日月年止。**

| 劫 | 时长 | 方式 |
|---|---|---|
| 刀兵劫 | 7日 | 人寿10岁时，害心增长，手中物全变兵器→相互残杀→全堕地狱 |
| 疾疫劫 | 7月7日 | 人寿10岁，非天不悦→瘟疫→堕地狱 |
| 饥馑劫 | 7年7月7日 | 人寿10岁，天人不降雨→箧盒/木条/白骨三大饥荒→堕饿鬼 |

**三饥荒**：
- **箧盒饥荒**：得微少粮即装箧为后代保存
- **木条饥荒**：母子都用木条来吃少量食物
- **白骨饥荒**：死人骨头粗糙无光彩

**恢复**：7日/7月7日/7年7月7日后——山林之人来城→相互见而仁慈→奉行十善→寿上增至20→渐次增到8万→再减到10→又造恶/疫/饥……循环。

### 坏劫的旁述（第74课）

#### 三种坏劫

> **坏劫有以火毁坏，与以水风毁三种。**

| 坏 | 身体顶 | 原因 |
|---|---|---|
| 火毁 | 二禅（身体顶） | 初禅寻伺如火 |
| 水毁 | 三禅 | 二禅喜乐如水 |
| 风毁 | 四禅 | 三禅呼吸如风 |

**四禅无毁**：无此三过患，毫无动摇。
**四禅无量殿非常有**：与众生一起出现与毁灭（同生同灭）。

#### 坏灭的次第

> **以火七毁水一毁，如是七水毁灭后，亦复以火毁七次，最终以风而毁灭。**

- 连续7次火毁→1次水毁→再7次火→1次水→……
- 如此火7+水1循环7次后→7次火→最终1次风毁

**大劫量**：
- 一禅以下（成住坏空）：1大劫
- 二禅毁灭：8大劫
- 三禅毁灭：64大劫

## 修行关联

- **人寿百岁下佛陀出世**——释迦牟尼佛出世于人寿百岁之末，是最后的机缘窗口
- **我们正处**住劫第9中劫（或类似中间），需在佛教住世期间精进
- **佛法住世的脆弱性**——中劫末可出现刀兵/疾疫/饥馑三劫，7日内即灭
- **转轮王的"不伤命征服"**——大乘理想王政的原型
- **初劫演化叙事**：从化生光明身到贪味→沉重有日月→侮辱→偷盗→王政→家庭→私产——**业感共业**的动态展示
- **四禅无毁**——为何修行者发愿至少得四禅以上的原因（躲过水火风三毁灭）

## 俱舍宇宙论与现代天文学的张力

教学中反复强调：
1. **缘起空性**：这些量非胜义实有，是业力感召的共业显现
2. **不可以现代天文望远镜"物理空间"衡量**
3. **天文历法三体系**（俱舍/戒律/时轮金刚）推算不同，不构成矛盾
4. 俱舍立此空间-时间结构，中观要破的是其**自性**而非其**显现**

## 本章对"珍宝人身难得"的贡献

- 南赡部洲人寿10~8万岁（对比北俱卢洲千岁固定）——**无常且有变化**
- 中劫末7日7月7年刀兵疾疫饥馑的警示
- 佛陀出世仅在**下减百岁**这一窗口
- 住劫20中劫中，真正具足佛陀住世的中劫极其稀少

此量化叙事构成《大圆满前行》"暇满难得"与"寿命无常"的法相基础。
