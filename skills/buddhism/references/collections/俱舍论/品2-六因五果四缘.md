# 六因五果四缘

## 六因总览（第44-46课）

> **能作因与俱有因，同类因及相应因，遍行因与异熟因，承许因有此六种。**

六因分总因（能作因）和别因（余五）；别因分同时因（俱有因/相应因）和非同时因（同类因/遍行因为因果同类，异熟因为因果不同类）。

### 一、能作因（kāraṇa-hetu）

> **除己之外能作因。**

除自己以外，对自生不作障碍的一切有为法与无为法。分有能力能作因（因缘条件起作用）和无能力能作因（仅不障碍）。范围最广——有部中安立为因不一定有果；经部以上认为有因必有果。

### 二、俱有因（sahabhū-hetu）

> **俱有因即互为果。**

同时产生的法互相成为因果。事相：大种群体间（四大互为俱有因）、心之随转（心所与心同时）、法相与事相。

八微中地水火风互为俱有因，但色香味触不是（本质不同）。因明不承认俱有因——因果不能同时。

### 三、同类因（sabhāga-hetu）

> **同类因即因果同，自类地摄前已生。**

前同类法产生后同类果。须满足：自类（五类所断中同类）、同地（自地所摄）、前已生（过去与现在，未来无次序故无同类因）。

**无漏法特殊**：九地无漏道以平等和殊胜方式互为同类因。加行善（闻思修慧）也以平等和殊胜方式：闻→闻为平等，闻→思/修为殊胜。

### 四、相应因（samprayukta-hetu）

> **相应因即心心所，一切所依相应生。**

俱有因的一部分，专指心和心所以五种相应方式取境，互为因果。

### 五、遍行因（sarvatraga-hetu）

> **所谓遍行烦恼生，自地所摄遍行前。**

从同类因中分出。烦恼产生烦恼之共因——不限于自类所断，跨类亦可。三界各十一种（共三十三种）。

遍行因不一定是同类因（跨类时只是遍行因），自类烦恼生自类烦恼时既遍行又同类。

### 六、异熟因（vipāka-hetu）

> **异熟因即唯不善，以及一切有漏善。**

唯不善法和有漏善法。无记法如腐烂种子无力量，无漏法虽有力量但无爱之湿性——均不产生异熟果。

**三时**：遍行因与同类因仅过去+现在二时；其余四因具三时。

## 五果（第46-47课）

> **果摄有为及离系，无为法则无因果。**

### 异熟果（vipāka-phala）

无覆无记法、有记（善不善）所生、众生相续所摄。因异熟因而得名——因异（善恶）时（经时间）熟（成熟为无记果）。

### 等流果（niṣyanda-phala）

自相续同类后来产生之法。事相为除初果圣者外的一切有为法。分感受等流果（前世杀生今世短命）和同行等流果（前世习气今世重复）。

### 增上果（adhipati-phala）

非前生之新生有为法——范围最广。唯有为法可为增上果。与士用果之别：增上果是不障碍自生，士用果是作者对果起作用。

### 士用果（puruṣakāra-phala）

因缘具足依士夫作用产生之果。分无间生（前刹那引后刹那）、同时生（心心所同时起作用）、间断生（春播秋收）。

### 离系果（visaṃyoga-phala）

以智慧力灭尽所断有漏法所获灭法——即抉择灭。唯无为法。

### 因果对应

| 果 | ← 因 |
|----|------|
| 异熟果 | 异熟因 |
| 等流果 | 同类因 + 遍行因 |
| 增上果 | 能作因 |
| 士用果 | 俱有因 + 相应因 |
| 离系果 | （无有为因，以智慧力获得） |

### 执果与生果

五因（除能作因）在现在以种子方式受持果。俱有因和相应因生现在果（同时）；同类因和遍行因以现在和过去生果；异熟因唯过去生果。

## 四缘（第48课）

> **经中说缘有四种。**

### 因缘（hetu-pratyaya）

除能作因外的五因——对法产生有直接力量的因。

### 等无间缘（samanantara-pratyaya）

一切心与心所，除阿罗汉最后心。前心为后心让路——平等（前后同类）、无间（不被他法阻断）。

### 所缘缘（ālambana-pratyaya）

一切法均可为心识对境。有部：所缘缘唯对现在果起作用（因三时实有，对境以现在方式存在）。

### 增上缘（adhipati-pratyaya）

即能作因。与所缘缘之别：增上缘可同时产生、有缘无缘均可安立；所缘缘不能同时、必须有缘。

### 几缘生何法

- 四缘 → 心心所
- 三缘（除所缘缘）→ 二定（无想定、灭尽定）
- 二缘（因缘+增上缘）→ 色法与不相应行

## 十二心与等无间缘（第49-52课）

> **欲界善与不善心，有覆以及无覆心，色界以及无色界，除不善外二无漏。**

十二心：欲界四心（善、不善、有覆无记、无覆无记）+ 色界三心（善、有覆、无覆）+ 无色界三心 + 有学无漏心 + 无学无漏心。

**八时**（等无间缘发生的八种情境）：相续时、同地时、结生时、入定时、出定时、染污定逼恼时、入化心时、出化心时。

品2以极其精密的矩阵详列十二心的相生关系——每种心可产生几心、由几心产生。例如：
- 欲界善心→产生9心，由8心产生
- 欲界不善心→由10心产生，产生4心
- 有学心→由4心产生，产生5心（含金刚定末际无间产生无学灭尽智）

**二十心**：十二心扩展——三界善心各分俱生善/加行善，欲界无覆无记分四（异熟生/威仪/工巧/化心），色界无覆分三（除工巧），无色界无覆唯异熟生。

俱生善与加行善的区分理由：死心必在俱生善上（与生俱来），入定必在加行善上（须经修持）。

**十二心新得**（得绳角度）：欲界烦恼心现前→六心新得；色界烦恼心→六心；无色界烦恼心→二心。其余六心仅得本身。
