# 九十八随眠·六根本烦恼·遍行·形象差别

## 定位

品5 的核心任务是**"为何轮回不止"**——品3-4讲了轮回的场所与业的动力，但若无烦恼，业不能感果（阿罗汉有不定业而不转生）。品5 先建立烦恼的精密分类体系（六→七→十→九十八随眠），再讨论它们的运作（遍行/非遍行、见断/修断）、形象差别（观待界/所缘/本体/时间/境）与生起规律。

这是阿毗达磨**烦恼地图**最精微的部分——九十八随眠是有部的标志性分类。

> **有之根本即随眠。**

## 一、根本随眠（第103-104课）

### 六种

> **六种即贪如是嗔，慢无明见以及疑。**

| 随眠 | 本质 |
|---|---|
| 贪（rāga） | 缘可爱境的爱执 |
| 嗔（pratigha） | 缘不可爱境的敌对 |
| 慢（māna） | 自高 |
| 无明（avidyā） | 不明真谛的心所他法 |
| 见（dṛṣṭi） | 五见（详下） |
| 疑（vicikitsā） | 二心 |

"如是贪"——缘某法能增上贪，嗔等也以同样方式增长。

### 七种（贪分二）

> **彼六贪分说七种。二界所生为有贪，内观之故为遣彼，解脱之想而宣说。**

贪分为：
- **欲贪**（kāma-rāga）：缘欲界色香等欲妙，唯欲界有
- **有贪**（bhava-rāga）：上两界生起的贪

**为何上两界贪合为一（有贪）**：
1. 上两界有情多数由贪爱等至内观入定（共同特征）
2. 皆为有覆无记法

**目的**：遣除"上两界是解脱"的想法——这两界仍在轮回。

### 十种（见分五）

> **见五坏聚见邪见，边执见与见取见，以及戒禁取见十。**

**五见**：
| 见 | 本体 |
|---|---|
| 坏聚见（satkāya-dṛṣṭi） | 将近取蕴视为我与我所 |
| 边执见 | 视近取蕴为常或断 |
| 邪见 | 认为业因果不存在 |
| 见取见 | 本为下劣却视为殊胜 |
| 戒禁取见 | 非因执为因、非道视为道（如大自在、五火、绝食） |

**坏聚见释名**：近取蕴是坏灭的聚合，视为我与我所。

## 二、九十八随眠（第105课·品5核心计算）

> **此十各七除三见，八种除去二种见，以见欲苦等渐断，四种随眠修所断，色无色界除嗔有，如是承许九十八。**

### 见断分类表（三界×四谛×随眠数）

**欲界见断**：
| 谛 | 见断数 |
|---|---|
| 苦谛 | **10**（十种全部——直接或间接颠倒执著近取蕴苦谛） |
| 集谛 | **7**（除坏聚见、边执见、戒禁取见三见） |
| 灭谛 | **7**（同集谛） |
| 道谛 | **8**（除坏聚见、边执见——它们不颠倒执著道） |
| 小计 | **32** |

**欲界修断**：贪/嗔/慢/无明的俱生部分 = **4**

**欲界所断总计**：32+4 = **36**

**色界**：除嗔以外各减一 = 32-5+4-1 = **31**
- 见断：苦9+集6+灭6+道7 = 28
- 修断：3
- 合计：31

**无色界**：同色界 = **31**

**三界总计**：36+31+31 = **98**

**见断88 + 修断10 = 98**

> **见欲界苦等四谛各自自相可依次断除。**

四种随眠（贪嗔慢无明）——遍计部分仅依现见真谛断，俱生部分是修断。

## 三、确定所断（第106课）

> **依有顶中所产生，忍毁唯一为见断，余生为见修二断，非忍所毁唯修断。**

| 随眠 | 断法 |
|---|---|
| 有顶所生+类智忍摧毁的见断 | **唯见道断**（世间道无法断有顶烦恼） |
| 剩余八地所生+忍摧毁 | **见道+修道二断**（圣者见谛；凡夫将见断修断合一，以世间修道断） |
| 非为忍所毁而由智所毁 | **唯修道断**（非见贪等四种俱生部分：有漏非见断） |

## 四、各见的法相（第106课）

> **我与我所常与断，无有下劣视为胜，非因非道视因道，此等即是五种见。**

详见 §一 ·十种。

## 五、与末见相关之争论

> **于大自在等非因，妄执为因由我常，颠倒执著而生故，唯是苦谛之见断。**

**他宗质难**：大自在非因妄执为因，应成**集谛见断**（颠倒执著苦因）。
**回应**：实际上是**苦谛见断**——由常有与我的颠倒执著中产生。

## 六、四颠倒（第106课）

> **由三见立四颠倒，颠倒计度分别念，以及增益皆具故。**

**四颠倒**：无常执为常、不净执为净、苦执为乐、无我执为我。

| 颠倒 | 出自 |
|---|---|
| 无常→常 | 边执见 |
| 不净→净 | 见取见 |
| 苦→乐 | 见取见 |
| 无我→我 | 坏聚见 |

**为何其他见不立为颠倒**：须俱全三条件——
1. 于对境始终颠倒执著
2. 具计度分别
3. 为增益见

戒禁取见对境不始终颠倒；边执见的断见+邪见是损减见（非增益）——故不立。

**质难**：经说"于无常认为常有是颠倒心/想/见"——"心/想"如何为颠倒？
**回应**：从根本随眠的基础而言不立；经中指"颠倒心/想与邪见相应"而说为颠倒。

## 七、七慢与九慢（第107课）

> **七慢九慢摄三中，以见及修而灭尽。**

### 七慢

| 慢 | 对象 | 心态 |
|---|---|---|
| 傲慢 | 下者 | 我胜一筹 |
| 过慢 | 同等者 | 我胜过 |
| 慢过慢 | 殊胜者 | 我更高一筹 |
| 我慢 | 近取蕴 | 视为我 |
| 增上慢 | 未获功德 | 认为已获 |
| 卑慢 | 特别优秀者 | 我略显低下 |
| 邪慢 | 过患 | 认为获得功德 |

### 九慢（《入智论》）
- **观待他立己三**：我已胜他 / 我与他平 / 我比他低
- **观待自立他三**：他高我 / 他等我 / 他低我
- **观待自否他三**：无比我高 / 无等我 / 无比我低

**九慢摄于七慢之三**：傲慢、过慢、卑慢。

### 断法
- 慢以见断+修断二者灭尽
- **圣者不现起七慢与九慢**（以见增上故，圣者已断见）
- **圣者不现起**：修断之杀生等缠、生爱（想成护地象子）、灭爱（想速死）
- 我慢+九种慢也不明显
- 不善的恶作亦不现起（以疑增长，圣者已断疑）

## 八、遍行与非遍行（观待因之分类，第108-109课）

### 何为遍行

> **见苦集断见与疑，相应不共之无明，即是同分界遍行，中除二见九缘上。**

**遍行**（sarvatraga）：在三界每一地的十一种随眠——
| 遍行 | 数量 |
|---|---|
| 见苦谛所断五见 | 5 |
| 见集谛所断二见（邪见、见取见） | 2 |
| 见苦集之疑 | 2 |
| 与见、疑相应的无明 | 1（合为一类） |
| 不共无明（苦集之独立无明） | 1 |
| 合计 | **11** |

三界各11遍行 = **33种同分界遍行**。

### 不同分界遍行

**11遍行除去坏聚见+边执见 = 9**——**不同分界遍行**（可缘上界）。

为何坏聚见/边执见不是不同分界遍行？因为此二见只将"自相续之蕴"看作我/常断，**上下相续不同**。

### 遍行与得绳

一切遍行 + 十一种随眠随行俱生的心/心所/受/相应法/法相 = 遍行，**但得绳不包括**（因随眠与得绳之因的果非一体）。

### 遍行与非遍行的意义
- **遍行**：与苦集谛的法相相违而执著——**直接颠倒执著**
- **非遍行**：缘"直接颠倒执著"并以贪/嗔/慢寻求、恼害、自满——**再度颠倒执著**

## 九、观待所缘境（第108课）

> **见灭道谛之所断，邪见疑与彼相应，不共无明共六种，即是无漏行境者。**

### 缘无漏的随眠
- 见灭谛+道谛所断：邪见×2，疑×2 = 4
- + 相应无明 = 5
- + 不共无明 = 6
- 每界6，三界**18种无漏行境者**（缘无漏法）

### 缘无漏的地域限制

#### 缘灭谛
灭谛见断**只缘自地灭谛**（不同地灭谛是不同实体）。

#### 缘道谛
**不同**——道谛可缘他地：
- 依六地（初禅未至定+正禅四+殊胜禅+上未至定）**法智方面**的一切道谛：缘法智欲界见断的邪见/疑/无明三者（6地道谛互为平等因+殊胜因+法智是欲界对治类）
- 依六地+上三无色界此九地**类智方面**：缘彼道有境上地三见断

### 为何其他随眠不缘灭道谛

| 随眠 | 原因 |
|---|---|
| 贪 | 灭道非所断 |
| 嗔 | 灭道不害他者 |
| 慢 | 灭道是寂灭法 |
| 戒禁取见 | 灭道本清净（非不清净执为清净） |
| 见取见 | 灭道本殊胜（非下劣执为殊胜） |

### 随眠的增长

> **所有遍行之随眠，以所缘于自地增，非遍行于同类中，无漏上地有境非。**

| 类别 | 增长方式 |
|---|---|
| 遍行随眠 | 所缘方式依一切自地增长 |
| 非遍行随眠 | 所缘方式于同类中增长 |
| 无漏行境的6/18随眠 | 缘但**不增长**（非爱对境、非我所对境，是对治法） |
| 上地有境的9不同分界遍行 | 缘但不增长 |
| 与受等相应的随眠 | 相应方式增长（未断坏爱故） |

## 十、观待本体（第109课）

### 无记法与不善法

> **上界随眠欲界中，坏聚见与边执见，相应无明均无记，余此欲界皆不善。**

| 界 | 无记 | 不善 |
|---|---|---|
| 上两界所有随眠 | 全部（有覆无记，不产生不悦异熟+染污） | — |
| 欲界坏聚见+边执见+相应无明 | 有覆无记（坏聚见可为自安乐持戒布施；边执见可与解脱相符） | — |
| 欲界其他随眠 | — | 不善（产生不悦意异熟） |

### 不善与无记法的根本

> **欲界所有贪嗔痴，即是不善之根本，无记根本有三种，彼为爱与无明慧。**

**不善根本**：欲界贪/嗔/痴（3）——自之本体即不善+是其他不善之根本。

**无记根本（3种）**：
1. 上两界禅定与无量宫等的一切**爱**
2. 三界的一切有覆**无明**
3. 三界有覆与无覆的一切**智慧**+相应一切无记智慧

**为何疑/慢不是无记根本**：
- 疑：三心二意不稳固；根本须稳固
- 慢：贡高趋向高处；根本须居低处

**为何嗔不是无记根本**：嗔必不善。

### 西方论师4种说

| 无记根本 | 内容 |
|---|---|
| 1 | 上两界的爱 |
| 2 | 上两界一切见+欲界坏聚见/边执见 |
| 3 | 上两界的慢 |
| 4 | 一切痴（对禅定爱重/见重/慢重均由无明生） |

## 十一、询问无记法旁述（第109课）

> **一向分辨与反诘，以及放置而授记，诸如死生与殊胜，及我与蕴一异等。**

经说**十四无记见**（佛未授记）：
- 我与世间：常/无常/二/非二
- 世间：有无边/二/非二
- 善逝圆寂后：出现/不出现/二/非二
- 身体=命 / 命=身外他法（身命一体/异体）

**问答四式**：
| 式 | 例问 | 例答 |
|---|---|---|
| 一向 | 众生有生死吗？ | 所有生的有情都不能脱离死亡 |
| 辨答 | 众生死后生还是不生？ | 有烦恼者生、无烦恼者不生 |
| 反诘 | 此人殊胜否？ | 你观待什么问？观待天人则下劣，观待恶趣则殊胜 |
| 置答 | 人我与蕴一体异体？ | 既非一体亦非他体（人我本不成立如石女儿） |

## 十二、观待时间之具理（第109课）

（此节的详细讨论延续到下一个三世实有论辩，归入 `品5-三世实有论辩.md`）

> **过去现在贪嗔慢，已生未断则具彼，未来意之彼等者，其余五识具自时，未生之法具三时，剩余随眠悉皆具。**

### 自相烦恼 vs 总相烦恼
- **自相烦恼**（3）：贪/嗔/慢——由美丑高低对境产生，非缘所有对境
- **总相烦恼**（3）：无明/疑/见——缘有漏蕴于一切时分产生

### 具足三时对境的规则
| 所具随眠 | 时相 |
|---|---|
| 过去/现在出现的贪嗔慢 | 缘三时五类所断任何境产生——未断前对那补特伽罗**具足**（如绳拴牛于桩） |
| 意识群体中未来贪嗔慢 | 缘三时一切对境**具足**（意的对境有三时法） |
| 五根识群体中未来/将生贪嗔 | 只具未来**自时**实法（不缘过去未来） |
| 不生的一切法（三时对境） | 对那补特伽罗**具足** |
| 剩余无明/疑/见 | 三时对境均具足（总相烦恼） |

"具足"含义：**以缘那些对境的方式具足所断**（非得绳具足）。

## 十三、境与有境之差别（第111课）

> **欲界所生苦集谛，见断以及修所断，自地三心色界一，无垢心识之行境。**

随眠以所缘方式增长——每一种随眠成为几心的对境？

### 总原则
- 对境随眠：三界×5类所断 = 15 + 无漏法 = **16**
- 有境：同样16种（缘15+缘无漏的）

### 欲界所生的随眠作为几心所缘

| 所缘类 | 几心所缘 |
|---|---|
| 欲界苦谛见断 | 自地遍行+非遍行相应（1）+集谛见断遍行（1）+修断闻思善（1）+色界（1）+无漏（苦法智，1）= **5心** |
| 欲界集谛见断 | 同样5心 |
| 欲界修断 | 同样5心 |
| 欲界灭/道谛见断 | **只有无漏心**（颠倒执著+再度颠倒执著都不缘；上地烦恼不缘下地；无色界远离善法不缘） |

### 色界所生
| 所缘 | 所缘心数 |
|---|---|
| 色界苦集见断+修断 | **8心**（自地3+欲界不同分遍行2+欲界修断善1+空无边处未至定善1+无垢类智1） |

### 无色界所生
| 所缘 | 所缘心数 |
|---|---|
| 无色界苦集见断+修断 | **10心**（自地3+欲界色界不同分遍行4+二修断+无漏类智） |

### 灭/道谛见断
- 欲界：6心（前面5+缘同类直接颠倒执著心）
- 色界：9心
- 无色界：11心

### 无漏法
- 10心所缘（三界各3+无漏法智/类智）

## 十四、心具随眠之理（第110课）

> **具随眠心有二种，染污心与非染心，染污心亦有二种，非染污心随增长。**

### 染污心
| 类型 | 内容 |
|---|---|
| 既具随眠也增随眠 | 凡夫染污心（所缘+相应方式增长） |
| 只具随眠不增随眠 | 阿罗汉已断的过去烦恼+未来未生烦恼（仍一起存在但不增长） |

### 非染污心
| 类型 | 内容 |
|---|---|
| 有漏善法+无覆无记法 | 以所缘方式增长，具随眠（作自地烦恼所缘不可能不增长） |
| 无漏心 | 作随眠所缘，**既不具随眠也不增随眠** |

## 十五、生起烦恼之理·三因（第111课）

> **痴中生疑后邪见，坏聚边执戒见取，尔后则于自之见，生起贪慢于他见，嗔恚如是依次生。**

### 随眠产生的顺序
1. 对真谛愚昧（无明）
2. 生**思虑苦谛有无的怀疑**（疑）
3. 依止恶知识，颠倒讲闻
4. 生**无苦谛的邪见**
5. 将蕴执为我与我所——**坏聚见**
6. 执蕴为常或断——**边执见**
7. 将边执见视为正确——**戒禁取见**
8. 因执为正确而生**见取见**
9. 对自见贪执+生慢
10. 对他见生嗔

### 生烦恼三因

> **由具尚未断随眠，境住非理之作意，此三因中生烦恼。**

生起烦恼的必要条件：
1. **随眠未断**（种子仍在）
2. **对境现前**（外境出现）
3. **非理作意**（心颠倒趋入）

三缘和合→烦恼生起。**断任一即可阻止**。

## 关键颂词

1. **有之根本即随眠。**（品5总纲）
2. **六种即贪如是嗔，慢无明见以及疑。**
3. **彼六贪分说七种。二界所生为有贪。**
4. **见五坏聚见邪见，边执见与见取见，以及戒禁取见十。**
5. **此十各七除三见，八种除去二种见……如是承许九十八。**（九十八随眠核心计算）
6. **我与我所常与断，无有下劣视为胜，非因非道视因道，此等即是五种见。**
7. **由三见立四颠倒。**
8. **见苦集断见与疑，相应不共之无明，即是同分界遍行。**（11遍行）
9. **欲界所有贪嗔痴，即是不善之根本。**
10. **痴中生疑后邪见，坏聚边执戒见取……**（随眠生起次序）
11. **由具尚未断随眠，境住非理之作意，此三因中生烦恼。**（三因）

## 修行关联

- **五见**的分析为空性修法提供精微的**所破对象**——中观破"人我/法我"，具体要破的就是五见所执
- **九十八随眠**的精确分类是五道四果体系的对治坐标
- **生烦恼三因**中"非理作意"是修行可控部分——断种子需见道，控制外境有限，唯"如理作意"是凡夫的对治武器
- **圣者不现起慢、邪恶作**——圣凡最明显的区别
- **无漏心不增随眠**——为什么修无漏道不会再增烦恼的法相依据
