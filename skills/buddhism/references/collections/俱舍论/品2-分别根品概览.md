# 分别根品概览

## 在全论中的位置

品1建立了蕴界处的静态分类框架——"有哪些法"。品2的任务是在此框架上安装**动态引擎**——"法如何运作"。具体回答三个问题：

1. 什么是法的**功能枢纽**（根）？→ 甲一：二十二根
2. 有为法**如何产生**？→ 甲二·乙一：色法微尘共生、心所法五类、不相应行十四种
3. 产生的**因果机制**是什么？→ 甲二·乙二：六因、五果、四缘

品1与品2合为"总说有漏无漏法"的存在论基础。品2完成后，全论的概念工具箱基本齐备，后续六品在此基础上展开具体问题。

## 论证脉络

### 第一部分：根之安立（第28-34课）

二十二根（indriya）是贯穿全论的功能枢纽：

> **传说五根四功用，余四各具二功用，五根八根彼一切，分具染净之功用。**

**根的分类逻辑**：
- 六根（五有色根+意根）→ 认知功能（心之所依）
- 男女根 → 身体差别功能
- 命根 → 存续功能
- 五受根 → 烦恼增长功能（染污）
- 信等五根+三无漏根 → 解脱功能（清净）

品2对二十二根进行多维分析：有漏/无漏、异熟/非异熟、善恶无记、三界分布、见断/修断、得舍之理、四果所需根数、必具与会具关系。这些分析与品1的界分类形成呼应——品1从法的角度分类，品2从功能的角度分类。

### 第二部分：有为法产生之理（第35-43课）

从"有哪些法"转向"法如何产生"：

**色法产生**（第35课前半）：有部认为欲界色法最少以八微（地水火风色香味触）同时共生。

**心所法五类**（第35-37课）：
- 遍大地法10种（受想思欲触慧念作意胜解定）→ 随一切心
- 大善地法10种（信不放逸轻安舍惭愧无贪无嗔无害精进）→ 随善心
- 大烦恼地法6种（痴放逸不信懈怠昏沉掉举）→ 随烦恼心
- 不善地法2种（无惭无愧）→ 随不善心
- 小烦恼地法10种（怒恨谄嫉恼覆吝诳骄害）→ 唯修断

**不相应行十四种**（第38-43课）：得绳、非得、同类、无想（天）、无想定、灭尽定、命、生衰住灭（四法相）、名身词句文字。这些"既非心也非色"的法，是有部独特的本体论承诺——经部以上大多认为仅是假立。

### 第三部分：六因五果四缘（第44-52课）

全品最核心的理论贡献——阿毗达磨的因果体系：

**六因**：
| 因 | 含义 | 时间 |
|----|------|------|
| 能作因 | 除自身外不障碍自生的一切法 | 三时 |
| 俱有因 | 同时产生互为因果 | 三时 |
| 同类因 | 前同类法生后同类果 | 过去+现在 |
| 相应因 | 心心所五种相应互为因 | 三时 |
| 遍行因 | 烦恼产生烦恼之共因 | 过去+现在 |
| 异熟因 | 不善与有漏善产生异熟果 | 三时 |

**五果**：异熟果、等流果、增上果、士用果、离系果

**四缘**：因缘（五因）、等无间缘（前心引后心）、所缘缘（一切法可为对境）、增上缘（能作因）

品2最后以**十二心/二十心**的等无间缘相生关系收束——这是一幅精密的心识流转图，详列每种心可产生几心、由几心产生，建立了轮回中心识连续的完整模型。

## 有部/经部张力线

| 议题 | 有部 | 经部（世亲自宗） |
|------|------|------------------|
| 五根功用 | 四种（庄严、保护、取境、名言因） | 仅取境生识一种 |
| 十遍大地法 | 任何心必具十法 | 仅五遍行必具（大乘） |
| 得绳 | 实有不相应行 | 如石女儿，无实体 |
| 同类/命/法相 | 成实物 | 假立名称 |
| 三时实有 | 过去未来法实有 | 仅现在成实 |
| 因果同时 | 俱有因因果同时 | 因果不同时（因明） |
| 无想天/灭尽定中心 | 无心（有部）/种子（随教经部） | 有细微心（随理经部） |

## 与其他品的关联

- **品1→品2**：品1建立蕴界处框架，品2在其上安装根与因果机制
- **品2→品3**：品2的十二缘起预告在品3详述；品2的异熟因果在品3器世界成因中运用
- **品2→品4**：品2的六因框架是品4分析业之因果的基础
- **品2→品5**：品2的小烦恼地法和大烦恼地法在品5详析；遍行因对应品5的遍行随眠
- **品2→品6**：品2的三无漏根（未知当知根/已知根/具知根）对应品6的见道/修道/无学道
