# 分别世间品概览

## 在全论中的位置

品1-2建立了法的分类框架和因果机制。品3回答"众生在**何处**轮回"——以品1的蕴界处和品2的因果体系为工具，描绘轮回的完整地图。品3与品4合为"世间品群"：品3是轮回的**地图**，品4是轮回的**燃料**（业）。

品3具有全论最强的"百科全书"特征——涵盖宇宙论、生命观、天文历法、物理度量。

## 论证脉络

### 甲一：生者——众生世界（第53-63课前半）

**三界五趣分类**（第53-54课）：欲界二十处（善趣十+恶趣十）、色界十七天（四禅所摄）、无色界四处无固定住所。五趣（地狱/饿鬼/旁生/人/天）为无覆无记法。七识处、有情九处、四识处等辅助框架。

**识入之理——四生与中有**（第55-57课前半）：四生（卵胎湿化），化生最殊胜但佛陀示现胎生有五理由。**中有**（antarābhava）以理证（破大众部）和教证（寻香、五不来果）成立其存在。中有身同来世本有、诸根具足、无碍而行、不可退转。入胎以颠倒心趋入——胎卵生对父母生贪嗔，化生湿生欲香处。

**识住之理——十二缘起**（第57-61课）：品3最核心的理论建构。分位缘起以五蕴不同阶段定义十二支：前世（无明、行）→今世（识至有）→后世（生、老死）。归摄为烦恼三（无明/爱/取）、业二（行/有）、事七。前际因果与后际因果可互相推断。经部定义无明为智慧违品之心所他法。

**识住之理——四食**（第62课）：段食（欲界）、触食、思食、识食（三界有漏）。前二利今世、后二引来世。

**识去之理**（第63课前半）：断善根/结合/离贪/退失/死心/生心唯意识。死心为等舍心。识从足（恶趣）/脐（人道）/心间（阿罗汉）离身。

### 甲二：生处——器世界（第63后半-74课）

**宇宙结构**（第63-65课）：风轮→水轮→金轮→须弥山。七金山七香海、四大洲八小洲、铁围山外大海。八热地狱/十六近边地狱/八寒地狱。

**天界**（第66-69课）：四大天王天（日月运行、昼夜季节）、三十三天善见城。六欲天行淫方式递减。色界十七天名义。

**身量寿量**（第70-71课）：四洲人身量倍增，欲天至色界递增。人寿至天寿以人间年倍增基础。地狱寿量以芝麻喻递增。

**劫**（第72-74课）：成住坏空各二十中劫=八十中劫=一大劫。三无数大劫成佛。火水风三灾以规律毁世（七火一水循环七次后风毁=六十四大劫毁一次三禅）。初劫人类文明演化史。

## 核心概念

| 概念 | 含义 |
|------|------|
| 十二缘起 | 无明→行→识→名色→六处→触→受→爱→取→有→生→老死 |
| 中有 | 死有与生有间的五蕴相续，化生，寻香而行 |
| 四有 | 生有（入胎刹那）、本有（生至死间）、死有（死亡刹那）、中有 |
| 四生 | 卵/胎/湿/化 |
| 四食 | 段食/触食/思食/识食 |
| 分位缘起 | 有部以五蕴不同阶段定义缘起各支 |
| 须弥山世界 | 风水金三轮→须弥山→七金山→四大洲→铁围山 |
| 劫 | 成住坏空各二十中劫=一大劫 |

## 有部/经部关键分歧

- **缘起各支本体**：有部=五蕴不同阶段（分位缘起）；经部=各有独立心所本义
- **无明定义**：有部=前世烦恼现行阶段五蕴；经部=智慧违品的独立心所
- **中有时间**：世友论师七天说 vs 不固定说

## 品3独特贡献

品3的宇宙论与现代科学的张力是全论最突出的教学特色。讲解中反复以"一碗水六道各见不同"之喻和缘起空性说化解矛盾：须弥山世界非物理实在，而是业力感召的共业显现。天文历法（日月运行/昼夜/月盈亏）涉及俱舍/戒律/时轮金刚三种不同推算体系的比较。
