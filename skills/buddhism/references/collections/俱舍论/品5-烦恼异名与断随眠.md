# 烦恼异名·结缚漏瀑流·断随眠·九遍知

## 定位

品5 后半——在九十八随眠分类与三世实有论辩之后，转入**烦恼异名系统**与**断随眠的方式**。

经中对烦恼有多种名称（漏/瀑流/近取等），对法论又有结/缚/缠/垢等分类——**同一套烦恼从不同侧面标定**。本章先梳理这套异名网络，再讲如何对治（见断三法+修断九品+四对治），最后以**九遍知**（断随眠的九种果）收束品5。

## 一、经中的烦恼异名（第112-115课）

### 漏（āsrava）

> **欲界除痴惑及缠，乃为欲漏色无色，唯一随眠为有漏。**

**三漏**：
| 漏 | 所摄 | 数量 |
|---|---|---|
| 欲漏 | 欲界除"痴惑"（无明）外的烦恼：12见+4疑+贪嗔我慢各5=31随眠+10缠 | 41 |
| 有漏 | 色界+无色界各除十无明外的26种 | 52 |
| 无明漏 | 三界的15无明 | 15 |

**为何合色无色界为"有漏"**：三原因
1. 为无记法相同
2. 内观相同
3. 是入定之地相同

**必要**：三界均属三有非解脱——明确此二界**不是寻求解脱**的（遣除二界为解脱之想）。

**为何单独立无明漏**：轮回的根本即无明，断无明则其他随眠皆断——单独立为漏以明解此点。

### 瀑流结合（第113课）

> **瀑流结合亦复然，为明诸见另宣说，漏非无有助伴见，则与安置不相符。**

**四瀑流四结合** = 漏替换：
- 欲漏 = 欲瀑流 = 欲结合
- 有漏 = 有瀑流 = 有结合
- + 单独立**见**（三界一切见共36种）
- 无明瀑流/结合 = 无明漏（15）

**为何有瀑流是4但漏是3**？
- 漏：能安置于轮回——"单独见（无贪等助伴）"与"安置轮回"之义不符合，故漏中不单列见
- 瀑流/结合：不须此义，故单列见

**数量**：
| | 数 |
|---|---|
| 欲瀑流结合 | 29 |
| 有瀑流结合 | 28 |
| 见瀑流结合 | 36 |
| 无明瀑流结合 | 15 |

### 近取

> **如是所说及无明，诸近取即见分二，所谓无明非能取，是故混合而宣说。**

**四近取**：
| 近取 | 内容 | 数 |
|---|---|---|
| 欲近取 | 欲瀑流结合+无明 | 34 |
| 我所近取 | 有瀑流结合+无明 | 38 |
| 见近取 | 前四见 | 30 |
| 戒禁取见近取 | 6个第五见 | 6 |

**为何戒禁取见单独立**：执非道为正道，成正见怨敌，经众多无义苦行欺惑在家出家——过患重。

**为何无明不单立**：无明不明显，无助伴单独不能取后世——故混合不独立。

### 释名

> **彼等极为微细故，随系二种随增故，及跟随故称随眠。彼等能置与能漏，能冲能粘近取故。**

- **随眠（anuśaya）**：极细微、**随系**（从无始时与得绳相联）、**随增**（以所缘+相应方式增长）、**跟随**（连续不断与相续辗转）
- **漏**：能安置轮回、从六处伤口漏到轮回
- **瀑流**：能冲到后世
- **结合**：能与后世粘连
- **近取**：能取受轮回

## 二、论中的烦恼异名（第113-115课）

> **此等亦分为结等，故复宣说有五种。**

对法论分烦恼为**5种**：结、缚、随眠、随烦恼、缠。

### 结（九种）

> **实体与取相同故，安立见取之二结，二唯不善自在故，嫉悭单独称二结。**

**九结** = 六根本烦恼 + 取结 + 嫉妒结 + 悭吝结

**为何见取见+戒禁取见单独立为"取结"**（与一般"见结"分开）：
1. **实体相同**：前三见与后两见各有18种实体
   - 三见：苦谛边/戒禁×3界+集谛邪见×3界+灭谛邪见×3界+道谛邪见×3界= 6+6+6 = 18
   - 后二见：见取见18 + 戒禁取见18 = 同
2. **取相同**：
   - 前三见：执为殊胜**之所取**
   - 后二见：执为殊胜**之能取**

**为何嫉妒+悭吝在缠外单独立**：
- 完全是不善业
- **自在而起**（不观待其他根本）

#### 五顺下分结

> **五种顺下分之结，以二令不离欲界，由三能令复返回。**

**五顺下分结**：坏聚见、戒禁取见、怀疑、欲贪、害心——与三界最下欲界随顺。

| 作用 |
|---|
| 欲贪+害心：令不离欲界 |
| 坏聚见+戒禁取见+怀疑：虽一次转生上界但又返回欲界 |

**为何预流果只断此5**？
- 预流果实已断五见+疑六种——经中只说断坏聚见、戒禁取见、疑
- **直接**：断第一门的坏聚见、前后两门的戒禁取见、四门的怀疑
- **间接**：边执见（随坏聚见）、邪见（随疑）、见取见（随戒禁取见）同时断
- 贪等根本随之断（缘这些是其有境）

**世亲论师的阿阇黎式喻**：
- 不愿到别地 → 坏聚见（担忧断我）
- 迷路 → 戒禁取见（依他道）
- 对路途疑虑 → 对解脱怀疑
- 断此三障则趋向解脱无碍——故从主要角度说只断此三

#### 五顺上分结

> **顺上分结亦有五，色界无色界所生，二贪掉举慢无明，令不超离上界故。**

| 顺上分结 | 内容 |
|---|---|
| 色界贪 | 1 |
| 无色界贪 | 1 |
| 掉举 | 1 |
| 我慢 | 1 |
| 痴（无明） | 1 |

作用：令不离上界。

### 缚（三种）

> **因以苦乐舍三受，安立贪嗔痴三缚。**

| 缚 | 缘 |
|---|---|
| 贪缚 | 缘乐受 |
| 嗔缚 | 缘苦受 |
| 痴缚 | 缘舍受 |

### 随烦恼（第113-114课）

> **所谓根本烦恼外，染污心所之行蕴，一切均称随烦恼，彼等不应称烦恼。**

除六根本烦恼外，其他**染污性心所行蕴**的烦恼。

**数量**：
- 10缠 + 6垢 + 放逸、懈怠、不信 = **19种**
- 《杂事律》：不喜、打呵欠等**24种**

**为何不称根本**：因不是根本（是根本的等流果）。

### 缠（八/十种）

> **八缠无惭无愧嫉，悭掉举悔昏沉眠，或十复加忿与覆。**

**八缠**（决定数）：
| 违 | 缠 |
|---|---|
| 戒律违品 | 无惭、无愧 |
| 利他违品 | 嫉妒、悭吝 |
| 智慧违品 | 掉举、悔心 |
| 定违品 | 昏沉、睡眠 |

**十缠**（有部加）：上8 + 忿 + 覆

### 缠由根本生

> **无惭掉举与悭吝，皆由贪生覆有诤，无愧昏眠无明生，悔心乃由怀疑生，忿与嫉妒由嗔起。**

| 缠 | 根本 | 原因 |
|---|---|---|
| 无惭 | 贪 | 贪自法不敬他法 |
| 掉举 | 贪 | 贪欲妙使心放荡 |
| 悭吝 | 贪 | 贪资具执为殊胜 |
| 覆 | 贪（有诤）/无明/二者 | 担忧失利养/不知过患 |
| 无愧 | 无明 | 不知罪果苦而不畏罪 |
| 昏沉 | 无明 | 明了入所缘者无昏沉 |
| 睡眠 | 无明 | 同上 |
| 悔心 | 疑（不善） | 对善果安乐心生怀疑→对行善起悔 |
| 忿 | 嗔 | 由嗔引发恼 |
| 嫉 | 嗔 | 对他圆满无法忍受 |

### 六垢（第114课）

> **此外烦恼有六垢，谄诳骄恼恨与害。**

**六垢** = 诳、谄、骄、恼、恨、害——如身体污垢。

**根本**：
| 垢 | 根本 | 原因 |
|---|---|---|
| 诳 | 贪 | 担忧失利养作诈现 |
| 骄 | 贪 | 于自法贪执心生安乐 |
| 恨 | 嗔 | 对境生损害心 |
| 害 | 嗔 | 对境生损害心 |
| 恼 | 见取见 | 自见执殊胜对他见起损恼 |
| 谄 | 邪见 | 对业果邪见导致恶心虚伪 |

## 三、缠/垢的差别（第114-115课）

### 缠的法之差别

> **其中无惭与无愧，昏沉睡眠掉举二，其余则是修所断，自在如是一切垢。**

| 缠 | 见/修 |
|---|---|
| 无惭、无愧、昏沉、睡眠、掉举（5） | 见断+修断相应 |
| 嫉、悭、悔、忿、覆（5） | 唯修断+自在性 |
| 六垢 | 均修断+自在性 |

**自在性**：不观待贪等取境；与无明相应者也因愚痴而不取境。

### 善恶分类（第115课）

> **欲界不善三者二，彼上则为无记法。**

- 欲界：无惭等7缠+6垢——不善；昏沉、掉举、睡眠（3）——不善与无记
- 欲界以上：所有随烦恼——有覆无记

### 地之差别

> **欲界一禅有谄诳，梵天欺惑马胜故。**

- **欲界+一禅**：有谄、诳
- **公案**：圣者马胜不知四大灭于何处，问大梵天——大梵天也不知，却让人传话"我是梵天，梵天伟大……"——正是**谄诳**
- **三界**：昏沉、掉举、骄
- **唯欲界**：其他11种随烦恼

## 四、与何识相应（第115-116课）

> **见断以及睡眠慢，即是意识之地起，自在随烦恼亦尔，余者均是依六识。**

| 随眠/随烦恼 | 所依 |
|---|---|
| 内观的见断+修断的睡眠/慢 | 唯**意识**（分别念+不明显，不于意识以外产生） |
| 自在性后5缠+6垢（11种修断） | 唯**意识**（分别念） |
| 睡眠/慢以外的修断三毒/无惭/昏沉/掉举/放逸/懈怠/不信 | **六识**均依 |

## 五、与何受相应（第116课）

> **诸喜乐与贪相应，嗔心与之正相反，无明相应一切受。**

### 根本烦恼
| 随眠 | 相应受 |
|---|---|
| 贪 | 身乐受+意乐受（喜相趋入） |
| 嗔 | 身苦受+意苦受（厌恶相趋入） |
| 无明 | 一切五受（不违一切趋入） |
| 邪见 | 意乐受+意苦受（见恶业无果→意乐；见善无果→意苦） |
| 疑 | 意苦受（对寻义生疑→意不安乐） |
| 其余四见+慢 | 意乐受（意识地自喜相趋入） |

此为**欲界所生明显心**。**不明显心**与**舍受**相应（舍受与一切不违）。

### 上界
- 二禅以下：意乐受
- 三禅：心乐受
- 四禅：舍受

### 随烦恼（第116课）
| 随烦恼 | 相应受 |
|---|---|
| 悔、嫉、忿、害、恨、恼 | 意苦受（意识地+忧愁） |
| 悭 | 意乐受（贪生+喜相） |
| 谄、诳、覆、眠 | 意乐受+意苦受 |
| 骄 | 三禅心乐受/二禅下意乐受/四禅上舍受（乐相应） |
| 舍受 | 遍行一切地/识/随烦恼 |
| 无惭、无愧、昏沉、掉举 | 五受（不善地+大烦恼地+六识俱起） |

## 六、五障（第116-117课）

> **欲界诸障共有五，违品作用因同故。二二合一有害蕴，及怀疑故唯立五。**

**五障**（经中）：欲贪、害心、昏眠、掉悔、怀疑——唯对**欲界**中存在的静虑与定作障碍。

经云："此乃不共圆满不善蕴，即五障。"

### 为何昏沉-睡眠/掉举-悔两两合一

**三原因**：
1. 违品对治法相同
2. 能生因相同
3. 作用相同

#### 昏眠
- **因**：不喜、瞢、频申、食不平性、心怯懦
- **对治**：光明想
- **作用**：令心怯懦

#### 掉悔
- **因**：回忆亲属/对境/不死的妄念+昔日喜乐
- **对治**：寂止
- **作用**：令心不寂静

### 为何只立五（而非所有烦恼都是障）

| 障 | 害蕴 |
|---|---|
| 欲贪+害心 | 令不离欲界和罪业——害**戒蕴** |
| 昏眠 | 远离胜观——害**慧蕴** |
| 掉悔 | 远离寂止——害**定蕴** |
| 无戒定慧 | 对真谛产生**怀疑** |

## 七、见断之断法（第117-118课）

> **由以遍知所缘境，真实灭尽彼能缘，以及断除所缘境，生起对治而灭惑。**

**见断三法**（同一时间反体不同）：

### 1. 以遍知所缘境断（彻底了知所缘）

通过趋入所缘四谛+完全了知见道自相而断。
**范围**：
- 欲界自地的17种苦集见断
- 上界前两类30种苦集见断
- 唯灭道见断无漏的12种有境+6无明（=缘灭道的18种随眠）

**喻**：知道是假人而去除为人的执著。

### 2. 以灭尽能缘断

真实灭尽了缘不同分界9种遍行对境的**有境**苦集见断烦恼——有境断则9种对境必将断（断有境：凡缘自地的并不需了知上界真谛；无因则无果）。

（欲界而言；色界缘无色界的9种依同时见苦集二谛了知所缘断。）

### 3. 以断除所缘境断

由断除所缘境而断有境烦恼：
- 对三界灭道谛直接颠倒执著的见断（灭谛4+道谛5=9，上界各7）
- 加**再度颠倒执著**
- 共23种
- 缘彼之有漏部分的6无明
- 通过断对境"灭道谛的邪见等"而断（无对境→无有境）

**重要说明**：这三种断除方式是**同一时间中反体不同的角度**，非前后次第——见道不是两次生起的。

## 八、修断之断法（第118-119课）

### 四对治

> **所谓对治有四种，断持远分与厌患。**

| 对治 | 内容 | 例 |
|---|---|---|
| 断对治 | 断绝烦恼得绳的**无间道** | 苦法忍 |
| 持对治 | 受持离得的对治 | 苦法智、解脱道、胜进道、其上无间道 |
| 远分对治 | 远离烦恼得绳的道 | 解脱道以上一切道，如苦类忍 |
| 厌患对治 | 视为过患厌无常等相 | 缘苦集的行相 |

### 断的方式

> **许以所缘可断惑。**

厌烦所缘境而想断烦恼——断那一烦恼，缘它的有境也不产生。
对治与所断**非相应方式断**（若相应则二者不能分开），而是**所缘方式**（可分开）。

### 断除的标准

- 自相续一切烦恼得绳已斩断+种子灭尽
- 非染污性的有漏善法+有覆无记法：缘它们的自相续烦恼已断，故它们也断

**见断一时断，修断次第断**。

### 远分的四种

> **法相不同与违品，境相隔绝与时间，如大种戒时方远。**

| 远分 | 例 |
|---|---|
| 法相不同而远 | 四大（一群体中但法相异） |
| 违品与对治而远 | 具戒与破戒 |
| 对境行相隔断而远 | 东海与西海 |
| 时间而远 | 过去未来久远 |

### 诸惑灭尽的机缘（第119课）

> **诸惑一次即灭尽，彼等离得复殊胜，生起对治与得果，以及练根诸时中。**

**钝根者**：6时中离得殊胜——
1. 生起各对治的解脱道
2-5. 获得沙门四果
6. 钝根→利根练根

**利根者**：5时（钝根前5）。

无间道时：只向生起离得方向趋近，未真正生起离得，故唯有解脱道是殊胜。

## 九、九遍知（断除之果，第119-120课）

### 分类

> **所谓遍知有九种，灭欲苦集谛为一，灭后二谛各为一，上三遍知余亦三，灭顺下分与色界，及灭诸漏三遍知。**

**见断6遍知**（忍之果）：
| 遍知 | 内容 |
|---|---|
| 1 | 灭欲界苦集见断（2合1） |

|---|---|

| 9 | 漏尽遍知（灭无色界修断） |

### 是何道之果

> **其中六种为忍果，余三即是智之果。**

- 前6：忍（无间道）之果
- 后3：智（修道——修道无忍）之果

**是何地之果**：
| 地 | 果 |
|---|---|
| 一禅未至定 | **9遍知全部**（一切道的基础） |
| 一切静虑正禅 | 5遍知（断上两界见修所断）或妙音尊者说8（先已离贪的不来果见道在正禅） |
| 空无边处未至定 | 第8遍知（色贪） |
| 空无边处等无色界前三无漏正行 | 第9遍知 |

### 世间道/出世道之果

> **一切均是圣道果，世间之道果有二。**

- 全部9遍知均可为**圣道**之果
- **世间道果**：第7+第8（圣者依世间道远离欲界/色界贪）
- **修道类智**：后2遍知
- **修道法智**：后3遍知（法智是三界对治）
- **法智+同品**：6遍知
- **类智+同品**：5遍知

### 定数（为何只9）

> **获得无漏之离得，失毁有顶一分惑，二因一切均摧毁，故立遍知复超界。**

安立**断见断**遍知的3条件：
1. 无漏离得重新获得
2. 有顶一分烦恼失毁
3. 苦集谛见断的两类遍行因全部摧毁

苦法忍具1、苦类智具1+2、**集法智具3条件** → 安立为断欲界苦集一遍知。

安立**断修断**最后3遍知还需第4因：**超界**（超欲界/色界/三界）。

### 为何不来/阿罗汉的多离法综合为1遍知
- **超界**：不来超欲界，阿罗汉超三界
- **沙门法重新得果**
- 两条件综合→多断法综合为一遍知

### 得舍（第120课）

> **有者一二五六舍，得者亦然除五外。**

**舍**（同时）：
- **1**：退失未离色界贪不来果时舍第7
- **2**：获得阿罗汉果时同时舍第7+第8
- **5**：以前离贪不来果获得16刹那类智时，舍欲界见断3遍知+上界见断2遍知
- **6**：渐次圣者远离欲界贪欲时获第7遍知，舍前6

**得**（同时）：
- **1**：集法智时得第1；离贪道类智时得第6；具贪欲界第9解脱道得第7；获阿罗汉时得第9
- **2**：以无色界烦恼退失的阿罗汉得第7+第8
- **6**：从渐次不来果退失得前6
- **不能5**

## 关键颂词

1. **欲界除痴惑及缠，乃为欲漏色无色，唯一随眠为有漏。**（三漏）
2. **瀑流结合亦复然，为明诸见另宣说。**（四瀑流）
3. **如是所说及无明，诸近取即见分二。**（四近取）
4. **彼等极为微细故，随系二种随增故。**（随眠释名）
5. **五种顺下分之结，以二令不离欲界，由三能令复返回。**（顺下分的作用）
6. **因以苦乐舍三受，安立贪嗔痴三缚。**
7. **八缠无惭无愧嫉，悭掉举悔昏沉眠，或十复加忿与覆。**
8. **由以遍知所缘境，真实灭尽彼能缘，以及断除所缘境，生起对治而灭惑。**（见断三法）
9. **所谓对治有四种，断持远分与厌患。**（四对治）
10. **所谓遍知有九种，灭欲苦集谛为一。**（九遍知）

## 有部/经部在此章的分歧

| 论题 | 有部 | 经部 |
|---|---|---|
| 离得 | 实有不相应行法 | 不承认实有 |
| 断与离 | 断了不一定离（遍行仍缘已断法） | 大乘不承认已断法还能被缘 |
| 断随眠次第 | 见断一时断，修断次第断 | 基本一致 |

## 修行关联

- **见断三法**（了知所缘/灭尽能缘/断除所缘境）在《现观庄严论》中**完整继承**——见道断三十六断的理论基础
- **五障**直指坐禅时的常见违缘——昏沉睡眠/掉举悔恨/欲贪/害心/怀疑
- **缠/垢的分类**为日常自省提供颗粒度更细的心态法相
- **顺下分结**——预流果的断证标准（尤其断"三结"即断坏聚见/戒禁取见/疑）
- **九遍知**——从预流到阿罗汉的九个断证里程碑

## 与《现观庄严论》的对应

品5的**见断三法**+**修断四对治**完整入《现观》的见道/修道章——大乘修道论的俱舍法相基础就在此。

## 与唯识的对应

- 大乘六根本烦恼一致
- 随烦恼：大乘标准20种（俱舍19/24近似）
- 缠、垢：大乘未单独立这些分类，但法义相通

## 与中观的对应

中观破烦恼"自性"——如何破？以**离一多因**破烦恼的实一或实异——这正是中观破有部结/缚/缠/垢的"实有不相应行"。见 `topics/madhyamaka/reasoning-methods.md §离一多因`。
