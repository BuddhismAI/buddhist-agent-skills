# 器世间·须弥山宇宙·三界五趣

## 定位

品3 的"百科全书"之一——俱舍宇宙论。全论最具"现代科学张力"的一章：须弥山、四大洲、六欲天、八热八寒地狱的详细构造。索达吉堪布讲解中反复以"一碗水六道各见不同"与缘起空性化解与现代天文学的冲突——须弥山世界非物理实在，而是**业力感召的共业显现**。

此章也涵盖三界五趣的法相分类（第53-54课）与器世界结构（第63-69课）——将二者合为一章，因为前者是众生所居的**法相框架**，后者是具体的**地理展开**。

## 一、三界的分类（第53课）

### 欲界

> **地狱饿鬼与旁生，人类以及六欲天，即是欲界分二十，由地狱洲之差别。**

**欲界**（kāmadhātu）：能增上不善业烦恼界。

**欲界二十处**（按地狱/洲之差别分）：
- 八热地狱（寒地狱与近边地狱为热地狱从属，不单独计算）
- 四大洲
- 饿鬼（一处）
- 旁生（一处）
- 六欲天（六处）

**六欲天**：四大天王天、三十三天、离诤天、兜率天、化乐天、他化自在天。

**释名**：
| 趣 | 释 |
|---|---|
| 地狱（那绕嘎） | "绕嘎"是欢喜，"那"是否定——非欢喜 |
| 饿鬼 | 心念饮食前去寻觅不复返 |
| 旁生 | 头横着行走 |
| 人类 | 意识占主要（"人" = 作意） |
| 天（得瓦） | 拥有喜乐尽情享受 |

### 色界

> **此上住所十七处，即为色界于其中，初三静虑各三处，第四静虑有八处。**

色界（rūpadhātu）：功德与住所远胜欲界，超离欲界并能增上自地烦恼。

**色界十七处**：
| 禅 | 处 |
|---|---|
| 初禅（各3处） | 梵众天、梵辅天、大梵天 |
| 二禅（各3处） | 少光天、无量光天、极净光天 |
| 三禅（各3处） | 少净天、无量净天、遍净天 |
| 四禅（8处） | 无云天、福生天、广果天、无热天、无烦天、善现天、善见天、色究竟天 |

**四禅的特殊结构**：
- 前三处（无云/福生/广果）：凡夫住处
- 后五处（无热~色究竟）：**五净居天**——唯圣者住

### 无色界

> **无色界则无住所，由转生而分四种，彼处同分与命根，即心相续之所依。**

**无色界**（arūpadhātu）：
- 无单独其他处所（获得无色界等持未退失者无论死于何处，立即形成四名蕴转生无色界）
- 分为空无边处、识无边处、无所有处、非想非非想处
- 众生心相续依于**同类与命根**（不相应行法），因远离色之想而力量强大
- 下二界的心依色法，不依同类与命根

## 二、五趣（第53-54课）

> **彼中地狱等五趣，经中宣说各名称，非烦恼性非有记，所谓众生非中有。**

五趣 = 地狱/饿鬼/旁生/天/人。

**本体**：五趣是**无覆无记法**（非烦恼性有覆无记+非不善+非有记善）——若是烦恼性或善法则圣者与断绝善根者被排除。

**非天归摄**：按圣者无著观点，非天属天界（身体受用与天相比，是天人联姻友伴；失天法故名"非天"——下劣加否定词）。

## 三、识住的三层分类（第53-54课）

### 七识处

> **身不同与识不同，身不同与识一致，相反以及身识同，无色界中初三处，此等即是七识处。**

| 识处 | 内容 |
|---|---|
| 身不同识不同 | 人类与诸欲天（颜色/形状/行相各异，识有苦乐舍） |
| 身不同识相同 | 梵众天新生天人（身体较大梵天小；识一致——大梵天认为"这些天人是我意愿所造"，新生者也觉得"我们由大梵天意愿所生"） |
| 身相同识不同 | 二禅天（身体同是光的自性；识有正行意乐/未至定二种） |
| 身相同识相同 | 三禅天 |
| 无色界初三处 | 空无边处/识无边处/无所有处（"彼处无色界众生……"） |

**识处含义**：识以**爱**的方式增上。

**为何三恶趣/第四禅/有顶不是识处**：
- 三恶趣：以痛苦摧毁识
- 第四禅无想天：因果无想定摧毁
- 有顶：灭尽定摧毁

### 有情九处

> **有顶无想天众生，经说有情之九处。**

七识处 + 有顶 + 无想天 = 有情九处。

**为何三恶趣+第四禅不叫有情处**：
- 三恶趣：业魔干扰在不情愿中安住（如关监狱）
- 第四禅凡夫：想到无想天去；圣者：想到涅槃

### 四识处

> **四处有漏之四蕴，自地独识未说处。**

**四识处** = 恶趣到有顶之间的有漏四蕴（非色识以外四蕴）。

**为何识蕴非识处**：此处说"住所与住者不同"——识是住者，不是住处（喻：国王不是自己的坐垫）。

### 两种识处的归摄（四句分别）
1. 为七识处所摄不为四识处所摄——七识处中的识蕴
2. 为四识处所摄不为七识处所摄——四识处中的第四禅/有顶/恶趣的有漏蕴
3. 既非七识处亦非四识处——第四禅/有顶/恶趣的识蕴
4. 既为七识处亦为四识处——七识处中识蕴以外的有漏四蕴

## 四、器世间结构（第63-65课）

### 风轮→水轮→金轮

> **器世最下风轮厚，十六洛叉广无数。**

器世界最下方：
1. **空虚空**：一切形成前的基础
2. **风轮**：形成前兆微风四起→多年狂风→金刚不摧的风轮
   - 厚度：一百六十万由旬（16洛叉）
   - 宽度：一无数（阿僧祇）

> **水深一百十二万，尔后深度八十万，剩余凝结成金轮。**

3. **水轮**：风轮上，雨流如车轴降下
   - 初深：112万由旬（具金子精华）
   - 最终深：80万（其余32万变成金子，不流淌因众生业力）
4. **金轮**：水轮上，以风搅水变坚硬→金刚大地

**水轮/金轮直径**：1,203,450由旬，周围圆形（是直径的三倍）。

### 九山与七海（第63课）

> **须弥山与持双山，持轴山与担木山，善见山与马耳山，象鼻山与持边山。尔后乃是一切洲，彼等之外铁围山，七山为金彼为铁，须弥四宝之自性。**

**九山**（环绕须弥山，由强到弱风形成）：
| 山 | 形象 |
|---|---|
| 须弥山 | 王（四宝自性：东银、南琉璃、西红宝石、北金） |
| 持双山 | 车轭 |
| 持轴山 | 金轴 |
| 担木山 | 沉香林严饰 |
| 善见山 | 山顶见而生喜之饰 |
| 马耳山 | 马耳形 |
| 象鼻山 | 象鼻 |
| 持边山 | 轮辐 |
| 铁围山 | 最外围（铁） |

**须弥山四方虚空颜色**：须弥山那一面是什么珍宝，那方虚空也显现那颜色——**南方琉璃**→南赡部洲天空**蓝色**。

**山的量**：
- 九山各有8万由旬没入水中
- 须弥山：水面上8万由旬
- 后八山：高度各减半（持双山露4万，铁围山312.5由旬）
- 厚度/宽度=各自高度

### 七香海

> **彼等七间为七海，初者八万由旬宽。**

须弥山至持边山的**七个间隔**=七香海（所有龙王的游戏处）。

**八功德水**：凉/柔/轻/香/澄/无臭/饮不损喉/饮不伤腹。

**香海的量**：
- 深度：各8万由旬
- 第一（须弥山~持双山）：宽8万
- 后六海：宽度后者是前者的一半（4万→2万→……→1250）

**外大海**：持边山至铁围山，宽32万由旬（充满盐味）。

### 四大洲（第63-64课）

> **南赡部洲之三边，二千由旬马车形，一边乃三由旬半，东胜身洲如半圆。**

| 洲 | 方位 | 形 | 尺寸 |
|---|---|---|---|
| 南赡部洲 | 南 | 马车形 | 三边各2000由旬，一边3.5由旬 |
| 东胜身洲 | 东 | 半圆 | 三边同南赡部（2000），一边350 |
| 西牛货洲 | 西 | 圆形 | 周长7500，直径2500 |
| 北俱卢洲 | 北 | 四方 | 每边2000 |

**释名**：
- 南赡部洲：因赡部树果落海发出"赡部"响声
- 东胜身洲：身量超南赡部洲两倍
- 西牛货洲：享用如意牛与珍宝
- 北俱卢洲：死前七日出现难听声音

### 八小洲

> **彼等之间八小洲，身洲胜身声不美。**

| 从属洲 | 八小洲 |
|---|---|
| 东胜身洲 | 身洲、胜身洲 |
| 北俱卢洲 | 声不美、声不美月洲 |
| 南赡部洲 | 拂洲、妙拂洲 |
| 西牛货洲 | 行洲、胜道行洲 |

拂洲是罗刹居住，其他都是人居住。

### 南赡部洲的特法

> **此向北越九黑山，即是所谓之雪山。**

南赡部洲中央 = **印度金刚座**。向北：
- 九黑山→雪山→香醉山
- 香醉山十由旬内：**无热恼海**（50由旬方形）盈满八功德水——**四大河**：
  - 东：恒河
  - 南：信度河
  - 西：缚刍河
  - 北：徙多河

所有河右绕无热恼海汇入外海。

## 五、恶趣（第64课）

### 热地狱

> **此下二万由旬处，即是无间地狱处。彼之上方七地狱。**

南赡部洲下方2万由旬 = **无间地狱**（房顶）；高度广度各2万。

**八热地狱**（从下到上）：
1. 无间地狱（最下）
2. 极热地狱
3. 烧热地狱
4. 大嚎叫地狱
5. 嚎叫地狱
6. 众合地狱
7. 黑绳地狱
8. 复合地狱（最上）

### 十六近边地狱

> **四边煻煨尸粪泥，利刃原等无滩河。**

八热地狱四面各四近边：
- **煻煨坑**：火烬没膝
- **尸粪泥**：利嘴昆虫刺身
- **利刃原**（含剑叶林/铁刺树林）：林中布满刀刃；剑叶林有具铁齿杂色狗啃食；铁柱山乌鸦以铁喙啄眼球
- **无滩河**：难以逾越的热灰

### 寒地狱

> **其他具疱地狱等，即是八种寒地狱。**

**八寒地狱**：具疱/疱裂/紧牙/阿啾啾/呼呼/裂如青莲花/裂如红莲花/裂如大莲花。

### 容纳问题
此洲下方何能容纳无间等地狱？答：所有洲如谷堆下端广大。

### 旁生与饿鬼

**旁生**：根本住所是外大海，天界人间居住的是散居旁生。

**饿鬼**：根本住处是**王舍城**（南赡部洲下500由旬处）——阎罗法王以不善引业牵，以善满业感为地狱众生显示业果。饿鬼有36种或归纳为**外障/内障/食障**三种。

## 六、天界：与地相连（第65课）

### 四大天王天

#### 依无量宫：日月位置

> **日月位于山王半，五十五十一由旬。**

- 位置：须弥山半（4万由旬）处
- 月径：50由旬（水晶，清凉）
- 日径：51由旬（火晶，发热，害眼）
- 周长=径×3
- 厚度：5.5+1/18由旬

**日照四洲的方式**：北方午夜=东方日落=南方正午=西方日初（同时）。

**夏冬日回**：
- 夏季二月末（5月下旬~6月上旬）：白昼18顷刻，夜12顷刻——极长
- 冬季四月末（11月下旬~12月上旬）：夜18顷刻，白昼12顷刻——极短
- 从9日起以"2漏分3秒"递变

**长短的原因**：太阳在南方六月运行时，越过南赡部洲边缘（从海上运行快速，东西距离短，白天短，阳光洒落海上气候寒）；在北方六月时（越过大海从洲上缓慢运行，东西距离长，白天长，阳光照洲上气候炎热）。

**月亮盈亏**：月与日最近时日光射月，自影遮月→显亏；越远越圆。月基底是水晶，围墙金子，阳光照墙影投基水晶→见月盈亏。

#### 依山

> **须弥山有四层级，间距一万一千也。**

须弥山四层，间距1.1万由旬。各层向外突出：
| 层 | 突出 | 居住 |
|---|---|---|
| 第一 | 16000由旬 | 持盆药叉（手持甘露宝盆） |
| 第二 | 8000 | 持鬘药叉 |
| 第三 | 4000 | 常醉药叉（恒为美酒迷醉） |
| 第四 | 2000 | **四大天王天**（持国东、增长南、广目西、多闻北） |

七金山处也是四大天王的天界。

### 三十三天（第66课）

> **须弥山顶卅三天，彼边八万由旬也。**

须弥山顶：
- 每边8万由旬，周长32万
- **三十三天**名：八财神+威猛十一天+日神十二天+娄宿二子=33（或善法堂等33处）
- 四方四隅四层楼（边225、高500）——**手持金刚药叉**保护天人
- **善见城**：中央，每边2500，周长2万，高1.5；墙99901门；每门500铠甲守护神

#### 尊胜宫（帝释住处）
- 每边250，周长1000，高0.5+1/18
- 101座宫殿，每殿100层，每层7房，每房7天女，每女7侍女，每侍女7铙钹

#### 四林苑
- 东：众车苑
- 南：粗恶苑
- 西：杂林苑
- 北：喜林苑
- 各苑20由旬处设妙地

#### 大香树与善法堂
- 东北隅：**大香树**（向下50由旬，向上100，枝叶50，芳香顺风100/逆风50）
- 前有骰子形一合石（边50，周200，高5.5+1/18）——上设天王狮子座+32天宝座（众天人夏四月居此）
- 护地神象（幻化32头撑起32天）
- 西南隅（隔12由旬）：**善法堂**（直径300，周900，高3.5）——天王与32天宝座，众天人思维人天之事与正法

## 七、天界：与地不连（第67-69课）

### 欲界余天的行淫方式（第67课）

> **二二相交抱执手，发笑目视而行淫。**

| 天 | 行淫方式 |
|---|---|
| 四大天王天 | 二根双运（无不净同类因→不出精，依出气行淫） |
| 三十三天 | 二根双运 |
| 离诤天 | 仅拥抱 |
| 兜率天 | 仅拉手 |
| 化乐天 | 仅发笑 |
| 他化自在天 | 仅目视 |

**天子天女生处**：四大天王天~他化自在天——身材达5岁到10岁孩童身量时迅速化生，在他们的怀中出生。

### 色界诸天

身体圆满，有惭愧，着衣出生，死时着衣而死。

**释名**：
- 梵众天：罪净、属大梵天眷属
- 梵辅天：大梵天前如婆罗门诵吠陀
- 大梵天：最初生、最后死，身材较大
- 少光天：威光范围小

## 品3 器世论的"现代科学张力"

教学讲解反复说明：
1. **一碗水六道各见不同**——同一境缘，众生业感显现不同（天见甘露、人见水、饿鬼见脓血、鱼见住所）
2. **须弥山非物理实在**——是业力感召的共业显现，不可以现代天文望远镜的"物理空间"衡量
3. **天文历法的三种体系**（俱舍/戒律/时轮金刚）对昼夜/日月运行的推算不同，并不构成矛盾
4. **"缘起空性"化解冲突**——不可执此器世结构为胜义实有，也不可以"现代科学未见"破俱舍（二者所见之境不同）
