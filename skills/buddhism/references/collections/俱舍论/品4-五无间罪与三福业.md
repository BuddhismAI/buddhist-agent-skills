# 五无间罪·近无间罪·引业满业·三福业

## 定位

此科目是品4的后半——从业果转入**重罪与大福**。五无间罪（杀父/母/罗汉/破僧/出佛身血）是最重的不善业定业；三福业之事（布施/持戒/修）是积累解脱资粮的三大途径。引业与满业讲业如何决定投生（总业）与一生苦乐（别业），其中"一业引一世"（有部）vs"一业引多世"（经部）的分歧在此展开。

## 一、引业与满业（第96课）

> **一业能引一生世，满业则可有多种。无心之定非引业，得绳亦非为引业。**

### 有部立场
- **引业**（ākṣepaka-karma）：一业只能引**一世**（决定转生何趣）
- **满业**（paripūraka-karma）：可有**多种**（决定一生苦乐）

画家喻：一位技艺高超的画家先进行素描（引业），之后其他人用各种颜料圆满完成（满业）。

### 四类引满业组合
| 引业 | 满业 | 例 |
|---|---|---|
| 善 | 不善 | 欲界具苦人天 |
| 不善 | 善 | 阎罗法王、财丰龙王之子 |
| 善 | 善 | 帝释天王 |
| 不善 | 不善 | 地狱众生 |

### 经部立场
经部等承认**一业也能引多世**、一业多世、多业一世等多种说法。

### 非引业
- **无心之定**（灭尽定/无想定）：虽有异熟，但非引业（无心与心不俱生，业力微弱）。是长寿天与有顶的**满业**，而二者的引业是四禅与有顶定。
- **得绳**：虽与业共存，但非引业（从无心状态顺便产生，力量微弱，如树皮与业异体）。有漏善法与不善业的得绳均是**满业**。
- 圣者相续中的有漏业与加行道四顺抉择分：**只是满业**。

## 二、三障（第96课）

> **无间罪之诸业障，强烈烦恼与恶趣，无想天众北俱洲，承许此等为三障。**

| 障 | 内容 | 原因 |
|---|---|---|
| 业障 | 五无间罪 | 障圣道与暖相等加行善 |
| 烦恼障 | 猛烈烦恼（由小到中、中到大连续不断） | 得不到对治机会；猛然涌现的烦恼易断故非障 |
| 异熟障 | 三恶趣众生、无想天、北俱卢洲 | 宿业牵引成恶相续者 |

### 例外
兰禅巴、指鬘、难陀等有深重烦恼但生起圣道——因**得道时已不存在深重烦恼**（障未断前未生圣道，断后方生）。

### 三障分布
- **业障**：三洲有（北俱卢洲无佛教不造无间罪；恶趣罪恶意乐不稳+业果现前；天人不杀父母）
- **无间罪**：三洲的黄门、两性黄门等不会造（生来缺根者，父母未生身根俱全者无慈恩、晚辈无惭愧）
- **异熟障**：恶趣、无想天、北俱卢洲具有
- **烦恼障**：一切众生（五趣）均有

## 三、五无间罪（第96-99课）

### 破僧（最重）
> **破僧即令不和睦，自性不相应行法，非烦恼性无记法，彼与所破僧众具。**

#### 破法轮僧的特殊结构
- **"破僧"本体**：双方僧团不和睦关系**实有存在的一种不相应行**（有部立场）
- **性质**：非烦恼性、非有记善、而是**无覆无记法**（因此断烦恼者与断善根者也可具）
- **具者**：破僧者+所破僧众双方均具
- **无间罪真正之因**：能挑拨僧众的**妄语**（语业）——这才是破僧者提婆达多等真实具足的无间罪

> **彼之罪业为妄语，彼与破者真实具，中劫成熟无间狱，造多无间害亦增。**

#### 异熟果
破僧者在**一中劫**中感受无间地狱之苦。

#### 多造无间的果报
有部解释：两三种无间罪虽只有一世转生地狱，但按业数量身体变为两倍并柔嫩庞大，痛苦也成倍增。前面是引业，后面是满业。

### 破僧的严格条件（第97-98课）

> **比丘见行具戒者，破于他处凡夫众，堪忍他师与他道。**

**造破僧者的资格**：
- 比丘（非在家）
- 见行者（非爱行者）
- 持戒者（非破戒者）
- 须与佛陀抗衡，须利根（否则僧众不信）

**破僧的处所**：佛所住之处**以外**的地方（破僧者无法堪忍佛陀威严）。

**所破**：凡夫僧众（非圣者——圣者已诚信佛陀不动）。

**破僧的标准**：远离佛陀与佛道，亲近提婆达多，堪忍并接受提婆达多制定的**五条学处**：
1. 不享用酸奶
2. 断肉
3. 不使用盐
4. 穿著未裁剪衣
5. 住于城内经堂中

**持续时间**：只一日（自然规律）。

### 两种破僧

> **许彼为破法轮僧，赡部洲有九等成，破羯磨僧三洲有，彼以八者以上成。**

| 类别 | 区域 | 最低人数 | 无间罪否 |
|---|---|---|---|
| 破法轮僧 | 佛住世南赡部洲 | 9人（双方各4+1挑拨者） | 是 |
| 破羯磨僧 | 除北俱卢洲外三洲 | 8人以上（界限内不和之心作长净等羯磨） | 非无间罪 |

### 六不出现时间（第98课）
> **初末二大尊者前，佛已涅槃未结界，于上此等六时中，破法轮僧不出现。**

1. 佛陀最初转法轮不久——众生喜悦和睦
2. 佛陀接近涅槃时——悲伤
3. 佛教见解/戒律未受损前——无损而住
4. 在二大尊者（舍利弗/目犍连等）之前——无能调解者
5. 佛陀涅槃后——无竞争对象
6. 未作大小结界——界内不存在破法轮僧

### 杀父母的特殊问题（第98课）

> **于利益及功德田，舍弃令彼无有故。变性杀之亦犯罪，月经中生者为母。**

**为何杀父母也是无间罪**：父母生育了相合解脱的身体（利益之田）；罗汉、僧众、佛陀是功德之田（殊胜福田）——存心舍弃+行动杀害。

**变性父母**：父变女、母变男，杀害也犯无间（已生育殊胜身体者）。

**交叉胎生问题**：一女人胎中凝酪注入另一女人胎中→"两个女人中自己从第一个月经产生"→**第一个女人是真正的母亲**，杀她犯无间罪（主要是生育相合解脱之身）。

### 恶心出佛身血（第98课）

> **以打佛心非无间，击后得罗汉亦非。**

- 恶心出佛身血须**利刃猛刺**（想杀佛）才构成无间罪
- 只想以兵器击打痛击：不犯无间（无杀意乐）
- 用兵刃击中未得阿罗汉者后被击者成阿罗汉死亡：不犯无间（对阿罗汉未采取行为）

### 造无间罪后
> **造无间罪则无法，获得离贪之果位。**

造无间罪者未断除此罪前**不能获得离贪果**（意乐极度相违）。

### 最重/最轻的无间（第99课）

> **为破僧众说妄语，承许彼为最重罪。**

**五无间罪的严重次序**（从重到轻）：
1. 破僧（说妄语）——最重
2. 恶心出佛身血
3. 杀罗汉
4. 杀母
5. 杀父——最轻

**破僧最重的原因**：
- 利刃击如来法身
- 障出离、得果、离贪、漏尽
- 断禅定、听闻、持诵、思业
- 中断善趣与解脱
- 三界含天界一片混乱
- 果报一中劫无间地狱

**所有善业中最大果报**：
- 世间善业：**有顶的善果**（异熟果八万大劫息灭一切痛苦）
- 有漏善业士用/离系果：**世间胜法位**
- 无漏法：**金刚喻定**（无同类因直接引见道，断三界烦恼的断得）

## 四、近五无间罪（第99-100课）

> **染污母亦无学尼，杀定菩萨有学圣，夺僧合食毁佛塔，此等即近无间罪。**

| 近无间罪 | 相近无间 |
|---|---|
| 以邪淫侮辱既是母又是无学尼者 | 近杀母 |
| 杀住定菩萨 | 近杀父 |
| 杀七种有学圣者补特伽罗任一 | 近杀罗汉 |
| 夺僧和合食 | 近破僧 |
| 以嗔心等毁坏佛像/佛经/佛塔 | 近出佛身血 |

造此等罪者**绝对堕地狱**。

### 异熟障的三障差别（第100课）

> **于得无来罗汉忍，成为极重之障碍。**

经中"具有能障异熟的三种业"：
- 对获加行道忍位者：转恶趣的业成极障
- 对不来果者：转生欲界的业成极障
- 对阿罗汉者：转三界的业成极障

（"要前往他乡，债主就会前来索债"——离业还未成熟时就障碍成熟）

### 住定菩萨（第100课）

> **从修妙相之业起，即名住定之菩萨，投生善趣贵族家，根足转男忆宿世，不退彼即赡洲男，观现量佛思所生，余百劫中方能引，每一妙相百福生。**

**住定菩萨的界限**：从积累成熟三十二妙相的业时起（到一百劫必获佛果）。

**住定菩萨六特点**：
1. 见道后转生善趣
2. 投生大萨拉树般王族/高贵种姓
3. 诸根具全
4. 转为男身
5. 能回忆生生世世
6. 百劫不退转而获佛果

**成熟妙相业的六特点**：
1. 基：转赡部洲、男根士夫
2. 对境：缘佛陀的善心（真实住世的佛陀）
3. 因：**思所生善**（非闻所生）
4. 时：离成佛还剩一百大劫
5. 业：能引能积累
6. 量：每一相由百百福德中生

### 本师释迦佛的三阿僧祇劫（第100课）

> **宝髻燃灯毗尸佛，三阿僧祇劫末出，最初乃大释迦佛。**

- 最初供养**大释迦佛**一碗米、一双鞋、布絮，发愿为陶师之子光明童子
- **第一阿僧祇劫末**：宝髻佛（承侍75000如来）
- **第二阿僧祇劫末**：燃灯佛（承侍76000如来）
- **第三阿僧祇劫末**：毗尸佛（承侍77000如来）

（大乘承认每一阿僧祇劫承侍无量如来。）

### 六度圆满的公案
- **布施度**：以大悲心将身体受用施舍给一切众生
- **安忍+持戒度**：具贪情况下被砍断肢体也一心不乱（安忍仙人传记）
- **精进度**：婆罗门童子见星胜如来入火遍处定，单足站立七日赞叹"大丈夫如牛王您沙门"——圆九劫资粮
- **静虑度**：未现前菩提前无间入金刚喻定时的等持
- **智慧度**：当时的智慧

## 五、三福业之事（第100-102课）

> **三福称为福德业，彼事如同业道也。**

三福业 = **布施、持戒、修（所生慧）**
- **福德**：能产生相应悦意果
- **业**：业的本体
- **事**：趋入等起之思的根本

喻：身语七业既是业又是道，意三业唯是道。

### 布施（第100-101课）

#### 定义与四分类
> **何者能舍名为施，以欲供养饶益心，身语之业及等起，彼果具足大受用。**

布施 = 善心将财物施与他众。
**等起**：供养善妙对境+救济可怜众生（非为解脱怖畏或希求回报）。
**果**：具足大受用。

**四种布施**：
1. 自利布施：尚未离贪的圣者/凡夫以世间道离贪，供养佛塔
2. 他利布施：除顺现法受业外远离欲界贪欲的圣者布施他众
3. 二利布施：未离贪圣者/凡夫布施未离贪他众
4. 非二利布施：除顺现法受业外远离欲贪圣者供养佛塔（恭敬报恩心）

#### 布施的殊胜差别
> **施主殊胜具信等，以恭敬等作布施。**

**施主殊胜**：具足信心+持戒+慷慨+听闻+知惭+有愧+智慧+知足少欲八功德。
**布施方式**：恭敬+亲手+应时+不害他众。
**果报对应**：
| 方式 | 果 |
|---|---|
| 恭敬 | 后世受眷属恭敬爱戴 |
| 亲手 | 受用丰厚 |
| 应时 | 应时获财 |
| 不害他众 | 无违缘获受用 |

#### 施物殊胜
> **色等圆满之施物，由彼美貌具名声。**

施物殊胜 = 色等圆满之物。果：相貌端严/美名远播/味香悦意/所触柔软/随时安乐之身。

#### 田殊胜
> **田殊胜由趣痛苦，利益功德之差别，解脱施脱或菩萨，其中布施第八胜。**

**殊胜田**：
- 趣之差别（善趣的人）
- 痛苦田（病人等）
- 利益田（父母）
- 功德田（具戒者、预流果等圣者）

**最殊胜布施**：
- **解脱者布施解脱田**：阿罗汉布施阿罗汉（无任何过患）
- **菩萨施主作布施**：虽二者都非解脱者，但为饶益一切有情、获得圆满菩提

**八种布施**（经中）：
1. 布施随至者
2. 以怖畏而施
3. 以曾施我而施
4. 以将施我而施
5. 以往昔父辈祖辈曾施而施
6. 为善趣而施
7. 为名声而施
8. **为心之庄严/资具/瑜伽资粮/殊胜之义而施**——最为殊胜

释义：
- 心之庄严 = 神变
- 心之资具 = 八正道
- 瑜伽资粮 = 寂止与胜观
- 殊胜之义 = 阿罗汉或涅槃

#### 四种殊胜对境（非圣者）
> **虽非圣者然而于，父母病人说法者，最后有者之菩萨，供养功德无有量。**

1. 父母——利益之田
2. 病人——痛苦之田
3. 说法者——于法身相续中修行之田
4. 最后有者菩萨——功德之田

果报**不可估量**。

### 作已积集业的六因（第101-102课）

> **由结行田基加行，思与意乐之大小，业亦变为轻与重。**

**业轻重的六因**：
1. 后行（随正行而连续行持）
2. 田（利益等福田）
3. 基（如杀生/妄语/邪见）
4. 加行（恒时所造）
5. 思（究竟业道）
6. 意乐（发心广大）

> **故意圆满无悔心，无有对治具从属，异熟作已积集业。**

**作已积集业的六条件**：
1. 故意
2. 支分圆满
3. 不悔
4. 无有忏悔等对治法
5. 具足随喜等从属
6. 必定感受异熟果报

其余称**作已不积业**。

### 供佛塔与恶田布施之诤（第102课）

> **供佛塔因所生福，虽无接受如慈等，因中生果无谬故，恶田亦有悦意果。**

**大众部质难**：供佛塔希求心供养不得福德（佛塔无接受享用能力）。

**回答**：
- 虽无接受享用的福德，但有**从供养佛塔之因中所生的福德**
- 如慈等四无量心、修正见——无接受者直接利益他众能力，但依自心力获无量福德

**恶田布施问题**：
- 有漏善因产生悦意异熟果——不错乱
- 布施恶劣田必产生悦意异熟果
- 然恶劣田使果变小（贫瘠田子不应时成熟且减小）

### 持戒所生福（第102课）

> **断破戒恶佛制罪，二色戒具四净德。**

- **所断**：不善有表无表色（恶戒）
- **戒的对治**：有表与无表色的善法
- **非自性罪但断佛制罪**（如过午不食）也是戒律——有表+无表

**戒律四功德**：
1. 不被破戒所染
2. 破戒的因（根本+随眠烦恼）未出现
3. 依止对治破戒的四念住
4. 依止回向解脱之寂灭

### 修行所生福

> **入定之善即为修，能熏染心之缘故。**

**修**：心专注于所缘境入定所生与等持相应的善业。"修"之义——如芝麻被花香熏染，心染上善的习气。

### 戒与修的轻重

> **为得善趣戒重要，为离烦恼修重要。**

- 善趣角度：戒律重要
- 离烦恼角度：修重要
- 二者均不可缺。

### 梵天福德（第102课）

> **感数劫住善趣故，四种梵天之福德。**

经说"四种补特伽罗生起梵天福德"：
1. 于以前未有佛塔之处建造佛塔（第一梵天福德）
2. 供养四方僧众欢喜园建造经堂（第二）
3. 调解声闻分裂（第三）
4. 修慈无量心（第四）

"梵天"指**梵辅天**。福德：在欲界善趣延续寿量+**四十中劫**期间享受安乐。

### 法布施

> **法施非为染污性，如实开示契经等。**

- **意乐等起**：不为名闻利养恭敬爱戴等染污心
- **行为**：如理无误讲解十二部契经或三藏

## 六、三随分（第102课）

> **随福德分随解脱，随抉择分三种善。**

| 随分 | 内容 | 所依道 |
|---|---|---|
| 随福德分 | 轮回中悦意果报 | — |
| 随解脱分 | 资粮道中向往涅槃的善业（必成涅槃之法） | 资粮道 |
| 随抉择分 | 暖/顶/忍/胜法四加行位 | 加行道 |

**随解脱分的外相**：闻轮回过患与解脱功德时生起强烈信心以致汗毛竖立。

## 七、如理而入之业（第102课）

> **如理三业具等起，次第乃为书字印，计算诗歌与数目。**

世间的"书写文字、刻印、计算、诗歌、数量"都是以五蕴为体的如理业：
- 书写/刻印 → 身业
- 计算/诗歌 → 语业
- 十乘十等于百等的数量合计 → 意业（四蕴性）

### 对法论名词（异名）

> **有覆下劣与具罪，此等染污法异名，无漏善业为善妙，有为善法当敬依，解脱乃为最无上。**

**染污性法的异名**：
| 名 | 义 |
|---|---|
| 有覆 | 烦恼有遮障作用 |
| 下劣 | 具染污黑色+圣者呵责 |
| 具罪 | 与烦恼同住 |

**无漏善法** = 善妙（无垢最清净）
**恭敬依止**：有为法中的有漏+无漏善法
**不该依止**：无为法（不能相续生起）、不善业、有覆无记法

**法之居上次第**（前前后后居上）：
不善业与有覆无记 → 有为无覆无记 → 有漏善法 → 有为无漏法 → 无为无覆无记（虚空/非抉择灭） → **解脱的抉择灭**（至高无上）

## 关键颂词

1. **一业能引一生世，满业则可有多种。**（有部立）
2. **无间罪之诸业障，强烈烦恼与恶趣，无想天众北俱洲。**（三障）
3. **破僧即令不和睦，自性不相应行法。**
4. **为破僧众说妄语，承许彼为最重罪。**（五无间次序）
5. **染污母亦无学尼，杀定菩萨有学圣，夺僧合食毁佛塔，此等即近无间罪。**
6. **何者能舍名为施，以欲供养饶益心。**（布施定义）
7. **由结行田基加行，思与意乐之大小，业亦变为轻与重。**（六因）
8. **故意圆满无悔心，无有对治具从属，异熟作已积集业。**（作已积集业六条件）
9. **随福德分随解脱，随抉择分三种善。**
10. **解脱乃为最无上。**（法之顶峰）

## 修行关联

此章是品4**果报学与福业积累**的集中：
- **破僧最重**——僧团和合的重要性（对藏传/汉传僧团都适用）
- **"百福生一相"**——佛陀每一相好的福德量
- **作已积集业六条件**——"不悔"作为业减重的关键（忏悔可使定业不定）
- **布施八种**——以"心之庄严/资具/瑜伽资粮/殊胜之义"为最胜
- **戒-修二者平衡**：善趣靠戒，离烦恼靠修
- **法布施**的两条件：非染污等起+如理讲解
- 加行道的随抉择分是"必生涅槃"的善根
