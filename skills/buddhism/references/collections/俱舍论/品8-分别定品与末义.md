# 分别定品与末义

## 在全论中的位置

品7回答"断烦恼的工具是什么"（智），品8回答"智慧在**何处**运作"——四禅八定是智慧的容器。品8是全论最后一品，以禅定为核心，收束全论从基（品1-5）到道（品6）到果（品7-8）的完整体系。

## 科判结构（第151-160课）

### 甲一：真实定（第151-156课）

**四禅八定**：色界四禅+无色界四定。分因定（善心一缘专注）与果定（转生后自地现前），又各分染/净/无漏三类（有顶仅二类）。

**禅定支分**：
| 禅 | 支分 | 说明 |
|----|------|------|
| 初禅 | 寻伺喜乐等持 | 五支，对治欲界五盖 |
| 二禅 | 内净喜乐等持 | 四支，离寻伺 |
| 三禅 | 念正知舍乐等持 | 五支，离喜 |
| 四禅 | 念舍受等持 | 四支，离八过患（最殊胜不动） |

从名十八种，从实体归十一。分对治支/功德支/安住支。

**三定互生规则**：净定起可入染/净/无漏；染定起可入自地染/净或无间下一地染；无漏定起可入净/无漏。顺退/住/胜/抉择四分等持。

**超越定**：远加行（顺逆序历遍八地）→近加行（超一地顺逆）→正行（有漏无漏不同类交替超越）。所依唯不动法阿罗汉。

**三解脱门**：空性等持（苦谛·空/无我）、无相等持（灭谛四相）、无愿等持（余十行相）。另有空空/无愿无愿/无相无相三种出离等持。

### 甲二：定所摄之功德（第157-159课）

**四无量心**：慈（无嗔·愿得乐）、悲（无嗔·愿离苦）、喜（意乐·愿不离悦）、舍（无贪·平等心）。所缘为欲界有情，远分对治。

**解脱·胜处·遍处三层递进**：
| 层次 | 含义 | 关系 |
|------|------|------|
| 八解脱 | 背离贪欲 | 因等持 |
| 八胜处 | 随意信解而胜过 | 自在 |
| 十遍处 | 所缘行相遍满一切 | 果等持 |

八解脱=前二不净观（一/二禅）+净色解脱（四禅）+四无色解脱+灭尽解脱。前七通凡圣，第八唯圣者。

**功德获得方式**：因力/业力/法尔力/传教力四种。

### 末义（第159-160课）

**佛法住世期**：教法（三藏）+证法（三学/三十七道品）。持教法者唯讲闻者，持证法者唯实修者。佛法住世五千年三期：果期（前1500年，三果渐降）→修期（中1500年，慧定戒）→教期（后1500年，论经律）→末五百年唯形象。

**谦虚自述**：世亲声明依克什米尔有部理而造，过错归己，正法理量唯诸佛。

**教诫不放逸**：佛已涅槃，阿罗汉多已入灭，正法如命至喉，求解脱者切莫放逸。引佛言"不放逸为无死处"作结。

> **俱舍论第八分别定品释终**

## 核心概念

| 概念 | 含义 |
|------|------|
| 四禅八定 | 色界四禅+无色界四定，禅定的完整体系 |
| 禅定三分 | 染定/净定/无漏定 |
| 超越定 | 有漏无漏交替超越，唯不动法阿罗汉 |
| 三解脱门 | 空性/无相/无愿等持 |
| 四无量 | 慈悲喜舍 |
| 解脱·胜处·遍处 | 背离→胜过→遍满 |
| 佛法教证二法 | 教法=三藏，证法=三学 |

## 有部/经部关键分歧

- **无色界色法**：有部=绝无；随理经部=有细微色法
- **初二禅乐支**：有部=轻安（非身乐受）；经部=可有身乐受
- **有顶无漏定**：有部=无（想极不明显）

## 品8独特贡献

品8以禅定三分法（染/净/无漏）和三层递进（解脱→胜处→遍处）构建了完整的定学体系。超越定的三阶修法是《现观庄严论》菩萨超越定的基础。末义以"不放逸为无死处"作结，回归全论的修行本怀。
