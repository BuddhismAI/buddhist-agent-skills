# 分别智品概览

## 在全论中的位置

品6建立修道次第，品7回答"断烦恼的**工具**是什么"——智慧的体系。品7以十智为核心，扩展至佛陀不共功德与六种神通，是全论对"果位"描述最集中的一品。

## 科判结构（第139-150课）

### 甲一-二：智之基与自性（第139-142课）

**十智体系**：法智（缘欲界四谛）、类智（缘上二界四谛）、世俗智（缘一切法，范围最广）、他心智、苦智、集智、灭智、道智、尽智、无生智。

**智·忍·见三者辨析**：八忍非智（所断未离）；尽智无生智是智非见（无计度分别）；法智至道类智既智又见。

**十六行相**：苦集灭道各四。有部十六实体，经部仅七实体。行相本体：有部以慧为体，经部以智之执相（如镜中影像）。

**七理安立十智**：自性/对治/行相/行境行相/加行/所作/因圆。

### 甲三-四：具智与得智（第142-144课）

见道十六刹那中具几智；何道何地得几智的详尽矩阵分析。

### 甲五：智所摄之功德（第145-150课）

**不共功德·十八不共法**：十力+四无畏+三念住+大悲。

**十力**：处非处智力、业异熟智力、静虑解脱等持等至智力、根上下智力、种种胜解智力、种种界智力、遍趣行智力、宿住随念智力、死生智力、漏尽智力。

**四无畏**：正等觉无畏、漏尽无畏、说障法无畏、说出道无畏。

**大悲与悲心八差别**：本体（无痴vs无嗔）、所缘（三界vs欲界）、行相（离三苦vs离苦苦）、所依（唯四禅vs四静虑）、相续（唯佛vs声缘亦有）、获得/救济/趋入各异。大悲以五理称"大"。

**六种神通**：神境通（三摩地性）、天耳通、他心通、宿住随念通、天眼通、漏尽通。前五依四禅，漏尽通九地皆有。三明=宿住+天眼+漏尽（对治前/后/中际无明）。三神变中教诫最胜。

**化心与幻化**：十四种化心（一禅2+二禅3+三禅4+四禅5），自地化心做自地幻化事。神境通五种来源：修得/生得/咒成/药成/业成。

## 核心概念

| 概念 | 含义 |
|------|------|
| 十智 | 法/类/世俗/他心/苦/集/灭/道/尽/无生 |
| 尽智/无生智 | 已遍知苦断集现灭修道 / 苦无所知集无所断灭无所现道无所修 |
| 十八不共法 | 十力+四无畏+三念住+大悲，唯佛具足 |
| 六通 | 神境/天耳/他心/宿住/天眼/漏尽 |
| 三明 | 宿住+天眼+漏尽（对治三际无明） |
| 四无碍解 | 法/义/词/辩无碍 |

## 有部/经部关键分歧

- **十六行相实体数**：有部十六实体 vs 经部七实体
- **行相本体**：有部=慧本身 vs 经部=智之执相
- **自证**：有部不承认（须二刹那回忆方知） vs 经部承认
- **诸佛色身**：小乘仅法身与化身，不承认常有报身

## 品7独特贡献

十智的"交叉分析法"是有部阿毗达磨对法进行穷尽式分类的典型范式。佛果功德的系统阐述（十力四无畏大悲六通）构成小乘佛陀论的核心。讲解中贯穿大小乘对比的教学风格——将每一论题与《入中论》《现观庄严论》《经庄严论》对照。
