# 十二缘起·四有·四食

## 定位

品3 从"众生在何处轮回"（器情世间分类）过渡到"众生如何轮回"——核心是**识入之理、识住之理、识去之理**三层：
- 识入之理：四生+中有
- 识住之理：十二缘起+四食
- 识去之理：死心/生心/断善根等（归入品3概览）

十二缘起是品3最核心的理论建构，有部采用**分位缘起**（以五蕴不同阶段定义各支），这是"有部安立法体"在品3的体现。

## 一、四生（第55课）

> **有情具卵等四生，人与旁生具四种，一切地狱与天界，中有唯有一化生，饿鬼亦有胎生二。**

三界有情的**四生**：卵生、胎生、湿生、化生。

| 趣 | 所具四生 |
|---|---|
| 人类 | 四种都有（初劫人化生、我乳国王湿生、卵生如船毁商人与鹤所生的长老札、胎生如大多数今世普通人） |
| 旁生 | 四种都有（四生的大鹏以四生的龙为食） |
| 一切地狱 | 唯化生 |
| 天界 | 唯化生 |
| 中有 | 唯化生 |
| 饿鬼 | 多数化生，也有胎生（饿鬼对目犍连言晚夜各生五子皆吃仍不足） |

**四生优劣**（前前善妙）：化生>湿生>胎生>卵生
- 化生：不害自他二者
- 湿生：不害他者
- 胎生：自己一次受苦
- 卵生：自己两次受苦（一次破壳、一次出壳）

## 二、中有（antarābhava, 第55-57课）

中有是死有与生有之间的过渡。此科目是品3第一场正面论辩——与大众部（不承认中有）的交锋。

### 本体

> **死有中有间蕴生，未至应至之境故。**

中有 = **五蕴相续**在死有与生有之间产生。中有不是后世——因为尚未到达应到的目的地。

### 能立（理证）

> **中有并非是后世，如谷相续同法故。影像本体不成立，及不同故非比喻，若一无有二共存，非相续因二法生。**

**大众部立**：中阴不存在——生有被死有稍稍中断便出生。

**回应**：中有不是被死有稍稍中断而产生——就像**种子→苗芽→茎**不间断产生一样，粮食的相续与相续趋入是相同的。

**大众部喻**：镜中面像是面容相续但由面容中断进入。

**破喻**：
1. 镜中面像**本体不成立**——一位置两个异体色法不可能同存
2. 所谓"镜中面像是面容相续"不成立——镜中像实由面容+清洁镜子两个主因生，但中有只由死有单一产生

### 能立（教证）

> **佛亲说故有寻香，说五众生经亦成。**

**教证一**：《宣说七有经》："有有七种，即五趣有、业有、中有。"

**教证二**：经云"若现前三处，则子住于母胎中……寻香"——"寻香"即中有（非他者）。

**经部质难**："寻香"是五蕴灭尽的别名。
**回应**：《达林之子经》云"汝之寻香即刹帝利种姓或……"——五蕴灭法不可能有种姓。

**教证三**：经说不来果五种中的"中般涅槃"。
**对方**："这只针对天人种性"。
**回应**：若如此，生般涅槃等也应如此。

### 身相

> **此一引业相同故，如当本有之身形，本有即是死有前，居生有之刹那后。**

| 四有 | 含义 |
|---|---|
| 生有 | 结生第一刹那的五蕴 |
| 本有 | 生有之后至死有之前的五蕴相续 |
| 死有 | 死心最后刹那 |
| 中有 | 死有与生有之间的五蕴相续 |

生有与死有均为**刹那性**；本有与中有是**相续存在**。

**中有身的颜色**：
| 趣 | 中有色 |
|---|---|
| 地狱中有 | 如被火焚的树干 |
| 旁生 | 如烟 |
| 饿鬼 | 如水 |
| 欲天/人类 | 金色 |
| 色界 | 白色 |
| 无色界 | 无中有（无色法） |

**中有身量**：
- 欲天与人类：如五六岁孩童
- 色界：身体圆满+着衣而生（有惭愧）
- 欲界：裸体降生（无惭愧）；白衣比丘尼由昔愿着衣降生
- 最后有者菩萨：具足相好严饰的童子相+着妙衣

### 法之差别

> **同类具天眼者见，具有业之神变力，诸根具足具无碍，不可退转即寻香。**

| 特点 | 内容 |
|---|---|
| 可见性 | 同类互见；修行所得清净天眼可见（俱生眼根不能见） |
| 神变 | 业力所生神变，快速行空，无人能阻 |
| 诸根 | 五根具全 |
| 无碍 | 山岩等处无碍穿行（但**不能穿过母胎与金刚座**） |
| 不退转 | 中有形成后不被他缘转变（与本有由同一业力牵） |
| 寻香 | 欲界中有由善恶缘分而寻香气或臭气 |

### 入生之理

> **生起颠倒之心故，欲行淫往所去境，湿生化生欲香处，地狱中阴头朝下。**

#### 胎卵入胎
- 中有从远处见父母交媾
- **男胎**：对父生嗔、对母生淫欲颠倒心——想与母交媾而前往入胎，入胎后在母右侧面朝背部蹲坐
- **女胎**：对母生嗔、贪恋父——在母左侧面朝前

#### 其他
- 湿生：希求清净/不清净生处的香而前往
- 化生：见生处生起欲望爱心而入
- 地狱中有：阎罗卒绳索系头朝下带走
- 天人：头向上
- 人：如人等

### 四种入胎（明不明知胎事）

> **一者明知而入胎，他者亦有知住胎，亦有知生余均迷，卵生亦有恒不知。**

| 补特伽罗 | 知入胎 | 知住胎 | 知出胎 |
|---|---|---|---|
| 转轮王 | ✓ | ✗ | ✗ |
| 独觉（麟角喻） | ✓ | ✓ | ✗ |
| 佛陀 | ✓ | ✓ | ✓ |
| 卵生独觉（众部独觉，如近岩尊者等） | ✗ | ✗ | ✗ |
| 其他众生 | ✗ | ✗ | ✗ |

理由：
- 转轮王福德广大
- 缘觉百劫积资+精通六事
- 佛陀三大阿僧祇劫积福慧

## 三、十二缘起（第57-61课·品3核心）

### 略说

> **此即缘起十二支，分为三份前与后，各自均有二缘起，中间八种寿圆具。**

十二支 = 无明→行→识→名色→六处→触→受→爱→取→有→生→老死。

梵语 "札德达雅萨莫巴达"——**缘起**：引申为因缘果聚合后真实出现。

**三部分**：
| 部分 | 支 | 所属世 |
|---|---|---|
| 前世 | 无明、行 | 前世 |
| 中间 | 识~有（8支） | 今世 |
| 后世 | 生、老死 | 后世 |

一个补特伽罗具足十二缘起的条件：**胎生+寿命圆满**（其他生与中间死亡者不具足）。

### 各自法相（分位缘起）

> **烦恼现时为无明，宿业诸识名为行，识即结生之诸蕴，名色显露六处前，六处三者聚前蕴，触能知苦乐等前，淫爱之前乃为受，爱即行淫之贪者，取为得财而奔波，有为能生后世果，结生之蕴即是生，至受之间为老死。**

**分位缘起**（有部立场）：各支以**五蕴的不同阶段**定义——即每一支都是"那一阶段的五蕴"。

| 支 | 阶段 |
|---|---|
| 无明 | 前世烦恼现行阶段的五蕴（=烦恼） |
| 行 | 前世造善等三业阶段的五蕴（=业） |
| 识 | 今生结生到母胎刹那的五蕴 |
| 名色 | 结生刹那起到显露六处前的五蕴（名=受想行识，色=凝酪等） |
| 六处 | 对境/根/识三聚合前的五蕴 |
| 触 | 三者聚合断定对境起到了知乐/苦/舍前的五蕴 |
| 受 | 能取舍三受起到不能行淫前的五蕴 |
| 爱 | 未真实享用欲妙/行淫/寻觅对境前的五蕴（=行淫的贪） |
| 取 | 为获成爱对境的受用奔波时的五蕴 |
| 有 | 奔波者造后世善不善业阶段的五蕴 |
| 生 | 此世死去结生后世时的五蕴 |
| 老死 | 中间今世留下的后世名色/六处/触/受四者 |

**"传说"问题**：世亲论师以"传说"标识——这是**有部分位缘起观点**；经部认为无明等各有独立心所本义。

**四种缘起**：
1. 刹那缘起
2. 相续缘起
3. 相联缘起
4. **分位缘起**（此处所讲）

### 定数之理由

> **为能遣除前后际，以及中间之愚痴。**

分十二支三份有必要——断除**三种愚痴**：
- 前际：我以前是否生/生什么/如何生
- 中间：现在如何
- 后际：未来产生与否/产生什么/如何产生

### 观察归摄

> **烦恼有三业有二，七事如是亦为果。前际后际略因果，由中因果可推断。**

**三法归摄**：
| 类 | 支 |
|---|---|
| 烦恼 | 无明、爱、取（3） |
| 业 | 行、有（2） |
| 事 | 识、名色、六处、触、受、生、老死（7） |

七事既是因也是果（由业惑生，又产生业惑）。

### 前后际因果

| | 因 | 果 |
|---|---|---|
| 前际 | 无明+行（2） | 识~受（5） |
| 后际 | 爱+取+有（3） | 生+老死（2） |

"前际略因果"：直接略说前际因（2）与后际果（2）；"后际详因果"：后际因（3）与中间果（5）——由**比量**可推出各缺分。

### 定数之理由（生起轮回的规律）

> **烦恼中生烦恼业，从彼业中产生事，事中生事与烦恼，此乃有支之规律。**

| 生起 | 例 |
|---|---|
| 烦恼→烦恼 | 爱→取 |
| 烦恼→业 | 取→有；无明→行 |
| 业→事 | 行→识；有→生 |
| 事→事 | 识→名色……触→受；生→老死 |
| 事→烦恼 | 受→爱；老死→无明（爱=无明故） |

**破前后无始/有始**：
- 无明之因=老死
- 老死之果=无明

《起生经》："三者（烦恼3）中生二（业2），二中产生七（事7），七中亦生三（烦恼3），有轮复旋转。"

### 缘起与缘生

> **此许缘起即是因，缘生承许为果法。**

- **缘起**：产生后世之因的支分
- **缘生**：以前所生之果的支分
- 各支都成为因果二者，但观待不同境：**观待儿子的父亲不是彼儿子的儿子**

### 宣说无明（第60课）

> **智慧违品之他法，无明如妄非亲等。**

**无明本体**：成为如实了达智慧之违品的一种心所他法（不只是"无觉知"或非遮许）。

喻：
- 非亲=唯一指亲友的违品敌人
- 妄=不真实的一种他法

**无明的教证标识**：经云"无明之结/缚/随眠/瀑流/行"——若只是"无觉知"则无法安立这些结构。

**辩破**：
- 他宗立"劣慧坏聚见即无明"
- 破：坏聚见是见，见与无明相应，两个智慧实体相应不可能；经说"无明染污智慧"，智慧染污自己自相矛盾

### 宣说名与触（第60课）

> **名即非为色法蕴。触六聚合而产生。**

- **名**：受想行识四蕴（随同名称趋入意义之故）
- **触**：六种——眼触~意触，由境/根/识三者聚合（能断定美境）而产生

**触的分类**：
| 类别 | 内容 |
|---|---|
| 有碍触 | 前五触（依有碍根生） |
| 名触 | 第六意触（缘名称） |
| 明触 | 与无垢无漏智慧相应 |
| 无明触 | 与染污智慧相应 |
| 其他触 | 有漏善法+无覆无记法 |

**无明触再分**：与害心相应 / 与随贪相应。

**所有触归纳**：三类——感受乐/苦/舍。

### 细述受（第60-61课）

> **从中所生之六受，前五身受余心受。**

从六触生六受——前五身受，第六心受。**近心之因中生总共十八种**受（悦意/不悦/等舍×六境）。

### 十八受的界地分布

详细计算略，简而言之：
| 界 | 受的数量 | 所缘 |
|---|---|---|
| 欲界（缘自） | 18 | — |
| 欲界缘色界 | 12 | （色界无香味） |
| 欲界缘无色界 | 3 | （无色界无五境） |
| 一禅/二禅 | 12（无意苦受） | 缘欲界 |
| 色界缘自界 | 8 | — |
| 色界缘无色 | 2 | — |
| 三禅/四禅 | 6（仅舍受） | 缘欲界 |
| 三禅四禅缘自地 | 4 | — |
| 三禅四禅缘无色 | 1 | — |
| 空无边处未至定 | 4 | — |
| 空无边处以上未至定 | 1 | — |
| 无色界四正行 | 1 | — |

十八受均是**有漏法**（能增上爱）。

### 为何只广说无明/名/触/受
- 色/识/六处——品1已讲
- 行/有/爱/取——品4/品5将讲

### 以比喻说明三支（第61课）

> **于许烦恼如种子，如龙根树与糠秕，业如具皮果药花，事如成熟之饮食。**

十二缘起的**烦恼/业/事**三摄可以五喻说明：

| 比喻 | 烦恼 | 业 | 事 |
|---|---|---|---|
| 种子→茎叶果 | 能生三类 | — | — |
| 龙→大海不干 | 烦恼在，三有海不干 | — | — |
| 树根→树叶 | 未断则生轮回 | — | — |
| 树→复生叶 | 能再度生业事 | — | — |
| 糠秕→苗芽 | 由烦恼得绳之业生果 | 具糠秕果（同上） | — |
| 药一次性 | — | 业只产生一次异熟 | — |
| 花→果近因 | — | 业是事近因 | — |
| 烹调饮食 | — | — | 只是所享用的 |

### 摄义（第61课）

> **四有之中生有惑，自地根本诸烦恼，其余三有具三种，无色界则无中有。**

| 四有 | 本体 |
|---|---|
| 生有 | 必烦恼性（以自地一切根本烦恼结生，非他地） |
| 中有/本有/死有 | 具善/不善/无记三种 |
| 欲界/色界 | 具四有 |
| 无色界 | 只三有（无中有） |

## 四、四食（识住之理，第62课）

> **有情依食而存住，段食欲界三处性。**

众生依靠**食**而存住。经云："本师成佛不久，则恩赐一法，即众生以食而存。"

### 四食

| 食 | 本体 | 界 |
|---|---|---|
| 段食（kavaḍīkāra-āhāra） | 香味触三处（欲界） | 欲界 |
| 触食 | 触（有漏） | 三界 |
| 思食 | 思（有漏） | 三界 |
| 识食 | 识（有漏） | 三界 |

### 段食的特殊性
**段食**：鼻舌身一口一口分段享用。**色法不属段食**——色处不利于眼根且对产生所依无利。

**地狱段食**：铁球与铜汁；近边地狱/孤独地狱有如人食。

**饥饿所逼见食的质难**：仅见食物就有利吗？
回答：仅见食而悦意只成**触食**（乐受感受），非色食；若仅见食便可满足，则离欲圣者仅见食就应满足，实际上仅见色处连从烦恼解脱尚且无济。

**后三食**：触/思/识均是**有漏食**（无漏法能灭世间，非食）。三界均有。

### 求生者异名

> **意成求生及寻香，中有以及现成名。**

中有别名：
| 名 | 释 |
|---|---|
| 意成 | 由意化生，不依父母不净种子 |
| 求生者 | 寻觅本有的形象 |
| 寻香 | 寻找食之香气 |
| 中有 | 住于死有与生有之间 |
| 现成 | 即将现前来生 |

### 四食的必要

> **前二食为利此世，增长所依及能依，为引他世与形成，次第而说此四食。**

| 食 | 必要 |
|---|---|
| 段食 | 增长此世心心所的所依身（根） |
| 触食 | 增长能依的心与心所（主要为此世） |
| 思食 | 引后世 |
| 识食 | 依业牵引之识形成后世 |

**间接作用**：段食/触食也间接引后世（产生烦恼→积业）。思食/识食也能令众生住留（公案：严重饥荒父亲挂装灰尘口袋称面粉，二子心存念未死，知真相后死去）。

## 五、识住之摄义（延续）

见《品3-分别世间品概览》§识去之理（死心/生心/断善根/圣者邪道）。

## 关键颂词

1. **有情具卵等四生，人与旁生具四种，一切地狱与天界，中有唯有一化生。**
2. **死有中有间蕴生，未至应至之境故。**
3. **此即缘起十二支，分为三份前与后。**
4. **烦恼中生烦恼业，从彼业中产生事，事中生事与烦恼。**
5. **智慧违品之他法，无明如妄非亲等。**
6. **此许缘起即是因，缘生承许为果法。**
7. **有情依食而存住，段食欲界三处性。**

## 与其他典籍的对应

- **《入中论》**：缘起观以中观破自性，俱舍立有部分位缘起即"所破对象"
- **《宝性论》**：十二缘起的还灭次第（反向）在修道论中继承
- **《前行》**：十二缘起的流转与还灭是共同外加行"轮回过患"的法相基础
- **《中论》·观因缘品**：品3的四生（卵胎湿化）+十二缘起的有部自性见，正是《中论》所破

## 修行关联

- **分位缘起**给修行以清晰的对治地图：断一支即断轮回（"无明"是总对治）
- **中有身相**（欲界中有同五六岁孩童）——为中阴救度提供法相依据
- **四食**框架：修行须断段食贪（止）+触食贪（观）+思食贪（定）+识食贪（慧）
- **入胎四种知**（转轮王→独觉→佛陀）：显示福慧的真实差别
