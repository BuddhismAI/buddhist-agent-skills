# 分别圣道品概览

## 在全论中的位置

品5剖析"为何轮回不止"（烦恼），品6开出"药方"——以四谛为所缘建立从资粮道到无学道的修行次第，以及四向四果的证悟体系。品5与品6构成因果对应：烦恼的分类对应道的对治。

## 四大科判（第121-138课）

### 甲一：所缘境圣谛（第121-122课）

四谛十六行相——苦（无常/苦/空/无我）、集（因/集/生/缘）、灭（灭/静/妙/离）、道（道/如/行/出）。现证次第：先苦后集再灭终道（如病人先知病→病因→康复目标→服药）。三苦遍一切有漏法。旁述二谛（胜义谛/世俗谛）。

### 甲二：现证真谛之次第（第122-127课）

**资粮道**：闻思修三慧为体，以四念住观身受心法自相总相。

**加行道四位**：
| 位 | 功德 | 所缘 |
|----|------|------|
| 暖位 | 必得涅槃 | 四谛十六行相 |
| 顶位 | 不断善根 | 四谛十六行相 |
| 忍位 | 不堕恶趣 | 上品仅缘欲界苦谛 |
| 胜法位 | 世间最胜法，一刹那性 | 仅缘欲界苦谛 |

**见道**：十六刹那无漏智。前十五刹那为见道，第十六刹那有部判为修道。法忍法智→类忍类智×四谛=十六刹那。

### 甲三：现证真谛之补特伽罗（第127-135课）

**离贪者与渐次者**：离贪者先以世间道断八地贪后修见道；渐次者从见道后次第获四果。

**四向四果**：预流（极七返/家家）→一来（断六品）→不来（七善士趣、五种行色界者、轮番修、身现证）→阿罗汉（六种：退法/思法/护法/安住/堪达/不动）。

**阿罗汉退转**：有部独特观点——前五种钝根阿罗汉会从果位退失（但退后必恢复、死前必重得果）。经部以上不承认。

### 甲四：宣说现证之道（第136-138课）

**四道**：加行道/无间道/解脱道/胜进道。

**三十七菩提分法**：名称三十七，实体归十类。资粮道对应前三组（念住/正断/神足），加行道对应五根五力，见道七觉支，修道八正道。七觉支与八正道唯无漏。

**金刚喻定**：有顶第九品烦恼的无间道，摧毁三界一切随眠，属有学道末尾。

**解信四种**：对佛法僧戒的不退信心，见三谛得法解信与戒解信，见道谛得佛僧解信。

## 核心概念

| 概念 | 含义 |
|------|------|
| 四谛十六行相 | 苦集灭道各四相，见道所证核心 |
| 见道十六刹那 | 法忍法智类忍类智×四谛 |
| 离贪者/渐次者 | 先世间道后出世道 vs 纯出世道次第 |
| 加行道四位 | 暖/顶/忍/胜法，功德递增 |
| 三十七菩提分法 | 名三十七实体十，显密共许唯一道 |
| 金刚喻定 | 有学道末尾最胜无间道 |
| 六种阿罗汉 | 退法至不动法，对应六种根性 |

## 有部/经部关键分歧

- **世间道断烦恼**：有部=断根本；经部=仅暂时压制
- **阿罗汉退转**：有部=前五种会退；经部以上=不退
- **离得实有**：有部=实有法；经部=假立
- **见道刹那数**：有部内部有一/四/八/十二刹那异说

## 品6独特贡献

品6是全论"修道论"的核心——从资粮道到无学道的完整路径图，四向四果的详尽分类（尤其不来果的九种分类），以及三十七菩提分法的系统阐释。这些内容在《现观庄严论》中被完整继承并在大乘框架中扩展。
