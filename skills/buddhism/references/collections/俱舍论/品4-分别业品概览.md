# 分别业品概览

## 在全论中的位置

品3描绘"众生在何处轮回"，品4回答"以何力轮回"——业（karma）是轮回的动力引擎。品4是八品中最大的一品（28课、918行论释），反映了业在阿毗达磨体系中的核心地位。

> **形形色色世间界，皆由众生业所生。**

## 论证脉络

### 乙一：业之本体（第75-84课）

**思业与思作业**：一切业归根结底由心（思心所）发起——思业即意业，思作业即身语业。

**有表业与无表业**（第75-77课）：有表业为外显动作/声音，有部认为是形色/声音（色法），经部安立于心。无表业为不可见的善恶色法，有部认为实有色法，经部认为是心上习气。

**三种戒律**（第79-84课）：
- **别解脱戒**（prātimokṣa）：八种归四类（比丘/比丘尼/正学女/沙弥沙弥尼/居士/斋戒），以羯磨仪轨获得
- **禅定戒**：色界善法随心产生
- **无漏戒**：圣者随心产生
- 三戒异体不违——可同时具足

别解脱戒的重要细则：斋戒八分支、居士一分至圆分、皈依三宝为前提。

### 乙二：经中所说名称（第85-102课）

**业的多维分类**：
- 果之方式：福德业/非福德业/不动业
- 受业分类：顺乐受/顺苦受/顺舍受
- 定业与不定业：顺现法/顺次生/顺后世/不定——定业三条件（强烈心/殊胜对境/恒常造）
- 因果方式：黑业/白业/杂业/无漏业

**十善业道与十不善业道**（第90-95课）：品4最核心的实践内容。身三（杀生/不予取/邪淫）语四（妄语/离间/粗语/绮语）意三（贪心/害心/邪见）。详析杀生三要素、邪淫四种、四名言。

**五无间罪与近无间罪**（第96-99课）：杀父/杀母/杀阿罗汉/破和合僧/出佛身血。破和合僧为最重罪（详析法轮僧vs羯磨僧、六条件、六不出现时间）。

**引业与满业**（第96课）：引业决定投生何趣（总业），满业决定一生苦乐（别业）。有部一业引一世，经部一业引多世。

**三福业之事**（第100-102课）：布施/持戒/修行。布施之施主/施物/福田三殊胜。六因决定业轻重。法布施最殊胜。

## 核心概念

| 概念 | 含义 |
|------|------|
| 思业/思作业 | 意业/身语业，一切业以心为本 |
| 有表业/无表业 | 可见动作声音/不可见善恶色法 |
| 等起（因时/彼时） | 最初发心/正行时之心 |
| 定业 | 强烈心+殊胜对境+恒常造→必定受报 |
| 引业/满业 | 总业（决定趣）/别业（决定苦乐） |
| 断善根 | 唯诽谤因果邪见可断，以怀疑或正见恢复 |
| 六因决定业轻重 | 后行/田/基/加行/思/意乐 |
| 作已积集业 | 故意+支分圆满+不悔+无对治+具从属+必定异熟 |

## 有部/经部关键分歧

| 议题 | 有部 | 经部 |
|------|------|------|
| 身有表业 | 实有形色 | 应安立于心 |
| 无表色 | 实有色法 | 心上习气 |
| 戒律本体 | 无表色（色法） | 断除恶心（心法） |
| 意三不善 | 非业而为恶行 | 亦为业 |
| 引业 | 一业引一世 | 一业引多世 |

## 修行关联

品4是全论实践性最强的一品：十善业道的具体标准为日常持戒提供判断依据；定业条件为因果定解提供框架；军兵共业原则（"一切人均如作者"）警示集体业力；三福业之事总结了积资粮的三大途径。
