# 三戒体系·别解脱戒·禅定戒·无漏戒

## 定位

品4 无表业的主要落脚点是**戒律**——在有部框架下，戒是一种"断除破戒相续"的无表色。此科目是律藏法相的直接基础：比丘戒、比丘尼戒、居士戒、斋戒的界限与得舍，都在此确定。大乘戒律学（菩萨戒）在此之上另立戒体，但法相分析工具（羯磨、学处、四根本罪）仍沿用俱舍。

> **所谓三种无表色，即戒恶戒与中戒。**

无表色有三类：
- **戒**（律仪, saṃvara）：遮止破戒相续的善无表色
- **恶戒**（不律仪）：如屠夫长期造不善业的不善无表色
- **中戒**：暂时受持二戒任一（如某日受斋戒、某日猛烈造恶业）

## 三戒的总法相

> **戒有所谓别解脱，无漏禅定所生戒。**

| 戒 | 法相 | 所摄 |
|---|---|---|
| 别解脱戒（prātimokṣa） | 包括在欲界中的一种出离戒 | 欲界 |
| 禅定戒（dhyāna-saṃvara） | 属于色界的一种善法戒 | 色界 |
| 无漏戒（anāsrava-saṃvara） | 无漏法的戒律 | 无漏 |

## 别解脱戒

### 八种归四类

> **所谓八种别解脱，实际戒体唯有四，除名称外无变故，彼等异体不相违。**

八种别解脱戒：比丘、比丘尼、正学女、沙弥、沙弥尼、男居士、女居士、斋戒。

归纳为**四类戒体**：
1. 比丘 / 比丘尼（同类戒体，变性不舍）
2. 正学女 / 沙弥 / 沙弥尼
3. 男居士 / 女居士
4. 斋戒（一日戒，时间短故不分男女）

**戒体异体不相违**：次第得受断杀等戒时，生起异体的三戒（如受比丘戒时仍保有沙弥戒的断杀戒体）——还比丘戒后沙弥戒仍可存在。一补特伽罗相续中可同时具足三种异体的断杀戒，**受后戒不是舍前戒之因**。

### 五戒次第

> **受持断除五八十，以及一切诸所断，依次立名为居士，斋戒沙弥与比丘。**

| 戒名 | 戒数 | 时长 |
|---|---|---|
| 居士戒 | 断四根本罪+饮酒=5 | 有生之年 |
| 斋戒 | 居士五+歌鬘+高广床+非时食=8 | 一日 |
| 沙弥戒 | 斋戒分开歌鬘+取金银=10 | 有生之年 |
| 比丘戒（近圆戒） | 身语一切所断 | 有生之年 |

### 释词

| 名 | 含义 |
|---|---|
| 律仪（saṃvara） | 消除破戒酷热获得清凉 |
| 妙行 | 蒙受圣者赞叹 |
| 业 | 所作的自性 |
| 戒律 | 禁止身语的非事 |
| 别解脱 | 得戒第一刹那的有表与无表色开始脱离恶业 |
| 道 | 彼业的途径 |

### 居士
"居士"即应作为比丘、比丘尼的仆人去承侍他们。

### 沙弥
勤勤恳恳为上师做事。

### 近圆（比丘戒）
圆=远离轮回所有损害的涅槃，与之趋近，故称近圆。

## 禅定戒与无漏戒

> **生禅定者具禅戒，圣者具有无漏戒。**

- **禅定戒**：具足禅定所生等持者即具足。
- **无漏戒**：圣者有情具有。

### 随心戒 vs 非随心戒

> **最后二者随心戒，无间道中生彼二，未至定中称断除。**

- **随心戒**：禅定戒、无漏戒——随心是否存在而变化，依赖入定。
- **非随心戒**：别解脱戒——散乱与无心阶段也存在。
- **断除之戒**：禅定戒与无漏戒在无间道中生起，若依一禅未至定的无间道，称**断道**。

### 意戒与根戒
经中说"身戒善妙、语戒善妙、意戒善妙"，意戒指从属于意识的正知正念；根戒指随从根识产生的正知正念（"眼根以戒守护而住"）。

## 具戒之理（第79-80课）

### 具无表色之理

> **乃至守别解脱者，未舍间具现无表，一刹那后具过去。**

| 补特伽罗 | 无表色之有 |
|---|---|
| 守别解脱者 | 未舍前具现在无表色；一刹那后兼具过去无表色 |
| 守恶戒者 | 同上（名词转换） |
| 禅定戒者 | 第一刹那起恒具过去与未来无表色（以前戒的得绳牵引） |
| 圣者第一刹那 | 不具过去无表色（以前未生无漏戒） |
| 入定/住圣道 | 具现在无表色；出定时不具 |
| 中戒者 | 不一定有；若有，第一刹那具现在中戒无表色，之后具过去+现在两种 |
| 守恶戒顶礼 | 具善无表色 |
| 守戒者杀生 | 具不善无表色 |
| 无表色具时 | 乃至未舍强烈清净心或强烈烦恼心之前 |

### 四类具戒（第80课）

> **守中戒者心弱故，若作则具有表色，舍弃有表尚未生，圣者则无有表色。**

| 类别 | 情形 |
|---|---|
| 具有表不具无表 | 以前不具无表色守中戒者，现心微弱只身语行事 |
| 不具有表具无表 | 前世有表色已舍、后世有表色未生的圣者补特伽罗，只具无漏无表 |
| 既具有表又具无表 | 普通众生（如我等） |
| 既不具有表也不具无表 | 无色界等众生 |

## 戒律之得法（第80-83课）

### 禅定戒与无漏戒
> **禅定戒依定地得，无漏戒依圣道获。**

- 禅定戒依未至定与正禅而得。
- 无漏戒依圣者无漏道而得（具无漏静虑六地之一）。

### 别解脱戒
> **所谓别解脱戒者，依他有表色等得。**

七种别解脱戒（除斋戒）从亲教师等他人的有表色中得到。

**十种近圆**（"等"字所指）：
1. 佛与缘觉——自然近圆（依证自然智慧生比丘戒）
2. 五比丘——趋入真谛
3. 妙誉尊者等——依靠善来
4. 大迦叶——承认导师
5. 善施——答问心喜
6. 众生主母等——宣说八难断法
7. 供施女——依僧传言
8. 中土十人、边地五人——白四羯磨
9. 六十善群比丘——许三皈依
10. 自然近圆与趋入真谛近圆不观待有表色

### 戒恶戒时间固定
> **即于有生之年中，及一日内真受戒，无有一日之恶戒。**

七种别解脱戒须有生之年真实受，斋戒一日内受。恶戒无一日受法。

### 斋戒之特点（第81课）

#### 受法
- 受戒者在较传戒者低的坐垫上蹲坐合掌
- 跟随传戒者所说重复三遍
- 不佩带华丽装饰
- 至第二天日出前
- 具足八分支
- 在比丘等他者面前受
- 时间是早餐之前

#### 分支固定（为八）
| 戒数 | 分类 | 理由 |
|---|---|---|
| 断四根本罪（四） | 戒律之支 | 断除非理作意 |
| 戒酒（一） | 不放逸之支 | 酒致放逸、丧失正念 |
| 歌鬘/高广床/非时食（三） | 禁行之支 | 处于厌离心，防傲慢 |

### 居士戒之特点（第82课）

#### 受法之诤
经云："身着白衣具根之在家男女诵'皈依佛、皈依法、皈依僧，祈请摄受吾为居士'而成居士。"

- **经部宗**：仅此不能生起居士戒。
- **克什米尔的论师（有部）**：先皈依再承诺为居士，则能生起戒体。

#### 皈依境
> **所谓皈依三宝尊，即是皈依能成佛，无学法与有无学，僧众以及涅槃法。**

皈依三宝 =
- 皈依**能成佛的无学法**（尽智、无生智），不皈依色身（色身与凡夫无异）。
- 皈依所有的佛（诸佛道相相同），同理僧众。
- 皈依**唯一的涅槃灭谛**（法）。

#### 支分固定
为何居士五戒立"断邪淫"而非"断非梵行"？
1. 邪淫侮辱他人之妻，是堕恶趣因，极受谴责。
2. 在家身份易断邪淫，非梵行难断。
3. 圣者于他世获不行邪淫之戒，非梵行非如此。

为何四语业中只立"断妄语"？因若已违越所有学处、别人问是否犯戒时会再说妄语，断妄语可防止双重罪。

为何佛制罪中唯戒酒？**饮酒是所有学处的总违缘**——断酒即守护其他所有学处。

### 戒律之基

> **欲界所摄一切戒，依二者与现众得。**

别解脱戒由加行、正行、后行中获得，依靠断除**自性罪与佛制罪**二者之基**众生**与**非众生**而得。

| 类别 | 例 |
|---|---|
| 自性罪基众生 | 杀生 |
| 自性罪非众生 | 盗金 |
| 佛制罪基众生 | 接触女人 |
| 佛制罪非众生 | 割草 |

依**现在众生**而得，非过去未来。

禅定戒与无漏戒只依断不善业的**正行**获得（不观待加行后行），并**依三时一切时间**（含过去未来众生）。

> **戒依一切有情得，支分与因有差别。**

戒律依一切众生而得（断除对一切众生的罪业）。支分与因有差别：
- 其他戒从身语四所断中得；比丘戒从身语七所断中得。
- 三根本善作因，可三者同时并存；三品等起（下中上）不能共存，仅从任一等起中获得。

### 恶戒与中戒之得

> **恶戒则由行彼事，或由承诺而获得。**

恶戒得法：
1. 转生屠夫种姓，开始杀业即得。
2. 面对境主人发誓"我要以此维生"，生颠倒意乐并着手做。

> **剩余无表则依田，承诺恭敬而获得。**

中戒之得：
1. 由田得（如七种实生福）
2. 由承诺得（如发誓未顶礼佛不用餐）
3. 由恭敬心得（猛烈发心）

## 戒律之舍法（第83-84课）

### 别解脱戒之舍

> **还戒以及死亡时，出现两性断善根，抑或已经过一夜，则已舍别解脱戒。**

1. 在知言解义者前诚心还戒（生与受戒相违之有表色）
2. 死亡（所依不存）
3. 顿时出现两性（失所依）
4. 断了善根（断绝戒之根本）
5. 过一夜舍斋戒（时间圆满）

#### 犯根本戒是否舍戒？
- **尼洪（西方）论师**：犯任何根本戒，所有戒律舍。
- **红衣部（东方）论师**：正法隐没舍别解脱戒。
- **世亲**：正法隐没是戒律羯磨不存，不能重新得戒，非说已得之戒重新舍。
- **克什米尔论师（有部主流）**：犯根本罪后**既具戒又破戒**（如人既负债又有财产）——忏悔堕罪而得阿罗汉。

### 禅定戒与无漏戒之舍

> **禅定所摄之善法，由从转生退失舍。**

- 禅定戒：转生或从等持退失时舍彼地戒。
- 无色所摄：同上。
- 圣戒：得果时舍四向所摄无漏戒；修炼根时舍钝根所摄戒；从果退失时舍殊胜所摄戒。

### 恶戒之舍

> **恶戒乃由得戒死，出现两性而舍弃。**

1. 得戒时（生起强对治）
2. 死亡
3. 顿时出现两性

### 中戒之舍

> **中戒则由力所受，事寿根本中断舍，欲界所摄非色善，由断善根转上舍。**

1. 清净、烦恼力量中断
2. 承诺时间已过
3. 时间未过但未按诺言去做
4. 所依佛塔或渔网等已毁
5. 死亡
6. 断绝善根

欲界非色善由断善根或转生上界舍；三界非色烦恼性依生各自无间道对治而失。

## 具戒之补特伽罗（第84课）

> **北俱卢洲二黄门，两性除外一切人，可具恶戒戒亦尔。**

| 补特伽罗 | 具戒情况 |
|---|---|
| 除黄门、北俱卢洲人、两性外的所有人 | 可具恶戒与戒 |
| 黄门等 | 善恶心不稳固，不具恶戒与戒 |
| 欲天 | 可具戒 |
| 人类 | 可具三戒 |
| 转生欲色诸天人 | 具禅定戒 |
| 除中定与无想天外的色无色天人 | 具无漏戒 |

## 有部/经部/大乘对戒体的分歧

| 立场 | 戒体 |
|---|---|
| 有部 | 无表色（实有色法） |
| 经部 | 断除恶心的心法/种子（非实色法） |
| 大乘菩萨戒 | 相续断恶修善的菩提心愿力 |

回答"戒体是什么"问题时须辨明所据立场。（详见 `topics/abhidharma/index.md §Correctness Anchors · 无表色不是大乘戒律观`）

## 关键颂词

1. **所谓三种无表色，即戒恶戒与中戒。**
2. **戒有所谓别解脱，无漏禅定所生戒。**
3. **所谓八种别解脱，实际戒体唯有四。**
4. **受持断除五八十，以及一切诸所断，依次立名为居士，斋戒沙弥与比丘。**
5. **所谓别解脱戒者，依他有表色等得。**
6. **还戒以及死亡时，出现两性断善根，抑或已经过一夜，则已舍别解脱戒。**

## 修行关联

本章的实践意义：
- 受戒前**皈依为前提**（居士/斋戒均须先皈依）
- **三戒异体不违**——可同时具居士+沙弥+比丘三层断杀戒
- **断酒**是守一切学处的根本违缘对治
- **过一夜舍斋戒**——斋戒时间圆满后不自动延续，须重新发心
- **犯根本戒具戒vs破戒并存**（克什米尔论师）为忏悔还净留下依据
