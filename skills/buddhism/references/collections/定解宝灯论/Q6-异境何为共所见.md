# 问六·异境何为共所见

## 核心问题

六道众生面对同一外境（如水）所见各异——地狱众生见铁水、饿鬼见脓血、人见水、天人见甘露。是否存在一个"共同所见"？若有，是什么？

此问包含全论最系统的**量论（pramāṇa）框架**——三层量的架构、量之观待成立原则、净见量教义。

## 六道所见对照

| 众生层次 | 所见 |
|---------|------|
| 地狱众生 | 滚滚铁水 |
| 饿鬼 | 腥臭脓血 |
| 人类/大多数旁生 | 清净水 |
| 天人（阿修罗） | 清净甘露 |
| 具修证持明者 | 玛玛格佛母（水大佛母） |
| 佛陀（如所有智） | 离一切戏论之法界 |
| 佛陀（尽所有智） | 清净法界明分 |

## 破三种他宗

### 第一他宗：水为共同所见（格鲁派）

> **有谓共见乃为水，见彼有境均正量。**

宗喀巴大师（《善解密意疏》）及弟子们主张：一碗水有许多支分，六道各见其中一分。

**破斥一——量非量无差别**：

> **若水少许有自性，则无正量与非量。**

若水有少许自性存在，六道众生见水的心识均为正量，则正量与非量的差别不复存在——饿鬼见脓亦应为正量（因为水上也有脓的一分）。

**破斥二——共同所见不可能**：

> **倘若各自有情前，现境悉皆不相同，则如见柱瓶眼识，共同所见不容有。**

六道所见本质完全不同（铁水、脓、水、甘露），如同一人见柱子另一人见瓶子——根本不能承许一个共同所见。

萨迦派果仁巴的补充批评：若六道各见水之一分，则一碗水已成六碗；或喝一碗水时只喝了六分之一。

### 第二他宗：湿性为共同所见

> **有谓共见为湿性。**

班钦索照、克主杰、蒋扬夏巴等主张：铁水、脓、水、甘露皆具湿润本性，故"湿性"为共同所见。

**破斥一——以现为多种而破**：

> **共同所见仅湿性，不灭真实而存在，现多不同所见境，一者现见另未见，水脓等基为何者？**

饿鬼所见之脓人根本未见，人所见之水饿鬼完全见不到——这个"共同之湿性"究竟在哪里？

**破斥二——以未周遍而破**：

> **此外空无边处众，如何会见湿性境？**

无色界空无边处众生整日安住于虚空禅定，根本不存在任何湿性对境。湿性说无法周遍一切众生。

**破斥三——以一异体而破**：

> **湿性若与水一体，则脓等物不能现，除水等外不同体，任何湿性不可得。**

- 若湿性与水一体：则除水外脓、甘露等不应显现
- 若湿性是水之外的异体：具眼识者应能现见独立的"湿性"，然而无人见过——因明中"可见而未得"的过失

> **各自不同所见前，共同所见不能有，共同所见之事物，不可能现不同故。**

### 第三他宗：无共同所见（类唯识立场）

> **若无共同所见境，则如唯识需承认，无有外境识为境，此种观点不合理。**

**破唯识——三重论证**：

**（一）心境等性原则**：

> **所取能取此二者，世俗显现相同故。于诸显现若观察，二取不应分有无，如有现境然虚妄，心识亦不成立故。**

月称论师《入中论》核心：在世俗中能取心与所取境平等显现，若外境不存在则心识亦应不存在。若以胜义观察则二者均不成立。不应割裂为一有一无。

> **无色不应执有心，有心不可执无色。**（月称《入中论》）

若无色法（外境），不应执著有心识；若有心识，不应执著无色法。《般若经》中佛将外境与心识双双遮破（皆空）。

**（二）梦境喻破斥**——击破唯识最常用的论据：

唯识宗以梦境比喻说"外境虽无、心识存在"。月称论师以精确逻辑击破此喻：

> **若谓如梦当思择，若时我说梦无心，尔时汝喻即非有。**

月称的反驳：梦中心与外境**同时存在**（梦中有山有人有感受），梦醒后二者**俱不存在**（醒后既无梦中之山，也无梦中之心）。梦境恰恰证明了心与境**同起同灭**，而非"有心无境"。唯识以此喻证明"有心无境"，恰恰自相矛盾。

**（三）焚山过失**（无垢光尊者）：

《大圆满心性休息大车疏》："若承许外境即是心识本身，那山若被火焚烧，则心识亦应被火焚烧"——荒谬不通。

**宁玛派立场的微妙区分**："外境并非心，但是心的显现"（无垢光尊者《如意宝藏论》）——承认外境是心的显现（法界明分之显现），但不等于外境就是心。

## 自宗：法界明分

### 暂时自宗

> **同见以及不同见，共同所见为明分，成立现基不能无，犹如观看戏剧等。**

**"法界明分"的内涵**：一切万法本体是现空无二（显宗）或觉空无二（大圆满）的法界。将法界光明之分与空性分析述，称其显现部分为"法界明分"——即缘起性空的显现分。实际上明分与空性始终无二无别。

**为何安立明分为共同所见——戏剧喻**：

- **时间维度**：一名演员上午扮天人下午扮魔鬼，虽显现不同但演员本身不可缺少
- **众生维度**：数百观众观同一戏剧感受各异，但"有演员表演"这一事实不可否认

> **除此存在明分外，他处存在不可能，是故若无此明分，一切不现如虚空。**

若不承认法界明分为共同所见，则名言中一切因果、业感皆无从安立。

**众生未能如实见真义的理由**：

> **内外诸缘所障故，不能如实见真义，如依魔术之咒语，木块现为马与象。**

凡夫被内（无始无明、烦恼习气）外（无具相善知识、罪障未净）障蔽，如同魔术师以咒力迷惑眼根——本来是法界明分，因业障而见为各种不净相。

### 究竟自宗

> **自宗不堕现与空，本基一切均不成，任何显现平等故，一法亦可现种种。**

共同所见既不堕于现也不堕于空，即现空双运本基。万法本体为空故能随业感显现为多种形态：若本体实有不空，只能显现为一法。

> **何者现空若圆融，则彼一切皆合理，何者现空不圆融，则彼一切均非理。**

## 量论体系——跨主题核心材料

此节建立了一套完整的三层量架构，是宁玛派量论的系统表述。

### 三层量架构

```
量
├── 胜义量
│   ├── 真实胜义量：中观应成派——离一切戏论
│   └── 假胜义量：中观自续派——抉择"空"
└── 名言量
    ├── 净见量：圣者/具证修者的清净认知
    └── 观现世量（凡夫名言量）
        ├── 现量：未被迷乱因染污的根识直接感知
        └── 比量：依逻辑推理间接获得
```

**1994年版明确说明**：因明七论（《释量论》等）仅抉择到观现世量。净见量在弥勒菩萨《宝性论》比较明显，《大幻化网总说光明藏论》主要抉择净见量。胜义量在中观论著中。分类依麦彭仁波切《解义慧剑》。

### 量之观待成立——核心哲学原则

> **故以观待量成立，独立自主之诸法，并非以量可成立，若成则应成实相。**

| 类型 | 能否以量成立 | 理由 |
|------|-------------|------|
| 观待而安立之法（如人前之水） | 能 | 观待特定众生业感而成立 |
| 独立自主（不观待任何法）之法 | 不能 | 若成立则成胜义实相——但无此法 |

> **一水执水量成立，不相观待独不成，胜义观察不成立，饿鬼前亦不成立。**

水对人是正量——但这是观待人类业感而成立：
1. **胜义量前不成立**：水的一滴也无法以自性成立
2. **饿鬼前亦不成立**：观待饿鬼，水根本不存在（只有脓）

故知水并非独立自主成立之法。

### 暂时名言量的安立

> **暂时饿鬼因业障，清净水亦见为脓，除障方见真水故，观待人见为正量。依靠他缘转变故，暂时安立水为量。**

暂时可安立：观待人类→水为正量；观待饿鬼→脓为量。但这种安立依靠他缘转变，不是独立自主成立。

### 净见量：上上为正量原则

> **是故障碍之外缘，愈来愈加清净后，观待下面所见言，当许上上为正量。**

障缘越清净，所见越为正量：

地狱 < 饿鬼 < 人 < 天人 < 持明 < 诸地菩萨 < 佛陀

上上所见观待下下为净见量意义上的正量。

> **究竟理证而观察，彼等皆为习气现，因水亦可现见为，清净刹土佛母身，故而人类所见者，不能决定是正量。**

以净见量进行究竟观察：一切不清净法均可转依为清净——人所见之水在持明者面前可见为玛玛格佛母。故人类所见不能决定为究竟正量。

### 究竟一正量：自然本智

> **究竟法性之真如，独一无二唯一性，能见正量亦唯一，第二正量不可有。**
> **实相双运唯一谛，正量自然之智慧，唯一所断即无明，仅觉未觉之差别。**

- 究竟实相：净等双运唯一真谛
- 能见此实相的正量：唯一的自然本智（遍知佛陀的智慧）
- 佛与众生的差别：**仅是觉与未觉的差别**，实相本身无差别

荣索班智达："众生和佛只是明与不明之别，众生因不了知诸法实相从而流转轮回，佛陀则已彻知万法本相，了知正于迷乱之此时亦根本未曾迷乱过。"

### 与解义慧剑的关系

2003年版第28课明确提及：麦彭仁波切《解义慧剑》对观现世量与净见量作了细致分析，同时对器情世界显现为清净的道理做了详细阐述。本品量论内容与《解义慧剑》构成直接呼应——量的分类体系、四种理的应用、证成理的展开皆同一框架。

## 大清净教义的建立

> **故以量理而建立，诸现自性净天尊，唯有前译派自宗，全知荣索班智达，犹如狮吼之善说，他宗对于此观点，无有如理说法故，如何承许均矛盾。**

宁玛派不共主张：**以净见量的名言量建立，一切显现的本来自性即是清净刹土与天尊。**

> **修此道时依净见，诚信不净现自解，现有清净之法身，通达金刚之教义。**

修行次第：不净现→依净见量认识本自清净→不净显现自解脱→通达"现有即清净法身"。

> **是故称为净见量，断除一切障碍故。诸法无谬之实相，唯佛现见无有他。**

## 主要教证

1. **月称《入中论》**："鬼见脓河心亦尔"（缘起例证）；"无色不应执有心，有心不可执无色"（破唯识）；"总如所知非有故，应知内识亦非有"
2. **龙树《中论》**："以有空义故，一切法得成"
3. **法称《释量论》**："心性为光明，诸垢客尘性"
4. **《大幻化网续》**："五蕴如幻无偏现，彼为清净天尊相"——净见量的密续依据
5. **荣索班智达《显现立为天尊》**：以净见量建立一切显现为本尊的不共观点
6. **麦彭仁波切《解义慧剑》**：观现世量与净见量的细致分析
7. **《宝性论》**：净见量的显宗依据

## 关键术语

| 术语 | 定义 |
|------|------|
| **法界明分** | 一切法本体（现空无二之法界）的光明显现分。暂时共同所见的自宗安立，也是如来藏本面 |
| **净见量** | 圣者通过清净见地对实相的如实现见。因明七论不涉及此量。分层：初地净见→佛陀彻见等性 |
| **观现世量** | 凡夫名言中未被暂时迷乱因染污的根识及推理所得正量。含现量+比量。因明主要范围 |
| **胜义量** | 抉择实相：真胜义量=应成派离一切戏论；假胜义量=自续派抉择"空" |
| **现基** | 各种显现所依托的本基——法界明分。若无现基一切如虚空 |
| **量之观待成立** | 一切正量均观待特定条件而成立，非独立自主。不观待之法以量不可成立 |
| **上上为正量** | 障缘越清净所见越为正量，上层众生的认知观待下层为正量 |
| **等净无二** | 大清净与大等性双运=显现清净分与胜义平等性无二无别 |
