# 定解宝灯论·结构总览

## 这部论在做什么

《定解宝灯论》(Nges shes sgron me) 是麦彭仁波切七岁时所造的一部论典，被宁玛派视为显密一切教法的如意宝藏。它不是一部系统建构的哲学论著，而是以**七个最令修行者困惑的难题**为线索，将中观见、修行方法、量论体系与密法精要编织成一个完整的"见修行果"架构。

这部论的独特价值在于：它是宁玛派将**中观空性**与**大圆满修证**统一起来的关键文本。每个问题都不是孤立的学术辩论，而是修行道路上必须获得"定解"才能前进的关卡——没有定解，一切修行都是"蒙而迷于幻网中"。

## 七问架构：见修行果

麦彭仁波切以"阿绕巴匝那德"六字（对应文殊心咒六字）统摄七个问题，按见修行果排列：

### 见（抉择正见 + 遣除见的歧途）

**问一·见解无遮或非遮**（阿字·无生门）——格鲁派主张究竟见为无遮（自空），觉囊等派主张非遮（他空）。麦彭仁波切超越二元：真实胜义中无遮与非遮皆不成立，二者仅是心之假立；但若单论空性，则应承许无遮——本净大空性。关键在于现空双运：空性不离显现，显现不离空性。此问奠定了全论的见解基础。

**问二·声缘证二无我耶**（绕字·离垢门）——声闻缘觉是否证悟了法无我？格鲁派主张三乘见道相同（圆证二无我），嘎玛拉希拉等主张完全未证。自宗取中道：声缘证悟了人无我及部分法无我（粗分），但未圆满证悟极细微法无我。此问是见解的"歧途标识"——提醒修行者不可止于声缘所证的有限空性。

### 修（抉择修行本体 + 修行方式）

**问三·入定有无执著相**（巴字·显现胜义门）——入定时应否保留"空"的执著？宗喀巴主张须保留无我执，部分前派大德主张一上来就无任何执著。麦彭仁波切的回答有次第：初学者须有无我定解之执著（否则形同睡眠），但究竟入定时一切执著（包括"空"的执著）皆须放下——因为真正的胜义离四边戏论。

**问四·观察修或安置修**（匝字·无生无死门）——修行应以观察分析为主，还是直接安住？此问与问三相似但角度不同：问三讨论入定的本体（有无执著），问四讨论达到入定的方式（观察还是安住）。麦彭仁波切的回答同样有次第：初期须观察修以生起定解，中期观察与安住交替，后期自然安住于先前观察所生之定解。此问包含大量**大圆满**内容——阿底约嘎为续之究竟、乘之极顶，直断与顿超的正行。

### 行（抉择见行统一 + 认知共同基础）

**问五·二谛何者为主要**（那字·离名称门）——胜义谛与世俗谛孰轻孰重？有三种他宗：偏重胜义、偏重世俗、或搁置胜义而只重世俗。自宗：二谛本来双运，不可偏废。此问从二谛双运推展到**密续分类**——以见解高低安立九乘次第，事行瑜伽各续的差别在于是否证悟现有大净等。此处有大量**密法**内容。

**问六·异境何为共所见**（德字·甚深智慧门，与问七共用）——六道众生面对同一外境（如水）所见各异，是否存在共同所见？他宗三种立场（水为共见、湿性为共见、无共见）皆被驳斥。自宗：暂时以"法界明分"为共同所见，究竟则现空双运之实相。此问包含大量**量论**内容——正量/非量、净见量/观现世量/胜义量的完整体系，与解义慧剑的量论框架形成呼应。

### 果（抉择中观的最终立场）

**问七·中观有无承认否**（德字·甚深智慧门，续）——中观自宗究竟有无承认？这是全论的收束，也是宁玛派中观立场的最终表述。格鲁派部分大德主张有承认，应成派的"无承认"则被一些人理解为虚无主义。麦彭仁波切依龙钦饶降的立场：实相大中观无有承认（入定智慧离一切戏论），现相名言中分二谛各有承认。此问还包含显密道共同要点与见解差别的总摄。

## 为什么七问是这个顺序

这不是随机排列，而是一条修行者的心路：

1. 先确立正见（问一）→ 避开见解歧途（问二）
2. 知道见解后如何入定（问三）→ 用什么方式达到入定（问四）
3. 入定后如何在行持中统一二谛（问五）→ 以什么量来认知世界（问六）
4. 最终安住于何种境界、以何种方式利他（问七）

每一问都在前一问的基础上推进。没有问一的现空双运见，问三的"离四边入定"无从理解；没有问四的观察修与安住修的次第，问五的二谛双运就缺少修行方法论的支撑。
