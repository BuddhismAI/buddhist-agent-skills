# 初善首义·定解与七问总纲

## "定解"——全论的核心概念

"定解"(nges shes) 不是普通的信心或知识，而是通过闻思修对佛法基道果任何一法所生起的**不退转的胜解**。麦彭仁波切以宝灯为喻：定解如珍宝之灯，能遣除无始以来心相续中的无明痴暗。

### 定解的两个维度

1994年版特别提出，定解分为**名言定解**与**胜义定解**两个维度：

- **名言定解**：对因果、三宝、轮回等世俗法确信不疑。"对三宝的信心及对因果的取舍有很大作用"——无论他人如何动摇，自己始终不变。
- **胜义定解**：证悟法的本性，"非常清明，任何人作违缘也不可能动摇"。

此二分法贯穿全论：七问中的名言问题（如Q6共同所见）对应名言定解，胜义问题（如Q1无遮非遮）对应胜义定解。

### 真实定解 vs 相似定解

> **于基道果一切法，生起真实之定解，听闻生信此二者，犹如道与彼影像。**

| | 真实定解 | 相似定解 |
|---|---------|---------|
| 来源 | 自己通过闻思修深入抉择后生起 | 听他人说而生起 |
| 稳固性 | 坚固如金刚，无人可夺 | 容易随外缘退转 |
| 比喻 | 真实的道路——能行至目的地 | 地图上的影像——起不到真正作用 |
| 举例 | 认识上师多年，了知其功德后生起的信心 | 听说某上师了不起，便前往朝拜的信心 |

2003年版强调：利根者容易生起真实定解（凭自智抉择后生信），钝根者容易生起相似定解（随他人语言而转）。修行者应当"凭借自己的智慧获得定解"。

## 定解的双柱来源：因明与中观

> **稀有法称月称尊，善说日光同现于，佛教广阔虚空中，摧毁疑惑重重暗。**

麦彭仁波切将定解追溯到两位论师的善说，建立了"因明＋中观"的双柱框架：

| | 法称论师 | 月称论师 |
|---|---------|---------|
| 方向 | 名言/世俗谛 | 胜义谛 |
| 核心论典 | 《释量论》《因明七论》 | 《入中论》《显句论》 |
| 量的类型 | 观现世量（现量＋比量） | 胜义量 |
| 作用 | 无误取舍名言法；成立本师为量士夫 | 抉择万法本来实相——离戏空性 |
| 比喻 | 如日光——照明名言 | 如月光——显现胜义 |

此双柱框架是全论的认识论基础。七问中的每一问都在这两种量的张力中展开：

- Q1（无遮非遮）、Q2（声缘二无我）→ 偏重胜义量（中观）
- Q6（共同所见）→ 偏重名言量（因明）中的净见量扩展
- Q3-Q5 → 两种量交互运用

2003年版引《中观宝灯论》颂说："乘骑二谛之骏马，睁开二量之双目，挥舞精进之长鞭，奔至佛果之城邑。"——修行者必须同时具足二种智慧方能入道。

> **相关知识库**：解义慧剑的四种理（观待理、作用理、证成理、法尔理）与此处的"四种理"完全一致（见 `collections/解义慧剑/结构总览.md`）。定解宝灯论 L2 明确提到这四种理作为回答七问的方法论基础。

## 仙人与流浪者——论的戏剧框架

麦彭仁波切将自己的智慧分成两个角色：

- **常住仙人**：代表长期闻思修后安住的智慧
- **流浪者**：代表在辩论和质疑中行走的智慧（"分别念"的一种显现）

流浪者向仙人提出七个难题后"消于法界"——喻指分别念融入自然本性。仙人默然不答，如末劫风摇须弥山——**不是真的答不出，而是在示现观察法器**，如释迦牟尼佛于菩提树下证悟后四十九天不说法。

随后仙人祈祷文殊菩萨，获得加持后"心中现如黎明"——答案在智慧中自然显现。此处1994年版用"诚恳祈祷"，2003年版更强调这是文殊智慧于麦彭仁波切相续中的"自然流露"。

### 回答的方法论承诺

流浪者要求仙人：

1. **教证不违**：不违背显宗经论与密宗续部
2. **理证成立**：以事势理成立——现量、比量、四种理（观待理、作用理、证成理、法尔理）
3. **快如闪电**：不拖延犹疑，展现真正智慧而非寻思辩论

特别提到寻思者"以荆棘矛百般破立亦不解"——仅靠词句辩论的人无法真正进入这些问题的核心，因为七问的答案都涉及甚深空性的亲证。

## 七问总纲

> **如是七种难解题：见解无遮或非遮？声缘证二无我耶？入定有无执著相？观察修或安置修？二谛何者为主要？异境何为共所见？中观有无承认否？**

### 文殊六字对应

七问以文殊心咒"嗡·阿·绕·巴·匝·那·德"后六字统摄：

| 字 | 门 | 对应问题 |
|----|-----|---------|
| 阿 | 无生之法门 | Q1 见解无遮或非遮 |
| 绕 | 远离诸垢门 | Q2 声缘证二无我耶 |
| 巴 | 显现胜义门 | Q3 入定有无执著相 |
| 匝 | 无生无死门 | Q4 观察修或安置修 |
| 那 | 远离名称门 | Q5 二谛何者为主要 |
| 德 | 甚深智慧门 | Q6 + Q7（共用） |

### 见修行果架构

2003 L2 明确指出七问按"见修行果"排列：

```
见 ─┬─ Q1 见解无遮或非遮 → 确立正见（现空双运）
    └─ Q2 声缘证二无我耶 → 遣除见的歧途（勿止于声缘所证）

修 ─┬─ Q3 入定有无执著相 → 入定的本体（最终离四边执著）
    └─ Q4 观察修或安置修 → 入定的方式（初观察·中交替·后安住）

行 ─┬─ Q5 二谛何者为主要 → 见行统一（二谛双运，不偏一方）
    └─ Q6 异境何为共所见 → 认知基础（法界明分为共同所见）

果 ── Q7 中观有无承认否 → 最终立场（实相无承认，现相有承认）
```

### 七问皆不离空性

2003 L2 特别强调，尽管七问涉及修行、量论、续部等不同领域，但"都是基于空性理而发问的"：

- Q1 直接讨论空性的性质（无遮/非遮）
- Q2 讨论不同层次行者所证空性的范围
- Q3 讨论入空性定时的状态
- Q4 讨论达到空性证悟的方式
- Q5 讨论空性（胜义）与显现（世俗）的关系
- Q6 讨论空性视角下的认知基础
- Q7 讨论空性证悟后的最终立场

这意味着全论是一部**空性见的完整修行手册**，而非七个独立的哲学辩论。
