# 渐次次第论

## 方法定义

面对"A还是B"的二元对立命题，麦彭仁波切的回答模式不是选择其中一方，而是建立一套**渐次的修行次第**——初期选A，中期A与B交替，后期选B——从而超越二元对立，将两个看似矛盾的观点整合为一条完整的道路。

此推理模式在定解宝灯论中反复出现，至少在三个问题中系统运用。

## 模式结构

```
他宗甲：始终A（如：始终有执著 / 始终观察 / 始终重胜义）
他宗乙：始终B（如：始终无执著 / 始终安住 / 始终重世俗）
自宗：
  初期 → A为主（对治无始习气，建立基础）
  中期 → A与B交替（定解渐增，交替巩固）
  后期 → B自然成就（定解圆满，无需A的工具）
```

## 具体应用

### 应用一：入定有无执著相（Q3）

| 阶段 | 立场 | 理由 |
|------|------|------|
| 初期 | 有执著（修无我） | 对治实执的工具，"一切初学者无有错谬之入门" |
| 中期 | 断无我执著 | "由修无所有，后亦断空执" |
| 后期 | 离四边无执著 | "远离四边戏，若于四边皆不执，除四边外无执相" |

**论证关键**：初学者"放松"是加重轮回（"众生平庸过放松，流转三界轮回中"），但永远有执著则永远无法超越分别念。渐次次第化解了这个矛盾。

### 应用二：观察修或安置修（Q4）

| 阶段 | 立场 | 理由 |
|------|------|------|
| 初期 | 观察修为主 | "若未以妙观察引，岂能生起妙定解"——六重连锁推导 |
| 中期 | 观察安住交替 | "定解以及增益心，二者相互矛盾故"——需反复以观察驱除增益 |
| 后期 | 自然安住 | "若知绳子无有蛇……仍说无蛇复观察，难道不是愚笨"——如蚕作茧自缚 |

**论证关键**：以绳蛇喻说明后期不需观察的合理性；以蚕茧喻说明后期继续观察的过失。

### 应用三：二谛何者为主要（Q5）

| 阶段 | 立场 | 理由 |
|------|------|------|
| 初期 | 区分二谛（轮番认识） | "何时分别各执著，无欺缘起之显现，远离所许之空性"——初学者先分开认识空与现 |
| 中期 | 二谛交融 | "一旦无有轮番时"——对现空双运生起定解 |
| 后期 | 二谛双运（等性） | "断除一切执著相，尔时见解即圆满"——现空无二自然现前 |

## 此模式的哲学基础

渐次次第论的深层依据是**中观四步境界**（宁玛派特有框架）：

1. 显现 → 空性（单空）
2. 空性 → 双运
3. 双运 → 离戏
4. 离戏 → 等性

每一步都是前一步的深化，而非否定。初期的"有执著"或"观察修"不是错误，而是通向后期"离戏"的必经之路。

## 判断标准（月亮盈亏喻）

> **是故未加辨别时，偏说有无执著相，均有错误正确分，犹如月形之盈亏。**

此颂同时出现在Q3和Q4的摄义处，说明此模式是有意识的方法论——片面说"始终A"或"始终B"，均有对有错，如月既有盈也有亏。关键不在选A或B，而在**辨别修行者当前处于哪个阶段**。

## 与宗喀巴大师的关系

宗喀巴《三主要道论》："何时分别各执著……尔时未证佛密意。一旦无有轮番时……断除一切执著相，尔时见解即圆满。"——这正是渐次次第论的精确表述。两家在此方法论上完全一致，差异仅在暂时侧重（宗喀巴暂时强调初期的观察/执著阶段，麦彭仁波切在此基础上更充分展开后期的离戏/安住阶段）。
