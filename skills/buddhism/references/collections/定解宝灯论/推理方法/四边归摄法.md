# 四边归摄法

## 方法定义

将对方的执著或见解**归入四边（有、无、亦有亦无、非有非无）之中**，然后指出：既然一切执著都可归于四边，而中观正见是远离四边的离戏，则对方的执著终究未超出分别念范畴——无论其自称多么高妙。

此法不是直接破四边（那是Q1中的四边破/金刚屑因），而是**将对方不自觉地归入四边**以揭示其未超越戏论的本质。

## 逻辑结构

```
对方声称：我的见解/修行已超越戏论
分析：对方心中有X概念作为所缘
X必然是以下之一：
  ① 有（执存在）
  ② 无（执不存在）
  ③ 亦有亦无（二俱）
  ④ 非有非无（二非）
∴ X仍在四边之内 → 对方未超越戏论
```

## 具体应用

### 应用一：破"非有非无心即光明法身"（Q3）

> **若详观察汝观点，既不敢说存在有，亦不能说不存在，实则有无此二边，抑或非有非无边，无论如何不超此。**

对方声称：心既非有又非无，此即光明法身。

归摄分析：
- "不敢说有"→ 心中明确排除了"有"
- "不能说无"→ 心中明确排除了"无"
- 排除有和无之后剩下什么？要么是"亦有亦无"（第三边），要么是"非有非无"（第四边）
- **无论如何都还在四边之内**

> **彼心非有非无者，心中常存此念头，彼与不可思议我，仅名不同义无别。**

更进一步：对方心中始终存着"非有非无"这个**念头**作为执著对象——这本身就是第四边（非有非无边）的执著。与外道承许的"不可思议我"（也是非有非无、超越言思的自我）**仅名不同义无别**。

### 应用二：靶子喻——揭示隐藏的所缘

> **汝者所谓离是非，如同靶子住心前。**

对方口说"离是非"（离戏），但"离是非"这三个字本身成了心中的执著对象——如同靶子摆在心前。只要心中有一个概念作为所缘（哪怕这个概念叫"无所缘"），就仍在四边之内。

### 应用三：自宗的"真正离四边"

> **如是远离四边戏，若于四边皆不执，除四边外无执相，故许无有执著相。**

自宗的离四边与对方的区别：

| | 对方的"离戏" | 自宗的离四边 |
|--|------------|-----------|
| 心中 | 有"离戏"的概念作为所缘 | 无任何概念作为所缘 |
| 实质 | 第四边（非有非无）的执著 | 超越四边的圣者智慧 |
| 体验 | 分别念中的"非有非无"体验 | 明空无二，如目视虚空 |
| 断证 | 烦恼不减，对因果信心可能退失 | 断证功德持续增上 |

**关键区分**：自宗离四边不是"第五个选项"，而是**四边的穷尽导致的自然灭尽**——有无二俱二非四种可能性全部排除后，不剩任何执著相。

## 与中论四边破的关系

中论的四边破（金刚屑因变体）是从本体论角度直接破除四边——诸法不自生、不他生、不共生、不无因生。

定解宝灯论的四边归摄法是从认识论角度运用——不是证明法的本体远离四边（这已由Q1建立），而是**检验修行者的心相续是否真正远离四边**。这是从"见解"到"修行"的转化。

## 使用要点

1. 此法的目的不是破四边本身（那是其他推理的工作），而是**揭示对方自以为超越戏论而实未超越**
2. 关键在于找到对方心中的**隐藏所缘**——即使对方说"无所缘"，"无所缘"本身也可能成为所缘
3. 外道"不可思议我"是最有力的类比——对方一旦被归入"与外道不可思议我等同"，其修法的问题就显而易见
4. 自宗的离四边不是概念上的"排除四边"，而是**断证功德增上**的实际修行状态——以断证推知（烟→火）来判断
