# 太过归谬法

## 方法定义

不直接建立自宗，而是**从对方的前提出发，推出对方自己也不能接受的结论**（太过/absurd consequence），从而迫使对方放弃原始立场。这是中观应成派的标志性论证方法。

定解宝灯论中大量使用此法，特别是在破斥他宗观点时。

## 逻辑结构

```
对方前提P → 推出结论C
C与对方另一承许Q矛盾（或C本身荒谬）
∴ 前提P不成立
```

常见变体：
- **教证相违**：P → C违反对方自承认的经典
- **理证相违**：P → C违反逻辑
- **自许相违**：P → C违反对方自己的其他立场

## 具体应用

### 应用一：三种相违——破"柱以柱不空"（Q1）

> **呜呼彼不以彼空，空基不空已留剩，与色以色空教义，以及理证皆相违。**

对方前提：柱子以柱子不空（以成实空）。

| 太过类型 | 推导 | 过失 |
|---------|------|------|
| **教证相违** | →《般若经》说"色以色空，眼以眼空"；若柱以柱不空则与此直接矛盾 | 违反对方也承认的佛经 |
| **理证相违** | → 以离一多因等五大因观察，无一法不以自体空 | 违反对方也使用的胜义理证 |
| **自许相违** | → 对方一面承许万法皆空，一面说柱以柱不空 | 自语矛盾 |

### 应用二："无所断"过——破"三乘见道同"（Q2）

> **则于前入小乘者，获得大乘见道时，毫无所断等过失，为理所害不可驳。**

对方前提：声缘已圆满证悟二无我，三乘见道智慧相同。

推导链：
1. 声缘已断烦恼障（阿罗汉位）→ ✓ 对方承认
2. 声缘已圆证二无我 → 对方前提
3. 所知障从八地才断 → 对方承认（格鲁派定义）
4. ∴ 回小向大者从一地到七地**毫无所断**——烦恼障已断，所知障还没到断的时候
5. 中间一个阿僧祇劫做什么？→ **荒谬**

萨迦派香穹巴为此发出"十八大包袱"，果仁巴指出六十四个过失。

### 应用三："日升遣暗"之荒谬（Q2）

> **未证与许证相违，证违未断诸所断，日虽升起遣黑暗，观待他缘真稀奇！**

对方辩解：虽已证悟，但断障仍需积累资粮作助缘。

太过：智慧（太阳）已具，所断（黑暗）应自消。说"太阳已升起却还需灯光才能遣除黑暗"——**任何人都不会接受此结论**。

### 应用四：圣者入定成灭法因——破偏空（Q5）

> **汝许见空修彼道，证悟唯一单空界，圣者入定空性智，成灭法因亦有过。**

对方前提：胜义谛（空性）为主要，世俗显现是所断。

太过：若修单空"消灭"显现 → 圣者入根本慧定时一切显现法被消灭 → **圣者入定智慧成为毁灭一切法之因** → 这正是中观应成派对自续派的"三大太过"之一。

### 应用五：若五蕴全证则无实法未证（Q2·破果仁巴）

> **若已证悟五蕴空，则除无为法之外，有何未证他实法？**

对方前提：声缘证悟了自相续的全部五蕴空性（含无分微尘）。

太过：一切有实法皆摄于五蕴 → 除无为法外无实法未证 → 无为法于经部以上亦为假立 → 实质上等于圆满证悟法无我 → **与对方"未证他法"的立场矛盾**。

## 与中论回遮论证的关系

太过归谬法是中观应成派的核心方法，在中论中系统运用（见 `collections/中论/推理方法/回遮论证.md`）。定解宝灯论继承并具体化了这一传统——不是在抽象层面破"自性"，而是针对具体宗派的具体论点进行精确归谬。

## 使用要点

1. **从对方前提出发**，不预设自宗立场——这是应成派与自续派的根本方法论差异
2. 推出的结论必须是**对方自己也不接受的**（否则不构成太过）
3. 三重相违同时发出（教证+理证+自许）时论证力度最大
4. 太过本身不建立自宗——自宗通过"排除一切不合理后的唯一合理选项"间接建立
