## Editorial Policy — REQUIRED reading before answering

This topic carries three structural cautions:

### 1. 三种戒律 (三士道戒) 是层层增上、不相违

宁玛派 / 萨迦班智达《三律仪论》(sdom gsum) 立场：

```
共道           不共大乘             不共密乘
─────          ───────              ─────
别解脱戒  →  菩萨戒 (在别解脱基础上加发心) →  密乘戒 (在菩萨戒基础上加见行)
```

三戒之间的关系**不是相违取舍，而是层层涵盖**。受密乘戒者亦持菩萨戒、亦持别解脱戒；后受不能舍前受。回答时**绝不可说「菩萨戒高于别解脱戒所以可以不持」**或「密乘戒可以舍菩萨戒」— 这是宗派外道的颠倒说，宁玛/萨迦/格鲁三派共破。

详见 萨迦班智达《三律仪论》(本知识库尚未处理) 与 宗喀巴《菩萨戒品释》。

### 2. 戒体 (śīla-svabhāva) 的传统差异

| 传统 | 戒体观 |
|---|---|
| **有部俱舍** | 无表色 — 实有的色法，由受戒仪式生起，于心中相续 |
| **经部** | 种子习气说 — 否认无表色为实体，戒体是心相续中的种子功能 |
| **唯识** | 第八识中的种子 — 与经部接近但更系统化 |
| **大乘菩萨戒** | 以菩提心愿力为戒体 — "若失菩提心，菩萨戒即破" |
| **密乘三昧耶** | 以见行无离无合为体 — 破誓言即破证悟根基 |

回答「戒体是什么」时**必须问语境**。默认大乘语境用菩提心戒体；默认俱舍 (有部) 语境用无表色。

详见 `collections/俱舍论/品4-分别业品概览.md` (有部立场) + `collections/入行论/品02-菩萨戒总义与供养顶礼.md` (大乘立场)。

### 3. 性罪 vs 遮罪

- **性罪** (prakṛti-sāvadya)：本性即是罪 — 杀、盗、淫、大妄语等。无论受戒与否、对象身份如何，造作即得罪。佛制戒只是显示其罪性。
- **遮罪** (pratikṣepaṇa-sāvadya)：佛因度生需要而制 — 受戒者犯则得罪，未受戒者不得罪。如比丘不得手挖土地、不得过中食。

回答「破戒会怎样」时须辨明：性罪须四力忏悔且果报极重；遮罪只对受戒者有约束，舍戒后即无。

详见 `collections/俱舍论/品4-分别业品概览.md`。

## Concept Map

### 戒律的法相基础 (来自 俱舍论)

#### 三戒分类 (品4)

| 戒 | 摄属 | 何识相应 | 何道获得 |
|---|---|---|---|
| **别解脱戒** | 欲界 | 与意识相应 | 受戒仪式获得 |
| **禅定戒 (静虑律仪)** | 色界 | 与定心相应 | 入定时自然伴生 |
| **无漏戒** | 无漏 | 与无漏智相应 | 见道以上自然伴生 |

后二戒不需别授戒仪式，是定/慧伴生的功德。

#### 业道 / 十善十不善

**十不善业道**：
- 身三：杀生 / 偷盗 / 邪淫
- 语四：妄语 / 离间语 / 恶语 / 绮语
- 意三：贪欲 / 嗔恚 / 邪见

**十善业道**：上述十种的"不"作 + 其反面之善行。

四要素分析每业之轻重：基 (对象)、意乐、加行、究竟 (`collections/大圆满前行/因果不虚·十不善业与果报.md`)。

#### 五无间罪

杀父 / 杀母 / 杀阿罗汉 / 出佛身血 / 破和合僧 — 五者皆下无间地狱无中阴。

详见 `collections/俱舍论/品4-分别业品概览.md`。

### 菩萨戒 (来自 入行论)

#### 三聚净戒 (大乘戒律框架)

| 净戒 | 内容 |
|---|---|
| **摄律仪戒** | 摄持别解脱戒 — 防非止恶 |
| **摄善法戒** | 修行六度万行 — 积聚善根 |
| **饶益有情戒** | 以四摄法度众生 — 利他事业 |

#### 四对治力 (大乘还净法)

破菩萨戒后的还净方法：
1. **依止力** — 皈依三宝、发菩提心
2. **厌患力** — 真实悔过
3. **遮止力** — 誓不再犯
4. **现行对治力** — 修空性 / 念诵心咒 / 礼拜 / 供曼茶 / 等

### 密乘戒 / 三昧耶 (来自 大圆满前行 + 定解宝灯论)

#### 五部三昧耶 (大圆满外续部分类)

具体内容因传承不同而有别。共许的核心：
- **不舍上师** — 视上师为佛
- **不舍法门** — 不弃所修密法
- **不舍誓言尊** — 守护本尊观修
- **不与异道行人交往** (依传承程度)
- **如法持咒、修法**

阿底约嘎层面的三昧耶简化为**「无所修」**为最深三昧耶 — 一切相皆是法身游舞，无可舍亦无可取。但此境界需以前阶三昧耶圆满为基础，**不是「不持戒」**。

## Correctness Anchors

### 三戒不相违 — 后受不舍前受

常见误解：受了菩萨戒就可以不持别解脱戒；受了密乘戒就可以放弃菩萨戒。

正确立场：宁玛 / 萨迦 / 格鲁三派共许 — 三戒**层层涵盖, 后摄前**, 不可舍前受。这是萨班《三律仪论》立基命题，亦是宗喀巴《菩萨戒品释》明文。

任何「越上越自由 / 上层戒可以舍下层戒」的说法即是颠倒外道。

### 菩萨戒以菩提心为戒体

常见误解：菩萨戒是18根本戒+46支分戒等条文清单，背熟即持戒。

正确立场：菩萨戒的**根本是菩提心**。"若失菩提心,菩萨戒即破" — 失菩提心则一切戒条形同虚设；菩提心未失则一切方便皆为菩萨行。条文是菩提心的具体施设，不是戒的根本。

详见 `collections/入行论/品04-不舍菩提心与谨慎暇满.md`。

### 性罪须忏悔, 遮罪舍戒即无

常见误解：所有犯戒都须严格忏悔。

正确立场：
- **性罪**（杀盗淫妄）— 自性即罪，无论受戒与否、修行与否，造作必招果报。须**四对治力 + 修空性**真实净化。
- **遮罪**（如比丘不持金钱、不挖地等）— 佛因度生制定，仅约束受戒者。舍戒后即无遮罪。

回答 「破了一条戒会怎样」时须先辨明性罪还是遮罪。

### 大圆满「无所修」≠「不持戒」

常见误解 (尤其见于大圆满外学者)：阿底约嘎说「无所修」、「自然解脱」 — 那就可以不持戒了。

正确立场：「无所修」是**三昧耶圆熟之果**, 不是修学起点。前阶 (别解脱+菩萨+下密三戒) 必须圆满, 才进入「无所修」的法身境界。任何以「我修大圆满,不持戒」为辞者, 既非大圆满, 亦非戒律。

参见 `topics/tantra/index.md` §Editorial Policy。

### 戒体的传统差异不可粗糙等同

回答 「戒体是什么」时, 须辨语境：
- 俱舍语境 → 无表色 (有部) 或 种子 (经部)
- 大乘语境 → 菩提心
- 密乘语境 → 见行无二 / 三昧耶

混同三者会产生大错 (如「无表色就是菩提心」 — 错；「菩提心就是三昧耶」 — 部分对但层次不同)。
