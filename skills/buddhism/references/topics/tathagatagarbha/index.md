# Tathāgatagarbha (如来藏) — Topic Index

This topic covers 如来藏 (buddha-nature / tathāgatagarbha / 佛性) teachings — the "third turning of the wheel" doctrine asserting that all sentient beings possess the seed of buddhahood. The foundational text is 弥勒菩萨《究竟一乘宝性论》(Uttaratantra / Ratnagotravibhāga).

Read this after the root SKILL.md to orient yourself in this knowledge space. **Read §Editorial Policy below before drafting any answer** — this topic carries live doctrinal tension that the other topics resolve differently.

## Editorial Policy — REQUIRED reading before answering

This topic sits at a live doctrinal fault-line. **The collection wiki preserves two interpretive layers you must not flatten:**

| Layer | Source | Position | Use |
|---|---|---|---|
| **朵洛瓦 (觉囊派)** | 朵洛瓦《大乘宝性论释》(the commentary 索达吉堪布 translates and bases his teaching on) | **义他空** — 能取所取空, 如来藏**不空**. 如来藏 is a positive reality (周遍常恒, 具足功德) distinct from 自空中观's 离戏空性. Third-turning is the definitive teaching; second-turning is provisional. | Always preserve when quoting the commentary. Tag explicitly as "朵洛瓦/觉囊派说" or "依朵洛瓦注释". |
| **索达吉堪布 (宁玛派视角重讲)** | Oral teaching transcripts (课程103) | Presents 朵洛瓦's system faithfully at each step, but interpretively reconciles it with 麦彭仁波切's 宁玛 position: 如来藏 as **离戏大双运 / 自性光明大圆满**, not a 不空 substance. Second and third turnings point at the same 实相 from different angles (空分 vs 光明分). | Tag as "索达吉堪布解释说" or "宁玛派通达方式" when the oral teaching reinterprets a 觉囊 reading. |

**The Nyingma correctness anchor** (麦彭仁波切《如来藏大纲要狮吼论》):

> 凡是承许无变之法界为成佛种性，首先需要认识所谓的法界是于何施设之基——真胜义二谛大双运极为不住的中观义。

> 以胜义理论观察诸法无实之后得出一个成实之法，犹如由光明产生黑暗，必无是处。

Read together: 如来藏 is the 施设基 of 大双运, not a positive substance beyond 自空; any reading that makes 如来藏 a 成实法 is 义他空 and must be rejected.

**What this means for answers:**

1. When quoting 宝性论 / 朵洛瓦 directly, preserve the 觉囊 language ("如来藏不空"/"常恒不变"/"功德本具"). Do not silently rewrite it into 自空 vocabulary.
2. Then — when the question touches interpretive stakes (是否究竟？与自空矛盾吗？) — add the 宁玛 reconciliation: the 不空如来藏 that Nyingma accepts is 离戏大双运 / 自性光明, not 义他空的实有法性.
3. Cross-reference `topics/madhyamaka/classifications.md §1 自空/他空` and `topics/madhyamaka/debates.md §1 第九诤辩` so the reader sees both topics committing to one coherent position.
4. If the user is asking from a 觉囊 framework with no cross-tradition comparison, answer within that framework — don't force Nyingma critique where it wasn't invited.

## The text's own intellectual architecture

《宝性论》's argument is structured around **七金刚处 (seven vajra points)** — seven topics that are 坚固不坏, rich with meaning, difficult to penetrate:

```
佛宝 ── 三宝          ← 所得果 (goal of the path)
法宝 ──  (abhidharma)
僧宝 ──

如来藏 ── 四处         ← 因果安立 (cause-and-fruit structure)
菩提  ──  (法界/转依/
功德  ──   二身三身/
事业  ──   任运事业)
```

The three-ratna are the **goal**; the four-vajra unpack **how that goal is reached**. Specifically:

- **如来藏** (有垢真如) = the *cause* — buddha-nature present in every sentient being, obscured by 客尘
- **菩提** (无垢真如) = the *fruit* — buddha-nature when 客尘 is fully removed. 转依 moment.
- **功德** = the *attributes* of the fruit — 64 merits (32 离系 + 32 异熟)
- **事业** = the *activity* that the fruit manifests for others

This structure is the text's core claim: 如来藏 is not a metaphysical speculation — it is the necessary cause-side of the logic that buddhahood is attainable. **Remove 如来藏 → buddhahood becomes 无因生.** The 九喻 (nine analogies) are the text's pedagogical move to make this cause-nature comprehensible.

Commentary chapters (朵洛瓦) map to these as:
- **总说品** — 七金刚处 framework + 三宝本体
- **第一如来藏品** — 四处之首 (cause)
- **第二菩提品** — 四处之二 (fruit)
- **第三功德品** — 四处之三 (attributes)
- **第四事业品** — 四处之四 (activity)
- **第五利益品** — why this text's teaching is supreme

See `collections/宝性论/结构总览.md` for the full intellectual map and lesson mapping.

## Correctness Anchors

These positions are where the tradition differs from common or training-data views. When your knowledge conflicts with these, your knowledge is wrong.

### 如来藏不是外道的"我" (神我 / ātman)

常见误解：如来藏 = 常恒不变、独立存在 → 与印度教梵我不二论相同。

正确立场：《宝性论》区分如来藏与外道我的关键是"**四果**"对应"**四颠倒之对治**"：
- 凡夫颠倒"无常/苦/无我/不净"→执无常法为常、苦为乐、无我为我、不净为净
- 声缘颠倒"常/乐/我/净"→偏执空寂的对治（仅遮颠倒而未安立如来藏之真四果）
- **大乘正见**：如来藏具清净/自我/乐/常四德

这里的"我"是**大我/真我/胜义自在**，以无我的空性智慧为其本性（非五蕴聚合上假立之我）。《大般涅槃经》："我者即是如来藏义。一切众生悉有佛性，即是我义。如是我义，从本已来常为无量烦恼所覆，是故众生不能得见。"

此"我"必须以**经过二无我抉择**为前提——先以《中观》破尽五蕴之我与法我，所余之**离戏自性光明**方名如来藏"真我"。**绕过自空中观而直说"我"即外道**。麦彭仁波切："凡是承许无变之法界为成佛种性，首先需要认识所谓的法界是于何施设之基——真胜义二谛大双运极为不住的中观义。"

### 如来藏不是阿赖耶识

常见误解（尤其受唯识框架影响者）：如来藏 ≈ 阿赖耶识 / 染净种子依。

正确立场：阿赖耶识属世俗名言法（瑜伽行中观在名言中承许）；如来藏是胜义法性（离戏自性光明）。前者生灭，后者不生不灭。《胜鬘经》："如来藏者，非我，非众生，非命，非人……如来藏者，是法界藏，是法身藏，是出世间上上藏，是自性清净藏。"法界藏与识藏（阿赖耶）判然有别。

具体对应关系：见 `topics/madhyamaka/yogachara-madhyamaka.md §阿赖耶`。

### 如来藏以自空中观为前行

常见误解：学了如来藏就可以超越空性；如来藏是比空性更高的见解。

正确立场：麦彭仁波切《如来藏大纲要狮吼论》——先通达自空中观（第二转），再入如来藏（第三转）。跳过自空直说如来藏，即成义他空过失（保留空基不空，与《般若经》"色以色空"直接相违）。

《宝性论》自身亦如此构造：必先安立三宝，再转入四处（如来藏为四处之首）。三宝的胜义皈依处即佛宝——是已通达空性之果位功德。

### 不空如来藏 = 离戏大双运

常见误解（义他空倾向）：如来藏是实有法性，为空性不能遮破之实法。

正确立场：如来藏所说"不空"，是"不空于功德"——具足如海佛不共法（十力/四无畏/十八不共等），而非具足一个实有本体。自性光明与空性无二无别，此即大双运，即大中观。故不空如来藏 = 自空中观本身所指，只不过从光明分侧面安立。

参见：`topics/madhyamaka/classifications.md §他空中观`（"就无为光明显现分的侧面抉择，由此成为连结显密之津梁的中观"）。

### 第二转与第三转所诠实相无二

常见误解（受朵洛瓦强他空立场影响）：第三转胜第二转，二者所诠不同。

正确立场（宁玛/麦彭/萨迦共许）：**二转法轮所诠实相无二无别**，差别在诠说侧面。
- 第二转（《般若经》《中论》）：从**空分**侧面诠实相 → 离戏自空
- 第三转（《如来藏经》《宝性论》）：从**光明分**侧面诠实相 → 自性光明如来藏

萨迦班智达、麦彭、无垢光皆许此立场。朵洛瓦《宝性论释》倾向**以第三转为了义，第二转为不了义**——此立场索达吉堪布在讲解中以宁玛视角作柔和调整（保留朵洛瓦原义，同时以麦彭立场解释二转无二）。

### 三身（法身/报身/化身）显密安立的层次差别

常见误解：显宗三身与密宗三身完全相同概念。

正确立场：显宗《宝性论》所说三身侧重**断证功德**（法身 = 断证圆满，报身 = 受用圆满于十地菩萨前，化身 = 随化现于众生前）。密宗三身（尤大圆满）侧重**心性本体**（法身 = 本体空性，报身 = 自性光明，化身 = 大悲周遍）。名同而安立角度有层次差别——显密可贯通，但不可粗糙等同。详参 `collections/宝性论/09-菩提-三身恒常.md §1.7`（此 anchor 的 locus classicus）。

### 佛外无涅槃（究竟唯一乘）

常见误解：声闻/缘觉有各自的究竟涅槃，不回大乘亦可。

正确立场：《宝性论》明示"**除佛以外没有其他涅槃**"——一切众生终当成佛。此为**一乘究竟立场**。声缘涅槃（无余涅槃）非究竟——终须重起菩提心回入大乘成佛。参见 `collections/宝性论/05-如来藏-恒常无变与功德无别.md §庚八 日轮喻`；法王如意宝圆寂即在宣此颂时，有深密意。

## Interpretation Pitfalls

When constructing answers, watch for:

- **觉囊派义他空混入宁玛立场**：宝性论原文语句若不作辨别，极易滑入义他空。须以§Editorial Policy中双层立场辨明。
- **"我"混同外道神我**：解释"大我四果"时，若未先提二无我抉择，读者易误以为如来藏是印度教式梵我。应在安立"大我"前明确已通过自空中观。
- **"不空"理解为实有**：朵洛瓦常说"如来藏不空"，若不补宁玛补遗（不空于功德而非不空于自性/本体离戏），答案即义他空。
- **断见与常见两端**：如来藏教设置本为防学空者堕断见，但若过度强调常恒不变，又堕常见。须以"空性不离光明，光明不离空性"的双运立场平衡。
- **九喻取象过实**：九种比喻意在说明众生虽具佛性却被垢染所覆，不是说众生心中有一尊小佛像。喻义对应不可僵硬。
- **三身显密粗糙等同**：见上条 correctness anchor。
- **跳过自空直讲如来藏**：宁玛次第严禁。如来藏教必以般若空性为基础。

## Cross-Topic Connections

### → madhyamaka (中观)

最深的连接。《宝性论》自身即 madhyamaka 所说"**他空中观**"的代表论典（见 `topics/madhyamaka/classifications.md §他空中观`）。跨 topic 必读：

- `topics/madhyamaka/classifications.md §1 自空/他空` — 定位《宝性论》的教判位置，及宁玛对句他空/义他空的精细辨析
- `topics/madhyamaka/debates.md §1 第九诤辩` — "如来藏不空"这一诤辩在中观框架下的裁定（依《如来藏大纲要狮吼论》）
- `collections/定解宝灯论/Q1-见解无遮或非遮.md` — 麦彭对句他空/义他空的 locus classicus

跨 topic 路由原则：**哲学诤辩（自空/他空是否相合？如来藏是否究竟？）归 madhyamaka 回答；如来藏的具体内涵（十义/九喻/三身/功德）归 tathagatagarbha 回答**。回答交叉问题时，应同时引用两个 topic 的相关 doc。

### → yogachara (唯识/瑜伽行) [强 — 弥勒五论兄弟]

**弥勒五论内部兄弟关系**. 如来藏 与 唯识 同属 弥勒菩萨五论 体系——四部偏 唯识 (大乘经庄严论 / 辩中边论 / 辨法法性论 / 现观庄严论) + 一部偏 如来藏 (宝性论). 核心 对 应:

- **种姓 ↔ 如来藏** (因位 两种开显): 本论 种姓品 从 "谁 能入 大乘道" 的 唯识式 角度; 宝性论 如来藏品 从 "一切众生 本具 佛性" 的 如来藏式 角度. **同源但非同义** — 可 互 表 里 却 不可 简单 画等号. 详见 `collections/大乘经庄严论/04-种姓品.md` + `collections/宝性论/03-如来藏-总纲十义.md`.
- **三身恒常 ↔ 三身四智** (果位 两种 安立): 宝性论 从 如来藏 光明分 立 三身恒常 (法身 3 理由 + 色身 7 理由); 《大乘经庄严论》 从 唯识转依 立 三身 (三平等 + 三恒常). 详见 `collections/大乘经庄严论/11-菩提品-三身四智.md` + `collections/宝性论/09-菩提-三身恒常.md`.

交叉 topic 路由: 涉及弥勒五论整体架构 / 种姓/如来藏 因位辩 / 三身显教对比 的问题, 同时引两个 topic 的 index. 详细立场协调见 `topics/yogachara/index.md §Relationship with tathagatagarbha`.

### → abhidharma (俱舍/对法) [轻 — 功德对应]

《宝性论》功德品的 **64 功德** (32 离系 + 32 异熟 = 十力 + 四无畏 + 十八不共法 + 32 相) 与 《俱舍论》品7 "分别智品" 的 **佛不共功德** (十力 / 四无畏 / 三念住 / 大悲 等) 在内容上直接对应——两框架以不同视角 (如来藏的 "本具显发" vs 阿毗达磨的 "果位法相") 对同一组佛功德作系统化整理。交叉查询时, 宜兼读二处。

- `topics/abhidharma/index.md §Cross-Topic Connections` — 反向 cross-topic 指引
- `collections/俱舍论/品7-分别智品概览.md` — 阿毗达磨的佛不共功德系统
- `collections/宝性论/10-功德品.md` — 如来藏的 64 功德广说

### → foundations (基础/加行) [轻度]

如来藏见对修行的影响：见解提升（我具佛性）→自信（四颠倒之对治）→出离（轮回颠倒之远离）。《大圆满前行》中"皈依"章节讲胜义皈依处即如来藏，与宝性论总说品三宝本体对应。

### → tantra (密法) [强度]

密宗立场中如来藏 = 自性光明大圆满（义同名不同，参《定解宝灯论》Q7）。《宝性论》藏文名"极喇嘛"（无上续），带密续意味——朵洛瓦宗派即以此为显密之津梁。法王如意宝亦认为《宝性论》特别深奥，需先受密法灌顶方可听闻（见课程103第1课）。

### → prajnaparamita (般若) [中]

《维摩诘经》的几处教义是 如来藏 思想的 sutra-level 依据:
- **品1 "佛土常净若此, 为度下劣人故示众恶不净"** — 本净论 的 经教 核心. 与 《宝性论》"如来藏不因轮涅而有增减" 同构. 详见 `collections/维摩诘经/品01-心净国土净.md`.
- **品8 "一切烦恼为如来种" / 烦恼即菩提** — 如来藏 的 关键 经证 之一. 但 须 严守 correctness anchor "烦恼即菩提不是放任烦恼" — 须以密乘本来清净见地诠释. 详见 `collections/维摩诘经/品08-佛道品·非道是佛道.md`.
- **品12 观佛实相 法身 41 门** — 法身 无相 的 经教 依据, 与 《宝性论》 菩提品 三身恒常 对应. 详见 `collections/维摩诘经/品12-见阿閦佛品.md`.
