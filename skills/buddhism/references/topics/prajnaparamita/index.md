# Prajnaparamita (般若) — Topic Index

This topic covers **般若 (perfection of wisdom) sutras and Mahāyāna narrative sutras centered on emptiness, non-duality, and bodhisattva wisdom**. Currently anchored by the full 维摩诘经 wiki (14 品 / 21 docs); will expand as other major sutras (《金刚经》《心经》《大般若经》等) are processed.

## Scope and Rationale

The topic groups together two partially overlapping genres:

1. **般若部 sutras proper** — Heart Sutra (《心经》), Diamond Sutra (《金刚经》), 般若摄颂, 大般若经 etc. Direct exposition of emptiness and perfection of wisdom.
2. **大乘方等 sutras that primarily teach emptiness / non-duality / bodhisattva wisdom through narrative** — 《维摩诘经》, 《法华经》 parts, 《思益梵天所问经》 etc. Technically 方等部 but share 般若's philosophical anchors and are routinely cited alongside.

A single topic handles both because they share (a) a common vocabulary of 空性 / 不二 / 不可得 / 无所得 / 无生法忍; (b) a common audience concern (how the bodhisattva knows and acts); (c) a common routing pattern. When a dedicated 般若部 collection is processed, scope may be revisited — including whether to split narrative 方等 sutras into their own topic.

## Relationship with madhyamaka

般若 is the **sutra source**; madhyamaka is the **treatise tradition** (龙树 / 月称 / 静命 etc.) that systematizes 般若's teaching.

- Questions about **"佛经中如何说空/不二"** → this topic
- Questions about **"中观如何以因理抉择空"** → madhyamaka topic
- Many questions span both; cross-reference generously

Specifically: 《维摩诘经》品9《入不二法门品》provides the 经教 basis for 不二法门, while `collections/中论/` and `collections/中观庄严论/` provide the 理证 systematization. Both together yield complete understanding.

```

──────────                                            ─────

品7 观众生 空观及"如"义                               品11 菩萨行 (秽土行道纲领)

                                                      品14 嘱累 (付嘱流通)
```

**核心命题**: 若以一句话概括, 即是 — **染净不二、生死涅槃不二、烦恼菩提不二、凡圣不二、能所不二；此「不二」之终极表达不是对立统一的话，而是寂默**。

一切品目均服务于此: 品1 "心净国土净" 立**能所不二**之地基；品2-4 以叙事展示一切"住着"(住空/住定/住戒/住慢) 皆为二；品5-9 层层破二；品9 以默然显不二；品10-14 以不二之见回照秽土行道。

## Correctness Anchors

### "心净国土净" 不是主观唯心

**常见误解**："心净国土净" = 西方哲学的 solipsism / subjective idealism (一切由我心决定).

**正确立场**：佛经中的 "心净国土净" 并非否认众生共业所现的共许境 (三千大千世界的丘陵坑坎), 而是 **同一境在不同心的观者前显不同相**. 螺髻梵王见娑婆如他化自在天宫, 舍利弗见其丘陵坑坎 — 不是二人各自造了净/秽土, 而是 **同一国土因观者心的净染而显不同相**. 这是缘起缘生的自然运作, 非个人主观创造.

详见 `collections/维摩诘经/品01-心净国土净.md §六、§七`.

**唯识框架的互补处理**: 《大乘经庄严论》品20 功德品 §心净国土净 以 **唯识立场 (阿赖耶转依)** 处理同一教义——八识转四智后心体清净, 国土随之显净相, 与 维摩诘经 的 般若/不二立场 互补. 两者共许 "外境无实", 差在 能显机制: 唯识 偏 **心识转依** (本论), 般若 偏 **同一境因观者心有高下** (维摩诘经). 详见 `collections/大乘经庄严论/23-功德品-修行之功德.md §心净国土净` + `topics/yogachara/index.md §心净国土净`。

### "观清净心" 不是把不净观为净

**常见误解**：观清净即强行观不净为净.

**正确立场**：**清净观是认识法本即净**, 非改造一个不净之法. 麦彭仁波切《定解宝灯论》"吐物瓶涂香"警示：若未达本来清净之理即观净, 如在装呕吐物的瓶外涂香. 须先从理论 (五蕴即五方佛、五大即五佛母) 与 修证 (大圆满/大手印) 建立"法本来即净"的见解, 观清净心才有真实所依.

与 `topics/tantra/index.md §等净无二` 的 "洗黑炭喻 / 画灯喻 / 太阳外道反证" 三重反证 同源.

### "烦恼即菩提" 不是放任烦恼

**常见误解**：不必断烦恼, 或可纵容贪嗔为 "修行".

**正确立场**：此教义 **不能脱离密乘本来清净的见地来讲**. 显宗如《坛经》有"烦恼即菩提"之说但未详释, 与"烦恼是所断"的通常显宗安立有表面冲突. 唯密乘"本来清净、法尔俱生"的见地才能如理诠释 — 烦恼的本性即空明不二的智慧, 非承许纵欲为道.

**三层判准** (品8 堪布):
- 吉藏 "方便" 判: 菩萨以空性见转烦恼为方便, 未证者强行即堕
- 圣者行三要: 空性定解 + 方便善巧 + 大悲坚固 — 三者 缺一不可
- 戒律次第烦恼三位: 凡夫位 断之 / 菩萨位 转之 / 圣位 不二之

详见 `collections/维摩诘经/品08-佛道品·非道是佛道.md` + `topics/tantra/index.md §Editorial Policy`.

### "不二" 不是 "一"

**常见误解**：不二 = 合二为一, 消除差别.

**正确立场**：**不二 ≠ 一**. 一 是 二 的 对立面, 落 "一" 即 入 "一多" 二分. 真 不二 是 **离 一 离 多 / 离 有 离 无 / 言默 相即**. 维摩诘 默然 即 此 教学 的 现量 化 — 不 是 "不说话" (那 仍 是 "说话/沉默" 的 二), 而是 言 与 默 同时 消亡.

详见 `collections/维摩诘经/品09-入不二法门品·文殊维摩诘与默然.md`.

### "维摩诘居士" 不等同普通在家人

**常见误解**：维摩诘是居士, 故在家修行与出家等同, 出家无必要.

**正确立场**：维摩诘 是 "示现居士相" 的 大菩萨 (妙喜国无动如来座下菩萨 / 古佛金粟如来 再来), 非普通在家人. 本经借此身份展示的是 "菩萨道 本 不 拘 身份" — 但 并 不 贬 低 出家. 序分 即 "八千 大 比丘 众" + "三万二千 菩萨" 并 列 庄严 法会. 本经 指向 在家 与 出家 皆 可 行 的 大乘 菩提, 非 主张 "在家 高于 出家" 或 "出家 可 废".

详见 `collections/维摩诘经/品12-见阿閦佛品.md` (维摩诘来历揭示).

### "秽土修行一世胜净土百千劫" 不是否定净土法门

**常见误解**：本经说秽土修行胜于净土, 所以修净土没有意义.

**正确立场**：此 教义 强调 **大悲 在 逆境 速 增 的 修行 机制**, 非 判 净土 法门 无效. 品12 明 示 维摩诘 从 妙喜国 来, 并 印证 八千 菩萨 愿 生 妙喜国. 发愿 往生 与 秽土行道 并 行 不违 — 皆 为 大乘 方便 门.

详见 `collections/维摩诘经/品10-香积佛品.md` + `品12-见阿閦佛品.md §八千菩萨愿生妙喜国`.
