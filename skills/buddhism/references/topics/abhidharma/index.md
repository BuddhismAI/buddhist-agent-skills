## Editorial Policy — REQUIRED reading before answering

This topic carries one main interpretive caution:

### 有部 (Sarvāstivāda) vs 经部 (Sautrāntika) — 持续的暗线张力

《俱舍论》名义上以**克什米尔有部**理论为基础 ("说此对法我多依，克什米尔有部理")，但世亲论师在关键处以「**许**」字标识**经部**自宗立场。**回答俱舍论问题时不可粗糙说「俱舍说……」— 须辨明是有部立场、经部立场，还是世亲自身的取舍。**

| 论题 | 有部立场 | 经部立场 |
|---|---|---|
| **三世实有** (品5 高潮) | 三世法皆为实体，过去未来法亦实有 | 仅承认现在法实有 |
| **无表色** (品1, 品4) | 大种所造色，真实色法 | 种子/习气说，非实体色法 |
| **得绳 (prāpti)** (品2) | 实有不相应行法，确保业果连接 | 否认实体性，以种子说取代 |
| **极微 (paramāṇu)** | 真实存在的最小实体单位 | 假立 (中观破极微的逻辑由此延伸) |
| **心所法分类** | 大地法/大善地法/大烦恼地法等十类 46 种 (有部精确化) | 较简化的分类 |
| **不相应行** | 14 种实有 | 多为假立 |

这条张力在品5 (分别随眠品·旁述分析三时之理) 达到高潮。世亲论师后期转向唯识 (《唯识二十颂》《三十颂》)，其在《俱舍论》中对经部立场的同情已埋伏笔。

### 与中观的关系：俱舍是中观的所破对象

《中论》《中观庄严论》破「实有」时，所破的具体内容很大程度上即是有部俱舍所安立的法体 (蕴、界、处、心心所、六因五果、不相应行……)。**学俱舍是学中观的必要前行 — 不知道有部立什么，无从理解中观破什么**。

详见 `topics/madhyamaka/reasoning-methods.md` 五大因 (尤其离一多因如何破极微、破心识)。

### 与因明的关系

《俱舍论》的认识论框架 (六识 / 心心所 / 现量 vs 比量的早期分类) 是因明的前身。萨班《量理宝藏论》大量吸收俱舍学说。蒋扬洛德旺波尊者本身即是《量理宝藏论》注释者，其法相分析方法贯穿《俱舍论释》。

详见 `topics/pramana/index.md`。

### 与戒律 (vinaya) 的关系

品4 (分别业品) 的戒律内容直接对应律藏 — 别解脱戒、禅定戒、无漏戒、十善业道、五无间罪。**戒律的法相基础在俱舍，戒律的具体行持在律藏。**

## The text's intellectual architecture — 八品 follows 四谛 logic

```
┌─ 存在论基础 ────────  品1 分别界品   "有哪些法"  → 蕴/界/处分类穷尽万法
│                       品2 分别根品   "法如何运作" → 22根+心所+不相应行+六因五果
│
├─ 苦谛 (轮回果) ─────  品3 分别世间品 "众生在哪里轮回" → 器世界+情世界+十二缘起
│
├─ 集谛 (轮回因) ─────  品4 分别业品   "以何力轮回"   → 业 (karma) 的本体+种类+果
│                       品5 分别随眠品 "为何轮回不止" → 九十八随眠+烦恼异名
│
├─ 道谛 (道) ─────────  品6 分别圣道品 "如何断除烦恼" → 四谛+五道+四向四果+菩提分法
│
└─ 道之工具 ──────────  品7 分别智品   "断烦恼工具是什么" → 十智+佛不共功德
                        品8 分别定品   "智慧在何处运作"  → 四禅八定+解脱胜处遍处
```

「先知苦 (品1-3)，再识集 (品4-5)，后修道证灭 (品6-8)」— 八品本身即是一部完整的解脱论。

## Concept Map

### 法的存在论框架 (品1-2)

#### 五事 (一切法的最大分类)
1. **色法** (rūpa) — 五根+五境+无表色 = 11 种
2. **心法** (citta) — 心王 (1 种, 等同于识)
3. **心所法** (caitasika) — 有部 46 种 (大地法 10, 大善地法 10, 大烦恼地法 6, 大不善地法 2, 小烦恼地法 10, 不定地法 8)
4. **不相应行法** (citta-viprayukta-saṃskāra) — 14 种 (得/非得/同分/无想果/无想定/灭尽定/命根/生住异灭/名身/句身/文身)
5. **无为法** (asaṃskṛta) — 3 种 (虚空/择灭/非择灭)

## Correctness Anchors

### 俱舍 ≠ 究竟见

常见误解：俱舍说三世实有 / 极微实有 / 心所实有 — 那大乘佛教也承认。

正确立场：俱舍 (尤其有部立场) 是**第一转法轮的精细化**，是修学的**所破对象**而非了义见。中观破的「法我」具体内容很大程度上即是俱舍所立的法体。学俱舍是为了：(a) 知道中观破什么，(b) 行持戒律的法相基础，(c) 理解修证次第 (五道四果) 的精确划分。

不可因俱舍立法而否定中观空性，亦不可因中观破法而否定俱舍施设。**前一立场是不解空，后一立场是不解假**。

### 三世实有不是宁玛/大乘正见

常见误解 (受有部框架影响)：过去未来法实有 → 有「时间上的实体」。

正确立场：三世实有是有部独家立场，**经部、唯识、中观皆破**。世亲论师本人在品5以经部立场作详细破斥。

### 无表色不是大乘戒律观

常见误解：戒律以「无表色」为体 — 一种实有的色法。

正确立场：有部以无表色解释戒体的连续性；经部以种子习气说取代；大乘戒律 (尤其菩萨戒) 以「相续断恶修善的菩提心愿力」为戒体。**回答戒律的「戒体是什么」时须辨明立场**。

### 阿罗汉退转的传统差异

有部 (尤其上座部分支) 许阿罗汉退转 (六种阿罗汉中前五种皆可退)。

经部 / 大乘：纯粹的阿罗汉果不退 — 退者非究竟阿罗汉。

回答时须指明立场。详见 `collections/俱舍论/品6-分别圣道品概览.md`。

### 心所数量的传统差异

| 传统 | 心所数 |
|---|---|
| 有部俱舍 | 46 |
| 经部 | 数目较少 (具体不固定) |
| 唯识 (《百法明门论》) | 51 |
| 大乘《大乘阿毗达磨集论》 | 51 |

回答「心所有多少种」时须问语境 — 默认大乘语境用 51, 默认俱舍语境用 46.

### 俱舍诠释依据：蒋扬洛德旺波《俱舍论释》

索达吉堪布讲解依据是宁玛派蒋扬洛德旺波尊者《开显世亲密意灯》— 不是有部毗婆沙派的注释，亦不是真谛 / 玄奘汉译俱舍系的诠释传统。回答时若涉及具体诠释立场，以蒋扬洛德旺波尊者立场为准；与汉传俱舍 (普光、法宝、圆晖三大注疏) 不同处应标注。

## Cross-Topic Connections

### → madhyamaka (中观) [极强]

俱舍是中观的所破对象，中观是俱舍的胜义破立。两者构成「立法相 vs 破自性」的对偶。**学好俱舍是学中观的必要前行。**

- `topics/madhyamaka/reasoning-methods.md` — 五大因如何破俱舍所立的极微、心识、不相应行
- `collections/中观庄严论/13-破外境-粗大与微尘.md` — 离一多因破极微 (针对俱舍极微论)
- `collections/中论/品04-观五阴品.md` — 直接破五蕴自性

### → pramana (量理/因明) [强]

俱舍认识论是因明前身。蒋扬洛德旺波同时是《量理宝藏论》注释者，其法相分析方法贯穿《俱舍论释》。

- `topics/pramana/index.md` §四理 §现量四种 — 现量比量的精细化
- `collections/解义慧剑/证成理·现量四种.md` — 因明的现量四分基于俱舍六识框架

### → vinaya (戒律) [强]

品4 (分别业品) 是律藏的法相基础。

- 别解脱戒 / 禅定戒 / 无漏戒的分类

### → foundations (基础/加行) [中]

俱舍的轮回观、业果观、四谛观是加行 (尤其共同外加行) 的法相依据。
