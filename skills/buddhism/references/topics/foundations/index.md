# Foundations (基础/加行) — Topic Index

This topic covers the foundational Buddhist practices — preliminary contemplations (加行/前行), bodhisattva conduct (菩萨行), and the graduated path (道次第) from renunciation through enlightenment. Read this after the root SKILL.md to orient yourself in the foundations knowledge landscape.

### 大圆满前行 — 修行次第的完整操作手册

这是宁玛派最核心的加行论典，涵盖从闻法心态到临终往生的全部修行次第。堪布索达吉2008年版讲解144课。

**结构**：

```
闻法方式（心态准备）→ 共同外前行（生起出离心）→ 不共内加行（实修积累）→ 往生法（临终保障）
```

知识文档路径：`collections/大圆满前行/`，从 `结构总览.md` 开始。

### 入行论 — 菩萨道的哲学与实践

寂天菩萨的菩萨行论典，十品覆盖从发菩提心到证悟空性的完整大乘修行次第。堪布索达吉2006年版讲解201课，全部处理完毕。29篇知识文档。

**四层递进结构（华智仁波切归纳）**：
- **未生令生（品1-3）**：菩提心功德论证 → 忏罪净化 → 正式受持
- **已生不退（品4-6）**：不放逸 → 正知正念 → 安忍（面对逆缘守护菩提心）
- **不退增上（品7-9）**：精进 → 静虑（自他平等/自他相换） → 智慧（中观空性）
- **善根回向（品10）**：功德回向众生

**入行论独特贡献**：
- 自他平等与自他相换（品8）——入行论对大乘佛教最独特的修行贡献
- 唯识vs中观完整辩论结构（品9）
- 大乘经典成立佛说辩论（品9）
- "入行论四品"（品1-3+品10）为藏传佛教必背内容

知识文档路径：`collections/入行论/`，从 `结构总览.md` 开始。

### 菩萨道体系（入行论维度）

入行论从另一个角度展开菩萨道的完整体系：

```
品01 菩提心利益 → 品02 忏悔罪业 → 品03 受持菩提心 → 品04 不放逸 →
品05 正知正念 → 品06 安忍 → 品07 精进 → 品08 静虑 → 品09 智慧 → 品10 回向
```

全十品已处理完毕（29篇文档）。详见 `collections/入行论/结构总览.md`。

### 关键修行原则

**三殊胜**（贯穿一切修行）：
- 加行发心殊胜：以菩提心摄持
- 正行无缘殊胜：不被违缘毁坏
- 结行回向殊胜：以回向印持善根

详见 `collections/大圆满前行/闻法方式.md`。

**日修框架**（心滴派传统）：
- 晨起→日间四/六座→晚睡，每座含前行（毗卢七法+排垢气+发心祈祷）、正行（观修引导文）、后行（回向）

详见 `collections/大圆满前行/日修闭关要诀.md`。

## Correctness Anchors

### 前行不是"低级"法门

常见误解：前行只是入门级别，高级修行者不需要。

正确立场：华智仁波切、法王如意宝、顶果钦哲仁波切等大成就者终生重视前行——"前行既是前行，也是正行"。法王规定学院每年讲一遍《大圆满前行》。没有前行基础，一切高深修法都如冰上建房。

### 出离心是大乘的基础而非小乘专属

常见误解："出离心是小乘的，大乘直接修菩提心就行。"

正确立场：《地藏十轮经》："不习二乘法，何能学大乘？"《前行》将出离心（共同外前行）置于菩提心（不共内加行）之前。没有出离心的菩提心只是口头禅。

### 因果取舍不因见解高而放松

常见误解："证悟空性后因果不重要了。"

正确立场：莲花生大士黄金法则——"是故见比虚空高，取舍因果较粉细。"见解越高，行为上的因果取舍越要细致。详见 `collections/大圆满前行/因果不虚·业之自性.md`。

### 菩提心不是口头禅

常见误解：念十万遍发心偈就算生起了菩提心。

正确立场："发心不为主，生心乃为主"——关键是内心是否真正生起了利他心。检验标准：看到快乐众生是否无嫉妒而欢喜？看到痛苦众生是否自发流泪？详见 `collections/大圆满前行/发殊胜菩提心·四无量心与发心.md`。

## Cross-Topic Connections

### → madhyamaka (中观)

大圆满前行的智慧度部分（六度之般若）涉及空性见解：八种幻喻、悲智双运的必要性。但这是从修行实践角度讲空性，非哲学论证。因果不虚·业之自性中莲花生大士的"见行平衡"教言对中观修行者极为重要。

入行论品09智慧品是完整的中观哲学论述（5篇文档），涵盖二谛安立、唯识vs中观辩论、人无我与法无我（四念住）、破实执。详见 `collections/入行论/品09-*.md`。

### → tantra (密法)

大圆满前行的不共内加行本质上是密法修行（金刚萨埵观想、供曼茶罗、上师瑜伽、往生法），但以加行的角度呈现。密宗四层皈依（共同乘→不共密乘→金刚藏→究竟实相）在皈依.md中有完整展开。

### → abhidharma (俱舍/对法)

轮回过患中的六道苦（尤其是地狱寿量计算、四大隐没次第）与俱舍论的宇宙论和生死学有直接对应。因果不虚的十不善业果报体系也与俱舍论的业果论对接。
