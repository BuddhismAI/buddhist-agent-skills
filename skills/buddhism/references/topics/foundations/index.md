### 大圆满前行 — 修行次第的完整操作手册

这是宁玛派最核心的加行论典，涵盖从闻法心态到临终往生的全部修行次第。堪布索达吉2008年版讲解144课。

**结构**：

```
闻法方式（心态准备）→ 共同外前行（生起出离心）→ 不共内加行（实修积累）→ 往生法（临终保障）
```

知识文档路径：`collections/大圆满前行/`，从 `结构总览.md` 开始。

### 入行论 — 菩萨道的哲学与实践

寂天菩萨的菩萨行论典，十品覆盖从发菩提心到证悟空性的完整大乘修行次第。堪布索达吉2006年版讲解201课，全部处理完毕。29篇知识文档。

**四层递进结构（华智仁波切归纳）**：
- **未生令生（品1-3）**：菩提心功德论证 → 忏罪净化 → 正式受持
- **已生不退（品4-6）**：不放逸 → 正知正念 → 安忍（面对逆缘守护菩提心）
- **不退增上（品7-9）**：精进 → 静虑（自他平等/自他相换） → 智慧（中观空性）
- **善根回向（品10）**：功德回向众生

**入行论独特贡献**：
- 自他平等与自他相换（品8）——入行论对大乘佛教最独特的修行贡献
- 唯识vs中观完整辩论结构（品9）
- 大乘经典成立佛说辩论（品9）
- "入行论四品"（品1-3+品10）为藏传佛教必背内容

知识文档路径：`collections/入行论/`，从 `结构总览.md` 开始。

### 修心七要 — 菩提心的窍诀化归纳

恰卡瓦·耶喜多杰格西将金洲大师传给阿底峡尊者的菩提心教授归纳为七个要点，加哲无著尊者作《耳传略释》注解。索达吉堪布2005年8天讲解，8篇知识文档。

**七要结构**：

```
前行法 → 正行修菩提心（胜义+世俗） → 恶缘转为道用 → 归结终生五力
   →  修心标准 → 修心誓言（破）+ 修心学处（立）
```

**修心七要独特贡献**：
- **自他相换的窍诀化**——把入行论品08的理证体系浓缩为每日可执行的修法（特别是"彼二乘风息"呼吸配合的tonglen）
- **临终菩提心修法**——狮子卧+大乘死教言五力提供不依赖颇瓦的临终修法
- **终生五力框架**——把菩提心修持系统化为五种势（引发/串习/善种/破斥/发愿）
- **菩提心的验证标准**——四个可测试的修心圆满指标（诸法摄一要/二证取上首/恒当依欢喜/能散即圆满）
- **菩萨戒的窍诀化**——十组誓言（破）+十八组学处（立）的日常行为规范
- **六颠倒的揭示**——忍耐/意乐/品味/悲悯/营求/随喜六种世间人常犯的修行误区

知识文档路径：`collections/修心七要/`，从 `结构总览.md` 开始。

### 佛子行 — 菩提心三大论的压缩手册

藏地无著贤菩萨（1297–1371，观世音菩萨化身，**不是印度 Asanga 无著**）造《佛子行三十七颂》，根桑秋札尊者（法王如意宝前世）著《甘露妙瓶》注疏，索达吉堪布 2022 年翻译并讲授（53课）。14篇知识文档。

**37颂结构**：

```
礼赞立誓 → 前行七支（颂1-7）→ 三士道
  ├── 小士（颂8）
  ├── 中士（颂9）
  └── 大士（颂10-37）
        ├── 意乐发菩提心（颂10）
        ├── 加行·世俗菩提心（颂11-21：入定·自他相换 + 后得·恶缘转道用10颂）
        ├── 加行·胜义菩提心（颂22-24：入定·离戏 + 后得·断实执）
        └── 学处（颂25-37：六度+经中四法+断烦恼+正知念+回向）
```

**佛子行独特贡献**（区别于入行论/修心七要）：

- **恶缘转道用十颂是全论重心**（颂12-21，占37颂的27%）——四不欲（未得利养/痛苦/恶名/诽谤）+ 两难忍（恩将仇报/凌辱）+ 兴衰 + 贪嗔，每颂针对一种具体逆境情境的转化窍诀，比入行论品06的理证与修心七要要03的框架更落地。
- **37颂可整部背诵**——藏传传统早晚课的核心念诵之一，是"菩提心三大论"中最便于日常对照的精要版。
- **胜义菩提心三颂（颂22-24）**——以"一切境现唯心造，心性本来离戏边"一颂收摄整个中观止观，配合"悦意视如彩虹/违缘视为梦幻"两颂作后得修持，是 foundations → madhyamaka 的关键桥梁。
- **经中四法（颂31-34）**——专门防护菩萨行人的"修道堕落四边界"：自观错误 / 不说菩萨过 / 断施主贪 / 杜绝粗语，补充六度未覆盖的防护性学处。
- **噶当六格西反复援引**——注疏密集引用仲敦巴、博朵瓦、金厄瓦、普穹瓦、喀绕巴、夏哦瓦六位格西的修行语录，是藏传实修传统的具象化。

知识文档路径：`collections/佛子行/`，从 `结构总览.md` 开始。

### 二规教言论 — 做人根基的大树十德

麦彭仁波切《二规教言论·积聚天德之宝》，由索达吉堪布译讲（1998/2003/2006 多次宣讲；本 collection 以 2003 版精校本 + 2006 版 12 课为双源）。共 13 篇知识文档。

**核心命题**：`世法即是佛法根`——世间贤善人格不是大乘"预备动作"，而是高深佛法的生长土壤。再殊胜的法注入人格低劣者相续中，如同将雪山狮子乳汁注入陶器，陶器必裂。六祖"佛法在世间，不离世间觉，离世觅菩提，恰如求兔角"为全论纲领。

**大树十德结构**（全论以一棵大树为喻，十种功德缺一不可）：

```
树果：安乐 + 声誉（得果）
  ↑
树花：信心 + 布施
  ↑
树叶：知恩 + 利他
  ↑
树枝：正直 + 誓言坚定
  ↑
树茎：有愧 + 不放逸
  ↑
树根：妙慧（智慧） + 稳重
```

**二规教言论独特贡献**（区别于入行论/修心七要/佛子行）：
- **人格前于菩萨行的明确立论**——本论是 foundations 中唯一以"做人根基"为主题的论典，堪布在 2006 年讲《入行论》期间配讲本论，原因即"《入行论》虽然甚深，但没有人格基础就如金器盛乳汁倒进陶罐会裂开"。其他前行/修心/菩萨行论典都假设"合格人格"已具足，本论补足此假设。
- **护法神与人格的绑定**——独特地将"护法神居堡即自身心"作为全论贯穿线（总论占篇幅四分之一）。身心不净则护法远离、做事不成——这为"修行人为何事事违缘"提供了行为学解释。
- **稳重可胜智慧的命题**（02-稳重.md）——"若具稳重虽无余，亦能圆满百功德，若无世法此根本，则无余法可希冀"。金山寺持律师公案（起初因愚笨被骗去讲经、老实背经几十年终成大法师）是稳重胜智慧的典范。
- **誓言三等（石/土/水）**（06-誓言坚定.md）——龙猛菩萨《亲友书》"石刻文字 / 土中写字 / 水中花纹"三喻被本论转化为日常修行标尺。
- **无愧九相 + 放逸九过**（03-有愧.md + 04-不放逸.md）——对末法时代修行人具体行为的最细致诊断清单，每条都可对号入座。
- **护法神、业力、信心三者不可思议的关联**（09-信心.md）——三宝为唯一不欺的皈依境、信心是福德根本、因果取舍为信心的实际表达——把抽象的"信心"落实为可检验的行为。

**本论在 foundations topic 中的定位**：不在五加行/菩提心/六度任一传统框架内，而是所有这些框架能否发挥作用的"底层土壤"。堪布在 2006 年第 12 课末总结："我这次传讲……主要是看到外面很多居士……基本的做人不太如法，所以很希望大家都懂得这个道理。"

知识文档路径：`collections/二规教言论/`，从 `结构总览.md` 开始。

## Correctness Anchors

### 前行不是"低级"法门

常见误解：前行只是入门级别，高级修行者不需要。

正确立场：华智仁波切、法王如意宝、顶果钦哲仁波切等大成就者终生重视前行——"前行既是前行，也是正行"。法王规定学院每年讲一遍《大圆满前行》。没有前行基础，一切高深修法都如冰上建房。

### 出离心是大乘的基础而非小乘专属

常见误解："出离心是小乘的，大乘直接修菩提心就行。"

正确立场：《地藏十轮经》："不习二乘法，何能学大乘？"《前行》将出离心（共同外前行）置于菩提心（不共内加行）之前。没有出离心的菩提心只是口头禅。

### 因果取舍不因见解高而放松

常见误解："证悟空性后因果不重要了。"

正确立场：莲花生大士黄金法则——"是故见比虚空高，取舍因果较粉细。"见解越高，行为上的因果取舍越要细致。详见 `collections/大圆满前行/因果不虚·业之自性.md`。

### 菩提心不是口头禅

常见误解：念十万遍发心偈就算生起了菩提心。

正确立场："发心不为主，生心乃为主"——关键是内心是否真正生起了利他心。检验标准：看到快乐众生是否无嫉妒而欢喜？看到痛苦众生是否自发流泪？详见 `collections/大圆满前行/发殊胜菩提心·四无量心与发心.md`。

## Cross-Topic Connections

### → madhyamaka (中观)

大圆满前行的智慧度部分（六度之般若）涉及空性见解：八种幻喻、悲智双运的必要性。但这是从修行实践角度讲空性，非哲学论证。因果不虚·业之自性中莲花生大士的"见行平衡"教言对中观修行者极为重要。

入行论品09智慧品是完整的中观哲学论述（5篇文档），涵盖二谛安立、唯识vs中观辩论、人无我与法无我（四念住）、破实执。详见 `collections/入行论/品09-*.md`。

修心七要的胜义菩提心（要02四颂：思诸法如梦/观心性无生/对治亦自解/道体住赖耶）与madhyamaka抉择心性的方法同源；"迷境观四身"（要03）直接把破生住灭与法报化本性身对应，是窍诀化的中观修持。详见 `collections/修心七要/要02-正行修菩提心.md` 与 `要03-恶缘转道用.md`。

### → tantra (密法)

大圆满前行的不共内加行本质上是密法修行（金刚萨埵观想、供曼茶罗、上师瑜伽、往生法），但以加行的角度呈现。密宗四层皈依（共同乘→不共密乘→金刚藏→究竟实相）在皈依.md中有完整展开。

### → abhidharma (俱舍/对法)

轮回过患中的六道苦（尤其是地狱寿量计算、四大隐没次第）与俱舍论的宇宙论和生死学有直接对应。因果不虚的十不善业果报体系也与俱舍论的业果论对接。
