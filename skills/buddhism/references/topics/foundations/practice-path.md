# 修行道次第——跨集综合修行地图

本文综合大圆满前行与入行论两部论典，呈现从初学到深入的完整修行次第。两部论典一纵一横：大圆满前行提供从出离心到密宗实修的完整纵向次第，入行论提供菩提心与六度的横向深度展开。理解二者的关系，是掌握基础/加行修行全貌的关键。

修心七要与佛子行作为"菩提心三大宗经"的窍诀与压缩版，在此修行道的不同阶段提供日用级的对照手册——修心七要侧重终生五力的执行框架，佛子行侧重一颂对应一事的日常查检。详见 `bodhicitta-methods.md` 的五源分工对照表。

## 一、三士道框架

藏传佛教将修行者依发心层次分为三种士夫（三士道）。大圆满前行的结构精确对应此框架：

### 1.1 小士道——追求善趣

**核心转变**：看破今世、追求来世善趣。

修行内容：
- **暇满难得**：认识人身之珍贵——八无暇、十圆满、暇满之比喻与数目差别。不生起"我拥有珍宝"的定解，后面一切修行缺乏驱动力。详见 `collections/大圆满前行/暇满难得.md`。
- **寿命无常**：认识珍宝之易失——外器世界无常、内情众生无常、殊胜正士无常、死缘无定。紧迫感是修行的发动机。详见 `collections/大圆满前行/寿命无常·四观无常.md` 与 `寿命无常·喻义与修行.md`。
- **因果不虚**：认识取舍之机制——十不善业（身三、语四、意三）及其四种果报（异熟果、等流果、增上果、士用果），十善业与业之自性四条（业决定、业增长、未作不遇、已作不失）。因果取舍是小士道的核心行为准则。详见 `collections/大圆满前行/因果不虚·十不善业与果报.md` 与 `因果不虚·业之自性.md`。

小士道的结果：断十不善、行十善，以此获得人天善趣。但这还不是修行的真正目标。

### 1.2 中士道——追求解脱

**核心转变**：看破来世善趣、追求出离轮回。

修行内容：
- **轮回过患**：认识善趣亦苦——地狱八热八寒之苦、饿鬼悬挂饿鬼与隐住饿鬼之苦、旁生之愚痴役使苦、人道八苦（生老病死、怨憎会、爱别离、求不得、五取蕴）、非天之嫉妒争斗苦、天人之死堕苦。认识到即使获得善趣也不过是苦海中暂时的喘息，由此生起真正的出离心。详见 `collections/大圆满前行/轮回过患·总述与三恶趣.md` 与 `轮回过患·三善趣.md`。
- **解脱利益**：认识出路所在——脱离轮回苦海、获得声闻缘觉圆满菩提任一果位。详见 `collections/大圆满前行/依止上师.md`（前半部分"解脱利益"）。

中士道的结果：发起出离心，追求个人解脱。但如《地藏十轮经》云"不习二乘法，何能学大乘"——出离心不是小乘专属，而是大乘的必要基础。

### 1.3 大士道——追求佛果度众生

**核心转变**：不仅追求自己解脱，更发愿度化一切众生。

修行内容：
- **依止上师**：认识独自走不通——观察上师、依止上师、修学上师意行。上师是从中士道升级为大士道的关键枢纽。详见 `collections/大圆满前行/依止上师.md`。
- **菩提心**：将修行动力从个人解脱提升为利益一切众生——四无量心（舍→慈→悲→喜）、愿菩提心与行菩提心、自他平等/自他相换/自轻他重。详见 `collections/大圆满前行/发殊胜菩提心·四无量心与发心.md` 与 `发殊胜菩提心·行菩提心六度.md`。入行论品1-3从另一角度完整展开，详见 `collections/入行论/品01-共同功德与六喻.md`、`品01-愿行二心与发心者赞.md`、`品02-忏罪四对治力.md`、`品03-受持菩提心.md`。
- **六度**：菩萨行的具体实践——布施、持戒、安忍、精进、静虑、智慧。大圆满前行在行菩提心学处中概述六度，入行论品4-9以每品一度（品4-5合为戒律）逐一深入展开。
- **密宗加行**：金刚萨埵百字明忏悔（清障）、供曼茶罗/古萨里（积资）、上师瑜伽（加持融入）、往生法（临终保障）。这是大圆满传承独有的实修体系。详见 `collections/大圆满前行/金刚萨埵.md`、`积累资粮.md`、`上师瑜伽.md`、`往生法.md`。

大士道的结果：发菩提心、修六度、行密宗加行，最终证悟佛果、度化无量众生。

### 1.4 三士道在大圆满前行中的对应

```
┌──────────┬──────────────────────────────────┬──────────────────────┐
│ 士道层次 │ 大圆满前行对应章节                 │ 核心心理转变          │
├──────────┼──────────────────────────────────┼──────────────────────┤
│ 小士道   │ 暇满难得 + 寿命无常 + 因果不虚     │ 看破今世→善恶取舍     │
│ 中士道   │ 轮回过患 + 解脱利益               │ 看破来世善趣→追求解脱 │
│ 大士道   │ 依止上师 + 不共内加行 + 往生法     │ 发菩提心→追求佛果     │
└──────────┴──────────────────────────────────┴──────────────────────┘
```

需要注意：三士道不是三种人的分类，而是同一修行者内心递进的三个阶段。没有小士道的出离基础，中士道的厌离不会真实；没有中士道的出离心，大士道的菩提心只是口头禅。华智仁波切在全论结构中严格贯彻了这个递进逻辑。

## 二、大圆满前行、入行论与修心七要的互补关系

### 2.1 两部论典各自的定位

**大圆满前行**：从出离心到密宗实修的**完整纵向次第**。

```
闻法方式 → 共同外前行（四种厌世心 + 依师） → 不共内加行（五加行） → 往生法
```

这是一部"操作手册"——华智仁波切不掺杂自己的分别念，全部记录如来芽尊者的口传教言，每一段都落实到可观修、可操作的层面。其特色在于次第完备——从怎么听法的心态准备，到临终时的往生保障，中间不漏一个环节。

**入行论**：菩提心与六度的**横向深度展开**。

```
品01 菩提心功德 → 品02 忏悔 → 品03 受持 → 品04-05 戒律 → 品06 安忍 → 品07 精进 → 品08 静虑 → 品09 智慧 → 品10 回向
```

这是一部"深度论证"——寂天菩萨以严密的逻辑论证和精妙的修辞，将菩提心的价值、受持、守护、增上展开到极致深度。全论以"未生令生→已生不退→不退增上→善根回向"四层递进结构运行。详见 `collections/入行论/结构总览.md`。

**修心七要**：菩提心的**极简窍诀归纳**。

```
前行 → 正行修菩提心（胜义+世俗） → 恶缘转为道用 → 归结终生五力（生+死）
   → 修心标准 → 修心誓言（破）+ 修心学处（立）
```

这是一部"窍诀化归纳"——恰卡瓦格西将金洲大师传给阿底峡尊者的菩提心教授提炼为七个要点，加哲无著尊者以注释方式展开。其特色是**极简、可日用、含临终方案**——特别是自他相换浓缩为呼吸配合的tonglen修法（彼二乘风息），以及五力终生框架与狮子卧临终法。详见 `collections/修心七要/结构总览.md`。

**三部论典的定位差异**：

| 维度 | 大圆满前行 | 入行论 | 修心七要 |
|------|-----------|--------|---------|
| 体量 | 10万字级引导文 | 十品诗颂广说 | 几百字颂文+8课讲记 |
| 形式 | 操作手册 | 哲学论证+诗颂 | 窍诀归纳 |
| 重点 | 完整修行次第 | 菩提心与六度 | 菩提心修法窍诀 |
| 临终 | 往生法/颇瓦 | 无独立章节 | 狮子卧+大乘死教言 |
| 加行 | 五加行完整展开 | 仅散布品中 | 仅三句话（暇满/无常/因果） |
| 自他相换 | 完整施受法 | 品08理证+修法 | 彼二乘风息（tonglen） |
| 菩萨戒 | 概述 | 无著vs龙猛两系统 | 十组誓言+十八组学处（窍诀化） |
| 适合对象 | 闭关实修者 | 广大闻思者 | **忙碌者/一切根基** |

**三部互补的时间线建议**：

- 初学——先学修心七要建立菩提心的总体图景（8课即可通览）
- 深化理论——进入入行论广大论证（品1-3建立信心，品8-9深化修持）
- 系统实修——用大圆满前行的完整次第落实每日修行（70课共同前行+五加行）
- 日常抓手——长期以修心七要的彼二乘风息、五力、四标准作为日常检测


### 2.2 重叠交汇之处

两部论典在以下主题上有直接交汇，可以互参互补：

**菩提心/发心**：
- 大圆满前行（第92-114课）：以四无量心为修心基础，以发心偈十万遍为定量修行，侧重"如何修心"——舍→慈→悲→喜的次第、知母→念恩→报恩的层层推进。详见 `collections/大圆满前行/发殊胜菩提心·四无量心与发心.md`。
- 入行论（品01-03）：以五项共同功德+六喻密集论证菩提心价值，以四对治力净化障碍，以七支供+正式仪轨受持菩提心，侧重"为什么发心"的理论说服和"如何受戒"的仪轨方法。详见 `collections/入行论/品01-共同功德与六喻.md`、`品03-受持菩提心.md`。
- **互参建议**：先学入行论品1建立对菩提心的信心与渴求，再学大圆满前行深化四无量心的实修内涵。

**忏悔/四对治力**：
- 大圆满前行（第115-118课）：以金刚萨埵本尊观想为忏悔方法——头顶观想金刚萨埵、甘露降注净化罪障、念诵百字明十万八千遍。这是密宗观想层面的忏悔实修。详见 `collections/大圆满前行/金刚萨埵.md`。
- 入行论（品02）：以四对治力为忏悔框架——厌患对治力（后悔）、所依对治力（皈依+菩提心）、现行对治力（念修善法）、返回对治力（发誓不再造）。以死亡无常的五重论证构建情感驱动力。这是显宗哲学层面的忏悔体系。详见 `collections/入行论/品02-忏罪四对治力.md`。
- **互参建议**：入行论提供忏悔的完整理论框架和心理动力，大圆满前行提供具体的密宗观想方法。理论明白了再去修，修行更有力量。

**六度**：
- 大圆满前行（第101-114课）：在行菩提心学处中概述六度，以修心为主线。详见 `collections/大圆满前行/发殊胜菩提心·行菩提心六度.md`。
- 入行论（品04-09）：以每品一度逐一深入展开（品4不放逸+品5正知正念=戒律，品6安忍，品7精进，品8静虑，品9智慧），每一度都有数十颂的精密论证和丰富公案。**品4（已处理）**以11课论证守护菩提心的不放逸态度——三层递进从"为什么守护"（舍弃菩提心之过患）到"守护什么"（暇满人身）到"谁在威胁"（烦恼），将烦恼人格化为内在敌人，论证"能战且必胜"。详见 `collections/入行论/品04-不舍菩提心与谨慎暇满.md` 与 `品04-谨慎所断烦恼.md`。品8静虑品（42课）是全论篇幅最长的品，包含自他平等与自他相换这一入行论最独特的修法。品9智慧品（38课）是完整的中观哲学论述。
- **互参建议**：大圆满前行的六度是修行指南式的概述，入行论的六度是深度教科书式的展开。前者告诉你做什么，后者告诉你为什么这样做、遇到困难怎么办。

**依止上师**：
- 大圆满前行（第73-83课）：系统论述观察上师的法相、依止上师的方法、修学上师意行的原则，并以常啼菩萨、那若巴、米拉日巴等公案说明依师的深度。详见 `collections/大圆满前行/依止上师.md`。
- 入行论（总序）：以寂天菩萨与无著菩萨的师承传统为背景，建立对传承的信心。详见 `collections/入行论/总序-造论因缘与传承.md`。

### 2.3 各自独有之处

**大圆满前行独有——密宗实修体系**：

入行论作为显宗论典，不涉及以下内容：
- **金刚萨埵观想**：头顶观想本尊、甘露降注净化的完整密宗忏悔方法。详见 `collections/大圆满前行/金刚萨埵.md`。
- **供曼茶罗**：以须弥山王等供品积累福慧二资粮、三十七堆曼茶与三身曼茶的观想方法。详见 `collections/大圆满前行/积累资粮.md`。
- **古萨里积资（断法/施身修法）**：以身体布施鬼神的独特积资方式，是玛吉拉准尊者的传承。详见 `collections/大圆满前行/积累资粮.md`。
- **上师瑜伽**：明观莲花生大士福田、七支供、专心祈祷、四灌顶融入——这是前行与正行之间的枢纽，也是宁玛派修行的核心。详见 `collections/大圆满前行/上师瑜伽.md`。
- **往生法（颇瓦法）**：从利根者法身往生到平凡者三想往生的四层临终方案——修行者的最后保险。详见 `collections/大圆满前行/往生法.md`。
- **共同外前行的完整次第**：暇满难得、寿命无常、轮回过患、因果不虚——这四种厌世心的系统观修在入行论中不作为独立主题展开，而是嵌入菩提心修持的不同环节中（品01以暇满建立渴求、品02以死亡无常驱动忏悔、品04以暇满难得防止懈怠、以烦恼过患建立断除决心），大圆满前行则将其展开为70课的完整引导。入行论品4对暇满难得的再次论述特别有趣——与品1角度不同：品1是"建立渴求"（你有这个珍宝），品4是"防止懈怠"（你正在浪费它），两处暇满教法互相呼应却服务于不同的修行阶段。详见 `collections/入行论/品04-不舍菩提心与谨慎暇满.md`。

**入行论独有——深度哲学论证与独特修法**：

大圆满前行作为窍诀式修行手册，对以下内容仅略述或未涉及：
- **菩提心的系统论证**：品01以五项功德+六喻+教证理证构成密集的说服策略——这种层层推进的论证深度是大圆满前行所没有的。详见 `collections/入行论/品01-共同功德与六喻.md`。
- **菩萨戒的完整体系**：品02系统比较无著与龙猛两大菩萨戒传承、四种堕罪类型、还净方法——大圆满前行仅在行菩提心学处中提及学处框架。详见 `collections/入行论/品02-菩萨戒总义与供养顶礼.md`。
- **自他平等与自他相换修法**（品08）：这是入行论最独特的贡献——将菩提心从理念转化为体验的核心技术，42课的深度展开在其他论典中无对应内容（待处理）。
- **品09空性哲学**（38课）：完整的中观应成派立场——二谛、世俗如幻、胜义空性、人无我、法无我（四念住）、破实执。大圆满前行的般若度部分仅以八种幻喻和悲智双运概述，远不及此深度（待处理）。
- **七支供的完整仪轨与论证**：品02-03将供养、顶礼、皈依、忏悔、随喜、请转法轮、祈请不涅槃、回向逐一展开论证，每一支都有丰富的教证和修行指导。详见 `collections/入行论/品02-菩萨戒总义与供养顶礼.md` 与 `品03-受持菩提心.md`。

## 三、修行的实际顺序

### 3.1 传承上师的教言

修行次第不是理论推演，而是历代成就者验证的实修路径。以下是几位关键上师对前行修行的态度：

**法王如意宝晋美彭措**：规定喇荣五明佛学院每年讲一遍《大圆满前行》。法王在为本论所作序言中指出：此《前行》涵摄了藏传四大教派的修法引导——萨迦派《三显三续引导文》、格鲁派《菩提道次第广论》《密宗道次第广论》、噶举派《解脱庄严论》、宁玛派《大圆满心性休息大车疏》——的道次第精要。法王一生以身作则，终生重视前行修法——"前行既是前行，也是正行"。详见 `collections/大圆满前行/闻法方式.md`（序言部分）。

**堪布阿琼仁波切**：造《前行备忘录》时记录了具体实修时间要求。从人身难得观修开始，到菩提心的四无量心修法为止，至少需要146天的观修时间。这不包括后面的金刚萨埵、供曼茶罗、上师瑜伽等不共加行的数量修行。《前行备忘录》对每个引导文都给出了观修天数的建议，体现了"不赶进度、修到量为止"的原则。

**顶果钦哲仁波切**：终生随身携带《大圆满前行》，无论走到哪里都以此为日常修行的根本。堪布引用此事说明：不是只有初学者才需要前行，越有证量的修行者越重视前行。

**华智仁波切本人**：亲闻如来芽尊者讲授此法二十五次，"不增不减"记录而成此论。他在序言中反复强调：不修前行而直趋正行的修行者，无论当下感觉多好，最终必然出问题。

### 3.2 实际修行次第

基于大圆满前行的结构和传承教言，修行者的实际次第如下：

**第一步：闻法方式（心态准备）**

一切修行的基础是正确的闻法态度。三种过失（覆器、污器、漏器）、六种垢染、五种不持——这些不是"初级知识"，而是贯穿一切修行的心态检查。每座修行前以三殊胜摄持（加行发心殊胜、正行无缘殊胜、结行回向殊胜）。详见 `collections/大圆满前行/闻法方式.md`。

具体日修框架：晨起→日间四或六座→晚睡，每座含前行（毗卢七法+排垢气+发心祈祷）、正行（观修引导文）、后行（回向）。详见 `collections/大圆满前行/日修闭关要诀.md`。

**第二步：共同外前行（生起出离心）**

按顺序逐一观修六个引导文。每个引导文修到相应定解在相续中稳固，再进入下一个：

1. 暇满难得——修到发自内心珍惜人身。详见 `collections/大圆满前行/暇满难得.md`。
2. 寿命无常——修到不再拖延、当下即修。详见 `collections/大圆满前行/寿命无常·四观无常.md` + `寿命无常·喻义与修行.md`。
3. 轮回过患——修到对六道一切受生都生起厌离。详见 `collections/大圆满前行/轮回过患·总述与三恶趣.md` + `轮回过患·三善趣.md`。
4. 因果不虚——修到在取舍善恶上谨慎细致。详见 `collections/大圆满前行/因果不虚·十不善业与果报.md` + `因果不虚·业之自性.md`。
5. 解脱利益——修到对解脱生起真实希求。详见 `collections/大圆满前行/依止上师.md`。
6. 依止上师——修到对上师生起不退信心。详见 `collections/大圆满前行/依止上师.md`。

前四项共同作用生起出离心。只有出离心稳固，后面的菩提心才不会流于空谈。

**第三步：不共内加行（实修积累）**

在出离心的基础上进入密宗加行的实修。五加行有严格的递进关系：

1. **皈依**（皈依偈10万遍）：正式进入修行之门。以三宝为依怙，建立修行的容器。详见 `collections/大圆满前行/皈依.md`。
2. **发菩提心**（发心偈10万遍）：将修行升级为大乘道。四无量心修心+正式发菩提心。详见 `collections/大圆满前行/发殊胜菩提心·四无量心与发心.md` + `发殊胜菩提心·行菩提心六度.md`。同时可参学入行论品1-3，加深对菩提心的理解和信心。
3. **金刚萨埵**（百字明10万8千遍）：清除修行障碍。以金刚萨埵本尊观想+百字明念诵净化罪障。详见 `collections/大圆满前行/金刚萨埵.md`。入行论品2的四对治力可作为理论基础。
4. **供曼茶罗**（供曼茶10万遍）：积聚修行顺缘。以三十七堆曼茶/三身曼茶积累福慧二资粮。详见 `collections/大圆满前行/积累资粮.md`。
5. **上师瑜伽**（莲师心咒1千万遍）：将传承加持直接融入自心。这是前行与正行之间的枢纽——既是五加行的最后一项，也是一切修法的究竟方便。详见 `collections/大圆满前行/上师瑜伽.md`。

关键原则："只有当心获得一种证量或达到一个定量，才可以说圆满了加行。否则，光是完成一个数量，自心却没有得到任何调伏，那还算不上是'圆满'。"（详见 `collections/大圆满前行/皈依.md`）

**第四步：往生法（临终保障）**

往生法看似"附加"内容，实则是修行者的临终保险。即使前行和正行修得不够圆满，凭借往生法也能保证不堕恶趣。四层方案适应不同根器：利根者法身往生、中根者报身往生、下根者化身往生、平凡者三想往生。详见 `collections/大圆满前行/往生法.md`。

**第五步：正行（大圆满/大手印等）**

完成以上全部前行后，方可正式进入上师传授的正行法门。华智仁波切反复强调：没有前行基础的正行修法如冰上建房——看似华美，必然坍塌。

### 3.3 入行论在修行次第中的位置

入行论不是独立于上述次第之外的另一套修法，而是在次第中的"菩提心与六度"环节提供深度资源：

- **修共同外前行时**：入行论品02的死亡无常论证可加深对寿命无常的理解；品04的暇满难得重述可巩固珍惜人身的定解。
- **修不共加行之发菩提心时**：入行论品01-03是最好的辅助读物。品01建立对菩提心的信心，品02清净障碍，品03正式受持——与大圆满前行的四无量心+发心修法形成完美互补。
- **修行菩提心六度时**：入行论品04-09是六度的深度教材。尤其品06安忍（嗔恨是菩提心最直接的破坏者）、品08自他相换（将菩提心从理念转化为体验）、品09智慧（空性见解的完整论证）。
- **日常修行中**：入行论品05正知正念提供了在一切行为中保持觉察的具体方法，品07精进提供了对治懈怠的策略。

## 四、《前行备忘录》的位置

堪布阿琼仁波切所造《前行备忘录》是大圆满前行修行体系中的第三层资源，其定位是**实修窍诀层面的注释**。

### 4.1 三源互参体系

大圆满前行的知识建立在三层来源之上：

1. **原论**（华智仁波切《普贤上师言教》）：精确的论证骨架——全论由如来芽尊者的口传教言不增不减记录而成。
2. **课程讲解**（索达吉堪布2008年版144课）：深入解读——大量引用经论旁证、公案比喻、当代应用，使窍诀与广大经论形成互参。
3. **前行备忘录**（堪布阿琼仁波切造）：实修窍诀——提供原论和讲解中未展开的具体观修步骤。

详见 `collections/大圆满前行/结构总览.md`（三源互参部分）。

### 4.2 前行备忘录的独特贡献

前行备忘录在多个关键修行环节提供了原论和讲解所未展开的实修细节：

- **四无量心的具体修法次第**：知母→念恩→报恩的层层推进法、有缘无量心与无缘无量心的区分、四无量心与四梵住的关键辨析。详见 `collections/大圆满前行/发殊胜菩提心·四无量心与发心.md`。
- **四种罪障的精确分类**：业障、烦恼障、所知障、习气障——及其各自障碍什么果位的分析。详见 `collections/大圆满前行/金刚萨埵.md`。
- **皈依的深层含义**：大乘行人畏惧有寂二边而皈依——不仅畏惧轮回之苦，也不安住于小乘的涅槃寂灭。皈依与戒律的递进关系：皈依→戒体→戒律→禅定→证悟。详见 `collections/大圆满前行/皈依.md`。
- **上师瑜伽的五种了知**：对上师具足五种认知的具体修法。详见 `collections/大圆满前行/上师瑜伽.md`。
- **具体观修天数建议**：为每个引导文设定最低观修天数，体现"修到量"而非"赶数量"的原则。

### 4.3 使用建议

初学者以原论为纲、以讲解为主体学习；当进入实修阶段，需要具体观修步骤时，查阅前行备忘录的对应内容。知识文档在涉及前行备忘录内容时标注"（《前行备忘录》）"。

## 六、核心修行原则

贯穿整个修行道次第的几条根本原则：

### 次第不可颠倒

华智仁波切反复强调：不修前行而直趋正行的修行者，最终必然出问题。前行不是"低级"法门，而是一切证悟的地基。共同外前行未修好就进入不共加行，等于没有出离心就修菩提心——结果只是口头禅。不共加行未修好就进入正行，等于地基未打就盖楼——无论外表多美，必然倒塌。

### 修到量而非赶数量

五加行各有数量要求（皈依10万、发心10万、百字明10万8千、供曼茶10万、莲师心咒1千万），但数量只是最低标准。关键是内心是否真正生起相应的证悟——"只有当心获得一种证量或达到一个定量，才可以说圆满了加行。"

### 见行平衡

莲花生大士黄金法则："是故见比虚空高，取舍因果较粉细。"见解越高，行为上的因果取舍越要细致。这个原则贯穿从小士道到大士道的全部修行。详见 `collections/大圆满前行/因果不虚·业之自性.md`。

### 三殊胜摄持一切

- **加行发心殊胜**：以菩提心摄持——无论修哪个引导文都为利益众生而修。
- **正行无缘殊胜**：不被分别念和违缘毁坏修行善根。
- **结行回向殊胜**：以回向印持——将一切善根回向众生，使善根不会被嗔恨等烦恼摧毁。

详见 `collections/大圆满前行/闻法方式.md`。入行论品10回向品即是结行回向殊胜的完整展开。

### 闻思修不可偏废

大圆满前行的结构本身就体现了闻思修的统一：闻法方式教你怎么听（闻），共同外前行让你反复思维直到生起定解（思），不共内加行让你在座上座下真正修行（修）。入行论同理：品1-3的论证是闻思层面的建立，品4-9的六度是修行层面的展开。两部论典合参，闻思修三者更为圆满。
