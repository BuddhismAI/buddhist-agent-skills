# Yogachara (唯识 / 瑜伽行) — Topic Index

This topic covers **唯识宗 / 瑜伽行派 (Yogācāra / Cittamātra)** teachings — the "third turning" Mahāyāna system centered on 阿赖耶识, 三自性, 唯识所现, 八识转四智, 转依, 三身. The foundational text currently processed is 弥勒菩萨《大乘经庄严论》(Mahāyāna-sūtrālaṃkāra) — one of 弥勒五论 — as taught by 索达吉堪布 over 111 lessons (2015-2018) based on 麦彭仁波切《密意疏·胜乘甘露喜宴》.

Read this after the root `SKILL.md` to orient yourself in the yogachara knowledge space. **Read §Editorial Policy below before drafting any answer** — this topic sits at a doctrinal fault-line (唯识 vs 中观, 名言实有 vs 宁玛重释) that every answer must navigate explicitly.

## Editorial Policy — REQUIRED reading before answering

This topic sits at a live doctrinal fault-line between **唯识宗的名言框架** and **中观/宁玛对唯识之破立**. The collection wiki preserves **three interpretive layers** that answers must not flatten:

| Layer | Source | Position | Use |
|---|---|---|---|
| **弥勒本论 / 唯识名言** | 弥勒菩萨《大乘经庄严论》本颂 + 无著/世亲/安慧注疏 | **唯识框架下的系统教义** — 以阿赖耶识、三自性、八识转四智、三身为真实教说. 在本论中这些不是方便说, 是弥勒菩萨抉择大乘道的首要架构. | Always preserve the 唯识 vocabulary when quoting 本论. Do not silently re-write 阿赖耶识 as "六识相续" or 转依 as "空性悟入". 标为 "本论立场" 或 "弥勒本论". |
| **索达吉堪布宁玛派重释** | 课程102 索达吉堪布讲解 + 麦彭仁波切《密意疏》 | **以宁玛视角疏通** — 保留唯识名言, 但将其限定在名言层面: 阿赖耶识**胜义中非实有**, 只名言承许; 三自性中圆成实可开显为离戏大中观 / 自性光明大圆满. 本论的"真如不空" 不等同觉囊派义他空. | 每当涉及胜义实有争议 (阿赖耶是否实有? 依他起是否成实? 真如是否存在?) 必须引入此层. 标为 "索达吉堪布说" 或 "宁玛派疏通". |
| **中观/应成对唯识实有派的破斥** | 静命论师《中观庄严论》第71-80课 破唯识 10 课, 月称《入中论》破唯识 / 应成因明 | **中观利器破唯识** — 破"识实一" "自证分实有" "依他起胜义自成立"; 但 **不否定名言唯心**. 瑜伽行中观派 (静命) 的立场是: "见跟龙猛 (中观), 行跟无著 (唯识)" — 名言随顺唯识, 胜义遮遣唯识实有. | 回答"中观派如何看唯识?" / "唯识宗是否究竟?" 类问题时必须引入. 标为 "中观自续派 / 应成派 / 静命瑜伽行中观派". |

**宁玛传承在弥勒五论上的独特立场** (课60 索达吉堪布明点):

> "《辩中边论》《辨法法性论》及《大乘经庄严论》, 弥勒菩萨都是用唯识宗的观点宣讲的, 因为整个框架是属于大乘唯识宗的法. 有时候如果按照大乘中观来讲会比较直接, 但意思都是相同的."

关键三点: (1) **框架是唯识, 意义与中观无二**. (2) **种姓品 ↔ 如来藏**: 本论"种姓品"(因位) 与《宝性论》"如来藏品"(亦因位) 为同一教义的两种开显. (3) **阿赖耶转依 ↔ 大圆满本来清净**: 本论"转依" 在宁玛判教中可开显为本来清净的显发, 非新造一个清净本体.

**What this means for answers:**

1. **忠实呈现唯识框架**. 不要因为个人/训练数据偏向中观就把唯识名言替换掉. 阿赖耶识是阿赖耶识, 不是"相续". 三自性是三自性, 不是"二谛". 八识转四智就是八识转四智, 不是"破我执得智慧".
2. **在胜义实有问题上明确宁玛立场**. 凡遇 "阿赖耶识实有吗?" / "依他起实有吗?" / "真如是一个存在的东西吗?" 之类问题, 必须答: 名言中如此安立, 胜义中同样以离一多因遮遣实有; 本论之唯识框架**不是实有唯识派**.
3. **与中观的关系以"圆融"而非"对立"呈现**. 唯识给名言建立, 中观给胜义遮遣. 瑜伽行中观是圆融点 (参 `topics/madhyamaka/yogachara-madhyamaka.md`). 不要制造"唯识 vs 中观二选一"的假对立.
4. **与如来藏的关系以"同源"而非"同义"呈现**. 本论种姓品 ↔ 宝性论如来藏品, 是弥勒五论内部"因位"的两种进路 — 唯识式 (以种姓决定能入道者) vs 如来藏式 (以佛性普遍覆). **不可简单画等号, 但可指出同源**.
5. **如果用户从"唯识实有派"语境提问** (比如汉传法相宗背景), 先在其框架内回答, 再在必要处补一句宁玛重释 — 不要粗暴把对方立场判为"错".

## The text's intellectual architecture — 七义 + 五种无上意义 + 二十一品

《大乘经庄严论》最精妙之处在于**开篇两颂已摄全论**. 理解本论 = 理解这两颂的总纲如何展开为 21 品.

### 七义 — 立誓语摄全论

第一偈"**知义讲义语句净, 为救苦性悯苦情, 示胜乘法宣行众, 圆五义理无上性**"以印度声明学**七格** (依言语关系) 对应七种意义, 实即"谁造 / 造什么 / 以何而造 / 为何而造 / 因何而造 / 属于什么 / 为谁而造"七维度定位本论.

| 意义 | 声明学七格 | 所指 |
|---|---|---|
| 知义 | 本体格 | 弥勒菩萨通达一切大乘教义 — 造论者身份 |
| 讲义 | 示宾格 | 所造之论 — 论体 |
| 语句净 | 施事格 | 八种无垢语 |
| 救苦 | 故为格 | 救度众生 — 造论目的 |
| 性 | 出处格 | 大悲心 — 造论动机 |
| 示胜乘法 | 所属格 | 所属大乘 — 论之范畴 |
| 宣行众 | 依存格 | 大乘信解者 — 所化众生 |

### 五种无上意义 — 21 品的五阶段递进

第二偈"**如金打造莲花开, 如饥饿人享美食, 如闻佳音启宝箧, 彼法此说亦最喜**"以五喻摄 21 品的"五令极喜之理" — 从**所建立 → 分别所了知 → 所思维 → 所获得 → 菩提分法之体性**五阶递进.

| # | 无上意义 | 比喻 | 对应品 | 课数 |
|---|---|---|---|---|
| 1 | **所建立** (大乘是佛语) | 如金打造饰品 | 品2 证成大乘品 | 3-5 |
| 2 | **分别所了知** (皈依/种姓/发心/修行) | 如莲花开启 | 品3-6 | 5-16 |
| 3 | **所思维** (真如/威力/成熟) | 如饥饿人享美食 | 品7-9 | 17-23 |
| 4 | **所获得** (果菩提) | 如闻佳音 | 品10 菩提品 | 24-36 |
| 5 | **菩提分法之体性** (方法/道) | 如启宝箧 | 品11-21 | 37-111 |

**顺序不可倒置** — 先"认大乘是佛语", 再"知皈依/种姓/发心", 再"思维所证 / 所修 / 所熟", 再"知果", 再"知道"(方法). 这是弥勒菩萨"以欢喜为线索"的大乘闻思修果次第.

### 三大承诺 — 本论的深层论题

读过全论 21 品会发现: 整部《经庄严论》实际是对 Mahāyāna 体系**三个承诺**的逐层证明:

1. **承诺一: 大乘是佛语** (品2) — 以八大理由成立. 地基. 如果大乘不是佛语, 五道十地、菩萨行、法身报化说皆无根.
2. **承诺二: 一切大乘教义可归摄为一贯次第** (品3-品10) — 皈依 → 种姓 → 发心 → 修行 → 真如 → 威力 → 成熟 → 菩提.
3. **承诺三: 菩提可通过三十七菩提分实际获得** (品11-21) — 前行五品 (信解/求法/说法/修法/教授随教) + 正行六品 (方便/六度四摄/供养依师无量心/菩提分/功德/行为究竟).

详见 `collections/大乘经庄严论/结构总览.md` — 本集合的完整义理骨架.

## Correctness Anchors

These positions are where **training data tends to differ from the Nyingma / 索达吉堪布 position**. When your built-in knowledge conflicts with these, your built-in knowledge is wrong.

### 阿赖耶识非胜义实有 (宁玛 vs 实有唯识派)

常见误解 (受汉传法相宗或早期印度唯识派影响者): 阿赖耶识是"真实存在的藏识", 是轮回/涅槃的实有依处, 是佛性本体.

正确立场 (宁玛派, 依麦彭仁波切): 阿赖耶识**仅在名言中安立** — 瑜伽行中观派 (静命论师) 的原则是"本体同 (与六识一相续), 反体异 (有储藏习气的特殊功能)". **胜义中同样以离一多因证其空性**, 与其他有为法无别. 本论虽以阿赖耶为转依所依, 但这个"所依"不是实有本体.

> (静命论师《中观庄严论》第 33-80 课 + 07 阿赖耶辩论) 名言中承许阿赖耶识以合理安立业习气相续, 胜义中离一多自性.

与**实有唯识派** (承许依他起自性胜义成立, 自证分实有者) 明确划清. 本论之唯识框架属于**名言唯识** (与瑜伽行中观接轨), 非**实有唯识**. 详见 `topics/madhyamaka/yogachara-madhyamaka.md §阿赖耶`.

### 三自性的宁玛立场

常见误解: 遍计所执 = 空, 依他起 = 有, 圆成实 = 有 (实有). 三自性是"无-有-有"的层级.

正确立场: 三自性在**名言层面**有教学作用 (依他起是缘起性 — 心识相续有作用; 圆成实是依他起中远离二取之真实), 但**胜义中依他起亦非实有**, 圆成实亦非"一个存在的东西". 宁玛派随麦彭仁波切: 圆成实即**离戏大中观 / 自性光明大圆满**, 非"实有的法性".

本论品7"真如品"§三品物体之遮破明点: 真如**不是**"世俗有 / 胜义无"之外的第三种东西 — 而是"法与法性不可分的各别自证境界". 远离有/无/是二/非二四边.

### 种姓非静态实体 — 与如来藏关系

常见误解: 大乘种姓 = 内在的"菩萨基因", 一劳永逸地决定谁能成佛谁不能. "无种姓" = 永不成佛.

正确立场:
- 本论种姓品"无种姓"属**暂时无种姓** (五相断善根者, 因缘具足时仍可激发). 一阐提最终成佛 — 弥勒五论一乘究竟立场.
- 种姓具"**自性住**" (本具基础) + "**增长**" (修行激发) 两面. 不是死板的分类.
- 本论种姓 ↔《宝性论》如来藏 — 弥勒五论内部"因位"的两种开显. 本论从"谁能入大乘道"的唯识式角度, 宝性论从"一切众生本具佛性"的如来藏式角度. **同源但非同义**.

参 `collections/大乘经庄严论/04-种姓品.md` + `collections/宝性论/03-如来藏-总纲十义.md §因`.

### 唯识所现 与 中观一切法空 — 圆融而非对立

常见误解 (中观与唯识对立观): 唯识说"万法唯心" 必承许心实有, 与中观"一切法空"相违. 故二宗择一.

正确立场: **瑜伽行中观 (静命论师) 是圆融范本** — "依于唯心已, 当知外无实; 依于此理已, 知彼亦无我". 两步次第:
1. 以唯识理破外境 (较易接受)
2. 以中观理破心识 (更深层次)

"见跟龙猛 (中观), 行跟无著 (唯识)" — 名言随顺唯识合理, 胜义遮遣唯识实有派. 本论虽以唯识为框架, 但讲解时以宁玛视角疏通, 不落实有唯识.

详见 `topics/madhyamaka/yogachara-madhyamaka.md` (**必读**).

### 四智 vs 五智 — 本论只讲四智

常见误解 (受密宗五方佛五智影响者): 佛有五智 — 大圆镜 / 平等性 / 妙观察 / 成所作 / **法界性** 智.

正确立场 (本论): 本论**只讲四智**, 法界清净不直接立为智慧, 而立为"智慧所依的法界" (课32). 这是本论与密宗"五智"判教的根本差别. 本论式"五法 = 1 (法界清净) + 4 (四智)".

不可粗暴等同"本论四智 = 密宗前四智". 此为显密法分野. 密宗五智别有自宗完整安立, 不与本论冲突但属不同判教. 参 `collections/大乘经庄严论/11-菩提品-三身四智.md §2 己一`.

### 本论三身 vs 宝性论三身 — 同义异入路

常见误解: 三身是一个统一概念, 哪里讲都一样.

正确立场:
- **本论三身** (品10): 从**唯识转依**立 — 八识转四智后的果位身. 三平等 + 三恒常.
- **宝性论三身** (品4-5): 从**如来藏光明**立 — 法身三理由 + 色身七理由恒常.

二者在实相上无二 (弥勒菩萨亲许), 但**入路不同, 名相侧重不同**. 显密三身又有层次差别 (参 `topics/tathagatagarbha/index.md §三身显密安立的层次差别`). 回答时需指明在哪个框架内讲.

### 本论虽唯识宗但可作大乘道次第总摄

常见误解: 本论 = 唯识宗专属论典, 只有修唯识者可用; 修中观 / 如来藏 / 密法 者不需. 或: 本论是"宗派书", 不是"道次第书".

正确立场: 麦彭仁波切明示"**这部论典, 无论谁去讲述、听闻, 都能获得讲闻一切大乘佛法的功德**"(课1 反复引). 本论的五种无上意义是**一条完整的大乘道次第** — 皈依 → 种姓 → 发心 → 修行 → 真如 → 威力 → 成熟 → 菩提 → 三十七菩提分. **不局限于唯识一宗**, 可作广义大乘道次第使用.

在宁玛传承中, 本论与《宝性论》并列为"显教大乘究竟教义" — 弥勒菩萨的两部"庄严论"与一部"宝性论"共同建立**显密融通之基**.

### 本论"真如不空" 不等同觉囊派义他空

常见误解: 本论说真如有 (非空), 故本论立场 = 觉囊派义他空 "如来藏不空".

正确立场: 本论讲真如五义"非有非无 / 不一不异 / 无生无灭 / 无减无增 / 非净成清净" — 关键是**非有非无**, 即离有无二边. 不是"有一个实有真如法存在"而是"离戏大中观义"(麦彭仁波切). 觉囊派义他空 (空基不空) 是另一立场, **不可等同本论**. 详见 `topics/tathagatagarbha/index.md §Editorial Policy` (朵洛瓦 vs 索达吉堪布 的双层处理).

## Relationship with other topics

### → madhyamaka (中观)

最深的 topic 关联. **唯识 / 中观的关系不是对立而是分工**:
- **唯识**: 名言建立 — 心识如何缘起、业习气如何载续、三自性如何分布、转依如何成办
- **中观**: 胜义遮遣 — 一切法 (包括阿赖耶、自证、依他起) 以离一多因证其空性

**瑜伽行中观 (静命论师) 是圆融范本**. 本 topic 与 madhyamaka topic 在以下文档处交会:
- `topics/madhyamaka/yogachara-madhyamaka.md` — 瑜伽行中观的完整阐述 (**必读, 双 topic 共享**)
- `collections/中观庄严论/07-阿赖耶辩论.md` — 阿赖耶的名言安立 / 胜义遮遣
- `collections/中观庄严论/14-16 破有外境派之识 + 破唯识所许实一之识` — 中观对唯识实有派 (真相/假相唯识) 的精细破析
- `collections/入行论/品09-二谛安立与遣除争论.md` — 入行论式破自证 (刀剑喻 / 灯火喻 / 蓝色喻)

**本 topic 与 madhyamaka topic 的分工原则**:
- **名言唯识建立** (八识 / 三性 / 转依 / 三身四智 / 种姓 / 道次第) → 本 topic 回答
- **对唯识实有派之破** / **瑜伽行中观如何融通二宗** → madhyamaka topic 回答
- **二者交汇** (如真如品的二无我以唯识方法论证) → 双 topic 并引

### → tathagatagarbha (如来藏)

**弥勒五论兄弟**关联. 本 topic 与 tathagatagarbha 同属弥勒五论体系:
- 《大乘经庄严论》《辩中边论》《辨法法性论》《现观庄严论》 = 四部论 (由无著 / 世亲系统开展的唯识学)
- 《宝性论》 = 一部论 (由朵洛瓦 / 觉囊派系统开展的如来藏学)

**种姓 ↔ 如来藏**: 弥勒五论内部"因位"的两种开显. 本论种姓品从"谁能入大乘道"角度, 宝性论如来藏品从"一切众生本具佛性"角度. 同源但非同义.

**三身对比**: 本论(唯识转依式) vs 宝性论(如来藏光明式). 实相上无二, 入路与恒常理由不同.

交叉 topic 路由: 涉及弥勒五论整体架构的问题, 建议同时引两个 topic 的 index.

### → foundations (基础 / 加行)

本论的**菩提心 / 六度 / 四摄 / 四无量心 / 菩提分 / 道次第** 内容与 foundations 有大量共享基础. 本 topic 不重复 foundations 已建立的基础, 而是呈现**唯识式的系统化安立**:

- **发心品** (品5, 22 喻) → 与 `topics/foundations/bodhicitta-methods.md` 互补: foundations 给修持方法, 本 topic 给弥勒菩萨的系统分类
- **六度** (品17 上) → 与 `collections/入行论/` 互补: 入行论讲修持实践, 本论讲六度如何"圆满自佛法"的理体系
- **四摄** (品17 下) → `topics/foundations/practice-path.md`
- **菩提分 / 三十七道品 / 四法印** (品19) → `topics/foundations/` + `collections/俱舍论/品6-四谛与加行道.md` (小乘式对比)
- **依止善知识** (品18) → `topics/foundations/`

原则: 涉及**修持怎么做**, 归 foundations; 涉及**教义系统如何安立 (本论式 / 弥勒菩萨式)**, 归本 topic.

### → pramana (量理 / 因明)

本论**求法品** (品12) 含"理安立 / 辩无碍解 / 寻求五明" — 其中"因明"之说与解义慧剑的四理 / 四法依对应. 本论**说法品** (品13) 的四秘密 / 四意趣是大乘**诠释学** (了义 / 不了义判分) — 与四法依"依了义不依不了义" 直接对照.

关联文档:
- `topics/pramana/index.md` — 因明总括
- `collections/解义慧剑/结构总览.md` / `四法依.md` — 因明 / 诠释学详展
- `collections/大乘经庄严论/13-求法品.md` / `14-说法品.md`

### → prajnaparamita (般若)

本论"**心净国土净**" (课101-102, 品10) 与《维摩诘经》品1 同名教义相对读:
- 本论侧重"**心净土净的唯识式建立**" — 八识转四智后心体清净, 国土随之显净相
- 维摩诘经侧重"**心净土净的般若/不二建立**" — 同一国土因观者心净染而显不同相

参 correctness anchor "心净国土净非主观唯心" — 双方共许, 路径不同.

此外, 本论"真如品"的"五义" (非有非无等) 与《心经》"不生不灭 / 不垢不净 / 不增不减" 同义异表.

### → abhidharma (俱舍 / 对法) [轻]

本论**菩提分品** (品19) 讲**三十七道品** — 与《俱舍论》品6 小乘三十七道品对比. 本论是**大乘视角重展** (同样的法目, 不同的修持动机 + 所缘 + 证果).

本论**四无量心** (品18) 亦可与小乘四无量心对比. 本论十地 / 五道 (品21) 亦与《俱舍论》五道 (加行 / 见道 / 修道 / 无学道) 对比 — 本论是大乘五道十地.

关联: `collections/俱舍论/品6-四谛与加行道.md`.
