# 量论种子内容·来自定解宝灯论

定解宝灯论第六问"异境何为共所见"包含宁玛派量论的系统表述。以下内容在未来建立 pramana 主题时可直接吸收。

## 三层量架构

麦彭仁波切（依《解义慧剑》体系）将量分为三层：

```
量
├── 胜义量
│   ├── 真实胜义量：中观应成派——离一切戏论
│   └── 假胜义量：中观自续派——抉择"空"
└── 名言量
    ├── 净见量：圣者/具证修者的清净认知
    └── 观现世量（凡夫名言量）
        ├── 现量：未被迷乱因染污的根识直接感知
        └── 比量：依逻辑推理间接获得
```

因明七论（《释量论》等）仅抉择到观现世量。净见量在弥勒菩萨《宝性论》中较明显，《大幻化网总说光明藏论》主要抉择净见量。胜义量在中观论著中。

此分类是宁玛派量论与格鲁派因明传统的关键差异点——格鲁派因明基本不涉及净见量层面。

## 量之观待成立

> **故以观待量成立，独立自主之诸法，并非以量可成立，若成则应成实相。**

一切正量均观待特定条件（众生业感、认知主体层次）而成立，并非独立自主成立。

以水为例：
- 观待人类→水为正量
- 观待饿鬼→水不成立（只有脓）
- 胜义量前→水的一滴也无法以自性成立

若水不观待他法而独立存在→天人应见水（不应见甘露）→饿鬼应见水（不应见脓）→但事实并非如此→∴ 水非独立自主成立。

此论证将缘起性空原理直接运用到认识论领域。

## 净见量：上上为正量

> **是故障碍之外缘，愈来愈加清净后，观待下面所见言，当许上上为正量。**

障缘越清净，所见越为正量：

地狱 < 饿鬼 < 人 < 天人 < 持明 < 诸地菩萨 < 佛陀

人所见之水，在持明者面前可见为玛玛格佛母（水大佛母）。故人类所见不能决定为究竟正量。

## 究竟一正量

> **究竟法性之真如，独一无二唯一性，能见正量亦唯一，第二正量不可有。**
> **实相双运唯一谛，正量自然之智慧，唯一所断即无明，仅觉未觉之差别。**

究竟实相（净等双运唯一真谛）只有一个，能见此实相的正量（佛陀自然本智）也只有一个。佛与众生的差别仅是觉与未觉。

## 与解义慧剑的关系

定解宝灯论Q6提供的是**具体论证案例**（以水为例展开三层量的运作），解义慧剑提供的是**抽象框架定义**（四种理、量的分类体系）。二者互补：

- 解义慧剑定义了什么是观现世量、净见量→定解宝灯论Q6展示了它们如何在"共同所见"辩论中实际运作
- 解义慧剑的证成理（现量四种、比量）→定解宝灯论Q6展示了量之观待成立如何从根本上限定正量的成立条件
