## Editorial Policy — REQUIRED reading before answering

This topic carries a key cross-tradition fault-line:

### 宁玛派量论的不共特色：量之四分

格鲁派因明传统 (基于《释量论》七论) 主要抉择**观现世量** (凡夫名言量)，对**净见量** (圣者清净量) 和**胜义量** (相似/真实) 的处理有限。

宁玛派 (麦彭仁波切《解义慧剑》体系) 将量分为四层：

```
                       ┌─ 真实胜义量 (中观应成派 — 离一切戏论)
              ┌─ 胜义量 ┤
              │        └─ 假/相似胜义量 (中观自续派 — 抉择「空」)
        量 ───┤
              │        ┌─ 净见量 (圣者/具证修者的清净认知)
              └─ 名言量 ┤
                       └─ 观现世量 (凡夫名言量)
                                 ├─ 现量 (未被迷乱因染污的根识直接感知)
                                 └─ 比量 (依逻辑推理间接获得)
```

- **因明七论** (《释量论》等) 仅抉择到观现世量
- **净见量** 在弥勒菩萨《宝性论》中较明显，《大幻化网总说光明藏论》主要抉择净见量
- **胜义量** 在中观论著中

此分类是宁玛派量论与格鲁派因明传统的**关键差异点** — 格鲁派因明基本不涉及净见量层面。

详见：`collections/解义慧剑/证成理·比量与量之四分.md` 和 `collections/定解宝灯论/Q6-异境何为共所见.md`

### 量必依止 — 不可孤立运用因明工具

宁玛派立场：因明 (量理) 是抉择见解的工具，但工具必须依止见解的整体方向。脱离中观正见而单独耽著因明形式分析，易堕入**戏论增益**或**只破不立**的过失。

- 因明七论的高峰指向「量士夫」(佛) — 量的终极成立处是佛语，反过来佛语的成立又依量。这是循环吗？不是 — 这是认识论的闭环 (`证成理·比量与量之四分.md` §一致摄义)。
- 《解义慧剑》最末四法依从「依法不依人」走到「依智不依识」— 在了义法中，应依无二智慧而非二取心识。换言之，**因明分别识是抵达无分别智的舟筏，不是终点**。

### 净见量 vs 观现世量是观待成立

> **故以观待量成立，独立自主之诸法，并非以量可成立，若成则应成实相。**

一切正量均观待特定条件 (众生业感、认知主体层次) 而成立，并非独立自主成立。

以水为例：
- 观待人类→水为正量
- 观待饿鬼→水不成立 (只有脓)
- 胜义量前→水的一滴也无法以自性成立

若水不观待他法而独立存在 → 天人应见水 (不应见甘露) → 饿鬼应见水 (不应见脓) → 但事实并非如此 → ∴ 水非独立自主成立。

**此论证将缘起性空原理直接运用到认识论领域**，是宁玛派量论的特色诤辩工具。

详见：`collections/定解宝灯论/Q6-异境何为共所见.md` 和 `collections/定解宝灯论/推理方法/量之观待论证.md`

## Concept Map

### 四理 (能量四理)

麦彭仁波切《解义慧剑》将一切名言抉择归摄为四种道理：

| 理 | 角度 | 内容 |
|---|---|---|
| **作用理** | 从因 | 因缘齐全必生果 |
| **观待理** | 从果 | 一切果必观待其因 |
| **法尔理** | 从本体 | 世俗法尔 — 诸法各住不共法相；胜义法尔 — 因果体皆空，三解脱法界 |
| **证成理** | 从验证 | 现量与比量验证前三理 |

前三理建立名言因果法则；证成理以现量比量验证。最终「理终归法尔，缘由无所觅」— 法尔如是，不可再追问。

详见：`collections/解义慧剑/二谛与前三理.md`

### 现量四种 (证成理之一)

| 现量 | 所缘 | 特点 |
|---|---|---|
| **根现量** | 五根所对外境 | 凡夫主要现量 |
| **意现量** | 意识所现 | 紧随根现量 |
| **瑜伽现量** | 修证所现 | 打破「只有感官能见」 — 通往净见量的桥梁 |
| **自证现量** | 心自身 | 一切认识活动的最终基础。无自证则其余三现量无法被确认 |

自证现量是宁玛/萨迦传统所共许，是量论的根基。格鲁派 (尤其月称应成派) 名言中不许自证 — 此为应成派与自续派一大分歧 (详见 `topics/madhyamaka/debates.md`)。

详见：`collections/解义慧剑/证成理·现量四种.md`

### 比量三要 (证成理之二)

| 要素 | 说明 |
|---|---|
| **以何心比量** | 名义混合的分别识 |
| **以何因比量** | 三相齐全的因 (宗法 / 同品定有 / 异品遍无) |
| **以何方式比量** | 建立式 (证成义理因) 与遮破式 (能破因) |

比量分类的多种维度：
- 以所量安立三/六种 (果因 / 自性因 / 不可得因 + 各分二)
- 以建立方式安立四种
- 以缘取方式安立二种

详见：`collections/解义慧剑/证成理·比量与量之四分.md`

### 量之四分 (宁玛派核心贡献)

| 量 | 类别 | 所对 | 何宗成立 |
|---|---|---|---|
| **观现世量** | 名言 | 凡夫根识所见 (现相 ≠ 实相) | 一切宗共许 |
| **净见量** | 名言 | 圣者清净所见 (现相 = 实相) | 大乘共许，密层尤为关键 |
| **相似胜义量** | 胜义 | 单空 (遮破所破后的不存在) | 中观自续派 |
| **真实胜义量** | 胜义 | 远离四边八戏的大空性 | 中观应成派 |

**为什么必须如此四分？**
- 若无净见量 → 圣者的清净显现就成虚妄，「诸根是量，圣道复益谁？」
- 若无二种胜义量 → 无法知道「二谛圆融」，胜义将堕入戏论边

详见：`collections/解义慧剑/证成理·比量与量之四分.md` + `collections/定解宝灯论/Q6-异境何为共所见.md`

### 净见量：上上为正量

> **是故障碍之外缘，愈来愈加清净后，观待下面所见言，当许上上为正量。**

障缘越清净，所见越为正量：

```
地狱 < 饿鬼 < 人 < 天人 < 持明 < 诸地菩萨 < 佛陀
```

人所见之水，在持明者面前可见为玛玛格佛母 (水大佛母)。故人类所见不能决定为究竟正量。

详见：`collections/定解宝灯论/Q6-异境何为共所见.md`

### 究竟一正量

> **究竟法性之真如，独一无二唯一性，能见正量亦唯一，第二正量不可有。**
> **实相双运唯一谛，正量自然之智慧，唯一所断即无明，仅觉未觉之差别。**

究竟实相 (净等双运唯一真谛) 只有一个，能见此实相的正量 (佛陀自然本智) 也只有一个。佛与众生的差别仅是觉与未觉。

### 四法依 (量的实践阶梯)

| 法依 | 内容 | 对治 |
|---|---|---|
| 依法不依人 | 选正法 | 邪师 / 名声 |
| 依义不依句 | 取所诠义 | 文字相 |
| 依了义不依不了义 | 以四秘密四意趣区分 | 方便说混同了义 |
| 依智不依识 | 依无二智慧 | 二取分别识 |

四秘密：令入秘密 / 相秘密 / 对治秘密 / 转变秘密
四意趣：平等意趣 / 别义意趣 / 别时意趣 / 补特伽罗意乐意趣

是处理佛经表面矛盾的方法论工具。

详见：`collections/解义慧剑/四法依.md`

### 八辩才 (量论修持的果)

正念藏 → 智慧藏 → 了悟藏 → 总持藏 → 辩才藏 → 正法藏 → 菩提心藏 → 修行藏

最终指向「量士夫」(佛) — 「量因佛陀语，依量成立故」— 形成认识论闭环：佛语是量 → 以量验证佛语 → 生起对量士夫的不退信心。

## Correctness Anchors

### 量必含净见量层

常见误解：因明的量就是凡夫名言量 (相当于格鲁派《因明七论》框架)。

正确立场 (麦彭仁波切《解义慧剑》)：量分四层 — 观现世量、净见量、相似胜义、真实胜义。**净见量不可缺** — 缺则圣者所见成虚妄，「诸根是量，圣道复益谁？」

### 自证现量的传统差异

宁玛 / 萨迦 / 自续派：名言中承许自证现量。
应成派 (月称) ：名言中亦不许自证。

此差异不是「谁对谁错」的简单判定 — 是不同观察层次的取舍。详见 `topics/madhyamaka/debates.md`。

### 量观待成立 — 不可执量为实有

量本身是缘起法 — 观待主体认识层次、所量客体业感而成立。任何执「量是独立自主成立」的立场，自身即陷入证违 (因为量若实有，则胜义量应破之)。

详见 `collections/定解宝灯论/Q6` 量之观待论证。

### 因明是抵达无分别智的舟筏

因明 (分别识层面) 是修证的工具，不是终点。四法依的最末「依智不依识」明确指向：在了义法中应依无二智慧而非二取心识。任何「因明就是究竟正见」的立场即违四法依。

### 量士夫 = 佛

量论的终极成立处是佛语；佛语的成立又依量。这是认识论闭环，不是循环论证 — 详见 `collections/解义慧剑/证成理·比量与量之四分.md §一致摄义` + `八辩才与衡量之果.md`。
